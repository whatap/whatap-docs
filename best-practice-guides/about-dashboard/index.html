대시보드 분석

학습 자료는 계속 업데이트해 제공할 예정입니다.

애플리케이션 모니터링​

애플리케이션 대시보드 살펴보기​

와탭의 애플리케이션 대시보드 메뉴의 차트형 위젯에 대한 분석 방법을 다음과 같은 순서로 안내합니다.

  • 애플리케이션 연결 상태 확인하기

  • 트랜잭션 분석하기

  • 사용자 및 서비스, 리소스 분석

히트맵 트랜잭션 살펴보기​

와탭의 히트맵 트랜잭션 메뉴를 활용해 서비스의 장애 현황을 파악하는 방법을 다음과 같은 순서로 안내합니다.

  • 패턴 분석

  • 구간 및 스택 분석

  • 호출 관계 분석

  • 트랜잭션 로그 분석

서버 모니터링​

리소스 보드 살펴보기​

와탭의 리소스 보드 메뉴의 차트형 위젯에 대한 분석 방법을 다음과 같은 순서로 안내합니다.

  • 서버 모니터링 핵심 지표

  • 서버 상태 요약

  • 시스템 운영 통계

  • 조기 경보 및 알림

브라우저(Browser) 모니터링​

브라우저 모니터링 활용하기​

와탭의 브라우저 모니터링을 활용하는 방법을 다음과 같은 순서로 안내합니다.

  • 페이지 로드 분석하기

  • AJAX 모니터링 및 분석

  • 사용자 접속 환경 분서

  • 에러 분석하기

애플리케이션 대시보드

애플리케이션 성능 모니터링 기능 중 하나인 애플리케이션 대시보드 메뉴에서 차트형 위젯의 분석 방법을 소개합니다.

히트맵 트랜잭션

히트맵 트랜잭션 메뉴를 활용해 웹 애플리케이션 서버(Web Application Server)의 문제를 파악하고 장애를 대응하는 방법을 소개합니다.

서버 리소스 보드

서버 성능 모니터링 기능 중 하나인 리소스 보드 메뉴에서 차트형 위젯 분석 방법을 소개합니다.

브라우저 모니터링

Browser 모니터링 서비스를 이용해 여러분의 웹 애플리케이션의 문제점을 빠르게 파악하고 대응하는 방법에 대한 안내 문서입니다.
