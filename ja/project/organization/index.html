組織管理

組織は、グループをグループ化できる親グループです。ほとんどの場合、プロジェクトとグループで十分です。運用管理サービスを専門とする企業であれば、複数の会社がグループとして存在します。グループやプロジェクトの数が非常に多くなる可能性があります。この場合、グループの概念だけでは不十分です。グループを統合管理できるコンセプトが必要です。組織の概念を使用します。

専門的な運用管理サービス企業は、複数のサブグループを1つの組織にグループ化することで管理できます。各グループメンバーに権限を付与してみてください。権限を付与されたメンバーは、個別にグループを管理できます。

組織生成​

  1. 画面右上のプロフィールアイコンを選択すると、ドロップダウンメニューが表示されます。

  2. 組織生成を選択します。

  3. 組織名とロゴ、詳細を入力して次へボタンを選択します。

  4. グループ一覧から組織に追加するグループを選択し、完了ボタンをクリックします。

問題なく組織の作成ができましたら、'組織を正常に生成しました。'メッセージが表示されます。

ノート

  • 組織名は必須入力です。ロゴと詳細は、入力せずに組織を作成できます。

  • 修正権限を持つグループのみ、組織に追加できます。

  • 修正権限がない場合はそのグループを選択できません。

  • グループを選択する必要なく組織を作成できます。

  • 修正権限のないグループに招待された場合、そのグループに対する修正権限はありません。

  • 親グループから削除したり、他のグループに移動したりすることはできません。

  • 組織ロゴのイメージの推奨サイズは、幅102ピクセル、高さ102ピクセルです。

  • 組織ロゴのイメージは、3MB以下の_png_、jpg、_jpeg_ファイルの拡張子のみ可能です。

  • メンバー権限の詳細については、次の文書を参照してください。

組織に属しているグループやプロジェクトを確認する​

ホーム画面の左上の非組織を選択します。組織一覧が表示されたら組織を選択します。プロジェクト全体メニューの下位で組織に属するグループを確認できるし、プロジェクトリストでは、組織に属するすべて全体グループのプロジェクトリストを確認できます。

組織管理​

  1. ホーム画面で左上の非組織を選択します。

  2. 組織の一覧が表示されたら、管理したい組織名にマウスポインターをかざします。

  3. ボタンが表示されたら、ボタンを選択します。

管理メニューに移動します。

  • 組織名領域

    ユーザーが所属している組織の一覧を確認するには、の組織名領域を選択します。組織リストが表示されると別の組織を選択するか、非組織リストに戻るを選択してプロジェクトリストに戻ることができます。

  • 組織リスト

    • 自分が属するすべての組織がパネルリストで表示されます。組織を変更する場合は、 移動を選択します。

    • 組織名とロゴ、説明を編集するには、ボタンをクリックします。

    • 組織を削除するには、ボタンをクリックします。所属するグループとプロジェクトは、非組織のグループに移動します。

    • 修正権限がない場合は、修正・削除ボタンが表示されません。

  • グループリスト

    現在選択されている組織に所属するすべてのグループが一覧として表示されます。ボタンをクリックすると、選択したグループの管理画面に移動できます。

  • 組織修正

    選択している組織の基本情報を修正するには、編集アイコンボタンを選択します。組織修正画面が表示されたら、組織名、ロゴ、説明などを修正します。修正を完了した後、保存ボタンを選択します。

組織メンバーを招待する​

  1. 管理メニューに移動します。

  2. サイドメニューでメンバーリストを選択します。

  3. 画面の右上にあるメンバー招待ボタンを選択します。

  4. 組織項目からサブグループやプロジェクト単位で選択し、メンバーを追加できます。

  5. ユーザー権限項目から招待するユーザーの権限を設定した後、適用ボタンを選択します。

  6. メール入力欄に招待するユーザーのメールアドレスを入力した後、追加ボタンを選択します。他のユーザーをさらに追加できます。

  7. 招待するユーザーをすべて追加して、完了ボタンを選択します。

ノート

  • メンバー招待機能は、メンバー管理権限のあるユーザーのみが利用できます。 メンバー権限の詳細については、次の文書を参照してください メンバー権限の詳細については、次の文書を参照してください。

  • WhaTapモニタリングサービスに登録していないアカウントには、新規アカウントタグを表示し、会員登録メールを送信します。

  • メンバーリストのEメールの右側には、MFA適用及びSSO連動可否によってMFAまたはSSOタグを表示します。

  • 招待を完了する前にメンバー招待をキャンセルする場合は、招待メンバーリストからボタンを選択します。

  • 形式が正しくないメールアドレスを入力する場合は、招待されません。

  • すでに招待されたメンバーは追加で招待できません。上位の組織やグループのメンバーは自動的に追加されます。

組織メンバーを確認する​

組織メニューからメンバーリストを選択します。組織のツリーでグループまたはプロジェクトを選択できます。選択したグループ、プロジェクトごとに所属するメンバーを確認できます。選択したグループ、プロジェクトごとにメンバーの追加や削除、権限変更ができます。

組織メンバーの権限を修正する​

  1. 管理メニューに移動します。

  2. サイドメニューでメンバーリストを選択します。

  3. 組織メンバー一覧から権限を変更するメンバーの右側にあるボタンを選択します。

  4. メンバー権限を修正する画面が表示されたら、メンバー権限と組織のグループ毎ユーザー権限を設定します。

    • メンバー権限：該当組織と組織のグループで許可する権限を設定できます。
    • 組織のグループ毎ユーザー権限：選択したメンバーの権限を組織に属するグループ別に設定できます。

  5. すべての設定が完了したら、適用ボタンを選択します。

ノート

  • 組織ツリー構造でグループまたはプロジェクトを選択し、グループ別、プロジェクト別に所属するメンバーの権限を修正できます。選択したグループまたはプロジェクトによって適用できる権限の範囲が異なる場合があります。

  • メンバー権限の修正機能はメンバー管理権限のあるユーザーのみ利用できます。メンバー権限の詳細は次の文書を参照してください。

組織メンバーを削除する​

  1. 管理メニューに移動します。

  2. サイドメニューでメンバーリストを選択します。

  3. 組織メンバー一覧から削除するメンバーの右側にあるボタンを選択します。

  4. メッセージが表示されたら、削除ボタンを選択します。

ノート

  • 下位グループまたは下位プロジェクトでもメンバーを削除する場合は、メッセージ画面からチェックボックスをクリックし、削除ボタンを選択します。

  • 組織ツリー構造でグループまたはプロジェクトを選択し、グループ別、プロジェクト別に所属するメンバーを削除できます。

  • メンバー管理権限がない場合、他のメンバーを削除できません。メンバー権限の詳細は次の文書を参照してください。

メンバーリストをダウンロードする​

メンバーリストをCSV形式のファイルにダウンロードする場合、次の手順に従います。

  1. 管理メニューに移動します。

  2. サイドメニューからメンバーリストを選択します。

  3. 画面右上のボタンを選択します。

メンバーリストをCSVファイル形式でユーザーのローカルディスクに保存します。

ノート

メンバーリストのダウンロード機能はメンバー管理権限のあるユーザーのみ利用できます。メンバー権限の詳細は次の文書を参照してください。
