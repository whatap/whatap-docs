統合管理

統合グループ管理​

ホーム画面 > 組織選択 > 管理 > 統合グループ管理

選択した組織のグループ一覧を表示できます。組織の修正権限がない場合、その機能は無効になります。

  • 組織を選択していない場合は、組織以外のグループとプロジェクトの一覧が表示されます。グループまたはプロジェクトを選択して、他の組織またはグループに移動できます。
  • グループまたはプロジェクトの修正権限がない場合、一覧は無効になります。
ノート
  • 所有していない組織やグループに、グループまたはプロジェクトを移動するには、その組織やグループに招待するか、WhaTapサポートチーム(help@whatap.io)にお問い合わせください。

グループを追加する​

  1. 統合グループ管理メニューから＋グループ追加を選択してください。

  2. グループ追加ウィンドウが表示されたら、グループ名、説明を順番に入力してください。グループ名は必須入力です。

  3. プロジェクト追加項目でグループに追加するプロジェクトを選択してください。

  4. 適用ボタンを選択します。

選択したプロジェクトをグループに追加します。

ノート

プロジェクト追加項目でテキストを入力し、グループに追加するプロジェクトをフィルタリングすることができます。 さらに、複数のプロジェクトを選択できます。

グループを移動する(組織からグループを削除)​

選択したグループを別の組織に移動したり、組織から削除したりできます。

  1. 別の組織に移動するグループを選択してください。複数のグループを選択できます。

  2. グループ移動ボタンが有効になっている場合は、グループ移動ボタンを選択してください。

  3. グループ移動ウィンドウが表示されたら、組織選択項目をクリックして移動する組織を選択してください。

  4. 確認ボタンを選択してください。

ノート
  • 修正権限のないグループは、グループ選択項目の一覧に表示されません。
  • 組織からグループを削除するには、組織選択項目で組織に属していないグループを選択してください。

プロジェクトを移動する(グループからプロジェクトを削除)​

選択したプロジェクトを他のグループに移動したり、グループから削除したりできます。

  1. 他のグループに移動するプロジェクトを選択してください。複数のプロジェクトを選択できます。

  2. プロジェクトの移動ボタンが有効になっている場合、プロジェクト移動ボタンを選択してください。

  3. プロジェクト移動ウィンドウが表示されたら、グループ選択項目をクリックして移動するグループを選択してください。

  4. 確認ボタンを選択してください。

ノート
  • 修正権限のないグループは、グループ選択項目の一覧に表示されません。
  • グループからプロジェクトを削除するには、グループ選択項目でグループに属していないプロジェクトを選択してください。

グループを修正する​

  1. グループ名と説明を修正したいグループにカーソルを合わせると、修正アイコンボタンが表示されます。

  2. ボタンを選択してください。

  3. グループ修正ウィンドウが表示され、修正を完了した後、適用ボタンを選択してください。

グループまたはプロジェクトを削除する​

  1. 削除するグループまたはプロジェクトにカーソルを合わせると、ボタンが表示されます。

  2. ボタンを選択してください。

  3. グループ削除またはプロジェクト削除ウィンドウが表示されると、メッセージを確認して確認ボタンを選択します。

注意
  • グループを削除する場合は、グループとユーザー一覧情報の両方が削除されます。グループに属していたプロジェクトは、グループに属していないプロジェクトに移動します。
  • プロジェクトを削除する場合は、ユーザー一覧情報と収集したデータが削除されます。削除したデータは復元できません。

統合メンバー管理​

ホーム画面 > 組織選択 > 管理 > 統合メンバー管理

選択した組織、グループ、プロジェクト別のメンバー一覧を確認、追加または削除できます。組織に修正権限がない場合、その機能は無効になります。

  • 組織を選択していない場合は、組織以外のグループとプロジェクトの一覧が表示されます。グループまたはプロジェクトを選択して、他の組織またはグループに移動できます。
  • グループまたはプロジェクトの修正権限がない場合、メンバーを追加または削除することはできません。
ノート

所有していない組織やグループのメンバーを確認するには、その組織やグループに招待するか、WhaTapサポートチーム(help@whatap.io)にお問い合わせください。

メンバーを招待する​

  1. 統合メンバー管理メニューから組織またはグループ、プロジェクトを選択してください。

  2. 画面右上のメンバー招待ボタンを選択してください。

  3. ユーザー権限項目で招待するユーザーの権限を設定した後、適用ボタンを選択してください。

  4. 電子メール項目で招待するユーザーの電子メールを入力し、追加ボタンを選択してください。他のユーザーを追加することができます。

  5. 招待するユーザーをすべて追加して、完了ボタンを選択してください。

ノート

招待を完了する前にキャンセルする招待メンバーがいる場合は、招待メンバー一覧でボタンを選択してください。

メンバーの権限を修正する​

  1. 統合メンバー管理メニューから組織またはグループ、プロジェクトを選択してください。

  2. メンバー一覧で権限を修正したいメンバーの右側のボタンを選択してください。

  3. メンバー権限の修正ウィンドウが表示されると、メンバー権限と組織のグループ別のユーザー権限またはグループのプロジェクト別のユーザー権限を設定してください。

    • メンバー権限：組織と組織に所属するグループが許可する権限を設定できます。
    • 組織を選択した場合) 組織のグループ別のユーザー権限：選択したメンバーの権限を組織に属するグループ別に設定することができます。
    • グループを選択した場合) グループのプロジェクト別のユーザー権限：選択したメンバーの権限をグループに属するプロジェクト別に設定することができます。

  4. すべての設定が完了したら、適用ボタンを選択してください。

ノート
  • 選択した組織、グループまたはプロジェクトによって適用できる権限の範囲は異なる場合があります。
  • メンバー権限の詳細については、次の文書を参照してください。

メンバーを削除する​

  1. 統合メンバー管理メニューから組織またはグループ、プロジェクトを選択してください。

  2. メンバー一覧から削除するメンバーの右側のボタンを選択してください。

  3. メッセージウィンドウが表示されたら、削除ボタンを選択してください。

ノート
  • 組織を選択した場合、下位グループおよびプロジェクトでもメンバーを削除するには、メッセージウィンドウのチェックボックスをクリックし、削除ボタンを選択してください。
  • グループを選択した場合、下位プロジェクトからメンバーを削除する場合は、メッセージウィンドウのチェックボックスをクリックし、削除ボタンを選択してください。
