Preview 1.107.X

https://preview.whatap.ioで、次の新機能と変更した内容を事前に体験することができます。

Preview 1.107.17​

2023年8月21日

共通​

ダッシュボード​

トランザクション分析ウィンドウで次の内容を修正します。

  • Changeレコード要約タブの表示データに原本URL、URL/TX追加
  • Changeトランザクション一覧のカラム設定に原本URL、URL/TX追加
  • Changeテーブルビュー、ツリービュータブのトランザクション要約情報にクライアントIP追加
  • Changeレコード要約タブでエージェント名クリックすると環境変数メニューの該当エージェント情報の表示
  • Changeテーブルビュータブのカラムの英訳を修正

    • Memory Accumulation (byte) → MemAllocAcc (byte)

    • CPU Accumulation (ms) → CpuTimeAcc (ms)

  • Changeカラム選択機能のUI のスタイルを修正

Application​

  • Fixed統計/レポート>統計>トランザクションメニューでデータ照会範囲が5分以下の場合、照会されるデータの検索範囲を5秒から5分に修正
  • Fixedアプリケーション>インスタンスパフォーマンス分析メニューのエージェント状態タブで空のデータに対する例外処理を追加

Preview 1.107.16​

2023年8月21日

共通​

ダッシュボード​

Fixedトランザクション分析ウィンドウのトランザクション 一覧からエージェント名カラムが表示されないバグを修正

ネットワーク​

Fixedプロジェクト作成およびプロジェクトページでNPM商品のヘッダーにBETAタグを露出するよう修正

Preview 1.107.15​

2023年8月17日

Server​

  • Fixedダッシュボード>リソースボードメニューからエージェントステータスウィジェットに危険、アラートステータスが区分されて表示されるよう修正
  • Fixedサーバーリストメニューからディスク、ネットワークフィルタで文字入力時に画面点滅現象の改善
  • Fixedサーバーリストメニューからディスク、ネットワークフィルタ値を入力した後、リフレッシュ時にフィルタが適用されないバグを修正

Database​

MySQL​

Change管理>エージェントのインストールメニューのWhaTapデータベースエージェントのダウンロードセクションでwhatap.confファイルの作成例を追加

ネットワーク​

Fixed分析>TCPセッション/UDPセッションのセッション情報テーブルでForeignProcessType、ForeginAppNameカラムの一部の値が0で表示される問題の修正

Preview 1.107.14​

2023年8月17日

ネットワーク​

Newネットワークパフォーマンスモニタリング(Network Performance Monitoring)サービスをオープンします。 ご利用いただき誠にありがとうございます! 該当サービスはhttps://preview.whatap.ioで利用できます。 詳細については、次の文書を参照してください。

Database​

インスタンスV1​

Fixedウィジェットカウントの選択モーダルのチャートサイズが異常に現れる現象を修正

V2共通​

  • Changeテーブルカラム選択モーダルUI変更
  • Changeプロジェクトメンバーの中で修正権限のあるメンバーだけセッション中止ボタンが露出するように変更
  • Featureラインチャートにデータ最大値を表示を適用

マルチインスタンスV2​

  • Featureウィジェット別のメトリクス情報を示すインフォアイコンの追加
  • Feature製品環境別(RDS、XOS)タブ分離
  • ChangeXOS Disk Usageウィジェット内のテーブルのカラムのうちMount Source→Mount Pointに変更(最新エージェントのアップデートが必要)

データベースサイズV2​

FeatureCSVダウンロード追加

PG SQL統計​

Feature照会基準選択オプションの追加(temp blocks read、temp blocks written)

Log​

  • Fixed対数検索機メニューからpcodeタグをクリックするとドロップダウンメニューが露出しないように修正
  • Featureログ設定メニューからカテゴリおよびログ検出条件の選択UIの改善

Preview 1.107.13​

2023年8月11日

FixedFlexボードメニューのSeriesウィジェットオブジェクトの結合で最大、最小値が作動しない現象を修正

Cubrid​

  • whatap.confファイル内のインストールコマンド

  • ガイド文書リンクの追加

Preview 1.107.07​

2023年8月8日

プロジェクトリスト​

Fixedプロジェクトリストで拡張カードビューのときに表示されるアプリケーションテキストの日本語が短くなる現象を修正するために用語修正(アプリケーション → アプリ)

ダッシュボード​

トランザクション分析ウィンドウで次の内容を修正します。

  • FixedFlexボードのフィルター機能を活性化するとトランザクション分析画面が表示されないバグを修正
  • FixedテーブルタブのSQLステップ詳細を見るモダルでSQL変数復号化が作動しないバグを修正
  • Changeトランザクション一覧のNoカラム項目の設定を修正できないように変更

Log​

  • Feature隣接ログで基準となるログにスクロール位置移動およびハイライト機能を追加
  • Changeアラート通知>イベント設定>ログメニューからリアルタイムログイベント 追加生成時検索キーからすべて(all)項目を除去

Preview 1.107.06​

2023年8月7日

共通​

ダッシュボード​
