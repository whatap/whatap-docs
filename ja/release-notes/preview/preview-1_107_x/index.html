Preview 1.107.X

ノート

preview.whatap.ioで、次の新機能と変更した内容を事前に体験できます。

Preview 1.107.17​

2023年8月21日

共通​

ダッシューボード​

トランザクション分析ウィンドウで次の内容を修正します。

  • Change レコード要約タブの表示データに原本URL、URL/TX追加

  • Change トランザクション一覧のカラム設定に原本URL、URL/TX追加

  • Change テーブルビュー、ツリービュータブのトランザクション要約情報にクライアントIP追加

  • Change レコード要約タブでエージェント名クリックすると環境変数メニューの該当エージェント情報の表示

  • Change テーブルビュータブのカラムの英訳を修正

    • Memory Accumulation (byte) → MemAllocAcc (byte)

    • CPU Accumulation (ms) → CpuTimeAcc (ms)

  • Change カラム選択機能のUI のスタイルを修正

Application​

  • Fixed 統計/レポート>統計>トランザクションメニューでデータ照会範囲が5分以下の場合、照会されるデータの検索範囲を5秒から5分に修正

  • Fixed アプリケーション>インスタンスパフォーマンス分析メニューのエージェント状態タブで空のデータに対する例外処理を追加

Preview 1.107.16​

2023年8月21日

共通​

ダッシューボード​

Fixed トランザクション分析画面のトランザクション 一覧からエージェント名カラムが表示されないバグを修正

Network​

Fixed プロジェクト作成およびプロジェクトページでNPM商品のヘッダーにBETAタグを露出するよう修正

Preview 1.107.15​

2023年8月17日

  • Fixed ダッシュボード>リソースボードメニューからエージェントステータスウィジェットに危険、アラートステータスが区分されて表示されるよう修正

  • Fixedサーバーリストメニューからディスク、ネットワークフィルタで文字入力時に画面点滅現象の改善

  • Fixedサーバーリストメニューからディスク、ネットワークフィルタ値を入力した後、リフレッシュ時にフィルタが適用されないバグを修正

Database​
