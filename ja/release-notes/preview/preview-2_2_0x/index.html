Preview 2.2.0.a

ノート

preview.whatap.ioで、次の新機能と変更した内容を事前に体験できます。

Preview 2.2.0.a6​

2024年02月19日

共通​

  • Changed アカウント管理メニューのマルチファクター認証セクションでOTPキーを発行していない状態であればメールでOTPキーを送信するボタンを非活性化

  • Changedプロジェクト作成時、製品のおすすめバナーにNetwork Performance Monitoring追加、Java製品を除去

  • Deprecated 言語設定が英語または日本語の場合、アカウント情報管理 メニューからマーケティングオプションを削除

Application​

  • Changed ダッシュボード > アプリケーションダッシュボードメニューで本日TPS及び本日ユーザーウィジェットにAll APPS.タグを追加

    ノート

    All APPS.タグ付きウィジェットは、常にすべてのエージェントのデータを表示するように固定されています。

  • Feature ダッシュボード > TXマップメニューに新しいUIを適用し、機能を変更

    • データ照会区間の拡張：リアルタイムで最大30分、過去の時点で最大1日
    • トレース分析ウィンドウで最大照会可能データ数の増加(100件→1000件)
    • トランザクションとエラー数をリアルタイムで表示
    • Top 30リストでキーボードイベントを追加
    • チャートエリアにキーボードイベントを追加
    • 過去時点のデータを照会する場合、サーバーから受信するデータフィルタリングが可能
    ノート

    TXマップメニューの改編に関する詳細は、次のリンクを参考にしてください。

Database​

共通​

  • Feature XOS2指標の代替に関する案内事項を追加

  • Fixed アラート通知 > アラート設定 > データベースタブでイベント状態が解決されたら追加で通知する機能に対するツールチップメッセージを修正(エージェントバージョン1.6.22以降 → エージェントバージョン1.6.30以降)

MySQL V2​

  • Feature JSON形式でのPlan照会機能を追加

  • Feature 統計/レポート > データベースサイズメニューのテーブルサイズ(トップ50)テーブルでTABLENAMEカラム項目をクリックするとcolumn、index構成を確認できる機能を追加

Kubernetes​

  • Fixedホーム画面> Flexボードメニューでコンテナperformnaceテンプレート選択時に発生するホワイトアウト問題を修正

  • Fixed ダッシュボード > コンテナマップメニューでノードブロックを選択した後、詳細ボタンを選択すると発生するホワイトアウト問題を修正

Log​

Fixed アラート通知 > アラート設定 > ログメニューで複合ログイベントの保存に失敗する問題を修正

Preview 2.2.0.a5​

2024年02月15日

共通​

  • Featureプロフィールアイコン選択> アカウント管理メニューのマルチファクター認証セクションでOTP認証手段を紛失した場合に備えて復旧コード機能を追加

  • FeatureダッシュボードのApdex*100ウィジェットに複数プロジェクトのマージメトリクス項目を追加

Application​

Changed ダッシュボード > アプリケーションダッシュボードメニューでチャートエリアのデザイン要素を修正(ウィジェットタイトルのサイズとウィジェット間隔、チャート区間の説明テキストのサイズを修正)

Database​

共通​

  • Feature 管理 > エージェントリストメニューでデータベースサーバーのIPアドレスを表記するDB IPアドレスカラムを追加

  • Fixed 管理 > エージェントリストメニューでステータスアイコンの色を変更、名前(初期), DB IPアドレスカラムのテキストがカラム幅より長い場合、マウスオーバー時にツールチップで確認できるように修正

  • Fixed アラート通知 > アラート設定 > データベースタブでイベント状態が解決されたら追加で通知する機能に対するツールチップメッセージを追加(エージェントバージョン1.6.22以降に対応)

Database V2​

  • Changed ダッシュボード > インスタンスメニューで基本プリセット(Default(rds)、Default(xos)) を追加

PostgreSQL V2​

Fixed 分析 > Topオブジェクトメニューでtable_size及びindex_size、bloat_sizeカラムの値を正常に出力するように修正

Kubernetes​

  • Change ワークロード > Object Manifestメニューで比較する機能のUIを修正し、目次機能を追加

  • Change ダッシュボード > コンテナマップメニューでブロックサイズが小さい場合、UI表示を修正

  • Changed コンテナー > コンテナイメージメニューのカラム項目を変更

    • 追加：Registry、Name、Tag
    • 除去：Image Name、Image Tag

  • Fixed 管理 > エージェントインストールメニューでPHP、Goアプリケーションエージェントのインストール過程中にエージェント実行のために一部のコマンドを修正

  • Fixed 管理 > Kubernetesエージェントリストメニューで非活性化されたエージェントのデータが出力されない問題を修正

  • Feature 管理 > Kubernetesエージェントリストメニューでノード別エージェントのバージョンが異なる場合、画面上にバージョンの統一を薦める案内メッセージを追加

Browser​

  • Featureページロードヒットマップデータにネットワークステータスに関する指標を追加：有効接続タイプ、RTT、実効帯域幅

    • ダッシュボード > ブラウザ監視ダッシュボードメニューのPage Load Hitmapウィジェット
    • 分析 > Page Load Hitmap

  • Feature 分析 > Page Load Hitmapメニューのページロード 詳細ウィンドウでリソースデータに指標を追加：状態コード、Delivery Type

  • Feature AJAXヒットマップデータにネットワークステータスに関する指標を追加：有効接続タイプ、RTT、実効帯域幅、Delivery Type

    • 分析 > AJAX Hitmap

    • ダッシュボード > ブラウザ監視ダッシュボードメニューのAJAX Hitmapウィジェット

ノート

追加された指標の詳細については、次のリンクを参考にしてください。

  • 有効接続タイプ
  • RTT
  • 実効帯域幅
  • Delivery Type
  • 状態コード

  • Fixed メトリックスチャートメニューでNetwork Traffic関連チャートに表示されるbyte単位をbit単位で表示されるように修正
