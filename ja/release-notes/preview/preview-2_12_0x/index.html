Preview 2.12.0.a

ノート

preview.whatap.ioで、次の新機能と変更した内容を事前に体験できます。

Preview 2.12.0.a4​

2024年12月17日

共通​

  • Feature Flexボードまたは統合Flexボードメニューのアラートリストウィジェットに次の機能を追加

    • 解除されたイベントを見るオプションを追加

    • ダッシュボードに設定された時間とは別に照会時間を選択できるオプションを追加

  • Fixed アカウント管理メニューでアラート受信言語設定が'ja'と表示される問題を修正

  • Fixed一部のメニューで画面上部のタイトルが表示されない問題を修正

Application​

Feature アラート通知 > イベント設定 Newメニューでアプリケーションのエージェント重複イベントオプションを追加

Server​

  • Feature サーバーリスト > サーバー インベントリメニューでCSVファイルをアップロードしてサーバー一覧のユーザー定義コラムを交換する機能を追加

  • Changed ダッシュボード > サーバーインベントリマップメニューでグループ化オプション選択ツールの初期化方法を変更

    Clearボタンクリック方式から一覧で設定なし項目を選択する方式に変更

  • Changed サーバーリスト > サーバー インベントリメニューのフィルターオプションをサーバーインベントリマップメニューのフィルターオプションと同じUIに交替

  • Fixed サーバーリスト > サーバー 詳細メニューのエージェント名カラムに特定のオペレーティング システムのアイコンが表示されない問題を修正

Kubernetes​

Fixed ダッシュボード > 外部コールステータスメニューでリアルタイムモードを一時停止した場合でも、リアルタイムでデータを照会する問題を修正

Database​

共通​

  • Feature次のメニューパスのアクティブセッションテーブルにページネーションを適用

    • ダッシュボード > インスタンスモニタリング
    • ダッシュボード > マルチインスタンス

  • Fixed次のメニューパスのアクティブセッションテーブルに1分の範囲を超えるデータが表示される問題を修正

    • ダッシュボード > インスタンスモニタリング
    • 分析 > カウント推移比較

Database V2​

  • FeatureAPMプロジェクトのトレース分析ウィンドウでコピーしたバインド変数の値を連携データベース情報ウィンドウで貼り付けるボタンを追加

    ノート

    適用プラットフォーム：PostgreSQL、Oracle、Oracle Pro、MySQL

  • Changed 連携データベース情報ウィンドウでインスタンスモニタリングメニューに移動する時、開始時間、終了時間、セッション時間をクエリパラメータとして含めるように変更

    ノート

    適用プラットフォーム：PostgreSQL、Oracle、Oracle Pro、MySQL

  • Fixed ダッシュボード > インスタンスモニタリングメニューで照会するデータがない場合は、ローディングインジケーターが継続する問題を修正し、空のデータに対する案内メッセージを追加

  • Fixed ダッシュボード > インスタンスモニタリングメニューにチャートデータはあるが、テーブルデータがない場合の最近照会された履歴はありません。メッセージを追加

  • Fixed ダッシュボード > インスタンスモニタリングメニューのアクティブセッションテーブルでインスタンス名がカラムの幅より長くなる場合、省略記号(...)を表示するように修正

  • Fixed ダッシュボード > インスタンスモニタリングメニューで過去のデータを照会する時セッション詳細ウィンドウを開くとFull Textが照会できない問題を修正

    ノート

    適用プラットフォーム：SQL Server

  • Fixed 分析 > カウント推移比較メニューのウィジェットのツールチップに[基準]/[比較]情報とインスタンス情報を追加

Oracle Pro​

  • Feature ダッシュボード > インスタンスモニタリングメニューのSQL詳細ウィンドウにアクティブセッション情報を追加

  • Feature 分析 > SQL分析メニューのフィルターオプションにobject_name条件を追加

  • Fixed 管理 > エージェントインストールメニューのアカウント生成段階でヒドゥンパラメータの照会権限を付与するコマンドが誤って表記された内容を修正

MySQL​

Fixed 統計/レポート > MYSQL SQL統計メニューのSQL詳細ウィンドウでhash値がundefinedと表記される問題を修正

Browser​

Featureリソース別の性能データを視覚的に提供し、ロード時間、リソースサイズ、タイプ別の性能推移などを分析できるリソース負荷統計メニューを追加

ノート

この機能は、統計 > リソース負荷統計メニューで確認できます。

Preview 2.12.0.a3​

2024年12月12日

共通​

  • Feature個別メニューでタイムセレクターのコンポーネントを交替

    • キーボードを使用して日付と時刻を変更できます。

    • 日付及び時刻形式のテキストをコピーして貼り付ける方法で変更できます。

    ノート

    タイムセレクターの利用方法の詳細については次の文書を参照してください。

  • Featureタイムセレクターのカレンダーポップオーバーの下部にガイド文書に移動できるリンクを追加

  • Feature アラート通知 > 通知設定メニューで通知メッセージをAIを通じてユーザーフレンドリーな内容で送信できるSmartメッセージ(AI)オプションを追加

  • Feature アラート通知 > イベント設定 Newメニューで解消された通知オプションに関する説明を追加

  • Changed アラート通知 > イベント設定 > メトリックスタブのイベント追加・修正の画面で、イベント発生条件オプションの選択入力を選択した時、すべてのフィールドに${ }を適用するように変更

    ノート

    • ただ、フィールド名に{ または }が含まれる場合は適用しません。

    • イベントを新たに追加・修正する場合に限って適用します。

Application​

  • Changed .NET 4.6.1バージョン未満またはWindows 2008環境を使用中の.NETプラットフォームに対応するエージェントインストールファイルのダウンロード及び関連メッセージを追加

  • Fixedトランザクションの開始/終了時間の単位値が異なる場合トレース分析ウィンドウで発生する次の問題を修正

    例えば、トランザクションが17時59分に始まって 18時00分に終了する時に発生する問題です。

    • テーブルビュー > SQL詳細モーダルでパラメータバインディングができない問題を修正

    • テーブルビュー > アクティブスタックモーダルでアクティブスタックが見えない問題を修正

Server​

  • Changed 管理 > エージェントインストール > HP-UXでIA64カーネルの 11.31 バージョンに対応するエージェントのダウンロードボタンを追加

  • Fixed ダッシュボード > サーバーインベントリマップメニューのグループ要約パネルでサーバーアップタイムのチャートが断続的にクリックされない問題を修正

  • Fixed ダッシュボード > サーバーインベントリマップメニューでタイムセレクターのコンポーネントを交替した後、過去の時点を選択できるバグを修正

  • Fixed サーバーリスト > サーバーリストメニューでタイムセレクターのリアルタイムモードを一時停止すると時間がNaNで表示される問題と過去の時点を選択できるというバグを修正

  • Fixed サーバーリスト > サーバーリストメニューのカラム設定で初期化ボタンをクリックすると、全体のカラム選択が解除される問題を修正

Kubernetes​

  • Feature次のメニューパスの詳細情報で提供するノード 構成タブのCPU、メモリ、ディスクのtotal値を追加

    • ダッシュボード > ノードマップ
    • クラスター > ノードリスト
    • クラスター > ノードタイムライン

  • ChangedKubernetesイベントデータ(#K8sEvent)はログ個人情報の非識別化処理ができないように機能を修正

Database​

Database V2​

  • Feature ダッシュボード > マルチインスタンスメニューのカスタムウィジェットに選択した指標の単位を表示するように追加

  • Fixed ダッシュボード > インスタンスモニタリングメニューでウィジェットのY軸が変更した指標に合わせて変更されない問題を修正

  • Fixed ダッシュボード > マルチインスタンスメニューでイコライザーチャート形式のユーザー定義ウィジェットに表示されたインスタンスの順序を画面上部のインスタンス一覧の順序と一致するように修正

  • Fixed ダッシュボード > マルチインスタンスメニューのユーザー定義ウィジェットでクラウド指標のデータが出力されない問題を修正

Oracle、Oracle Pro​

Changed ダッシュボード > インスタンスモニタリングメニューでTOP STAT, TOP EVENTウィンドウスタイルを修正し、テーブル内容を変更

Log​

Changed次のメニュー パスでインデックスされない検索キー(.p, .piiなど)を含むログのタグをクリックして表示されるドロップダウンメニューから検索、除外検索項目を除去

  • ログ > ログTail
  • ログ > ログ検索

AWS Log​

  • Changed 管理 > エージェントインストールメニューのAWS Region選択一覧にジャカルタリージョンを追加

  • Fixed 管理 > エージェントインストールメニューでWhatTap Forwarderインストールページ ボタンのリンクパラメータエラーを修正

Preview 2.12.0.a2​

2024年12月05日

共通​

  • Feature アラート通知 > イベント設定メニューのイベント追加または修正画面で受信テスト機能を実行すると、イベント受信タグの選択事項を適用してテストするように修正

    ノート

    ウェブブラウザーで発生する通知は、イベントの受信タグとは関係ありません。

  • Feature 分析 > メトリクスチャート Newメニューで次の機能を追加

    • ダッシュボードのウィジェット凡例表示有無によって異なるUIを表示

      On: ウィジェットに凡例パネルが固定された形態

      Off: ウィジェットに凡例パネルがドロップダウンとして提供される形態

    • チャートラインが多い時、グラフのカラーがぼやけないように処理

Application​

  • Feature トレース分析ウィンドウのSQL要約、HTTPコール要約、メソッド要約、アクティブスタックタブでテキストダウンロード機能を追加

  • Feature トレース分析ウィンドウのテーブルビュータブでSQL詳細情報を確認する時、バインド変数コピー機能を追加

    ノート

    • この機能は連携データベース情報で使用できます。

    • パスワードが解除され、バインド変数の値が表示された時に使用できます。

Kubernetes​

  • Feature次のメニューでログから個人情報を照会します。権限のあるユーザーに個人情報の非識別化機能を追加:個人情報を表示・非表示にする機能

    • ダッシュボード > コンテナマップ > ログ > ログ検索
    • ダッシュボード > コンテナマップ > イベント
    • ダッシュボード > 外部コールステータス
    • クラスター > etcd ダッシュボード
    • クラスター > Kubernetesイベント

  • Changedネームスペースプロジェクトの管理 > プロジェクト管理メニューでプロジェクト名を修正できないように変更

  • Fixed クラスター > etcd ダッシュボードメニューでチャートインデックスに不要な情報値を削除する

Database​

共通​

  • Feature 実験室 > スクリプト·マネージャーメニューでユーザーがSQLを実行した場合ユーザー行為メニューで確認できる機能を追加

    ノート

  • Feature 管理 > エージェントアップデートメニューでユーザーがエージェントをアップデートするとユーザー行為メニューで確認できる機能を追加

    ノート

    この機能はOracle Proを除いたプラットフォームに適用されます。

  • ChangedデータベースV1のサポート期間を2024年12月31日から2025年6月26日に変更

  • Changed ダッシュボード > インスタンスモニタリングメニューでリアルタイムの照会範囲を変更すると、チャートに空白が生じる現象を修正

  • Changed 分析 > カウント推移比較メニューでアクティブセッションテーブルのタイトルを変更

  • Fixed XOSエージェントが収集するメモリサイズに関する指標の単位を変更

  • Fixed 統計/レポート > テーブルスペース容量増減メニューでセグメントサイズ(Top 50)セクションの単位表示のエラーを修正

  • Fixed ダッシュボード > インスタンスモニタリングメニューのアクティブセッションテーブルにセッション状態値が正常に出力されない問題を修正

Log​
