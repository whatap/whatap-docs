Preview 1.111.X

ノート

preview.whatap.ioで、次の新機能と変更した内容を事前に体験できます。

Preview 1.111.9​

2023年10月23日

共通​

User Interface​

  • Feature サービスカタログページでユーザー体験を向上するためのデザイン変更

  • Fixedユーザー言語が日本語の場合、サイドバーのサービス紹介ページのサービスタイトルを英語に変更

Flexボード​

Fixedリッチテキストウィジェットのテキストエディタから使用できない編集機能ボタンを削除

Container​

  • Featureノードエージェント1.3.7バージョンから新しい指標の追加と、UIデザインが改善されたノード一覧画面を提供

  • Fixedクラスタープロジェクトに対する照会権限がないメンバーの場合、ネームスペースプロジェクト画面上にプロジェクト名がプロジェクトコード(PCODE)で表示される現象を修正

  • Feature ダッシュボード>コンテナマップメニューから、ブロックカラー表示指標にアプリケーション関連フィールドも選択できるように追加

Browser​

  • Fixedユーザー言語が日本語の場合ダッシュボード>ブラウザ監視ダッシュボードメニューから日本語ウィジェットのタイトルを修正

  • Feature サイトマップ > ユーザー行為 メニューの追加

  • Feature 分析 > AJAX Hitmap メニューの追加

Preview 1.111.8​

2023年10月19日

共通​

  • Changed サイトマップ>ユーザー行為メニューのテーブルでRequest URI、 METHODカラムの削除

  • Fixed組織が選択された状態でプロジェクト作成時に、サービスカードをクリックすると選択された組織が解除される現象を修正

Application​

ダッシューボード​

  • Fixedトレース分析のSQL変数置換機能のバグを修正

  • Fixedトレース分析のマルチトランザクションビューに進入できないバグを修正

分析​

Fixedヒットマップメニューから検索ボタン連打防止のため、次の照会まで0.5秒の遅延を追加

統計/レポート​

Changed統計メニューからタブ移動時に、ユーザーが設定した検索時間を維持するように修正

Database​

共通​

  • Changedダッシュボード>インスタンスリストメニューから、テーブルがブラウザのサイズに合わせて表示されるように変更

  • Changed統計/レポート>SQL統計メニューのユーザビリティ向上のためのUI改善

    • 進捗バーの追加

    • テーブル領域にパネルを追加

    • テーブルサブチャートにChartIndexを追加

  • Featureダッシュボード>インスタンスモニタリングメニューから、非リアルタイムモードになると該当時間のSQL統計メニューへ移動できるボタンの追加

DB V2​

  • Changedマルチインスタンスモニタリングメニューから、アクティブセッションが漏れる問題を修正

  • FixedマルチインスタンスモニタリングメニューからSQL Elapse Mapウィジェットチャートの照会範囲の修正

MySQL​

Changed分析>データベースパラメータメニューのテーブルスクロールが、テーブル内部で作成されるように変更

PostgreSQL​

Fixed分析>Wait分析メニューから、アクティブセッションの詳細画面を通じてセッション履歴メニューへ移動する場合、Wait分析メニューの時間でセッション履歴を照会するように修正

SQL Server​

ChangedMS SQL Serverサービスの名前をSQL Serverに変更

Container​

  • Fixed管理>エージェントのインストールメニューからIstio環境の場合、追加リソースの作成に対してのyamlファイルのインデントを修正

  • Changed次のメニューからトレース分析機能を改善したバージョンに置き換え。変更点は次の文書を参照してください。

    • アプリケーション > マルチサービスヒットマップ

    • ダッシュボード > コンテナマップ

  • Fixed次のメニューから検索ボタン連打防止のため、次の照会まで0.5秒の遅延の追加

    • アプリケーション > ヒットマップ

    • アプリケーション > マルチサービスヒットマップ

Log​

  • Fixedログフィルター入力で値を修正する時大小文字の区分オプションが反映されない現象を修正

  • Fixedログ設定メニューからユーザーが選択した言語に合わせてログ設定ガイドを案内するように修正

Network​

Changedサイトマップメニューからユーザーの行為メニューリンクを表示するように修正

2023年10月6日

共通​

User Interface​

Fixedサーバーリスト>サーバーリストメニューから、エージェント検索時にエージェント一覧の背景色と実際の状態が一致しない問題の修正
