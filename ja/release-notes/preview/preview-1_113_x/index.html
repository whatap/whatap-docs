Preview 1.113.X

https://preview.whatap.ioで、次の新機能と変更した内容を事前に体験することができます。

Preview 1.113.11​

2023年11月21日

共通​

  • Feature管理>メンテナンス計画メニューの機能を改善

    • 新規メンテナンス計画メニューの追加、新機能のメンテナンス計画機能は新しいバージョンで見るボタンを選択してください。

    • エージェントごとにメンテナンス計画を追加できる対象エージェントオプションを追加しました。

    • 繰返しオプションはサーバープロジェクトでも使用できます。
  • Fixedリアルタイムアラートリストメニューから グループ化オプションを活性化すると、表示されるテーブルの視眼性の改善

Application​

Fixedダッシュボード>アクティブTXメニューからトランザクション一覧のいずれかを選択すると表示されるアクティブTXウィンドウのアクティブTXボタンを除去

Server​

  • Fixedサーバーリスト>サーバー 詳細メニューからプロセステーブルの最大(max)数値が断続的に誤った値が表示される問題を修正
  • Changedサーバーリスト>サーバー 詳細メニューからプロセステーブルの詳細画面照会時にディスクI/O(Bps)チャートに合計チャート線を共に表示するように変更
  • Fixedサーバーリスト>サーバーリストメニューから比較するオプションを活性化し、比較するサーバーのチェックボックスを選択すると、スクロール位置が初期化される問題を修正

Database​

共通​

Featureダッシュボード>インスタンス、マルチインスタンスメニューからアクティブセッションテーブルを折る(folding)ボタンを追加

PostgreSQL / MySQL​

Feature ユーザビリティのためのUI改善

  • クエリ照会領域とPlan照会領域を上/下/左/右に配置を変更できるように改善

  • Parameterバインドをデフォルトで実行するように変更

PostgreSQL​

  • FeaturePlan照会をjson形式で照会できる機能を追加
  • Featureアラート通知>アラート設定メニューからメトリクスイベントのテンプレートを追加
  • Changedダッシュボード>インスタンス、マルチインスタンスメニューでDefaultプリセットに保存されたアクティブセッションテーブルのカラム順序の変更

Oracle​

Changed統計/レポート>テーブルスペースサイズメニューからテーブルスペース情報を折りたたんで(folding)表示するように修正

Kubernetes​

  • FixedFixed次のメニューパスでトランザクション一覧のいずれかを選択すると表示されるアクティブTXウィンドウのアクティブTXボタンを除去
    • ダッシュボード>コンテナマップ>コンテナブロックをクリック>詳細>Active Transactionウィジェットのチャートをクリック
    • ダッシュボード>アクティブTXメニューからトランザクション一覧の項目をクリック
  • DeprecatedネームスペースプロジェクトのPod初期化性能メニューの削除
  • Featureネームスペースプロジェクトの実験室メニューでPending Pods追加

Browser​

FeatureAJAX Hitmapメニューやウィジェット、Ajax分析機能におけるユーザビリティのためのUIの改善

次のメニューパスでAJAX Hitmapウィジェットのチャート領域をドラッグすると新しくなったAjax分析ウィンドウを確認できます。

  • ブラウザ監視ダッシュボード>AJAX Hitmapウィジェット
  • 分析 > AJAX Hitmap

Preview 1.113.10​

2023年11月16日

共通​

メトリックス照会​

ChangedFieldsカラムでチャート表示機能で表示されるチャートデータを照会したテーブルデータベースに変更

Server​

  • Fixedアラート通知>アラート設定メニューからサーバー、プロセスイベントを追加または修正する場合、発行条件の持続時間を０分に設定すると、実際に反映するように修正
  • Changedアラート通知 >アラート設定メニューからデフォルト(Default)通知の名前を修正できるように変更し、'デフォルト' タグを別途表示するように修正
  • Changedサーバーリスト>サーバー 詳細メニューからプロセスの詳細画面照会時にLimitsテーブルを常に表示するように変更

Kubernetes​

  • Feature リソース > クラスターサマリー メニューの追加

    クラスターに関する情報を要約して提供します。 クラスタープロジェクトだけでなく、ネームスペースプロジェクトで分離された情報まで統合して確認することができます。

  • Featureダッシュボード >コンテナマップメニューから1分間消えたコンテナ/Podのブロックを表示する機能を追加

  • Changedリソース >ノードリストメニューを次のように改編

    • ノード情報指標の追加（クラスタープロジェクト）

    • CSVオプション追加

    • フィルタ検索機能の追加

    • テキスト検索を削除し、フィルター検索に置き換え

    • 時間更新間隔を1分から10秒に変更

    • 比較カラムを基本露出に変更し比較する ボタンの除去

    • 表示カラムを最大20個に制限

    • ユーザビリティのためのUI改善

  • Changedリソース >コンテナリストメニューを次のように改編
    • CSVオプション追加

    • 比較カラムを基本露出に変更し比較する ボタンの除去

    • 表示カラムを最大20個に制限
