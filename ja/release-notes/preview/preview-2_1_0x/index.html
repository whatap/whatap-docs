Preview 2.1.0.a

ノート

preview.whatap.ioで、次の新機能と変更した内容を事前に体験できます。

Preview 2.1.0.a4​

2024年01月19日

Application​

Fixed 分析 > キューブ及び性能推移 ニューでデザインを修正

Application, Kubernetes​

Fixed 統合APPトポロジー及びアプリケーショントポロジー内のボタングループ間の間隔エラーを修正

AWS Log​

  • 管理 > エージェントインストール メニューを次のように修正

    • Changeインストール画面でS3案内文を修正

    • Feature WhaTap Forwarderインストールタブの2番目のAWS Region一覧にVirginia(us-east-1) 項目を追加

Preview 2.1.0.a3​

2024年01月19日

共通​

  • Feature アラート通知 > イベント履歴メニューで次の機能を追加

    • アラート発生時刻カラムに進行中, メンテナンス中に発生タグ表示機能を追加

    • イベント解消時刻カラム表示のオプションを追加

    ノート

    この機能は、Server製品に既に適用されている機能であり、Server製品を除くすべての製品に拡大適用します。

  • Fixed アラート通知 > アラート設定メニューで次の機能を修正

    • メトリックスイベントタブの修正ウィンドウでガイド ショートカットボタンをクリックすると、言語設定別のガイド文書に移動するように修正

    • メトリックスイベントタブでJSONの書き出しまたは読み込みの過程で保存及びリストに追加するボタンクリック時に、書き出しまたは読み込みウィンドウが閉じられないイシューを修正

  • Fixed リアルタイムアラートリストメニューでグループ指定時に通知一覧が正しく表示されないイシューを修正

  • Changed一部のメニューで適用されなかった要素にダークテーマを追加適用

  • Fixed Safariブラウザーでお知らせメニュー内の断続的に一覧が表示されないイシューを修正

Server​

  • Fixed アラート通知 > イベント履歴 > イベント項目選択> イベント詳細メニューで処理履歴を追加する場合、作成したユーザーアカウントが登録されない問題を修正

  • Changed レポートメニューの月間サーバーレポート(詳細)テンプレート選択画面でエージェント選択のオプション名をアプリケーションからフィルターに変更

Database​

DB V2​

  • Feature 分析 > ロックツリーメニューで次の機能を追加

    • Lock Wait Sessionsウィジェットの右上に照会時間単位と最大値の表記タグを追加
    • 時間照会範囲が1か月前の状態で3時間以内のデータを照会する場合、照会不可メッセージの表示機能を追加

  • Fixed 分析 > ロックツリーメニューで大量のロックによりロジック上の負荷を招いて発生するホワイトアウト現象を修正

Kubernetes​

  • Changed リソース > マスターメタ情報メニュー名をObject Manifestに変更

  • Changed リソース > Object Manifestメニューで過去のオブジェクトマニフェストを照会できる機能を追加

Preview 2.1.0.a1​

2024年01月15日

共通​

  • Changed 分析 > メトリックスチャートメニューでエージェント全体の選択・解除する機能を追加

  • Fixed 統計/レポート > 統計メニューのSQLタブで他のタブのフィルターが適用されるバグを修正

  • Changed 利用内訳メニューで課金の固定価格のポリシーを変更

    ノート

    WhaTapの料金体系の変更に関する詳細は、次のリンクを参考にしてください。

Application​

  • Fixed トレース分析ウィンドウのツリービュータブでボタンを選択すると、設定された他のオプションが解除される現象を修正

  • Changed トレース分析ウィンドウのマルチTXタブでチャートとツリー設定の保存機能を追加

  • Changed トレース分析ウィンドウのマルチTXタブでプロジェクトの選択状態を保存する機能を追加

Kubernetes​

  • Fixed以下のメニューで抜けている多言語翻訳を追加

    • アプリケーション > マルチサービスヒットマップ
    • 統計/レポート > レポート

Log​

  • Feature ログトレンドメニューでフィルター一覧にない値を直接入力して選択可能

Preview 2.1.0.a0​

2024年01月11日

共通​

  • Changed プロジェクトリストメニューでプロジェクト削除時に表示されるポップアップウィンドウのUIを変更(プロジェクト名がカットされる現象を改善)

  • Changedサイドバーでプロジェクト一覧にマウスオーバーをする時、表示されるツールチップをクリックすると、そのプロジェクトに移動する現象を修正

  • Changed休眠状態またはロックされたアカウントをプロジェクトまたはグループに招待する時に表示されるフレーズを変更

  • Changed 利用内訳 > プロジェクト課金状況メニューの有料転換 ウィンドウでプロジェクト名が長い場合は、省略処理、マウスオーバー時のツールチップでプロジェクト名を表示するように改善

  • Changed FlexボードメニューでApdexウィジェットにチャート凡例設定を追加

  • Changed Flexボードメニューでダッシュボードのエクスポート機能を通じてJSONファイルとして保存する場合、ファイル名の特殊文字を以前は削除したが、下線(_)に変更

Database​

  • Fixed ダッシュボード > インスタンスリストメニューで正常状態のカラムを選択すると表示されるイベントウィンドウの照会機能のエラーを修正
