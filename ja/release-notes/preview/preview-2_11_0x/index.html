Preview 2.11.0.a

ノート

preview.whatap.ioで、次の新機能と変更した内容を事前に体験できます。

Preview 2.11.0.a4​

2024年11月20日

共通​

  • Fixedサイドメニューを閉じた状態で組織を選択し、お気に入りのFlexボードがあれば、統合Flexボードメニュー選択時に組織選択が解除される問題を修正

  • Fixed組織内で通知メッセージ設定移動後、組織内の照会が正常に動作しない問題を修正

Server​

  • Newクラウドおよびエンタープライズのオンプレミスインフラのモニタリングをサポートするサーバーインベントリマップメニューを追加

  • Changed サーバーリスト > サーバー インベントリメニュー名をサーバーインベントリに変更

Kubernetes​

  • Feature クラスター > Kubernetesイベントメニューに詳細表示機能を追加(Suggestions、隣接ログ、イベントタイムライン)

  • Feature クラスター > KubernetesイベントメニューにType、Reason個数表示および時間別イベント個数チャートにType区分表示を追加

Database​

共通​

Fixed ダッシュボード > インスタンスモニタリングメニューのアクティブセッションテーブルでカラム変更事項がCSVデータに反映されない問題を修正

Database V2​

Fixed 分析 > カウント推移比較メニューのメトリクス選択画面でメトリクスが見えない問題を修正

SQL Server​

New SQL Server V2サービス開始

Oracle、Oracle Pro​

Changed ダッシュボード > マルチインスタンスメニューでDisk Usageウィジェットの大きく表示ウィンドウで余白に対するUIを変更

Browser​

Feature 統計メニューの下位にIP負荷統計メニューを追加

Log​

Feature ログ > ログ設定メニューで次の機能を追加および修正

  • ログデータ保存期間を直接入力して変更できる機能を追加

  • ログデータ保存期間のデータ表示方式を修正

Preview 2.11.0.a3​

2024年11月14日

共通​

  • Feature リアルタイムアラートでCSVダウンロード機能を追加

  • Feature 利用状況 > プロジェクト課金状況メニューでプロジェクト全体および個別プロジェクトの課金単位の要約を追加

  • Changed アカウント情報 > SSO連動メニューでOktaに限定された案内文を変更

  • Fixed アカウント情報 > アラート受信設定セクションで照会エラー発生時に通知が終了しない現象を修正

  • Fixed 利用状況 > 決済情報メニューでAWS Marketplaceの再購入ボタンが有効化されない現象を修正

  • Fixed Flexボードメニューで特殊文字を含むメトリクスが統合されて表示される現象を修正

Application​

  • Fixed トレース分析画面のレコード要約タブでエラートランザクションでないにもかかわらず、エラーステップフィールドが表示される現象を修正

  • Fixed トレース分析画面のレコード要約タブでPod名、コンテナIDフィールドを別のグループで表示するように修正

  • Fixed トレース分析画面のテーブルビュータブでSQL詳細照会時にパスワードが解除されていない場合でも、バインド変数マッピング機能が有効になる現象を修正

Server​

  • Changedユーザー行為の履歴が残るAPIに置き換えてサーバーリスト > サーバー インベントリメニュー削除機能を改善

  • Fixed サーバーリスト > サーバー インベントリメニューでカスタムカラムを修正した後、保存されない現象を修正

Kubernetes​

  • New クラスター下位にetcd ダッシュボードメニューを追加

  • Feature コンテナマップおよび統合コンテナマップメニューで外部プロジェクトエージェントがインストールされたコンテナの識別とそのプロジェクト移動機能を追加

  • Fixedコンテナ数が多い時、メトリクスダッシュボードがロード状態を繰り返す問題を修正

Database​

  • Fixed 統計/レポート > SQL統計メニューでチャート領域の枠(border)を除去

  • Fixed ダッシュボード > インスタンスモニタリングメニューのメトリクス選択画面でイベント設定を選択時イベント設定 Newメニューに移動するように修正

PostgreSQL​

Fixed 統計/レポート > SQL統計メニューで空のクエリが照会される問題を修正

ノート

この修正事項は、DBXエージェント1.6.7バージョン以降に対応します。

Oracle、Oracle Pro​

  • Feature 分析 > SQL分析メニューのPlan Change Historyタブでsql_id、sql_hash_value、PLAN_HASH_VALUEで検索できるフィルター機能を追加

  • Feature 分析 > SQL分析メニューのPlan Change HistoryタブでPlan Diffを確認する際、新しいウィンドウで表示を追加

  • Fixed 分析 > SQL分析メニューのPlan Change Historyタブで時間間発生したデータが明確に区分されるようにチャートを修正

Preview 2.11.0.a2​

2024年11月07日

共通​

  • Featureサイドメニューの統合検索中Agent/IP検索時に無効なエージェント検索機能を提供

  • Feature プロジェクト一覧画面で設定ドロップダウンメニュー内のマウスホバースタイルを追加提供

  • Feature アラート通知 > イベント履歴メニューで進行中イベントのみ表示機能を提供

  • Feature 利用状況 > 決済情報メニューでAWS再購読の機能を提供

  • Changedサービスのお知らせに入ると、ヘッダータイトルを既存のお知らせ事項でサービスのお知らせ に変更

  • Changed 分析 > メトリクスチャート Newメニューでウィジェットのデータ表示形式を小数点2桁まで表示するように改善

  • Fixed 統計/レポート > SQL統計メニューで不要な横スクロールが表示される現象を修正

  • Fixed アラート通知 > イベント設定メニューでメトリクスイベントの案内文章の文字化けを修正

Application​

  • Fixed Java、Node.js、Python商品のアプリケーションエージェント設定値のうちencrypt_level英文を修正

  • Fixed 分析 > TX検索メニューの数字入力フォーム(経過時間、HTTP呼び出し件数、SQL件数、SQLパッチ件数)に一桁の数字のみある場合、値を削除できない現状を修正

Server​

  • Changed ダッシュボード > Flexボードメニューでserver_disk、server_processカテゴリ内のフィールドを使用してウィジェットを作成する場合、onameタグがidに必須で含まれないように変更

  • Changed アラート通知 > イベント履歴メニューUIを変更

  • Changed リアルタイムアラートでイベントを選択すると、イベント詳細画面に移動するように変更

Kubernetes​

Changed ダッシュボード > コンテナマップメニューのメトリックスタブを新しいメトリクスダッシュボードに変更

  • 時間範囲を選択できないように制限(10分固定)

  • コンテナビューのContainer CPU Total UsageウィジェットとPodビューのCPU Total Usageウィジェットはcoresでmillicoresに変更

Database​

共通​

Changedアクティブセッションの連携分析画面のThreadStat値を文字に変更

ノート

適用プラットフォーム：MySQL、PostgreSQL、Oracle V2、Oracle Pro

Database V1​

Fixed 分析 > ロックツリーメニューでSQL詳細画面のポップアップ時にクエリテキストが表示されない現象を修正

Oracle、Oracle Pro​

  • Feature ダッシュボード > マルチインスタンスメニューにDisk Usageウィジェットを追加

  • Changed 統計/レポート > SQL統計メニューのSQL詳細ポップアップ画面のRuntime PlanタブのUIを変更

  • Changed ダッシュボード > インスタンスリストとマルチインスタンスメニューのセッション詳細画面でセッションが終了してもsidとserial#情報が保持されるように変更

  • Changed ダッシュボード > インスタンスリストとマルチインスタンスメニューのセッション詳細画面のSession Infoにsql_hash_value値を追加

  • Changed ダッシュボード > インスタンスリストとマルチインスタンスメニューのセッション詳細画面のSession Infoにcommand値をステータス名で表示するように変更

  • Fixed 統計/レポート > SQL統計メニューのSQL詳細ポップアップ画面でプラン照会時の照会範囲を変更

Browser​

  • Feature 利用内訳 > 使用量と請求書プレビューでセッションリプレイ課金を表示

  • Feature アラート通知 > イベント設定メニューにAJAXヒットマップのパターン通知設定を追加

Log​

  • Changedログフィルターのお気に入り制限個数を10個から50個に変更

  • Changed ログ設定メニューのログモニタリングの開始タブデザインを修正

  • Changed ログ長期保存統計修正画面のタグ間の間隔を変更

Preview 2.11.0.a1​

2024年10月31日

共通​

  • Feature アラート通知 > イベント設定 Newメニューのテーブルリストに順番(No) カラムを追加

  • FeatureWhaTapユーザーの教育申請書(グーグルフォーム)に対する多言語支援を追加

  • Changed アラート通知 > イベント設定メニューで検索された結果通りにイベントの設定リストをJSON書き出し(export)するように変更

  • Fixedアラート通知>イベント設定NewメニューでJSON一括修正機能を利用する際、上書きボタンをクリックしても確認ウィンドウが表示されない問題を修正

  • Fixed 統合Flexボードまたはダッシュボード > Flexボードメニューでダッシュボードが反応型レイアウトの場合、印刷モードでウィジェットが表示されない問題を修正

  • Fixed 統合Flexボードまたはダッシュボード > Flexボードメニューで一部の環境によってPDFファイルをダウンロードする時、異常に描かれる問題を修正

  • FixedページネーションUIで選択ボックス(Select box) 要素のダークテーマを修正

Application​

  • Feature 分析 > マルチTXメニューで次の状況にボタンを無効にする機能を追加

    • 以前の検索と同じフィルター値(MTID/CUSTID、時間、選択したプロジェクト)を設定した場合

    • フィルター値(MTID/CUSTID、時間、選択したプロジェクト) のいずれか一つでも入力をしていない場合

  • Fixed 分析 > マルチTXメニューでプロジェクト名が実際のプロジェクト名と異なるように見える問題を修正

  • Fixed トレース分析ウィンドウテーブルビュー タブでSQLステップの高さが実際の領域よりも大きく出力される問題を修正

  • Fixed 分析 > マルチTXメニューで選択したプロジェクトが検索に反映されない問題を修正

  • FixedJavaプラットホームの管理 > エージェント設定メニューでsigar_enabledオプションのデフォルト値をtrueからfalseに変更

Kubernetes​

  • Feature クラスター > Ingress リストメニューでクバネティスエージェントバージョンが1.7.13未満の時、エージェント更新の案内画面を表示

  • Changed クラスター > ノードタイムラインメニューで最小限選択できる時間単位を5分に固定するように修正

  • Fixed サイトマップ > インスタンストポロジーメニューでコンテキストメニューのリンクが動作しない問題を修正

  • Fixedネームスペースプロジェクトでダッシュボード > コンテナマップメニューでコンテナのStdout/Stderrログが表示されない問題を修正

Database​

  • Feature アラート通知 > イベント履歴メニューでイベント名を選択すると、インスタンスモニタリングメニューに移動する機能を追加

  • Fixedリアルタイム通知で発生したイベントを選択すると、インスタンスモニタリングメニューに移動し、イベントの発生時刻を基準にデータを照会するように修正

  • Fixed 分析 > カウント推移比較メニューで次の事項を修正

    • 既存の日付と比較した日付間のチャートの色を異なって表示するように修正

    • 既存の時間が表示されるツールチップに日付を表示しないように修正

      例、既存：Innodb_buffer_pool_read_requests / 現在：Innodb_buffer_pool_read_requests (2024-10-29)

  • Fixed次のメニューのアクティブセッションテーブルでelapseカラムを削除

    • ダッシュボード > インスタンスモニタリング

    • ダッシュボード > マルチインスタンス

    • 分析 > カウント推移比較

    • 分析 > Wait分析

    適用プラットフォーム：PostgreSQL、OracleV2、Oracle Pro
