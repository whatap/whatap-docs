Service 2.20.0​

2025年08月27日

共通​

  • Feature アカウント管理 > アカウント情報 UTCオフセット変更機能を追加

  • Feature アカウント管理 アクセス制御メニューを追加し、アカウントごとのIPアクセス制御機能を追加

  • Feature アカウント管理 GPU、GPU MIG、Open Metric High、Open Metric Lowなど新しい使用量指標を請求書、課金、プロジェクト購読、メータリングサマリーに追加

  • Feature Flexボード > フィルター メトリクスフィルター、タグ選択に直接入力オプションを追加

  • Feature Flexボード 日次比較ウィジェットを追加

Application​

  • Feature ダッシュボード > アプリケーションダッシュボード アクティブHTTPC、アクティブDBウィジェットを追加

  • Feature 統計 > トランザクション トポロジーを開くボタンを追加

  • Feature ダッシュボード > ビジネスダッシュボード 詳細、トポロジータブを追加

  • Feature ダッシュボード > アプリケーションダッシュボード イベントウィジェットを追加

    • イベントをレベルごとに可視化し、エージェントごとに区分して表示

  • Changed APPトポロジー Beta UI改善

    • エッジラベルゲージ表現の追加および応答時間の閾値色表示を追加

    • 特定ノード（詳細）

      • TXトレース列設定追加および全体/アクティブ/マルチトランザクション/エラーフィルターを追加

      • 選択したノードと関連するノードのみ表示するポップアウトを追加

  • Changed app_context_stat 関連名称を統一（コンテキスト → 業務）

  • Changed アクティブスタックおよびマルチトランザクションアイコンの色を変更

    • マルチトランザクション: 青 → グレー

    • アクティブスタック: 赤 → 青

  • Fixed ダッシュボード > ビジネスダッシュボード エイリアス設定で、全エイリアス削除後保存しても変更が適用されない問題を修正

Java, Node.js, PHP​

  • Fixed ログ > ログ設定 ログモニタリング開始タブ「1. エージェント設定確認」の日本語翻訳を修正

.Net, Python​

  • Changed サポートバージョン情報とOS情報を修正

Java​

  • Changed エージェントインストール ページ改善

Server​

  • Feature GPUインベントリ キーワード検索フィルターを追加

  • Feature GPU性能サマリー 列設定にフィールド別説明ツールチップを追加

  • Feature サーバー 詳細 サーバーリストに検索入力フィールドを追加

  • Feature ダッシュボード ブラウザサイズに応じてウィジェットサイズを自動調整するモードをデフォルト適用し、スクロールが発生してもウィジェットを拡大して正常に表示できるようウィジェット高さ自動調整スイッチを追加

  • Changed GPU性能サマリー GPU Statusの参照時間範囲を従来の5分から1分に変更

  • Changed GPU性能サマリー Performance_State フィールド値をP0~P15で表示されるよう変更

  • Changed GPU性能サマリー > 比較する パネル

    • 単位がbytes/sのチャート値を自動的に単位変換するよう変更

    • Performance_State チャートのY軸範囲をP0~P15に固定

  • Changed サーバーリスト 下位の GPUインベントリ と GPU性能サマリー メニューを GPU グループ下位に移動

  • Changed Event v2 メトリクスイベントルールを追加する際、指標設定条件にバリデーションを追加し、問題がある場合はエラーメッセージを追加

  • Changed GPUインベントリ, GPU性能サマリー のエージェント最小バージョンを2.8.9から2.9.2に変更

  • Fixed サーバーリスト キーワードフィルター検索で項目がoidの場合フィルター項目が選択できない問題を修正

  • Fixed サーバー 詳細 サーバー選択後ページロード時、サーバーリストにマウスホバーしても即時表示されない問題を修正

  • Fixed メトリクスエクスプローラー GPUプロセスごとの指標選択時、凡例ラベルの形式を process_name, pid, (process_type) - index(uuid) に修正

  • Fixed サーバー インベントリ 新規エージェント収集フィールド追加時、テーブルヘッダーにフィールド名が表示されない問題を修正

  • Fixed サーバー 詳細 チャート詳細ビューのドロワーを閉じる際、選択サーバーではなくデフォルトサーバー選択に初期化される問題を修正

  • Fixed サーバーリスト 全体リフレッシュボタンをクリックした際、イベントによるサーバー状態が反映されない問題を修正

  • Fixed GPU性能サマリー > 比較する パネル

    • 単位が%のチャートのY軸最大値が100%を超えないよう修正

    • 単位がないチャートでタイトルに空の括弧が表示される問題を修正
