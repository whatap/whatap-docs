Service 1.112.0​

2023年10月25日

共通​

User Interface​

  • Feature サービスカタログページでより良いユーザーエクスペリエンスのためのデザインの再編成
  • Featureサイドバー幅の調整機能の追加

アラート通知​

ユーザー行為​

Changedサイトマップ>ユーザー行為メニューのテーブルでリクエストURI、METHODカラムの削除

Application​

  • Changed重複利用される指標の名称に対する統一化、標準化作業
    • URL, TX, TX名 → TX
    • Domain, ドメイン → ドメイン
    • DBC Time, DBコネクション時間, DB接続時間 → DBコネクション時間

    • CPU時間, CPU使用時間 → CPU使用時間

    • HTTPC, HTTPコール, HTTPコール件数 → HTTPコール件数
    • その他、重複する多数の用語及び用語に対する作成規則を策定し、標準化を進行
  • Changed統計/レポート>統計メニューからタブ移動時にユーザーが設定した検索時間を維持するように修正

サーバーリスト​
