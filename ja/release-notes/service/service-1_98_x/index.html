Service-1.98.0​

2023年3月29日

共通​

UX/UI​

ユーザーの満足度を向上させるためには、次のようにユーザーインターフェースのデザインを改善します。

  • Change 利用内訳 > 請求書プレビュー メニューのUI改善
    • プロジェクト、製品別にグループ化された請求書のプレビュー情報の提供
    • グループ化されたデータの関連商品情報及び、課金の集計量、請求額の合計などの確認が可能
  • Change 利用履歴 > プロジェクト有料プラン変更 メニューのUI改善
  • Change 画面の上部の「メニューパス」を「メニュー名」として表示するように変更
    • 変更前：上位パス > メニュー名
    • 変更後：メニュー名
  • Change 画面の右上のWhaTap技術文書のリンクボタンの追加し、チャンネルトークで技術文書リンクを削除します。
  • Fixed サービスロゴの横のブラウザー時間の不一致通知アイコンの位置を修正
  • Fixed プロジェクトがないグループを選択すると、プロジェクト作成ページに移動する問題を修正
  • Deprecate 統合ダッシュボード、統合メトリックボード メニューは、2023年5月1日以降、左サイドメニューから削除、サイトマップの移動に関する説明を表示

Flexボード​

  • Change ダッシュボードと一覧を画面に読み込む時に、読み込みインジケーターを適用
  • Feature Seriesウィジェットタイプのチャート設定で凡例表示を追加
  • Feature Seriesウィジェットタイプに累積チャートタイプを追加
  • Feature Seriesウィジェットタイプのチャート設定でチャート色の基準オプションを追加
    • 従来：タグを基準にチャートの色を表現
    • 追加：フィールドを基準にチャートの色を表示するオプション
  • Feature DB製品群から次のウィジェットテンプレートを追加

    • DB共通

      • CPU % (xos) ウィジェット
      • Memory % (xos) ウィジェット

    • MySQL

      • スロークエリ

    • PostgreSQL

      • スロークエリ
      • PostgreSQL Wait Event件数ウィジェット
      • PostgreSQL Client Ip件数ウィジェット
      • PostgreSQL統計テーブル
      • PostgreSQLアクティブセッション件数
      • PostgreSQLアクティブセッションテーブル

MXQL​

Fixed MXQLクエリのフィールド名に特殊文字が含まれている場合、MXQLデータを呼び出し時に発生するバグを修正するための、MXQLクエリ生成ロジックを修正

ログ​

  • Change ログ メニューからの機能をすばやく理解するためにメニュー名を変更
    • ライブTail → ライブテール
    • 検索機 → ログトレンド
    • 検索機 → ログ検索
  • Change ログ検索 メニューからクエリ内容を Timestamp を基準にソートする機能(新しい順/古い順) に変更
  • Change ログトレンド メニューから検索キーを最大2つまで設定できるように修正 (検索値は制限なし)
  • Change ログトレンド メニューから参照時間が変更された時に、選択したフィルターの結果がない場合は、上記のフィルターを解除できるよう改善

アラート通知​

  • Feature イベント設定 メニューから エージェント重複通知 オプションを追加

    モニタリング中のエージェントで、同じIDを設定したエージェントが2つ以上ある場合、メトリクスを重複して収集することによって発生するアラート通知です。 エージェント名が重複して設定されていないかエージェント設定で関連オプションを確認してください。ブルー/グリーンのようなデプロイ状況でエージェントが重複してアラート通知が発生した場合、この機能を使用することをお勧めします。 同じ名前のエージェントが同時に2つ以上起動した場合、選択した時間中は通知が発生しません。

  • Feature イベント受信設定 メニューから 受信タグの未設定通知を受け取る オプションを追加

    イベント受信タグが設定されていないアラート通知を受信しない場合は、そのオプションを選択してください。

  • Fixed イベント受信設定 メニューから登録したモバイル機器の数が表示されないバグを修正
  • Fixed イベント設定 メニューからログイベントを追加するときに イベント発生の一時停止 項目の説明文を修正

管理​

  • Feature ユーザー権限で ログ参照 アクセス権限の追加
  • Feature プロジェクト管理 メニューからアラート通知の言語を選択するオプションを追加
  • Fixed メンバー招待ウィンドウで メンバー管理 権限をオフできないバグを修正
  • Fixed チャンネルトーク > を修正。WhaTap支援チームの招待機能が動作しないバグを修正

    WhaTap支援チームの招待ボタンは、メンバー管理権限を保有しているユーザーのみ表示されます。

  • Fixed メンテナンス計画 メニューから 毎月 タブをクリックすると発生するホワイトアウト現象を修正
  • Change アカウント情報管理 > 会員退会 メニューを使用して会員退会する場合は、注意事項の更新や退会関連事項(未払金、権限情報) などの確認手続きを追加

利用内訳​

プロジェクト有料プラン変更 メニューから次の機能を追加および修正します。
  • Feature プロジェクトを複数選択して有料プランに変更する機能を追加
  • Change 月額見積費用 カラムを変更して、製品別の詳細を確認できるように修正

サービスカタログ​

  • Feature プロジェクト追加時 プロジェクトグループ 項目にグループ検索機能を追加
  • Change 組織を選択したプロジェクトを追加する場合 プロジェクトグループ 項目を必須で選択するように変更

Application​

アプリケーションダッシュボード​
