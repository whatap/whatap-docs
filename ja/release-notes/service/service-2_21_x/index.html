Service 2.21.20​

2025年10月27日

Server​

  • Fixed GPU インベントリ でカスタムカラムデータを修正する際、データがまれに全削除される問題を修正しました。

  • Fixed GPU インベントリ でフィルターを適用した際、検索結果の件数が更新されない問題を修正しました。

Service 2.21.18​

2025年10月22日

共通​

  • Fixed ユーザーグループ別表示設定で、Kubernetes 環境において 3 階層以上のメニューが表示されない問題を修正しました。

Service 2.21.17​

2025年10月20日

Browser​

  • Fixed 統計データのエラーを修正しました。

Service 2.21.15​

2025年10月16日

Database​

  • Fixed エージェントリスト でインスタンスが非アクティブ状態のときにページがホワイトアウトする問題を修正しました。

Service 2.21.14​

2025年10月16日

Server​

  • Fixed GPU インベントリ で同じサーバーの ipAddress が異なって表示される問題を修正しました。

  • Fixed メトリクスエクスプローラー の by GPU Process カテゴリでメトリクスを選択した際、チャート凡例に folded_id が表示される問題を修正しました。

2025年10月16日

  • Feature GPU インベントリ テーブルに Connection、External IP、OS Version カラムを追加しました。

  • Feature GPU パフォーマンスサマリー の性能指標比較チャート領域に期間選択セレクターを追加しました。

  • Fixed メトリクスエクスプローラー の Infiniband カテゴリでメトリクスを選択した際、チャート凡例に folded_id ではなく interfaceName が表示されるように修正しました。

Kubernetes​

  • Deprecated 管理 > Kubernetesエージェントのインストール において、オペレーター方式でエージェントをインストールする場合、追加機能として kube-apiserver は提供されなくなりました。

Server​

  • Fixed ユーザーグループ表示設定 の大メニュー配下のサブメニューが一切表示されないユーザーについては、大メニューも表示しないように修正しました。

  • Fixed ユーザーグループ表示設定 でレポートメニューの権限を設定できるように修正しました。

Service 2.21.0​

2025年9月24日

共通​

  • Feature ユーザーグループ別の表示設定 を追加

  • Changed サイト管理者 > リージョン 管理リストにルーター項目を追加

課金​

  • Changed プロモーションAPI応答変更に伴う割引量表示ロジックの変更（+ 商品コード表示）

  • Changed Cloud、ECS製品の課金集計ロジックを変更（API下位互換性対応）

    • 請求および支払い履歴 > 詳細 - 課金集計量
    • 請求書プレビュー - 課金集計量
    • プロジェクト使用量プレビュー - 課金集計量

  • Fixed サーバー無料利用超過案内ポップアップで支払い方法が登録されている場合にも、支払い方法登録案内が表示される問題を修正

OpenMetrics​

  • Feature OpenMetricsダッシュボードリスト 追加

    • 事前定義ダッシュボード提供: Istio, Nginx

  • Changed OpenMetricsエクスプローラ の参照可能期間を変更

Application​

  • Changed 管理 エージェントリストのテーブルカラム名称を修正

    • 開始時間 → アプリケーション開始時刻
    • 活性/非活性 → エージェント活性/非活性時刻

  • Fixed アプリケーション 本日ウィジェットで比較日付データがない場合、基準日付データが灰色で表示される現象を修正

Server​

  • Feature サーバー インベントリ に履歴照会機能を追加

    • インベントリ過去データ照会
    • 変更履歴照会
    • サーバーヒストリー日付別比較

  • Feature サーバー インベントリ 選択したテーブルカラムを対象に動作するデータ検索入力欄を追加

  • Feature エージェントインストール Windows 2008 R2用ZIP、EXEファイルダウンロードボタンを追加

  • Changed GPUインベントリ、GPU性能サマリー のLinuxエージェント最小バージョンを変更 (2.9.2 → 2.9.3)

  • Changed サーバー 詳細 > プロセスリスト の「ディスクI/O」を「プロセスI/O」に名称変更

  • Changed サーバー 詳細 > プロセス詳細 の「ディスクI/O (Bps)」を「プロセスI/O (Bps)」に変更

  • Changed GPUインベントリ > カラム設定 初期化時に自動収集カラムが全選択されるよう変更し、基本カラム順序を調整

  • Fixed GPUインベントリ でPhysical GPUのMIG Mode Current値がDisabledの場合、下位MIGインスタンスが表示されないように修正

Kubernetes​

  • Feature アプリケーション > ダッシュボード > ビジネスダッシュボード を追加

  • Feature 管理 > エージェントインストール にオペレーターインストール方法を追加

Database​

共通​

  • Feature 分析 > スクリプト·マネージャー にCSVダウンロード機能を追加

  • Feature ダッシュボード > インスタンスモニタリング に New 日次比較ウィジェットを追加

  • Feature ダッシュボード > インスタンスモニタリング、 分析 > カウント推移 でマルチDB構成時の dbname セレクターを改善

  • Feature 管理 > エージェントリスト にエージェント一時停止機能を追加（dbxエージェント v2.55.03以上対応）

  • Feature 統合Flexボード にインスタンスリストウィジェットを追加

  • Feature dbxc Linuxダウンロード用コピー枠を追加

  • Changed 分析 > メトリクス照会、 メトリックス異常検出 をサイトマップに移動

  • Fixed ダッシュボード > インスタンスリスト でインスタンスのReplicationやCluster状態変更時にM/S/Cアイコンが正常に変更されない問題を修正

  • Fixed ダッシュボード > インスタンスマップ にてTop SQLをサポートするデータベース製品のみTop SQLタブが表示されるよう修正

MySQL, Oracle, Oracle Pro, PostgreSQL, SQL Server​
