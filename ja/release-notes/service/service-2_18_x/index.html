Service 2.18.0​

2025年06月26日

共通​

  • Feature サービスタイトルの右側に WhaTap Docs リンクを追加

  • Feature Flexボード ウィジェットで非表示のプロジェクト名にツールチップ表示を追加

  • Feature Flexボード ウィジェット検索時、フォルダ内にウィジェットが存在しない場合は該当フォルダを非表示に

  • Feature 実時間アラートを新しい順に並べるように改善

  • Changed Flexボード 編集モードでウィジェット編集画面のスクロールを削除

  • Fixed グループ管理 現在のグループを削除した後、初期ページに移動するよう修正

  • Fixed 別のウィンドウから戻ったときに通知が自動で消えない問題を修正

  • Fixed 決済権限を移譲する際、同意案内のポップアップがプロジェクト選択ポップアップに隠れてしまう問題を修正

  • Fixed エージェント情報を誤った時刻で照会する現象を修正

Application​

  • Feature ダッシュボード > エージェントリソースダッシュボード 全体データ更新時にローディング表示を追加

  • Feature TXマップ フィルター保存機能を追加

  • Feature TXマップ リアルタイムモードでない場合、照会時間とフィルター情報をURLに保存

  • Feature 分析 > 性能推移 種類別・サーバ別のエージェントを選択できるように改善（種類に関係なく1つのみ選択可能）

  • Feature 実験室 > ビジネスダッシュボード メニュー追加

  • Changed 分析 > 性能推移 統計画面と同様にエージェントセレクターUIを変更

  • Fixed ダッシュボード > エージェントリソースダッシュボード リアルタイムモードで全体データにフィルターが適用されない問題を修正

  • Fixed 分析 > ヒットマップ チャート軸の範囲を超えたデータが表示される問題を修正

  • Fixed ダッシュボード > ヒットマップ ウィジェットのエージェントフィルターがヒットマップの一部にしか適用されない問題を修正

  • Fixed トレース分析 ポップアウトで内部データが間欠的に表示されない問題を修正

Server​

  • Feature GPUインベントリ、GPUパフォーマンス概要 の新メニューをリリース

  • Feature サーバーリスト > サーバー 詳細 過去の履歴を参照する際、削除されたエージェントもサーバー選択（セレクター）で表示されるように修正

  • Feature サーバーリスト 管理機能のボタン名を変更し、説明ツールチップを追加

    • 一時停止 → アラート停止： “選択されたサーバーのイベント発生を一時停止します"
    • 再起動 → アラート再開： “選択されたサーバーのうち、アラート停止状態のサーバーのイベントアラートを再度有効にします。”
    • 除外：“選択されたサーバーの動作を停止します。 サーバー一覧から除外されます。また、サーバーのステータスを確認できません。”

  • Feature サーバー 詳細 過去時点を照会する際、削除されたサーバーでもデータが残っている場合はサーバー選択セレクターに非アクティブ状態で表示

    • yard.confの show_deleted_servers_in_range オプションが有効なプロジェクトに適用

  • Fixed Featureプロジェクト > KAFKAダッシュボード 時間更新時にすべてのウィジェットで毎回ローダーが表示され、データが隠れる問題を改善

    • リアルタイムモードでは初回データ読み込み時にのみローダーを表示
    • 一時停止モードでは時間変更時にローダーを表示

Kubernetes​

  • Feature 分析 > OpenMetric Explorer を追加

  • Feature GPUモニタリング をリリース

  • Feature ダッシュボード > コンテナマップ GPU指標に基づく色表示機能を追加

  • Changed ワークロード > Podリスト > 詳細 メトリクスダッシュボードの照会期間を最大30日までに変更

  • Changed タイムラインメニューの5分単位時間選択制限を解除

    • コンテナー > コンテナタイムライン

    • ワークロード > Podタイムライン

    • ノード > ノードタイムライン

  • Fixed 管理 > エージェントインストール helm インストールコマンド修正

Database​

共通​

  • Feature 統計/レポート > SQL統計 指標にマウスオーバー時の説明表示機能を追加

  • Changed ログファイルビューア 1ページあたりの照会文字数を2,000文字 → 20,000文字に拡張

  • Fixed 統計/レポート > SQL統計 テーブルのフィールド名、カラムセレクタのフィールド名、フィルタ指標のフィールド名が異なって表示される不具合を修正

  • Fixed ダッシュボードウィジェットに不要なスクロールが発生する問題を改善

    • Wait Event
