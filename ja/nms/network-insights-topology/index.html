トポロジー

ネットワークインサイトのトポロジーは、ネットワーク機器とリンク構造を視覚的に表示します。

単に構造を見るだけでなく、リンクダウンなどリアルタイムで発生するイベントが即時に反映されるため、管理者はネットワーク状態を素早く把握できます。 トポロジーは2つのモードを提供します。

  • モニタリングモード: ネットワーク全体構造を確認し、リアルタイムイベントを監視する閲覧モード
  • 編集モード: 領域を作成したりノード属性を編集して画面レイアウトを管理

基本画面と設定​

トポロジー画面上部では、モード切替、マネージャー選択、表示形式変更など主要な制御機能を利用できます。

メニュー説明
モード選択モニタリングモード または 編集モード タブを選択
- モニタリングモード: ネットワーク全体構造を確認し、リアルタイムイベントを監視
- 編集モード: 領域を作成したりノード属性を編集して画面レイアウトを管理
マネージャー選択接続された NMS マネージャーを選択
位置保存ユーザーが調整した機器およびリンク位置を保存
表示形式物理 または 論理 トポロジーに切替
- 物理: 実際に機器間で接続されたすべてのポートを確認
- 論理: 機器間の論理的な接続構造を中心に確認
表示オプションノード IP 表示の有無を選択
設定初期化すべての画面設定を初期値に戻す
通知設定アイコンON / OFF でトポロジーに異常があるときサウンドで通知するかを設定

共通機能​

  • パン: 画面の空白部分をドラッグして移動

  • ズームイン/ズームアウト: マウスホイールで拡大・縮小

  • 要素移動: トポロジー上の要素をドラッグ＆ドロップで移動

モニタリングモード​

モニタリングモードでは、ネットワーク構造をリアルタイムで確認し、イベント変化を即座に把握できます。

  1. ネットワークインサイト > トポロジー に移動します。

  2. モニタリングモード タブをクリックし、メイントポロジービューと Recent Trap Messages を確認します。

メイントポロジービュー​

  • ネットワーク機器はアイコンで表示されます。

  • ルーター、スイッチ、ファイアウォールなど機器タイプによってアイコンの形が異なります。

  • 機器間の接続は線で表示され、状態によってスタイルが変化します。

    • 正常: 灰色の実線
    • リンクダウン: 赤色の実線

  • 機器の電源がオフまたは無効になると、アイコンの色が変わり状態をすぐ確認できます。

  • 領域を分けて機器をグループ化するカスタム機能を提供します。

Recent Trap Messages​

画面右側のパネルに、収集された Trap メッセージがリアルタイムで順次表示されます。機器やリンクのイベントをテキストログ形式で確認できます。

  • 例: 特定リンクダウン、機器の有効化または無効化

編集モード​

編集モードでは要素をクリックして編集または追加できます。NMS マネージャーを通じてノードやリンクを編集できます。

  1. ネットワークインサイト > トポロジー に移動します。

  2. 編集モード タブをクリックし、必要な要素を追加または編集します。

新しい要素の追加​

編集モードタブで画面左上の 新しい要素を追加 ボタンをクリックして、新しい要素を追加できます。

ノート

現在サポートされている機能は 領域 の追加のみで、ノード の追加は無効化されています。

領域の編集​

作成された領域のプロパティを編集できます。領域の背景色と枠線色を変更して視覚的に区別しやすくできます。

  • 領域名は アイコンをクリックして入力または変更できます。

  • 不要な領域は 削除 ボタンをクリックして削除できます。

ノードの編集​

ネットワーク機器（ノード）を選択すると、名前とデバイスタイプを編集できます。

  • ネットワークノード名は アイコンをクリックして入力または変更できます。

  • 提供されるデバイスタイプ: router, switch, hub, host, ap, phone, firewall, unknown
