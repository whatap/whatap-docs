サーバー詳細

ホーム画面 > プロジェクト選択 > サーバーリスト > サーバー詳細

WhaTapモニタリングサービスの初期画面でプロジェクトを選択し、サーバーリストメニューのサーバー詳細メニューを選択してください。サーバー詳細は、サーバーのリソース現況と実行プロセスの詳細情報を提供します。

上段オプション​

上段の領域でデータ更新時間および検索サーバーを指定することができます。データ更新時間およびサーバー指定オプションと当該サーバーの概要情報を確認できます。

データ更新時間​

サーバー詳細ページのデータは5秒ごとに自動的に更新されます。

  • 一時停止：ユーザーはデータの自動更新を停止することができます。

  • 再生：ユーザーは停止した自動更新を再開することができます。

  • 更新：ユーザーは現在の時間を基準にデータを更新することができます。

サーバー選択​

サーバー選択オプションを使用すると、詳細を表示するサーバーを指定できます。選択ウィンドウの一覧から実行するサーバーを選択し、次のの領域でサーバーのリソース状態や実行プロセスの詳細情報を確認できます。

サーバー詳細​

サーバーリソースの状態​

の領域では、 CPU Usage、Memory Usage、Network、Disk I/Oチャートを使用して、サーバーのリソース状況を確認することができます。

  • ：CSVアイコンを選択すると、チャート履歴をCSVファイルとして保存することができます。

  • ：カメラアイコンを選択して、画像形式でチャート画面を保存することができます。

  • ：詳細アイコンを選択すると、CPU使用率、メモリ使用率、ネットワーク、ディスクのI/Oなどの詳細チャートが表示されます。

    • CPU使用率

      CPU Usage, CPU Idle, CPU Nice, CPU I/O Wait, CPU Steal, CPU IRQ, CPU Soft IRQ, CPU Load

    • メモリ使用率

      Memory Usage, Memory Available, Memory SReclaimable, Memory SUnreclaim, Memory Slab, Memory Swap Used, Memory Page Faults

    • ネットワーク

      Traffic In/Out. Packet In/Out, Error In/Out, Dropped In/Out

    • ディスクI/O

      Disk I/O, IOPS Read/Write, Disk Bph Read/Write, Used Space, Queue Length, Inode Used, Free Space

プロセスリスト​

配置​

  • プロセス一覧上で選択した区分に従って、降順または昇順のソートを指定して参照することができます。

  • プロセス一覧の上で選択したメトリクスによって上位の10個、20個、30個、全体プロセスを指定して確認することができます。

グループ化​

プロセスグループは、インスタンスで同じ動作を実行する複数のプロセスをまとめたものです。グループ化ボタンを選択してプロセスグループを追加することができます。プロセスのグループ化の詳細については、次の文書を参照してください。

サーバー情報パネル​

サーバー情報パネルで確認できる内容は次のとおりです。ディスプレイ名とメモの場合サーバー情報パネルのみ編集が可能です。

  • サーバーアラートリスト

    サーバー情報パネルの上のサーバー通知一覧ボタンを選択すると、現在のサーバーに割り当てられた通知を確認することができます。

  • Display Name/ID

    選択したサーバーの表示名(ONAMEエイリアス)とサーバーID(OID)を確認できます。修正アイコンを選択すると表示名を変更することができます。

    ONAMEは、デフォルトでホスト名が設定されます。

  • OS

    選択したサーバーのOS情報を確認することができます。

  • IP Address

    選択したサーバーのIPアドレスを確認することができます。

  • Agent

    選択したサーバーのエージェントのバージョンを確認することができます。

  • Memo

    作成したメモを確認します編集アイコンを選択してメモを編集および追加します。
