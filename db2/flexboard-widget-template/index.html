Widget Templates

Flex boards let you build dashboards with widgets to visualize and link all data for monitoring targets. After selecting a dashboard in the Integrated Flex Board or Flex Board menu and moving to Edit Mode, you can check available widgets in Widget templates.

  • Integrated Flex Board: you can choose widgets for all product types in projects you belong to
  • Flex Board: you can choose only widgets for the product types of that project
Integrated Flex BoardFlex Board

Note

Available widgets in Widget templates may differ by a project’s product type.

  • For details on common options in the top right of a widget, see the product specific Flex board > Managing widgets document.

  • Widget features may be limited by member permission. For details, see the Project structure and member permissions document.

Graph widgets​

Visualize change over time or relationships between metrics to see performance change, distribution, and comparisons at a glance.

Time series​

The time series chart widget is the basic chart for checking metric change over time.

The time series chart widget can compare with a previous time.

  1. Click the icon in the top right of the widget.

  2. Set the time unit you want and click the Apply button. When you hover the mouse pointer over the chart, you can check comparison data in a tooltip.

Note

Compare with previous time is not provided by every widget and is available only in Edit Mode.

  • For details on data merge options, see the product specific Flex board > Managing widgets document.

Top N time series​

This widget shows metric change over time and metric rankings for monitoring targets.

You can set the number of top N items to display in the chart.

  1. In Edit Mode, click the more icon () in the top right of the widget.

  2. Click TopN and choose how many top N items to display.

Apdex​

The Application Performance Index (Apdex) widget shows application performance metrics.

Note

The Apdex widget is available in the application monitoring product family.

Pie​

This pie chart widget is useful for checking composition ratios against the whole of collected data. You can check values such as CPU and memory usage and error counts.

Hitmap​

The Hitmap widget shows a distribution of response times for completed transactions.

The horizontal axis is transaction end time and the vertical axis is execution time. Transactions with longer execution time are located at the top of the distribution. When you drag within the chart area in Hitmap, the Trace analysis window appears where you can check detailed transaction information.

The horizontal axis is end time and the vertical axis is execution time. Drag in the chart area to open the Trace analysis window for detailed transactions.

Note

The Hitmap widget is available in the application monitoring product family.

Bar​

This bar chart widget is useful for comparing the status of data collected from monitoring targets.

A vertical bar chart is provided by default, but you can switch to a horizontal layout. In Edit Mode, click the more button () in the top right of the widget, then click the button.

Arc equalizer​

This arc equalizer chart widget visualizes and compares transaction status by agent for monitoring targets.

An arc equalizer chart is provided by default, but you can switch to a vertical bar chart. In Edit Mode, click the more button () in the top right of the widget, then click the button.

Note

The arc equalizer chart widget is available in the application monitoring product family.

Treemap​

This widget visualizes server disk or memory usage. Each block’s size and color indicate usage and status.

When you hover the mouse pointer over blocks in the default chart, a tooltip shows that block’s value. To switch to a chart that displays values inside each block, click the more button () in the top right of the widget, then click the button.

Scatter​

A scatter plot visualizes the relationship between two variables. Each point represents a data point. It is useful for identifying patterns or trends in large datasets and for visually checking how data points are distributed. You can identify performance issues that occur at specific times.

  • The x axis represents time and shows when the data point was recorded.

  • The y axis represents processing time and shows how long the data point took.

  • Drag a region of the chart to view details.

Geomap​

This map chart widget visualizes spatial data. By combining data values with geographic locations, it helps you identify and analyze regional performance differences.

  • Use the mouse wheel or the slider to zoom in or out.

  • To view the world map, select the button. You can select a specific country to check regional data.

  • To view the data in a table, select the button. To return to the map view, select the button.

  • For regional maps, data is displayed by top level administrative divisions. (Korea: provinces and metropolitan cities)

Info

This site or product includes IP2Location LITE data available from https://lite.ip2location.com.

Speed chart​

The Speed chart widget visualizes the speed of active transactions that occur on the monitoring target. It shows the performance status of the monitoring target and is useful for analyzing performance metrics such as request processing time and response time.

Note

The Speed chart widget is available in the application monitoring product family.

Honeycomb​

This honeycomb chart widget shows performance and stability for monitoring targets. Each color indicates performance and stability. Blue means normal. Orange means warning. Red means slow. Gray means disabled.

Note

Collected data types and color meaning can differ by monitoring product family.

List widgets​

Table​

This widget displays a list in table form. It is useful for checking values and status of data collected from different monitoring targets. For each listed item, you can check additional related information through columns. Text color in the list indicates performance status.

Click the more button () in the top right of the widget to use the following features.

  • : set filter options to display only the data you want

  • : add or remove columns and change their order

Group widget​

This widget groups multiple widgets into one composite.

Place a group widget​

  1. In the Widget templates list, select ETC > Group Widget.

  2. In the top right of the placed group widget, select the Project selection button and choose the project that will provide data.

  3. In the top right of the group widget, select the grid button and set the layout you want.

  4. Select the Apply button to complete layout settings.

  5. In the top right of the group widget, select the Add widget button.

  6. When the Widget templates list appears on the right, choose the widgets you want in order.

Note

  • Only one project can be selected in a group widget.

  • Widgets are placed in the order added. To change order, click and drag the widget name to the desired position.

  • You can place up to 36 widgets in a group widget. You cannot add more widgets than the number set in the grid option.

  • When you have placed the maximum number of widgets, the grid button becomes disabled.

  • You can open a group widget in a new window. Enter View mode, then click the button in the top right of the widget.

Remove a widget in a group widget​

To delete an individual widget placed in a group widget, click the more button () in the top right of the widget, then click the button.

Collapse or expand a group widget​

You can collapse or expand a group widget. Click the or button in the top right of the widget.

This feature helps you use space efficiently when many widgets are placed on a dashboard. You can also expand only when you want to view information.

Alert and status widgets​

Alert​

Alert list​

You can track alerts in real time or set a specific query time for one or multiple projects.

Edit Mode

  • In Edit Mode, you can group and sort by Title, Project, and Application. To disable grouping, select the Timeline button.

  • To include resolved events, enable the View resolved events option.

  • You can set a separate time from the dashboard’s query time. Select the button. When the Select a time popup appears, uncheck the Use dashboard settings option.

Note

  • For details on the Select a time option, see the next document.

  • To toggle alert sounds on or off when alerts occur, select the button.

View mode​

In View mode, you can check alert status based on the settings from Edit Mode.

  • To view the alert list in full screen, select the button.

  • To open the alert list in a new window, select the button.

Event Timeline​

You can check alerts by time for one or multiple projects in chart form.

Edit Mode

View mode

  • In Edit Mode, you can set a separate time from the dashboard’s query time. Select the button. When the Select a time popup appears, uncheck the Use dashboard settings option.

  • In View mode, to view full screen, select the button. To open in a new window, select the button.

Status check​

This widget displays current status and results based on data collected from monitoring targets. You can monitor increase or decrease for a specific service.

Click the more button () in the top right of the widget to use the following features.

  • : add or remove the data you want and change order

  • : show only text information without the graph chart

  • : show text information with the graph chart

Information widgets​

Text​

This widget displays metric values in text form.

Click the more button () in the top right of the widget, then click the button to add or remove the data you want and change order.

Overview​

This widget presents project summaries or explanations of specific metrics.

Note

Displayed summaries differ by the selected project’s product type.

Page load Timeline​

This is a Browser product only widget that shows Page load Timeline information.

Note

For details on Page load Timeline, see the next document.

Other widgets​

Rich text​

You can enter any text and place it as a widget.

You can edit text and set formatting using options provided in the top right of the widget.

  • : use the Edit widget option to set widget background and border and font color. After finishing, select the Save button.

  • : a Rich text window appears on the right. You can add or edit text and use the provided toolbar to set formatting.

Maintenance plan​

This widget shows maintenance plans for the selected project.

Note

  • Only maintenance plans scheduled within 1 hour are displayed in the widget.

  • For details on the Maintenance Plan menu, see the next document.

URL list​

This is a URL product only widget that shows the list and status of registered URLs.

Note

For details on URL monitoring, see the next document.
