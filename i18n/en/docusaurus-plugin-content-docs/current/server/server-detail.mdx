Home &gt; Select Project &gt; <Cmdname sid="side_serverList" className="uitext" /> &gt; <Cmdname sid="side_serverDetail" className="uitext" />

Select a project in the initial screen of the WhaTap monitoring service, and then select <Cmdname sid="side_serverDetail" className="uitext" /> under <Cmdname sid="side_serverList" className="uitext" />. <Cmdname sid="side_serverDetail" className="uitext" /> provides detailed information on the server resource status and running processes.

<ImgLang img='server-detail.png' desc='Server details sc'/>

## ![](/img/number-01.png) Options at the top

In the ![number1](/img/number-01.png) area at the top, you can set the data for **_data update time_** and **_server_**. You can see the **_Summary Info_** of the server, along with the **_Data Update Time_** and **_Server_** options.

### Data update time

The data on the server detail page is automatically updated every 5 seconds. 

- ![Pause icon](/img/ic-s-pause.svg) **_Pause_** : You can stop automatic update of the data. 

- ![Playback icon](/img/ic-play-grn.svg) **_Playback_** : You can restart the auto update you have stopped.

:::note 

For more information on how to use the time selector, see [the following](usage-ui#timeselector).

:::

### Select Server

The <Cmdname sid="select_server" className="uitext" /> option allows you to set the server to view. It also set to perform <Cmdname sid="DSC09486" className="uitext" /> during a view. If you select a server to view from the list, you can see the server resources and detailed information of running processes in the ![number2](/img/number-02.png) area.

## ![](/img/number-02.png) Detail

### Server resources

In the ![number2](/img/number-02.png) area, you can see the resources of the server through the **_CPU Usage_**, **_Memory Usage_**, **_Network_**, and **_Disk I/O_** charts.

- ![Configuration icon](/img/ic-setting.svg): You can specify the maximum value of the Y-axis in the chart configuration. 

- ![csv icon](/img/ico-download.svg) : You can save the chart records in a CSV file.

- ![Camera icon](/img/ic-camera.svg) : The chart data is saved in an image data.

- ![Detail icon](/img/ic-zoom-in.svg) : You can see the charts for CPU utilization, memory utilization, network, and disk I/O. 

  - CPU Utilization

    **_CPU Usage_**, **_CPU Idle_**, **_CPU Nice_**, **_CPU I/O Wait_**, **_CPU Steal_**, **_CPU IRQ_**, **_CPU Soft IRQ_**, **_CPU Load_**

  - Memory utilization

    **_Memory Usage_**, **_Memory Available_**, **_Memory SReclaimable_**, **_Memory SUnreclaim_**, **_Memory Slab_**, **_Memory Swap Used_**, **_Memory Page Faults_**

  - Network

    **_Traffic in/out_**. **_Packet in/out_**, **_Error in/out_**, **_Dropped in/out_**

  - Disk I/O

    **_Disk I/O_**, **_IOPS read/write_**, **_Disk Bph Read/Write_**, **_Used space_**, **_Queue length_**, **_Inode Used_**, **_Free space_**

### Process List

#### Align

- You can search by setting a sorting in descending or ascending order according to the selected metric at the top of <Cmdname sid="sms_summary_process" className="uitext" /> list.

- According to the metric selected at the top of <Cmdname sid="sms_summary_process" className="uitext" /> list, you can check by setting the top ***10***, ***20***, ***30***, and <Cmdname sid="all" className="uitext" /> processes.

#### Grouping

A process group is a set of multiple processes with the same behaviors in an instance. You can add process groups by selecting the <Cmdname sid="groupby" className="uitext" /> button. For more information about process grouping, see [the following](server-detail-process-group).

### Server information panel

You can see the following in the <Cmdname sid="server_info" className="uitext" /> panel. In case of *** Agent Name*** and ***Memo***, you can edit only in <Cmdname sid="server_info" className="uitext" /> panel.

- ![Alert icon](/img/ic-alert.svg) <Cmdname sid="server_alert_list" className="uitext" />

  At the top of <Cmdname sid="server_info" className="uitext" /> panel, if you select ![Alert icon](/img/ic-alert.svg) <Cmdname sid="server_alert_list" className="uitext" />, the alerts for the current server appear.

- **_Agent Name / ID_**

  You can check the agent name (alias for ONAME) and ID (OID) of the selected server. You can modify the agent name by selecting the ![Edit icon](/img/ic-edit.svg) button. 

  :::note

  ONAME is set to the host name by default. 

  :::

- **_OS_**

  You can see the OS information of the selected server.

- **_IP Address_**

  You can see the IP address of the selected server.

- **_Agent_**

  You can see the agent version of the selected server.

- **_Memo_**

  By checking the written memo or selecting ![Edit icon](/img/ic-edit.svg), you can modify or add any memo.
