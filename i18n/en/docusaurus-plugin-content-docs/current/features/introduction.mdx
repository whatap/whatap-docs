WhaTap newly introduced **Feature Project** (<code class='Features'>Features</code>) so that users can easily check whether the desired software supports monitoring and easily start monitoring. The Feature Project provides an optimized monitoring solution by repackaging existing monitoring products to fit the software to monitor. Manage your services for stability and performance more effectively through the WhaTap's Feature Project. 

- **Integrated monitoring experience**

  **Feature Project** not only provides easy installation through a dedicated menu, but also allows you to easily integrate and manage the monitoring configuration. Experience the integrated monitoring with user-friendly interfaces. 

- **Intuitive user experience**

  **Feature Project** not only allows you to easily check whether your software is compatible, but also provides a customized dashboard for the software. Monitor your software easily without cumbersome settings. 

## KAFKA

KAFKA is a distributed streaming platform used for data stream processing. WhaTap KAFKA monitoring helps you trace the performance and status of the Kafka cluster to detect and resolve issues early as possible.

## Aerospike

Aerospike is a high-performance NoSQL database optimized for real-time data processing and large-scale data analysis. Through the Aerospike monitoring, it traces the cluster performance, response time, data consistency, and more in real time to help maintain and optimize the system stability. 

## Apache Pulsar

Apache Pulsar is a scalable, high-performance, open source messaging and streaming platform. Apache Pulsar monitoring helps you trace the cluster efficiently. 

## vCenter

vCenter is the VMware's centralized management platform that enables integrated management for various resources and tasks in the virtualized data center from a single location. Simplify IT infrastructure operations and increase management efficiency by efficiently managing multiple virtual machines (VMs), hosts, storage, and networks through a single interface.

## NGINX

NGINX is a high-performance HTTP and reverse proxy server that provides load balancing, caching, and HTTP caching for web applications, and can effectively manage traffic to optimize the server performance. NGINX is optimized to handle web traffic securely and is widely used to operate large-scale websites.

## APACHE

Apache Server is one of the most widely used web servers in the world. It processes HTTP requests and provides the content details. Through Apache Monitoring, you can effectively manage the server performance, request throughput, response times, and more.

## Milvus

Milvus is a database system for high-performance vector search. WhaTap Milvus monitoring allows you to see the resource usage of each component and the performance of searches and storage operations.
