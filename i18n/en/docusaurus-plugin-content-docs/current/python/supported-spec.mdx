## Python agent-supported environment

Before installing the WhaTap Python monitoring agent, check the support environment.

### Supported Versions

| Application Type | Python Version |
| --- | --- |
| WSGI Application | 3.6 ~ 3.13 |
| ASGI Application | 3.7 ~ 3.13 |

### Operating System

| OS | Minimum Version | Architecture |
| --- | --- | :---: |
| Debian / Ubuntu | 12.04 | 64bit |
| Red Hat / CentOS | 6.x | 64bit |
| macOS | 10.15 Catalina | - |

### Application Server

| Framework | Supported Version |
| --- | --- |
| Bottle | - |
| Celery | - |
| CherryPy | - |
| Django | - |
| FastAPI | 0.84.0 or later |
| Flask | - |
| Frappe | - |
| Nameko | - |
| Odoo | v16, v17 |
| Tornado | - |

### Libraries

| Category | Libraries |
| --- | --- |
| **External Calls** | HttpLib, Requests, Urllib, HTTPX |
| **Data Processing** | SQL Alchemy, GraphQL |
| **Logging** | smtplib, logging, loguru |
| **Messaging** | Kombu, Pika |
| **LLM** | OpenAI |

### Database

| Database | Driver |
| --- | --- |
| MongoDB | pymongo |
| MySQL | MySQLdb, pymysql |
| Neo4j | neo4j |
| Oracle | cxOracle |
| PostgreSQL | psycopg2, psycopg, psycopg_pool |
| Redis | redis |
| SQLite | sqlite3 |
