## Python agent-supported environment

Before installing the WhaTap Python monitoring agent, check the support environment.

- **Support versions**

  - WSGI Application (Python 3.6 or later, 3.13 or earlier)

  - ASGI Application (Python 3.7 or later, 3.13 or earlier)

- **OS**

  - Debian/Ubuntu 12.04 (64 bit) or later

  - OSX 10.15 Catalina

  - Red Hat/CentOS 6.x (64 bit) or later

- **Application Server**

  - Bottle

  - Celery

  - CherryPy

  - Django

  - FastAPI(0.84.0 or later)

  - Flask

  - Frappe

  - Nameko

  - Odoo (v16, v17)

  - Tornado

- **Library**

  - External Calls: HttpLib, Requests, Urllib, HTTPX

  - SQL Alchemy, GraphQL

  - smtplib, logging , loguru

  - Kombu, Pika

  - LLM: OpenAI

- **Database**

  - MongoDB(pymongo)

  - MySQL(MySQLdb, pymysql)

  - Neo4j(neo4j)

  - Oracle(cxOracle)

  - PostgreSQL(psycopg2, psycopg, psycopg_pool)

  - Redis(redis)

  - SQLite(sqlite3)
