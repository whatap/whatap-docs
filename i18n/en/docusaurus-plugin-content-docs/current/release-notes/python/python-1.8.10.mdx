August 22, 2025

- <Status priority='1'>New</Status> Added tracing support for the `psycopg` module

- <Status priority='1'>New</Status> You can now specify the path to the *whatap_python* executable using the `WHATAP_PYTHON_AGENT_PATH` environment variable.

    - If this environment variable is not set, it defaults to `WHATAP_HOME`.

    :::note

    For details, see [Kubernetes Agent v1.8.8](https://docs.whatap.io/release-notes/k8s/k8s-1_8_8).

    :::

- <Status priority='0'>Fixed</Status> Fixed issues in the `neo4j` module  

    * Resolved an issue where DB connection timing was not being collected properly  

:::note

Agent download: [whatap-python v1.8.10](https://pypi.org/project/whatap-python/1.8.10/)

:::
