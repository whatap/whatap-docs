July 16, 2025

- <Status priority='0'>Fixed</Status> Fixed an issue in PostgreSQL slow query where enabling `use_tcp=1` caused a 1-byte offset, leading to `[CPU]` field misalignment and repeated "unknown packet" errors
- <Status priority='0'>Fixed</Status> Fixed an issue in MySQL where setting `slow_query=` incorrectly changed the `use_midinfo` processing flag to 1  
  ※ Bug: SQL text was missing or corrupted in slow query logs
- <Status priority='0'>Fixed</Status> Modified PostgreSQL slow query handling to skip `00000:` error codes when `log_error_verbosity=verbose` is set  
  ※ Bug: SQL text was not displayed in slow query logs
