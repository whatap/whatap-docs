March 13, 2025

## MySQL

- <Status priority='1'>New</Status> Added the 1-minute option for the statistical data collection cycle (`sqlstat_send_interval=1`, default: `5`).

- <Status priority='0'>Fixed</Status> Modified the columns due to Master and Slave Status query changes in MySQL 8.4 or later.

## Oracle

- <Status priority='1'>New</Status> Added the 1-minute option for the statistical data collection cycle (`sqlstat_send_interval=1`, default: `5`).

- <Status priority='0'>Fixed</Status> Fixed the issue where the Oracle Active Session text is not displayed.

## Oracle Pro

- <Status priority='1'>New</Status> Added the 1-minute option for the statistical data collection cycle (`sqlstat_send_interval=1`, default: `5`).
