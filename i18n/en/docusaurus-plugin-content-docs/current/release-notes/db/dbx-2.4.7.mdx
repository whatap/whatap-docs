March 18, 2025

## MongoDB

<Status priority='0'>Fixed</Status> Changed the default value of metrics collection option, v2 from `false` to `true`.

- Previous default: `v2=false` → Present default: `v2=true`

- When `v2=true` is set, only `opcounters`, `globalLock`, `extra_info`, `connections`, `network`, `mem, opLatencies`, and `asserts` are collected and stored among the `serverStatus` values.

- If you want to collect additional metrics, you can proceed by specifying a `serverStatus` output option such as `category=network,repl,indexStats…`.
