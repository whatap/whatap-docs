March 14, 2025

## PostgreSQL

<Status priority='1'>New</Status> Added the `node_state` and `DB size` collection features for Yugabyte DB.

- Basically, it collects the DB size and index size by using the `yb_pg_database_size` and `yb_pg_indexes_size` functions.

- If you use a different function name, you can specify it through the `yb_pg_database_size=` and `yb_pg_indexes_size=` options.
