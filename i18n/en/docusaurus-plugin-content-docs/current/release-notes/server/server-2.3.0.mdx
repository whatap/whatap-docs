## Release details

Release date: 2023-01-11

-   <Status>Change</Status> Changed the TCP port check function to be asynchronous so that there is no effect on the performance collection upon access delay.

-   <Status>Feature</Status> Added the option to exclude files by the log monitoring plug-in.

-   <Status>Feature</Status> Added the function to reflect the AWS EC2 and Azure VirtualMachine meta data and tags on the metrics.
