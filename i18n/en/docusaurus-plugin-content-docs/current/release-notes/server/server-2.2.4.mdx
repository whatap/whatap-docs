Release date: 2022-07-29

## Improving the extension API function

Added the function for the extension API to delete the extension configuration file.

## Adding the inventory extension function

Added the kernel_version column to the server_inventory category.

## Removing the killall command from the RPM installation package

Fixed the stop error with the killall command not installed on the server using the RPM package.
