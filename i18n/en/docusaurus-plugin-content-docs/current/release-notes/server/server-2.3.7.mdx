## Release details

배포일: 2023년 6월 8일

<p><Status>Fixed</Status> <code>hostguid.enabled=true</code> 옵션을 적용하면 <b>AWS EC2</b>를 포함한 가상 머신의 <code>Guid</code> 값을 <code>oid</code>(에이전트 고유 번호) 생성 시 포함하도록 수정</p>

-   IP 주소가 재활용되더라도 <code>oid</code>가 달라지도록 변경하였고, 해지된 <code>oid</code>를 재활용할 때 <code>Guid</code> 값을 이용해 기본 알림 적용이 누락될 수 있는 오류를 방지합니다.
