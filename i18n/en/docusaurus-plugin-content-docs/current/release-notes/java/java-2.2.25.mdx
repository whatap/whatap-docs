December 20, 2023

- <Status>Feature</Status> Added the multi-transaction ID (<code>@mtid</code> tag) in the Java application log for log monitoring.

- <Status>Changed</Status> Modified to collect transaction IDs and multi-transaction IDs when the <code>logsink_enabled</code> option has been set (Previous: <code>logsink_trace_enabled</code> option).

  ```ini title="whatap.conf"
  # default: false
  logsink_enabled=false
  ```

- <Status>Changed</Status> Modified to collect appenders by default upon use of log4j.

  ```ini title="whatap.conf"
  # requires agent restart
  weaving=log4j-2.17

  # example: collect Console, AccessLog, ErrorLog 
  trace_log4j_appender_set=Console,AccessLog,ErrorLog
  ```

- <Status>Changed</Status> Modified to collect appenders by default upon use of logback.

  ```ini title="whatap.conf"
  # requires agent restart
  weaving=logback-1.2.8

  # example: collect ErrorAppender, FileAppender log
  trace_logback_appender_set=ErrorAppender,FileAppender
  ```
