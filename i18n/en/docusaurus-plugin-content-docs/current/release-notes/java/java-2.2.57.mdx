April 16, 2025

- <Status priority='1'>Feature</Status> When the JVM process starts, the JVM settings are displayed in the application log.

- <Status priority='1'>Changed</Status> Modified to detect whether the Valve tracing point has been selected through the setting due to Tomcat's java.security issue.

  ```bash
  # default: false
  hook_valve_http_enabled=true
  ```

  - To collect the spring-security exception and error status code in the spring-boot, add settings and then start the system again.

- <Status priority='0'>Fixed</Status> Prevented the parent txid from being truncated when tracing the spring async-annotation in the record summary.
