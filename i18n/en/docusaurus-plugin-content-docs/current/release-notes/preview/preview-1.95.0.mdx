## Official Release: 2023-01-16

### `Common`

#### Changing the terms

- <Status>Change</Status> Changed the term from <b>profile</b> to <b>trace</b>.

#### Log setting

- <Status>Change</Status> Deleted the storage column from the <UI>Data retention period by category</UI> table.

- <Status>Feature</Status> In the <UI>Data retention period by category</UI> table, added the <UI>Daily Log Count</UI> column and helps for each column.

- <Status>Feature</Status> In the <UI>Log Parser Settings</UI> section, when registering the status success rate parser and the 4xx, 5xx parser, added validation so that the parser cannot be registered twice in the same category.

#### Manages the account information

- <Status>Feature</Status> In <UI>My Profile</UI>, added the message window for final confirmation when enabling the multi-factor authentication.

#### Flex Board

<Status>Change</Status> Modified the <UI>Log RPS Board</UI> dashboard template.

- Change of the default value in the data lookup period: real-time 30 minutes → real-time 12 hours
- Modified the layout height of the **_Log RPS trend_** and **_server status_** widgets.
- Setting the **_4XX, 5XX Count_** widget's auxiliary table data units
- Changed the widget data call interval for some widgets from 5 seconds to 1 minute.

### `Application`

#### Transaction Detail

The items have been added and modified in the following menu path.

:::note

- Click `![See details icon](/img/right-arrow.svg)` on the right of the **_Dashboard_** > **_Application Dashboard_** > **_Active Transaction_** widget > select a finished transaction item > click **_Show Transaction_**.
- Click `![See details icon](/img/right-arrow.svg)` on the right of the **_Dashboard_** > **_Application Dashboard_** > drag the **_Hitmap Transaction_** chart > click a desired item from the **_TX Trace_** list.
- Drag the **_Dashboard_** > **_Transaction Map_** > **_Transaction_** chart > In the **_TX Trace_** list, click ![See details icon](/img/ico-detail-view.svg) on the upmost left.
- Select the finished transaction column on the **_Dashboard_** > **_Active Transaction_** > **_Transaction_** tab > Click **_Show Trace_**.
- Drag **_Analysis_** > **_Hitmap_** > **_Hitmap Transaction_** chart > Click a desired item in **_TX Trace_** list.

:::
