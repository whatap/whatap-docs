## Official Release: 2023-02-08

### `Common`

#### Flex Board

<Status>Feature</Status> In the dashboard list under <UI>Home</UI> > <UI>Flex Board</UI>, added the Share Dashboard (<img src='/img/ico-flexboard-share.svg' alt='sharing icon' />) function.

#### Analysis

- <Status>Fixed</Status> In <UI>Sitemap</UI> > <UI>Analysis</UI> > <UI>Metric Anomaly Detection</UI>, replaced the UI logic due to the UI bug upon selection of category and field.

- <Status>Fixed</Status> Modified not to change the <UI>start date</UI>, <UI>metrics of interest</UI>, and dashboard tab, if query is not finished in <UI>CUBE</UI>.

#### Transaction Detail `Application`

- <Status>Change</Status> Changed the <UI>SQL</UI> analysis tab name to <UI>SQL Summary</UI>.

- <Status>Change</Status> Changed the <UI>method analysis</UI> tab name to <UI>Method Summary</UI>.

- <Status>Feature</Status> Added the <UI>HTTP Calls</UI> tab: Provided the HTTP call count, time sum, and time average.

#### Management

- In **_Management_** > **_Project Member_**, the following functions have been added.

  - <Status>Feature</Status> Added the function to display the member affiliation type (<UI>group</UI>/<UI>project</UI>) for improved project member management.

  - <Status>Feature</Status> Added the tooltip (deletion is impossible) when the member deletion (<img src='/img/ico-trash.svg' alt='delete icon'/>) button is disabled.

- In **_My Usage_** and **_Billing Management_**, the following functions have been modified.

  - <Status>Change</Status> In <UI>Preview Invoice</UI> and <UI>Usage</UI>, changed the log display and expression units according to the change in log charging policy.

    - Change of the log display unit: LOGs ⇒ LOG UNITs
    - Change of the log representation unit: Modified to display the daily average for the summed log line count.
    - In <UI>Preview Invoice</UI>, changed to display the log usage in the <UI>Metered Units</UI> column.
    - In <UI>Usage</UI>, changed the metering summary column for usage details.
