-   <Status>Changed</Status> Modified to select <UI>Default</UI> among the <UI>Interval</UI> selection options, when any <UI>time</UI> search section is changed.

    When changing the search section, the **_Interval_** option is selected as **_Default_** in the following cases:

    -   The previous selection in the **_Interval_** option is **_Default_**.

    -   The previous selection in the **_Interval_** option is not **_Default_** and it is not in the changed **_Interval_** option list.

    -   The previous selection in the **_Interval_** option is not **_Default_** and the selected **_Interval_** option value is less than half of the search period.

-   <Status>Fixed</Status> Fixed the error where, in some situations, pressing <UI>OK</UI> in the <UI>Column settings</UI> pop-up displays an unknown error message and it does not work.

-   <Status>Fixed</Status> In <UI>Manage account INFO.</UI>, fixed the error where the country code number is defaulted to blank when entering a phone number using WhatsApp.
