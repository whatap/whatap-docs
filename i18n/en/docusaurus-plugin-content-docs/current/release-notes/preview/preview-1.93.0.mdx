Through <https://preview.whatap.io>, you can experience the following new features. 

## 2022-11-24

### `Common`

<Status>Change</Status> In <UI>Event Setting</UI> and <UI>Notification Setting</UI>, if the reception tag was deleted while using the <UI>Event Receiving Tag</UI> function, the alert reception was impossible. Accordingly, the tag deletion has been temporarily disabled in the following menus.

-   Application products: Metrics (including the composite metrics), hitmap patterns, anomaly detections, logs
-   Server products: Metrics (including the composite metrics), anomaly detections, logs
-   Kubernetes products: Metrics (including the composite metrics), hitmap patters, anomaly detections, logs
-   Database products: Metrics (including the composite metrics), anomaly detections

## 2022-11-16

### `Common`

-   <Status>Fixed</Status> While loading the data in Server Suite Project > <UI>Sever List</UI>, deleted the message, "No agent installed" and modified to display the loading spinner.

-   <Status>Fixed</Status> When going to the project list by selecting a group on the initial screen, modified to display a loading spinner when a delay occurs.

-   <Status>Feature</Status> In <UI>Setting</UI>, under <UI>Log Parser Setting</UI>, added the <UI>4xx, 5xx Status Code Parser</UI> and <UI>Status Code Success Rate Parser</UI>.

-   <Status>Fixed</Status> In <UI>Live Tail</UI>, fixed the malfunction of the content lookup function.

-   <Status>Fixed</Status> In <UI>Live Tail</UI>, fixed the malfunction of the content filtering (highlighting) function.

### Daily Application Status/Performance Trend

<Status>Fixed</Status> Fixed the malfunction of the lookup button in the following menus for the Kubernetes product suite projects.

-   <UI>Application</UI> > <UI>Daily Application Status</UI>
-   <UI>Application</UI> > <UI>Perf. Trend</UI>

### Agent Installation

<Status>Fixed</Status> Kubernetes Suite Project > <UI>Management</UI> > <UI>Agent Installation</UI> > <UI>Application Agent</UI>, changed the Java agent installation jar file name (whatap.agent-X.Y.Z.jar → whatap.agent.kube.jar).

### Maintenance Plan

In Project > <UI>Management</UI> > <UI>Maintenance Plan</UI>, added and modified the following functions.

-   <Status>Feature</Status> Added the function to display the maintenance plan notice at the top of the screen.
-   <Status>Change</Status> Maintenance Plan UI improvements
    * Changed the repeat option UI text: <UI>Daily</UI>/<UI>Weekly</UI>/<UI>Monthly</UI>/<UI>Yearly</UI>
    * Changed the maintenance plan table column order.
    * Improved the maintenance plan addition/modification modal.
    * Upon addition of any maintenance plan, modified to select <UI>Once</UI> by default.
    * Modified the maintenance plan guide texts and their locations.
-   <Status>Fixed</Status> Modified not to delete any server maintenance plan.

### Metrics Chart

<Status>Change</Status> In Kubernetes Suite Project > <UI>Analysis</UI> > <UI>Metrics Chart</UI>, fixed the no response status that occurs when searching a tag list that exceeds a specific size: changed the tag list search API `search` → `tags`.

### HITMAP (Transaction List)

<Status>Change</Status> In Kubernetes Suite Project > <UI>Application</UI> > <UI>Hitmap</UI>, modified to add the <UI>container ID</UI> and <UI>pod name</UI> columns.

### Flex Board

In Project > <UI>Dashboard</UI> > <UI>Flex Board</UI> or Home > <UI>Flex Board</UI>, added or modified the following functions.

### Project Subscription

### Invoice
