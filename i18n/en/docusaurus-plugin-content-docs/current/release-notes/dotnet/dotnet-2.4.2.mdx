April 26, 2025

- <Status priority='1'>New</Status> Modified to automatically select a version based on the .NET runtime version of the installed server.

    :::note 

    To manually change the applied version, enter the desired version in the `WHATAP_DOTNET_TARGETFRAMEWORK` environment variable and then restart IIS. You can select any of the following versions.

  - net45
  - net461
  - net462
  - net47
  - net472
  - net48

    :::

- <Status priority='0'>Fixed</Status> Fixed the bug where the hitmap for ISAPI filter-based transactions appears in blue even when an error occurs.
