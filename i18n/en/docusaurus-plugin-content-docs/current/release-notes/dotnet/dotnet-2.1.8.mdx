October 24, 2023

-   <Status>Fixed</Status> Fixed the initialization error that occurs intermittently during .NET agent service startup.

    :::note

    If an initialization error occurs, the WA172 log is repeated and then monitoring is not activated.

    :::

-   <Status>Fixed</Status> Fixed the error that occurs in a specific situation while collecting the HTTP request body (written in XML).

    :::note

    This error does not occur in case of JSON.

    :::
