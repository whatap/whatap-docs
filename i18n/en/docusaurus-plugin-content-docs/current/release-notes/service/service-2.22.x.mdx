### `Common`

- <Status priority='1'>Feature</Status> Added agent list and CSV download to <Cmdname sid="management" className="uitext" />

- <Status priority='1'>Feature</Status> Improved **GPU Inventory** by marking custom columns with a custom tag in the table header for easier distinction

- <Status priority='1'>Feature</Status> Added file monitoring event settings in <Cmdname sid="event_setting" className="uitext" /> > **File Keyword Watch**, **Windows Event Watch**, **File Watch** tabs

- <Status priority='0'>Fixed</Status> Fixed an issue where the **GPU Performance Summary** BAR1_Total_Memory field value was not formatted with commas

    - Added preset functionality (save sort and filter values)

- <Status priority='1'>Feature</Status> Added Whatap custom GPU metrics to the Pod list

- <Status priority='1'>Feature</Status> Added functionality in the **GPU Dashboard** drawer (opened by clicking the map chart) to query metrics for a selected time range

- <Status priority='1'>Changed</Status> In **GPU Dashboard**, M/P labels are now displayed even when shapes on the map chart are very small

- <Status priority='0'>Fixed</Status> Fixed an issue in <Cmdname sid="workload" className="uitext" /> where some Service information was not linked in the Ingress list
