June 18, 2025

- <Status priority='0'>Feature</Status> Reduced CPU load

  - Reduced CPU load by changing the criteria for monitoring targets.

  - Further reduced CPU load when user function monitoring is disabled.

    `whatap.profile_method_enabled=false`

- <Status priority='1'>New</Status> CLI parameter URL display

  - Collects URLs including parameters when running CLI commands.

  - Example: php /home/www/test.php aaa bbb â†’ Collected as /home/www/test.php aaa bbb

- <Status priority='1'>New</Status> Collection of file parameter info

  - Collects up to 20 parameters, each up to 256 bytes (partial data only).

- <Status priority='0'>Fixed</Status> Memory leak fix

  - Fixed memory leak caused by calling the error function when a DB function fails.

:::info
