March 10, 2025

<Status priority='1'>New</Status> Added the option to use the Prisma ORM.

- **prisma_read_func_name** <span class='type'>String</span>

  Default `read`

  Set the name of the function called to obtain a read-only client when using replication.

- **prisma_database_url_name** <span class='type'>String</span>

  Default `DATABASE_URL`

  Set the environment variable name in Prisma ORM where the database connection URL is stored.
