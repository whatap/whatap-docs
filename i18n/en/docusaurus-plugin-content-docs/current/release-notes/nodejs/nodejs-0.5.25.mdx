August 22, 2025

- <Status priority='1'>New</Status> **perfx_nodejs_gc_enabled** <span class='type'>Boolean</span> 

    * Default: `false`

    * Enables collection of Garbage Collection (GC) performance metrics. (Supported in Node.js version 16.7 or higher)

- <Status priority='1'>New</Status> Detects when the application process terminates and records the termination reason in the agent log.

    - **SIGTERM**: Normal termination request from the system or process manager  
    - **SIGINT**: Interruption by the user (Ctrl+C or kill command)  
    - **SIGHUP**: Terminal disconnection or parent process termination  
    - **SIGUSR1/SIGUSR2**: User-defined signals  
    - **uncaughtException**: Abnormal termination due to an unhandled exception  
    - **unhandledRejection**: Termination due to an unhandled Promise rejection
