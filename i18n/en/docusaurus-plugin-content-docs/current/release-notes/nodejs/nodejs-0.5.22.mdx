June 30, 2025

- <Status priority='1'>New</Status> <code>profile_mongodb_param_enabled</code> <span class='type'>Boolean</span> 

    * Default: <code>false</code>

    * Added an option to display actual query parameter values when tracing the mongoose module

- <Status priority='0'>Fixed</Status> Fixed an issue where three separate transactions (API request, Socket.IO handshake, and polling) were recorded for a single client request when using both an HTTP server and a Socket.IO server together

    * Now only the API request is recorded as a single transaction. Socket.IO handshake and polling requests are no longer recorded as separate transactions
