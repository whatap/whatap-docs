February 25, 2025

- <Status priority='1'>Feature</Status> Added the option to specify the span's `name` value as the transaction name.

  - **otel_tx_path_setby_name_enabled** <span class='type'>bool</span> (default value: `false`)

- <Status priority='1'>Feature</Status> Added the option for multi-transaction connection to recognize `SpanID` of the trace span as `ParentSpanID` for external connection.

  - **otel_mtrace_stepid_setby_trace_spanid_enabled** <span class='type'>bool</span> (default value: `false`)

- <Status priority='1'>Feature</Status> Added the log option to output the same content as the file log to `Stdout`.

  - **log_stdout_enabled** <span class='type'>bool</span> (default value: `false`)

  :::note 

  You can register and use environment variables in the Docker and Kubernetes environment. 

  :::

- <Status priority='0'>Fixed</Status> Fixed the issue where the "file not found" error log continues to be output when the *whatap.conf* file does not exist.

- <Status priority='0'>Fixed</Status> Fixed the issue where the transaction ID appears as `0`.

:::info
