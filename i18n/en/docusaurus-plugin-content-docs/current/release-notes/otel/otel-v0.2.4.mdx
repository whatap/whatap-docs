April 30, 2025

- <Status priority='1'>Feature</Status> The HTTP status error appears.

  - Setting to display an error when the HTTP status code is 400 or higher

    - Key: **transaction_status_error_enable** 
    - Type: <span class='type'>bool</span> 
    - Default: `false`
    - Collection interval: 10 seconds

- <Status priority='1'>Feature</Status> The HTTP status code statistics are collected.

  - Setting to collect the number of cases by status code for all URLs

    - Key: **tx_status_meter_enabled** 
    - Type: <span class='type'>bool</span> 
    - Default: `true`
    - Collection interval: 10 seconds

  - Setting to collect the number of status codes by URL 

    - Key: **stat_txstatus_enabled** 
    - Type: <span class='type'>bool</span> 
    - Default: `true`
    - Collection interval: 5 minutes

- <Status priority='1'>Feature</Status> The HTTP call's virtual span information for distributed tracing has been set for analysis.

  - The virtual span is activated when there is no HTTP call span information.

    - Key: **otel_virtual_httpc_enabled** 
    - Type: <span class='type'>bool</span> 
    - Default: `false`

  - The name is set when there is no HTTP call span. 

    - Key: **otel_virtual_httpc_name** 
    - Type: <span class='type'>string</span> 
    - Default: `external http call`

- <Status priority='1'>Feature</Status> The gateway information is collected.

  - The gateway information collection is activated.

    - Key: **otel_gateway_enabled** 
    - Type: <span class='type'>bool</span> 
    - Default: `false`
    - Collection items and cycle  
      - URL-based transaction, gateway, and API call's elapsed time collection cycle: 10 seconds  
      - Transaction/gateway/API call's elapsed time collection cycle by URL/status: 5 minutes

  - Set the elapsed time of the gateway with the value of a specified attribute.

    - Key: **otel_attr_tx_gw_time_keys** 
    - Type: <span class='type'>string</span> 
    - Default: `duration.kong`

  - Set the elapsed time of the API call with the value of a specified attribute. 

    - Key: **otel_attr_tx_api_time_keys** 
    - Type: <span class='type'>string</span> 
    - Default: `duration.proxy`

  - Set the response time of the gateway with the value of a specified attribute.  

    - Key: **otel_replace_tx_resp_time_enabled** 
    - Type: <span class='type'>bool</span> 
    - Default: `false`  
    - Key: **otel_attr_replace_tx_resp_time_keys** 
    - Type: <span class='type'>string</span> 
    - Default: `duration.kong`

- <Status priority='0'>Fixed</Status> Fixed the HTTP method ignore setting error.

  - `PATCH`, `OPTIONS`, `HEAD`, and `TRACE` methods are not collected into transactions.
