July 9, 2025

* <Status priority='2'>Feature</Status> Recognize Spans without a Parent as Transactions  
  * Spans without a parent are treated as transactions regardless of `kind`  
  * Previously, only `SPAN_KIND_SERVER` was treated as a transaction  
  * Related option: `otel_tx_root_span_enabled` (default: true)

* <Status priority='2'>Feature</Status> Improved metrics collection method  
  * Changed from log-based collection to Prometheus-compatible collection  
  * Can be viewed via the Flex board (MXQL data query)

* <Status priority='1'>Change</Status> Modified behavior of `query_string_enabled` option  
  * When `query_string_enabled` is true and `query_string_urls`, `query_string_keys` are not set,  
    query strings of all URLs are collected

* <Status priority='0'>Fixed</Status> Fixed error in collecting HTTP Status counters  
  * Fixed an issue where 5xx error status totals were not collected
