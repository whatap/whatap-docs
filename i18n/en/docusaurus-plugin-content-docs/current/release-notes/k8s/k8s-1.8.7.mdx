> **Includes Java Agent v2.2.61**

May 8, 2025

- <Status priority='1'>New</Status> Added tag counts to the `kube_pod` tag count category to check the Pod status and ownership.

  - `kube_display_status`: Display status of a Pod

- <Status priority='1'>New</Status> Expanded the tag count resource scope by adding the tag count for workload resources in addition to the existing `kube_pod`.

  - `kube_replicaset`
  - `kube_statefulset`
  - `kube_daemonset`

- <Status priority='1'>New</Status> Added the `container_gpu` field to support the MIG environment.

- <Status priority='0'>Fixed</Status> Fixed the bug where the `server_disk` data is missing.
