March 28, 2025

&nbsp; **Java v2.2.55**

- <Status priority='1'>New</Status> Added the `podUid` tag to the `containerStdOut` log.

- <Status priority='1'>New</Status> Added the `containerid` tag to the `kube_oom_event` tag count.

- <Status priority='1'>New</Status> Added the `GPU_I_PROFILE`, `GPU_I_ID`, and `DCGM_FI_DRIVER_VERSION` metrics to the `container_gpu` category in the MIG environment.

- <Status priority='0'>Fixed</Status> Fixed the issue where the `container_gpu` tag count is not collected in the environment where there are multiple GPU Compute instances, with a mix of active and unused GPUs.

- <Status priority='0'>Fixed</Status> Fixed the issue where the collection times are not sorted every 5 seconds when collecting the GPU data.
