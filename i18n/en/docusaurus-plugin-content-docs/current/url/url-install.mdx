## Project Creation

Create a project before adding the URL.

1. Log in [WhaTap monitoring service](https://service.whatap.io).

2. To create a project, on the left of the screen, select <Cmdname sid="BTN07332" className="uitext" /> &gt; **_+_** <Cmdname sid='BTN06696' className='uitext'/>.

3. In <Cmdname sid="TTL07073" className="uitext" />, create a URL monitoring project.

4. Configure the settings for <Cmdname sid="project_name" className="uitext" />, <Cmdname sid="data_server_region" className="uitext" />, and <Cmdname sid="project_time_zone" className="uitext" />.

   <ImgLang img='create-project-v3-url.png' desc='url monitoring project' />

5. In <Cmdname sid="TTL07727" className="uitext" />, select the language for alert messages.

6. After all settings are finished, select <Cmdname sid="creating_a_project" className="uitext" />.

7. Check the new project in the project list.

:::note

- A <Cmdname sid="data_server_region" className="uitext" /> refers to a region (a bundle of data centers installed to provide cloud services). Selecting a specific region stores your data in the datacenter that belongs to that region.

- <Cmdname sid="project_time_zone" className="uitext" /> is the reference time for generating alerts and reports.

- To group multiple projects for management, select a group from <Cmdname sid="project_groups" className="uitext" /> or add a group. For more information about grouping, see [the following](../project/group).

:::

:::caution

Up to 10 URL projects can be created. The number of URLs that can be registered in a URL project for free is unlimited.

:::

## Adding a URL{#add-url}

1. Select the generated URL project. A screen with no URL set appears.

2. Select **_+_** <Cmdname sid="add_url" className="uitext" /> on the upper right of the screen. 

3. Proceed to perform URL settings in the input fields.

   <ImgLang img='url-input-field.png' desc='URL input field' />

   - <Cmdname sid="name" className="uitext" />: Used as a unique identification for URL settings. Duplicates are not allowed, and if you register with the same name when resetting, statistical data with the name will be considered as data from the same URL.

   - <Cmdname sid="url" className="uitext" />: Register the URL to check. Enter the full path of URL to call. It supports the http/https protocol.

   - <Cmdname sid="check_location" className="uitext" />: Select an agent provided by WhaTap. It provides the regions such as Seoul, Mumbai, Tokyo, Singapore,Frankfurt, Califonia, and Jakarta.

     :::note

     Select <Cmdname sid="TTL08547" className="uitext" /> to check the IP address of the region. Check your firewall rules and allow the IP address if necessary. For more information, see <Link to="#select-confirm-region">the following</Link>.

     :::

   - <Cmdname sid="collection_cycle" className="uitext" />: Set the interval to collect monitoring data. The default value is <Cmdname sid="1_min" className="uitext" />(60000 ms). You can select up to 5 minutes in 1-minute increments.

4. After configuration, select <Cmdname sid="save" className="uitext" />.

:::note

- It takes about 1 minute for the added/modified/copied URL to be reflected in the list.

- For more information about <Cmdname sid="event_setting" className="uitext" />, see <Link to='url-event'>the following</Link>.

:::

## Running Monitoring{#url-st}

<ImgLang img='url-monitoring.png' desc='URL Monitoring screen' />

### Condition setting option

Select a project at Home screen and go to <Cmdname sid="url_list" className="uitext" />. Through the option at the top of ![number-1](/img/number-01.png), you can specify the view format and sorting method, or view the list through search.

- ![List View Icon](/img/ic-list.svg) <Cmdname sid="list_view" className="uitext" />: You can view the <Cmdname sid="url_list" className="uitext" /> in the list format. The default setting is <Cmdname sid="list_view" className="uitext" />. 

- ![Card View icon](/img/ic-project.svg) <Cmdname sid="card_view" className="uitext" />: You can view the <Cmdname sid="url_list" className="uitext" /> in the card format.

- <Cmdname sid="search" className="uitext" />: You can search based on the URL or the name of the specified URL in the input field.

In the ![number-2](/img/number-02.png) area, you can see and filter the counts for each final response status in the list of registered URLs. The view can be seen based on <Cmdname sid="status_total" className="uitext" />, **_9XX_**, <Cmdname sid="status_5xx" className="uitext" />, <Cmdname sid="status_4xx" className="uitext" />, <Cmdname sid="status_3xx" className="uitext" />, and <Cmdname sid="status_2xx" className="uitext" />. The default setting is <Cmdname sid="status_total" className="uitext" /> view.

### Exporting and importing URL List

In the ![number-3](/img/number-03.png) area, you can export or import the current <Cmdname sid="url_list" className="uitext" /> in Excel format. 

- ![Import icon](/img/ico-import.svg) <Cmdname sid="import" className="uitext" />: By selecting an Excel file, the URL list is imported. 

- ![Export icon](/img/ico-export.svg) <Cmdname sid="export" className="uitext" />: The current URL list is downloaded in an Excel file through the web browser.

### URL Monitoring Detail

In the URL list table of the ![number-4](/img/number-04.png) area, you can see the monitoring check results for each registered URL. The columns are as follows. To register or edit a URL, select ![Edit icon](/img/ic-edit.svg) <Cmdname sid="edit_url" className="uitext" /> to switch to the editing screen.

- <Cmdname sid="current_status" className="uitext" />

  It is the final check response code (HTTP Status Code). If you select each status code, the HTTP status code cleanup document appears.

- <Cmdname sid="name_and_url" className="uitext" />

  This is the unique ID and URL specified upon registration of the URL.

- <Cmdname sid="check_region" className="uitext" />

  Check location.

- <Cmdname sid="elapsed" className="uitext" />

  It has the 20-minute check result (response code and time) and final response time. When searching for a time range by dragging in the chart area, you can see the response codes and response times for the time period as follows.

  <ImgLang img='url-monitoring-dt.png' desc='Detail' />

:::note 

**Use of WhaTap URL Monitoring service**

URL Monitoring is provided as a free service until there is any change of payment policy. You do not need to install separate agents and can use the WhaTap agents and server repository.

:::
