## Creating a project{#create-project}

</Xclude>

Create a project before installing the agent.

1. Log in to the WhaTap Monitoring Service.

1. In the left side menu, click <Cmdname sid="BTN07332" className="uitext" /> > + <Cmdname sid='BTN06696' className='uitext' />.

1. On the <Cmdname sid="TTL07073" className="uitext" /> screen, select the product you want to install.

1. Enter or select the following items:

   * <Cmdname sid="project_name" className="uitext" />: Enter the project name.

   * <Cmdname sid="data_server_region" className="uitext" />: Select the data server region. A region is a group of data centers that provide cloud services. Your data will be stored in the selected region.

   * <Cmdname sid="project_time_zone" className="uitext" />: Set the time zone to be used for alerts and report generation.

   * <Cmdname sid="TTL07727" className="uitext" />: Set the language for alert notifications. (Supports Korean and English)

   * <Cmdname sid="project_groups" className="uitext" />: Group multiple projects for easier management. Select a group if applicable.

   * <Cmdname sid="project_description" className="uitext" />: Add additional details or a description for your project.

1. After completing all settings, click the <Cmdname sid="creating_a_project" className="uitext" /> button.

:::note

If you add a project while an organization is selected, you must set the <Cmdname sid="group_of_organization" className="uitext" /> field.

For more information about groups, see the related document.

:::
