:::note

If the data for specific columns does not appear, your Kubernetes node agent may be outdated. In this case, update the Kubernetes agent to the 1.3.7 or later so that enough data can be collected. 

:::

You can check the node information of the current project in a list.

- Real-time data is updated every 10 seconds.

- It provides the functions for node status summary, comparison, filtering, and _CSV_ file download.
  > CSV file name, _kubernetes\_node\_list\_\{pcode}_\{timestamp}.csv_

## Basic screen guide

<ImgLang img='k8s-resource-node-list-m.png' desc='Node list basic screen' />

You can see data by selecting a desired option at the top of <Cmdname sid="node_list" className="uitext" />.

- You can see the number of all nodes and the numbers of nodes for each state.

- If you select a desired column, you can search by sorting <Cmdname sid="node_list" className="uitext" /> based on the selected column.

## Comparing

The **_Compare_** option allows you to see a comparison chart of key metrics for multiple nodes. When you select the **_Compare_** check boxes of the nodes to view in the list, a window comparing details for the nodes appears on the right as shown below. 

<ImgLang img='k8s-resource-node-list-compare.png' desc='Compare' />

## View Detailed {#node-list-details}

Node details can be checked through the **_Detail_** option for details on the selected node. If you select the ![Detail icon](/img/ico-detail-view.svg) icon in the **_Detail_** column of the node to view in the list, the detailed information window for the node appears on the right.
