Before starting the Kubernetes monitoring, check out the following supports. 
WhaTap Kubernetes supports standard Kubernetes version 1.16 or later and can be installed in the following corresponding environments.

## Supported platforms

| Platform | Support |
| --- | :---: |
| Amazon EKS (Elastic Kubernetes Service) | ✅ |
| AKS (Azure Kubernetes Service) | ✅ |
| GKE (Google Kubernetes Engine) | ✅ |
| OKE (Oracle Container Engine for Kubernetes) | ✅ |
| Naver Cloud Kubernetes Service | ✅ |
| NHN Kubernetes Service (NKS) | ✅ |
| Red Hat OpenShift (3.x / 4.x) | ✅ |
| Cocktail | ✅ |
| Accordion | ✅ |
| PaaS-TA | ✅ |
| Kakao i Cloud | ✅ |
| Rancher | ✅ |
