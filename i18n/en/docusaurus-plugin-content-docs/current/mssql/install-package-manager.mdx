패키지 매니저를 이용한 와탭 데이터베이스 모니터링 에이전트 설치 방법을 안내합니다.

:::tip

**설치 방법 선택**

* `apt` 패키지 매니저를 사용하는 Debian/Ubuntu 계열 시스템의 경우 이 문서를 참고하세요.
* Docker 환경에서는 [Docker 설치](install-docker)를 참고하세요.
* 기타 환경에서는 [직접 설치](install-agent)를 참고하세요.

:::

:::note

**지원 데이터베이스**

MySQL, PostgreSQL, SQL Server
