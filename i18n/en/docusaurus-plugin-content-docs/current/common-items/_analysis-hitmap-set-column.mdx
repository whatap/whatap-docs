### Setting the columns in the trace list

You can additionally see the details about the search result by setting the columns in the <Cmdname sid="trace_list" className="uitext" /> section.

1. In the **TX Trace** area, click the ![Column Icon](/img/ico-column.svg) <Cmdname sid="column_select" className="uitext" /> button.  

1. Select or deselect the desired column items. The changes are immediately reflected in the table.  

    - <Cmdname sid="select_all" className="uitext" />: It selects all columns.

    - <Cmdname sid="deselect_all" className="uitext" />: All columns are unselected.

    - <Cmdname sid="initialize_default" className="uitext" />: The column is set as the default value.

1. Once finished, click the ![Close Icon](/img/ico-close.svg) icon in the upper-right corner.
