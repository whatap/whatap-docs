### Modifying and deleting the event reception tag

1. In the user list of the <Cmdname sid="alert_receive_settings_by_member" className="uitext" /> section, select <Cmdname sid="add_tag" className="uitext" /> or <LinkImage img="ico-add.svg" desc="add icon" />.

2. If the <Cmdname sid="event_tag" className="uitext" /> pop-up window appears, select <LinkImage img="ico-edit.svg" desc="Edit icon" /> of the item to modify or delete in <Cmdname sid="event_tag_list" className="uitext" />.

3. Modify the tag name or change the color and then select <Cmdname sid="apply" className="uitext" />.

   * To delete the tag, select ![Delete icon](/img/ico-trash-red.svg) <Cmdname sid="delete_tag" className="uitext" />.

:::note

You cannot delete the <Cmdname sid="event_tag" className="uitext" /> item applied to the event.

:::
