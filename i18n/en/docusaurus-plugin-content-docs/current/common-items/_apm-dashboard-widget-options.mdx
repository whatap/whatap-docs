## Using the widget options

The icons displayed on the widget function as follows:

<InDoc product="java,php,python,nodejs,dotent,golang">

- ![Information icon](/img/ic-info-circle.svg): You can see the features and information for major widgets.

- ![Merge icon](/img/ico-data-merge.svg) <Cmdname sid="TLT06734" className="uitext" />: The agent data for the widget is displayed individually or merged in a graph.

- ![Detail icon](/img/ico-detail.svg) <Cmdname sid="detail" className="uitext" />: A modal window appears where you can search the data of the widgets separately by agent.

</InDoc>

<InDoc product="postgresql,mysql,redis,mongodb,oracle-pro,oracle,altibase">

- ![Information icon](/img/ic-info-circle.svg): You can see the features and information for major widgets. (multilingual support planned)

- ![Up arrow icon](/img/up-arrow.svg) or ![Down arrow icon](/img/down-arrow.svg): You can enlarge or reduce the range of the vertical axis.

- ![Fullscreen icon](/img/ic-fullscreen.svg): You can view the widget's data on a wider screen.

- ![Detail icon](/img/ico-detail.svg) : A detail window appears where you can search the data of the widgets separately by agent.

</InDoc>

:::note

The provided options may differ depending on the widget.

:::
