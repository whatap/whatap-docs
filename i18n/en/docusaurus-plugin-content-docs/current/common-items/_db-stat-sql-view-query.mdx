### Checking the query-based execution trend

<Xclude product='postgresql,postgresql-v1'>

On the utmost left for each item of the table, select ![number1](/img/number-01.png) ►. A chart appears that displays the number of executions every 5 minutes and the average execution time of the query statement. If you hover your mouse over each bar graph, you can see the number of that time.

To classify and view data by usage category, select the desired tab from ![number2](/img/number-02.png).

</Xclude>
