## Downloading the DBX agent

Download the DBX agent file. Use the following two methods.

- You can download it by using the 'wget' command.

  <InDoc product="postgresql">

  ```bash title="BASH"
  wget -O whatap.agent.database.tar.gz "https://service.whatap.io/download/dbx_agent?type=postgresql&format=tar.gz"
  ```

  </InDoc>

- If you cannot download it with the command, select <Cmdname sid="download" className="uitext" /> on the WhaTap monitoring service screen.

Download the DBX agent file and then unzip it.

```bash title="BASH"
tar -zxvf whatap.agent.database.tar.gz
```

:::note

For users who cannot download tar files due to security settings, ZIP files are also provided. On the installation screen, select the <UI>.zip</UI> <Cmdname sid="download" className="uitext" /> button.

:::
