### Filtering the trace list

You can filter the results retrieved from the <Cmdname sid="trace_list" className="uitext" /> section using the following options:

<ImgLang img="analysis-hitmap-tx-list-filter.png" desc="TX trace" />

- <Cmdname sid="view_all" className="uitext" />: You can see all transactions in the viewed result.

- <Cmdname sid="active_stack" className="uitext" />: Only the transactions that include active stacks can be viewed in the searched result.

- <Cmdname sid="multi_transaction" className="uitext" />: Only the transactions including multi-transactions can be checked in the searched result.

- **Search URL**: Only transactions containing items that match the entered string are displayed. Enter a desired string and then press `Enter` on the keyboard.

    * <Cmdname sid="transactionUrl" className="uitext" />
    * <Cmdname sid="TTL06081" className="uitext" />
    * <Cmdname sid="TTL06101" className="uitext" />
    * <Cmdname sid="TTL06086" className="uitext" />
    * <Cmdname sid="TTL06088" className="uitext" />
