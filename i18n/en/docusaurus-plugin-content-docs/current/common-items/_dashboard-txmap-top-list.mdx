## Checking the upper list

In the **_Top 30_** section, the list of top 30 items with the highest transactions performed based on <Cmdname sid="transaction" className="uitext" />, <Cmdname sid="domain" className="uitext" />, <Cmdname sid="ip" className="uitext" />, and <Cmdname sid="agent" className="uitext" />. 

- <Cmdname sid="transaction" className="uitext" />: Top 30 items, aggregated by the transaction URL

- <Cmdname sid="domain" className="uitext" />: Top 30 items aggregated based on the Internet address specified in the address of the IP connected by the client

- <Cmdname sid="ip" className="uitext" />: Top 30 items, aggregated by the client IP address

- <Cmdname sid="agent" className="uitext" />: Top 30 items aggregated by the agent included in the project

If you select each item from the list, the data is applied based on the item selected in the <Cmdname sid="transaction_map" /> chart. To select multiple items, hold down the `Ctrl` (Windows/Linux) or `CMD` (Mac) key and then click desired items.

:::tip

You can move an item in the list of the **_Top 30_** section by pressing the up or down arrow button on the keyboard.

:::
