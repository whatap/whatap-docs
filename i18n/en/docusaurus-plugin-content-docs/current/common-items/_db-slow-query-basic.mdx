## Basic screen guide

<ProdImg img='db-slow-query.png' desc='slow query'/>

- ![Leader line 1](/img/number-01.png) <Cmdname sid="time" className="uitext" /> / <Cmdname sid="instance" className="uitext" />

  You can change the duration of an active session or select individual instances.

  :::note

  For more information on how to use the <Cmdname sid="time" className="uitext" /> option.

  :::

- ![Leader line 2](/img/number-02.png) **_Slow Query_**

  It displays the times of slow queries with a chart. If you drag a specific area with you mouse, you can see information details in [Leader line 3](/img/number-03.png) **_Slow Query List_**.

- ![Leader line 3](/img/number-03.png) **_Slow Query List_**

  The slow query details on the area dragged on the **_Slow Query_** chart are displayed in the list. If **_SQL_** is selected, you can see the (**_query_**)'s full text and **_plan_** information.
