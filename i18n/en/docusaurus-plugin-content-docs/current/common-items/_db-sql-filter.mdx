<Title level={props.level} hashid="filtering">Filtering the searched results</Title>

<Title level={props.level + 1}>Adding the filter conditions</Title>

<InDoc pages="stat">

You can filter the results by SQL text, agent name, or DB user name.

</InDoc>

<InDoc pages="analysis-sql">

You can filter the desired results based on the values ​​of the following items in the searched results.

:::info

- **_sql_id_** 

- **_sql_hash_value_**

- **_after_plan_hash_value_**

- **_before_plan_hash_value_**

:::

</InDoc>

1. In the <Cmdname sid="filter" className="uitext" /> option, select <LinkImage img="ico-add.svg" className="ico" />.

   <InDoc pages="analysis-sql,job-info,backup-recovery-history,procedure-stat"><ImgLang img='db-analysis-sql-plan-change-filter-01.png' desc='Adding condition' /></InDoc>

2. In <Cmdname sid="TTL06740" className="uitext" />, select a desired filtering criteria.

   - If the value of the selected item is text, you can select any of <Cmdname sid="includes" className="uitext" /> (blue) and <Cmdname sid="excludes" className="uitext" /> (red).

     <InDoc pages="stat,top-sql"><ImgLang img="db-sql-filter-condition-color.png" desc="example" multi /></InDoc>

     <InDoc pages="analysis-sql"><ImgLang img="db-sql-analysis-filter-condition-color.png" desc="example" multi /></InDoc>

   - If the value of the selected item is number, you can select any condition of `==` (equal to), `>=` (greater than or equal to), and `<=` (less than or equal to).

3. In <Cmdname sid="condition" className="uitext" />, select a condition.

4. Enter a string or number to match the condition.

5. Select <Cmdname sid="apply" className="uitext" />.

:::note

- To add filtering conditions, select <Cmdname sid="bsm_button_add" className="uitext" /> and then repeat Step 1 to 5. Added conditions are applied based on AND (`&&`).

- To delete some items while adding conditions, select ![Delete icon](/img/ico-trash.svg) on the right of the filter conditions. To delete all conditions, select ![Delete icon](/img/ico-trash.svg) <Cmdname sid="BTN06616" className="uitext" />.

- To quickly delete the conditions applied to the <Cmdname sid="filter" className="uitext" /> option, select <LinkImage img="ico-close.svg" className="ico" />.

<InDoc pages="stat,top-sql">

- The default setting is to exclude if `sqlText` contains 'WhaTap'.

  <ImgLang img='db-sql-filter-default.png' desc='WhaTap' />

</InDoc>

:::

<Title level={props.level + 1}>Modifying the filter conditions</Title>

<InDoc pages="analysis-sql,job-info,backup-recovery-history,procedure-stat"><ImgLang img='db-sql-analysis-filter-modify.png' desc='Edit filter' /></InDoc>

Click any applied to the <Cmdname sid="filter" className="uitext" /> option. If the <Cmdname sid="TTL06737" className="uitext" /> window appears, modify any desired items and then select <Cmdname sid="apply" className="uitext" />.
