Home > Select Project > **Dashboard** > **Instance List**

<ProdImg img="db-dashboard-instance-list.png" desc="Instance List" />

You can see the list of database servers in which the agent has been installed. You can check key performance metrics and status. The performance metrics are updated every 5 seconds, and the order of the columns in the list can be changed or it can be changed to the desired metrics through settings.

:::note

If you click the abnormal status icon in the **Status** column, the **Event** window appears. You can see the history of events in the instance.

:::
