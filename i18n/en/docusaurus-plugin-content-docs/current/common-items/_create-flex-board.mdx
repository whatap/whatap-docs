## Creating a dashboard

You can create your own dashboard by creating a Flex board and placing widgets on it.

1. From the <Cmdname sid="side_flexboard" className="uitext" /> menu, select <Cmdname sid="create_dashboard" className="uitext" /> on the upper right.

2. If the <Cmdname sid="create_dashboard" className="uitext" /> window appears, enter the dashboard name in the <Cmdname sid="name" className="uitext" /> field.

3. Select the widget placement method and then click <Cmdname sid="create_dashboard" className="uitext" />.

   - <Cmdname sid="fixed_flexboard" className="uitext" />: Widgets can be placed freely on a pixel basis.

   - <Cmdname sid="flexible_flexboard" className="uitext" />: It provides the grid responsive layout based on the browser size. If <Cmdname sid="flexible_flexboard" className="uitext" /> is selected, select a grid and then select <Cmdname sid="create_dashboard" className="uitext" />.

4. In the <Cmdname sid="widget_template" className="uitext" /> selection area, select widgets to place on the dashboard. 

   * To add any metrics widget, on the right of <Cmdname sid="widget_template" className="uitext" />, select **All metrics**.

   :::note

   - <Cmdname sid="widget_template" className="uitext" />: You can select from a list of predefined widgets to arrange the metrics that are important to you for general monitoring.
   - <Cmdname sid="all_metrics" className="uitext" />: It is used to create widgets based on all metrics data being collected from your project. For more information about the metrics widget, see [the following](flexboard-metric-widget).

   :::

5. Reposition or resize the placed widgets.

   - Hover your mouse cursor over the placed widget. When the cursor shape changes to a cross, click and drag the widget. You reposition it.

   - Move your mouse cursor to the lower right of the place widget. When the cursor shape changes to an arrow, click and drag the widget. You can resize it.

6. After placing the added widgets, select <Cmdname sid="view_mode" className="uitext" /> and then check the layout. To modify the layout again, select <Cmdname sid="edit_mode" className="uitext" /> and then reposition the widgets.

7. After all things are made, select the icon ![목록으로 가기 아이콘](/img/ico-exit.svg) (<Cmdname sid="go_to_list" className="uitext" />) on the upper right. In <Cmdname sid="dashboard_list" className="uitext" /> of the <Cmdname sid="side_flexboard" className="uitext" />, you can check the created board. By selecting a created board, you can check the new Flex board.

:::note

We want to add dashboard templates or widgets based on your feedback. If you need any dashboard templates or widgets, contact [support@whatap.io](mailto:support@whatap.io).

:::
