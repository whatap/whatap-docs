### Searching the agent {#searchagents}

If the agents connected to the project are numerous and difficult to find, use the agent search function. If you click ![Search icon](/img/ico-search.svg) <Cmdname sid="search" className="uitext" /> on the upper right of the screen, a box appears where you can enter a string.

<InDoc pages="dashboard"><ImgLang img="apm-dashboard-search-agent.png" desc="Agent search" /></InDoc>

The agents that match the entered string appear in the list. To filter the data in the dashboard based on the searched agents, click <Cmdname sid="select" className="uitext" /> on the right of the text entry box.

To reset the selected agent entries, click <Cmdname sid="close" className="uitext" /> on the right of the input field.
