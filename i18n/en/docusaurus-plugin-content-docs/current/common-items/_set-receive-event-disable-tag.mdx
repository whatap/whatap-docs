### Disabling the event reception tag

1. In the user list of the <Cmdname sid="alert_receive_settings_by_member" className="uitext" /> section, select <LinkImage img="ico-add.svg" desc="add icon" />.

2. If the <Cmdname sid="event_tag" className="uitext" /> pop-up window appears, select <LinkImage img="ico-close.svg" desc="close icon" /> next to the applied tag.

3. Close the <Cmdname sid="event_tag" className="uitext" /> pop-up window.
