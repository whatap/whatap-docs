After installation, you can select a namespace and a data collection cycle in **_Management_** > **_Agent Installation_**. 

<ProdImg img='cloud-namespace-datainterval.png' desc='Selecting the namespace and collection cycle' />

-   If you check in ![number1](/img/number-01.png) **_Active Service_**, you can see active namespaces in ![number2](/img/number-02.png). 
-   If you enter a keyword in the ![number1](/img/number-01.png) field, you can search any namespace in ![number2](/img/number-02.png). 
-   ![number2](/img/number-02.png) You can set the data collection cycle for each namespace and whether or not to enable data collection.
    -   Select ![down-arrow](/img/down-arrow.svg) in the **_Whatap Data Interval_** column to set the data collection cycle.
    -   If you select the toggle button in the **_Enable_** column, you can enable data collection ![Icon](/img/ic-toggle-on.svg) or disable it ![Icon](/img/ic-toggle-off.svg).
-   Select ![number3](/img/number-03.png) **_Save_** to save the settings.
