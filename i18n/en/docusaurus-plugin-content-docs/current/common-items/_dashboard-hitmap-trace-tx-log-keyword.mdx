#### Setting the keyword highlight{#anlaysis-trace-set-keyword}

1. To add keywords to highlight, select ![Highlight icon](/img/ic-highlight.svg).

   <ImgLang img="analysis-tx-transaction-log-highlight-keyword.png" desc="Keyword highlight" />

2. Enter a desired keyword.

3. Select the color icon and then select a desired color.

4. Press Enter.

The added keywords are added in the list.

#### Deleting the keyword highlight{#anlaysis-trace-delete-keyword}

To delete the items unused from the highlight list, select ![Delete icon](/img/ic-minus.svg).
