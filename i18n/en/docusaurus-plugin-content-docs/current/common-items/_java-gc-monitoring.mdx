- **gclog_enabled** <span class='type'>Boolean</span>

  Default `false`

  It enables the GC log monitoring.

- **gclog_file** <span class='type'>String</span>

  Set the GC log file path.

- **gclog_check_interval** <span class='type'>Int</span>

  Default `2000`

  It is the interval for monitoring the GC logs. At every interval, it checks whether any record has been added to the log file.

- **gclog_word** <span class='type'>String</span>

  Default `Full GC`

  Register the keyword to monitor. If `Full GC` occurs, data is transferred to the server and then is saved.
