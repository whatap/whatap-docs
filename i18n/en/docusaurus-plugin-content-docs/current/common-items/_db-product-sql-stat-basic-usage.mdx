1. Select the time and date to view data in <Cmdname sid="time" className="uitext" />. You can also select a lookup time by clicking the green button.

2. Select a target to view data in <Cmdname sid="instance" className="uitext" />.

3. Select ![Search icon](/img/ico-btn-search.png).

The top 50 entries appear in the table.

:::note

- Select the clock display area to query a random date and time. If you click the text range for date and time, the option appears to select the date and time.

- If you select each column in the table header, you can sort the list by the selected item.

- You can sort the searched results according to the items selected in <Cmdname sid="TTL07772" className="uitext" />. Select a desired item and then select <LinkImage img="ico-btn-search.png" className="ico" />.

- In <Cmdname sid="TTL07773" className="uitext" />, you can set the number of items to display in the table. After all settings are finished, select <LinkImage img="ico-btn-search.png" className="ico" />.

- To download the viewed data as a CSV file, select ![Download icon](/img/ico-download.svg).

- If you select **_query_** from the searched result list, the <Cmdname sid="TTL06224" className="uitext" /> window appears. For more information, see [the following](#query-details).

:::
