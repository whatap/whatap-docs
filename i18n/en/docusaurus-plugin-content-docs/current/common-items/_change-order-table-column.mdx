<Title hashid='change-order-column' level={props.level}>Reordering the table columns</Title>

<InDoc product="browser">

You can change the order of column headers in the browser error list table.

</InDoc>

1. Select ![Column icon](/img/ico-column.svg) on the upper right of the table.

2. When the <Cmdname sid="column_settings" className="uitext" /> window appears, change the order by dragging the desired items from the list on the right.

3. After all settings are finished, select <Cmdname sid="BTN06053" className="uitext" />. The set content is applied to the table.
