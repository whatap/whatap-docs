For backward compatibility, you can use the service screens before Database Monitoring V2. 

1. Go to <Cmdname sid="side_management" className="uitext" /> &gt; <Cmdname sid="side_projectManagement" className="uitext" />.

2. In the <Cmdname sid="TTL06144" className="uitext" /> section, select ***V1*** as <Cmdname sid="TTL06145" className="uitext" />.

3. Select <Cmdname sid="save" className="uitext" />.

:::note

- If there is data that cannot be viewed on the screen when inquiring past data, please change the <Cmdname sid="TTL06145" className="uitext" /> to ***V1*** and inquire with the previous version screen.

- For compatibility, this function is only available until December 31, 2024.

:::
