**_Metrics Cube_** provides the trend analysis of metrics field data for each category. You can see the metrics field data through the chart of square widgets as follows.

![Metrics Cube sc](/img/metrics-cube.png)

## Selecting a category

The **_Select Category_** function is provided in the ![number1](/img/number-01.png) area on the left of **_Metrics Cube_**. Search the category list and select a category in ![number1](/img/number-01.png). You can configure your dashboard in the ![number2](/img/number-02.png) chart widget area with the fields contained in the selected category.

## Filtering

It provides tag-based filtering in the ![number1](/img/number-01.png) area on the left of the **_Metrics Cube_** screen. At the top of ![number1](/img/number-01.png), select **_FILTER_** to display the filter list as follows. 

![Metrics Cube Filter sc](/img/metrics-cube-filter.png)

In the ![number3](/img/number-03.png) filter list, you can set the filtering criteria such as **_Equal_**, **_Includes_**, **_Excludes_**, **_Select All_**, and **_Deselect All_**. You can add a filter by selecting **_+ Add Filter_**. Select **_Apply_** to apply the setting and then configure the dashboard in the ![number2](/img/number-02.png) chart widget area. 

At the top of the filter list, select the ![Left arrow icon](/img/left-arrow.svg) icon to return to the category list. If you select the ![Left icon](/img/ic-metrics-cube-back.svg) icon, the filter list is hidden and the dashboard area is expanded to focus on the chart.

You can select an applied filter in the filter list on the left and at the top of the ![number2](/img/number-02.png) area to modify on the ![number4](/img/number-04.png) screen.

## Grouping

It provides the tag-based grouping. You can modify the chart data grouping through the grouping options at the top of the ![number2](/img/number-02.png) area on the right of the **_Metrics Cube_** screen. Two levels of grouping is provided. By selecting **_Step 2_**, you can set the tags for grouping. In the following example, two-level grouping is applied using the reference properties of `attachedInstance` and `Name` as tags.

![Metrics Cube Grouping sc](/img/metrics-cube-gp.png)
