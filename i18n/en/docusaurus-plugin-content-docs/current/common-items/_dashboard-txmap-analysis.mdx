## Analyzing a trace

If you drag the chart area of <Cmdname sid="transaction_map" />, the <Cmdname sid="TTL06209" className="uitext" /> window appears where you can check the transaction details. You can see the transaction list and the steps under each transaction at once.

After dragging the chart area, you can see up to 1,000 items in the <Cmdname sid="TTL06209" className="uitext" /> window.
