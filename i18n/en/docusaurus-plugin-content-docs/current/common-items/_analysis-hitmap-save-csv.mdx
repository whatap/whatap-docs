### Downloading the viewed list

You can download the results retrieved from the <Cmdname sid="trace_list" className="uitext" /> section as a CSV file. The formation of the downloaded CSV file is *profile*`{project_code}`*`YYYYMMDD`.csv*.

1. After viewing the transactions, select ![Download icon](/img/ic-csv-download-ms.svg).

2. Enter a desired value in the <Cmdname sid="maximum_csv_lines" className="uitext" /> option.

3. Select <Cmdname sid="download" className="uitext" />.
