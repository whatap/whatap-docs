Anomaly detection is a set of behaviors that look for unexpected patterns in data such as anomaly, outlier, and abnormal. Based on the metrics data, you can set alerts to be sent when the rising and falling patterns are not as usual. 

<ImgLang img='set-event-abnormal.png' desc='Anomaly Detection' />

## Creating the anomaly detection event

By selecting <Cmdname sid="add_alert_setting" className="uitext" />, you can perform the settings for the <Cmdname sid="anomaly_detection" className="uitext" /> event. After all settings are finished, select <Cmdname sid="save" className="uitext" />. By selecting the toggle button on the right of <Cmdname sid="anomaly_detection" className="uitext" />, you can select the activation.

<ImgLang img='set-event-anormal-create.png' desc='Anomaly Detection' />

### Event basic information {#event_basic_rule_help}

Enter the basic data for the anomaly detection event.

- <Cmdname sid="level" className="uitext" />: Select a level among <Cmdname sid="info" className="info" />, <Cmdname sid="warning" className="slow" />, and <Cmdname sid="critical" className="vslow" />.

- <Cmdname sid="message" className="uitext" />: Enter the alert message displayed upon occurrence. You can apply a variable to the message by entering `${Tag}` or `${Field}`. The key to enter for the variable must be a value contained in the selected metrics data, <Cmdname sid="category" className="uitext" />.

### Metrics {#metrics}

Select an event occurrence target.

- <Cmdname sid="category" className="uitext" />: The unit to identify the metrics data. It is a mandatory value for setting the metrics events.

- <Cmdname sid="field" className="uitext" />: Select a field to use as an event generation condition. You can select multiple items.

- <Cmdname sid="filter" className="uitext" />: Select an event condition target. You can select multiple items.

### Anomaly detection {#anomaly_detection}

You can select the severity of <Cmdname sid="rising_pattern" className="uitext" /> or <Cmdname sid="falling_pattern" className="uitext" />. This function can be enabled or disabled by selecting the toggle buttons for each item.

## Modifying the anomaly detection event

1. In <Cmdname sid="side_event" className="uitext" /> &gt; <Cmdname sid="side_eventSetting" className="uitext" />, select the <Cmdname sid="anomaly_detection" className="uitext" /> tab.

2. In the <Cmdname sid="anomaly_detection" className="uitext" /> list, select ![Edit icon](/img/ico-edit.svg) on the right of the item to modify.

3. If the <Cmdname sid="anomaly_detection" className="uitext" /> window appears, modify the desired items and then select <Cmdname sid="save" className="uitext" />.

## Deleting the anomaly detection event

In <Cmdname sid="side_event" className="uitext" /> &gt; <Cmdname sid="side_eventSetting" className="uitext" />, select the <Cmdname sid="anomaly_detection" className="uitext" /> tab. Select ![Delete icon](/img/ico-trash.svg) on the right of the item to delete. When the confirmation message appears, select <Cmdname sid="delete" className="uitext" />.
