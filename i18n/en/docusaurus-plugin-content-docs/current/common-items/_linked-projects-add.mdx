## Adding any linked project {#addlinkproject}

1. Go to <Cmdname sid="side_management" className="uitext" /> &gt; <Cmdname sid="side_correlatedProjectManagement" className="uitext" />.

2. In the <Cmdname sid="TTL07015" className="uitext" /> section, select the <Cmdname sid="project_selection" className="uitext" /> input field.

   <ImgLang img="linked-project-management-add-project.png" desc="Correlated Project Management" />

3. If you see a list of projects, select a project to link to. You can also enter a string to search for matching projects.

4. Select one or more projects and then select <Cmdname sid="add" className="uitext" /> that is active.

The selected project is added in <Cmdname sid="TTL07016" className="uitext" />.

:::note 

You need the **Edit** role on the project to add a linked project. If you have the **Edit** role, the <Cmdname sid="add" className="uitext" /> button is enabled in the <Cmdname sid="side_correlatedProjectManagement" className="uitext" /> menu. 

:::
