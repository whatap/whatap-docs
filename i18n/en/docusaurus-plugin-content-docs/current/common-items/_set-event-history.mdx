## Viewing the event records {#eventview}

Set the following options in order and then select ![Search icon](/img/ico-btn-search.png). You can search event records based on the set options.

- <Cmdname sid="time_selector" className="uitext" />: You can set the time range for events to search for. It searches based on <Cmdname sid="event_time" className="uitext" />.

  :::note

  For more information on how to use the <Cmdname sid="time_selector" className="uitext" /> option.

  :::

- <Cmdname sid="filter" className="uitext" />: It filters the list of events that contain the entered string based on the content of <Cmdname sid="event_title" className="uitext" /> or <Cmdname sid="message" className="uitext" />.

<Xclude product="browser,url">

- <Cmdname sid="agent" className="uitext" />: Filtering can be made by selecting only the events that occurred on a specific agent.

</Xclude>

## Registering the processed records {#alertacknowledge}

When you register the processing history, it makes it easy to update the event processing status and manage its history.

:::caution

When you register the processing history, the <Cmdname sid="TTL06149" className="uitext" /> of the event is suspended. For more information about <Cmdname sid="TTL06149" className="uitext" />, see [the following](set-receive-event#escalation).

:::

1. In the event to handle the processing history, select <Cmdname sid="sms_alert_acknowledge" className="uitext" />.

2. When the <Cmdname sid="sms_alert_acknowledge_add" className="uitext" /> window appears, enter the processed results.

    * Processed details can be entered up to 255 Bytes.

3. Select <Cmdname sid="registration" className="uitext" />.

    * The registered processing history appears as a list in <Cmdname sid="TTL08669" className="uitext" />. You can check the creation time, author, and details.

        <ImgLang img="event-history-add-acknowledge-history.png" desc="Acknowledge detail" />

    * You can check the number of processed records added in the event list.

        <ImgLang img="event-history-add-acknowledge-added.png" desc="Acknowledge" />

## Using additional options {#additionaloption}

- <Cmdname sid="BTN06441" className="uitext" />: You can add, delete, and change the order of columns in the event list.

- **CSV**: You can save the searched event history records as a CSV file. Click **CSV**, enter <Cmdname sid="maximum_csv_lines" className="uitext" />, and then select <Cmdname sid="download" className="uitext" />.

- <Cmdname sid="exclude_resolved_alerts" className="uitext" />: You can view only the **ongoing events** in the event history list.

- <Cmdname sid="event_setting" className="uitext" />: Go to <Cmdname sid="event_statistic" className="uitext" /> &gt; <Cmdname sid="event_setting" className="uitext" />.
