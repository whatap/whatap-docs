## Screen modes

<Cmdname sid="side_flexboard" className="uitext" /> provides the following screen modes. The functions that can be set differ depending on the screen mode.

- <Cmdname sid="edit_mode" className="uitext" />: From the widget template list, you can place, modify, delete, or reposition widgets on the Flex board layout.

- <Cmdname sid="admin" className="uitext" />: You can manage the attributes of widgets in json format. Select ![Code icon](/img/ico-custom-code.svg) on the upper right of the widget placed on the layout.

- <Cmdname sid="view_mode" className="uitext" />: After finishing the settings such as widget placement, resizing, and option selection, you can monitor data for each widget including the dashboard in real time.

You can use the following common features in <Cmdname sid="side_flexboard" className="uitext" />.

- Data filtering
- Real-time data lookup range setting
- Print mode
- Flex Board Management
- Data merging option

## Data filtering

In the Flex board, you can handle hundreds of types of data. Organize your Flex board by using the data filtering feature to narrow down what you want to monitor.

<ImgLang img='flexboard-filter.png' desc='Flex board - filter' />

1. Click ![Filter icon](/img/ico-filter.svg) <Cmdname sid="filter" className="uitext" /> on the screen. 

2. On the left of the screen, the <Cmdname sid="filter" className="uitext" /> window appears. Select a filtering target.

   - If the <Cmdname sid="agent" className="uitext" /> option has been selected, you can select filtering targets based on the agent type or application list. To select all as the filtering target, select <Cmdname sid="select_all" className="uitext" />. If necessary, you can select the targets by using the <Cmdname sid="search" className="uitext" /> function.

   - If the <Cmdname sid="metrics" className="uitext" /> option has been selected, you can select filtering targets based on the project and agent type in <Cmdname sid="category" className="uitext" />. In the <Cmdname sid="category" className="uitext" /> option, you can add more by selecting <UI>+ And</UI>. If <Cmdname sid="contain" className="uitext" /> or <Cmdname sid="exclude" className="uitext" /> has been selected, filtering targets are automatically selected with any string.

3. If filtering targets have been selected, select <Cmdname sid="apply" className="uitext" />. 

4. To save the filter settings for later use, select <Cmdname sid="save_filter" className="uitext" />. Enter the filter name and then select <Cmdname sid="save" className="uitext" />. <Cmdname sid="saved_filter" className="uitext" /> appears at the bottom of the <Cmdname sid="filter" className="uitext" /> window.

#### Filtering conditions

Filtering conditions are divided into the agent type and metrics type.

| Type                | Agent                                                                                                          | Metrics                                                                                                                                        |
| ------------------- | -------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------- |
| Option              | WhaTap project monitoring targets <br/>(applications, servers, DB,<br/> containers, and Kubernetes POD & node) | All metrics tags including the agent                                                                                                           |
| Application range   | All widgets                                                                                                    | All widgets using the metrics data                                                                                                             |
| Filtering condition | Checkbox                                                                                                       | <Cmdname sid="equal" className="uitext" />, <Cmdname sid="contain" className="uitext" />, <Cmdname sid="exclude" className="uitext" /> methods |

:::note

- The agent type provides only basic monitoring targets as options.

- The metrics type provides optional tags for all metrics data being collected in the project. Metrics types are useful when filtering is applied after collecting data in a non-agent manner, such as cloud monitoring.

- For more information about the metrics, see [the following](metrics-intro).

:::

## Setting the lookup time

You can set the time in which data is retrieved from the widget using the time selector at the upper right of the screen in the <Cmdname sid="side_flexboard" className="uitext" /> menu. You can stop the real-time mode and set the query time by selecting the ![Pause icon](/img/ic-pause-green.svg) button. To return to real-time mode, select the ![Playback icon](/img/ico-play-time.svg) button.

:::note

For more information on how to use the time selector, see [the following](usage-ui#timeselector).

:::

## Printing

The screen of the Flex board can be saved as a PDF file to be printed. Select ![Print icon](/img/ico-print.svg) on the upper right of the Flex Board screen. Switch the screen to Print mode. After setting the following options, select <UI>PDF</UI><Cmdname sid="download" className="uitext" />.

- <Cmdname sid="filter_visible_msg" className="uitext" />: You can turn on or off the filter option of the screen.
- <Cmdname sid="change_landscape_mode_msg" className="uitext" replace="horizontal" /> / <Cmdname sid="change_landscape_mode_msg" className="uitext" replace="vertical" />: The screen can be converted to landscape or portrait.

To end the Print mode, select ![End icon](/img/ico-close.svg) <Cmdname sid="close_print_mode" className="uitext" /> on the upper right.

## Flex Board Management

It provides management functions such as changing the Flex board layout, repositioning/resizing widgets, and changing the name. <Cmdname sid="flexboard_management" className="uitext" /> can also be accessed by selecting ![Edit icon](/img/ico-edit.svg) in <Cmdname sid="dashboard_list" className="uitext" /> of <Cmdname sid="side_dashboard" className="uitext" /> &gt; <Cmdname sid="side_flexboard" className="uitext" />.

1. Click ![Configuration icon](/img/ico-setting.svg) on the upper right of the Flex Board screen. The <Cmdname sid="flexboard_management" className="uitext" /> window appears. 

    - <Cmdname sid="rename" className="uitext" />: You can change the name of the Flex board. Enter the name to change.

    - <Cmdname sid="layout" className="uitext" />: You can reposition or resize the widgets. To delete a placed widget, click ![Delete icon](/img/ico-trash.svg).

    - <Cmdname sid="dont_allow_users_to_modify" className="uitext" />: You can set for other users not to modify the Flex board. Only members with the edit permission can modify, delete, or edit widgets. If you click the button, it is changed to <Cmdname sid="dont_allow_users_to_modify" className="uitext" /><UI>: On</UI>.

    - <Cmdname sid="widget" className="uitext" /><UI>.json</UI>: You can edit the widgets on the Flex board into json format for management.

    - <Cmdname sid="option" className="uitext" /><UI>.json</UI>: You can edit the Flex board settings into json format for management. You can edit the settings such as project information, Flex board filtering, and time selection.

    :::note

  <Cmdname sid="widget" className="uitext" /><UI>.json</UI> and <Cmdname sid="option" className="uitext" /><UI>.json</UI> are available only to the users with the dashboard administrator role.

  :::

2. After all settings are made, at the lower right of the screen, click <Cmdname sid="save" className="uitext" />.

## Registering the dashboard as a favorite

The created Flex board can be registered as a favorite under <Cmdname sid="side_dashboard" className="uitext" />.  
In <Cmdname sid="side_dashboard" className="uitext" /> > <Cmdname sid="side_flexboard" className="uitext" />, select ![Favorite icon](/img/ico-favorite.svg) from the <Cmdname sid="dashboard_list" className="uitext" />. The item set as a favorite is changed to ![Favorite setting icon](/img/ico-favorite-check.svg).
