- **stat_domain_enabled** <span class='type'>Boolean</span>

  Default `false`

  It enables the function to collect transaction statistics for each domain that connects to clients.

- **stat_domain_max_count** <span class='type'>Int</span>

  Default `7000`

  Maximum number of records of transaction statistics for each domain that collects for 5 minutes.

- **stat_referer_enabled** <span class='type'>Boolean</span>

  Default `false`

  It collects transaction statistics by referer.

- **stat_referer_max_count** <span class='type'>Int</span>

  Default `7000`

  Maximum number of records of transaction statistics for each referer that collects for 5 minutes.
