## Log tagging options {#log-tag-options}

When you select a log tag, the following drop-down menu appears. You can find the <Cmdname sid="search" className="uitext" />, <Cmdname sid="exclusion_search" className="uitext" />, <Cmdname sid="TTL06582" className="uitext" /> and <Cmdname sid="TTL06527" className="uitext" /> options. 

<ImgLang img='log-search-tag.png' desc='Log tag dropdown menu'/>

- Copy

- **Search**

  If you select the <Cmdname sid="search" className="uitext" /> option, the tags that correspond to <Cmdname sid="filter" className="uitext" /> are entered with the inclusion (`==`) condition.

- **Exclusion search**

  If you select the <Cmdname sid="exclusion_search" className="uitext" /> option, the tags that correspond to <Cmdname sid="filter" className="uitext" /> are entered with the exclusion (`!=`) condition.

- **Adjacent Log**

  If you select the <Cmdname sid="TTL06582" className="uitext" /> option, the <Cmdname sid="TTL06582" className="uitext" /> detail window appears. Search for the logs of the target server and the logs in time zones adjacent to the selected log. You can click <Cmdname sid="time_selector" className="uitext" /> to see the logs in adjacent time zones. Reference logs appear with a blue background.

  <ImgLang img='log-search-adjacent-log.png' desc='Log tag - adjacent logs'/>

- Transaction

  If the log contains the `@txid` and `@mtid` tags, you can see the information in the transaction Information details window by clicking the transaction information option.

  <ImgLang img='log-search-transaction-info.png' desc='로그 트랜잭션 정보'/>
