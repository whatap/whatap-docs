You can customize your dashboard by handling the widgets placed on the Flex board.

## Adding a widget

You can add any widget to the created Flex board.

1. In <Cmdname sid="dashboard_list" className="uitext" /> of <Cmdname sid="side_flexboard" className="uitext" />, select a dashboard to add the widget.

2. On the upper left of the screen, select <Cmdname sid="edit_mode" className="uitext" />.

3. Select a desired widget from the <Cmdname sid="widget_template" className="uitext" /> list on the left. To add a metrics widget, select <Cmdname sid="all_metrics" className="uitext" /> and then click a desired widget.

  - If you have entered <Cmdname sid="integratedFlexboard" className="uitext" /> on the Home screen and added a widget, click <Cmdname sid="project_selection" className="uitext" /> and then select target projects to collect.
  - If you have more widgets to add, repeat the same procedure.

4. After all steps are finished, select <Cmdname sid="view_mode" className="uitext" /> to check the layout and then select ![Go to list icon](/img/ico-exit.svg) (<Cmdname sid="go_to_list" className="uitext" />).

## Changing the widget attributes

In <Cmdname sid="dashboard_list" className="uitext" /> of <Cmdname sid="side_flexboard" className="uitext" />, select a board to change the widget properties. It goes to <Cmdname sid="edit_mode" className="uitext" />.

### Changing the widget name

1. On the right of the widget name to change, select ![Edit icon](/img/ico-edit.svg). 

2. Set the widget values for <Cmdname sid="title" className="uitext" />, <Cmdname sid="font_size" className="uitext" />, and <Cmdname sid="color" className="uitext" />, and then select <Cmdname sid="save" className="uitext" />.

    - If the <Cmdname sid="apply_to_the_entire_widget" className="uitext" /> toggle button is enabled, the font size and color are applied to other widgets.

    - To lock the widget title, select [Configuration icon](/img/ico-invisible.svg) and then click <Cmdname sid="save" className="uitext" />.

### Time Setting

1. On the upper right of the widget to set the time, select ![Time settin icon](/img/ico-time-set.svg). 

2. Set the time in the <Cmdname sid="select_time_title" className="uitext" /> window.

    - To apply the set time to the dashboard, check <Cmdname sid="use_dashboard_settings" className="uitext" />. The time setting on the dashboard is located on the upper right.

    - To apply a desired time setting, uncheck <Cmdname sid="use_dashboard_settings" className="uitext" />. Use ![Left arrow icon](/img/left-arrow.svg) or ![Right arrow icon](/img/right-arrow.svg) to set the desired time.

3. After setting the time, select ![End icon](/img/ico-close.svg).

:::note

Depending on the data type of the widget, the time setting may not be available.

:::

### Setting the output data options

By setting the output data options of the chart widget, you can select the desired data. 

1. At the top right of the widget, click the <img src="/img/ico-more.svg" alt="More icon" /> icon to configure the output data options.

    - You can filter data by selecting types for project, agent, application, etc.

    - You can change the type of the graph displayed on the chart by selecting individual or merged data.

    - Select ![More icon](/img/ico-more.svg) &gt; ![Configuration icon](/img/ico-setting.svg). You can change the option details through the <Cmdname sid="chart_settings" className="uitext" /> window. The options that can be set are as follows.

      - <Cmdname sid="chart_type" className="uitext" />: This option allows you to display the current aggregated figures on the chart.

      - <Cmdname sid="y_axis" className="uitext" /> <Cmdname sid="chart_max_value" className="uitext" />: The chart maximum value on the Y axis can be selected with one of <Cmdname sid="auto" className="uitext" /> and <Cmdname sid="fix" className="uitext" />.

      - <Cmdname sid="set_horizontal_line" className="uitext" />: Select <Cmdname sid="add" className="uitext" /> to display the threshold or range as a horizontal line. You can apply multiple items added. After adding the horizontal line, select <Cmdname sid="apply" className="uitext" /> to apply it to the chart.

      - <Cmdname sid="show_chart_max_value" className="uitext" />: The maximum value of the graph is displayed on the chart.

      - <Cmdname sid="sub_chart" className="uitext" />: You can add a sub-chart to the main charts of the widget. Select a type of sub-chart and then select its layout, size, and whether or not to merge data. To apply to other widgets, select <Cmdname sid="apply_to_the_entire_widget" className="uitext" />. Sub-charts are useful to view data in various ways.

2. After setting all options, on the left of the <Cmdname sid="chart_settings" className="uitext" /> window, click ![Close icon](/img/ico-close.svg) or click the outside of the <Cmdname sid="chart_settings" className="uitext" /> window.

:::note

Depending on the widget's data type, available options may differ.

:::

### Changing the metrics widget data{#changedata-metric-widget}

You can change the data query conditions of the metrics widget. 

1. In <Cmdname sid="side_flexboard" className="uitext" /> > <Cmdname sid="dashboard_list" className="uitext" />, select the dashboard where you want to change the widget data.

2. In <Cmdname sid="edit_mode" className="uitext" />, click the <img src="/img/ico-more.svg" alt="More icon" /> icon at the top right of the widget you want to modify.

3. Click the <img src="/img/ico-edit-metric.svg" alt="Edit icon" /> icon to open the data settings edit window.

4. Change the settings referencing the following, and then select ![Apply icon](/img/left-arrow.svg) <Cmdname sid="apply_to_the_widget" className="uitext" />.

   - Metrics options that are created by default when creating a widget appear on the tab.

   - Metrics options are <Cmdname sid="category" className="uitext" />, <Cmdname sid="field" className="uitext" />, and <Cmdname sid="tag" className="uitext" />.

   - The <Cmdname sid="category" className="uitext" /> currently restricts the editing function.

   - In <Cmdname sid="field" className="uitext" /> and <Cmdname sid="tag" className="uitext" />, multiple items can be selected. The chart data is grouped using the selected tag options.

   - Select ![Search icon](/img/ico-btn-search.png) to query the original data.

   - For more information about<Cmdname sid="object_merge" className="uitext" /> and <Cmdname sid="time_merge" className="uitext" />, see the [Data merge options](#datamerge-option).

### Data merge options {#datamerge-option}

The metrics widget placed on the <Cmdname sid="all_metrics" className="uitext" /> list of the Flex board provides the data merge options. Data merge provides the following methods: <Cmdname sid="object_merge" className="uitext" /> and <Cmdname sid="time_merge" className="uitext" />.

- <Cmdname sid="object_merge" className="uitext" />

  It is used to group the chart data based on the tag (e.g. agent name) selected on the left. At this time, if the tags match among the data with different fields (e.g. CPU), the corresponding data is merged.

- <Cmdname sid="time_merge" className="uitext" />

  It is mainly used if the lookup time is too long. Data with the same fields in the original data are merged at regular time intervals (5 minutes or 1 hour) depending on the time range. This method is called "Time Merge."

- **AWS ColudWatch Metrics**

  In case of AWS monitoring metrics, the statistical method is added after the field name in the form of suffix. Data merging defaults reference this suffix.

  <ImgLang img='flexboard-awsec2-metrics.png' desc='AWS ColudWatch Metrics' />

1. In <Cmdname sid="edit_mode" className="uitext" />, select <Cmdname sid="all_metrics" className="uitext" /> and then change the <Cmdname sid="widget_template" className="uitext" /> list to the <Cmdname sid="all_metrics" className="uitext" /> list.

2. Place the metrics widget on the layout.

3. On the upper right of the metrics widget, select ![더보기 아이콘](/img/ico-more.svg).

4. If a tag (e.g. agent name) option is selected, the data merge options appear. Depending on the widget size, the metrics option's location may differ.

## Copying a widget

1. Click ![More icon](/img/ico-more.svg) on the upper right of the widget. 
2. Click ![Copy icon](/img/ico-copy.svg).
3. If the <Cmdname sid="copy" className="uitext" /> window appears, click ![추가 아이콘](/img/ico-add.svg) <Cmdname sid="add_widget" className="uitext" />.
4. Change the widget name in the list.
5. Click <Cmdname sid="apply" className="uitext" />.
6. In <Cmdname sid="edit_mode" className="uitext" />, you can see the copied widget.

## Delete Widget

1. Click ![More icon](/img/ico-more.svg) on the upper right of the widget. 
2. Click ![Delete icon](/img/ico-remove.svg).
3. If the confirmation message appears, click <Cmdname sid="delete" className="uitext" />. To cancel the deletion, select <Cmdname sid="cancel" className="uitext" />.
