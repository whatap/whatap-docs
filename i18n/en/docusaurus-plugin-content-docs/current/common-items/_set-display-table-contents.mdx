<Title hashid='setting-table-contents' level={props.level}>Setting to display the table content</Title>

<InDoc product="browser">

You can set the content displayed in the browser error list. It can be used when too much information appears so it is difficult to see, or it is required to add necessary information.

</InDoc>

1. Select ![Configuration icon](/img/ico-setting.svg) on the upper right of the table.

2. When the <Cmdname sid="ITM08349" className="uitext" /> window appears, set the desired options.

   - Select <UI>content</UI> or <Cmdname sid="tag" className="uitext" /> to determine 
     whether or not to display the log content.

   - <Cmdname sid="tag_management" className="uitext" />: Tags are displayed in the log in the order of addition. The order of tags can be changed by dragging. Disabled tags do not appear in the logs.

3. After all settings are finished, click ![Close icon](/img/ico-close.svg). The set content is applied to the table.

:::note

For more information about <Cmdname sid="ITM08349" className="uitext" />, see [the following](../log/log-search#set-table).

:::
