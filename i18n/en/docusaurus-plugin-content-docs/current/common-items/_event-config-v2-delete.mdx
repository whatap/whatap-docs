## Deleting event

You can delete the embedded or added events. 

1. Select ![Edit icon](/img/ico-edit.svg) in the event to delete from the event list. 

2. In the event modification screen, select <Cmdname sid="delete" className="uitext" /> on the upper right of the screen.

3. If the deletion confirmation window appears, select <Cmdname sid="delete" className="uitext" /> again.

:::note

The deleted event cannot be restored.

:::
