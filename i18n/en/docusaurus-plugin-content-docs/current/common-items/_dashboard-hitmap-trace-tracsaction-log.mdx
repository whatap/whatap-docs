### Transaction log {#anlaysis-trace-transaction-log}

To see the logs of the transaction, select the <Cmdname sid="TTL06128" className="uitext" /> tab.

<ImgLang img="analysis-tx-transaction-log.png" desc="Transaction log" />

- ![Statistics icon](/img/ic-static.svg): You can see the <Cmdname sid="transaction_statistics" className="uitext" /> information. If a graph on the chart is selected, the <Cmdname sid="side_analysis" className="uitext" /> &gt; <Cmdname sid="side_txSearch" className="uitext" /> menu appears. You can see the list of transactions for the selected time zone.

- **Keyword search**: Enter the desired text to search in the keyword search field, and then press Enter or select the ![Search icon](/img/ico-search.svg) button. It highlights the keywords that match the entered text.

- ![Highlight icon](/img/ic-highlight.svg) <Cmdname sid="keyword_highlight" className="uitext" />: If you set the keyword and color, the texts that match the keyword are automatically highlighted in the log list. For more information, see [the following](#anlaysis-trace-set-keyword).

- ![Column icon](/img/ico-column.svg) <Cmdname sid="column_settings" className="uitext" /> : You can add columns or change the order of columns displayed in the log list table. For more information, see [the following](#display-or-hide-column).

- ![Configuration icon](/img/ico-setting.svg) <Cmdname sid="ITM08349" className="uitext" />: You can configure the items to be displayed in the log list. For more information, see [the following](#setting-table-contents).

:::note

- To enable the <Cmdname sid="TTL06128" className="uitext" /> tab, apply the agent setting options related to the logs. For more information about the log settings, see [the following](../log/log-function).

- The <Cmdname sid="TTL06128" className="uitext" /> tab can be accessed by only the members with the <Cmdname sid="log_read" className="b600" /> role. For more information about the member roles, see [the following](../project/project-structure#member-auth).

:::
