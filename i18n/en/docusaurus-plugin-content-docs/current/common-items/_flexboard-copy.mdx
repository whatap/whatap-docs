## Copying the Flex board {#copyflexboard}

You can copy the created Flex board and then deliver it to other members.

1. In the <Cmdname sid="integratedFlexboard" className="uitext" />'s <Cmdname sid="dashboard_list" className="uitext" />, click the <LinkImage img="ico-copy.svg" className="ico" desc="Copy icon" /> icon.

1. Select either the <Cmdname sid="copy_to_me" className="uitext" /> or <Cmdname sid="copy_to_others" className="uitext" /> option.

1. The copied Flex board will be added to the <Cmdname sid="dashboard_list" className="uitext" />.

   * If you selected the <Cmdname sid="copy_to_me" className="uitext" /> option:  

        An item with the `_copied` suffix will be added to the <Cmdname sid="dashboard_list" className="uitext" />.

   * If you selected the <Cmdname sid="copy_to_others" className="uitext" /> option:

        a. After selecting the <Cmdname sid="copy_to_others" className="uitext" /> option, click <LinkImage img="ico-add.svg" className="ico" /> <Cmdname sid="account" className="uitext" />.  
        b. Select the email of another member, then click the <Cmdname sid="copy_dashboard" className="uitext" /> button.  
        c. An item with the `_copied` suffix will be added to the memberâ€™s <Cmdname sid="dashboard_list" className="uitext" /> list.

## Sharing as a JSON file

You can save the Flex board settings as a JSON file and pass them to another member or import the settings of the member.

### Export

1. Go to <Cmdname sid="integratedFlexboard" className="uitext" /> or <Cmdname sid="flexboard" className="uitext" /> for the project.

2. In <Cmdname sid="dashboard_list" className="uitext" />, select <LinkImage img="ico-export.svg" desc="Export icon" className="ico" />.

3. Select <Cmdname sid="BTN07940" className="uitext" /> or <Cmdname sid="BTN07941" className="uitext" />.

   - <Cmdname sid="BTN07940" className="uitext" />: You can include and share project information set on the dashboard in the JSON file.

   - <Cmdname sid="BTN07941" className="uitext" />: You can share project information set on the dashboard without including it in the JSON file.

4. Once the JSON file has been downloaded, forward it to other members for sharing.

### Import

1. Go to <Cmdname sid="integratedFlexboard" className="uitext" /> or <Cmdname sid="flexboard" className="uitext" /> for the project.

2. On the upper right of the screen, select ![Import icon](/img/ico-import.svg) <Cmdname sid="import" className="uitext" />.

3. Select a JSON file to download.

:::note

This feature is available only to the members who have the **Edit** role on the project.

:::
