When creating the **default plan user file**, you can configure the agent to automatically view the plan by using the predefined DB information. After checking the box for <Cmdname sid="ITM09107" className="uitext" />, search for the plan through the specified DB connection information as follows.

</Xclude>

<Tabs>
<TabItem value='linux-dfp' label='Linux' default>

1. Add the `plan_db` option to the agent configuration (*whatap.conf*) file and then enter the DB name.

   ```ini title='whatap.conf'
   plan_db=(dbname)
   ```

2. After configuring the DB, use the `planuid.sh` script to set the user and password.
