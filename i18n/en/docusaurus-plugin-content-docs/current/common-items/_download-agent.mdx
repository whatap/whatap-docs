## Downloading the latest version of agent

1. Log in the WhaTap monitoring service (https://service.whatap.io).

2. Select a project to update the agent.

3. On the left of the screen, select <Cmdname sid="management" className="uitext" /> > <Cmdname sid="agent_update" className="uitext" />.

4. Click ![다운로드 아이콘](/img/ico-download.svg) <Cmdname sid="latest_version" className="uitext" /> or ![다운로드 아이콘](/img/ico-download.svg) <Cmdname sid="latest_version_for_all" className="uitext" />.

5. Download the agent and then restart the application server. Update is automatically applied.
