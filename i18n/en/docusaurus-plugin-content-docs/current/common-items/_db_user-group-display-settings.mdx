<Cmdname sid="users_group_display_settings" className="uitext" /> is a feature that allows you to restrict certain menus or presets to specific user groups.  
After creating a display group, the administrator can associate it with the desired menu or preset, and configure view permissions. This allows unnecessary items to be hidden and only the necessary information to be exposed according to each user's role or permissions.

:::note

Only users with member management permissions in database product projects can configure this setting.

:::

## Main Screen

<Cmdname sid="users_group_display_settings" className="uitext" /> can be used to allow only tagged users to view specific presets, menus, or features.

<ImgLang img='db-user-group-display-settings.png' desc='User group display settings main screen' />

### View Permission Settings

If no display group is configured, the default view setting is **![Active icon](/img/ic-radio.svg) Show to all**. Select **Show to display group only** to restrict view access to that group.

- **Show to display group only**: Only users in the display group can see the page

- **Show to all**: All users can see the page, regardless of display group configuration

### Add Display Role

When adding a display group, you can assign display roles for specific pages.

1. In the **User Group Display Settings** screen, click the ![Add icon](/img/ic-s-plus.svg) **Add Display Role** button at the top right.

2. In **Create Display Group**, enter the **Display Group Name** and **Description**.

    a. Click the ![Add icon](/img/ic-s-plus.svg) **Add** button to add users by email or username.

    b. Activate the **Add to Display Group** toggle for the **Menu/Preset** or **Function** to add it to the display group.

3. Click **Confirm** to create the **Display Group**.

## Add Display Group

1. In **User Group Display Settings**, click the display group ![Add icon](/img/ic-s-plus.svg) of the **Menu/Preset** you want to link.

2. From the dropdown, click ![Add icon](/img/ic-s-plus.svg) **Create Display Group**.

3. In **Create Display Group**, enter the group name and description.

    a. Click the ![Add icon](/img/ic-s-plus.svg) **Add** button to add users by email or username.

    b. Activate the **Add to Display Group** toggle for the **Menu/Preset** or **Function** to add it to the display group.

4. Click **Confirm** to create the display group.

5. After selecting the created display group, click **Apply** to apply the settings.

## Edit Display Group

1. To edit a display group, click ![Add icon](/img/ic-s-plus.svg) from the display group list.

2. In the dropdown, click the ![Edit icon](/img/ic-edit.svg) icon of the group you want to edit.

## Delete Display Group

1. Click the **Display Group** of the **Menu/Preset** that contains the group you want to delete.

2. In the dropdown, click the ![Edit icon](/img/ic-edit.svg) icon of the group you want to delete.

3. Click the ![Delete icon](/img/ic-delete-critical.svg) **Delete Display Group** button.

4. Click the **Apply** button to delete the display group.

    - If there is no display group, all pages and features will be visible.
