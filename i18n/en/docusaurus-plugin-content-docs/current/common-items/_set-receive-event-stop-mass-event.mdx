## Avoiding a large number of alerts

When numerous alerts occur, it temporarily stops notifying alerts for a set period. In <Cmdname sid="event_statistic" className="uitext" /> > <Cmdname sid="eventNoti" className="uitext" />, go to the <Cmdname sid="aboids_large_number_alerts" className="uitext" /> section.

<ImgLang img='set-event-huge-message.png' desc='Avoids mass notifications' />

- Select the <Cmdname sid="enable" className="uitext" /> toggle button to enable or disable the function.
- If the events occur more than <Cmdname sid="number_of_detections" className="uitext" /> during <Cmdname sid="detection_time" className="uitext" />, it stops alert notification for <Cmdname sid="pause_time" className="uitext" />.

For example, alert notification stops for 5 minutes when 20 events occur within 5 minutes. If the set <Cmdname sid="pause_time" className="uitext" /> elapses, the Avoids a large number of alerts function is disabled.

:::caution

If the number of text notifications exceeds 200 per day, the notification is paused and the following message appears. To disable text notification, select <Cmdname sid="terminate_now" className="uitext" />.

<ImgLang img='notification-set-large-message-warning.png' desc='Unblock SMS' />

:::
