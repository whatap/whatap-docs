The <Cmdname sid="side_counter" className="uitext" /> <code className="newfunc">New</code> menu is a core feature of the WhaTap monitoring solution, which provides the data collected from various monitoring targets in a visualized time-series chart. This allows users to effectively monitor the system performance, quickly identify anomalies, and gain insights for resolving issues.

- You can visually check various metrics via time series charts, and analyze multiple metrics upon failure at a time.

- You can select a specific time range to view past data and analyze the performance at that period.

- You can filter data based on specific time zones or conditions for more accurate analysis.

- Users can place their desired charts on the screen and freely adjust the layout.

- Users can arrange desired metrics and save them as presets for loading at any time.

:::note

## Changing the query time

To change the chart query time range for an individual widget, select ![Left arrow icon](/img/left-arrow.svg) or ![Right arrow icon](/img/right-arrow.svg) at the top of the widget. The time change range is the same as the time reference set in the <Cmdname sid="time" className="uitext" /> option. For example, if the <Cmdname sid="time" className="uitext" /> option is set to <Cmdname sid="1_day" className="uitext" /> and you select ![Left arrow icon](/img/left-arrow.svg) or ![Right arrow icon](/img/right-arrow.svg), you can change the time to a daily interval.

<ImgLang img="metrics-chart-new-change-time.png" desc="Chart widget" />

Alternatively, by dragging a specific area of the chart, you can change the time according to the dragged range for a view.

<ImgLang img="metrics-chart-new-chart-drag-time.png" desc="Change time" />

:::tip

- You can change the time for all widgets by changing for each widget with the <Cmdname sid="CNT08055" className="uitext" /> option enabled at the top of the screen.

  <ImgLang img="metrics-chart-new-chart-reset-time.png" desc="Time batch reset" />

- To change the query time range and then change it back to a recent time, select <LinkImage img="ic-recent-time-active.svg" className="ico active" />. 

  If the <Cmdname sid="CNT08055" className="uitext" /> option is enabled at the top of the screen, the ![Recent time icon](/img/ic-recent-time-active.svg) <Cmdname sid="BTN08155" className="uitext" /> button appears. By selecting this button, you can reset the time for all widgets.

:::

## Setting presets {#setpreset}

You can save and load the settings and layouts for widgets on the dashboard. You can create a new preset by adjusting its size and placing it on the desired position.

:::note

You can save or modify the presets with the <Cmdname sid="edit_project" className="uitext" /> role. For more information about the member roles, see [the following](../project/project-structure#member-auth).

:::

### Creating a new preset

1. Place the widgets in the desired format on the dashboard. You can also resize and place only the widgets to be frequently checked.

2. On the upper right of the screen, select ![Save icon](/img/ic-save.svg).

3. Enter a new preset name.

   <ImgLang img="metrics-chart-new-preset-save.png" desc="Preset" />

4. Select <Cmdname sid="save" className="uitext" />.

You can see the newly saved preset from the preset list.

:::note

- If any changes are made to the newly created preset, save the preset again. Select ![Save icon](/img/ic-save.svg) and then save the preset with the same name. Any changes are overwritten in the existing preset.

- If you go to another menu without saving changes on the dashboard, the changes are not saved.

- Presets are saved on a project basis and can be shared with other users.

- The targets saved in the preset are the layout of the dashboard, the widgets placed on the dashboard, and the options (<Cmdname sid="target" className="uitext" />, <Cmdname sid="interval" className="uitext" />) modified in the widgets. The <Cmdname sid="time" className="uitext" /> option at the top of the screen is not saved.

- To change the widget query time after loading a preset, use the <Cmdname sid="BTN08226" className="uitext" /> feature. For more information, see [the following](#batchchanges).

:::

### Deleting presets {#removepreset}

If you have unused presets, you can delete them from the preset list. Select ![Delete icon](/img/ico-remove-white.svg) on the right of the item to delete from the preset list.

## Editing a dashboard widget

The widgets placed on the dashboard can be placed and resized to the user's desired location.

### Moving a widget

When you move the top part of the widget by using the mouse pointer, it changes to the ![Move icon](/img/ic-move-pointer.svg) shape. At this time, you can move the widget by dragging with your mouse to the desired position.

If <Cmdname sid="layout" className="uitext" /> among the dashboard options is set to <Cmdname sid="ITM08273" className="uitext" />, you can resize the widget.

<ImgLang img="metrics-chart-new-size-widget.png" desc="Move a widget" />

:::note

If <Cmdname sid="layout" className="uitext" /> is not set to <Cmdname sid="ITM08273" className="uitext" />, you can adjust only the height of the widget. Resize the widget with the bottom center of the widget clicked with your mouse.

:::
