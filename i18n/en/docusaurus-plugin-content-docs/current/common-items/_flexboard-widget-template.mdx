## List widgets {#list}

### ![](/img/ic-table.svg) Table {#table}

This widget displays a list in table form. It is useful for checking values and status of data collected from different monitoring targets. For each listed item, you can check additional related information through columns. Text color in the list indicates performance status.

![Table widget](/img/fb-table-chart-widget.png)

Click the more button (![아이콘](/img/ico-more.svg)) in the top right of the widget to use the following features.

* ![Filter icon](/img/ico-filter.svg) : set filter options to display only the data you want

* ![Column icon](/img/ic-column.svg) : add or remove columns and change their order

### ![](/img/ic-templete.svg) Group widget {#group}

This widget groups multiple widgets into one composite.

![Group widget](/img/fb-group-widget-apm.png)

#### Place a group widget {#addgroup}

1. In the <Cmdname sid="widget_template" className="uitext" /> list, select <Cmdname sid="etc" className="uitext" /> &gt; <Cmdname sid="group" className="uitext" /> <Cmdname sid="widget" className="uitext" />.

1. In the top right of the placed group widget, select the <Cmdname sid="project_selection" className="uitext" /> button and choose the project that will provide data.

1. In the top right of the group widget, select the <Cmdname sid="grid" className="uitext" /> button and set the layout you want.

1. Select the <Cmdname sid="apply" className="uitext" /> button to complete layout settings.

1. In the top right of the group widget, select the <Cmdname sid="add_widget" className="uitext" /> button.

1. When the <Cmdname sid="widget_template" className="uitext" /> list appears on the right, choose the widgets you want in order.

:::note

* Only one project can be selected in a group widget.

* Widgets are placed in the order added. To change order, click and drag the widget name to the desired position.

* You can place up to 36 widgets in a group widget. You cannot add more widgets than the number set in the <Cmdname sid="grid" className="uitext" /> option.

* When you have placed the maximum number of widgets, the <Cmdname sid="grid" className="uitext" /> button becomes disabled.

* You can open a group widget in a new window. Enter <Cmdname sid="view_mode" className="uitext" />, then click the ![Popup icon](/img/ic-popup.svg) button in the top right of the widget.

:::

#### Remove a widget in a group widget {#removewidget}

To delete an individual widget placed in a group widget, click the more button (![아이콘](/img/ico-more.svg)) in the top right of the widget, then click the ![Delete icon](/img/ico-remove.svg) button.

#### Collapse or expand a group widget

You can collapse or expand a group widget. Click the ![Collapse icon](/img/ic-caret-down.svg) or ![Expand icon](/img/ic-caret-up.svg) button in the top right of the widget.

![Group widget](/img/fb-group-widget-fold.png)

This feature helps you use space efficiently when many widgets are placed on a dashboard. You can also expand only when you want to view information.
