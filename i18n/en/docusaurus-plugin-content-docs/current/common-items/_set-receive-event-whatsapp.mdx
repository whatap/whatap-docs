### Adding to receive WhatsApp notifications

The notifications can be received through WhatsApp.

<ImgLang img='set-event-whatsapp-notification.png' desc='WhatsApp'/>

1. Select the profile icon on the upper right of the screen. 

2. If a pop-up menu appears, select <Cmdname sid="MNU07360" className="uitext" />. 

3. In the <Cmdname sid="account_user_phone" className="uitext" /> section, select <span class="uitext">WhatsApp</span>.

4. Enter the phone number to receive authentication numbers in <Cmdname sid="phone" className="uitext" />.

5. Select <Cmdname sid="BTN06578" className="uitext" />.

6. Enter the 6-digit verification code sent through your WhatsApp application.

7. Select <Cmdname sid="BTN06501" className="uitext" />.

:::note

To change the registered phone number, select <Cmdname sid="BTN06594" className="uitext" /> and then repeat the above steps.

:::
