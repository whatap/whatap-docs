## Setting the alert language

You can change the language for alert messages that occur in the project.

1. In Home screen, select a project to change the language for alert messages.

2. In the menu on the left of the screen, select <Cmdname sid="management" className="uitext" /> > <Cmdname sid="project_management" className="uitext" />.

3. In the <Cmdname sid="TTL06021" className="uitext" /> section, select a desired language.

4. Select <Cmdname sid="save" className="uitext" /> on the lower right of the screen.
