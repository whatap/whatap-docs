To use the extension pack function after installation, create the configuration file and then restart the agent as follows.

### MySQL instances

The MySQL/MariaDB monitoring agent is available.

### Checking the web URL performance

The WhaTap infrastructure agent accesses the given web address and then collects the access status and response time. The collected web URL performance can be checked in <Cmdname sid="category" className="uitext" /> `server_webcheck`. Add web addresses to check with the following command. For more information about URL monitoring, see [the following](../url/url-intro).

<Tabs groupId='server-user-script'>
<TabItem value="lxs" label="Linux Shell">

```bash title=SH
echo "web address to check" | sudo tee -a /usr/whatap/infra/conf/webcheck.conf
echo "webcheck.enabled=true" | sudo tee -a /usr/whatap/infra/conf/whatap.conf
sudo service whatap-infra restart
```

</TabItem>
<TabItem value="wps" label="Windows Powershell">

```bash title=SH
Add-Content -Path "C:\Program Files\WhatapInfra\webcheck.conf" -Value "web address to check"
Add-Content -Path "C:\Program Files\WhatapInfra\whatap.conf" -Value "webcheck.enabled=true"
Restart-Service "Whatap Infra"
```

</TabItem>
</Tabs>

### Redis Instances

The WhaTap infrastructure agent accesses the given Redis address and collects multiple performance metrics including the number of connections. The collected performance data can be checked in <Cmdname sid="category" className="uitext" /> `server_redis`. You can add the connection data to the Redis server to be checked with the following command.

<Tabs groupId='server-user-script'>
<TabItem value="lxs" label="Linux Shell">

```bash title=SH
sudo echo "addr=Redis address:port" | sudo tee /usr/whatap/infra/conf/redis.conf
sudo echo "password=Redis password" | sudo tee -a /usr/whatap/infra/conf/redis.conf
sudo service whatap-infra restart
```

</TabItem>
<TabItem value="wps" label="Windows Powershell">

```bash title=SH
Add-Content -Path "C:\Program Files\WhatapInfra\redis.conf" -Value "addr=Redis address:port"
Add-Content -Path "C:\Program Files\WhatapInfra\redis.conf" -Value "password=Redis password"
Restart-Service "Whatap Infra"
```

</TabItem>
</Tabs>
