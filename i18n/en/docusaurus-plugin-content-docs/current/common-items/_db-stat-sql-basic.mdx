1. In ![number 1](/img/number-01.png) <Cmdname sid="time" className="uitext" />, select the time and date to view the data. You can select a lookup time after selecting the green button.

2. In ![number 1](/img/number-01.png) <Cmdname sid="instance" className="uitext" />, select a target to view the data.

3. Select ![Search icon](/img/ico-btn-search.png).

The top 50 entries appear in the table.

:::note

- Select the clock display area to query a random date and time. If you click the date and time text area, the selectable options for the date and time appear.

- If you select each column in the table header, you can sort the list by the selected item.

- You can sort the searched results by the item selected in <Cmdname sid="TTL07772" className="uitext" />. Select a desired item and then select <LinkImage img="ico-btn-search.png" className="ico" />.

- In <Cmdname sid="TTL07773" className="uitext" />, you can set the number of entries to display in the table. Finish the settings and then select <LinkImage img="ico-btn-search.png" className="ico" />.

- To download the viewed data as a CSV file, select ![Download icon](/img/ico-download.svg).

- For more information on how to use the <Cmdname sid="time" className="uitext" /> option, see [the following](usage-ui#timeselector).

<InDoc product="oracle-pro">

- You can hide the table header columns or add any desired items. You can also change the order of the columns. For more information, see [the following](usage-ui#change-column).

</InDoc>
