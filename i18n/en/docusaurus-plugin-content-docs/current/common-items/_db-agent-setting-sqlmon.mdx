### User Metric Collection (SQLMON)

SQLMON periodically collects the results of user-defined SQL queries and stores them as metrics.  
It is suitable for lightweight queries that return numeric results.

* **sqlmon** <span class='type'>Int</span>

  Default value: `1`

  - `1`: Run both dbx and sqlmon simultaneously  
  - `2`: Run only sqlmon  

<Xclude product="oracle-pro">

  Supported version: 2.3.5 or later

</Xclude>

<InDoc product="oracle-pro">

  Supported version: dmx 1.0.4y or later

</InDoc>

#### Configuration

1. Create a `sqlmon` directory in the agent installation path.  
2. Save the query to be collected as a `.sql` file.  
3. Enable SQLMON in `whatap.conf`.  
4. Restart the agent.  
5. In the ![아이콘](/img/ic-sitemap.svg) <Cmdname sid="sitemap" className="uitext" /> &gt; <Cmdname sid="side_metricsSearch" className="uitext" /> menu, check the sqlmon category.

```properties
# whatap.conf
sqlmon=1
```
