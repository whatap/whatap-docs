### Monitoring each classified agents

You can easily monitor multiple agents by categorizing them in Agent Settings.

- <Cmdname sid="ITM06005" className="uitext" />: Individual Agent

- <Cmdname sid="ITM06006" className="uitext" />: Group classified by the agent type (e.g. web server, database). Set it to <code>whatap.okind</code>.

- <Cmdname sid="ITM06007" className="uitext" />: Group categorized by the server (node). Set it to <code>whatap.onode</code>.

- <Cmdname sid="ITM06008" className="uitext" />: List of agents classified as <code>whatap.okind</code>.

- <Cmdname sid="ITM06009" className="uitext" />: List of agents classified as <code>whatap.onode</code>.

:::note

For more information about the agent configuration, see [the following](agent-name).

:::
