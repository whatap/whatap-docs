### Viewing HTTP parameters {#view-http-parameter}

You can view the HTTP parameters of the transaction in the <Cmdname sid="TTL06126" className="uitext" /> tab.

1.  Scroll down the page to the final step of the transaction. 

2.  Select the ![Lock icon](/img/ic-lock.svg) <Cmdname sid="TTL06520" className="uitext" /> item.

3.  If the **_HTTP-PARAMETERS_** window appears, select <Cmdname sid="BTN06764" className="uitext" />.

4.  Enter the specified Param Key.

You can see the hidden parameters.

<Xclude product='java,kubernetes'>

:::note

-   For the agent settings for HTTP parameters, see [the following](agent-transaction#profile_http_parameter).

-   <Cmdname sid="BTN06764" className="uitext" />: You can see the decrypted parameter value. The password is a 6-digit string in the <em><code>WHATAP_HOME</code>/paramkey.txt</em> file. You can change it to another string. The key in <em>paramkey.txt</em> is required to view the SQL variables and HTTP queries, and stop the threads.

:::

</Xclude>
