Use the pre-configured templates. You can easily perform the initial configuration and change the settings for use.

1. Select <Cmdname sid="side_flexboard" className="uitext" />.

2. Select a desired template from the template list on the left.

3. If the <Cmdname sid="flexboard_management" className="uitext" /> window appears, enter the name in <Cmdname sid="rename" className="uitext" />.

   * Select a widget from the <Cmdname sid="layout" className="uitext" />and then drag it to the desired location.

   <ImgLang img='flexboard-template-window.png' desc='Flex board management' />

4. Click <Cmdname sid="save" className="uitext" />. The Flex board is created based on the selected template. 

   * You can see the list of boards you have created in <Cmdname sid="side_flexboard" className="uitext" />.
