## Download Agent{#download-agent}

If the **access key** has been granted, in the <Cmdname sid="JAVA-agent-setup-1-subject" className="uitext" /> section, the <Cmdname sid="download" className="uitext" /> button is enabled.

1.  Select <Cmdname sid="download" className="uitext" /> to download the WhaTap agent file.

    :::tip

    You can directly download it from the server by using the following command:

    ```bash
    wget https://service.whatap.io/agent/whatap.agent.java.tar.gz
    ```

    :::

2.  Connect the server to be monitored.

3.  Unzip the agent file at the desired location of the target server to monitor. The unzipped path becomes `$WHATAP_HOME`.

    ```bash
    tar -zxvf whatap.agent.java.tar.gz
    ```
