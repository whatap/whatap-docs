By using the **_Filter_** option, you can select a desired node list for view. Through the `and` condition, multiple filters can be applied.

1.  If you select ![Add filter icon](/img/ico-add.svg) **_Add icon_** in the ![Filter icon](/img/ico-filter.svg) **_Filter_** field, the **_Add filter_** window appears as follows:

    <ImgLang img='k8s-resource-container-list-add-filter.png' desc='Add Filter' />

    -   Select **_Filter Key_** and **_Condition_**.

    -   Select a value that meets the conditions.

2.  Configure a filter and then select **_Apply_** to apply the filter.

3.  To add multiple filters, select ![Add filter icon](/img/ico-add.svg) **_Add icon_** in the **_Filter_** field or **_Add filter_** window.
