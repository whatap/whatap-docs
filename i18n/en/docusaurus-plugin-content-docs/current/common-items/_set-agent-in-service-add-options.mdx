### Adding options

<ProdImg img='app-set-agent-in-service.png' desc='Agent CONFIG.' />

1. In <Cmdname sid="side_agents" className="uitext" />, select the ![number1](/img/number-01.png) agent to add options to.

2. In ![number2](/img/number-02.png) <Cmdname sid="write_options" className="uitext" />, select options to add.

   - You can find options to add in <Cmdname sid="search" className="uitext" />. Enter text to filter matching options.
   - If you select <Cmdname sid="direct_input" className="uitext" />, you can enter the option keys and values.

3. Check the description and default value of the selected option key, and then enter the setting.

   <ImgLang img='add-agent-option-value.png' desc='' />

   - To cancel the selected option, select ![Delete icon](/img/ico-trash.svg).
   - To add new options, select <Cmdname sid="adding" className="uitext" /> and then repeat Step 2.

4. After adding the desired options, select <Cmdname sid="apply" className="uitext" /> on the upper right of the screen.

Apply the selected options and settings to agents.

:::note

- To initialize the content in modification, select ![Reset icon](/img/ico-refresh.svg) on the upper right of the screen.
- If you select <Cmdname sid="apply" className="uitext" /> without entering options, the options are deleted.
- The added options cannot be selected in the option list.
- The option keys that can be applied may differ depending on the application type and agent version.
- The example images may differ depending on the application type and agent version.

:::
