Home > Select Project > **_STAT/REPORT_** > **_Tablespace Size_**

You can view the maximum capacity and usage for each tablespace over time. It is provided in the form of a graph chart, and you can check the trend data from 1 day to 3 weeks ago.

<ProdImg img='db-stat-table-space.png' desc='' />

1.  Set the time to search in **_Time_**.

2.  Select an instance name to view the trend data of the instance.

3.  Select ![Search icon](/img/ico-btn-search.png).

:::note

You can select a lookup time (up to 3 weeks). Select the clock display area to query a random date and time. If you click the date and time text area, the selectable options for the date and time appear.

:::
