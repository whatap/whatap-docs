### Custom monitoring {#custom}

It provides the feature to check only the selected items in the response time distribution chart by adding specific transactions, domains, and IPs to the <Cmdname sid="custom" className="uitext" /> tab. This allows users to efficiently monitor and analyze specific problematic items.

1. Click the <Cmdname sid="custom" className="uitext" /> tab.

2. <Cmdname sid="transaction" className="uitext" /> or <Cmdname sid="domain" className="uitext" />, <Cmdname sid="ip" className="uitext" /> can be entered directly, selected from the <UI>Top 30</UI> list, or searched.

3. Click the added item. 

   <ImgLang img="txmap-custom-filter-select.png" desc="Custom" />

You can see the response time distribution for the selected items filtered in real time on the chart on the right.

:::note

To delete any added custom items, click ![Delete icon](/img/ico-trash.svg).

:::
