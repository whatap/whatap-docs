The <Cmdname sid="sms_alert_status" className="uitext" /> feature provides real-time pop-up information when an event added through the <Cmdname sid="side_eventSetting" className="uitext" /> menu occurs. This allows you to be notified immediately when an event occurs, allowing you to quickly recognize and respond to issues. Additionally, you can check the history of notifications that occurred already or were missed to identify and resolve situations that are difficult to respond to in real time.

## Checking the real-time alert

Real-time notifications generated through event settings appear as pop-ups on the upper right of the screen.

<ImgLang img="ui-home/main-ui-realtime-notification-popup.png" desc="Alert status" />

- You can see the notification creation time, title, and message.

- To close an individual notification, select ![Close icon](/img/ico-close.svg) on the upper right of the message.

- To close all pop-up notifications on the screen, select <Cmdname sid="BTN08345" className="uitext" />.

## Checking the notification content

To check the notification history, select a project and then select ![Notification icon](/img/ico-noti.svg) at the top of the screen on the following screen. The <Cmdname sid="sms_alert_status" className="uitext" /> window appears on the right of the screen. You can see up to 100 notifications for the last 24 hours of the current project.

<ImgLang img="ui-home/main-ui-realtime-notification.png" desc="Alert status" />

- If you select an individual event, the corresponding menu appears.

- You can go to the event by selecting a desired time zone from the hourly chart at the top of the real-time notification screen.

  <ImgLang img="ui-home/main-ui-realtime-notification-select-time.png" desc="Alert status" className="width-400" />

- To see all notification records, select <Cmdname sid="total_event_history" className="uitext" />. Go to <Cmdname sid="side_event" className="uitext" /> &gt; <Cmdname sid="side_eventHistory" className="uitext" />.

  <ImgLang img="ui-home/main-ui-realtime-notification-all-history.png" desc="Alert status" className="width-400" />

  :::note

  - The <Cmdname sid="total_event_history" className="uitext" /> button can be checked when nothing appears in the <Cmdname sid="sms_alert_status" className="uitext" /> list.

  - For more information about <Cmdname sid="side_eventHistory" className="uitext" />, see [the following](set-event-history).

  :::

## Clearing events in batch

From the <Cmdname sid="sms_alert_status" className="uitext" /> list, all event in progress can be cleared in batch. It takes about 30 seconds to 1 minute to end the event resolution.

- The <Cmdname sid="add_ack_to_project" className="uitext" /> feature can generate a large number of clearing notifications for events in progress.

- Even after using the <Cmdname sid="add_ack_to_project" className="uitext" /> feature, if the event rule is enabled and the data exceeds the threshold, the event can occur again.

- The event clearing notification can be disabled through event modification in <Cmdname sid="side_event" className="uitext" /> &gt; <Cmdname sid="side_eventSetting" className="uitext" />. Disable the <Cmdname sid="CNT08320" className="uitext" /> or <Cmdname sid="CNT08326" className="uitext" /> option.

1. Select ![Notification icon](/img/ico-noti.svg) on the upper right of the screen.

2. Select ![Notification clearing icon](/img/ic-check-circle.svg) on the upper right of the <Cmdname sid="sms_alert_status" className="uitext" /> window.

3. If the message window appears, select <Cmdname sid="DSC08323" className="uitext" />.

## Setting the real-time notification {#setrealtimenoti}

You can set which items are to be displayed in the list by using the <Cmdname sid="TTL08411" className="uitext" /> feature. Select ![Configuration icon](/img/ico-setting.svg) on the upper right of the <Cmdname sid="sms_alert_status" className="uitext" /> screen.

- <Cmdname sid="ITM08412" className="uitext" />: It provides notifications in pop-up form even when the <Cmdname sid="sms_alert_status" className="uitext" /> window is closed.

- <Cmdname sid="ITM08413" className="uitext" />: You can see the cleared events in the <Cmdname sid="sms_alert_status" className="uitext" /> list.

- <Cmdname sid="DSC08409" className="uitext" />: You can hear the notification sound in advance that occurs in your browser. Select ![Sound icon](/img/ico-noti-sound.svg) next to <span className="vslow">Critical</span> or <span className="slow">Warning</span>.
