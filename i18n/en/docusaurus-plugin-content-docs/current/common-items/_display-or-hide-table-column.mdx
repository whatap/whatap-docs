<Title hashid='display-or-hide-column' level={props.level}> Displaying or hiding the table columns</Title>

<InDoc product="browser">

You can display or hide column header items in the browser error list table.

</InDoc>

1. Select ![Column icon](/img/ico-column.svg) on the upper right of the table.

2. If the <Cmdname sid="column_settings" className="uitext" /> window appears, select or unselect the desired item in the list on the left.

   You can also hide columns by selecting the ![Delete icon](/img/ico-close.svg) button in the list on the right.

3. After all settings are finished, select <Cmdname sid="BTN06053" className="uitext" />. The set content is applied to the table.
