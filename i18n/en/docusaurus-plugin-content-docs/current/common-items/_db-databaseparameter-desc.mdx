Using the database configuration inquiry, you can view the database settings of the past. You can compare parameter values in the database by date through the comparison function. By default, you can see the data of the current time. 

If a problem occurs due to a parameter in the database, the cause can be quickly recognized through the database parameter comparison function.

<ProdImg img='db-databaseparameter.png' desc='Database parameter'/>

1. Set values for <Cmdname sid="reference_date" className="uitext" /> and <Cmdname sid="comparison_date" className="uitext" />.

2. Select a desired instance in <Cmdname sid="instance" className="uitext" />.

3. Click ![Search icon](/img/ico-btn-search.png). 

To check the changes between the current time and the selected past date at a glance, select <Cmdname sid="view_changes" className="uitext" />.
