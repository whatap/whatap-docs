### Options at the top

In the ![number1](/img/number-01.png) area, you can specify the time range of the chart and the agent to be monitored through the options at the top of <Cmdname sid="side_counter" className="uitext" />.

- <Cmdname sid="time" className="uitext" />: You can specify the start and end of the X-axis as the total range in time.

- <Cmdname sid="interval" className="uitext" />: You can specify the X-axis data interval as the time interval.

- <Cmdname sid="time_merge" className="uitext" />: As one of the data merging methods, you can merge data within the time specified by intervals.

  > For example, <Cmdname sid="merge_avg" className="uitext" /> indicates the average value of data within 1 hour.

- ![Icon](/img/ic-pageview.svg) <Cmdname sid="agent" className="uitext" />: You can specify the agents to query. If not specified, all items are queried.
