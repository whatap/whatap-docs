## PHP agent support environment

Before installing the WhaTap PHP monitoring agent, check the support environment.

### Supported Versions

| Category | Support |
| --- | --- |
| **Operation Mode** | PHP extension module |
| **PHP Version** | PHP 5.2.0 or later, PHP 7, PHP 8 |
| **Thread Safe** | NTS / ZTS (Zend Thread Safe) |

### Operating System

| OS | Minimum Version | Architecture |
| --- | --- | :---: |
| CentOS / Red Hat | 6.x | 64bit |
| Debian / Ubuntu | 12.04 | 64bit |
| Alpine Linux | - | 64bit |
| FreeBSD | 10.x | 64bit |
| Linux | - | ARM64 (aarch64) |

### Application Server

| Server | Support |
| --- | :---: |
| Apache | ✅ |
| PHP-FPM (Nginx) | ✅ |

### Libraries

| Library | Support |
| --- | :---: |
| PDO | ✅ |
| MySQLi | ✅ |

| mssql | ✅ |

### Database
