## PHP agent support environment

Before installing the WhaTap PHP monitoring agent, check the support environment.

- **Support versions**

  - Operates by the OPHP extension module
  - PHP PHP 5 >= 5.2.0, PHP 7, PHP 8
  - NTS / ZTS (Zend Thread Safe)

- **OS**

  - CentOS/Red Hat 6.x (64 bit) or later
  - Debian/Ubuntu 12.04 (64 bit) or later
  - Alpine Linux (64 bit)
  - FreeBSD 10.x (64 bit) or later
  - Linux-aarch64 (ARM64)

- **Application Server**

  - Apache
  - PHP-FPM (Nginx)

- **Library**

  - PDO
  - MySQLi
  - cURL
  - oci8
  - mssql
  - sqlsrv
  - psql
  - phpredis
  - CUBRID
  - odbc

- **Database**

  - Oracle (OCI8)
  - MySQL
  - SQL Server (mssql, sqlsrv)
  - PostgreSQL (psql)
  - Redis (phpredis)
  - CUBRID
  - Tibero (odbc)
