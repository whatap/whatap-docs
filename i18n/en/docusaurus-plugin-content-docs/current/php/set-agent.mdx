In the PHP agent, the required settings can be written and modified in _whatap.ini_. For user convenience, the agent configuration function is provided in [WhaTap Monitoring Service](https://service.whatap.io).

:::note

Settings are usually reloaded at runtime. However, due to the setting mechanism, the options applied when Apache or PHP-FPM are started may need to be restarted.

:::
