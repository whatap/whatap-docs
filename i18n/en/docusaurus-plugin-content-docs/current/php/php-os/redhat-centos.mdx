1.  Register the WhaTap **repository**.

    ```bash title=SH
    $ sudo rpm -Uvh http://repo.whatap.io/centos/5/noarch/whatap-repo-1.0-1.noarch.rpm
    ```

2.  Install the **package** with the following command. 

    ```bash title=SH
    $ sudo yum install whatap-php
    ```
