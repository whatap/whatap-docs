- **whatap.tx_caller_meter_enabled** <span class='type'>Boolean</span>

  Default `false`

  It generates topology statistics with the transaction data.

- **whatap.sql_dbc_meter_enabled** <span class='type'>Boolean</span>

  Default `false`

  It creates the topology statistics based on the database connection information.

- **whatap.httpc_host_meter_enabled** <span class='type'>Boolean</span>

  Default `false`

  It creates the topology statistics based on the outbound HTTP calls.
