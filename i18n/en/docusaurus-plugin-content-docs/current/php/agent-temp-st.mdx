To pause the PHP monitoring service, you must stop the PHP extension module and whatap-php service. See the following.

1.  Stop the PHP extension module. 

    Comment out the `extension=` syntax in _whatap.ini_. Set the comment by using a semicolon (_;_).

    ```bash
    $ sudo vi whatap.ini

    extension=whatap.so

    ;Comment
    ;extension=whatap.so
    ```

    Even if manual configuration is made in _php.ini_, comment out the `extension=` syntax. 

    :::note

    Apache or PHP-FPM service is started.

    :::

2.  Stop the whatap-php service.

    <Tabs>
    <TabItem value="rdct" label="Redhat/CentOS" default>

    ```bash
    $ sudo service whatap-php stop
    ```

    </TabItem>
    <TabItem value="dbub" label="Debian/Ubuntu">

    ```bash
    $ sudo service whatap-php stop
    ```

    </TabItem>
    <TabItem value="frbsd" label="FreeBSD">

    ```bash
    $ service whatap_php stop
    ```

    </TabItem>
    </Tabs>
