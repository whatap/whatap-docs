This option creates the topology map by using the data collected by the agents.

- **tx_caller_meter_enabled** <span class='type'>Boolean</span>

  Default `false`

  It generates topology statistics with the transaction data.

- **sql_dbc_meter_enabled** <span class='type'>Boolean</span>

  Default `false`

  It creates the topology statistics based on the database connection information.

- **httpc_host_meter_enabled** <span class='type'>Boolean</span>

  Default `false`

  It represents the HTTPC outbound data from the topology map.
