If a framework or open source is used in user applications, the settings to enable the tracing through the Java agent are provided. It can be used by adding the `weaving` option to the Java agent configuration file (_whatap.conf_).

```ini title='${WHATAP_HOME}/whatap.conf'
weaving=  
```

When using spring-boot-3.x, feign-client-11, or okhttp3-4.4 with a framework or open source, set the options as follows:

```ini title='whatap.conf'
weaving=spring-boot-3.0,feign-11,okhttp3-4.4
```
