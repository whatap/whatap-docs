The items must be checked after installing the agent. If any problem occurs during installation, see [the following](agent-troubleshooting).

## Restarting WAS

Restart the application server. If the following WhaTap character log appears in a running log, it indicates that the `JAVA_OPTS` setting has been completed.

```bash

INFO: Deployment of web application directory /var/lib/tomcat7/webapps/ROOT has finished in 577 ms
Nov 16, 2016 3:06:40 AM org.apache.coyote.AbstractProtocol start
INFO: Starting ProtocolHandler ["http-bio-8080"]
Nov 16, 2016 3:06:40 AM org.apache.catalina.startup.Catalina start
```
