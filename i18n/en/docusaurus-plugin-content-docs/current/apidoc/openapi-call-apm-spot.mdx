## Viewing all information

<span class='apitype'>GET</span> https://api.whatap.io/open/api/json/spot

```bash title='command'
curl -L -w "\n" -H "x-whatap-token: {{PROJECT API TOKEN}}" -H "x-whatap-pcode: {{projectCode}}" "https://api.whatap.io/open/api/json/spot"
```

```json title='response'
{
    "pcode": 8,
    "act_agent": "6",
    "inact_agent": "6",
    "host": "1",
    "cpucore": "2",
    "txcount": "459",
    "tps": "93.07",
    "user": "1007",
    "actx": "149",
    "rtime": "1873",
    "cpu": "29.35",
    "threadpool_active": "0",
    "threadpool_queue": "0",
    "dbconn_total": "600.0",
    "dbconn_act": "239.0",
    "dbconn_idle": "361.0",
    "act_method": "51",
    "act_sql": "24",
    "act_httpc": "64",
    "act_dbc": "10",
    "act_socket": "0"
}
```

| Response Field    |            Field Type           | Description                                                 |
| ----------------- | :-----------------------------: | ----------------------------------------------------------- |
| pcode             | <span class='api'>number</span> | Queried projected code                                      |
| act_agent         | <span class='api'>string</span> | [Active agent count](#active-agent)                         |
| inact_agent       | <span class='api'>string</span> | [Inactive agent count](#deactive-agent)                     |

| cpucore           | <span class='api'>string</span> | [Sum of CPU cores of the host](#sum-cpu)                    |

| actx              | <span class='api'>string</span> | [Active transactions](#active-transaction)                  |
| rtime             | <span class='api'>string</span> | [Average response time](#response-time)                     |
| cpu               | <span class='api'>string</span> | [Average CPU utilization](#usage-cpu)                       |
| threadpool_active | <span class='api'>string</span> | [Active thread count of the thread pool](#count-threadpool) |
| threadpool_queue  | <span class='api'>string</span> | Queuing thread count of the thread pool                     |
| dbconn_total      | <span class='api'>string</span> | Total DB connection count                                   |
| dbconn_act        | <span class='api'>string</span> | Active DB connection count                                  |

| act_method        | <span class='api'>string</span> | Active method count                                         |
| act_sql           | <span class='api'>string</span> | Active SQL count                                            |
| act_httpc         | <span class='api'>string</span> | Active HTTP call count                                      |

| act_socket        | <span class='api'>string</span> | Active socket count                                         |

## Active/inactive agent information

<span class='apitype'>GET</span> https://api.whatap.io/open/json/agents

```bash title='command'
$ curl -L -w "\n" -H "x-whatap-token: {{PROJECT API TOKEN}}" -H "x-whatap-pcode: {{projectCode}}" "https://api.whatap.io/open/json/agents"
```

```json title='response'
{
  "data": [
    {
      "host_ip": "127.0.0.1",
      "oname": "TC-0-1-8087",
      "active": true
    },
    {
      "host_ip": "127.0.0.1",
      "oname": "TC-0-1-8086",
      "active": true
    }
  ]
}
```

| Response Field |       Field Type                 | Description                                                                |
| -------------- | :------------------------------: | -------------------------------------------------------------------------- |
| host_ip        |  <span class='api'>string</span> | IP address of the host where the agent is located                          |
| oname          |  <span class='api'>string</span> | Agent name                                                                 |
| active         | <span class='api'>boolean</span> | Whether the agent is active (Active: `true`, Inactive: `false`)            |
