The Whatap agent writes the necessary settings for each agent in *whatap.conf*. The agent locates and loads the configuration file through the environment variables.

If any separate setting, the agent has been implemented to find *whatap.conf* in the path where it is located. It checks whether *whatap.conf* has been changed every 5 seconds and then reloads the file.

For user convenience, [Whatap Monitoring Service](https://service.whatap.io/) also provides the agent configuration function.
