The Whatap agent writes the necessary settings for each agent in *whatap.conf*. The agent locates and loads the configuration file through the environment variables. When the Node.js server operates in a container or PaaS environment, the agent settings are configured using environment variables. For user convenience, the [WhaTap Monitoring Service](https://service.whatap.io) also provides the agent configuration function.
