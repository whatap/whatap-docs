## Page group filter

You can apply the filtering options for each page group. You can set the filter options by selecting the <Cmdname sid="page_group" className="uitext" /> filter button at the top left of the dashboard screen. 

- <Cmdname sid="equal" className="uitext" />: You can view the data that matches the selected page group.

  <ImgLang img='browser-dashboard-filter.png' desc='Page group filter' />

- <Cmdname sid="contain" className="uitext" />: You can query data for a group of pages that contain a specific string.

  <ImgLang img='browser-dashboard-filter-include.png' desc='Page group filter' />

- <Cmdname sid="exclude" className="uitext" />: You can query data for a group of pages that exclude a specific string.

  <ImgLang img='browser-dashboard-filter-include.png' desc='Page group filter' />

- <Cmdname sid="add_filter" className="uitext" />: To apply multiple filters, select <Cmdname sid="add_filter" className="uitext" /> and then enter the desired conditions. You can add filters with the **AND** condition.

  <ImgLang img='browser-dashboard-filter-and.png' desc='Page group filter' />
