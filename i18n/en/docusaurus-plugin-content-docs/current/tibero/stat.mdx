## Tablespace Size

Home > Select Project > **_STAT/REPORT_** > **_Tablespace Size_**

<ImgLang img="tibero-tablespace-size.png" desc="Tablespace Size" />

Set the **_time_** and **_instance_** and then select ![Search icon](/img/ico-btn-search.png). Depending on the settings, the searched data is provided in a chart and a table.

-   **_Time_**
    -   If you select the green icon, you can search up to 3 weeks. 
    -   With the lookup period selected, select ![Left arrow icon](/img/left-arrow.svg) or ![Right arrow icon](/img/right-arrow.svg). Then you can change the lookup time at the set time intervals.
    -   If you select the date and time, you can set the desired time in detail. After configuration, select **_OK_**.

-   **_Instance_**: You can select an instance that is connected with the project.

-   **_Tablespace Size_** Chart

    The maximum capacity and usage trend are provided for each tablespace in a cumulative chart. You can check the trend from 1 day up to 3 weeks. You can change the chart data by selecting **_Total_** or **_Used_** on the upper right corner.

-   **_Tablespace Detail_**

    The searched data is provided in a table. You can compare the start times and end times based on the lookup time, and you can see the change in data usage size. 

    -   To check the details in a tablespace, select ![Right arrow icon](/img/right-arrow.svg) located on the left of each item.
    -   If you select each item in the table header, you can sort the table by the selected item.
