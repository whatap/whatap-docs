WhaTapモニタリングサービスを利用するには、まず、会員登録を行ってください。

:::tip

-   WhaTapモニタリングを15日間無料で体験してみてください。 決済方式の詳細については、[次の文書](../billing/billing.mdx)を参照してください。 
-   会員登録、プロジェクト作成後、プロジェクトアクセスキーを発行した日から無料体験を開始します。
-   体験期間終了後にサービスを続けて利用したい場合は、決済情報を登録してプロジェクトを有料に切り替えてください。:::

1.  [WhaTapホームページ](http://www.whatap.io/) へ移動してください。
2.  右上の**_[無料で始める](https://www.whatap.io/ko/signup/)_**)ボタンをクリックしてください。
3.  名前、メール、パスワード、会社名を順番通りに入力してください。
4.  **_アカウント作成_**ボタンをクリックしてください。

:::note
通知をSMSで受信するにはログインし、右上のプロファイルアイコン > **_アカウント情報管理_**の順に選択してください。 **_ユーザーの電話番号_**セクションに移動し、電話番号を入力してください。
:::
