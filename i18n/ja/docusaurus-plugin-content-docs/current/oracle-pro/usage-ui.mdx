## セッション情報を確認する {#checksessioninfo}

### <Cmdname sid="session_detail" /> {#session-details}

次のメニュー経路でセッションの項目を選択すると確認できる<Cmdname sid="session_detail" className="uitext" />画面について案内します。

- <Cmdname sid="side_dashboard" className="uitext" /> &gt; <Cmdname sid="side_dbMonitoring" className="uitext" />

- <Cmdname sid="side_dashboard" className="uitext" /> &gt; <Cmdname sid="side_multiInstance" className="uitext" />

- <Cmdname sid="side_analysis" className="uitext" /> &gt; <Cmdname sid="side_dbTrendComparison" className="uitext" />

- <Cmdname sid="side_analysis" className="uitext" /> &gt; <Cmdname sid="side_dbLockTree" className="uitext" />

- <Cmdname sid="side_analysis" className="uitext" /> &gt; <Cmdname sid="side_dbWaitAnalysis" className="uitext" />

セッションのテーブル一覧でセッション項目を選択すると<Cmdname sid="session_detail" className="uitext" />画面が表示されます。 選択したセッションについての詳細を確認できます。

<InDoc product="oracle-pro,oracle"><ImgLang img='db-session-details-info-oracle-pro.png' desc='セッション詳細' /></InDoc>

- セッションに関する情報は、既定値でリアルタイムで照会されています。 タイムセレクターの![静止アイコン](/img/ico-pause.svg)ボタンを選択すると、リアルタイム照会を中止し、中止した時間ベースのデータを確認できます。 リアルタイム照会を中止した状態でデータを更新するには、![更新アイコン](/img/ico-refresh.svg)ボタンを選択します。

- ***Kill Session***：選択したセッションを停止できます。 <Cmdname sid="session_kill" className="uitext" />画面が表示されたらパスワード(***Parameter Key***)を入力し、<Cmdname sid="apply" className="uitext" />ボタンを選択します。

  :::note

  この機能は、リアルタイム照会モードでのみ利用できます。

  :::

- <Cmdname sid="TTL06721" className="uitext" />：選択したセッション履歴が確認できる<Cmdname sid="side_sessionHistory" className="uitext" />メニューに移動できます。

  :::note

  <Cmdname sid="side_sessionHistory" className="uitext" />メニューの詳細については、<Link to="analysis-session-history">次の文書</Link>を参照してください。

  :::

- ***Session Stat***: 選択したセッションの様々な性能メトリクスと統計情報を提供します。 セッションで実行された作業のタイプと回数、リソース使用量などが含まれます。

- ***Delta*** / ***Sigma***: 選択したセッションの変化率と統計的分析情報を提供します。 セッションのアクティビティに対する変化量を確認できます。

- ***Session Info***: 選択したセッションの基本情報とステータスを含む詳細を確認できます。

- ***Wait Info***: 選択したセッションの待機イベントと待ち時間の情報を提供します。 特定の作業を実行するために、データベースシステムで待機する時に発生する待機イベントに関する情報も確認できます。

- ***SQL Text*** / ***Plan***：選択したセッションのSQLクエリ文とPlan情報を確認できます。 詳細については[次の文書](#query-details)を参照してください。

### <Cmdname sid="TTL06224" /> {#query-details}

現在実行中のセッションまたは過去のセッションのSQLクエリの情報を確認するためには<UI>query_text</UI>のカラム項目を確認します。 クエリー文とSQL Planを詳細に確認できる<Cmdname sid="TTL06224" className="uitext" />画面が表示されます。

<ImgLang img="db-sql-details-new-oracle-pro.png" desc="SQL 상세" />

- <Cmdname sid="BTN06225" className="uitext" />：該当クエリーを基準に統計データを照会できる<Cmdname sid="side_dbSqlStat" className="uitext" />メニューに移動できます。

- ***Formatting***: SQLクエリ文にインデントおよびフォーマットを適用して可読性を高めます。

- ***Runtime Plan***: 選択されたSQLクエリの実行計画とランタイム情報を提供します。 実行回数、平均実行時間、平均物理的読み取りなどの詳細を提供します。

  - <Cmdname sid="select_1_hour" className="uitext" /> / <Cmdname sid="select_1_day" className="uitext" />：最近の1時間または1日を基準に統計情報を提供します。

- ***Explain Plan***: オプティマイザーが予測した実行計画を示します。 費用、作業、オブジェクト名、カーディナリティなどの情報を提供します。

- ***Plan History***: データベースで実行されたSQLクエリの実行計画に対する履歴を確認できます。

- ***Bind Capture***: データベースで実行されたSQLクエリに使用されたバインド変数の値を確認できます。 これにより、クエリ実行の実際の内容を確認できます。

  :::note

  リアルタイムで実行されたbind値ではなく、データベースにキャプチャされた値(`v$sql_bind_capture`)です。 件数が多いと、100個まで表示します。

  :::

### チューニングガイド{#tunning-guide}

<Cmdname sid="TTL06224" className="uitext" />画面のPlanツリーでチューニングガイドを確認できます。 ![チューニングアイコン](/img/ic-chat-red.svg)ボタンを選択します。 提供するチューニングガイドを参照して、データベースの最適化を進めてください。

<ImgLang img="db-sql-details-tunning-guide-oracle-pro.png" desc="Tunning guide" />

:::note

チューニングガイドは、Plan Treeノードが`TABLE ACCESS (FULL)` / `INDEX (SKIP SCAN)`である場合とノードに`CARTESIAN`キーワードが含まれている場合、確認できます。

:::
