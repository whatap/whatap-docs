<Cmdname sid="side_analysis" className="uitext" /> > <Cmdname sid="side_dbCompareTrends" className="uitext" /> で、データベースの主要指標を時間帯ごとに確認し、パフォーマンスの変化を追跡します。  
異なる日付の指標を比較することで、運用の傾向を分析できます。  
<Cmdname sid="active_sessions" /> テーブルでは、現在実行中のセッションと長時間実行中のセッションの両方を確認できます。
