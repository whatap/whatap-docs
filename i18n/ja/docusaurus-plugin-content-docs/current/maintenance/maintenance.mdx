ホーム画面 > プロジェクト選択 > **_管理_** > **_メンテナンス計画_**

![メンテナンス計画](/img/maintenance.png)

メンテナンス計画を登録して、登録したメンテナンス時間には受信設定した通知を一時停止できます。繰り返し設定して定期的なメンテナンス計画を設定してください。

:::note
メンテナンス計画を追加して修正・削除するには、プロジェクトに対する**修正**権限が必要です。メンバー権限の詳細については、[次の文書](../project/project-structure#project-auth)を参考にしてください。
:::

## メンテナン計画を登録する

1.  画面右上の**_+ 計画追加_**ボタンを選択してください。

2.  **_メンテナンス計画の追加_**ウィンドウが表示されたら**_作成者_**、**_メンテナンス履歴_**の順に入力してください。

  ![メンテナンス計画追加](/img/maintenance-add.png)

1.  **_繰り返し_**項目で周期的な繰り返し間隔を選択してください。

    -   **_1回のみ_**：設定した時間中1回のみメンテナンス計画の通知が動作します。繰り返し設定が必要ない場合は選択してください。
    -   **毎日**：毎日繰り返すメンテナンス計画です。
    -   **毎週**：毎週選択した曜日ごとに繰り返すメンテナンス計画です。
    -   **_毎月_**：毎月選択した日ごとに繰り返すメンテナンス計画です。
    -   **毎年**：毎年選択した月、日ごとに繰り返すメンテナンス計画です。

2.  開始時刻と終了時刻を設定してください。

3.  設定をすべて完了した後、**_確認_**ボタンを選択してください。

メンテナンス計画一覧で追加した項目を確認できます。 

メンテナンス計画を登録した後、プロジェクト内のメニューの使用中1時間以内に進行予定または進行中のメンテナンス計画があれば、画面上にバナー形態でメンテナンス計画を表示します。

![メンテナンス計画ヘッダーバナー](/img/maintenance-header.png)

-   進行中のメンテナンス計画はオレンジ色、1時間以内に進行予定のメンテナンス計画は灰色の背景として表示されます。

    ![進行中](/img/progress-banner-1.png)

    ![進行予定](/img/before-banner.png)

-   上段のメンテナンス計画バナーを選択すると、右側に**_メンテナンス計画_**一覧が表示されます。1時間以内に進行予定または進行中のメンテナン計画の一覧を確認することができます。

-   メンテナンス計画の一覧で一つを選択すると、**_メンテナンス計画確認_**ウィンドウが表示され、詳細情報が確認できます。そのウィンドウでは、メンテナンス備計画を修正できません。

## メンテナンス計画を修正する

1.  **_メンテナンス計画_**一覧で修正しようとする項目の右側の![編集アイコン](/img/ico-edit.svg)ボタンを選択してください。 

2.  **_修正_**ウィンドウが表示されたら、必要な項目を修正してください。 

3.  修正を完了したら**_確認_**ボタンを選択してください。

## メンテナンス計画を非活性化する

**_メンテナンス計画_**一覧で活性化されている一覧の一番右のトグル ボタンを選択すると、非活性化されます。そのメンテナンス計画の項目は機能しません。

## メンテナンス計画を削除する

1.  **_メンテナンス計画_**一覧で削除しようとする項目の一番左のチェックボックスを選択してください。重複選択できます。

2.  画面右上の**_削除_**ボタンが非活性化されます。 

3.  **_削除_**ボタンを選択すると、チェックしたメンテナンス計画を削除できます。

:::note

-   すべてのメンテナンス計画を選択するには、一覧上部の一番左のチェックボックスを選択してください。
-   別途の削除確認メッセージは表示されません。削除する前に注意してください。
    :::

## ダッシュボードにメンテナンス計画ウィジェットを追加する

**_Flexボード_**からウィジェットテンプレートとして**_メンテナンス計画_**ウィジェットを提供します。 

![メンテナンス計画ウィジェット](/img/maintenance-widget.png)

1.  ホーム画面 > プロジェクト選択 > **_ダッシュボード_** > **_Flexボードメニュー_**に移動した上で**_ダッシュボード一覧_**でウィジェットを追加するダッシュボードを選択してください。

2.  **_修正モード_**を選択してください。

3.  **_ウィジェットテンプレート_**一覧で**_メンテナンス計画_**を選択してください。

4.  **_メンテナンス計画_**ウィジェットをご希望の位置に配置してください。

:::note

-   **_メンテナンス計画_**ウィジェットで一覧のいずれかを選択すると、詳細情報を確認できるウィンドウが表示されます。そのウィンドウでは、メンテナンス計画を修正することはできません。
-   **_メンテナンス計画_**ウィジェット右上の![もっと見るアイコン](/img/ico-more.svg) > ![ショートカットアイコン](/img/ico-shortcut.svg)ボタンを選択すると、当該プロジェクトの**_管理_** > **_メンテナンス計画_** メニューに移動します。
    :::
