WhaTapモニタリングサービスの無料利用期間の満了またはサービスによる無料利用基準を超過した状態で利用を継続したい場合は、決済情報を登録して有料転換しなければなりません。

## 決済情報の登録{#register-pay}

決済はカード定期決済、税金計算書発行の2種類から選択できます。 決済を進める前に、次の事項を事前に準備してください。

-   **決済**権限アカウント

-   カード決済の場合、決済カード番号及び有効期限

-   税金計算書発行の場合、事業者登録証、計算書受信メール

:::note

**決済**権限の詳細については、[次の文書](../project/project-structure#project-auth)を参考にしてください。

:::

### クレジットカード

1.  [WhaTapモニタリングサービス](https://service.whatap.io)で**決済**権限を持つアカウントでログインしてください。

2.  ホーム画面左下の**_利用内訳_**を選択してください。

    <ImgLang img='tax_invoice_1.png' desc='利用履歴' />

3.  **決済情報\***ページへ移動すると、**_決済情報登録_**ボタンをクリックしてください。

4.  **_決済情報登録_**ウィンドウが表示されたら、まず国を選択してください。

5.  会社名または法人名を入力してください。

6.  **_電話番号_**と**_請求書言語_**項目を確認し、必要な場合は変更してください。

7.  ミドルネームを入力してください。 必須事項ではありません。

8.  **_次へ_**ボタンを選択してください。

#### 国内ユーザー

1.  決済方法を**_クレジットカード(ナイスペイ)_**で選択し、**_次へ_**ボタンを選択してください。

2.  クレジットカードのカード番号、有効期限、パスワードの前の2桁の数字を順番に入力してください。 

3.  **_カードタイプ_**項目で**_企業_**または**_個人_**を選択してください。

    -   **_企業_**：法人カードで、カードの実物に会社名が入力されています。 事業者登録番号と商号を入力する必要があります。
    -   **_個人_**：個人カードで、カードの実物にカード所有者の名前が入力されています。 会社から支給された場合でも、個人名が明記されていれば個人カードです。 生年月日を8桁で入力してください。

4.  カード情報の入力が完了したら**_請求書の受信者設定_**ボタンをクリックしてください。

5.  請求書を受け取る担当者情報を順番に入力し**_完了_**ボタンをクリックしてください。

:::note

複数のメールで請求書を受け取るには、メールアドレスにコンマ（,）を使用して複数入力できます。 例）a@whatap.io, b@whatap.io, c@whatap.io

:::

#### 海外ユーザー

1.  決済方法を**_クレジットカード(Stripe)_**で選択し、**_次へ_**ボタンを選択してください。

2.  クレジットカード所有者の名前とクレジットカード番号、有効期限、CVC番号を順番に入力してください。

3.  チェックされた約款情報を確認してください。

4.  カード情報の入力が完了したら**_請求書の受信者設定_**ボタンをクリックしてください。

5.  請求書を受け取る担当者情報を順番に入力し**_完了_**ボタンをクリックしてください。

:::note

複数のメールで請求書を受け取るには、メールアドレスにコンマ（,）を使用して複数入力できます。 例）a@whatap.io, b@whatap.io, c@whatap.io

:::

### 税金計算書

1.  [WhaTapモニタリングサービス](https://service.whatap.io)で**決済**権限を持つアカウントでログインしてください。

2.  ホーム画面左下の**_利用内訳_**を選択してください。

    <ImgLang img='tax_invoice_1.png' desc='利用履歴' />

3.  **決済情報\***ページへ移動すると、**_決済情報登録_**ボタンをクリックしてください。

4.  **_決済情報登録_**ウィンドウが表示されたら、まず国を選択してください。

5.  会社名または法人名を入力してください。

6.  **_電話番号_**と**_請求書言語_**項目を確認し、必要な場合は変更してください。

7.  ミドルネームを入力してください。 必須事項ではありません。

8.  **_次へ_**ボタンを選択してください。

#### 国内ユーザー

1.  決済方法を**_請求書発行_**で選択し、**_次へ_**ボタンを選択してください。

2.  計算書発行情報を入力してください。

    -   事業者登録証上の法人名、代表者名、事業者登録番号、電子税金請求書発行メールなどを順に入力してください。

    -   税金計算書の発行日の基準は次のとおりです。

        -   **_25日_**：使用月の25日付けで請求書を発行します。
        -   **_月末_**：使用月の最後の日付で請求書を発行します。
        -   **_請求書の発行日(翌月の初め)_**:使用した月を基準に翌月初めの請求書を発送する日付です。

    -   必要に応じて、業種と業態を入力してください。

3.  請求書の入力を完了したら**_請求書の受信者設定_**ボタンをクリックしてください。

4.  請求書を受け取る担当者情報を順番に入力し**_完了_**ボタンをクリックしてください。

    **_CC_**項目に一緒に受信するメールアドレスを追加できます。 2つ以上入力する場合は、コンマ（,）を区切り文字として利用してください。

:::note

-   事業者登録番号を入力し**_認証する_**ボタンをクリックして認証を進めてください。

-   税金請求書の逆発行を希望する場合は、billing@whatap.ioまでお問い合わせください。

:::

#### 海外ユーザー

1.  決済方法を**_請求書の発送_**で選択し、**_次へ_**ボタンを選択してください。

2.  **_担当者名_**と**_メール_**項目をご確認ください。 必要に応じて修正できます。 

    **_CC_**項目に一緒に受信するメールアドレスを追加できます。 2つ以上入力する場合は、コンマ（,）を区切り文字として利用してください。

3.  請求書の発行日を選択してください。 基準は次のとおりです。

    -   **_25日_**：使用月の25日付けで請求書を発行します。

    -   **_月末_**：使用月の最後の日付で請求書を発行します。

    -   **_請求書の発行日(翌月の初め)_**:使用した月を基準に翌月初めの請求書を発送する日付です。

4.  **_住所_**項目を入力してください。

5.  **_適用_**ボタンを選択します。

## 請求書受信メールアドレスの修正

ホーム画面 > **_利用内訳_** > **_決済情報_**>

<ImgLang img='invoice_mail_1.png' desc='決済アカウントの修正' />

担当者が変更されて請求書の受取人を変更するには、決済アカウントを修正してください。 

**_決済アカウント_**でメールアドレスの右側の![編集アイコン](/img/ico-edit.svg)**_修正_**ボタンをクリックしてください。 **_修正_**ウィンドウが表示されたら、担当者情報(**_名前_**、**_連絡先_**、**_請求書の受信メール_**)を順番に修正し、**_登録_**ボタンをクリックしてください。

:::note

複数のメールで請求書を受け取るには、メールアドレスにコンマ（,）を使用して複数入力できます。 例）a@whatap.io, b@whatap.io, c@whatap.io

:::

## プロジェクト有償転換

決済情報の登録が完了したら、**_プロジェクト有料転換_**メニューに移動し、保有しているプロジェクト全体は有料に自動転換します。 ユーザーが登録したプロジェクト全体の一覧を確認できます。

<ImgLang img='billing-pay-list.png' desc='プロジェクト有償転換' />

-   プロジェクトステータスが無料、トライアル、使用制限の場合は**_有料転換_**ボタンがアクティブになります。 プロジェクトを有料に切り替えて使用したい場合は、次の方法を利用してください。

    -   各プロジェクト項目の**_有料転換_**ボタンをクリックすると、個別に有料転換することができます。 
    -   **_有料_**状態ではなく、プロジェクト全体を一括で有料に転換するには![指示線1](/img/number-01.png)**_有料に切り替える_**ボタンをクリックしてください。
    -   各プロジェクト項目の左側にあるチェックボックスを選択し、**_有料転換_**ボタンをクリックしてください。 複数のプロジェクトを選択できます。

    **_有料転換_**ウィンドウが表示されたら、プロジェクト一覧を確認して**_確認_**ボタンを選択してください。

-   削除プロジェクトを一覧に含めて確認するには![指示線1](/img/number-01.png)**_削除プロジェクトを含む_**トグルボタンを選択してください。

-   **決済権限**を持っていないプロジェクトの一覧を確認するには![指示線1](/img/number-01.png)**_決済権限未保有プロジェクトを含む_**トグルボタンを選択してください。

-   プロジェクトの一覧をCSVファイルとしてダウンロードするには![指示線1](/img/number-01.png)**_CSVダウンロード_**ボタンをクリックしてください。

:::note

-   **決済権限**は有料転換が可能で、決済手段を登録でき、請求書を受け取ることができる権限です。
-   有料転換時点が7日未満のプロジェクトの場合、有料転換を取り消すことができます。 **_有料キャンセル_**ボタンをクリックしてください。
-   有料転換時点から7日以降のプロジェクトを削除できる**_プロジェクト削除_**ボタンを提供します。

:::

## 決済情報の確認

ホーム画面 > **_利用内訳_** > **_決済情報_**>

<ImgLang img='credit_card_8.png' desc='決済情報' />

ユーザーが登録した決済情報を確認できます。 **_修正_**ボタンを選択して他の決済手段に変更することができます。 決済情報登録の詳細については、[次の文書](#register-pay)を参照してください。

請求書は毎月5営業日以内に発送され、実際の決済は25日に行われます。 正常な決済のため、25日前に入金されるようにしてください。

## 請求及び納付履歴

ホーム画面 > **_利用内訳_** > **_請求及び納付履歴_**

納付履歴、未納金、請求履歴を確認できます。 請求書は、決済情報に入力した請求書受信メールアドレスからも確認できます。

<ImgLang img='usage-estimate-history.png' desc='請求及び納付履歴' />

一覧で一番右の**_請求書_**コラムで**_詳細_**ボタンをクリックすると詳細を確認することが。

:::note

WhaTapモニタリングサービスは時間単位で課金します。 毎月の基準時間は、当該月の日数✕24時間で計算します。 以下は、1 か月の時間を計算したものです。

-   28日：28✕24 = 672時間
-   29日：29✕24 = 696時間
-   30日：30✕24 = 720時間
-   31日：31✕24 = 744時間

:::

## 使用量

ホーム画面 > **_利用内訳_** > **_使用量_**

プロジェクト別の使用量を確認できます。 プロジェクト別の月別詳細使用量を確認するには、左側のプロジェクト一覧でプロジェクトを選択してください。

<ImgLang img='billing-usage.png' desc='사용량'/>

1.  ![数字1](/img/number-01.png)から使用量を確認したい期間を選択してください。

2.  画面左側の![数字2](/img/number-02.png)プロジェクト一覧でプロジェクトを選択してください。 画面右上のテキスト入力欄を通じて、ご希望のプロジェクトを検索することができます。

3.  ![数字3](/img/number-03.png)領域で選択したプロジェクトの月間使用量に関する要約情報を確認できます。

4.  日別の詳細使用量を確認するには、![数字4](/img/number-04.png)日別使用量一覧で![もっと見るアイコン](/img/ico-expander.svg)ボタンを選択してください。

<ImgLang img='billing-usage-details.png' desc='사용량 상세'/>

:::note

-   月間使用量の要約情報および詳細使用量情報は、プロジェクトの製品種類によって異なる場合があります。

-   使用量情報は、課金方式に基づいて測定されます。 課金方式の詳細については、[次のリンク](https://www.whatap.io/ko/pricing/)を参考にしてください。

プロモーションコードを登録するには、決済情報を登録してください 決済情報登録の詳細については、[次の文書](#register-pay)を参照してください。

## WhaTapモニタリングサービスの利用料金の計算方法

WhaTapサービスは後払い決済方法を導入しています。 毎月5営業日以内に前月利用金額の請求書を**決済**権限を持つアカウントのメールで受け取ることができます。 前月の利用分は1日から末日まで利用した内容です。 請求書は、アカウントに登録した有料サービスのリストに基づいてアカウントごとに作成します。

もし有料サービスを月の途中から加入した場合、利用した期間を日割り計算して請求します。

## 利用料金の決済方法

毎月5営業日以内に前月利用料金請求書を受け取ることができます。 この請求書に基づき、[WhaTapモニタリングサービス](https://service.whatap.io)ページで次のような方法で料金を支払うことができます。

1.  画面左側のメニューから、**_利用内訳_** > **_請求及び納付履歴_**を選択してください。 納付状況と明細書を確認することができます。

2.  未納状態の場合、**_納付_**ボタンをクリックすると決済ウィンドウが表示されます。 たとえば、1月の使用分を2月25日までに決済しない場合、以降は未納状態になります。

3.  決済方法は、クレジットカードとリアルタイム口座振替のどちらかを選択できます。

4.  **_決済_**ボタンをクリックすると、決済方式別の認証手続きを経た後、納付を完了することができます。

:::note

WhaTapは現在、CMS自動振替をサポートしていません。

:::
