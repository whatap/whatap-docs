## 統合グループ管理

ホーム画面 > 組織選択 > **_管理_** > **_統合グループ管理_**

選択した組織のグループ一覧を確認できます。組織の修正権限がない場合、その機能は非活性化されます。

-   組織を選択していない場合は、**組織に属していない**グループとプロジェクトの一覧が表示されます。グループまたはプロジェクトを選択して、他の組織またはグループに移動できます。
-   グループまたはプロジェクトの修正権限がない場合、一覧は非活性化されます。

:::note

-   所有していない組織やグループへとグループやプロジェクトを移動するには、その組織やグループに招待するか、WhaTapサポートチーム（help@whatap.io）に問い合わせてください。
:::

### グループを追加する

![統合グループ管理](/img/manage-integrated-group.png)

1.  **_統合グループ管理_**メニューで**_＋グループ追加_**を選択してください。

2.  **_グループ追加_**ウィンドウが表示されたら、**_グループ名_**、**_説明_**を順番に入力してください。グループ名は必須入力です。

3.  **_プロジェクト追加_**項目でグループに追加するプロジェクトを選択してください。

4.  **_適用_**ボタンを選択します。

選択したプロジェクトをグループに追加します。

:::note
**_プロジェクト追加_**項目でテキストを入力し、グループに追加するプロジェクトをフィルタリングすることができます。 さらに、複数のプロジェクトを選択できます。
:::

### グループを移動する（組織からグループを削除）

![グループ移動](/img/manage-group-move-group.png)

選択したグループを別の組織に移動したり、組織から削除したりできます。

1.  別の組織に移動するグループを選択してください。複数のグループを選択できます。

2.  **_グループ移動_**ボタンが活性化されたら**_グループ移動_**ボタンを選択してください。

3.  **_グループ移動_**ウィンドウが表示されたら、**_組織選択_**項目をクリックして移動しようとする組織を選択してください。

4.  **_確認_**ボタンを選択してください。

:::note

-   修正権限のないグループは、**_グループ選択_**項目の一覧に表示されません。
-   組織からグループを削除するには、**_組織選択_**項目で**_組織に属していないグループ_**を選択してください。:::

### プロジェクトを移動する（グループからプロジェクトを削除）

![プロジェクト移動](/img/move-project-from-group.png)

選択したプロジェクトを他のグループに移動したり、グループから削除したりできます。

1.  他のグループに移動するプロジェクトを選択してください。複数のプロジェクトを選択できます。

2.  **_プロジェクトの移動_**ボタンが活性化されたら、**_プロジェクト移動_**ボタンを選択してください。

3.  **_プロジェクト移動_**ウィンドウが表示されたら、**_グループ選択_**項目をクリックして移動しようとするグループを選択してください。

4.  **_確認_**ボタンを選択してください。

:::note

-   修正権限のないグループは、**_グループ選択_**項目の一覧に表示されません。
-   グループからプロジェクトを削除するには、**_グループ選択_**項目で**_グループに属していないプロジェクト_**を選択してください。
:::

### グループを修正する

1.  グループ名と説明を修正したいグループにマウスオーバーすると、[修正アイコン](/img/ico-edit.svg)ボタンが表示されます。

2.  ![修正アイコン](/img/ico-edit.svg)ボタンを選択してください。 

3.  **_グループ修正_**ウィンドウが表示され、修正を完了した後、**_適用_**ボタンを選択してください。

### グループまたはプロジェクトを削除する

1.  削除したいグループまたはプロジェクトでマウスオーバーすると、![削除アイコン](/img/ico-trash.svg)ボタンが表示されます。

2.  ![削除アイコン](/img/ico-trash.svg)ボタンを選択してください。

3.  **_グループ削除_**または**_プロジェクト削除_**ウィンドウが表示されたら、メッセージを確認して**_確認_**ボタンを選択してください。

:::caution

-   グループを削除する場合は、グループとユーザー一覧情報の両方が削除されます。グループに属していたプロジェクトは、グループに属していないプロジェクトに移動します。
-   プロジェクトを削除する場合は、ユーザー一覧情報とこれまで収集したデータの両方が削除されます。削除したデータは再復旧できません。
:::

## 統合メンバー管理{#member}

ホーム画面 > 組織選択 > **_管理_** > **_統合メンバー管理_**

![統合メンバー管理](/img/integrated-manage-member.png)

選択した組織、グループ、プロジェクト別のメンバー一覧を確認・追加・削除できます。組織の修正権限がない場合、その機能は非活性化されます。

-   組織を選択していない場合は、**組織に属していない**グループとプロジェクトの一覧が表示されます。グループまたはプロジェクトを選択して、他の組織またはグループに移動できます。
-   グループまたはプロジェクトの修正権限がない場合、メンバーを追加・削除することはできません。

:::note

所有していない組織やグループのメンバーを確認するには、その組織やグループに招待するか、WhaTapサポートチーム（help@whatap.io）に問い合わせてください。

:::

### メンバーを招待する

1.  **_統合メンバー管理_**メニューで組織またはグループ、プロジェクトを選択してください。

2.  画面右上の**_メンバー招待_**ボタンを選択してください。

3.  **_ユーザー権限_**項目で招待するユーザーの権限を設定した後、**_適用_**ボタンを選択してください。

4.  **_メール_**項目で招待するユーザーのメールを入力し、**_追加_**ボタンを選択してください。他のユーザーをさらに追加することができます。

5.  招待しようとするユーザーをすべて追加したら、**_完了_**ボタンを選択してください。

:::note
招待を完了する前にキャンセルしたい招待メンバーがいる場合は、**_招待メンバー一覧_**で! [削除アイコン](/img/ico-trash-red.svg)ボタンを選択してください。
:::

### メンバーの権限を修正する

1.  **_統合メンバー管理_**メニューで組織またはグループ、プロジェクトを選択してください。

2.  メンバー一覧で権限を修正したいメンバーの右側の![編集アイコン](/img/ico-edit.svg)ボタンを選択してください。 

3.  **_メンバー権限の修正_**ウィンドウが表示されたら、**_メンバー権限_**と**_組織のグループ別のユーザー権限_**または**_グループのプロジェクト別のユーザー権限_**を設定してください。

    -   **_メンバー権限_**：該当組織と組織に属するグループが許可する権限を設定できます。
    -   組織を選択した場合）**_組織のグループ別のユーザー権限_**：選択したメンバーの権限を組織に属するグループ別に設定することができます。
    -   グループを選択した場合）**_グループのプロジェクト別のユーザー権限_**：選択したメンバーの権限をグループに属するプロジェクト別に設定することができます。

4.  すべての設定が完了したら、**_適用_**ボタンを選択してください。

:::note

-   選択した組織、グループまたはプロジェクトによって適用できる権限の範囲は異なる場合があります。
-   メンバー権限の詳細については、[次の文書](project-structure#project-auth)を参考にしてください。
:::

### メンバーを削除する

1.  **_統合メンバー管理_**メニューで組織またはグループ、プロジェクトを選択してください。

2.  メンバー一覧で削除しようとするメンバーの右側の![削除アイコン](/img/ico-trash-red.svg)ボタンを選択してください。 

3.  メッセージウィンドウが表示されたら、**_削除_**ボタンを選択してください。

:::note

-   組織を選択した場合、下位グループおよびプロジェクトでもメンバーを削除するには、メッセージウィンドウでチェックボックスをクリックし、**削除**ボタンを選択してください。
-   グループを選択した場合、下位プロジェクトでもメンバーを削除するには、メッセージウィンドウでチェックボックスをクリックし、**_削除_**ボタンを選択してください。
:::
