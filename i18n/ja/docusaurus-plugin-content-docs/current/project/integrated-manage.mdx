## 統合グループ管理

ホーム画面 > 組織選択 > **_管理_** > **_統合グループ管理_**

選択した組織のグループ一覧を表示できます。組織の修正権限がない場合、その機能は無効になります。

-   組織を選択していない場合は、**組織以外の**グループとプロジェクトの一覧が表示されます。グループまたはプロジェクトを選択して、他の組織またはグループに移動できます。
-   グループまたはプロジェクトの修正権限がない場合、一覧は無効になります。

:::note

-   所有していない組織やグループに、グループまたはプロジェクトを移動するには、その組織やグループに招待するか、WhaTapサポートチーム(help@whatap.io)にお問い合わせください。

:::

### グループを追加する

<ImgLang img='manage-integrated-group.png' desc='統合グループ管理' />

1.  **_統合グループ管理_**メニューから**_＋グループ追加_**を選択してください。

2.  **_グループ追加_**ウィンドウが表示されたら、**_グループ名_**、**_説明_**を順番に入力してください。グループ名は必須入力です。

3.  **_プロジェクト追加_**項目でグループに追加するプロジェクトを選択してください。

4.  **_適用_**ボタンを選択します。

選択したプロジェクトをグループに追加します。

:::note

**_プロジェクト追加_**項目でテキストを入力し、グループに追加するプロジェクトをフィルタリングすることができます。 さらに、複数のプロジェクトを選択できます。

:::

### グループを移動する(組織からグループを削除)

<ImgLang img='manage-group-move-group.png' desc='グループ移動' />

選択したグループを別の組織に移動したり、組織から削除したりできます。

1.  別の組織に移動するグループを選択してください。複数のグループを選択できます。

2.  **_グループ移動_**ボタンが有効になっている場合は、**_グループ移動_**ボタンを選択してください。

3.  **_グループ移動_**ウィンドウが表示されたら、**_組織選択_**項目をクリックして移動する組織を選択してください。

4.  **_確認_**ボタンを選択してください。

:::note

-   修正権限のないグループは、**_グループ選択_**項目の一覧に表示されません。
-   組織からグループを削除するには、**_組織選択_**項目で**_組織に属していないグループ_**を選択してください。

:::

### プロジェクトを移動する(グループからプロジェクトを削除)

<ImgLang img='move-project-from-group.png' desc='プロジェクトの移動' />

選択したプロジェクトを他のグループに移動したり、グループから削除したりできます。

1.  他のグループに移動するプロジェクトを選択してください。複数のプロジェクトを選択できます。

2.  **_プロジェクトの移動_**ボタンが有効になっている場合、**_プロジェクト移動_**ボタンを選択してください。

3.  **_プロジェクト移動_**ウィンドウが表示されたら、**_グループ選択_**項目をクリックして移動するグループを選択してください。

4.  **_確認_**ボタンを選択してください。

:::note

-   修正権限のないグループは、**_グループ選択_**項目の一覧に表示されません。
-   グループからプロジェクトを削除するには、**_グループ選択_**項目で**_グループに属していないプロジェクト_**を選択してください。

:::

### グループを修正する

1.  グループ名と説明を修正したいグループにカーソルを合わせると、[修正アイコン](/img/ico-edit.svg)ボタンが表示されます。

2.  ![修正アイコン](/img/ico-edit.svg)ボタンを選択してください。 

3.  **_グループ修正_**ウィンドウが表示され、修正を完了した後、**_適用_**ボタンを選択してください。

### グループまたはプロジェクトを削除する

1.  削除するグループまたはプロジェクトにカーソルを合わせると、![削除アイコン](/img/ico-trash.svg)ボタンが表示されます。

2.  ![削除アイコン](/img/ico-trash.svg)ボタンを選択してください。

3.  **_グループ削除_**または**_プロジェクト削除_**ウィンドウが表示すると、メッセージを確認して**_確認_**ボタンを選択してください。

:::caution

-   グループを削除する場合は、グループとユーザー一覧情報の両方が削除されます。グループに属していたプロジェクトは、グループに属していないプロジェクトに移動します。
-   プロジェクトを削除する場合は、ユーザー一覧情報と収集したデータが削除されます。削除したデータは復元できません。

:::

## 統合メンバー管理{#member}

ホーム画面 > 組織選択 > **_管理_** > **_統合メンバー管理_**

<ImgLang img='integrated-manage-member.png' desc='統合メンバー管理' />

選択した組織、グループ、プロジェクト別のメンバー一覧を確認、追加または削除できます。組織に修正権限がない場合、その機能は無効になります。

-   組織を選択していない場合は、**組織以外の**グループとプロジェクトの一覧が表示されます。グループまたはプロジェクトを選択して、他の組織またはグループに移動できます。
-   グループまたはプロジェクトの修正権限がない場合、メンバーを追加または削除することはできません。

:::note

所有していない組織やグループのメンバーを確認するには、その組織やグループに招待するか、WhaTapサポートチーム(help@whatap.io)にお問い合わせください。

:::

### メンバーを招待する

1.  **_統合メンバー管理_**メニューから組織またはグループ、プロジェクトを選択してください。

2.  画面右上の**_メンバー招待_**ボタンを選択してください。

3.  **_ユーザー権限_**項目で招待するユーザーの権限を設定した後、**_適用_**ボタンを選択してください。

4.  **_電子メール_**項目で招待するユーザーの電子メールを入力し、**_追加_**ボタンを選択してください。他のユーザーを追加することができます。

5.  招待するユーザーをすべて追加して、**_完了_**ボタンを選択してください。

:::note

招待を完了する前にキャンセルする招待メンバーがいる場合は、**_招待メンバー一覧_**で![削除アイコン](/img/ico-trash-red.svg)ボタンを選択してください。

:::

### メンバーの権限を修正する

1.  **_統合メンバー管理_**メニューから組織またはグループ、プロジェクトを選択してください。

2.  メンバー一覧で権限を修正したいメンバーの右側の![編集アイコン](/img/ico-edit.svg)ボタンを選択してください。 

3.  **_メンバー権限の修正_**ウィンドウが表示されると、**_メンバー権限_**と**_組織のグループ別のユーザー権限_**または**_グループのプロジェクト別のユーザー権限_**を設定してください。

    -   **_メンバー権限_**：組織と組織に所属するグループが許可する権限を設定できます。
    -   組織を選択した場合) **_組織のグループ別のユーザー権限_**：選択したメンバーの権限を組織に属するグループ別に設定することができます。
    -   グループを選択した場合) **_グループのプロジェクト別のユーザー権限_**：選択したメンバーの権限をグループに属するプロジェクト別に設定することができます。

4.  すべての設定が完了したら、**_適用_**ボタンを選択してください。

:::note

-   選択した組織、グループまたはプロジェクトによって適用できる権限の範囲は異なる場合があります。
-   メンバー権限の詳細については、[次の文書](project-structure#project-auth)を確認してください。

:::

### メンバーを削除する

1.  **_統合メンバー管理_**メニューから組織またはグループ、プロジェクトを選択してください。

2.  メンバー一覧から削除するメンバーの右側の![削除アイコン](/img/ico-trash-red.svg)ボタンを選択してください。 

3.  メッセージウィンドウが表示されたら、**_削除_**ボタンを選択してください。

:::note

-   組織を選択した場合、下位グループおよびプロジェクトでもメンバーを削除するには、メッセージウィンドウのチェックボックスをクリックし、**削除**ボタンを選択してください。
-   グループを選択した場合、下位プロジェクトからメンバーを削除する場合は、メッセージウィンドウのチェックボックスをクリックし、**_削除_**ボタンを選択してください。

:::
