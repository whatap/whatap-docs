## 統合グループ管理

ホーム画面 &gt; 組織選択 &gt; <Cmdname sid="side_management" className="uitext" /> &gt; <Cmdname sid="side_integratedGroupManagement" className="uitext" />

選択した組織のグループ一覧を表示できます。組織の修正権限がない場合、その機能は無効になります。

- 組織を選択していない場合は、**組織以外の**グループとプロジェクトの一覧が表示されます。グループまたはプロジェクトを選択して、他の組織またはグループに移動できます。
- グループまたはプロジェクトの修正権限がない場合、一覧は無効になります。

:::note

- 所有していない組織やグループに、グループまたはプロジェクトを移動するには、その組織やグループに招待するか、WhaTapサポートチーム(help@whatap.io)にお問い合わせください。

:::

### グループを追加する

<ImgLang img='manage-integrated-group.png' desc='統合グループ管理' />

1. <Cmdname sid="side_integratedGroupManagement" className="uitext" />メニューで![追加アイコン](/img/ico-add.svg)<Cmdname sid="add_group" className="uitext" />を選択します。

2. <Cmdname sid="add_group" className="uitext" />画面が表示されたら、<Cmdname sid="group_name" className="uitext" />、<Cmdname sid="description" className="uitext" />を入力します。グループ名は必須項目です。

3. <Cmdname sid="add_project" className="uitext" />項目でグループに追加するプロジェクトを選択します。

4. <Cmdname sid="apply" className="uitext" />ボタンを選択します。

選択したプロジェクトをグループに追加します。

:::note

<Cmdname sid="add_project" className="uitext" />項目でテキストを入力し、グループに追加するプロジェクトをフィルタリングできます。複数のプロジェクトを選択できます。

:::

### グループを移動する(組織からグループを削除)

<ImgLang img='manage-group-move-group.png' desc='グループ移動' />

選択したグループを別の組織に移動したり、組織から削除したりできます。

1. 別の組織に移動するグループを選択します。複数のグループを選択できます。

2. <Cmdname sid="TLT07929" className="uitext" />ボタンが有効になったら、ボタンを選択します。

3. <Cmdname sid="TLT07929" className="uitext" />画面が表示されたら、<Cmdname sid="select_organization" className="uitext" />項目をクリックして移動する組織を選択します。

4. <Cmdname sid="confirm" className="uitext" />ボタンを選択します。

:::note

組織からグループを除外するには、<Cmdname sid="select_organization" className="uitext" />項目で<Cmdname sid="groups_not_belong_to_organization" className="uitext" />を選択します。

:::

### プロジェクトを移動する(グループからプロジェクトを削除)

<ImgLang img='move-project-from-group.png' desc='プロジェクトの移動' />

選択したプロジェクトを他のグループに移動したり、グループから削除したりできます。

1. 他のグループに移動するプロジェクトを選択します。複数のプロジェクトを選択できます。

2. <Cmdname sid="BTN08282" type="project" replace="({count})" className="uitext" />ボタンが有効になったら、ボタンを選択します。

3. <Cmdname sid="group_management_move" type="project" className="uitext" />画面が表示されたら、<Cmdname sid="select_group" className="uitext" />項目をクリックして移動するグループを選択します。

4. <Cmdname sid="confirm" className="uitext" />ボタンを選択します。

:::note

- <Cmdname sid="edit" className="b600" />権限のないグループは、<Cmdname sid="select_group" className="uitext" />項目の一覧に表示されません。

- グループからプロジェクトを削除するには、<Cmdname sid="select_group" className="uitext" />項目から<Cmdname sid="no_group_projects" className="uitext" />を選択します。

:::

### グループを修正する

1. グループ名と説明を修正したいグループにカーソルを合わせると、[修正アイコン](/img/ico-edit.svg)ボタンが表示されます。

2. ![修正アイコン](/img/ico-edit.svg)ボタンを選択します。 

3. <Cmdname sid="edit_group" className="uitext" />画面が表示され、修正を完了した後、<Cmdname sid="apply" className="uitext" />ボタンを選択します。

### グループまたはプロジェクトを削除する

1. 削除するグループまたはプロジェクトにカーソルを合わせると、![削除アイコン](/img/ico-trash.svg)ボタンが表示されます。

2. ![削除アイコン](/img/ico-trash.svg)ボタンを選択します。

3. <Cmdname sid="group_management_delete" type="group" className="uitext" />または<Cmdname sid="group_management_delete" type="project" className="uitext" />画面が表示されたら、メッセージを確認して<Cmdname sid="confirm" className="uitext" />ボタンを選択します。

:::caution

- グループを削除する場合は、グループとユーザー一覧情報の両方が削除されます。グループに属していたプロジェクトは、グループに属していないプロジェクトに移動します。
- プロジェクトを削除する場合は、ユーザー一覧情報と収集したデータが削除されます。削除したデータは復元できません。

:::

## 統合メンバー管理{#member}

ホーム画面 &gt; 組織選択 &gt; <Cmdname sid="side_management" className="uitext" /> &gt; <Cmdname sid="side_integratedMemberManagement" className="uitext" />

<ImgLang img='integrated-manage-member.png' desc='統合メンバー管理' />

選択した組織、グループ、プロジェクト別のメンバー一覧を確認、追加または削除できます。組織に修正権限がない場合、その機能は無効になります。

- 組織を選択していない場合は、**組織以外の**グループとプロジェクトの一覧が表示されます。グループまたはプロジェクトを選択して、他の組織またはグループに移動できます。
- グループまたはプロジェクトの修正権限がない場合、メンバーを追加または削除することはできません。

:::note

所有していない組織やグループのメンバーを確認するには、その組織やグループに招待するか、WhaTapサポートチーム(help@whatap.io)にお問い合わせください。

:::

### メンバーを招待する

1. <Cmdname sid="side_integratedMemberManagement" className="uitext" />メニューから組織またはグループ、プロジェクトを選択します。

2. 画面の右上にある<Cmdname sid="invite_member" className="uitext" />ボタンを選択します。

3. <Cmdname sid="user_role" className="uitext" />項目から招待するユーザーの権限を設定した後、<Cmdname sid="apply" className="uitext" />ボタンを選択します。

4. メール入力欄に招待するユーザーのメールアドレスを入力した後、<Cmdname sid="add" className="uitext" />ボタンを選択します。他のユーザーをさらに追加できます。

5. 招待するユーザーをすべて追加して、<Cmdname sid="complete" className="uitext" />ボタンを選択します。

### メンバーの権限を修正する

1. <Cmdname sid="side_integratedMemberManagement" className="uitext" />メニューから組織またはグループ、プロジェクトを選択します。

2. メンバー一覧で権限を修正するメンバーの右側にある![編集アイコン](/img/ico-edit.svg)ボタンを選択します。 

3. <Cmdname sid="edit_member_role" className="uitext" />画面が表示されると、<Cmdname sid="member_role" className="uitext" />と<Cmdname sid="user_role_per_group_of_organization" className="uitext" />または<Cmdname sid="user_role_per_project_of_group" className="uitext" />を設定します。

   - <Cmdname sid="edit_member_role" className="uitext" />：組織と組織に所属するグループが許可する権限を設定できます。

   - 組織を選択した場合) <Cmdname sid="user_role_per_group_of_organization" className="uitext" />：選択したメンバーの権限を組織に属するグループごとに設定できます。

   - グループを選択した場合) <Cmdname sid="user_role_per_project_of_group" className="uitext" />：選択したメンバーの権限をグループに属するプロジェクトごとに設定できます。

4. すべての設定が完了したら、<Cmdname sid="apply" className="uitext" />ボタンを選択します。

:::note

- 選択した組織、グループまたはプロジェクトにより適用可能な権限の範囲が異なります。

- メンバー権限の修正機能は<Cmdname sid="member_management" className="b600" />権限のあるユーザーのみ利用できます。メンバー権限の詳細は[次の文書](project-structure#project-auth)を参照してください。

:::

### メンバーを削除する

1. <Cmdname sid="side_integratedMemberManagement" className="uitext" />メニューから組織またはグループ、プロジェクトを選択します。

2. メンバー一覧から削除するメンバーの右側にある![削除アイコン](/img/ico-trash-red.svg)ボタンを選択します。 

3. メッセージが表示されたら、<Cmdname sid="delete" className="uitext" />ボタンを選択します。

:::note

- 組織を選択した場合、下位グループや下位プロジェクトからもメンバーを削除するには、メッセージ画面からチェックボックスをクリックし、<Cmdname sid="delete" className="uitext" />ボタンを選択します。

- グループを選択した場合、下位プロジェクトからもメンバーを削除するには、メッセージ画面からチェックボックスをクリックし、<Cmdname sid="delete" className="uitext" />ボタンを選択します。

- <Cmdname sid="member_management" className="b600" />権限がない場合、他のメンバーを削除できません。メンバー権限の詳細は[次の文書](project-structure#project-auth)を参照してください。

:::

## メンバーリストをダウンロードする{#downlaodmemberlist}

メンバーリストをCSV形式のファイルにダウンロードする場合、次の手順に従います。

1. <Cmdname sid="side_integratedMemberManagement" className="uitext" />メニューに移動します。

2. 画面左側の一覧から組織またはグループ、プロジェクトを選択します。

3. 画面右上の![CSV icon](/img/ic-csv-download-ms.svg)ボタンを選択します。

メンバーリストをCSVファイル形式でユーザーのローカルディスクに保存します。

:::note

メンバーリストのダウンロード機能は<Cmdname sid="member_management" className="b600" />権限のあるユーザーのみ利用できます。メンバー権限の詳細は[次の文書](project-structure#project-auth)を参照してください。

:::
