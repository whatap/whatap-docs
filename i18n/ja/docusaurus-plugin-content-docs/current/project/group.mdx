プロジェクトをグループ化して管理できます。ユーザーは、複数のプロジェクトを所有できます。複数のプロジェクトを管理するためにグループという概念を提供します。2つ以上のプロジェクトがある場合、プロジェクトごとにユーザーを追加したり、権限を付与したりするのは面倒な作業になります。この場合、2つのプロジェクトを1つのグループにまとめ、グループとしてユーザーを管理できます。

:::note

-   1つのプロジェクトは、1つのグループに属します。
-   プロジェクトは、グループに属していない可能性があります。
:::

プロジェクトユーザーはグループ ユーザーを継承し、グループに属するユーザーはグループ内のすべてのプロジェクトをモニタリングできます。特定プロジェクトに限って権限が付与されたユーザーなら、グループの有無に関係なく、そのプロジェクトだけをモニタリングできます。

## グループを追加する

1.  [WhaTapモニタリングサービス]（https://service.whatap.io）画面左上の**_＋グループ_**ボタンを選択してください。

2.  作成するグループの名前を**_グループ名_**に入力してください。

3.  グループに関する説明を**_グループ説明_**に入力してください。**_グループ説明_**は必須入力ではありません。

4.  **_プロジェクト追加_**一覧でグループに追加するプロジェクトを選択してください。プロジェクトを検索して選択することもできます。

5.  プロジェクトの追加が完了したら、**_グループ追加_**ボタンをクリックしてください。

:::note
グループに追加できるプロジェクトの一覧は、修正権限があるプロジェクトまたはグループに属していないプロジェクトです。
:::

## グループにプロジェクトを追加する

1.  ホーム画面の**_プロジェクト一覧_**のプロジェクト項目右上に![設定アイコン]（/img/ico-setting.svg）ボタンを選択してください。 

  ![プロジェクト設定]（/img/project-add-group.png）

1.  **_グループに追加_**を選択してください。

2.  プロジェクトを追加するグループを選択してください。

3.  **_グループに追加_**ボタンを選択してください。

## 選択したグループのプロジェクト一覧を見る

ホーム画面で**_全体プロジェクト_**メニューの下位グループの中で一つを選択してください。選択したグループに含まれるプロジェクトが一覧に表示されます。

![プロジェクトグループを選択する]（/img/project-select-group.png）

再度プロジェクト一覧を確認するには、**_全体プロジェクト_**を選択してください。グループに属していないプロジェクトを確認するには、全体プロジェクトの領域で下にスクロールしてください。**_グループに属していないプロジェクト_**の一覧を確認することができます。

## グループからプロジェクトを削除する

1.  ホーム画面の**_プロジェクト一覧_**のプロジェクト項目右上に![設定アイコン]（/img/ico-setting.svg）ボタンを選択してください。 

  ![グループから削除]（/img/remove-project-from-group.png）

1.  **_グループから削除_**を選択してください。

2.  **_グループからプロジェクトを削除_**ウィンドウが表示されたら、**_確認_**ボタンをクリックしてください。

## グループを削除する

1.  ホーム画面で**_全体プロジェクト_**メニューの下位グループの中から一つを選択してください。

2.  **_グループ管理_**を選択してください。

3.  **_グループ情報_**セクションの右上の**_グループ削除_**を選択してください。

4.  **_グループ削除_**メッセージウィンドウの内容を確認した後、**_削除_**ボタンを選択してください。

:::caution

-   グループを削除すると、ユーザー一覧の情報も削除されます。 
-   グループに含まれるプロジェクトは、グループ情報のみが削除されます。 
-   削除した内容は復旧できません。
:::

## グループメンバーを招待する{#add-member}

1.  ホーム画面で**_全体プロジェクト_**メニューの下位グループの中から一つを選択してください。

2.  画面左側のメニューで**_グループメニュー_** >**_グループメンバー_**を選択してください。

3.  画面右上の**_メンバー招待_**ボタンを選択してください。

4.  **_グループ_**項目で下位プロジェクト単位を選択してメンバーを追加することもできます。

5.  **_ユーザー権限_**項目で招待するユーザーの権限を設定した後、**_適用_**ボタンを選択してください。

6.  **_メール_**項目で招待するユーザーのメールを入力し、**_追加_**ボタンを選択してください。他のユーザーをさらに追加することができます。

7.  招待しようとするユーザーをすべて追加したら、**_完了_**ボタンを選択してください。

:::note
招待を完了する前にキャンセルしたい招待メンバーがいる場合は、**_招待メンバー一覧_**で! [削除アイコン]（/img/ico-trash-red.svg）ボタンを選択してください。
:::

## プロジェクト別のメンバーを確認する

![プロジェクト別のメンバー確認]（/img/group-memer-by-project.png）

ホーム画面で**_全体プロジェクト_**メニューの下位グループの中から一つを選択してください。画面左側のメニューで**_グループメニュー_** >**_グループメンバー_**を選択してください。グループ下位のプロジェクトを選択すると、プロジェクトに属しているメンバーを確認できます。

## グループメンバーの権限を修正する

1.  ホーム画面で**_全体プロジェクト_**メニューの下位グループの中から一つを選択してください。

2.  画面左側で**_グループメニュー_** >**_グループメンバー_**を選択してください。

3.  **_グループメンバー_**一覧で権限を修正するメンバーの右側の![編集アイコン]（/img/ico-edit.svg）ボタンを選択してください。 

4.  ***メンバー権限の修正***ウィンドウが表示されたら、***メンバー権限***と***グループのプロジェクト別のユーザー権限***を設定してください。

    -   **_メンバー権限_**：当該グループとグループに属するプロジェクトで許可する権限を設定できます。
    -   **_グループのプロジェクト別ユーザー権限_**：選択したメンバーの権限をグループに属するプロジェクトごとに設定できます。

5.  すべての設定が完了したら、**_適用_**ボタンを選択してください。

:::note
メンバー権限の詳細については、[次の文書]（project-structure#project-auth）を参考にしてください。
:::

## グループメンバーを削除する

1.  ホーム画面で**_全体プロジェクト_**メニューの下位グループの中から一つを選択してください。

2.  画面左側のメニューで**_グループメニュー_** >**_グループメンバー_**を選択してください。

3.  **_グループメンバー_**一覧で削除するメンバーの右側の![削除アイコン]（/img/ico-trash-red.svg）ボタンを選択してください。

4.  メッセージウィンドウが表示されたら、**_削除_**ボタンを選択してください。

:::note
下位プロジェクトでもメンバーを削除するには、メッセージウィンドウでチェックボックスをクリックし、**削除**ボタンを選択してください。
:::
