プロジェクトをグループ化して管理できます。ユーザーは、複数のプロジェクトを所有できます。複数のプロジェクトを管理するためにグループという概念を提供します。2つ以上のプロジェクトがある場合、プロジェクトごとにユーザーを追加したり、権限を付与したりするのは面倒な場合があります。この場合、2つのプロジェクトを1つのグループにまとめ、グループとしてユーザーを管理できます。

:::note

- 1つのプロジェクトは、1つのグループのみに属します。
- プロジェクトは、グループに属していない可能性があります。

:::

プロジェクトユーザーはグループ ユーザーを継承し、グループに属するユーザーはグループ内のすべてのプロジェクトをモニタリングできます。特定プロジェクトに限って権限が付与されたユーザーなら、グループの有無に関係なく、そのプロジェクトのみをモニタリングできます。

## グループを追加する

1. サイドメニューで<Cmdname sid="BTN07332" className="uitext" /> &gt; ![追加アイコン](/img/ico-add.svg) <Cmdname sid="BTN06697" className="uitext" />ボタンを選択します。

   <ImgLang img="group-add.png" desc="グループ追加" />

2. <Cmdname sid="side_addGroup" className="uitext" />画面で作成するグループの名前を<Cmdname sid="group_name" className="uitext" />に入力します。

3. グループに関する説明を<Cmdname sid="group_description" className="uitext" />に入力します。<Cmdname sid="group_description" className="uitext" />は必須ではありません。

4. <Cmdname sid="add_project" className="uitext" />一覧からグループに追加するプロジェクトを選択します。プロジェクトを検索して選択することもできます。

5. プロジェクトの追加が完了したら、<Cmdname sid="add_group" className="uitext" />ボタンをクリックします。

:::note

グループに追加できるプロジェクトの一覧は、修正権限があるプロジェクトまたはグループに属していないプロジェクトです。

:::

## グループにプロジェクトを追加する

1. ホーム画面の<Cmdname sid="side_projectList" className="uitext" />のプロジェクト項目の右上に![設定アイコン](/img/ico-setting.svg)ボタンを選択します。 

   <ImgLang img='project-add-group.png' desc='プロジェクト設定' />

2. <Cmdname sid="add_to_group" className="uitext" />を選択します。

3. プロジェクトを追加するグループを選択します。

4. <Cmdname sid="add_to_group" className="uitext" />ボタンを選択します。

## 選択したグループのプロジェクトリストを表示する

<ImgLang img='project-select-group.png' desc='プロジェクトグループを選択する' />

1. サイドメニューの<Cmdname sid="BTN07332" className="uitext" />メニューを選択してグループとプロジェクトリストのパネルを開きます。

2. パネルで照会を希望するグループの右側に![右方向アイコン](/img/right-arrow.svg)ボタンを選択します。

   選択したグループに含まれるプロジェクトのみリスト画面に表示されます。この状態で<Cmdname sid="side_groupMember" className="uitext" />または<Cmdname sid="side_groupManagement" className="uitext" />のメニューを利用できます。

3. もう一度全体のプロジェクトリストを確認するには、サイドメニュー下段の![ホームアイコン](/img/ic-home.svg)<Cmdname sid="TLT07384" className="uitext" />ボタンを選択します。 

:::note

グループに属していないプロジェクトを確認するには、リストパネルを下にスクロールします。

:::

## グループからプロジェクトを除外する

### ホーム画面から除外する

1. ホーム画面の<Cmdname sid="side_projectList" className="uitext" />のプロジェクト項目の右上に![設定アイコン](/img/ico-setting.svg)ボタンを選択します。 

   <ImgLang img='remove-project-from-group.png' desc='グループから削除' />

2. <Cmdname sid="delete_from_group" className="uitext" />を選択します。

3. <Cmdname sid="delete_project_from_group" className="uitext" />画面が表示されたら、<Cmdname sid="ok" className="uitext" />ボタンをクリックします。

### グループ管理から除外する

グループに属しているプロジェクトを別のプロジェクトに移動または除外できます。

1. サイドメニューの<Cmdname sid="BTN07332" className="uitext" />メニューを選択してグループとプロジェクトリストのパネルを開きます。

2. パネルから削除するグループの右側に![右方向アイコン](/img/right-arrow.svg)ボタンを選択します。

3. サイドメニューで<Cmdname sid="side_groupManagement" className="uitext" />を選択します。

4. <Cmdname sid="group_management" className="uitext" />セクションから除外するプロジェクトを選択します。

   <ImgLang img="group-exclude-project.png" desc="グループ管理" />

5. <Cmdname sid="BTN08282" type="project" replace="({count})" className="uitext" />ボタンが表示されたら、ボタンを選択します。

6. <Cmdname sid="group_management_move" type="project" className="uitext" />画面が表示されたら<Cmdname sid="select_group" className="uitext" />項目で移動するグループを選択または<Cmdname sid="no_group_projects" className="uitext" />を選択します。

7. <Cmdname sid="confirm" className="uitext" />ボタンを選択します。

:::note

<Cmdname sid="select_group" className="uitext" />項目で<Cmdname sid="no_group_projects" className="uitext" />を選択すると、そのプロジェクトは所属するグループから除外されます。

:::

## グループを削除する

1. サイドメニューの<Cmdname sid="BTN07332" className="uitext" />メニューを選択してグループとプロジェクトリストのパネルを開きます。

2. パネルから削除するグループの右側に![右方向アイコン](/img/right-arrow.svg)ボタンを選択します。

3. サイドメニューで<Cmdname sid="side_groupManagement" className="uitext" />を選択します。

4. <Cmdname sid="group_info" className="uitext" />セクションで<Cmdname sid="delete_group" className="uitext" />を選択します。

   <ImgLang img="group-remove.png" desc="グループ削除" />

5. <Cmdname sid="delete_group" className="uitext" />メッセージ画面の内容を確認し、<Cmdname sid="delete" className="uitext" />ボタンを選択します。

:::caution

- グループを削除すると、ユーザー一覧の情報も削除されます。 
- グループに含まれるプロジェクトは、グループ情報のみ削除します。 
- 削除した内容は復元できません。

:::

## グループメンバーを招待する {#add-member}

1. サイドメニューの<Cmdname sid="BTN07332" className="uitext" />メニューを選択してグループとプロジェクトリストのパネルを開きます。

2. パネルからメンバーを招待するグループの右側に![右方向アイコン](/img/right-arrow.svg)ボタンを選択します。

3. サイドメニューで<Cmdname sid="side_groupMember" className="uitext" />を選択します。

4. 画面の右上にある<Cmdname sid="invite_member" className="uitext" />ボタンを選択します。

5. <Cmdname sid="group" className="uitext" />項目から下位プロジェクト単位を選択してメンバーを追加することもできます。

6. <Cmdname sid="user_role" className="uitext" />項目から招待するユーザーの権限を設定した後、<Cmdname sid="apply" className="uitext" />ボタンを選択します。

7. メール入力欄に招待するユーザーのメールアドレスを入力した後、<Cmdname sid="add" className="uitext" />ボタンを選択します。他のユーザーをさらに追加できます。

8. 招待するユーザーをすべて追加して、<Cmdname sid="complete" className="uitext" />ボタンを選択します。

## プロジェクト別のメンバーを確認する

<ImgLang img="group-member-by-project.png" desc="グループメンバー" />

1. サイドメニューで<Cmdname sid="BTN07332" className="uitext" />メニューを選択してグループとプロジェクトリストのパネルを開きます。 

2. パネルでメンバーリストを確認するグループの右側に![右方向アイコン](/img/right-arrow.svg)ボタンを選択します。

3. サイドメニューで<Cmdname sid="side_groupMember" className="uitext" />を選択します。

4. グループに所属するプロジェクトを選択すると、プロジェクトに所属するメンバーを確認できます。

## グループメンバーの権限を修正する

1. サイドメニューの<Cmdname sid="BTN07332" className="uitext" />メニューを選択してグループとプロジェクトリストのパネルを開きます。

2. パネルで権限を修正するメンバーを含むグループの右側に![右方向アイコン](/img/right-arrow.svg)ボタンを選択します。

3. サイドメニューで<Cmdname sid="side_groupMember" className="uitext" />を選択します。

4. <Cmdname sid="side_groupMember" className="uitext" />一覧で権限を変更するメンバーの右側の![編集アイコン](/img/ico-edit.svg)ボタンを選択します。

5. <Cmdname sid="edit_member_role" className="uitext" />画面が表示されたら、<Cmdname sid="member_role" className="uitext" />と<Cmdname sid="user_role_per_project_of_group" className="uitext" />を設定します。

   - <Cmdname sid="member_role" className="uitext" />：該当グループとグループに属するプロジェクトで許可する権限を設定できます。
   - <Cmdname sid="user_role_per_project_of_group" className="uitext" />：選択したメンバーの権限をグループに属するプロジェクトごとに設定できます。

6. すべての設定が完了したら、<Cmdname sid="apply" className="uitext" />ボタンを選択します。

:::note

メンバー権限の修正機能は<Cmdname sid="member_management" className="b600" />権限のあるユーザーのみ利用できます。メンバー権限の詳細は[次の文書](project-structure#project-auth)を参照してください。

:::

## グループメンバーを削除する

1. サイドメニューの<Cmdname sid="BTN07332" className="uitext" />メニューを選択してグループとプロジェクトリストのパネルを開きます。

2. パネルから削除するメンバーを含むグループの右側に![右方向アイコン](/img/right-arrow.svg)ボタンを選択します。

3. サイドメニューで<Cmdname sid="side_groupMember" className="uitext" />を選択します。

4. <Cmdname sid="side_groupMember" className="uitext" />一覧から削除するメンバーの右側にある![削除アイコン](/img/ico-trash-red.svg)ボタンを選択します。

5. メッセージが表示されたら、<Cmdname sid="delete" className="uitext" />ボタンを選択します。

:::note

- 下位プロジェクトからメンバーを削除する場合は、メッセージ画面でチェックボックスをクリックし、<Cmdname sid="delete" className="uitext" />ボタンを選択します。

- <Cmdname sid="member_management" className="b600" />権限がない場合、他のメンバーを削除できません。メンバー権限の詳細は[次の文書](project-structure#project-auth)を参照してください。

:::

## メンバーリストをダウンロードする{#downlaodmemberlist}

メンバーリストをCSV形式のファイルにダウンロードする場合、次の手順に従います。

1. サイドメニューの<Cmdname sid="BTN07332" className="uitext" />メニューを選択してグループとプロジェクトリストのパネルを開きます。

2. パネルから削除するメンバーを含むグループの右側に![右方向アイコン](/img/right-arrow.svg)ボタンを選択します。

3. サイドメニューで<Cmdname sid="side_groupMember" className="uitext" />を選択します。

4. 画面右上の![CSV icon](/img/ic-csv-download-ms.svg)ボタンを選択します。

メンバーリストをCSVファイル形式でユーザーのローカルディスクに保存します。

:::note

メンバーリストのダウンロード機能は<Cmdname sid="member_management" className="b600" />権限のあるユーザーのみ利用できます。メンバー権限の詳細は[次の文書](project-structure#project-auth)を参照してください。

:::
