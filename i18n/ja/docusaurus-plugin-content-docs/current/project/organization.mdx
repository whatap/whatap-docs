組織は、グループをグループ化できる親グループです。ほとんどの場合、プロジェクトとグループで十分です。運用管理サービスを専門とする企業であれば、複数の会社がグループとして存在します。グループやプロジェクトの数が非常に多くなる可能性があります。この場合、グループの概念だけでは不十分です。グループを統合管理できるコンセプトが必要です。組織の概念を使用します。

専門的な運用管理サービス企業は、複数のサブグループを1つの組織にグループ化することで管理できます。各グループメンバーに権限を付与してみてください。権限を付与されたメンバーは、個別にグループを管理できます。

## グループ生成

1. 画面右上のプロフィールアイコンを選択すると、ドロップダウンメニューが表示されます。

2. <Cmdname sid="create_organization" className="uitext" />を選択してください。

3. <Cmdname sid="organization_name" className="uitext" />と<Cmdname sid="site_logo" className="uitext" />、<Cmdname sid="description" className="uitext" />を入力して<Cmdname sid="next" className="uitext" />ボタンを選択してください。

4. グループ一覧から組織に追加するグループを選択し、<Cmdname sid="complete" className="uitext" />ボタンをクリックします。

問題なく組織の作成ができましたら、'<Cmdname sid="success_create_organization" className="uitext" />'メッセージが表示されます。

:::note

- <Cmdname sid="organization_name" className="uitext" />は必須入力です。<Cmdname sid="site_logo" className="uitext" />と<Cmdname sid="description" className="uitext" />を入力することなく組織を作成することができます。

- 修正権限を持つグループのみ、組織に追加できます。

- 修正権限がない場合はそのグループを選択できません。

- グループを選択する必要なく組織を作成できます。

- 修正権限のないグループに招待された場合、そのグループに対する修正権限はありません。

- 親グループから削除したり、他のグループに移動したりすることはできません。

- 組織<Cmdname sid="site_logo" className="uitext" />イメージの推奨サイズは、幅102ピクセル、高さ102ピクセルです。 

- 組織<Cmdname sid="site_logo" className="uitext" />画像は、3MB以下容量の_png_、_jpg_、_jpeg_ファイルの拡張子のみ可能です。

- メンバー権限の詳細については、[次の文書](project-structure#project-auth)を参照してください。

:::

## 組織に属しているグループやプロジェクトを確認する

ホーム画面の左上の<Cmdname sid="sidebar_org_selector_no_organization" className="uitext" />を選択してください。組織一覧が表示されたら目的の組織を選択してください。<Cmdname sid="BTN07332" className="uitext" />メニューから組織に属するグループを確認することができます。<Cmdname sid="side_projectList" className="uitext" />では、組織に属する全体グループのプロジェクトリストを確認することができます。

## 組織管理 {#manage}

1. ホーム画面で左上の<Cmdname sid="sidebar_org_selector_no_organization" className="uitext" />を選択します。

   <ImgLang img='project-organize-manage.png' desc='組織管理' />

2. 組織の一覧が表示されたら、管理したい組織名にマウスポインターを持っていきます。

3. ![設定アイコン](/img/ico-setting.svg)ボタンが表示されたら、ボタンを選択します。

<Cmdname sid="side_organizationManagement" className="uitext" />メニューに移動します。

<ImgLang img='project-organize-manage-group.png' desc='組織管理' />

- ![指示線1](/img/number-01.png) 組織名領域

  ユーザーが所属している組織の一覧を確認するには、![指示線 1](/img/number-01.png)の組織名領域を選択してください。<Cmdname sid="organization_list" className="uitext" />が表示されると別の組織を選択するか、<Cmdname sid="return_non_organization_list" className="uitext" />を選択してプロジェクトリストに戻ることができます。

- ![指示線2](/img/number-02.png) <Cmdname sid="organization_list" className="uitext" />

  - すべての組織がパネル一覧に表示されます。組織を変更する場合は、![指示線 a](/img/char-a.svg) <Cmdname sid="organization_management_change_to" className="uitext" />を選択します。

  - 組織名とロゴ、説明を編集するには、![編集アイコン](/img/ico-edit.svg)ボタンをクリックしてください。

  - 組織を削除するには、![削除アイコン](/img/ico-trash.svg)ボタンをクリックしてください。所属するグループとプロジェクトは、組織に所属していないグループに移動します。

  - 修正権限がない場合は、修正・削除ボタンが表示されません。

- ![指示線3](/img/number-03.png) <Cmdname sid="group_list" className="uitext" />

  現在選択されている組織に所属するすべてのグループが一覧として表示されます。![右方向アイコン](/img/right-arrow.svg)ボタンをクリックすると、選択したグループの管理画面に移動できます。

- ![指示線4](/img/number-04.png)組織修正

  選択している組織の基本情報を修正するには、[編集アイコン](/img/ico-edit.svg)ボタンを選択します。<Cmdname sid="edit_organization" className="uitext" />画面が表示されたら、組織名、ロゴ、説明などを修正します。修正を完了した後、<Cmdname sid="save" className="uitext" />ボタンを選択します。

## 組織メンバーを招待する

1. <Cmdname sid="side_organizationManagement" className="uitext" />メニューに移動します。

2. サイドメニューで<Cmdname sid="side_organizationMembers" className="uitext" />を選択します。

3. 画面右上の<Cmdname sid="invite_member" className="uitext" />ボタンを選択します。

4. <Cmdname sid="organization" className="uitext" />項目でサブグループやプロジェクト単位を選択してメンバーを追加することもできます。

5. <Cmdname sid="user_role" className="uitext" />項目で招待するユーザーの権限を設定した後、<Cmdname sid="apply" className="uitext" />ボタンを選択してください。

6. メール入力欄に招待するユーザーのメールアドレスを入力した後、<Cmdname sid="add" className="uitext" />ボタンを選択します。他のユーザーをさらに追加できます。

7. 招待するユーザーをすべて追加して、<Cmdname sid="complete" className="uitext" />ボタンを選択してください。

## 組織メンバーを確認する{#member-list}

<ImgLang img='organization-member.png' desc='組織のメンバーを特定する' />

組織メニューから<Cmdname sid="side_organizationMembers" className="uitext" />を選択します。組織のツリーでグループまたはプロジェクトを選択できます。選択したグループ、プロジェクトごとに所属するメンバーを確認できます。選択したグループ、プロジェクトごとにメンバーを追加・削除したり権限を修正できます。

### 組織メンバーの権限を修正する

1. <Cmdname sid="side_organizationManagement" className="uitext" />メニューに移動します。

2. サイドメニューで<Cmdname sid="side_organizationMembers" className="uitext" />を選択します。

3. <Cmdname sid="organization_member" className="uitext" />一覧で権限を変更するメンバーの右側の![編集アイコン](/img/ico-edit.svg)ボタンを選択してください。

4. <Cmdname sid="edit_member_role" className="uitext" />画面が表示されたら、<Cmdname sid="member_role" className="uitext" />と<Cmdname sid="user_role_per_group_of_organization" className="uitext" />を設定します。

   - <Cmdname sid="member_role" className="uitext" />：組織と組織に所属するグループが許可する権限を設定できます。
   - <Cmdname sid="user_role_per_group_of_organization" className="uitext" />：選択したメンバーの権限を組織に属するグループ別に設定することができます。

5. すべての設定が完了したら、<Cmdname sid="apply" className="uitext" />ボタンを選択してください。

:::note

- 組織ツリー構造でグループまたはプロジェクトを選択し、グループ別、プロジェクト別に所属するメンバーの権限を修正できます。選択したグループまたはプロジェクトによって適用できる権限の範囲が異なる場合があります。

- メンバー権限の修正機能は<Cmdname sid="member_management" className="b600" />権限のあるユーザーのみ利用できます。メンバー権限の詳細は[次の文書](project-structure#project-auth)を参照してください。

:::

## 組織メンバーを削除する

1. <Cmdname sid="side_organizationManagement" className="uitext" />メニューに移動します。

2. サイドメニューで<Cmdname sid="side_organizationMembers" className="uitext" />を選択します。

3. <Cmdname sid="organization_member" className="uitext" />一覧から削除するメンバーの右側の![削除アイコン](/img/ico-trash-red.svg)ボタンを選択してください。

4. メッセージが表示されたら、<Cmdname sid="delete" className="uitext" />ボタンを選択します。 

:::note

- 下位グループまたは下位プロジェクトでもメンバーを削除する場合は、メッセージ画面からチェックボックスをクリックし、<Cmdname sid="delete" className="uitext" />ボタンを選択します。

- 組織ツリー構造でグループまたはプロジェクトを選択し、グループ別、プロジェクト別に所属するメンバーを削除できます。

- <Cmdname sid="member_management" className="b600" />権限がない場合、他のメンバーを削除できません。メンバー権限の詳詳細[次の文書](project-structure#project-auth)を参照してください。

:::

## メンバーリストをダウンロードする{#downlaodmemberlist}

メンバーリストをCSV形式のファイルとしてダウンロードする場合、次のステップを実行します。

1. <Cmdname sid="side_organizationManagement" className="uitext" />メニューに移動します。

2. サイドメニューで<Cmdname sid="side_organizationMembers" className="uitext" />を選択します。

3. 画面右上の![CSV icon](/img/ic-csv-download-ms.svg)ボタンを選択します。

メンバーリストをCSVファイル形式でユーザーのローカルディスクに保存します。

:::note

メンバーリストのダウンロード機能は<Cmdname sid="member_management" className="b600" />権限のあるユーザーのみ利用できます。メンバー権限の詳細は[次の文書](project-structure#project-auth)を参照してください。

:::
