### アカウント設定

本格的なWhaTapモニタリングサービスを開始する前に、いくつかの考慮事項があります。

-   **プロジェクト**<br/>
    プロジェクトとは、WhaTapのモニタリング単位を意味します。エージェントをインストールするには、まずプロジェクトを作成する必要があります。プロジェクトの詳細については、[次の文書](../project)を参考にしてください。
-   **データサーバー地域の設定**<br/>
    登録時にはデータ サーバー地域、つまりリージョンを選択する必要があります。特定のリージョンを選択すると、そのリージョンに属するデータセンターにユーザー データを保存します。 
-   **決済情報**<br/>
    WhaTapモニタリングサービスは、登録後15日間無料サービスを提供します。無料利用の基準を超えても利用を継続したい場合は、決済情報を登録して有料転換する必要があります。決済方式の詳細については、[次の文書](../management/billing)を参考にしてください。
