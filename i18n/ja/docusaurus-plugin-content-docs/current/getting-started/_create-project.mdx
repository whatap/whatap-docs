## プロジェクト作成{#create-project}

エージェントをインストールするには、まずプロジェクトを作成する必要があります。

1.  [WhaTapモニタリングサービス](https://service.whatap.io)に移動し、ログインします。
2.  モニタリングプロジェクトを作成するには、左側のメニューで**_+ プロジェクト_**ボタンを選択します。
3.  プロジェクト名を入力し、利用するプラットフォームを選択してください。
4.  近くのデータ サーバー地域（AWSまたはAzure）を選択してください。
5.  **_タイムゾーン_**を選択します。<br/>デフォルト値は(GMT +9:00)Seoul、Tokyo、Yakutskです。
6.  **_保存_**ボタンを選択します。

:::note

-   WhaTapモニタリングプロジェクトは、モニタリング単位を意味します。**プロジェクト**に関する詳細については、[次の文書](../project)を参照してください。
-   データサーバー地域は、リージョンを意味します。
-   クラウドでリージョンとは、クラウドサービスを提供するために設定したデータセンターです。
-   特定のリージョンを選択すると、そのリージョンに属するデータセンターにデータが保存されます。
    :::
