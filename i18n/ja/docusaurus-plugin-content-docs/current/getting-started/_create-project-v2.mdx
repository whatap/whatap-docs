## プロジェクト作成{#create-project}

</Xclude>

エージェントをインストールする前にプロジェクトを作成してください。

1. [WhaTapモニタリングサービス](https://service.whatap.io)にログインしてください。

2. 左側のサイドメニューで <Cmdname sid="BTN07332" className="uitext" /> &gt; + <Cmdname sid='BTN06696' className='uitext'/> ボタンをクリックします。

3. <Cmdname sid="TTL07073" className="uitext" /> 画面でインストールする製品を選択します。

4. 以下の項目を入力または選択します。

   - <Cmdname sid="project_name" className="uitext" />：プロジェクト名を入力します。

   - <Cmdname sid="data_server_region" className="uitext" />：データサーバーが所在するリージョンを選択します。リージョンとは、クラウドサービスを提供するデータセンターの集合体です。選択したリージョンにユーザーデータが保存されます。

   - <Cmdname sid="project_time_zone" className="uitext" />：アラートおよびレポート生成時の基準となる時刻を設定します。

   - <Cmdname sid="TTL07727" className="uitext" />：アラート通知メッセージの言語を設定します。（韓国語、英語に対応）

   - <Cmdname sid="project_groups" className="uitext" />：複数のプロジェクトをグループとしてまとめて管理できます。所属するグループがある場合は選択してください。

   - <Cmdname sid="project_description" className="uitext" />：プロジェクトに関する補足説明や詳細情報を入力します。

5. すべての設定が完了したら、<Cmdname sid="creating_a_project" className="uitext" /> ボタンをクリックします。

:::note

- グループに関する詳細は、[次のドキュメント](/project/group)を参照してください。

- 組織を選択してプロジェクトを追加する場合は、<Cmdname sid="group_of_organization" className="uitext" />を必須に設定する必要があります。

:::
