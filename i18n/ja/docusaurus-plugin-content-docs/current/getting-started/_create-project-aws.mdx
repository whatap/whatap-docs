エージェントをインストールするには、まずプロジェクトを作成する必要があります。

1. [WhaTapモニタリングサービス](https://service.whatap.io)に移動し、ログインします。
2. モニタリングプロジェクトを作成するには、左側のメニューで**+ プロジェクト**ボタンを選択します。
3. **製品カタログ**からインストールする製品を選択します。 
4. **プロジェクト名**を入力してください。 
5. 近接した**データサーバーのリージョン**を選択します。
6. **タイムゾーン**を選択します。<br/>既定値は(GMT +9:00)Seoul、Tokyo、Yakutskです。
7. **保存**ボタンを選択します。

:::note

- WhaTapモニタリングプロジェクトは、モニタリング単位を意味します。**プロジェクト**に関する詳細については、[次の文書](../project/project-manage)を参照してください。
- データサーバー地域は、リージョンを意味します。
- クラウドでリージョンとは、クラウドサービスを提供するために設定したデータセンターです。
- 特定のリージョンを選択する場合、そのリージョンに属するデータセンターにデータを格納します。

:::
