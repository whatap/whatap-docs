:::note

-   各ウィジェットで右上の ![詳細アイコン](/img/ico-detail-mode.svg)ボタンをクリックして、ウィジェットデータを全画面サイズで表示できます。

-   ウィジェットの右上にある ![設定アイコン](/img/ico-setting.svg)ボタンをクリックすると、チャート設定画面が表示されます。 チャートに表示するオプションを設定できます。

-   ウェブブラウザーの画面サイズによってウィジェット右上のアイコンの表示が異なる場合があります。 ![もっと見るアイコン](/img/ico-more.svg)ボタンを選択すると、追加アイコンが表示できます。

-   ウィジェット設定の詳細については [次の文書](dashboard-widget-setting)を参照してください。

:::
