ダッシュボード メニューに配置したウィジェットをユーザーが希望するオプションに設定できます。

:::note
ウェブブラウザーの画面サイズによってウィジェット右上のアイコンの形が異なる場合があります。 ![もっと見るアイコン](/img/ico-more.svg)ボタンを選択すると、追加アイコンを確認できます。
:::

## グラフチャート基準データの変更

![위젯](/img/widget-sample.png)

ウィジェット右上の![指示線1](/img/number-01.png)メトリクスボタンを選択してください。 **_メトリクス_**ポップアップウィンドウが表示されます。 ご希望の項目を選択してグラフチャートのデータを変更できます。

:::note

-   **_メトリクス_** ポップアップウィンドウで選択できる項目は、ウィジェットによって異なる場合があります。
-   ウィジェットのデータによってメトリックボタンを提供しない可能性があります。
    :::

## フルスクリーン表示

フルスクリーンで見たいウィジェットの右上の![詳細を見るアイコン](/img/ico-detail-mode.svg)ボタンをタップしてください。 ウェブブラウザー上でウィジェットのグラフチャートをフルスクリーンサイズで確認できます。

![ウィジェットフルスクリーン](/img/widget-fullscreen-mode.png)

-   画面右上の ![指示線 1](/img/number-01.png)オプションボタンを選択して、データを様々な方法で照会することができます。
-   画面右の ![指示線 2](/img/number-02.png)領域は、メトリクスボタンを通じて選択できるデータタイプです。 [次の文書](#change-data)を参考にしてください。

:::note
ウィジェットのデータによっては、選択できるオプションが異なる場合があります。
:::

### データ照会時間の変更

現在進行中のデータの流れをリアルタイムで確認でき、過ぎ去った時間のデータを確認することもできます。 ![時間選択アイコン](/img/ico-time-set.svg)を選択してください。 **_時間選択_**ポップアップウィンドウが表示されます。

![時間選択](/img/widget-select-time.png)

-   グラフチャートのデータ照会時間を変更するには **_ダッシュボード設定に従う_**のチェックボックスを解除してください。
-   グラフチャートをリアルタイムでモニタリングするには ![リアルタイムアイコン](/img/ico-play-time.svg)ボタンを選択してください。
-   照会時間は ![一時停止アイコン](/img/ico-pause.svg)ボタンを選択して変更できます。
-   ![左方向アイコン](/img/left-arrow.svg)または![右方向アイコン](/img/right-arrow.svg)ボタンを選択して時間を移動できます。 移動間隔は、右側に設定された![指示線1](/img/number-01.png)時間間隔です。
-   ![指示線 1](/img/number-01.png)時間間隔ボタンをクリックして時間間隔を選択できます。
-   日付と時刻を直接選択するには、日付と時刻領域をクリックしてください。

### 以前時間と比較する

過去の時間のデータを現在の時間と比較できます。 ![以前時間との比較アイコン](/img/ico-timing.svg)ボタンを選択してください。 **_前の時間との比較_**ポップアップウィンドウが表示されます。

![以前時間と比較](/img/widget-compare-time.png)

秒、分、時間、日単位で時間を比較でき、ご希望の日付を選んで比較することもできます。 ご希望のオプションを選択した後、**_適用_**ボタンをクリックするとグラフチャートに適用できます。 以前時間データとの比較を取り消すには、**_未設定_**をクリックして適用ボタンを選択してください。

### 基準データの変更{#change-data}

![메트릭스](/img/widget-metric.png)

1.  右上のオプションのうちウィジェットのメトリクスに該当するテキストを選択してください。 (通常は3番目にあります)

2.  メトリクスポップアップウィンドウが表示されます。

3.  項目を選択してください。

グラフチャートのデータを選択した項目に変更します。

### 上位Nデータを確認する

-   **_上位5_**ボタンをクリックすると、最大上位10項目まで選択できます。 選択した数値だけ高い順にグラフチャートにデータを適用できます。

    ![上位N](/img/widget-n-data.png)

-   ![上位Nアイコン](/img/ico-topn-graph.svg)ボタンを選択し、上位Nのデータ項目をグラフチャートにオーバーレイ形式で表示できます。 グラフチャートで上位Nのデータ項目を削除するには、![グラフアイコン](/img/ico-normal-graph.svg)ボタンを選択してください。

### MXQLでデータを照会する

MXQLを利用してデータを照会するには ![メトリクス編集アイコン](/img/ico-edit-metric.svg)ボタンを選択してください。 MXQL編集ウィンドウが表示されます。

:::note
MXQLについての詳細は [次の文書](../mxql/mxql-overview)を参考にしてください。
:::

### 前日のデータ表示

![前日のデータ表示](/img/widget-before-data.png)

-   前日データと比較してグラフチャートを確認するには ![前日データ比較アイコン](/img/ico-before-data.svg)ボタンを選択してください。 ![指示線](/img/number-png)画面左上に前日データの値を表示できます。
-   画面右上の![指示線2](/img/number-02.png)**平均**ボタンをクリックしてグラフチャートのデータタイプを変更できます。 選択できる項目は次のとおりです。 デフォルト値は**_平均_**です。
    -   **_最近値_**
    -   **_最大値_**
    -   **_平均_**
    -   **_合計_**

### テーブルカラムの変更

データテーブルウィジェットの場合、フルスクリーンでカラムを変更できます。 右上の![カラムアイコン](/img/ico-column.svg)ボタンを選択してください。 **_カラム編集_**ウィンドウが表示されたら、テーブルに表示するカラムを非表示または追加できます。 設定を完了した後、**_保存_**ボタンを選択するとデータテーブルチャートに適用できます。

![カラム編集](/img/widget-column.png)

## チャート設定

y軸の最大値を設定し、水平線を追加してしきい値を設定するには、ウィジェットの右上の ![設定アイコン](/img/ico-setting.svg)ボタンを選択してください。

![チャート設定](/img/widget-chart-set.png)

-   **_y軸チャート最大値_**：**_自動_**または**_固定_**を選択してy軸の最大値を設定することができます。 **_固定_**を選択した場合、数値を入力できます。 数値を入力したら**_適用_**ボタンを選択してください。

-   **_水平線設定_**：グラフチャートに水平線を追加してしきい値を設定し、効率よくモニタリングできます。

    -   **_+追加_**ボタンを選択した後**_ラベル_**と**_しきい値_**項目に数値を入力し、色を選択してください。 **_しきい値_**項目では、**_範囲_**を選択して水平線エリアを設定できます。 **_適用_**ボタンを選択して設定を完了してください。

    -   水平線を削除するには！[削除アイコン](/img/ico-trash-red.svg)ボタンを選択してください。

-   **_チャート最大値表示_**：グラフチャートに最大値を表示できます。 最も高い数値をチャートにテキストとして表示します。
