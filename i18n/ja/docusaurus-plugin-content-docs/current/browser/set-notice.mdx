- ***イベント設定***メニューにより、しきい値ベースの通知条件および受信設定を行うことができます。
- ***イベント受信設定***メニューでは、個人通知の受信チャネルおよびタイムゾーンを設定できます。
- ***イベント履歴***メニューでは、通知発生履歴を確認できます。
