**_페이지 로드_** 대시보드에서는 페이지 로드와 관련한 코어 웹 바이탈(Core Web Vitals, CWV) 및 리소스, AJAX 등의 성능 데이터를 실시간으로 모니터링할 수 있습니다. 이를 통해 최종 사용자의 웹 애플리케이션 경험에 영향을 미치는 요인들을 파악하고 개선점을 도출하는데 도움을 받을 수 있습니다.

{@include: _dashboard-note.mdx}

## コアウェブバイタル関連ウィジェット

コアウェブバイタル（Core Web Vitals、CWV）は、エンドユーザー経験に影響を及ぼす様々な測定値のうち、グーグルで重要だという3つのメトリクスを意味します。

![コアウェブバイタル]（/img/rum-core-web-vitals.png）

:::note

グーグルの[コアウェブバイタルレポート]（https://support.google.com/webmasters/answer/9205520）文書で、ページ性能がエンド ユーザー エクスペリエンスに与える影響を確認してください。

:::

### **_平均最大コンテンツレンダリング時間_**

**최대 콘텐츠풀 페인트(Largest Contentful Paint, 이하 LCP)**는 처음 페이지를 불러오기 시작한 시점을 기준으로 브라우저 뷰포트 내에 있는 가장 큰 이미지 또는 텍스트 블록의 렌더링 시간을 보고합니다. 최종 사용자가 감지하는 페이지 로드 시간을 측정할 수 있는 중요한 사용자 중심 지표입니다. LCP가 빠르면 최종 사용자가 해당 페이지를 사용할 수 있다고 인지하는 데 도움이 됩니다.

![平均最大コンテンツレンダリング時間]（/img/rum-lcp.png）

최적화된 사용자 경험을 제공하려면 웹 애플리케이션의 LCP가 **2.5초** 이하여야 합니다. **_평균 최대 콘텐츠 렌더링 시간_** 위젯은 페이지 그룹 또는 전체 페이지 기준으로 평균 LCP를 확인할 수 있습니다. 페이지 그룹 기준으로 차트를 확인할 경우 상위 5개 페이지 그룹의 평균 LCP를 제공합니다.

### **_平均最大入力反応時間_**

**최초 입력 지연(First Input Delay, 이하 FID)**는 최종 사용자가 웹 애플리케이션의 페이지와 처음 상호 작용할 때(링크를 클릭하거나 버트을 탭하거나 사용자 지정 Javascript 기반 컨트롤을 사용할 때)부터 해당 상호 작용에 대한 응답으로 브라우저가 실제로 이벤트 핸들러 처리를 시작하기까지의 시간을 측정합니다. 사용자가 응답하지 않는 페이지와 상호 작용할 때 느끼는 경험에 대한 지표입니다. FID가 낮아야 사용자는 해당 페이지를 정상적으로 사용할 수 있다고 인지합니다.

![平均最大入力反応時間]（/img/rum-pageload-fid.png）

최적화된 사용자 경험을 제공하려면 웹 애플리케이션의 FID가 100ms 이하여야 합니다. **_평균 최대 입력 반응 시간_** 위젯은 페이지 그룹 또는 전체 페이지 기준으로 평균 FID를 확인할 수 있습니다. 페이지 그룹 기준으로 차트를 확인할 경우 상위 5개 페이지 그룹의 평균 FID를 제공합니다.

### **_平均レイアウト変更点数_**

누적 레이아웃 이동(Cumluative Layout Shift, 이하 CLS)은 최종 사용자가 예상치 못한 레이아웃 이동을 경험하는 빈도를 수량화하여 시각적 안정성을 측정할 때 중요한 사용자 중심 지표입니다. CLS가 낮으면 우수한 사용자 경험을 보장하는데 도움이 됩니다.

![平均レイアウト変更点数]（/img/rum-pageload-cls.png）

최적화된 사용자 경험을 제공하려면 CLS가 0.1 이하를 유지해야 합니다. **_평균 레이아웃 변경 점수_** 위젯은 페이지 그룹 또는 전체 페이지 기준으로 평균 CLS를 확인할 수 있습니다. 페이지 그룹 기준으로 차트를 확인할 경우 상위 5개 페이지 그룹의 평균 CLS를 제공합니다.

### **_コアウェブバイタルテーブル_**

![コアウェブバイタルテーブル]（/img/rum-pageload-cwv-table.png）

조회 시간 기준 페이지 그룹(**_Page Group_**) 단위로 평균 LCP(**_Avg. LCP_**), 평균 FID(**_Avg. FID_**), 평균 CLS(**_Avg. CLS_**)를 테이블로 확인할 수 있습니다. 테이블 헤드 컬럼을 클릭해 컬럼별로 정렬할 수 있습니다.

## ページロードタイミング関連ウィジェット

エンドユーザーがウェブアプリケーションに接続したときのページロード時間とロード数を通じて、ページロード時間が長くかかるケースと負荷が発生するページを確認できます。

### **_平均ページロード時間_** / **_ページロード数_**

![平均ページロード時間、ページロード数]（/img/rum-pageload-timing.png）

최종 사용자가 웹 애플리케이션의 웹 페이지를 접속했을 때 실시간 또는 선택한 시간 기준으로 **_평균 페이지 로드 시간_**과 **_페이지 로드 수_**를 확인할 수 있습니다. 페이지 그룹 단위로 로드 시간과 로드 수를 기본 제공합니다.

-   ページグループ別の照会時、上位10ページグループの平均ページロード時間とページロード数を提供します。

-   ウィジェットチャートにマウスオーバーすると、ツールチップ形式でページグループと平均ページロード時間、ロード数の情報を提供します。

### **_平均ルーター変更時間_** / **_ルーター変更回数_**

![平均ルーター変更時間、ルーター変更回数]（/img/rum-pageload-router.png）

최종 사용자가 Single Page Application(SPA)로 구성되어 있는 웹 애플리케이션에서 페이지 전환을 했을 때 실시간 또는 선택한 시간 기준으로 **_평균 라우터 변경 시간_**과 **_라우터 변경 횟수_**를 실시간으로 확인할 수 있습니다. 페이지 그룹 단위로 라우터 변경 시간과 변경 횟수를 기본 제공합니다.

-   ページグループごとの照会時に、上位10ページグループの平均ルーターの変更時間と変更回数を提供します。

-   ウィジェットチャートにマウスオーバーすると、ツールチップの形でページグループと平均ルーター変更時間と変更回数の情報を提供します。

### **_ページロードタイミングテーブル_**

![ページロードタイミングテーブル]（/img/rum-pageload-timing-table.png）

페이지 그룹(**_Page Group_**) 단위로 페이지 로드 수(**_Load count_**), 평균 로드 시간(**_Avg. Load Time_**), 평균 백엔드 처리 시간(**_Avg. Backend Time_**), 평균 프론트엔드 처리 시간(**_Avg. Frontend Time_**)을 테이블로 확인할 수 있습니다. 테이블 헤드 컬럼을 클릭해 컬럼별로 정렬할 수 있습니다.

## リソース関連ウィジェット

페이지 로드에 영향을 주는 리소스를 식별할 수 있습니다. 리소스의 로드 시간은 800ms를 넘지 않는게 좋습니다.

### 平均リソースレスポンス時間 / リソース件数

![平均リソースレスポンス時間、リソース件数]（/img/rum-pageload-resource.png）

페이지 로드 시 발생했던 평균 리소스 응답시간 및 리소스 건수를 차트로 제공합니다. 위젯 오른쪽 위에 **_리소스 Path_** 버튼을 선택해 **_리소스 Host_**, **_전체_** 리소스 기준으로 확인할 수 있습니다.

-   **_リソースパス_**、**_リソースホスト_** を基準に、照会時に上位10に該当する平均リソースレスポンス時間およびリソース件数を提供します。

-   ウィジェットチャートにマウスオーバーすると、ツールチップの形で平均リソースレスポンス時間とリソース件数情報を提供します。

### リソーステーブル

![リソーステーブル]（/img/rum-pageload-resource-table.png）

**_페이지 로드 - 리소스 테이블_** 위젯에서는 리소스 URL의 Path 기준과 요청한 페이지 그룹 기준을 테이블로 확인할 수 있습니다. 위젯 오른쪽 위에 **_리소스 Path_** 버튼을 선택해 **_페이지 그룹_** 기준으로 전환해 데이터를 조회할 수 있습니다.

## AJAX関連ウィジェット

페이지 로드에 영향을 주는 AJAX를 식별할 수 있습니다. 렌더링에 필요한 데이터를 AJAX를 통해 제공 받는 경우 응답이 느리다면 화면의 렌더링이 오래 걸릴 수 있습니다.

### 平均AJAXレスポンス時間 / AJAX件数

![AJAXレスポンス時間、AJAX件数]（/img/rum-pageload-ajax.png）

페이지 로드 시 발생한 평균 AJAX 응답 시간 및 AJAX 건수를 차트로 제공합니다. 위젯 오른족 위에 **_AJAX Path_** 버튼을 선택해 **_AJAX Host_**, **_전체_** AJAX 기준으로 전환해 데이터를 조회할 수 있습니다.

-   **_AJAX Path_**、**_AJAX Host_** 基準で照会時、上位10個に該当する平均AJAXレスポンス時間とAJAX件数を提供します。

-   ウィジェットチャートにマウスオーバーすると、ツールチップの形で平均AJAXレスポンス時間とAJAX件数の情報を提供します。

### AJAXテーブル

![AJAXテーブル]（/img/rum-pageload-ajax-table.png）

**_페이지 로드 - AJAX 테이블_** 위젯에서는 AJAX URL의 Path 기준과 요청한 페이지 그룹 기준을 테이블로 확인할 수 있습니다. 위젯 오른쪽 위에 **_AJAX Path_** 버튼을 선택해 **_페이지 그룹_** 기준으로 전환해 데이터를 조회할 수 있습니다.
