## デプロイ履歴

### Linuxモニタリングエージェント

リリース：2022-12-27

<Status>New</Status> CentOS Stream 9 / Oracle Linux 9  - <code>systemctl</code>のサポートが追加しました。

### Windowsモニタリングエージェントの午前2時頃の自動再起動エラーを修正

リリース：2022-12-14

<Status>Fixed</Status> 2.2.8バージョンのコードエラーによる、Windowsモニタリングエージェントが自動的に再起動するエラーがあります。午前2時頃に停止され、Agent NoDataの誤検出が発生する可能性があります。2.2.8バージョンを使用している場合は、アップデートをお勧めします。
