    <div class='.release-compare'>

    |                      変更前                     |                     変更後                     |
    | :------------------------------------------: | :-----------------------------------------: |
    | ![](/img/release/interval-option-before.png) | ![](/img/release/interval-option-after.png) |

-   <Status>Changed</Status><UI>時間</UI>照会区間が変更されるとき<UI>インターバル</UI>選択オプションから<UI>デフォルト</UI>項目を選択するようにロジックを修正

    照会区間を変更する際、**_インターバル_**オプションが**_デフォルト_**項目が選択される場合は、次のとおりです。

    -   **_インターバル_**オプションで以前の選択項目が**_デフォルト_**の場合

    -   **_インターバル_**オプションで以前の選択項目が**_デフォルト_**ではなく、変更された**_インターバル_** オプション一覧にない場合

    -   **_インターバル_**オプションで以前の選択項目が**_デフォルト_**ではなく、選択した**_インターバル_** オプション値が照会期間の半分以下の場合
