2025年07月09日

* <Status priority='0'>Fixed</Status> whatap-php.service の登録および実行に失敗する問題を修正

  * 新規インストール時、/etc/systemd/system/whatap-php.service に environment 値（Access Key およびサーバーアドレス）が欠落しており、エージェントが非活性化される現象が発生

  * /usr/whatap/php/install.sh 実行後、/usr/lib/systemd/system/whatap-php.service を /etc/systemd/system/whatap-php.service にコピーすることで解決可能

  * 既存のインストール環境では、アップデート時に正常にインストールされる

2025年06月18日

* <Status priority='0'>Feature</Status> CPU負荷の軽減

  * モニタリング対象判定基準の変更によりCPU負荷を削減

  * ユーザー関数モニタリングを無効化するとさらにCPU負荷が減少

    `whatap.profile_method_enabled=false`

* <Status priority='1'>New</Status> CLIパラメーターのURL表示

  * CLI実行時、パラメーターを含むURLを収集

  * 例）php /home/www/test.php aaa bbb → /home/www/test.php aaa bbbとして記録

* <Status priority='1'>New</Status> fileパラメーター情報の収集

  * 最大20個、各256バイトまでパラメーター情報を一部収集

* <Status priority='0'>Fixed</Status> メモリリーク修正

  * DB関数の失敗時、error関数呼び出しによるメモリリークを修正
