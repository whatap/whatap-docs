2025年06月18日

- <Status priority='0'>Feature</Status> CPU負荷の軽減

  - モニタリング対象の判定基準を変更することで、CPU負荷を軽減しました。

  - ユーザー関数モニタリング無効化時の追加負荷を軽減

    `whatap.profile_method_enabled=false`

- <Status priority='1'>New</Status> CLI呼び出しパラメータのURL表示

  - CLI呼び出し時にパラメータを含めてURLを収集

    - 例) php /home/www/test.php aaa bbb
      → /home/www/test.php aaa bbb として表示

- <Status priority='1'>New</Status> fileパラメーター情報の収集

  - 最大20個まで、各256byteまでのパラメーターの一部のみを収集します。

- <Status priority='0'>Fixed</Status> メモリリークを修正しました

  - DB関数の失敗時にerror関数が呼び出されることによるメモリリークを修正

:::info
