2025年07月29日

- <Status priority='1'>New</Status> **whatap_code_start** オプションを追加

    - デフォルト: `false`

    - `true` に設定すると、*sitecustomize.py* 内で `import whatap` の後に `whatap.agent()` を実行することで、whatap-start-agent コマンドなしでも、通常の Python 実行コマンドでエージェントが起動可能になります。

- <Status priority='0'>Fixed</Status> **open_file_descriptor_enabled** オプションを修正

    - `false` に設定すると、ファイルディスクリプタを追跡するスレッドも無効になります。

:::note

:::
