2025年9月12日

- <Status priority='1'>New</Status> LLMログ監視に `@step_id` フィールドを追加

- <Status priority='1'>New</Status> LLMメトリクス収集機能を追加

    - **llm_stat_enabled**  
        - `false`: デフォルト  
        - `true`: このオプションと **trace_llm_log_enabled**、**counter_thread_enabled** がすべて `true` の場合にメトリクス収集が有効化されます

    - **llm_stat_interval**  
        - LLMメトリクスの収集間隔を秒単位で設定するオプション  
        - `5`: デフォルト

- <Status priority='1'>Feature</Status> **counter_thread_enabled** オプションを追加  

    - `false`: デフォルト  
    - `true`: llm_stat と app_filedescriptor メトリクスを収集するスレッド（`counter_thread`）を有効化

- <Status priority='1'>Changed</Status> `psycopg_pool` トレースモジュールの安定性を向上

- <Status priority='1'>Changed</Status> `psycopg` と `psycopg_pool` の非同期トレース安定性を向上

- <Status priority='1'>Changed</Status> **open_file_descriptor_enabled** オプションは `counter_thread` の有効化に影響しなくなりました
