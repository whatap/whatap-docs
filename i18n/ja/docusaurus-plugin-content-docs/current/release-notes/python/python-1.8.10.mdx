2025年 8月 22日

- <Status priority='1'>New</Status> `psycopg` モジュールのトレース機能を追加

- <Status priority='1'>New</Status> `WHATAP_PYTHON_AGENT_PATH` 環境変数を使用して、*whatap_python* 実行ファイルのパスを指定できるようになりました。

    - この環境変数を設定しない場合は、`WHATAP_HOME` が代わりに使用されます。

    :::note

    詳細は [Kubernetes Agent v1.8.8](https://docs.whatap.io/release-notes/k8s/k8s-1_8_8) を参照してください。

    :::

- <Status priority='0'>Fixed</Status> `neo4j` モジュールの修正  

    * DB 接続のタイミングが正しく収集されない問題を修正  

:::note

エージェント ダウンロード: [whatap-python v1.8.10](https://pypi.org/project/whatap-python/1.8.10/)
