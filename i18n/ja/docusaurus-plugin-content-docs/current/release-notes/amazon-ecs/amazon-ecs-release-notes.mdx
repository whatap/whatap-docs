## v1.2.1

2022年12月11日

### 収集サーバーとの接続が切断された際、タスクが異常終了する原因となっていたコードエラーを修正しました。

WhaTapエージェントコードエラーによる収集サーバーの接続が切断されると、以下のタスクが異常終了する問題を修正しました。 

- エージェントがダウンした場合、タスクの実行スクリプトにコードを追加することで安全に再実行します。
- 収集サーバーとの接続時の再接続試行コードのエラー（非同期ルーチン）を修正
