- <Status priority='1'>New</Status> `ignore_env_variable_set` <span class='type'>String</span>

  - 既定値なし

  - 特定の環境変数の収集を無視するオプションを追加

  - カンマ（`,`）で区切って複数の値を指定可能

  - 使用例. `ignore_env_variable_set=ENV_1,ENV_2`

- <Status priority='0'>Fixed</Status> mysql2 v3.12.0以上のバージョンでモニタリングされない現象を修正
