2025年02月18日

<Status priority='1'>New</Status> GraphQLモジュール(@apollo/server)モニタリングを追加

- **profile_graphql_enabled** <span class='type'>Boolean</span>

  既定値 `true`

  GraphQLモニタリングの活性化有無を設定します。

- **profile_graphql_variable_enabled** <span class='type'>Boolean</span> 

  既定値`false` 

  トランザクションプロファイルにGraphQLリクエストのvariables情報を含めるかどうかを設定します。

- **ignore_graphql_operation** <span class='type'>String</span> 

  指定されたOperation Nameのモニタリングを無視するように設定します。 複数の値を対象とする場合は、コンマ(`,`)を区切り文字として使用します。 

  ```ini title='whatap.conf'
  ignore_graphql_operation=operationNameA,operationNameB
  ```
