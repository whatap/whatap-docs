<Status priority='1'>New</Status> Prisma ORM使用のためのオプションを追加

- **prisma_read_func_name** <span class='type'>String</span>

  デフォルト値`read`

  Replication使用時にRead-onlyクライアントを取得するために呼び出す関数名を設定します。

- **prisma_database_url_name** <span class='type'>String</span>

  デフォルト値`DATABASE_URL`

  Prisma ORMで、データベース接続URLが保存された環境変数名を設定します。
