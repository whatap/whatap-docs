2025年02月25日

- <Status priority='1'>Feature</Status> Spanの`name`値をトランザクション名として指定するオプションを追加

  - **otel_tx_path_setby_name_enabled** <span class='type'>bool</span> (デフォルト値 `false`)

- <Status priority='1'>Feature</Status> Trace Spanの`SpanID`を外部接続の`ParentSpanID`として認識するマルチトランザクション接続関連オプションを追加

  - **otel_mtrace_stepid_setby_trace_spanid_enabled** <span class='type'>bool</span> (デフォルト値 `false`)

- <Status priority='1'>Feature</Status> ファイルログと同じ内容を`Stdout`で出力するログオプションを追加

  - **log_stdout_enabled** <span class='type'>bool</span> (デフォルト値 `false`)

  :::note 

  Docker及びKubernetes環境で環境変数を登録して使用できます。 

  :::

- <Status priority='0'>Fixed</Status> *whatap.conf*ファイルがない場合、ファイルなしエラーログが出力され続ける現象を修正

- <Status priority='0'>Fixed</Status> トランザクションIDが`0`と表記される現象を修正

:::info
