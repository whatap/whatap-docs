2025年07月09日

* <Status priority='2'>Feature</Status> ParentのないSpanをトランザクションとして認識  
  * ParentがないSpanは、kindに関係なくトランザクションとして処理されます  
  * 以前は`SPAN_KIND_SERVER`のみがトランザクションとして処理されていました  
  * 関連オプション：`otel_tx_root_span_enabled`（デフォルト：true）

* <Status priority='2'>Feature</Status> メトリクス収集方式の改善  
  * 従来のログベース収集からPrometheus互換方式に変更  
  * Flexボード（MXQLデータ照会）で確認可能

* <Status priority='1'>Change</Status> `query_string_enabled`オプションの動作変更  
  * `query_string_enabled`がtrueで、`query_string_urls`と`query_string_keys`が設定されていない場合、  
    すべてのURLのクエリストリングが収集されます

* <Status priority='0'>Fixed</Status> HTTPステータスカウンター収集エラーを修正  
  * 500番台エラーの合計値が収集されなかった問題を修正
