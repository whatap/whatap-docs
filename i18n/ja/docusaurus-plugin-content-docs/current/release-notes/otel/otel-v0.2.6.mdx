2025年9月1日

* <Status priority='1'>Change</Status> Spanデータ処理方式の改善  
  * Spanデータを親子のツリー構造で収集  
  * 中間の親Spanデータがなくても後続の子Spanデータを処理できるように変更  

* <Status priority='1'>Change</Status> 属性キー名を追加  
  * HTTPステータスコードを収集する属性キー名 `http.response.status_code` を追加  
    * `otel_attr_tx_status_keys`  
      * デフォルト: `http.status_code`, `http.response.status_code`  

  * エラー情報を収集する属性キー名 `error` を追加  
    * `otel_attr_error_keys`  
      * デフォルト: `error`, `error.type`
