## `共通`

**_クラウドモニタリング_** - Azureのインストール機能の追加

**_ログモニタリング_**

- アプリケーションGoプロジェクトのログモニタリング機能を追加
- ログ検索画面のユーザビリティの改善

**_Flexボード_**

- プロジェクトカードウィジェットの追加
- サーバーネットワークウィジェットの追加
- シリーズチャートウィジェットは、以前の時間の比較機能を追加
- ウィジェットポップアップウィンドウ機能の追加
- フィルター機能の使用性を改善

**_イベント設定_** - メトリクスイベント設定の編集ウィンドウのユーザビリティを改善

**_組織管理_** - 決済権限の変更時に決済情報を確認する検証ロジックを修正

## アプリケーション

**_メトリクスチャート_** - ウィジェット削除エラーを修正

## サーバー

**_サーバーリスト_** - 一覧の更新時にサーバー選択状態を保持し、比較する機能を改善

**_マスターメタ情報_** - ユーザビリティ改善画面の刷新

**_Kubernetesプラットフォームビュー_** - デザインガイド適用
