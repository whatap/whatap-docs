- <Status priority='1'>Feature</Status> <Cmdname sid="side_serverDetail" className="uitext" /> 過去時点を照会する際、削除されたサーバーでもデータが残っている場合はサーバー選択セレクターに非アクティブ状態で表示

- <Status priority='0'>Fixed</Status> **Featureプロジェクト** &gt; **KAFKAダッシュボード** 時間更新時にすべてのウィジェットで毎回ローダーが表示され、データが隠れる問題を改善
