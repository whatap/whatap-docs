## Service 2.18.0

2025年06月26日

<div class='indentTab'>

### `共通`

- <Status priority='1'>Feature</Status> サービスページタイトルの右側に WhaTap Docs リンクを追加
- <Status priority='1'>Feature</Status> <Cmdname sid="side_flexboard" className="uitext" /> ウィジェットに隠されたプロジェクト名のツールチップを追加
- <Status priority='1'>Feature</Status> <Cmdname sid="side_flexboard" className="uitext" /> ウィジェット検索時、フォルダ内にウィジェットがない場合はそのフォルダを非表示

- <Status priority='0'>Fixed</Status> <Cmdname sid="group_management" className="uitext" /> で現在のグループを削除後、初期ページに移動するよう修正
- <Status priority='0'>Fixed</Status> 別ウィンドウから戻った際にアラートが自動的に消えなかった問題を修正
- <Status priority='0'>Fixed</Status> 権限移譲時の同意ポップアップがプロジェクト選択ポップアップに隠れていた問題を修正
