リリーズ：2022-05-11

## `共通`

**_ログモニタリング_**

-   トランザクションプロファイルのログタブの追加
-   ログの構文解析機能の追加 
-   ログ読み取り状態メニューの追加

**_Flexボード_**

-   ユーザー作成ダッシュボードを組織単位で管理
-   アクティブトランザクションのウィジェットチャートのクリック時の詳細な参照機能の追加 
-   ダッシュボード管理およびプロジェクト選択機能の利便性の改善
-   適用されたフィルターオプションの選択状態の未表示とダッシュボードの上部に追加した情報が表示される問題を修正
-   ウィジェット編集機能のユーザビリティの改善

**_請求と決済履歴_** - CSVダウンロード機能の追加 

**_メトリクスイベント設定_** - イベント発生条件および対象フィルタリング条件入力の操作性改善

**_組織管理_**

-   メニューから移動およびプロジェクト削除時に選択した組織情報が消去する現象を修正
-   Kubernetesネームスペースプロジェクト組織及びグループ移動時、一覧に反映されない問題を修正 

## アプリケーション

**_トランザクションプロファイル_** - ログ検索機能の追加

**_トランザクションマップ_** - フィルター機能の修正 

**_アクティブトランザクション_** - チャート初期の読み込み遅延の修正

**_統計 - ドメイン参照_** - 空きデータの参照時に読み込み状態が続く現象を修正

## サーバー

**_プロセスイベント設定_** - 継続時間の設定時、0分~999分と入力値を制限

## データベース

**_カウンター推移_** - タイムオプション変更を連続するとチャートデータが隠れる現象を修正

## コンテナ

**_ネームスペース管理_**

-   Kubernetesマスタープロジェクトの **_管理 - ネームスペース管理_** メニューの追加
-   ネームスペースプロジェクト追加機能
-   Node、コンテナ、Podステータスの照会

**_ネームスペース状況_**

-   プロジェクトリストでクラスタープロジェクトをクリック時に移動していたKubernetesマスターグループビューをマスタープロジェクト内の **_リソース - ネームスペースステータス_** メニューに移動します。
-   左側のプロジェクトリストでは、既存のKubernetesマスターグループの代わりに、ユーザーが指定したプロジェクトグループとして表示されます。 

**_コンテナマップ_**

-   Container、Podカテゴリのメトリクス詳細な検索ウィジェットの追加
-   チャートの読みやすさの改善

## URLモニタリング

**_無料_** - URL登録及びプロジェクト作成の無制限利用

**_Flexボード_** - URL一覧ウィジェットの追加
