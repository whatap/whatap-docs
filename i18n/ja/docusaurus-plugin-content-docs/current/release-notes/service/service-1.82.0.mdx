## `共通`

***Flexボード*** - アクティブトランザクションのイコライザーチャートのクリック時にアクティブトランザクションを表示する機能を追加

***エージェント設定*** - オプション追加で移動ボタンを追加

***ログモニタリング*** - ログ一覧のスタイルを修正

***イベント検索*** - メトリクス通知などの他のタイプの通知での処理履歴機能を追加

***レポート*** - アプリケーションのカスタマイズされたレポートから不要な選択を削除

***Flexボード*** - テーブルウィジェットフィルターが適用されないエラーを修正

***メトリクス検索*** - メタなしのカテゴリを選択すると、メトリクスエクスプローラーのショートカットリンクが機能しないエラーを修正

***ログエクスプローラー*** - スクロールを下げるとデータの不整合が発生するエラーを修正

***ログ検索機*** - 検索結果がない場合、以前に検索されたデータが表示されるエラーを修正

***メンバー一覧*** - メンバーの検索時、検索ウィンドウに入力された値が残るエラーを修正

***プロジェクト設定*** - プロジェクト名を変更した後に他のメニューに移動すると変更前のプロジェクト名が表示されるエラーを修正

***プロジェクトリスト*** - 組織の選択時、プロジェクトカード一覧と組織のプロジェクトが一致しないエラーを修正

***エージェント設定*** - エージェントのクリック時、エージェント一覧 ツリーが閉じるエラーを修正

## アプリケーション

***トランザクション追跡*** - ツリービュー機能の追加・補足

***ヒットマップ*** - ヒットマップの x軸移動ボタンのクリック時の時間変更エラーを修正

## コンテナ

***コンテナマップ*** - 以前のコンテナログを表示するタブを追加

***コンテナマップ*** - 発生したイベントに一致するタブにフォーカスする機能を追加

***コンテナマップ*** - 正常(青) のしきい値のゾーンによる3段階の区分を削除し、単一色に修正

***コンテナマップ*** - Podカテゴリで最新のログメッセージタブが表示されないエラーを修正

***ノード詳細情報*** - コンテナ一覧のカラムとデータを変更

***コンテナ一覧*** - MemWsByRequestカラムデータエラーの修正
