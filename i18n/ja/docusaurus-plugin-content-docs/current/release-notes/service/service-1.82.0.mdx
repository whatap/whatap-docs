## `共通`

**_Flexボード_** - アクティブトランザクションのイコライザーチャートのクリック時にアクティブトランザクションを表示する機能を追加

**_エージェント設定_** - オプション追加で移動ボタンを追加

**_ログモニタリング_** - ログ一覧のスタイルを修正

**_イベント検索_** - メトリクス通知などの他のタイプの通知での処理履歴機能を追加

**_レポート_** - アプリケーションのカスタマイズされたレポートから不要な選択を削除

**_Flexボード_** - テーブルウィジェットフィルターが適用されないエラーを修正

**_メトリクス検索_** - メタなしのカテゴリを選択すると、メトリクスエクスプローラーのショートカットリンクが機能しないエラーを修正

**_ログエクスプローラー_** - スクロールを下げるとデータの不整合が発生するエラーを修正

**_ログ検索機_** - 検索結果がない場合、以前に検索されたデータが表示されるエラーを修正

**_メンバー一覧_** - メンバーの検索時、検索ウィンドウに入力された値が残るエラーを修正

**_プロジェクト設定_** - プロジェクト名を変更した後に他のメニューに移動すると変更前のプロジェクト名が表示されるエラーを修正

**_プロジェクトリスト_** - 組織の選択時、プロジェクトカード一覧と組織のプロジェクトが一致しないエラーを修正

**_エージェント設定_** - エージェントのクリック時、エージェント一覧 ツリーが閉じるエラーを修正

## アプリケーション

**_トランザクション追跡_** - ツリービュー機能の追加・補足

**_ヒートマップ_** - ヒートマップの x軸移動ボタンのクリック時の時間変更エラーを修正

## コンテナ

**_コンテナマップ_** - 以前のコンテナログを表示するタブを追加

**_コンテナマップ_** - 発生したイベントに一致するタブにフォーカスする機能を追加

**_コンテナマップ_** - 正常(青) のしきい値のゾーンによる3段階の区分を削除し、単一色に修正

**_コンテナマップ_** - Podカテゴリで最新のログメッセージタブが表示されないエラーを修正

**_ノード詳細情報_** - コンテナ一覧のカラムとデータを変更

**_コンテナ一覧_** - MemWsByRequestカラムデータエラーの修正
