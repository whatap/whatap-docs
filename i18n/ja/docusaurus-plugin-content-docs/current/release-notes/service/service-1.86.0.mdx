## `共通`

**_Flexボード_**

- ダッシュボードのPDFエクスポート機能を追加 
- ウィジェット間のグループ化機能を追加(ウィジェットテンプレート > その他 > ウィジェットグループのグループ化)
- お気に入りのダッシュボード間のメニューを移動する場合の時間表示範囲を更新します。

**_統合ダッシュボード_** - トランザクションプロファイルの連続照会ができなかった問題を修正

**_時間セレクタ_** - 時間移動時の問題を修正

## インテグレーション

**_インテグレーションプロジェクト_**

- クラウドサービスを含む各種モニタリング対象を１つのプロジェクトで統合的に管理するための新規製品の追加
- インテグレーションアイテム別のダッシュボードと収集メトリクスを用いた分析メニュー、イベント設定機能を提供
- インテグレーション項目
  - Amazon CloudWatch
  - Amazon ECS
  - Naver Cloud Monitoring
  - Oracle Cloud Monitor
  - Azure Monitor

## アプリケーション

**_マルチトランザクション追跡_** - プロジェクトリストの階層化表示エラーを修正

**_トランザクションプロファイル_** - ダイアグラムフォントサイズの調整問題を修正

## コンテナ

**_コンテナマップ_**

- Container画面のActiveTX操作ルールの改善
- 詳細画面 > イベントNormal、Warningタイプのフィルタリング機能を追加
- 詳細画面 > カード上部のステータス別の数量の部分表示スタイルを改善

**_Out of Memory Killedコンテナ_** - 照会時間範囲の選択オプションを削除およびテーブルソート機能を修正 

## ログ

**_ログ設定_** - 現在の使用状況を表示する機能を追加 

## サイト管理者

**_決済管理_** > **_プロジェクト有料変更_** - 製品種類、プラットフォームのカラムをマージ

**_決済管理_** > **_請求管理_**、**_アカウント状態_** - 「open」から 「正常」にカラム名を変更

**_決済管理_** > **_請求書_** - 「Sub Total」から「使用額」にカラム名を変更、顧客表示(設定)、顧客表示(状態)、割引情報表示カラムの非表示・表示メッセージの修正
