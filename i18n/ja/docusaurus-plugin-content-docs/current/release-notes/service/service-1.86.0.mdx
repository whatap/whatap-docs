## `共通`

***Flexボード***

- ダッシュボードのPDFエクスポート機能を追加 
- ウィジェット間のグループ化機能を追加(ウィジェットテンプレート > その他 > ウィジェットグループのグループ化)
- お気に入りのダッシュボード間のメニューを移動する場合の時間表示範囲を更新します。

***統合ダッシュボード*** - トランザクションプロファイルの連続照会ができなかった問題を修正

***時間セレクタ*** - 時間移動時の問題を修正

## インテグレーション

***インテグレーションプロジェクト***

- クラウドサービスを含む各種モニタリング対象を１つのプロジェクトで統合的に管理するための新規製品の追加
- インテグレーションアイテム別のダッシュボードと収集メトリクスを用いた分析メニュー、イベント設定機能を提供
- インテグレーション項目
  - Amazon CloudWatch
  - Amazon ECS
  - Naver Cloud Monitoring
  - Oracle Cloud Monitor
  - Azure Monitor

## アプリケーション

***マルチトランザクション追跡*** - プロジェクトリストの階層化表示エラーを修正

***トランザクションプロファイル*** - ダイアグラムフォントサイズの調整問題を修正

## コンテナ

***コンテナマップ***

- Container画面のActiveTX操作ルールの改善
- 詳細画面 > イベントNormal、Warningタイプのフィルタリング機能を追加
- 詳細画面 > カード上部のステータス別の数量の部分表示スタイルを改善

***Out of Memory Killedコンテナ*** - 照会時間範囲の選択オプションを削除およびテーブルソート機能を修正 

## ログ

***ログ設定*** - 現在の使用状況を表示する機能を追加 

## サイト管理者

***決済管理*** > ***プロジェクト有料変更*** - 製品種類、プラットフォームのカラムをマージ

***決済管理*** > ***請求管理***、***アカウント状態*** - 「open」から 「正常」にカラム名を変更

***決済管理*** > ***請求書*** - 「Sub Total」から「使用額」にカラム名を変更、顧客表示(設定)、顧客表示(状態)、割引情報表示カラムの非表示・表示メッセージの修正
