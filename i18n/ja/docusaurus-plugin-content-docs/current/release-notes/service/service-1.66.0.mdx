リリース：2022-03-23

## `共通`

**_クラウドモニタリング_** - Naver Cloudインストール機能の追加

**_グループ管理_** - グループ間でプロジェクトの移動の利便性向上

**_LogSink_** - ログフィルタリング機能の改善

## アプリケーション

**_マルチトランザクション_** - ブラウザのサイズ変更時のチャートの歪みを修正

**_イベント履歴_** - テーブル内の通知メッセージが長くなる場合に改行処理をします。

## サーバー

**_整備計画_** - プロジェクト整備計画の日次、週次、年次の繰り返し設定機能を追加 

**_ イベント設定_** - プロセスイベント設定時にPIDを入力してプロセス名を検索する機能を改善しました。

**_サーバーリスト_** - プロセスのグループ化を解除する機能を追加
