## `共通`

***Flexボード***

- 反応型フレックスボード(グリッド型でウィジェット配置) 機能を追加
- サーバープロセスCPU、Memory TopNウィジェットの追加
- サーバーリソースPlusテンプレートのウィジェット構成の変更、テンプレートを使用したユーザー作成ダッシュボードのフィルタリング適用バグの修正
- 統合Flexボードを他のユーザーと共有する場合、入力したメール情報を表示 

***エージェント設定*** - 新規エージェント設定メニューの追加

- アプリケーション、コンテナプロジェクト - エージェント設定 - 新規エージェントの設定リンクの提供
- 設定値入力機能の有用性を改善
- エージェントオプションの検索機能の強化
- JSONを使用したエクスポートおよび、インポート機能の追加
- エージェントの種類およびプロジェクト単位のエージェント設定(既定値) 編集機能の追加(プロジェクト修正権限がある場合)
- すべてのエージェント適用機能の改善

***プロジェクトリスト*** - プロジェクト別のカードビュースタイルの改善 

***メンバー管理*** - メンバー招待時のメール検証機能の改善

***MXQL実行機*** - テーブルデータ表現の修正

## アプリケーション

***エージェントのインストール*** - Goモニタリングライブラリの使用方法のリンクを修正

***マルチトランザクション*** - 日付選択オプションの追加

## サーバー

***保守プラン*** - メンバー修正権限の適用、画面エラーの修正

***サーバーエージェントの移動*** - 組織変更時のプロジェクト選択一覧の更新エラーを修正

## コンテナ

***コンテナマップ***

- Podカテゴリ選択時にポッド別のコンテナ情報を追加、コンテナステータス(ConState) フィールド選択機能の追加
- Kubernetesイベント参照機能の追加
- メトリクス別の説明ガイドポップアップの追加
- しきい値の設定時、チャートのしきい値の色に重みづけし、フィールド値が0の場合は正常状態として表示を変更

***マルチサービスヒットマップ*** - ウィジェットの配置をソート
