## `共通`

***組織管理*** - プロジェクトグループ間の移動機能の操作性の改善

***ログモニタリング***

- LiveTailフィルター機能の追加
- ログデータの保管サイクルオプションの追加
- ダークテーマの修正、デザインガイドの適用

***メトリクスチャート*** - CSV書き出し機能の修正

***メトリクスイベント設定*** - 条件オートコンプリート入力のタイプミスを修正

***プロジェクトリアルタイム通知*** - 通知発生時のサウンド再生機能の復旧

## アプリケーション

***ヒットマップ*** - チャートデータ検索時間の制限を基本15日に延長

***マルチトランザクション*** - 複数のプロジェクトの選択時にチャートが壊れる現象を修正

## サーバー

***異常値探知*** - カテゴリー選択オプションの追加、デザインガイドの適用

***異常値検出イベント設定*** - カテゴリ選択オプションの追加

***サーバー詳細*** - チャートデータの間隔を表示

## コンテナ

***メトリクスイベント設定*** - Kubernetesマスタープロジェクトの通知情報の表示

## データベース

***カウンタートレンド比較*** - 時間オプション変更時のチャート間隔の選択異常現象を修正

***カウンタートレンド*** - プロセス時間選択の異常を修正
