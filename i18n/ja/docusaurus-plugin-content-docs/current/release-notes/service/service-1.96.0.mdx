## Official Release：2023-02-22

### `共通`

#### 用語変更

<Status>Change</Status> <b>プロファイル</b> 用語を <b>トレース</b>に変更します。

#### Flexボード

-   <Status>Change</Status> <UI>Log RPS Board</UI> ダッシュボードテンプレートの修正

    -   データ参照期間のデフォルト値変更：リアルタイム30分→リアルタイム12時間
    -   **_Log RPS trend_**、**_サーバステータス_** ウィジェットのレイアウトの高さを修正
    -   **_4XX、5XX Count_** ウィジェットの補助テーブルのデータ単位設定
    -   一部のウィジェットデータの呼び出し間隔を5 秒から1分に変更

-   <Status>Feature</Status> <UI>アクティブトランザクションTopN</UI>ウィジェットを追加

-   <Status>Feature</Status> <UI>ヒットマップ</UI> ウィジェット：トランザクショントレーステーブルの <UI>URL</UI> カラム項目に <b>コピー</b>(<img src='/img/ico-copy.svg' alt='복사 아이콘' />) ボタンを追加

-   <Status>Feature</Status> <UI>ホーム画面</UI> > <UI>Flexボード</UI> メニューのダッシュボード一覧からダッシュボード共有(<img src='/img/ico-flexboard-share.svg' alt='공유 아이콘' />) 機能を追加

#### 分析

-   <Status>Feature</Status> <UI>メトリクスチャート</UI> メニューからウィジェット上部に時間マージとオブジェクトマージを表示されます。

-   <Status>Fixed</Status> <UI>サイトマップ</UI> > <UI>分析</UI> > <UI>メトリクス異常検出</UI> メニューからカテゴリ、フィールドを選択時の UIバグによる UIロジックを変更

-   <Status>Fixed</Status> <UI>キューブ</UI> メニューからで検索が終わっていない場合 <UI>開始日</UI>, <UI>対象メトリクス</UI>, のダッシュボードタブを変更できないように修正

#### ログの設定

-   <Status>Change</Status> <UI>カテゴリ別のデータ保持期間</UI> テーブルの保存量カラムを削除

-   <Status>Feature</Status> <UI>カテゴリ別データ保持期間</UI> テーブルで <UI>本日のログ数</UI> カラムの追加およびカラム別ヘルプの追加

-   <Status>Feature</Status> <UI>ログ構文解析の設定</UI> セクションでステータス成功率の構文解析、4xx、5xx構文解析の登録時に同じカテゴリの構文解析を重複登録を防ぐ有効性検査を追加

#### アカウント情報管理

-   <Status>Feature</Status> <UI>アカウント情報</UI> メニューで多要素認証を有効にする場合、最終確認のメッセージウィンドウを追加

#### 管理

-   <Status>Fixed</Status> <UI>メンテナンス計画</UI> メニューで登録したメンテナンス計画項目を削除できないバグを修正

-   **_管理_** > **_プロジェクトメンバー_** メニューから、次の機能を追加します。

    -   <Status>Feature</Status> プロジェクトメンバーの管理性の向上のために、メンバーの所属(<UI>グループ</UI>/<UI>プロジェクト</UI>) タイプを表示
    -   <Status>Feature</Status> メンバー削除(<img src='/img/ico-trash.svg' alt='삭제 아이콘'/>) ボタンを無効にする場合、ツールチップ(削除できない理由)を追加

-   **_使用履歴_**、**_決済管理_** メニューで次の機能を修正します。

    <Status>Change</Status> <UI>請求書プレビュー</UI> / <UI>使用量</UI> メニューからログ課金の変更に伴う、ログ表示単位および式単位の変更

    -   ログ表示単位の変更：LOGs ⇒ LOG UNITs
    -   ログ表示単位の変更：毎時間カウントしたログ行数の1日あたりの平均値の表示に変更
    -   <UI>請求書プレビュー</UI> メニューから <UI>課金集計量</UI> カラムのログ使用状況の表示を変更
    -   <UI>使用量</UI> メニューから使用状況の詳細のメータリング概要カラムを変更

#### 利用内訳

**_請求書プレビュー_** メニューに次の機能を追加します。

-   <Status>Feature</Status> プロジェクト、製品名、カラム別のグループ化ボタンを追加：グループ化したデータの時間合計、費用平均、請求費用の合計情報をプレビューで提供

-   <Status>Feature</Status> 請求書テーブルの上に <UI>製品種類</UI>、<UI>プラットフォーム</UI>、<UI>製品通貨</UI>、<UI>製品単価</UI>、<UI>適用為替</UI>、<UI>請求通貨</UI> カラムの表示有無を選択できるオプションを追加
