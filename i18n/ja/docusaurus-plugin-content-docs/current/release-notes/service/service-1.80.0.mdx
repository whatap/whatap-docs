リリース：2022-07-13

## `共通`

**_イベント設定_** - メトリクスイベントにイベントルールテスト機能を追加

**_レポート_** - レポートの再作成機能を追加

**_Flexボード_** - シリーズチャートメトリクスオプションにプロジェクトオプションを追加

**_ログ検索機能_** - スクローリングによる次ページの照会エラーの修正

**_Flexボード_** - お気に入りを追加する場合、間違ったボードが表示されるエラーを修正

## アプリケーション

**_ アプリケーション固定トポロジ / アプリケーショントポロジー_** - エージェントフィルター選択エラーの修正

## サーバー

**_サーバー詳細_** - チャート画像保存エラーを修正

## データベース

**_Mongo/Mysql/PostgreSQL_** - 「統計-データベースサイズ」メニューの追加

**_Cubrid_**

-   「統計-データベースサイズ」画面を追加
-   「統計-テーブルサイズ増減」画面を追加

**_SGAサイズ増減_** - 「SGAサイズ」とメニュー名を変更

**_テーブルスペースサイズ増減_** - 「テーブルスペースサイズ」とメニュー名を変更

**_Mongo/Redis_** - 「マルチインスタンスモニタリング」のカテゴリ選択のメトリクスプレビューエラーを修正

## コンテナ

**_共通_** - ヘッダーメニューによるネームスペースプロジェクト間の移動機能を追加

**_コンテナマップ_** - トレースチャートを追加

-   Apdex \* 100, Transaction Total Count, HTTP Call Error Count, SQL Error Count, Transaction Error Count, GC Old Generation Count

**_コンテナマップ_** - Kubernatesイベントにノードイベントタブを追加

**_コンテナマップ_** - 単一コンテナの基本情報照会機能を追加

**_コンテナマップ_** - 下部の追加情報(グループフィールド) を非表示にする機能を追加

**_コンテナマップ_** - しきい値選択オプションの場所を変更
