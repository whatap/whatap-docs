リリース：2022-07-13

## `共通`

***イベント設定*** - メトリクスイベントにイベントルールテスト機能を追加

***レポート*** - レポートの再作成機能を追加

***Flexボード*** - シリーズチャートメトリクスオプションにプロジェクトオプションを追加

***ログ検索機能*** - スクローリングによる次ページの照会エラーの修正

***Flexボード*** - お気に入りを追加する場合、間違ったボードが表示されるエラーを修正

## アプリケーション

*** アプリケーション固定トポロジ / アプリケーショントポロジー*** - エージェントフィルター選択エラーの修正

## サーバー

***サーバー詳細*** - チャート画像保存エラーを修正

## データベース

***Mongo/Mysql/PostgreSQL*** - 「統計-データベースサイズ」メニューの追加

***Cubrid***

- 「統計-データベースサイズ」画面を追加
- 「統計-テーブルサイズ増減」画面を追加

***SGAサイズ増減*** - 「SGAサイズ」とメニュー名を変更

***テーブルスペースサイズ増減*** - 「テーブルスペースサイズ」とメニュー名を変更

***Mongo/Redis*** - 「マルチインスタンスモニタリング」のカテゴリ選択のメトリクスプレビューエラーを修正

## コンテナ

***共通*** - ヘッダーメニューによるネームスペースプロジェクト間の移動機能を追加

***コンテナマップ*** - トレースチャートを追加

- Apdex \* 100, Transaction Total Count, HTTP Call Error Count, SQL Error Count, Transaction Error Count, GC Old Generation Count

***コンテナマップ*** - Kubernatesイベントにノードイベントタブを追加

***コンテナマップ*** - 単一コンテナの基本情報照会機能を追加

***コンテナマップ*** - 下部の追加情報(グループフィールド) を非表示にする機能を追加

***コンテナマップ*** - しきい値選択オプションの場所を変更
