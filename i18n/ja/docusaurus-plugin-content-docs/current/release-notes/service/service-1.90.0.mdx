## 2022-10-24

### コンテナマップ

<Status>Fixed</Status> ContainerカテゴリとPodカテゴリのヒットマップチャート表示される情報が異なる現象を修正

### `共通`

<Status>Fixed</Status> <UI>アカウント情報</UI> - 既存のパスワードが9文字以下の場合、新しいパスワードを変更できない問題を修正

## 2022-10-20

### ログ

-   <Status>Fixed</Status> 無料試用期間が画面に表示される現象を修正(NaN → 0)
-   <Status>Fixed</Status> ログモニタリングの非アクティブ化されたプロジェクトに入力する場合、コンテンツのない「ログ監視のアクティベーションガイダンスモーダル」の問題を修正
-   <Status>Fixed</Status> <UI>ライブTail</UI> メニューで構文解析されたキーを検索するAPIを呼び出さない現象を修正

### URL

<Status>Fixed</Status> URL一覧 - 名前変更ボタンの誤字(cancle → cancel) を修正

## 2022-10-18

### チャート

-   <Status>Feature</Status> Out of Memory メトリクスチャートの移動ロジックを追加 - 分析フローを生成のため、行をクリックするとメトリクスチャートに移動するよう修正
-   <Status>Fixed</Status> ヒットマップのダークテーマを修正し、画面処理を改善するよう修正

### リソース

-   <Status>Change</Status> <UI>リソース</UI> > <UI>マスターメタ情報</UI> > 「データなし」メッセージを <UI>マスターメタデータなし</UI>に修正

### コンテナマップ

-   <Status>Change</Status> 概要ビューウィンドウが開いた状態でステータス別のテキスト数が修正されます。
-   <Status>Change</Status> ログ検索タブで(ログカテゴリが存在する場合) ログ対象のカテゴリを修正
-   <Status>Fixed</Status> カテゴリがPodの場合にコンテナが複数あると、一部のコンテナ情報のみが表示されるバグを修正

### アプリケーションの現状/性能推移

-   <Status>Fixed</Status> エージェント情報が誤って表示されるエラーを修正

### ログ

-   <Status>Feature</Status> <UI>ライブTail</UI>, <UI>検索機</UI>、<UI>エクスプローラー</UI>にテーブルカラムの順序変更機能を追加
-   <Status>Change</Status> ログ設定のデータ保持期間オプションを修正

    -   トライアルプロジェクト：最大3日までデータを保持
    -   有料プラン変更プロジェクト：最大40日のデータを保持
    -   _front.conf_ ファイルに`ui_log_retention_limit`オプションを追加すると、その値までデータを保持するように修正

### Kubernetes

-   <Status>Change</Status> クラスタープロジェクトでヘッダーメニューをネームスペース移動時に、既存メニューを維持した状態で移動できるように修正
-   <Status>Change</Status> 性能サマリーのページウィジェットデータを修正

### `共通`

-   <Status>Change</Status> 照会および分析権限をオフできないように非アクティブ処理を修正
-   <Status>Fixed</Status>通知受信設定 - サードパーティーのSlackにredirect urlが正常に接続されないバグを修正
-   <Status>Fixed</Status> <UI>統合レポート</UI> メニューから初期入力時、時間選択の値が正常に初期化されないバグを修正
