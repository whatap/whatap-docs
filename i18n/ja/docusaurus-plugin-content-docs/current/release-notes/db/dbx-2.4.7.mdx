2025年03月18日

## MongoDB

<Status priority='0'>Fixed</Status>指標収集オプションv2のデフォルト値を`false`から`true`に変更

- 既存のデフォルト値: `v2=false` → 変更後のデフォルト値: `v2=true`

- `v2=true`設定時`serverStatus`値のうち`opcounters`、`globalLock`、`extra_info`、`connections`、`network`、`mem, opLatencies`、`asserts`のみ収集及び保存

- 追加指標の収集を希望する場合`category=network,repl,indexStats…`などの`serverStatus`出力オプションを指定して追加収集可能
