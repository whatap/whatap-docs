2025年06月10日

- <Status priority='1'>New</Status> シェルスクリプトを定期的に実行し、結果値を収集できる `shmon` 機能を追加

  - `xos.conf` に `shmon=1` を設定して使用します。

  - `agent` ディレクトリの下に `shmon` ディレクトリを作成し、その中に `.sh` ファイルを配置すると、定期的に実行されて結果を収集できます。

- <Status priority='1'>New</Status> AIXマシンで Oracle Pro 11.2.0.4、19.3 のサポートが可能になりました。

- <Status priority='1'>Feature</Status> script manager をスレッドで処理

- <Status priority='1'>Feature</Status> SQL トレース時に tkprof をスレッドで処理

- <Status priority='1'>Feature</Status> エージェントオプション trace_tkprof_path= を追加

- <Status priority='0'>Fixed</Status> イベント収集時に ‘event in waitclass Other’ の収集を除外（まれに大きな値が入る Oracle のバグあり）

- <Status priority='0'>Fixed</Status> planchange、plansave 時に 11g 以下で SQL エラーにより収集されない問題を修正
