2025年04月03日

- <Status priority='1'>New</Status>韓国語が文字化けした場合、コード変換処理オプションを追加

  - `codefrom=`及び`codeto=`文字エンコーディング指定

  - `iconv -l`コマンドの結果値の中から選択して指定

  - 例示: `codefrom=EUC-KR`、`codeto=UTF-8`

- <Status priority='1'>Feature</Status> `plan_elapse` `bind_elapse` デフォルト値を5秒 → 3秒に変更

- <Status priority='1'>Feature</Status> `sqlstat`伝送周期のデフォルト値を5分 → 1分に変更

- <Status priority='1'>Feature</Status> `sqlstat`に`!rectype`、`@id`、`@stime`を追加

- <Status priority='0'>Fixed</Status> 19 以降のバージョンでParallelquery情報が収集されない問題を修正

- <Status priority='0'>Fixed</Status> pdbが1つのとき、multi DBごとに指標が照会されない問題を修正

- <Status priority='0'>Fixed</Status>エージェントをリアルタイムで呼び出す`session list request/reply`に`result2`を追加

- <Status priority='0'>Fixed</Status> `temp tablespace`の数値が異常に出力される問題を修正

  - Oracleバグ(18c、19c)影響により、`distinct`で収集クエリロジックを修正

- <Status priority='0'>Fixed</Status> 5分データの生成が不要な`_no_5m_hour_`カテゴリーを整理

- <Status priority='0'>Fixed</Status> `plan_hash`値でプランをリアルタイムで照会する時に`child_number`がない場合、`timestamp`最新の値で照会するように修正
