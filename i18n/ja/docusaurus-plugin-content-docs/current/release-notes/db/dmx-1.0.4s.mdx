2025年05月21日

- <Status priority='1'>New</Status> sqlstatにwait情報を追加

  - Event別のwait time及びwait class別のwait timeを追加

  - Event名をdb_index_eventカテゴリーに1日2回転送

- <Status priority='1'>New</Status> Linux用、AIX用Oracle 10.2.0.5バージョンに対応

- <Status priority='1'>New</Status> ASM diskサイズ情報を追加

- <Status priority='1'>New</Status> Multi Script Manager機能を追加

- <Status priority='1'>New</Status> Plan情報にindexのtable名を追加

- <Status priority='1'>New</Status> SQL trace on/off機能を追加

- <Status priority='0'>Changed</Status> sessionにport情報を追加

- <Status priority='0'>Fixed</Status> sqlstat情報を修正

  - cpu_time, wait_time値に`/ 100`処理

  - include_txnsessionオプションを追加

  - Memory leak修整
