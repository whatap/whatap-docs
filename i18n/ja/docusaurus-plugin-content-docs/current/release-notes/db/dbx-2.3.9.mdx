2025年02月18日

## Tibero

- <Status priority='1'>Changed</Status> Plan情報を伝送する時、`parent_id`が`null`の場合`-1`で伝送 (20250220バージョンから適用)

- <Status priority='0'>Fixed</Status> Tibero 5(rev. 125305からrev. 127709まで) `v$session`のstatusカラム名のエラーを処理

## Oracle

- <Status priority='1'>Feature</Status>次のカテゴリに`!rectype`、`@id`フィールドを追加

  > `db_oracle_active_session`, `db_oracle*xos*disk_usage`

- <Status priority='1'>Changed</Status> Plan情報を伝送する時、`parent_id`が`null`の場合`-1`で伝送

- <Status priority='1'>Changed</Status> 次のカテゴリに対して5分統計を生成しないように変更

  > `db_oracle_active_session`, `db_oracle_xview`, `db_oracle_bind_save`, `db_oracle_lockinfo`, `db_oracle_plan_save`, `db_oracle_pq_tree`, `db_oracle_segments`, `db_oracle_tablespace`, `db_oracle_sga`
