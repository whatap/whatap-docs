2025年7月16日

### 共通

- <Status priority='2'>Feature</Status> Agentinfo にデータベース種別を示す `sub_type` フィールドを追加
- <Status priority='2'>Feature</Status> Autoscale エージェントからも `AgentinfoPack` を送信できるように対応
- <Status priority='1'>New</Status> シェルスクリプト実行をサポートする `shellmon` 機能を追加

### MySQL

- <Status priority='0'>Fixed</Status> MySQL 5.6 バージョンでアクティブセッション収集時に発生していたエラーを修正
- <Status priority='0'>Fixed</Status> slave DB が破損した場合、`node_state` の値を 2 から -2 に変更するように修正
- <Status priority='2'>Change</Status> `max_connection` 指標の収集間隔を 5 分に変更
- <Status priority='0'>Fixed</Status> タイムスタンプ形式によって slow query のテキストが収集されない問題を修正

### PostgreSQL

- <Status priority='0'>Fixed</Status> タイムスタンプ形式によって slow query のテキストが収集されない問題を修正
