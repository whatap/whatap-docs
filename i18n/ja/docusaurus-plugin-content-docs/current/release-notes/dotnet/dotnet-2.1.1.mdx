#### 2023年7月17日

- <Status>New</Status>Active Stackサポート

  <details>
  <summary>エージェント設定(whatap.conf)</summary>

  **active_stack_enabled** <span class='type'>Boolean</span>

  既定値 `true`

  アクティブスタックの追跡を有効にします。 **_分析_**>**_スタック_**メニューの**_トップスタック_**および**_ユニークスタック_**、**_アクティブスタック_**が該当します。

  </details>

- <Status>New</Status>ドットネットサポート環境に分析ツール機能の追加

  IISサーバーにインストールされているサイトを分析し、ドットネットエージェントのサポートが可能かどうかをより詳細に分析できる機能を追加しました。
