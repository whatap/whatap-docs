2025年04月26日

- <Status priority='1'>New</Status>インストールされるサーバーの.NETランタイムバージョンに従って自動的にバージョンが選択されるように修正

    :::note 

    適用バージョンを手動で変更するには、環境変数`WHATAP_DOTNET_TARGETFRAMEWORK`にご希望のバージョンを入力し、IISを再起動する必要があります。 選択できるバージョンは次のとおりです。

  - net45
  - net461
  - net462
  - net47
  - net472
  - net48

    :::

- <Status priority='0'>Fixed</Status> ISAPIフィルターベースのトランザクションでエラーが発生した場合も、ヒットマップに青色で表示されるバグを修正
