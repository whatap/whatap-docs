#### 2023년 7월 6일

-   <Status>New</Status> Web Service 설정 옵션 추가

    -   **webservice_method_enabled** <span class='type'>Boolean</span>

        デフォルト値`false`

        값을 `true`로 설정하면 Web Service를 추적할 수 있습니다. 설정한 값을 적용하려면 IIS를 다시 시작해야 합니다.

    -   **webservice_method_prefix** <span class='type'>String</span>

        추적 대상이 되는 메소드의 이름을 네임스페이스와 함께 입력하세요. Web Service는 직접 추적할 수 없기 때문에 Web Service를 사용한 메소드를 직접 추적 대상으로 입력해야 합니다. 설정한 값을 적용하려면 IIS를 다시 시작해야 합니다.

        次の例を参照してください。

        ```ini title='whatap.conf'
        webservice_method_prefix=Test.Demo, WebServiceDemo.WebService.
        ```

        :::caution

        -   조건에 맞는 메소드가 Web Service를 사용하지 않아도 추적 대상이 됩니다. 다만 Web Service를 사용하는 메소드가 아니면 에러가 발생해도 에러는 검출되지 않고 트랜잭션이 끝나지 않은 상태로 유지될 수 있습니다.

        -   메소드가 소속된 클래스까지만 입력하면 해당 클래스 내의 모든 메소드가 추적 대상이 됩니다.

        :::
