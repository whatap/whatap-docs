2024年07月24日

- <Status>Feature</Status> <Cmdname sid="side_cluster" className="uitext" /> &gt; <Cmdname sid="side_nodeDiskList" className="uitext" />メニューの追加

  WhaTap Kubernetesモニタリングの<Cmdname sid="side_nodeDiskList" className="uitext" />で各ノードにマウントされた個別ディスクの現在の使用量、余裕空間、性能指標をリアルタイムでモニタリングし、ディスクの状態を簡単に確認できます。 これにより、クラスターのディスク リソースを効率的に管理し、潜在的な問題を事前に把握して対応できます。

  :::note

  WhaTap Kubernetesエージェントの起動時にマウントポイントを持つディスクの一覧を表示します。

  - マウントポイントが存在しないディスクは、一覧に表示されません。

  - WhaTap KubWhaTap エージェントの起動後に追加されたディスクは、一覧に表示されません。

  :::

- <Status>Feature</Status> <Cmdname sid="side_workload" className="uitext" /> &gt; <Cmdname sid="side_kubeServiceList" className="uitext" />メニューの追加

  Kubernetesクラスター内で実行中のすべてのサービスの状態と情報を総合的に提供します。 WhaTap Kubernetesモニタリングの <Cmdname sid="side_kubeServiceList" className="uitext" />メニューにより、クラスター内のネットワーク構成を容易に把握し、各サービスの構成と現状を効率的にモニタリングできます。

:::info

Agent download: [kube_mon_1.7.8.tar](https://repo.whatap.io/docker/kube_mon_1.7.8.tar)

:::
