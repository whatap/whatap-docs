2024年09月11日

- <Status priority='1'>Feature</Status>タグカウント`kube_stat`のカテゴリにPod状態の指標を追加

  - `pending_pod`：待機中のPod数です。

  - `running_pod`：実行中のPod数です。

  - `succeeded_pod`：成功裏に完了したPod数です。

  - `failed_pod`：失敗したPod数です。

  - `unknown_pod`：状態の分からないPod数です。

- <Status priority='1'>Feature</Status>タグカウント`kube_node`のカテゴリに各ノードの生成時間を示す`creationTimeStamp` フィールドを追加

  `creationTimeStamp`フィールドを使用して、ノードが作成された時間を追跡できます。
