> **Java Agent v2.2.61 を含む**

2025年11月7日

- <Status priority='0'>Fixed</Status> **CPU 統計の改善**

    - **CPU 統計の負の値を防止**：カウンターリセットや競合状態により CPU メトリクスが負の値を示す可能性がある問題を修正。

        - 複数の CPU メトリクスに適用：`user`、`sys`、`nice`、`irq`、`softirq`、`steal`、`guest`、`guest_nice`

        - 計算された値が負の場合は 0 を返し、すべての CPU 統計が負の値にならないよう保証。

- <Status priority='0'>Fixed</Status> **Ingress バックエンドサービスポート処理**

    - **バックエンドサービスポート処理の改善**：Ingress 構成処理時のエッジケースを修正。

        - バックエンドサービスの数値ポートと名前付きポートを正しく処理。
