2024年12月24日

&nbsp; **Java v2.2.47**

<Status priority='1'>New</Status>新しいタグカウントメトリックを追加

- **kube_scheduler** 

  サンプリング周期`1分`

  Kubernetesコントロールプレーンコンポーネントであるスケジューラ関連メトリックを収集します。

- **kube_pv** 

  サンプリング周期`30秒`

  KubernetesストレージPersistent Volume(PV)関連メトリックを収集します。

- **kube_pvc** 

  サンプリング周期`30秒`

  KubernetesストレージPersistent Volume Claim(PVC)関連メトリックを収集します。

- **kube_pod_volume_list\{h1\}** 

  サンプリング周期`1時間`

  Kubernetes Podのボリュームマウント情報を収集します。

- **kube_container_volume_list\{h1\}** 

  サンプリング周期`1時間`

  Kubernetesコンテナのボリューム情報を収集します。
