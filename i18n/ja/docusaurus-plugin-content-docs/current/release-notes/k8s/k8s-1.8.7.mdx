> **Java Agent v2.2.61 を含む**

2025年05月08日

- <Status priority='1'>New</Status> `kube_pod`タグカウントカテゴリにPod状態及び所有情報を確認するためのタグカウント項目を追加

  - `kube_display_status`: Podの表示状態(Display Status)

- <Status priority='1'>New</Status> 既存の`kube_pod`の他に、ワークロードリソースに対するタグカウントを追加してタグカウントのリソース範囲を拡張

  - `kube_replicaset`
  - `kube_statefulset`
  - `kube_daemonset`

- <Status priority='1'>New</Status> `container_gpu` MIG環境に対応できるようにフィールドを追加

- <Status priority='0'>Fixed</Status> `server_disk`データが欠落していたバグを解決

:::
