2025年03月28日

&nbsp; **Java v2.2.55**

- <Status priority='1'>New</Status> `containerStdOut`ログに`podUid`タグを追加

- <Status priority='1'>New</Status> `kube_oom_event`タグカウントに`containerid`タグを追加

- <Status priority='1'>New</Status> MIG環境で`container_gpu`カテゴリーに`GPU_I_PROFILE`、`GPU_I_ID`、`DCGM_FI_DRIVER_VERSION`メトリクスを追加

- <Status priority='0'>Fixed</Status> GPU Compute Instanceが複数ある場合、使用中のGPUと使用されていないGPUが混在した環境で `container_gpu`タグカウントが収集されない現象を修正

- <Status priority='0'>Fixed</Status> GPUデータ収集時に収集時間が5秒単位でソートされない現象を修正
