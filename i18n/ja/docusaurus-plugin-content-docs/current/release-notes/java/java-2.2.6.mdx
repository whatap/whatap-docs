2023年6月1日

## 新機能 <sup><Status>New</Status></sup>

-   spring-boot追跡機能の追加

    -   resttemplate追跡

        spring-bootでRestTemplateを使用する場合、Http Clientドライバーに関係なくHttp Callを追跡できるように機能追加

    -   spring-cloud-gateway追跡

        spring-cloud-gatewayでfilterを使用する場合は追跡

    -   spring-kafka追跡

        ```ini title='whatap.conf'
        # spring-boot-2.7以降の場合
        weaving=tomcat9,spring-boot-2.7.x

        # spring-boot-3.0以降の場合
        weaving=tomcat10,spring-boot-3.x
        springboot_tomcat10_enabled=true
        ```
-   open-feign追跡

    -   feign-client-11追跡

        OpenFeign Clientを使用する場合、Http Clientドライバーに関係なくHttp Callを追跡できるように機能追加

        ```ini title='whatap.conf'
        # feign-client
        weaving=feign-11
        ```
