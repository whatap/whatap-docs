2023年10月10日

- <Status>Feature</Status>quarkus-reactive-1.13追跡

  _quarkus-resteasy-reactive-1.13.6.FINAL.jar_以降

  ```ini title='whatap.conf'
  weaving=quarkus-reactive-1.13
  ```

- <Status>Feature</Status>quarkus-reactive-2.10追跡

  _quarkus-resteasy-reactive-2.10.0.FINAL.jar_以降

  ```ini title='whatap.conf'
  weaving=quarkus-reactive-2.10
  ```

- <Status>Feature</Status>Spring Boot 2.5以降のWebfluxサービスにおけるアクティブスタックの追跡

- <Status>Feature</Status>トランザクションのthread id追跡

  ```ini title='whatap.conf'
  # default: false
  trace_thread_id_enabled=false
  ```

- <Status>Changed</Status>JAVAアプリケーションのメトリクス指標(<code>java_memory</code>, <code>java_gc</code>, <code>java_os</code>)収集の活性化

  ```ini title='whatap.conf'
  # default: true
  perfx_java_enabled=true
  ```

- <Status>Changed</Status>Java 20バージョンのsystem property値の対応

- <Status>Changed</Status>Java 6、Java 7、Java 8のASM apiバージョンをASM5からASM6に修正

- <Status>Fixed</Status>spring-boot-3.0追跡時に正しくないバージョンのログが表示されるバグの修正
