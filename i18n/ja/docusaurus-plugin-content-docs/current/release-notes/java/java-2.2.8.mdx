2023年6月9日

## 新機能 <sup><Status>New</Status></sup>

- io.asyncer追跡 (r2dbc-mysqlオープンソースドライバー)

  - spring-boot-2.7.xオプション、spring-boot-3.xオプションに含まれる

    ```ini title='whatap.conf'
    # spring-boot-2.7以降の場合
    weaving=tomcat9,spring-boot-2.7.x

    # spring-boot-3.0以降の場合
    weaving=tomcat10,spring-boot-3.x
    ```

  - spring-boot-2.7.x

    - io.asyncer:r2dbc-mysql-0.9.3

  - spring-boot-3.x

    - io.asyncer:r2dbc-mysql-1.0.2

## アップデート <sup><Status>Change</Status></sup>

- spring-boot-3.x非同期追跡時、オプション修正
  デフォルト値`false`から`true`に修正
  ```ini title='whatap.conf'
  #デフォルト値
  springboot_tomcat10_enabled=true
  ```
