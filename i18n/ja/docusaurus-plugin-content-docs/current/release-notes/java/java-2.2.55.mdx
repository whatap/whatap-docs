2025年03月27日

- <Status priority='1'>Feature</Status> `undertow-server 2.3.0`以降のバージョンでStatus Code収集機能を追加し、Exception追跡範囲を拡張

- <Status priority='1'>Changed</Status> `mtid`がない場合、traceparentヘッダーを生成しないように変更

- <Status priority='0'>Fixed</Status> `undertow-server 2.3.0`以降で`spring-webflux 6.0`以降を実装した場合、トランザクションデータが重複収集される現象を修正

  ```makefile
  ## example using spring-boot-3.4.3 on undertow-2.3.18.Final
  weaving=spring-boot-3.2,undertow-2.3.0  
  ```

- <Status priority='0'>Fixed</Status> `undertow-server 2.3.0`以降でJakarta Filterを呼び出すと、トランザクションデータが重複収集される現象を修正

- <Status priority='0'>Fixed</Status> `tomcat`または`embedded-tomcat`実行時`whatap.oname`を指定していない場合、収集サーバーに接続できない現象を修正

- <Status priority='1'>Deprecated</Status> `webflux`設定の対応を終了、既存設定は`spring-boot`設定に置き換え

  > `webflux-5.3` → `spring-boot-2.1` or `spring-boot-2.5` or `spring-boot-2.7`
