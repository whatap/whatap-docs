AWS Logプロジェクトで使用するAWSリソースを完全に削除するためには、次の3種類のリソースをすべて削除する必要があります。

- **Cloud Formation**で作成された**AWSリソース**
- **CloudWatch LogGroup**に追加された**Subscription filters**
- **S3 Bucket**に追加された**Event Notifications**

## CloudFormationで作成されたAWSリソースの削除

![cloudformationリソース削除](/img/aws-log-cloudformation-resource.png)

1. AWSコンソールで***CloudFormation***を選択します。

2. 削除する***Stack***を選択します。

3. ***Stack***によって作成されたすべてのリソースを削除するには***Delete***ボタンを選択します。

## CloudWatch LogGroup의 Subscription filters削除

![subscription filters削除](/img/aws-log-delete-subscription-filter.png)

1. AWSコンソールで***CloudWatch***を選択します。 

2. 購読した***LogGroup***を検索します。

3. ***LogGroup***の![number1](/img/number-01.png)***Subscriptionfilters***タブを選択します。 

4. ![number 2](/img/number-02.png) ***Subscription filters***リストから設定を選択します。 

5. ![number 3](/img/number-03.png) ***Delete***ボタンを押してから、削除してください。

## S3 Bucket의 Event Notifications削除

![event notifications削除](/img/aws-log-delete-event-notification.png)

1. AWSコンソールで ***S3***を選択します。 

2. 購読した***S3 Bucket***を検索します。

3. ***Buckets***の![number 1](/img/number-01.png) ***Properties***タブに移動してください。 

4. ***Properties***タブで設定された![number 2](/img/number-02.png) ***Event notifications***の履歴を選択します。 

5. ![number 3](/img/number-03.png) ***Delete***ボタンを押してから、削除してください。
