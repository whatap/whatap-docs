**クラッシュ統計メニュー**は、モバイルアプリで発生したクラッシュや例外を多様な基準で分析できるダッシュボードです。  
OS、アプリバージョン、デバイス、例外タイプなどでグループ化し、問題パターンを把握できます。  

- 上部エリア: 照会時間設定、共通フィルター  
- 下部エリア:  
  - 左側: グループ化の選択  
  - 右側: 選択したグループ化基準ごとの詳細画面（ロード数推移、クラッシュ推移グラフ、クラッシュ一覧テーブルを提供）  


## 主な構成

### グループ化の選択

照会結果をさまざまな基準でグループ化できます。  
各グループ化項目ごとのクラッシュ比率は**円グラフ**、時間別のクラッシュ発生件数は**棒グラフ**で表示されます。  
グループ化を適用しない場合、プロジェクト別のロード数推移グラフが表示されます。

| 項目         | 説明 |
| ------------ | ---- |
| アプリバージョン | アプリバージョン別のクラッシュ発生比率と推移 |
| OS           | OS別のクラッシュ発生比率と推移 |
| デバイス     | デバイス別のクラッシュ発生比率と推移 |
| 例外タイプ   | 例外クラス/タイプ別の発生比率と推移 |
| スレッド名   | 各スレッドで発生したクラッシュ比率の分析 |
| 例外メッセージ | 例外メッセージ別のクラッシュ発生比率と推移 |

**活用**: 特定基準に基づくクラッシュパターンの分析  

### クラッシュ一覧テーブル

照会されたクラッシュを詳細テーブルで提供します。  

| 項目       | 説明 |
| ---------- | ---- |
| グループ   | グループ化基準の値（例: iOS、Android） |
| 例外メッセージ | 発生した例外クラスおよびメッセージ |
| 件数       | 該当クラッシュの総発生回数 |
| アプリバージョン | クラッシュが発生したアプリバージョン |
| OS         | クラッシュが発生したOSの種類 |
| デバイス   | クラッシュが発生したデバイス |
| 例外タイプ | 例外の分類 |
| スレッド名 | 例外が発生したスレッド名 |
