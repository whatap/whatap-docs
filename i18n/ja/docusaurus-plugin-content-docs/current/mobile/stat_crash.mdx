크래시 통계 메뉴는 모바일 앱에서 발생한 크래시와 예외를 다양한 기준으로 분석할 수 있는 대시보드입니다.  
운영체제, 앱 버전, 디바이스, 예외 타입 등으로 그룹화하여 문제 패턴을 파악할 수 있습니다.  
상단 영역에서는 조회 시간, 필터링 기능을 공통으로 제공합니다. 


## 주요 구성

### 그룹화 선택

- **기능**: 운영체제, 앱 버전, 디바이스, 예외 타입 등으로 그룹화  
- **활용**: 특정 기준별 크래시 패턴 분석  

### 크래시 분포 (Pie Chart)

- **표시**: 그룹화 기준에 따른 크래시 건수 비율  
- **툴팁**: 마우스 오버 시 그룹명과 건수(%) 표시  
- **상호작용**: 클릭 시 해당 그룹만 필터링  

### 시간별 크래시 추이 (Bar Chart)

- **표시**: 시간대별 크래시 발생 건수  
- **단위**: 1시간 또는 자동 조정  
- **상호작용**  
  - 바 클릭 → 특정 시간대 강조  
  - 드래그 → 선택 구간으로 테이블 필터링  

### 크래시 목록 테이블

- 컬럼별 정렬  
- 텍스트 검색  
- CSV 다운로드 지원
