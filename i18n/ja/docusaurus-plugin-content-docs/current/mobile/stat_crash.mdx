クラッシュ統計メニューは、モバイルアプリで発生したクラッシュや例外を多様な基準で分析できるダッシュボードです。  
OS、アプリバージョン、デバイス、例外タイプなどでグループ化し、問題のパターンを把握できます。  
上部エリアでは共通の時間選択とフィルタリング機能を提供します。  

## 主な構成

### グループ化オプション

- **機能**: OS、アプリバージョン、デバイス、例外タイプなどでグループ化  
- **活用**: 特定基準ごとのクラッシュパターン分析  

### クラッシュ分布 (円グラフ)

- **表示**: グループ化基準に基づくクラッシュ件数の割合  
- **ツールチップ**: マウスオーバーでグループ名と件数(%)を表示  
- **インタラクション**: クリックすると該当グループのみフィルタリング  

### 時間別クラッシュ推移 (棒グラフ)

- **表示**: 時間帯ごとのクラッシュ件数  
- **単位**: 1時間または自動調整  
- **インタラクション**  
  - 棒をクリック → 特定時間帯を強調表示  
  - ドラッグ → 選択区間でテーブルをフィルタリング  

### クラッシュ一覧テーブル

- カラム別ソート  
- テキスト検索  
- CSV ダウンロード対応
