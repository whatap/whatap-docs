### チャットでのお問い合わせ {#invisible-chat}

> チャットの問い合わせアイコンが表示されません。

ユーザーが使用しているウェブブラウザの広告遮断プラグインまたは設定によりCrispに関連するJavaScriptのロードが遮断され、画面右上の![チャットアイコン](/img/ico-help.svg) **チャットでのお問い合わせ**アイコンが表示されない場合があります。 

![](/img/support-faq/whale-none-crisp-chat-icon.png)

<div className="faq-about-billing">

#### ウェールブラウザ{#chat-icon-whale}

[ウェールブラウザ](https://whale.naver.com)を使用している場合は、次のガイドに従ってウェブブラウザ オプションを設定します。

1. アドレスバーでロックボタンを選択します。

   <ImgLang img='support-faq/whale-prevent-trace-01.png' desc='' />

2. 追跡防止された項目があるか確認します。 **遮断された追跡機**を選択します。

3. 追跡防止されたコンテンツが**client.crisp.chat**か確認します。

   <ImgLang img='support-faq/whale-prevent-trace-02.png' desc='' />

4. 前の段階に戻り、追跡防止オプションを解除します。

   <ImgLang img='support-faq/whale-prevent-trace-03.png' desc='' />

5. ウェブブラウザで画面を更新します。

画面右上の![チャットアイコン](/img/ico-help.svg) **チャットでのお問い合わせ**アイコンが表示されていることを確認できます。

![](/img/support-faq/whale-visible-crisp-chat-icon.png)

#### クロームブラウザ{#chat-icon-chrome}

[クロームブラウザ](https://www.google.com/chrome)を使用している場合は、クロームで広告遮断プラグインの確認が必要です。 次の[リンク](https://help.crisp.chat/en/article/the-chatbox-does-not-work-or-appear-on-my-website-7v450s/)で、措置方法を参考に設定を変更します。

</div>

### 会員登録{#email-verification}

> 会員登録認証メールが来ません。 

認証メールは基本的にno-reply@whatap.ioから発送されます。

```
* **정상적으로 발송된 경우**, 메일 특성상 추적이 어렵습니다. 반송된 이력이나 메일 발송 여부는 확인할 수 있습니다. 인증 메일이 스팸 메일함으로 수신되지 않았는지 또는 외부망에서 수신되도록 허용되어 있는지 확인이 필요합니다. 

* **인증 메일을 받지 못하였거나 인증 과정에 문제가 있는 경우**, 로그인에 필요한 이메일 또는 인증 해제가 필요한 이메일에서 se@whatap.io로 계정 인증 해제 요청 메일을 전달합니다. 이메일 인증을 위해 본인 인증 절차를 진행한 후, 인증을 해제하고 안내해 드립니다.

<br/>  
<br/>
```

### ログインアカウント変更{#change-account}

> ログインアカウントを変更したいです。 

現在はアカウント変更機能を別途提供していません。 SaaS環境では、実際に使用するEメールで直接会員登録及びEメール認証を完了した後、アカウントが活性化されます。 Eメールの変更を希望する場合は**新しいEメール**で**会員登録**を進めてください。
