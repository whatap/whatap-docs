### 無料サーバープロジェクト利用時のデータ保存及び照会制限

> 過去の週間/月間報告書や以前の日付の日刊報告書が照会されません。

WhaTapサーバーモニタリングを無料で使用している場合、**データ保存期間が1日に制限**され、保存期間を超えた過去のデータは削除されます。
過去のデータを含むレポートを照会できますが、過去のデータはないように見えます。 また、削除されたデータはレポートだけでなく、すべてのメニューで照会できません。 

:::note

無料利用時に提供される項目は[WhaTapサイト > 料金案内](https://www.whatap.io/ko/pricing/)のサーバータブで確認できます。

:::
