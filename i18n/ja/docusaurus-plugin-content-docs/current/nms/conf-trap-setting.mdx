SNMP Trapは、ネットワーク機器で発生したイベントをNMSでリアルタイムで送信する機能です。 <Cmdname sid="side_trapSetting" className="uitext" />メニューでTrapメッセージを収集する時に必要な設定ができます。 デバイス別に設定したTrapポートとコミュニティ文字列情報を通じてトラップメッセージをNMSに送信します。

{/* <ImgLang img="nms-conf-trap-st.png" desc="Trap Setting" /> */}

- **Port**: SNMP Trapメッセージは、基本的にUDP162番ポートを通じて送信されます。 必要に応じて、別のポートを設定できます。

- **Community**: Trapメッセージを収集するために必要なコミュニティ文字列を設定します。 この文字列は、データへのアクセス権限を決定します。

- **Activate**: ![トグルアイコン](/img/ic-toggle-on.svg)活性化トグルボタンを使用して、その設定の活性化有無を選択します。
