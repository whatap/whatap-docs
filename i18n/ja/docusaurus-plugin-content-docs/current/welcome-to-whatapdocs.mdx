**WhaTapモニタリングサービス**は、様々なプラットフォームに対応する統合モニタリングを提供します。 数多くのデジタルソースからデータを収集し、ユーザーがシステムとその改善策を把握できるよう支援します。 

- ユーザーは、**WhaTapと**様々なデータを統合することができます。 WhaTapエージェントと拡張ツールを活用してデータを収集し、単一のUIでユーザー独自のデジタルエコシステムを構築することができます。 

- ユーザーは、**WhaTapで**データを分析することができます。 収集したデータに基づいて、カスタマイズ可能なダッシュボードおよび各種チャートまたは強力なクエリ言語を使用してトラブルの根本原因を特定し、システムを最適化できます。

- ユーザーは、**WhaTapを使用して、**システムトラブルに迅速に対応することができます。 トラブルの兆候を最初に検出し、トラブルが発生する前にユーザーに通知することができます。

### サービス

[製品](./#shortcut-product)メニューは、ユーザーが選択した製品に関するインストールから設定・活用指針まで統合案内文書のショートカットを提供します。 

- エージェントのインストール前に**サポート環境**を確認してください。 

- WhaTapモニタリングサービスを使用するためのエージェント**インストール**文書を参照してください。

- インストールチェック事項を確認し、インストール中に問題が発生した場合は、**インストールの問題解決**文書を参照してください。 

- エージェント設定オプションとアップデート、削除などの**管理**方法を確認してください。 

- <Cmdname sid='dashboard' className='uitext' />、<Cmdname sid='flexboard' className='uitext' />、<Cmdname sid='report' className='uitext' />、<Cmdname sid="event_statistic" className="uitext" />設定方法などを調べてみてください。

各製品別の文書は[Docsホーム画面](./)内の[製品](./#shortcut-product)のショートカットと固定された上部の**製品別文書のショートカット**メニューで確認できます。

### 管理

[管理](manage) メニューは、WhaTapモニタリングサービスのアカウント、プロジェクト、グループなどを管理する方法と決済方法を説明します。 

- セキュリティが強化されたWhaTapのマルチファクタ認証方式を確認してください。 

- WhaTapモニタリングサービスのモニタリング単位であるプロジェクトの階層構造と管理方法について説明します。 

- WhaTapモニタリングサービスのメンテナンス計画の管理方法についてご確認ください。 

- WhaTapモニタリングサービスの無料利用期間の15日が過ぎた場合、決済情報を確認し、有料サービスへの切り替えについての説明を確認してください。
