## イベント削除 {#remove-event}

基本的に提供されたイベントまたは新しく追加されたイベントを削除できます。 

1. イベント一覧から削除したいイベント項目の![修正アイコン](/img/ico-edit.svg)ボタンを選択します。 

2. イベント修正画面に移動し、画面右上の<Cmdname sid="delete" className="uitext" />ボタンを選択します。

3. 削除確認メッセージが表示されたら、もう一度<Cmdname sid="delete" className="uitext" />ボタンを選択します。

:::note

削除したイベントは復元できません。

:::
