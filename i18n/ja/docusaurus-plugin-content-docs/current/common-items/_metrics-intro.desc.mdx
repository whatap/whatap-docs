## メトリクスとは?

WhaTapは、モニタリング対象からデータを収集し、ユーザーに提供します。エージェントから収集されるデータを**メトリクス**と呼びます。 

<ImgLang img='metrics-overview.png' desc=''/>

**メトリクス**は、次の情報で構成されています。

- ***Category***：関連する指標をバインドする単位
- ***Tags***：収集対象を特定できる個別情報を含むデータ
- ***Fields***：エージェントから収集された指標
- ***Time***：メトリクスが収集された時間
- ***Oid***：メトリクスを収集したエージェントの固有番号
