## メトリクスウィジェット {#metric-widget}

<Cmdname sid="all_metrics" className="uitext" /> から追加できるウィジェットは二種類です。カテゴリのタグとフィールド情報を一覧するテーブル形式ウィジェットと、単一フィールドの推移を表示するトレンドグラフです。

:::note

**メトリクスとは**

プロジェクトで収集している数値の性能指標を **メトリクス** と呼びます。メトリクスは **カテゴリ** で分類され、カテゴリ別タグでデータを探索できます。メトリクスの詳細は[次のドキュメント](metrics/common-metrics-intro)を参照してください。

* **カテゴリ**: 関連指標をまとめる単位
* **タグ**: 収集対象を区別する固有情報を含むデータ
* **フィールド**: エージェントから収集した指標

:::

### テーブルウィジェット

選択したメトリクスカテゴリのデータを表形式で提供します。

  ![テーブルウィジェット](/img/metric-widget-table.png)

### シリーズウィジェット

選択したメトリクスカテゴリ内の選択フィールドのデータをシリーズチャートで提供します。

  ![シリーズウィジェット](/img/metric-widget-series.png)

:::note

メトリクスウィジェットの設定の詳細は、製品別 **Flex ボード > ウィジェット管理** ドキュメントを参照してください。

:::
