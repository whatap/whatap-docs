## メトリクスとは?

プロジェクトによって収集される数値のパフォーマンス指標を**メトリクス**と呼びます。メトリックは「カテゴリ」に分類され、カテゴリ別のタグを使用してデータを検索できます。

- **カテゴリ**(Category)：関連する指標をグループ化する単位
- **タグ**(Tags)：収集対象を識別できる固有情報が含まれたデータ
- **フィールド**(Fields)：エージェントから収集した指標

メトリクスの詳細については、[次の文書](metrics-intro)を参照してください。

## メトリクスウィジェットの種類

**_すべてのメトリクス_**で追加できるウィジェットタイプは、カテゴリのタグとフィールド情報を一覧表示するテーブル形式のウィジェット、１つのフィールドの推移グラフに分かれます。

- **テーブルウィジェット**：選択したメトリックカテゴリのデータを表形式で提供します。

  ![テーブルウィジェット](/img/metric-widget-table.png)

- **シリーズウィジェット**：選択したメトリックカテゴリの選択フィールドに、シリーズチャート形式でデータを提供します。

  ![シリーズウィジェット](/img/metric-widget-series.png)
