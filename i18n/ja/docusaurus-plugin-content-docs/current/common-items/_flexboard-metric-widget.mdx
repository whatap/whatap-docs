## メトリクスウィジェットの種類

<Cmdname sid="all_metrics" className="uitext" />で追加できるウィジェットタイプは、カテゴリのタグとフィールド情報をリストアップするテーブル形式のウィジェットと１つのフィールドに対する推移グラフがあります。

:::note

**メトリクスとは**

プロジェクトで収集されるパフォーマンス指標を**メトリクス**と呼びます。メトリクスは「カテゴリ」に分類され、カテゴリ別のタグからデータを検索できます。 メトリクスの詳細については、[次の文書](metrics-intro)を参照してください。

- **カテゴリ**(Category)：関連する指標をグループ化する単位
- **タグ**(Tags)：収集対象を識別できる固有情報が含まれたデータ
- **フィールド**(Fields)：エージェントから収集した指標

:::

### テーブルウィジェット

選択したメトリクスカテゴリのデータを表形式で提供します。

  ![テーブルウィジェット](/img/metric-widget-table.png)

### シリーズウィジェット

選択したメトリクスカテゴリの選択フィールドに対するデータをシリーズチャート形式で提供します。

  ![シリーズウィジェット](/img/metric-widget-series.png)

:::note

メトリクスウィジェットの設定に関する詳細は、製品別の **Flexボード > ウィジェット管理** ドキュメントを参照してください。

:::
