***フィルター***オプションを活用する時、希望する<InDoc pages="resource-node-list">ノード一覧</InDoc><InDoc pages="resource-pod-list">Pod一覧</InDoc><InDoc pages="resource-application-list">アプリケーション一覧</InDoc><InDoc pages="resource-container-list">コンテナ一覧</InDoc>を選択して照会できます。 `and`条件を使用すると、複数のフィルターが適用できます。

1. ![フィルターアイコン](/img/ico-filter.svg)***フィルター***入力ウインドウで![フィルター追加アイコン](/img/ico-add.svg)***追加アイコン***を選択すると、次のように***フィルターを追加する***ウィンドウが表示されます。

   <ImgLang img='k8s-resource-container-list-add-filter.png' desc='フィルター追加' />

   - ***フィルターキー***と***条件***を選択します。

   - 条件に合った値を選択します。

2. フィルター設定後、***適用***ボタンをクリックし、フィルターを適用します。

3. 複数のフィルターを追加するには、***フィルター***入力ウインドウまたは***フィルターを追加する***ウィンドウで![フィルター追加アイコン](/img/ico-add.svg)***追加アイコン***を選択します。
