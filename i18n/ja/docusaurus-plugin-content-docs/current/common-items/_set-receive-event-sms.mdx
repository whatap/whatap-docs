### SMS通知受信を追加する

SMS通知受信が必要な場合、<Cmdname sid="MNU07360" className="uitext" />でユーザーの電話番号を設定してください。 

<ImgLang img='set-event-sms-notification.png' desc='ユーザー電話番号' />

1. 画面の右上のプロフィールアイコンを選択します。 

2. ポップアップメニューが表示されたら<Cmdname sid="MNU07360" className="uitext" />ボタンを選択します。 

3. <Cmdname sid="account_user_phone" className="uitext" />セクションで<Cmdname sid="ITM06575" className="uitext" />ボタンを選択します。

4. <Cmdname sid="phone" className="uitext" />に認証番号を受信する電話番号を入力します。

5. <Cmdname sid="BTN06578" className="uitext" />ボタンを選択します。

6. ユーザーの携帯電話で6桁の認証番号を入力します。

7. <Cmdname sid="BTN06501" className="uitext" />ボタンを選択します。

<InDoc pages="account-manage">

:::note

- 登録した電話番号の変更についての詳細は[次の文書](#changecallnumber)を参照してください。

- SMSを通知で受信できる電話番号は、**韓国の携帯番号**のみ登録できます。

:::

</InDoc>
