- **Node**: [ノード詳細](node-details)、[オブジェクトマニフェスト](object-manifest)、[Nod要約分析](container-map-target-info#node-summary-analysis) 情報を確認し、[ノード一覧](resource-node-list) メニューに移動できます。

- **Deployment**: [Object Manifest](object-manifest) 情報を確認し、[Deployment リスト](deployment-list) メニューへ移動できます。

- **ReplicaSet**: [Object Manifest](object-manifest) 情報を確認できます。 

- **Pod**：[Pod開始解析](/kubernetes/pod-init-perform-v2)、[オブジェクトマニフェスト](object-manifest)、[Pod要約分析](container-map-target-info#pod-summary-analysis)情報を確認し[Pod一覧](resource-pod-list)メニューに移動できます。 

- **ContainerId**: **コンテナ情報**、[コンテナ要約分析](container-map-target-info#container-summary-analysis) 情報を確認し、[コンテナリスト](resource-container-list) メニューへ移動できます。

- **Image**: **コンテナイメージ**情報を確認し、[コンテナイメージ](container-images) メニューへ移動できます。 

- **ServiceName**: [Object Manifes](object-manifest) 情報を確認し、[Serviceリスト](service-list) メニューへ移動できます。 

- **Volume**：コンテナが使用するボリュームとマウント経路の情報を確認できます。

  - Podで使用する![指示線](/img/char-a.svg)ボリュームの名前と![指示線](/img/char-b.svg)ボリュームタイプを確認できます。 

    > 例、**Volume** ![指示線](/img/char-a.svg)`containerddomainsocket` ![指示線](/img/char-b.svg)`HostPath` 

  - **HostPath**ボリュームタイプを使用する場合![指示線](/img/char-c.svg)ホストシステムのファイルパスを確認できます。 このパスは、ファハドとホストシステム間のファイルシステムを接続するパスです。

    > 例、**Host Path** ![指示線](/img/char-c.svg)`/run/containerd/containerd.sock`
