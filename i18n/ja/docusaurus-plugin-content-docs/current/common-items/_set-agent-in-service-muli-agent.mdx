### 複数のエージェントに同時適用する

プロジェクトに所属する複数のエージェントに変更したオプションを同時に適用できます。

1. 画面右上の<Cmdname sid="applying_multiple_type" className="uitext" replace="ITM06005" /> チェックボックスを選択すると、各オプション項目にチェックボックスが表示されます。

2. 同時に適用するオプションのチェックボックスを選択します。複数選択できます。

3. 画面右上の<Cmdname sid="apply" className="uitext" />ボタンを選択します。

4. <Cmdname sid="applying_type" className="uitext" replace="ITM06005" />画面が表示されたら、変更したオプションを適用するエージェントを選択します。すべてを選択するには、<Cmdname sid="select_all" className="uitext" />チェックボックスを選択します。

   <ImgLang img='multiple-apply-agent.png' desc='' />

5. <Cmdname sid="apply" className="uitext" />ボタンを選択します。
