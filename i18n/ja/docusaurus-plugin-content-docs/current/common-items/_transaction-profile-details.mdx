## 詳細分析 {#details}

ホーム画面 &gt; プロジェクト選択 &gt; <Cmdname sid="side_analysis" className="uitext" /> &gt; <Cmdname sid="side_hitmap" className="uitext" />

<ImgLang img='tx-hitmap.png' desc='ヒットマップ' />

1. ![数字 1](/img/number-01.png) <Cmdname sid="time_selector" className="uitext" />で、ヒットマップを照会する期間を選択します。

2. ![数字1](/img/number-01.png)<Cmdname sid="option_type" className="uitext" />、<Cmdname sid="application" className="uitext" />から照会する対象を選択します。

3. ![検索アイコン](/img/ico-btn-search.png)ボタンを選択します。

4. ![数字2](/img/number-02.png)<Cmdname sid="hitmap_transaction" className="uitext" />から照会する領域を![数字3](/img/number-03.png)のようにドラッグします。

5. ![数字5](/img/number-05.png)<Cmdname sid="trace_list" className="uitext" />に一覧が表示されたら、分析するトランザクションを選択します。

トランザクションのパフォーマンス分析のためのクライアント情報などの属性、トランザクションの処理性能、各区間別の詳細な実行履歴などを確認できる<Cmdname sid="TTL06527" className="uitext" />画面が表示されます。

- <Cmdname sid="side_hitmap" className="uitext" />メニューの機能については、[次の文書](trs-view#details)を参照にしてください。

- アプリケーションダッシュボードのヒットマップウィジェットの詳細については、[次の文書](dashboard-hitmap-trace)を参照にしてください。

:::
