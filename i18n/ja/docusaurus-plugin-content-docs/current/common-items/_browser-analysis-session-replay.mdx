### セッションリプレイ {#TTL08103}

<InDoc pages="analyze-ajax-hitmap"><ImgLang img='browser-analysis-ajax-hitmap-session-replay.png' desc='セッションリプレイ' /></InDoc>

<InDoc pages="user-session-log-search"><ImgLang img="browser-analysis-user-session-log-details-session-replay.png" desc="セッションリプレイ" /></InDoc>

ユーザーがウェブサイトで行うすべてのイベントを記録し、再生できる機能を提供します。 <Cmdname sid="TTL08103" className="uitext" />機能を通じて、クリック、スクロール、入力、ページ切り替えなどのユーザー行動を再現できます。 これにより、ユーザーが実際にウェブサイトとどのように相互作用するかを、正確に把握できます。 

:::note

- エージェントオプションを通じて<Cmdname sid="TTL08103" className="uitext" />機能を適用する方法は、[次の文書](session-replay#set-agentoption)を参照してください。

- <Cmdname sid="TTL08103" className="uitext" />機能の使用についての詳細は、[次の文書](session-replay#usage)を参照してください。

:::
