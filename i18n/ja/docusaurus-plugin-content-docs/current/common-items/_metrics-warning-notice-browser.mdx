## テンプレート{#template}

次のようにアラート通知テンプレートを提供します。 

### メトリクスイベントテンプレート

**_アラート通知_** > **_イベント設定_** > **_メトリクス_**セクションで**_+ イベント追加_**ボタンを選択します。 **_テンプレート_**から希望する項目を選択します。

- **_ページロード時間の異常_**

  ページグループ(`${page_group}`)のページロード時間が10秒を超えたらアラート通知を送信します。

- **_ブラウザーのエラー数の異常_**

  ブラウザーエラーが10件を超えたらアラート通知を送信します。

- **_Ajax応答時間の異常_**

  AJAX Path(`${request_path}`)のAjax応答時間が30秒を超えたらアラート通知を送信します。

### 複合メトリクスのイベントテンプレート{#composite-metrics-template}

**_アラート通知_**>**_イベント設定_**>**_複合メトリクス_**セクションで**_+イベント追加_**ボタンを選択します。 **_テンプレート_**から希望する項目を選択します。

- **_分当りのページロード時間の異常_**

  ページグループ(`${page_group}`)の1分当りの平均ページロード時間が10秒を超えたらアラート通知を送信します。

- **_1分当りのページロード回数の異常_**

  ページグループ(`${page_group}`)のページロード数が100件を超えたらアラート通知を送信します。

- **_1分当りのルーター変更時間の異常_**

  ページグループ(`${page_group}`)の平均ルーター変更時間が10秒を超えたらアラート通知を送信します。

- **_分当りのAjax応答時間の異常_**

  AJAX Path(`${request_path}`)の1分当りの平均Ajax応答時間が30秒を超えたらアラート通知を送信します。

- **_分当りのAjaxリクエスト数の異常_**

  ホスト(`${request_host}`)の1分当りAjaxリクエスト数が1000件を超えたらアラート通知を送信します。

- **_分当りのAjax失敗件数の異常_**

  AJAX Path(`${request_path}`)の1分当りのAjaxの失敗件数が100件を超えたらアラート通知を送信します。

- **_分当りのブラウザーエラー数の異常_**

  ページグループ(`${page_group}`)の1分当りのブラウザエラー数が10件を超えたらアラート通知を送信します。
