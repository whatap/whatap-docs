### app_active_stat

* * *

アクティブトランザクションの区間指標です。

-   収集間隔：5秒
-   統計データ：5分、1時間

#### Tags

| タグ名          | 説明            | 備考    |
| ------------ | ------------- | ----- |
| container    | コンテナ名         | ユニーク値 |
| containerKey | コンテナキーの値      | ユニーク値 |
| host_ip      | Host IP       | ユニーク値 |
| okindName    | アプリケーションの種類名  | -     |
| oname        | エージェント名       | ユニーク値 |
| onodeName    | アプリケーションNode名 | -     |
| pid          | アプリケーションPID   | -     |
| type         | アプリケーションタイプ   | 言語名   |

#### Fields

| フィールド名 | 単位 | 説明                                 | 備考 |
| ------ | -- | ---------------------------------- | -- |
| dbc    | 件数 | Database Connectionを実行中のトランザクション件数 | -  |
| httpc  | 件数 | HTTP Callを実行するトランザクションの件数          | -  |
| method | 件数 | Methodロジックを実行するトランザクションの数数         | -  |
| socket | 件数 | Socket接続を実行するトランザクションの件数           | -  |
| sql    | 件数 | SQLクエリを実行するトランザクションの件数             | -  |

### app_counter

* * *

トランザクションの指標です。

-   収集間隔：5秒
-   統計データ：5分、1時間

#### Tags

| タグ名          | 説明            | 備考    |
| ------------ | ------------- | ----- |
| alias        | アプリケーションの別称   | ユニーク値 |
| container    | コンテナ名         | ユニーク値 |
| containerKey | コンテナキーの値      | ユニーク値 |
| host_ip      | Host IP       | ユニーク値 |
| okindName    | アプリケーションの種類名  | -     |
| oname        | エージェント名       | ユニーク値 |
| onodeName    | アプリケーションNode名 | -     |
| pid          | アプリケーションPID   | -     |
| type         | アプリケーションタイプ   | 言語名   |

#### Fields

| フィールド名          | 単位    | 説明                     | 備考                |
| --------------- | ----- | ---------------------- | ----------------- |
| active_tx_0     | 件数    | 3秒以下の区間トランザクション数       | -                 |
| active_tx_3     | 件数    | 3秒超過、8秒以下の区間トランザクション数  | -                 |
| active_tx_8     | 件数    | 8秒超過区間のトランザクション数       | -                 |
| active_tx_count | 件数    | 実行中のトランザクション数全体        | -                 |
| apdex_satisfied | 件数    | APDEX満足数               | -                 |
| apdex_tolerated | 件数    | APDEX許容数               | -                 |
| apdex_total     | 件数    | APDEXトランザクションの総数       | -                 |
| arrival_rate    | パーセント | トランザクションの要求率           | -                 |
| httpc_count     | 件数    | HTTP外部呼び出し数            | -                 |
| httpc_error     | 件数    | HTTP外部呼び出しのエラー数        | -                 |
| httpc_time      | ミリ秒   | HTTP外部呼び出しの平均時間        | -                 |
| metering        | コア    | アプリケーションが動作しているホストのコア数 | コンテナの場合、limit cpu |
| resp_time       | ミリ秒   | 平均応答時間                 | -                 |
| sql_count       | 件数    | 実行完了したSQL件数            | -                 |
| sql_error       | 件数    | SQLエラーの件数              | -                 |
| sql_fetch_count | 件数    | SQLFetchの件数            | -                 |
| sql_fetch_time  | ミリ秒   | SQLFetchの実行時間          | -                 |
| sql_time        | ミリ秒   | SQLの平均実行時間             | -                 |
| tps             | 件数    | 一秒あたりのトランザクション処理数      | -                 |
| tx_count        | 件数    | トランザクションの処理件数          | -                 |
| tx_dbc_time     | ミリ秒   | DB平均接続時間               | -                 |
| tx_error        | 件数    | トランザクションのエラー件数         | -                 |
| tx_httpc_time   | ミリ秒   | HTTPリクエストの平均時間         | -                 |
| tx_sql_time     | ミリ秒   | トランザクション別SQL実行時間合計の平均  | -                 |
| tx_time         | ミリ秒   | トランザクション実行時間           | -                 |

### app_host_resource

* * *

-   収集間隔：5秒
-   統計データ：5分、1時間

#### Tags

| タグ名          | 説明            | 備考    |
| ------------ | ------------- | ----- |
| alias        | アプリケーションの別称   | ユニーク値 |
| container    | コンテナ名         | ユニーク値 |
| containerKey | コンテナキーの値      | ユニーク値 |
| host_ip      | Host IP       | ユニーク値 |
| okindName    | アプリケーションの種類名  | -     |
| oname        | エージェント名       | ユニーク値 |
| onodeName    | アプリケーションNode名 | -     |
| pid          | アプリケーションPID   | -     |
| type         | アプリケーションタイプ   | 言語名   |

#### Fields

| フィールド名    | 単位    | 説明                  | 備考                               |
| --------- | ----- | ------------------- | -------------------------------- |
| cpu       | パーセント | ホストCPU使用率           | -                                |

| cpu_sys   | パーセント | ホストCPU SYS使用率       | -                                |
| cpu_usr   | パーセント | ホストCPU USER使用率      | -                                |
| cpu_wait  | パーセント | ホストCPU WAIT使用率      | -                                |
| disk      | パーセント | ルートファイルシステムのディスク使用率 | JAVAは設定として指定したファイルシステムパスのディスク使用率 |
| mem       | パーセント | ホストメモリの使用率          | -                                |
| swap      | パーセント | ホストSWAPの使用率         | -                                |
