## ![](/img/ic-templete.svg) グループウィジェット {#group}

複数のウィジェットを1つのグループにまとめて表示できるウィジェットです。

### グループウィジェットの配置{#addgroup}

1. <Cmdname sid="widget_template" className="uitext" />リストから<Cmdname sid="etc" className="uitext" /> &gt; <Cmdname sid="group" className="uitext" /> <Cmdname sid="widget" className="uitext" />を選択します。

2. 配置されたグループウィジェットの右上にある<Cmdname sid="project_selection" className="uitext" />ボタンを選択し、データを呼び出すプロジェクトを選択します。

3. グループウィジェットの右上にある<Cmdname sid="grid" className="uitext" />ボタンを選択してレイアウトを設定します。

4. <Cmdname sid="apply" className="uitext" />ボタンを選択してレイアウト設定を完了します。

5. グループウィジェットの右上にある<Cmdname sid="add_widget" className="uitext" />ボタンを選択します。

6. 画面の右側にある<Cmdname sid="widget_template" className="uitext" />リストが表示されたら、必要なウィジェットを選択して配置します。

:::note

- グループウィジェットでは、1つのプロジェクトのみを選択できます。

- ウィジェットは、追加された順番で配置されます。 ウィジェットの表示順を変更するには、ウィジェット名をクリックしたまま、希望の位置にドラッグします。

- グループウィジェットに配置できるウィジェットの数は最大36個です。 <Cmdname sid="grid" className="uitext" />オプションで設定した数以上のウィジェットを追加できません。

- 追加できるウィジェットの数をすべて配置すると、<Cmdname sid="grid" className="uitext" />ボタンは非活性化になります。

- グループウィジェットは、新しい画面を開いて確認できます。 <Cmdname sid="view_mode" className="uitext" />画面を開き、ウィジェット右上にある![ポップアップアイコン](/img/ic-popup.svg)ボタンを選択します。

:::

## 情報{#infomation}

### ![](/img/ic-text-widget.svg)テキスト{#text}

メトリクスの数値をテキスト形式で表示するウィジェットです。

![テキストウィジェット](/img/fb-text-widget.png)

ウィジェットの右上にある詳細(![アイコン](/img/ico-more.svg))ボタンを選択した後、![カラムアイコン](/img/ic-column.svg)ボタンをクリックして、必要なデータを追加したり、除外したりして表示順を変更できます。

### ![](/img/ic-vital-chart.svg)要約情報{#overview}

プロジェクトに要約情報や特定のメトリクスの説明を案内するウィジェットです。

![プロジェクト要約](/img/fb-project-overview-widget.png)

:::note

選択したプロジェクトの製品の種類によって、表示される要約情報は異なる場合があります。

:::

## その他

### ![](/img/ic-note.svg) <Cmdname sid="rich_text" /> {#richtext}

テキストを入力して、ウィジェットで配置できます。

![リーチテキスト](/img/fb-rich-text-widget.png)

ウィジェットの右上にあるオプションで、テキストの修正や書式の設定などができます。

- ![カラーアイコン](/img/ic-colors.svg) : <Cmdname sid="edit_widget" className="uitext" />オプションでウィジェットの背景と線、フォントの色を設定できます。 設定を完了した後、<Cmdname sid="save" className="uitext" />ボタンを選択します。

- ![修正アイコン](/img/ic-edit.svg) : 画面の右側に <Cmdname sid="rich_text" className="uitext" />画面が表示され、テキストを追加または修正できます。 また、提供されるツールバーを使用して書式を設定できます。

### ![](/img/ic-tool.svg) メンテナンス計画

選択したプロジェクトのメンテナンス計画の履歴を確認できるウィジェットです。

![メンテナンス計画](/img/fb-maintenance-plan-widget.png)

:::note

- 1時間以内に予定されているメンテナンス計画のみがウィジェットに表示されます。

- <Cmdname sid="side_maintenancePlan" className="uitext" />メニューの詳細は[次の文書](../management/maintenance)を参照してください。

:::
