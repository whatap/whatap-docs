### グループウィジェットの配置{#addgroup}

1. <Cmdname sid="widget_template" className="uitext" />一覧で<Cmdname sid="etc" className="uitext" /> &gt; <Cmdname sid="group" className="uitext" /> <Cmdname sid="widget" className="uitext" />を選択します。

2. 配置されたグループウィジェットの右上にある<Cmdname sid="project_selection" className="uitext" />ボタンを選択して、データを呼び出すプロジェクトを選択します。

3. グループウィジェットの右上にある<Cmdname sid="grid" className="uitext" />ボタンを選択して希望するレイアウトを設定します。

4. <Cmdname sid="apply" className="uitext" />ボタンを選択してレイアウト設定を完了します。

5. グループウィジェットの右上にある<Cmdname sid="add_widget" className="uitext" />ボタンを選択します。

6. 画面の右側に<Cmdname sid="widget_template" className="uitext" />一覧が表示されたら、希望するウィジェットを順番に選択します。

:::note

- グループウィジェットでは、1つのプロジェクトのみを選択できます。

- ウィジェットは、追加した順に配置されます。 ウィジェットの順序を変更するには、ウィジェット名の部分をクリックした状態で、希望の位置にドラッグします。

- グループウィジェットに配置できるウィジェットの数は最大36個です。 <Cmdname sid="grid" className="uitext" />オプションで設定した数以上のウィジェットを追加できません。

- 追加できるウィジェットの数をすべて配置すると、<Cmdname sid="grid" className="uitext" />ボタンは非活性化になります。

- グループウィジェットは、新しい画面を開いて確認できます。 <Cmdname sid="view_mode" className="uitext" />に進入してウィジェット右上に![ポップアップアイコン](/img/ic-popup.svg)ボタンを選択します。

:::
