### テーブルビュー {#anlaysis-trace-table-view}

<Cmdname sid="TTL06126" className="uitext" />タブでは、トランザクションの実行過程を時間順に確認できます。

<InDoc product='php,nodejs,golang'>

<ImgLang img="analysis-tx-table-view-mode-none-actx.png" desc="テーブルビュー" />

</InDoc>

- トランザクションを実行区間別に分類し、ダイアグラムで確認できます。 全経過時間のうち、各区間別に所要された時間、最も長くかかった区間を素早く把握できます。

<Xclude product='php,nodejs,golang'>

- ダイアグラムの各区間を選択すると、そのステップが位置するテーブルの一覧に移動します。

  <ImgLang img="analysis-tx-table-view-select-stack.png" desc="ステップ選択" />

</Xclude>

<InDoc product='php,nodejs,golang'>

- ダイアグラムの各区間を選択すると、そのステップが位置するテーブルの一覧に移動します。

  <ImgLang img="analysis-tx-table-view-select-stack-none-actx.png" desc="ステップ選択" />

</InDoc>

<Xclude product='php,nodejs,golang,kubernetes'>

- 青い丸(![アクティブスタックアイコン](/img/ic-c-tx-active-stack.svg))が位置する領域は、アクティブスタックが収集された時点です。 青い丸を選択すると、<Cmdname sid="active_stack" className="uitext" />ボタンがあるテーブル一覧に移動できます。 <Cmdname sid="active_stack" className="uitext" />ボタンをクリックすると、その区間で行われたステップ情報を確認できます。

  :::note

  - トランザクション一覧から![アクティブスタックアイコン](/img/ic-s-active-stack.svg)アイコンが表示された項目で確認できます。

  - アクティブスタックの詳細については、[次の文書](analysis-apm#active-stack)を参照してください。

  :::

</Xclude>

<InDoc product='kubernetes'>

- 赤い丸(![アクティブスタックアイコン](/img/ic-c-tx-active-stack.svg))が位置する領域は、アクティブスタックが収集された時点です。 赤い丸を選択すると、<Cmdname sid="active_stack" className="uitext" />ボタンがあるテーブル一覧に移動できます。 <Cmdname sid="active_stack" className="uitext" />ボタンをクリックすると、その区間で行われたステップ情報を確認できます。

  :::note

  - トランザクション一覧から![アクティブスタックアイコン](/img/ic-s-active-stack.svg)アイコンが表示された項目で確認できます。

  - アクティブスタックの詳細については、[次の文書](../java/analysis-apm#active-stack)を参照してください。

  :::

</InDoc>

- ![カラムアイコン](/img/ico-column.svg)<Cmdname sid="select_column" className="uitext" />：テーブル一覧にカラムでメモリ累積情報およびCPU累積情報を追加または非表示できます。 カラム情報は次のとおりです。

  - <UI>No</UI>：ステップの実行順序

  - <Cmdname sid="TTL06416" className="uitext" />：各ステップの開始時刻

  - <Cmdname sid="gap_time" className="uitext" />：直前ステップの開始時刻から現在のステップに移るまでの待ち時間、外部要因により遅延される場合、経過時間と異なる場合があります。

  - <Cmdname sid="TTL06375" className="uitext" />：各メソッドの開始から終了までの所要時間

  - <Cmdname sid="TTL06417" className="uitext" />：該当ステップの詳細な実行内容

- ![ダウンロードアイコン](/img/ico-download.svg)<UI>TXT</UI>：トランザクションの基本情報および区間ごとの実行情報をTXTファイルでダウンロードできます。

- ![ダウンロードアイコン](/img/ico-download.svg)<UI>SQL</UI>：トランザクションの基本情報およびSQL実行情報をTXTファイルでダウンロードできます。 <ScrollToLink anchor='downloadsql' />

  - <Cmdname sid="ITM08351" className="uitext" />: バインド変数値を元の値に置き換え、SQL統計データをダウンロードします。 セキュリティキー(`paramkey`)が設定されている場合は、セキュリティキー入力画面が表示されます。 セキュリティキーを入力しないとダウンロードできません。 セキュリティキーに関するエージェント設定の詳細については、[次の文書](agent-transaction#profile_http_parameter)を参照します。

  - <Cmdname sid="ITM08352" className="uitext" />: バインド変数値を置き換えない状態でSQL統計データをダウンロードします。

- ![拡大アイコン](/img/ic-outside.svg):トランザクションの要約情報とダイアグラムを非表示にし、テーブル一覧のみを確認できます。 ![縮小アイコン](/img/ic-inside.svg)ボタンをクリックすると、隠された情報を再表示します。 テーブル一覧が長い場合は、この機能を利用すると便利です。

- SQLステップを選択すると、パラメータを照会できる<UI>SQL</UI>画面が表示されます。 詳細については、[次の文書](#transaction-parameter)を参照してください。
