### ユーザーカスタム項目のモニタリング{#custom}

特定のトランザクション、ドメイン、IPを<Cmdname sid="custom" className="uitext" />タブに追加して、応答時間の分布図チャートで選択した項目だけを確認できる機能を提供します。 これにより、ユーザーは問題が発生する特定のアイテムを効率的にモニタリングして分析できます。

1. <Cmdname sid="custom" className="uitext" />タブをクリックします。

2. <Cmdname sid="transaction" className="uitext" />または<Cmdname sid="domain" className="uitext" />、<Cmdname sid="ip" className="uitext" />項目は直接入力、<UI>Top 30</UI>一覧内の選択、項目検索が可能です。

3. 追加した項目をクリックします。 

   <ImgLang img="txmap-custom-filter-select.png" desc="カスタム" />

選択した項目の応答時間の分布図が、右のチャートにリアルタイムでフィルタリングされることを確認できます。

:::note

追加したカスタム項目を削除するには、![削除アイコン](/img/ico-trash.svg)ボタンをクリックしまます。

:::
