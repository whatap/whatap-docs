WhaTapモニタリングサービスは、円滑なサービス提供と安定した運用、異常の監視と特定な情報を提供します。対象APIに接続して、すべての指標とクラウドイベントをリアルタイムで収集します。収集したデータをダッシュボードからリアルタイムに確認し、トラブルの原因を直接分析することができます。

ホーム画面からプロジェクトを選択してください。**_プロジェクトメニュー_**サブカテゴリは次のように構成されています。

- **項目別ダッシュボードメニュー**

  可視化されたチャートでプロジェクト全体のステータスを一目で確認できます。また、モニタリングされているリソースの合計量を確認し、対象リソースをフィルタリングすることができます。現在のサービスやシステムの正確な状態を直感的に把握できます。

- **Flexボード**

  基本的に提供されるアイテム別のダッシュボードに加えて、ユーザーは必要なさまざまなダッシュボードを構成できます。ウィジェットを追加し、サブチャートを利用できます。

- **メトリクス**

  メトリクスデータを使用した統合分析機能を提供します。各カテゴリーのメトリクスフィールドデータの傾向分析機能を提供する**_メトリクスキューブ_**、単一のメトリクス照会機能を提供する**_メトリクスチャート_**などを提供します。

  - **_メトリクスキューブ_**
  - **_メトリクスチャート_**
  - **_メトリクス異常検知_**

- **アラート通知**

  メトリクベースのイベント設定と照会機能を提供します。 

  - **_イベント設定_**
  - **_イベント受信設定_**
  - **_イベント履歴_**
