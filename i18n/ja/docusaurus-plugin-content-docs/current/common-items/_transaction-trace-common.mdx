-   **trace_normalize_enabled** <span class='type'>Boolean</span><br/>
    デフォルト値 `true`<br/>トランザクションURLを構文解析して正規化する機能を活性化します。

    :::note

    `false`に値を変更すると、パスパラメータの構文解析を非活性化します。この場合、統計データの意味が弱まるため、デバッグ用途に限ってしばらく使用することをお勧めします。

    :::

-   **trace_auto_normalize_enabled** <span class='type'>Boolean</span><br/>
    デフォルト値 `true`<br/>トランザクションURLを正規化するとき、パターン値をアノテーションから抽出して自動的に構文解析する機能を活性化します。

-   **trace_normalize_urls** <span class='type'>String</span><br/>
    正規化するトランザクションURLパターンを設定します。呼び出しURLパターンを構文解析してパスパラメータを削除します。

    :::note例）**/a/{v}/b**と宣言すると、**a/123/b** → **a/{v}/b**に置き換えます。複数登録するときは、コンマ（,）を区切り文字として使用します。置換パターンの整理後は、補完が必要です。:::

-   **web_static_content_extensions** <span class='type'>String</span><br/>
    デフォルト値 `js, htm, html, gif, png, jpg, css, swf, ico`<br/>静的クコンテンツであることを判断する拡張子を設定します。このオプションに設定した拡張子を持つトランザクションは、トレースの追跡とカウントから除外されます。

-   **trace_transaction_name_header_key** <span class='type'>String</span><br/>
    設定したhttp header keyから抽出した値をトランザクション名の最後に追加します。
