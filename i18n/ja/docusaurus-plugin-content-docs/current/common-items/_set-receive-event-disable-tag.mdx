### イベント受信タグを解除する

1. <Cmdname sid="alert_receive_settings_by_member" className="uitext" />セクションのユーザー一覧の一番左の<LinkImage img="ico-add.svg" desc="詳細アイコン" />ボタンを選択します。

2. <Cmdname sid="event_tag" className="uitext" />ポップアップウィンドウが表示されたら、適用されたタグの横の<LinkImage img="ico-close.svg" desc="閉じる" />ボタンを選択します。

3. <Cmdname sid="event_tag" className="uitext" />ポップアップ画面を閉じます。
