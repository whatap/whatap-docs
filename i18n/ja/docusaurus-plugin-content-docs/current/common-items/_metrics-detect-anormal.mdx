홈 화면 > 프로젝트 선택 > **_사이트맵_** > **_분석_** > **_메트릭스 이상 탐지_**

多様なメトリクス指標のパターンをAIが学習した予想パターンと比較することができます。予想パターンから外れた異常検知をグラフチャートで確認できます。過去のデータに基づいて繰り返されるパターンを確認し、今後の指標値予測に活用できます。

:::note

-   画面に配置したウィジェットは、他のメニューに移動する場合、保存されずに初期化します。
-   패턴 표시와 이상치 표시 기능을 제외하면 **_분석_** > **_메트릭스 차트_** 메뉴와 유사합니다.

:::

## ウィジェットを確認する

![메트릭스 이상 탐지](/img/metrics-detect-abnormal.png)

-   ![数字1](/img/number-01.png)明るい色のグラフ領域は、AIが分析した予想パターンです。 

-   ![数字2](/img/number-02.png) <span class='normal'>青色</span>グラフは、プロジェクトのメトリクス指標の推移です。

-   AIが分析した予想パターンから外れると<span class='slow'>オレンジ</span>、<span class='vslow'>赤色</span>の段階としてグラフに点を表示します。予想パターンの範囲を大きく外れた値を<span class='vslow'>赤色</span>として表示します。

## ウィジェットを配置する

![메트릭스 이상 탐지](/img/metrics-detect-abnormal-set-widget.png)

1.  ![数字1](/img/number-01.png) **_時間_**で、目的の時間間隔を設定してください。最大1か月間隔まで設定できます。

2.  ![숫자 2](/img/number-02.png)에서 메트릭스 지표의 범위를 선택하세요.

3.  아래 목록에서 모니터링하길 원하는 메트릭스 지표를 선택하세요.

選択したメトリクス指標を画面の右側に配置します。

:::note

-   モニタリング対象を選択し、メトリクス指標を区分して確認するには、![数字3](/img/number-03.png) **_タグ_**を選択してください。画面の右側にタグ選択一覧が表示されます。項目を選択した後、ウィジェットを追加してください。
-   画面に配置したウィジェットは、**_時間_**または**_タグ_**値を変更してもチャートに反映されません。
-   画面に配置したウィジェットを削除するにはウィジェット右上の![削除アイコン]\(/img/ico-remove.svg）ボタンを選択してください。
-   ウィジェットの左上を選択した状態でドラッグすると、ウィジェットの位置を変更できます。
-   ウィジェットの右下を選択した状態でドラッグすると、ウィジェットのサイズを調整できます。

:::

## 関連メトリクスを確認する

![메트릭스 이상 탐지](/img/metrics-detect-anormal-details.png)

ウィジェットでチャートの一部の領域をドラッグしてください。選択した領域で、該当メトリクス指標と関連した指標を一緒に確認することができます。
