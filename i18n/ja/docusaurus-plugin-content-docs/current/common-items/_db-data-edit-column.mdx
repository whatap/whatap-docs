テーブルのカラム項目を表示または非表示にしたり、順序を変更したりできます。

1.  テーブルの右上の![カラムアイコン](/img/ico-column.svg)ボタンを選択してください。 

2.  **_コラムの編集_**ウィンドウが表示されたら、テーブルのカラムを表示する順序を非表示、追加または、変更してください。

    -   カラム項目を選択し、[左方向アイコン](/img/left-arrow.svg)または![右方向アイコン](/img/right-arrow.svg)、![追加アイコン](/img/ico-add.svg)または![抜きアイコン](/img/ico-minus.svg)ボタンを選択してください。**_表示_**または**_非表示_**へ移動します。 
    -   カラム項目を選択した状態でドラッグすると、位置を移動できます。

3.  設定を完了した後、**_保存_**ボタンをクリックしてください。

:::note

設定した内容はブラウザのクッキー値として保存され、ページの更新後もステータスを維持します。クッキーの削除などの理由により設定間にエラーが発生した場合は、初期化されます。

:::
