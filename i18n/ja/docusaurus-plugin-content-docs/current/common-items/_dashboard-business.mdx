## ホーム > プロジェクト選択 > <Cmdname sid="dashboard" className="uitext" /> > **ビジネスダッシュボード**

**ビジネスダッシュボード**は、URLパターンやパスを基に定義されたビジネス単位（App Context）ごとに、アプリケーションのパフォーマンスをモニタリングできます。  
リアルタイムおよび過去のパフォーマンスデータを確認・分析することが可能です。  
TPS、応答時間、エラー率などの主要指標をビジネス単位で確認することで、特定業務やサービスのパフォーマンス状態を迅速に把握できます。

## 画面構成

<ImgLang img='apm-dashboard-business.png' desc='ビジネスダッシュボード' />

### 時間の選択

画面左上の時間セレクターで、データを確認する時間範囲を設定できます。  
その右側には、プロジェクトに接続されたエージェントの状態を確認でき、モニタリング対象サーバーの稼働状況を即座に把握できます。

- **リアルタイムモード**: 5秒ごとに自動更新され、直近10分間のデータを表示します。
- **過去データモード**: 最大1か月前までのデータを照会できます。
- **データ集計単位**:
    - 3時間未満: Rawデータ使用
    - 3時間～3日: 5分統計データ使用
    - 3日以上: 1時間統計データ使用

:::note

時間セレクターの使用方法の詳細は[次の文書](usage-ui#timeselector)をご覧ください。

:::

### エージェントの選択

時間セレクターの下には、プロジェクトに属するエージェント一覧が表示されます。  
エージェントをクリックすると青いタグが付き、そのエージェントまたはグループに関連するデータのみをフィルタリングして表示できます。

:::note

エージェント選択の詳細については[次の文書](dashboard#select-agent)をご覧ください。

:::

### 表示モードの切り替え

ビジネスダッシュボードは、好みに応じて2つのビューを切り替えてデータを確認できます。  
画面左上のアイコンをクリックして、以下のビューに変更できます。

* ![リストアイコン](/img/ic-list.svg) **テーブルビュー**  
  全ビジネス項目を表形式で一覧表示し、比較やソートが容易です。  
  全体状況を素早く把握する際に有効です。

* ![カードリストアイコン](/img/ic-card-list.svg) **カードビュー**  
  各ビジネス項目をカード形式で視覚的に表示します。  
  表示列は1～4列まで選択でき、画面サイズや環境に応じて柔軟に調整可能です。  
  特定ビジネスの状態を個別に確認したり、ダッシュボード用途での活用に適しています。

### 業務名の検索

検索エリアで業務名を入力し、特定ビジネスを素早く探せます。

### CSV ダウンロード

画面右上の検索エリア横にある ![CSVダウンロードアイコン](/img/ic-csv-download-ms.svg) をクリックすると、現在表示中のすべてのビジネスデータをCSVファイルとしてダウンロードできます。

### ビジネスパフォーマンス状況

各ビジネス単位ごとに、以下の主要パフォーマンス指標を確認できます。  
これらの指標はリアルタイム情報だけでなく、指定された時間範囲における累積統計も含まれます。  
**アクティブトランザクション**、**応答時間**、**エラー率** はチャートでも視覚的に確認できます。

- **アクティブトランザクション**: 現在処理中のトランザクション数をスピードメーター形式で表示し、リアルタイムの負荷状況を直感的に把握できます。
- **TPS（1秒あたりのトランザクション数）**: システム処理能力を示します。
- **トランザクション数**: 照会期間中に収集されたトータル数で、全体的なリクエスト規模を把握できます。
- **トランザクション/min**: 1分あたりの平均トランザクション数で、時間帯ごとのトラフィック傾向を分析するのに役立ちます。
- **応答時間**: 平均および最大応答時間を表示し、時間による変動をチャートで確認可能です。
- **エラー数**: 照会期間中に発生したエラーの総数です。
- **エラー率（%）**: 総トランザクション数に対するエラー発生の割合を示します。
- **エラー/min**: 1分あたりの平均エラー数です。

## ビジネス詳細分析

### トップトランザクション統計照会

テーブルビューやカードビューで詳細アイコンをクリックすると、選択したビジネスの上位30件のトランザクション統計が画面右のドロワーに表示されます。

:::tip

詳細ビューの開き方:

- テーブルビュー: 最左列の ![検索アイコン](/img/ic-pageview.svg) をクリック
- カードビュー: カード右上の矢印（>）をクリック

:::

1. テーブルまたはカードビューで詳細アイコンをクリックします。  
2. 右側にドロワーが開き、該当ビジネスのトップ30トランザクションが表示されます。  
3. 上部で並び替え基準を選択可能（初期値: 件数順）  
4. 表示項目:
    * トランザクション名
    * 詳細分析 (![検索アイコン](/img/ic-pageview.svg))
    * 件数
    * エラー数
    * 平均時間 (ms)
    * 最大時間 (ms)
    * 平均CPU使用時間
    * 平均メモリ使用量
    * SQL件数
    * 平均SQL時間 (ms)
    * SQLフェッチ件数
    * 平均SQLフェッチ件数
    * HTTP呼び出し件数
    * 平均HTTP呼び出し時間 (ms)

5. 上部検索バーで特定トランザクションを検索可能です。
