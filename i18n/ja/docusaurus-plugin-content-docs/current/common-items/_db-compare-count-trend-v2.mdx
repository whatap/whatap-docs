## カウント推移の比較

ホーム画面 &gt; プロジェクト選択 &gt; <Cmdname sid="side_analysis" className="uitext" /> &gt; <Cmdname sid="side_dbTrendComparison" className="uitext" />

<ProdImg img='db-compare-count-trend.png' desc='カウント推移' />

異なる日付のカウント推移を比較できます。 

1. **_インスタンス_**項目から目的のインスタンスを選択します。別のインスタンスを選択することもできます。

2. **_基準日_**と**_比較日_**を設定してください。 

3. ![検索アイコン](/img/ico-btn-search.png)ボタンを選択します。
