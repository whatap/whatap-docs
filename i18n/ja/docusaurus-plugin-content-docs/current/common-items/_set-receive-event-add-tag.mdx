### イベント受信タグを追加する

1. <Cmdname sid="alert_receive_settings_by_member" className="uitext" />セクションのユーザー一覧から<Cmdname sid="add_tag" className="uitext" />または<LinkImage img="ico-add.svg" desc="追加" />ボタンを選択します。

2. <Cmdname sid="event_tag" className="uitext" />ポップアップウィンドウが表示されたら、タグ入力欄にタグ名を入力して、Enterキーを入力または、<Cmdname sid="create_new_eventtag" className="uitext" />を選択します。

3. タグの色を選択します。

4. <Cmdname sid="create_eventTag" className="uitext" />ボタンを選択します。

<Cmdname sid="event_tag_list" className="uitext" />で作成したタグを確認できます。作成したタグを適用するには、作成したタグを選択します。
