## SpringBoot2.2以降、Tomcat JMXのOFF状態

SpringBoot 2.2バージョン以降でEmbedded Tomcatを使用する場合、既定値としてJMX関連の機能が動作しません。この場合、JMX機能を有効してください。[関連リンク](https://github.com/spring-projects/spring-boot/wiki/Spring-Boot-2.2-Release-Notes#jmx-now-disabled-by-default) を参照してください。

```properties title='application.properties'
spring.jmx.enabled=true
server.tomcat.mbeanregistry.enabled=true # tomcat embeddedの場合 #
```

```yml title='application.yml'
spring:
  jmx:
    enabled: true
server:
  tomcat:
    mbeanregistry:
      enabled: true #tomcat embeddedの場合
```
