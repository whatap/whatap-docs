## モニタリングの確認

サーバーから正常にログがアップロードされたことを確認し、コンソールに登録するには、[WhaTapモニタリングサービス](https://service.whatap.io)の初期画面に移動してください。確認したいプロジェクトを選択し、画面の左側の**_管理_** > **_エージェント一覧_**メニューを選択してください。**_エージェント一覧_**からデータベースサーバーの名前を確認することができます。データベースサーバーの名前を別に設定するには、[次の文書](agent-dbx-settings#naming)を参照してください。
