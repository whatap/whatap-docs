## app_proc_counter {#appproccounter}

アプリケーションのプロセス指標を収集したデータのカテゴリです。

- 収集間隔：5秒
- 統計データ：5分、1時間

### Tags {#appproccounter-tags}

<div className="c4-20-10-10-60">

| Fields       | Type | Unit | Description                          |
| ------------ | ---- | ---- | ------------------------------------ |
| alias        | -    | -    | アプリケーション別称(固有値)                      |
| container    | -    | -    | コンテナ名(固有値)                           |
| containerKey | -    | -    | コンテナキー値 (固有値)                        |
| host_ip      | -    | -    | ホストIPアドレス(固有値)                       |
| okindName    | -    | -    | whatap.okindに分類されたグループ名              |
| oname        | -    | -    | エージェント名 (固有値)                        |
| onodeName    | -    | -    | whatap.onodeに分類されたグループ名              |
| pid          | -    | -    | 特定プロセスの固有識別子(ID)                     |
| type         | -    | -    | プラットフォームタイプ(アプリケーションの開発言語)           |

</div>

### Fields {#appproccounter-fields}

<div className="c4-20-10-10-60">

| Fields                 | Type | Unit       | Description                                            |
| ---------------------- | ---- | ---------- | ------------------------------------------------------ |
| cputime                | -    | ミリ秒(ms)    | トランザクションのcpuの時間                                        |
| db_num_active          | -    | count      | DB Connection Pool Active 数                            |
| db_num_idle            | -    | count      | DB Connection Pool Idle数                               |
| gc_count               | -    | count      | GC発生回数                                                 |
| gc_oldgen_count        | -    | count      | Old Generation GCカウント                                  |
| gc_time                | -    | ミリ秒(ms)    | GCの実行時間                                                |
| heap_max               | -    | バイト        | ヒープメモリの最大サイズ(-Xmx値)                                    |
| heap_perm              | -    | バイト        | Perm領域のメモリ使用量                                          |
| heap_tot               | -    | バイト        | 現在割り当てられているヒープメモリの総量                                   |
| heap_use               | -    | バイト        | 現在使用中のヒープメモリの量                                         |
| pending_finalization   | -    | count      | GCでfinalizeの実行を待機しているオブジェクトの数                          |
| proc_fd                | -    | count      | プロセスが開いているファイル記述子(file descriptors)の数                  |
| proc_fd_max            | -    | count      | プロセスがオープンできる最大のファイルディスクリプタの数                           |
| thread_count           | -    | count      | JVMで実行中のスレッドの総数                                        |
| thread_daemon          | -    | count      | JVMで実行中のデーモンスレッドの数                                     |
| thread_peak_count      | -    | count      | JVMが実行中に観察されたスレッド数の最大値                                 |
| thread_total_started   | -    | count      | JVMが開始以降に生成されたスレッドの合計数                                 |
| threadpool_activeCount | -    | count      | 現在実行中のスレッドプールのスレッド数                                    |

</div>
