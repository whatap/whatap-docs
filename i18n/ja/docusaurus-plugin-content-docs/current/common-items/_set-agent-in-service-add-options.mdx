### オプションを追加する

<ProdImg img='app-set-agent-in-service.png' desc='エージェント設定' />

1. <Cmdname sid="side_agents" className="uitext" />からオプションを追加する![数字1](/img/number-01.png)エージェントを選択します。

2. ![数字2](/img/number-02.png)<Cmdname sid="write_options" className="uitext" />から追加するオプションを選択します。

   - <Cmdname sid="search" className="uitext" />で追加するオプションを確認できます。テキストを入力すると、一致するオプションをフィルタリングします。
   - <Cmdname sid="direct_input" className="uitext" />を選択すると、オプションキーと値を入力できます。

3. 選択したオプションキーの説明と既定値を確認し、設定値を入力します。

   <ImgLang img='add-agent-option-value.png' desc='' />

   - 選択したオプションをキャンセルするには![削除アイコン](/img/ico-trash.svg)ボタンを選択します。
   - オプションを追加設定するには<Cmdname sid="adding" className="uitext" />ボタンを選択し、上記2の手順を繰り返します。

4. 必要なすべてのオプションを追加した場合は、画面右上の<Cmdname sid="apply" className="uitext" />ボタンを選択します。

選択したオプションと設定値をエージェントに適用します。

:::note

- 変更内容を初期化するには、画面右上の![初期化アイコン](/img/ico-refresh.svg)ボタンを選択します。
- オプション値に何も入力せずに<Cmdname sid="apply" className="uitext" />ボタンを選択すると、該当オプションを削除します。
- すでに追加したオプションは、オプション一覧から選択することはできません。
- アプリケーションの種類やエージェントのバージョンによって、適用できるオプションキーが異なる場合があります。
- 例として提供された画像は、アプリケーションの種類やエージェントによって異なる場合があります。

:::
