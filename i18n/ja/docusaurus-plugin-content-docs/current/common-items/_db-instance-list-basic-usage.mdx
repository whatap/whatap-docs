## 基本画面ガイド

<ProdImg img='db-instance-list.png' desc='インスタンスリスト' />

- ![数字1](/img/number-01.png)領域は、プロジェクトに設定されているインスタンスの状態を表示します。表示されたステータス情報は、インスタンスの合計数と一致し、テーブルの<Cmdname sid="status" className="uitext" />カラム情報と一致します。

  <InDoc product="postgresql,mysql,oracle-pro">

  - <span class='snormal'><Cmdname sid="sms_alert_normal" /></span>：定常状態のエージェントの数です。

  - <span class='slow'><Cmdname sid="sms_alert_warn" /></span>: <Cmdname sid="side_event" className="uitext" /> &gt; <Cmdname sid="side_eventNoti" className="uitext" />メニューで設定したイベントのしきい値が<span class='slow'>警告</span>レベルを超えたエージェントの数です。

  - <span class='vslow'><Cmdname sid="sms_alert_fatal" /></span>: <Cmdname sid="side_event" className="uitext" /> &gt; <Cmdname sid="side_eventNoti" className="uitext" />メニューで設定したイベントのしきい値が<span class='vslow'>危険</span>レベルを超えたエージェントの数です。

  :::note

  <Cmdname sid="side_event" className="uitext" />メニューの詳細については、<Link to="set-notice">次の文書</Link>を参照してください。

  :::

  </InDoc>

- <Cmdname sid="status" className="uitext" />カラムで正常ではない状態のアイコンをクリックすると<Cmdname sid="event" className="uitext" />画面が表示されます。そのインスタンスで発生したイベントの履歴を確認できます。

- 各インスタンスを選択すると、<Cmdname sid="side_dbMonitoring" className="uitext" />メニューが表示されます。選択したインスタンスの詳細をリアルタイムでモニタリングできます。 

  :::note

  <Cmdname sid="side_dbMonitoring" className="uitext" />メニューの詳細については、<Link to="instance-monitoring">次の文書</Link>を参考にしてください。

  :::

- 主要なパフォーマンス指標をグラフチャート形式で提供します。最も高い数値をチャートに表示します。チャートにカーソルを合わせると、その場所の数値が表示されます。パフォーマンス指標の詳細については、[次の文書](metrics-data-list)を参照してください。

<Xclude product="oracle,oracle-pro,oracle-v1,altibase,mssql">

- <Cmdname sid="instance_list" className="uitext" />で<Cmdname sid="server_name" className="uitext" />カラムに表示された<span class='mark master'>M</span>、<span class='mark instance'>S</span>、<span class='mark master'>C</span>は、それぞれMaster、Slave、Clusterを意味します。Replicationで構成したDBの場合に表示します。

  Replicationインスタンスが異常な状態の場合<LinkImage img='ic-warinig-db.svg' desc='경고' className='ico' />アイコンが表示されます。

  <ImgLang img='db-instance-list-replication-warning.png' />

</Xclude>

- 画面右上の![通知アイコン](/img/ico-noti.svg)ボタンをクリックすると、リアルタイムの通知が確認できます。通知設定の詳細については、[次の文書](set-notice)を参照してください。
