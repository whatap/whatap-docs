ホーム画面 > プロジェクト選択 > **_メトリクス_** > **_メトリクスチャート_**

**_メトリクスチャート_**は、単一のメトリクス検索機能を提供します。メトリクスデータを次のようにチャートで確認できます。 

<ImgLang img='metrics-chart-item.png' desc='メトリクスチャート' />

### 上段オプション

![number 1](/img/number-01.png)領域の**_メトリクスチャート_**の上段オプションを使用してチャートの時間範囲とX軸の範囲を指定することができます。

- **_時間_**：時間の合計範囲でX軸の開始と終了を指定できます。

- **_間隔_**：時間間隔でX軸データの間隔を指定できます。

- **_オブジェクトのマージ_**：異なるフィールド値を持つデータからタグが一致するデータをマージしてチャートデータをグループ化することができます。

- **_フィルター_**：**_タグ_**を指定してフィルタリングできます。 

### 指標一覧

![number 2](/img/number-02.png)領域は、オプションを検索する指標の一覧です。指標を選択して![number 1](/img/number-01.png)領域の上段メニューで指定した時間範囲のデータをもとに![number 3](/img/number-03.png)領域にチャートを表示します。

:::note

**_時間_**と**_メトリクス_**を指定することは必須です。

:::

### チャートウィジェット

![number 3](/img/number-03.png)領域のチャートウィジェットの左上で指標名を確認できます。チャートウィジェットの右上で次のオプションを確認できます。

<ImgLang img='metrics-chart-widget-upper.png' desc='メトリクスチャートウィジェット上段' />

- タイムシフト：![left-arrow](/img/ic-caret-left.svg)![left-arrow](/img/ic-caret-right.svg)左矢印または右矢印ボタンを使用して、選択した時間範囲だけ-1または、+1ずつ移動することができます。 

  > 例えば、時間範囲が2月13日00:00\~2月14日00:00の場合、![left-arrow](/img/ic-caret-left.svg)左矢印を選択すると、2月12日00:00\~2月13日00:00までのデータが表示できます。

- 間隔/オブジェクトのマージ：![number 1](/img/number-01.png)上段メニューで指定した間隔と時間のマージを修正できます。

- 時間比較：![時間比較アイコン](/img/ic-hourgalss.svg)アイコンを選択すると、同じ指標の以前の時間帯の推移を比較できます。

- モニタリング対象：![モニタリング対象アイコン](/img/ic-filter-bl.svg)アイコンを選択して、モニタリング対象を指定できます。選択しない場合は、全体として表示されます。

- スナップショット：![スナップショットアイコン](/img/ic-camera.svg)アイコンを選択すると、ウィジェットのオプションを除くチャートのスナップショットを取得できます。

- 詳細をみる：![詳細アイコン](/img/ico-detail.svg)アイコンを選択して詳細を表示します。モニタリング対象が多く、モニタリング対象の指標推移を個別に確認できる場合に便利です。

- CSV：![CSVダウンロードアイコン](/img/ico-csv.svg)アイコンを選択して、チャートを描画したデータをCSVファイルとしてダウンロードできます。

:::note

メトリクスチャートウィジェットの上部にオプションが表示されない場合は、![その他のアイコン](/img/ico-more.svg)アイコンを選択します。

:::

### ウィジェットを削除する

画面に配置したすべてのウィジェットを削除するには、![削除アイコン](/img/ico-trash.svg)**_すべてのウィジェットを削除_**ボタンを選択します。
