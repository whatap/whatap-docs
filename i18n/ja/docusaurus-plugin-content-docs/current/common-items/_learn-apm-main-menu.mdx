WhaTapエージェントのインストールが完了しましたら、[WhaTapモニタリングサービス](https://service.whatap.io)にログインしてください。WhaTapモニタリングサービスは、円滑なサービス提供と安定した運用、異常検知モニタリングおよび把握のための情報を提供します。

アプリケーションモニタリングは、リアルタイムでトランザクションをモニタリングし、特定の時点を設定した分析機能および運用の利便性を高めるためのレポート機能、通知手段を提供します。

WhaTapモニタリングサービスは、次の主な機能を提供します。

- **リアルタイムモニタリング**

    リアルタイムトランザクションの状態／応答時間の分布図(ヒットマップ) / サーバーの現状／リアルタイムユーザ(Realtime User) の状態／トランザクションの状態／リソースの状態

- **トランザクション分析**

    トランザクショントレーシング／スタック分析／マルチトランザクショントレーズ(連携追跡) ／応答時間分布図(ヒットマップ) ／SQL追跡／HTTP Call追跡

- **事後分析**

    Cube分析／スタック分析／Performance Trend分析／応答時間の分布図(ヒットマップ) ／エラー統計

- **統計**

    Transaction／Error／SQL／Remote HTTP Call／Client IP／ブラウザ統計

- **管理**

    Agent実行方法／開発および運営環境の分離サポート／通知の設定／通知配信方法の設定

- **レポート**

    Daily Report／Daily Application Check／Weekly Report／Monthly Report
