異常検出（Anomaly Detection）とは、データ内のanomaly、outlier、abnormalなど予期せぬパターンを探す一連の活動です。ユーザーは、メトリクスのデータに基づいて、上昇パターンと下降パターンの動きが通常通りでない場合、アラート通知を送信するよう設定できます。 

<ImgLang img='set-event-abnormal.png' desc='異常値検知' />

## 異常値検出イベントの作成

<Cmdname sid="add_alert_setting" className="uitext" />ボタンを選択し、<Cmdname sid="anomaly_detection" className="uitext" />イベントを設定します。すべての設定が完了したら<Cmdname sid="save" className="uitext" />ボタンを選択します。<Cmdname sid="anomaly_detection" className="uitext" />ウィンドウのタイトルの右側にあるトグルボタンを選択し、活性化の有無を選択できます。

<ImgLang img='set-event-anormal-create.png' desc='異常値検知' />

### イベントの基本情報 {#event_basic_rule_help}

異常値検出イベントの基本情報を入力します。

- <Cmdname sid="level" className="uitext" />：<Cmdname sid="info" className="info" />、<Cmdname sid="warning" className="slow" />、<Cmdname sid="critical" className="vslow" />の中で一つのレベルを選択します。

- <Cmdname sid="message" className="uitext" />: 通知が発生した時に出力する通知メッセージを入力します。`${Tag}`または`${Field}`値を入力して、メッセージに変数を適用できます。変数に入力するキーは、選択したメトリクスデータ<Cmdname sid="category" className="uitext" />に含まれる値である必要があります。

### メトリックス {#metrics}

イベントのターゲットを選択します。

- <Cmdname sid="category" className="uitext" />：メトリクス データを区切る単位です。メトリクスイベント設定時に必須の設定値です。

- <Cmdname sid="field" className="uitext" />：イベント発行条件に使用するフィールドを選択します。複数選択が可能です。

- <Cmdname sid="filter" className="uitext" />：イベント条件のターゲットを選択します。複数選択できます。

### 異常検知 {#anomaly_detection}

<Cmdname sid="rising_pattern" className="uitext" />または<Cmdname sid="falling_pattern" className="uitext" />の感度を選択できます。各項目のトグルボタンを選択して機能をオン・オフできます。

## 異常値検知イベントの変更

1. <Cmdname sid="side_event" className="uitext" /> &gt; <Cmdname sid="side_eventSetting" className="uitext" />メニューで<Cmdname sid="anomaly_detection" className="uitext" />タブを選択します。

2. <Cmdname sid="anomaly_detection" className="uitext" />一覧で編集したい項目の右側の![編集アイコン](/img/ico-edit.svg)ボタンを選択します。

3. <Cmdname sid="anomaly_detection" className="uitext" />画面が表示されたら、修正が必要な項目を修正した後、<Cmdname sid="save" className="uitext" />ボタンを選択します。

## 異常値検知イベントの削除

<Cmdname sid="side_event" className="uitext" /> &gt; <Cmdname sid="side_eventSetting" className="uitext" />メニューで<Cmdname sid="anomaly_detection" className="uitext" />タブを選択します。一覧で削除する項目の右側にある![削除アイコン](/img/ico-trash.svg)ボタンを選択します。確認メッセージが表示されると<Cmdname sid="delete" className="uitext" />ボタンをクリックします。
