### イベント受信タグを修正および削除する

1. <Cmdname sid="alert_receive_settings_by_member" className="uitext" />セクションのユーザー一覧から<Cmdname sid="add_tag" className="uitext" />または<LinkImage img="ico-add.svg" desc="追加" />ボタンを選択します。

2. <Cmdname sid="event_tag" className="uitext" />ポップアップ画面が表示されたら<Cmdname sid="event_tag_list" className="uitext" />から修正または削除する項目の<LinkImage img="ico-edit.svg" desc="編集" />ボタンを選択します。

3. タグ名を修正するか、色を変更してから<Cmdname sid="apply" className="uitext" />ボタンを選択します。

   タグを削除するには![削除アイコン](/img/ico-trash-red.svg) <Cmdname sid="delete_tag" className="uitext" />ボタンを選択します。

:::note

イベントに適用中の<Cmdname sid="event_tag" className="uitext" />項目は削除できません。

:::
