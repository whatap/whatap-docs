- **大容量規模でも効率的な構造**

  複数のDBサーバーを管理する場合でも、一元化された性能データ収集および効率的なデータストレージ構造により、最適な分析環境を提供します。

- **クラウドDBモニタリング可能**

  クエリベースのモニタリングデータ収集方式で、DBサーバーの種類に制限があり暗線。AWS、GCP、Azureおよび、OCI環境のDBもモニタリングができ、クラウドのリソース指標との統合モニタリングにも対応しています。

- **統合モニタリング**

  Application、Server、Databaseモニタリング製品をすべてインストールすると、連携されたビューを使用して障害の原因を素早く分析し特定できます。

- **WhaTap SaaS**

  多様なDatabase製品を選択でき、サブスクリプションで利用期間を自由に設定できます。(オンプレミスタイプも提供可能)
