## 最新バージョンのエージェントをダウンロード

1. [WhaTapモニタリングサービス](https://service.whatap.io)に移動してログインしてください。

2. エージェントを更新するプロジェクトを選択します。

3. 左側のメニューから<Cmdname sid="management" className="uitext" /> > <Cmdname sid="agent_update" className="uitext" />を選択します。

4. <Cmdname sid="latest_version" className="uitext" /> または、 <Cmdname sid="latest_version_for_all" className="uitext" />ボタンをクリックします。

   <ImgLang img="java-update-agent.png" desc="エージェントアップデート" />

エージェントをダウンロードして、アプリケーションサーバーを再起動してください。アップデートを自動的に適用します。
