## 最新バージョンのエージェントをダウンロード

1. [WhaTapモニタリングサービス](https://service.whatap.io)に移動してログインしてください。

2. エージェントを更新するプロジェクトを選択します。

3. 左側のメニューから<Cmdname sid="management" className="uitext" /> > <Cmdname sid="agent_update" className="uitext" />を選択します。

4. ![다운로드 아이콘](/img/ico-download.svg) <Cmdname sid="latest_version" className="uitext" /> または、 ![다운로드 아이콘](/img/ico-download.svg) <Cmdname sid="latest*VERSION*for_all" className="uitext" />ボタンをクリックします。

5. エージェントをダウンロードして、アプリケーションサーバーを再起動してください。アップデートを自動的に適用します。
