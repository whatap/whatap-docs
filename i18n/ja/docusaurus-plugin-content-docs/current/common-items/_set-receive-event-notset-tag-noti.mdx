### 受信タグの未設定通知

<ImgLang img='receive-untag-event.png' desc='受信タグ' />

<Cmdname sid="event_tag" className="uitext" />が設定されていないアラート通知を受信する場合は、<Cmdname sid="ITM05991" className="uitext" />オプションを選択してください。<Cmdname sid="event_tag" className="uitext" />が設定されたアラート通知のみを受信する場合は、選択を解除してください。

:::note

すべてのアラート通知を受信しない場合は、オプションを解除します。<Cmdname sid="event_tag" className="uitext" />が選択されていないことを確認する必要があります。

:::
