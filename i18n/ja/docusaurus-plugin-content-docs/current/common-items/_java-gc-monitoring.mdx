- **gclog_enabled** <span class='type'>Boolean</span>

  デフォルト値`false`

  GCログモニタリングを有効にします。

- **gclog_file** <span class='type'>String</span>

  GCログファイルのパスを設定します。

- **gclog_check_interval** <span class='type'>Int</span>

  デフォルト値`2000`

  GCログをモニタリングする間隔です。定期的にログファイルにレコードが追加されたことを確認します。

- **gclog_word** <span class='type'>String</span>

  デフォルト値`Full GC`

  モニタリングするキーワードを登録します。「Full GC」が発生すると、情報をサーバーに送信して保存します。
