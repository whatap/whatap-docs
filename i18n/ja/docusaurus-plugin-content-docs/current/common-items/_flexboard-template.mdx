提供された事前構成テンプレートを使用します。 初期設定を簡単に実行し、設定を変更して利用することができます。

1. <Cmdname sid="side_flexboard" className="uitext" />メニューを選択します。

2. 左側のテンプレート一覧から目的のテンプレートを選択します。

3. <Cmdname sid="flexboard_management" className="uitext" />画面が表示されたら、<Cmdname sid="rename" className="uitext" />に名前を入力します。

   * <Cmdname sid="layout" className="uitext" />項目のウィジェットをマウスで選択し、目的の位置にドラッグして移動できます。

   <ImgLang img='flexboard-template-window.png' desc='Flex ボード管理' />

4. <Cmdname sid="save" className="uitext" />ボタンを選択します。 選択したテンプレート基準のFlexボードを作成します。

   * <Cmdname sid="side_flexboard" className="uitext" />メニューで作成したボードの一覧を確認できます。
