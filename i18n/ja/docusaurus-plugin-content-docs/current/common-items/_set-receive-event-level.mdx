## 受信レベルを選択する

警告通知レベルに応じて、通知を受信有無を選択できます。<Cmdname sid="alert_receive_settings_by_member" className="uitext" />および<Cmdname sid="3rd_party_plugin" className="uitext" />セクションで<Cmdname sid="receive_level" className="uitext" />を<Cmdname sid="all" className="uitext" />または<Cmdname sid="warn" className="uitext" />、<Cmdname sid="fatal" className="uitext" />を選択します。

| <Cmdname sid="alert_receive_settings_by_member" className="uitext" /> |     <Cmdname sid="3rd_party_plugin" className="uitext" />     |
| :-------------------------------------------------------------------: | :-----------------------------------------------------------: |
|     <ImgLang img="set-event-user-select-level.png" desc="受信レベル" />    | <ImgLang img="set-event-3rd-select-level.png" desc="受信レベル" /> |

- <Cmdname sid="all" className="uitext" />：すべてのアラート通知を受信できます。

- <Cmdname sid="warn" className="uitext" />：警告レベルの通知のみ受信できます。

- <Cmdname sid="fatal" className="uitext" />：重大レベルのアラート通知のみ受信できます。
