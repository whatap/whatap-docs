### トレース一覧カラムを設定する

<Cmdname sid="trace_list" className="uitext" />セクションのカラム設定により、照会された結果の詳細項目を追加で確認できます。 ![カラムアイコン](/img/ico-column.svg) <Cmdname sid="column_select" className="uitext" />ボタンを選択します。

<ImgLang img="analysis-hitmap-tx-list-select-column.png" desc="カラム選択" />

ご希望のカラム項目を選択または解除します。 設定した状態がテーブルにすぐ反映されます。 設定をすべて完了したら、右上の![閉じるアイコン](/img/ico-close.svg)ボタンを選択します。

- <Cmdname sid="select_all" className="uitext" />: すべてのカラム項目を選択します。

- <Cmdname sid="deselect_all" className="uitext" />: すべてのカラム項目を解除します。

- <Cmdname sid="initialize_default" className="uitext" />: カラム項目をデフォルトに設定します。
