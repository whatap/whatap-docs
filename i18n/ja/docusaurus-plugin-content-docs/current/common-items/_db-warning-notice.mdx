## 基本通知：Connection Failed

アラート通知を別途設定しなくても、エージェントがDB接続に15回以上失敗した場合、次のように通知を送信します。DBのシャットダウン(shutdown) 及びネットワーク障害、アカウントまたは権限の問題により接続できなくなる可能性があります。

<ImgLang img='db-default-alert-connection-failed.svg' desc='Connection Failed'/>

## アラート通知の設定

ユーザーが選択した<Cmdname sid="metrics" className="uitext" />指標項目によって入力した<Cmdname sid="fatal" className="uitext" />と<Cmdname sid="warn" className="uitext" />項目値の条件を満たすイベントが<Cmdname sid="number_of_detections" className="uitext" />だけ発生した場合は、警告通知を送信するように設定できます。

<ImgLang img='set-event-item-db.png' desc='イベント' />

1. <Cmdname sid="instance" className="uitext" />でアラート通知を設定するエージェントを選択します。

2. <Cmdname sid="add_notification" className="uitext" />ボタンを選択すると、画面の下にイベント項目を作成します。

3. <Cmdname sid="metrics" className="uitext" />項目から目的の指標を選択します。

4. <Cmdname sid="fatal" className="uitext" />と<Cmdname sid="warn" className="uitext" />項目に目的の数値を入力します。

5. <Cmdname sid="number_of_detections" className="uitext" />項目に目的の数値を入力します。

   - 5秒間に設定された回数だけ連続して発生した場合、アラートが発生します。  
   - はい、検知回数が3回に設定されている場合は、`5秒（1回目） - 5秒（2回目） - 5秒（3回目） → 合計15秒経過 → アラート送信`となります。 

6. 右端のトグルボタンを選択して、アラート通知をアクティブにしてください。

:::tip

<Cmdname sid="CNT08320" className="uitext" />

発生した<Cmdname sid="CNT08320" className="uitext" />送信可否を選択できます。トグルボタンを選択して機能をオンまたはオフできます。

- この機能は、DBXエージェント1.6.30バージョン以降に対応しています。

:::

## 通知のコピー

現在選択しているインスタンスで作成されたアラート通知イベントを別のインスタンスにコピーできます。

1. ![複製アイコン](/img/ico-copy.svg) <Cmdname sid="copy_notification" className="uitext" />ボタンを選択します。 

2. コピー対象のインスタンスを選択します。すべてのインスタンスにコピーするには、<Cmdname sid="all_selected" className="uitext" />を選択します。

3. <Cmdname sid="copy" className="uitext" />ボタンを選択します。

:::caution

コピー対象インスタンスに既に作成済みのアラート通知イベントがある状態で、<Cmdname sid="copy_notification" className="uitext" />機能を使用すると、既存のアラート通知イベントは削除されます。

:::

## 繰り返される警告通知を解除する

過剰なアラート通知の発生を防止できるオプションです。 最初のアラート通知から選択した時間にアラート通知を送信しません。 また、<Cmdname sid="event_history" className="uitext" />メニューに記録しません。

<Cmdname sid="BTN07592" className="uitext" />で目的の時間を選択し、右側のトグルボタンをクリックして機能をオンまたはオフにすることができます。

:::note

<Cmdname sid="BTN07592" className="uitext" />オプションを有効にした場合、設定した時間にエージェントからデータを受信できない場合は、<span class="uitext">INACTIVE MONITORING AGENT</span>アラート通知が発生します。

<ImgLang img='db-set-event-disable-alert.png' desc='INACTIVE MONITORING AGENT'/>

:::
