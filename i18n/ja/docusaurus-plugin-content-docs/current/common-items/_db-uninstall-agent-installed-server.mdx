### エージェントがインストールされたサーバーから削除する

<Tabs groupId='os'>
<TabItem value='linux' label='Linux' default>

サーバー上で起動されるJavaプロセスから、`whatap.agent.dbx`文字列を含むプロセスを終了します。エージェントをインストールしたパスから_stop.sh_を実行してください。その後、エージェントファイルとエージェントに登録したUser権限を削除してください。

</TabItem>
<TabItem value='windows' label='Windows'>

サーバー上で起動するJavaプロセスからエージェントファイル名のjarファイルで起動中のプロセスを見つけて停止してください。もし、batファイルをForegroundで実行中の場合は、そのbat実行スクリプトを終了してください。その後、エージェントファイルとエージェントに登録したUser権限を削除してください。

</TabItem>
</Tabs>
