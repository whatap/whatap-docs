## チーム別およびユーザー別の通知を設定する {#set-notice-by-team-or-users}

プロジェクトのメンバーのうち、特定メンバーまたはチームの通知の受信有無を設定します。<Cmdname sid="metrics" className="uitext" />、<Cmdname sid="anomaly_detection" className="uitext" />、<Cmdname sid="log" className="uitext" />イベント設定の<Cmdname sid="event_tag" className="uitext" />項目でタグを追加してください。 イベントごとにアラート通知を受信するメンバーまたはチームを選択できます。

逆に、イベント受信タグを設定しない場合、すべてのメンバーにアラート通知を送信できます。

<details>
<summary>イベント受信タグの使用例</summary>

イベント受信タグの使用例を使用して、チーム別またはユーザー別によるアラート通知を送信する設定方法について説明します。次のように、チームごとに異なるアラート通知を設定します。

| 通知の種類             |           サーバーチームの受信有無          |          DevOpsチームの受信有無         |
| ----------------- | :-----------------------------: | :-----------------------------: |
| メトリックアラート通知       | <span class='emoji-ok'>✅</span> | <span class='emoji-no'>❌</span> |
| 異常値検知の警告通知        | <span class='emoji-ok'>✅</span> | <span class='emoji-ok'>✅</span> |
| ログのリアルタイムアラート通知   | <span class='emoji-no'>❌</span> | <span class='emoji-ok'>✅</span> |

1. **イベント受信の設定** (<Cmdname sid="event_statistic" className="uitext" /> > <Cmdname sid="eventNoti" className="uitext" />)

   <ImgLang img='set-event-tag-01.png' desc='ユーザー別イベント受信設定' />

   プロジェクトに属するメンバーのうち、サーバーチーム所属は、`サーバーチーム`、DevOps所属は`DevOpsチーム`とイベント受信タグを設定してください。

2. **メトリクスアラート通知の設定** (<Cmdname sid="event_statistic" className="uitext" /> > <Cmdname sid="event_setting" className="uitext" /> > <Cmdname sid="metrics" className="uitext" /> > <Cmdname sid="add_notification" className="uitext" /> > <Cmdname sid="eventNoti" className="uitext" />)

   <ImgLang img='set-event-tag-02.png' desc='ユーザー別イベント受信設定' />

   <Cmdname sid="add_tag" className="uitext" />または<LinkImage img="ico-add.svg" desc="追加アイコン" />ボタンをクリックしてください。タグ一覧から目的のタグを選択するか、新しいタグを作成してください。メトリクスイベントの通知を`サーバーチーム`に設定されている場合です。

3. **ログのリアルタイムアラート通知設定** (<Cmdname sid="event_statistic" className="uitext" /> > <Cmdname sid="event_setting" className="uitext" /> > <Cmdname sid="log" className="uitext" /> > <Cmdname sid="add_notification" className="uitext" /> > <Cmdname sid="event_tag" className="uitext" />)

   <ImgLang img='set-event-tag-03.png' desc='ユーザー別イベント受信設定' />

   <Cmdname sid="add_tag" className="uitext" />または<LinkImage img="ico-add.svg" desc="追加アイコン" />ボタンをクリックしてください。タグ一覧から目的のタグを選択するか、新しいタグを作成してください。ログのリアルタイムイベントの通知を`DevOpsチーム`に設定した場合です。

4. 異常値検知イベントは、すべてのメンバーにアラート通知を送信するため、イベント受信タグを設定しません。

</details>
