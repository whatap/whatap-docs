:::note

-   この機能は、**修正**権限を持つメンバーが利用できます。**修正**権限のないメンバーは、設定内容の照会のみ可能です。

-   オプション値として設定できる形式は次のとおりです。

    -   Boolean形式の値は`true`または`false`を選択してください。
    -   数値形式の値は数字のみ入力できます。
    -   テキスト（String）形式の値を入力または修正する場合は、オプションの説明を詳しく確認してください。

-   修正できないオプションは選択できません。（例、`license`）

-   追加または修正、削除したオプションによっては、エージェントを再起動する必要があります。 

-   アプリケーションの種類やエージェントのバージョンによって、適用できるオプションキーは異なる場合があります。

:::
