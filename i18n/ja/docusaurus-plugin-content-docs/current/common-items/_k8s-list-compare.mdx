1. ![number a](/img/char-a.svg) **_比較する_**オプションを選択すると、<InDoc pages="resource-container-list">コンテナ</InDoc><InDoc pages="resource-node-list">ノード</InDoc>一覧に比較カラムが追加されます。

2. <InDoc pages="resource-container-list">コンテナ</InDoc><InDoc pages="resource-node-list">ノード</InDoc>一覧の左側<LinkImage img="char-b.svg" className="ico" />の比較コラムを通じて比較したい<InDoc pages="resource-container-list">コンテナ</InDoc><InDoc pages="resource-node-list">ノード</InDoc>を選択します。

   - <InDoc pages="resource-container-list">コンテナ</InDoc><InDoc pages="resource-node-list">ノード</InDoc>は、最大10個まで選択できます。

   - 選択した<InDoc pages="resource-container-list">コンテナ</InDoc><InDoc pages="resource-node-list">ノード</InDoc>ごとに異なる色が割り当てられます。

3. ![number c](/img/char-c.svg)領域で選択した<InDoc pages="resource-container-list">コンテナ</InDoc><InDoc pages="resource-node-list">ノード</InDoc>の比較チャートを確認できます。 

   - ![number b](/img/char-b.svg)比較カラムで選択した<InDoc pages="resource-container-list">コンテナ</InDoc><InDoc pages="resource-node-list">ノード</InDoc>を![number c](/img/char-c.svg)領域の上部で確認できます。

   - ![number c](/img/char-c.svg)上部をドラッグして比較チャート領域の高さを調整できます。
