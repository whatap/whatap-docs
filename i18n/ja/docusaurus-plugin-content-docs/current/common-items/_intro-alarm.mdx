イベント条件を設定し、メール、SMS、メッセンジャー、App Pushなどのさまざまな経路で通知を受信することができます。

-   エージェントと収集サーバーの通信損失およびリソース使用率、アクティブトランザクション、エラー、応答遅延、応答分布の異常などの通知設定ができます。
-   困難な状況の通知設定は、タグカウントの通知機能によって提供されます。
-   マシンラーニングベースのAI通知設定により、ヒットマップパターンを検知し、通知機能を提供します。AIモジュールがアプリケーション実行分布パターンを判断して通知を送信します。複雑な条件なしに問題の状況を迅速かつ正確に認識することができます。
-   **_イベント設定_**メニューにより、しきい値ベースの通知条件および受信設定を行うことができます。
-   **_イベント受信設定_**メニューでは、個人通知の受信チャネルおよびタイムゾーンを設定できます。
-   **_イベント履歴_**メニューでは、通知発生履歴を確認できます。
