### 分類別エージェントモニタリング

エージェント設定で分類した単位で複数のエージェントを簡単にモニタリングできます。

- <Cmdname sid="ITM06005" className="uitext" />: 個別エージェント

- <Cmdname sid="ITM06006" className="uitext" />: エージェントの種類(例: ウェブサーバー、データベースなど)に分類されたグループ、<code>whatap.okind</code>に設定

- <Cmdname sid="ITM06007" className="uitext" />: サーバ(ノード)別に分類されたグループ、<code>whatap.onode</code>に設定

- <Cmdname sid="ITM06008" className="uitext" />: <code>whatap.okind</code>に分類されたエージェントの一覧

- <Cmdname sid="ITM06009" className="uitext" />: <code>whatap.onode</code>に分類されたエージェントの一覧

:::note

エージェント設定の詳細については[次の文書](agent-name)を参照してください。

:::
