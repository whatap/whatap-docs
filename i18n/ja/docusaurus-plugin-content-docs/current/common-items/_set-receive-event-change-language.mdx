## アラート通知の言語設定

プロジェクトで発生するアラートメッセージの言語を変更できます。

1. ホーム画面でアラート通知の言語を変更するプロジェクトを選択します。

2. 画面の左側のメニューで<Cmdname sid="management" className="uitext" /> &gt; <Cmdname sid="project_management" className="uitext" />を選択します。

3. <Cmdname sid="TTL06021" className="uitext" />セクションで、使用する言語を選択します。

4. 画面の右下にある<Cmdname sid="save" className="uitext" />ボタンを選択します。
