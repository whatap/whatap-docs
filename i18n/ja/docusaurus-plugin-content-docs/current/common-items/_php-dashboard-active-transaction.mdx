ホーム画面でプロジェクトを選択し、**_ダッシュボード_**のサブメニューの中の**_アプリケーションダッシュボード_**を選択してください。次のような**\*アプリケーションダッシュボード**の**_アクティブトランザクション_**ウィジェット領域で![right-arrow](/img/right-arrow.svg)ボタンをクリックすると、**_アクティブトランザクション_**詳細ページに移動します。

![アクティトランザクション](/img/apm-dashboard-active-tx.png)

現在のトランザクションのURL、SQL、HTTP呼び出し情報で異常トランザクションをすばやく特定し、詳細なトレースを通じて遅延間隔を詳細に分析できます。

## 画面構成

![アクティトランザクション](/img/dashboard-active-transaction.png)

![指示線1](/img/number-01.png) **エージェント別アクティブトランザクション**

全体アプリケーションサーバーの一覧と進行中のトランザクションの合計数を表示します。チャート内の各グラフにマウスオーバーすると、トランザクションステータスの速度が表示されます。トランザクションステータスの速度は、<span class='normal'>Normal</span>、<span class='slow'>Slow</span>、<span class='vslow'>Very Slow</span>に分類します。

![指示線2](/img/number-02.png) **エージェント一覧**

アプリケーションサーバーの一覧と進行中のトランザクションの合計数を表示します。該当アプリケーション サーバーを選択すると![指示線3](/img/number-03.png)トランザクション一覧に詳細が表示されます。

![指示線3](/img/number-03.png) **トランザクション一覧**

進行中のトランザクションのURL、SQL、HTTP呼び出し情報の詳細を表示します。Top 10項目タブを選択すると、各項目別の経過時間が高い順にソートされている表を確認できます。同様に、トランザクションステータスの速度は<span class='normal'>Normal</span>、<span class='slow'>Slow</span>、<span class='vslow'>Very Slow</span>に分類します。

## エージェント一覧フィルタリング

![指示線2](/img/number-02.png) **エージェント一覧**でフィルタリング対象を選択・検索して、トランザクション一覧で個別に確認できます。 

-   現在活性化されているエージェントを確認するには、**_活性化エージェントを見る_**チェックボックスを選択してください。

-   **_全体エージェント_**一覧ボックスをクリックしてください。エージェントをカテゴリ別に分類して確認できます。

-   進行中のトランザクション情報をエージェント別に確認するには、エージェントを選択し、以下の**_適用_**ボタンをクリックしてください。選択を取り消すには、![リセットアイコン](/img/ico-reset.svg)ボタンをクリックしてください。

## 自動更新設定

進行中のトランザクションを設定した間隔ごとに自動更新できます。**_自動更新(秒)_**トグルボタンを選択してオプションを活性化し、右のテキスト入力ボックスに更新間隔を入力してください。

自動更新設定を非活性化するには、**_自動更新(秒)_** トグルボタンをもう一度選択してください。

:::note

-   時間は、秒単位で更新します。
-   自動更新設定を非活性化した状態でトランザクション一覧を更新するには、![自動更新アイコン](/img/ico-refresh.png)ボタンを選択してください。
    :::

## トランザクション一覧フィルタリング

![指示線3](/img/number-03.png) **トランザクション一覧**でURLまたはIPアドレスを検索してフィルタリングすることができます。右上の**_URL / IP_**テキスト入力ボックスにフィルタリングする対象を入力してからEnterキーを入力するか![検索アイコン](/img/ico-search.svg)ボタンを選択してください。入力したテキスト値を含むトランザクションの一覧でフィルタリングします。

フィルタリングした一覧を初期化するには右上の**_URL / IP_**テキスト入力ボックスのテキストを削除し、Enterキーを入力するか![検索アイコン](/img/ico-search.svg)ボタンを選択してください。
