ホーム画面でプロジェクトを選択し、***ダッシュボード***のサブメニューから***アプリケーションダッシュボード***を選択します。次のように***アプリケーションダッシュボード***の***アクティブトランザクション***ウィジェット領域で![right-arrow](/img/right-arrow.svg)ボタンをクリックすると、***アクティブトランザクション***詳細ページに移動します。

<ImgLang img='apm-dashboard-active-tx.png' desc='アクティブトランザクション' />

実行中のトランザクションのURL、SQL、HTTP呼び出し情報から異常トランザクションをすばやく特定し、詳細なトレースを通じて遅延部分を詳細に分析できます。

## 画面構成

<ImgLang img='dashboard-active-transaction.png' desc='アクティブトランザクション' />

![指示線1](/img/number-01.png) **エージェント別のアクティブトランザクション**

すべてのアプリケーションサーバーの一覧と稼働中のトランザクションの合計数を表示します。チャート内の各グラフにカーソルを合わせると、トランザクションステータスが表示されます。トランザクションステータスの速度<span class='normal'>Normal</span>、<span class='slow'>Slow</span>、<span class='vslow'>Very Slow</span>に分類します。

![指示線2](/img/number-02.png) **エージェント一覧**

アプリケーションサーバーの一覧と進行中のトランザクションの合計数を表示します。該当アプリケーション サーバーを選択すると![指示線3](/img/number-03.png)トランザクション一覧に詳細が表示されます。

![指示線3](/img/number-03.png) **トランザクション一覧**

進行中のトランザクションのURL、SQL、HTTP呼び出し情報の詳細を表示します。Top 10項目タブを選択すると、各項目別の経過時間が高い順にソートされている表を確認できます。同様に、トランザクションステータスの速度は<span class='normal'>Normal</span>、<span class='slow'>Slow</span>、<span class='vslow'>Very Slow</span>に分類します。

## エージェント一覧のフィルタリング

![指示線2](/img/number-02.png) **エージェント一覧**でフィルタリング対象を選択または検索して、トランザクション一覧で個々の対象を確認できます。 

- 現在、アクティブなエージェントを確認するには、***アクティブなエージェントのみ表示***チェックボックスを選択します。

- ***すべてのエージェント***一覧ボックスをクリックします。エージェントをカテゴリ別に分類して確認できます。

- 実行中のトランザクション情報をエージェントごとで確認するには、エージェントを選択し、***適用***ボタンをクリックします。選択を取り消すには、![リセットアイコン](/img/ico-reset.svg)ボタンをクリックします。

## 自動更新設定

実行中のトランザクションを設定した間隔で自動更新できます。***自動更新(秒)***トグルボタンでオプションを有効にして、右側の入力欄に更新間隔を入力します。

自動更新設定を無効にするには、***自動更新(秒)*** トグルボタンをもう一度選択します。

:::note

- 時間は秒単位で更新します。
- 自動更新設定を無効した状態でトランザクション一覧を更新するには、![自動更新アイコン](/img/ico-refresh.png)ボタンを選択します。

:::

## トランザクション一覧のフィルタリング

![指示線3](/img/number-03.png) **トランザクション一覧**でURLまたはIPアドレスを検索してフィルタリングできます。右上の***URL / IP***の入力欄にフィルタリングする対象を入力し、Enterキーまたは![検索アイコン](/img/ico-search.svg)ボタンを選択します。トランザクション一覧は、入力値でフィルタリングされます。

フィルタリングした一覧を初期化するには右上の***URL / IP***テキスト入力ボックスのテキストを削除し、Enterキーまたは![検索アイコン](/img/ico-search.svg)ボタンを選択します。
