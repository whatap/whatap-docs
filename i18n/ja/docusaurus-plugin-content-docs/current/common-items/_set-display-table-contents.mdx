<Title hashid='setting-table-contents' level={props.level}>テーブルコンテンツの表示設定</Title>

<InDoc product="browser">

ブラウザーエラー一覧に表示するコンテンツの内容を設定できます。 多くの情報が表示されて見づらい場合や必要な内容を追加したい場合に利用できます。

</InDoc>

1. テーブル右上の![設定アイコン](/img/ico-setting.svg)ボタンを選択します。

2. <Cmdname sid="ITM08349" className="uitext" />ウィンドウが表示されたら、目的のオプションを設定します。

   - <UI>content</UI>または<Cmdname sid="tag" className="uitext" />オプションを選択して、ログにコンテンツの表示有無を選択します。

   - <Cmdname sid="tag_management" className="uitext" />: タグは、追加した順にログに表示されます。 タグの順序はドラッグに変更できます。 非活性化されているタグはログに表示されません。

3. すべての設定を完了して![閉じるアイコン](/img/ico-close.svg)ボタンを選択します。 設定した内容がテーブルに適用されます。

:::note

<Cmdname sid="ITM08349" className="uitext" />の詳細については[次の文書](../log/log-search#set-table)を参照してください。

:::
