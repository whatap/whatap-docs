<Title hashid='change-order-column' level={props.level}>テーブルカラムの表示順番の変更</Title>

<InDoc product="browser">

ブラウザーエラー一覧のテーブルでカラム ヘッダーの順序を変更できます。

</InDoc>

1. テーブルの右上の![カラムアイコン](/img/ico-column.svg)ボタンを選択します。

2. <Cmdname sid="column_settings" className="uitext" />ウィンドウが表示されたら、右側のリストで目的の項目をドラッグして順序を変更します。

3. すべての設定を完了した後<Cmdname sid="BTN06053" className="uitext" />ボタンを選択します。

設定した内容がテーブルに適用されます。
