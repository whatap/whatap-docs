**要求事項**

WhaTap Kubernetesエージェント1.7.6バージョン以降が必要です。 タイムラインデータは、1.7.6バージョン以降で収集された時点から提供されます。

<UI>{props.menu}</UI>メニューは時間の経過に伴う{props.object}の状態変化を視覚的に追跡できる機能を提供します。 既に終了された{props.object}がいつ終了したのか、特定の時点で{props.object}がどのような状態であったかを視覚的に捉えることができます。 これにより、ユーザーは過去の状態を分析し、クラスターで発生した問題の原因把握および運営の安定性をモニタリングすることができます。

## 基本画面ガイド

<UI>{props.menu}</UI>メニューに入った後<Cmdname sid="TTL06740" className="uitext" />と<Cmdname sid="TTL08842" className="uitext" />を選択してデータを照会してください。 最近1時間の間に収集された{props.object}一覧を確認することができます。 時間軸に沿って{props.object}の状態変化を視覚的に確認できます。

</InDoc>

<ImgLang img="k8s-container-timeline.png" desc="コンテナタイムライン1" />

## {props.object} タイムライン詳細 {#timeline-details}

{props.object}タイムラインの詳細は、特定区間又は特定{props.object}を選択して確認できます。 

<InDoc pages="pod-timeline">

<ImgLang img="k8s-pod-timeline-details.png" desc="Pod タイムライン詳細" />

</InDoc>

<InDoc pages="container-timeline">

<ImgLang img="k8s-container-timeline-details.png" desc="コンテナタイムライン詳細" />

</InDoc>

- 詳細情報を照会したい区間を選択し、右側の **詳細を見る** 画面を通じて当該時点の{props.object}に対する詳細を照会できます。

- チャートの左側の一覧で照会したい{props.object}名前を選択して右側の**詳細を見る** 画面から該当{props.object}の全体の状態を見ることができます。

:::note

### <Cmdname sid="overview" className="uitext" />

<InDoc pages="pod-timeline">

<Cmdname sid="overview" className="uitext" />タブで選択したPod情報及び全般的なリソース使用量情報を要約して提供します。 当該Podで実行中のコンテナ、Podが実行されているノード、`ownerReference`などPod情報を総合的に確認できます。 ユーザーはPodのリソース使用状況と各構成要素の状態を把握し、パフォーマンスと運用安定性をモニタリングできます。

- ***AVG***、***MAX***ボタンをクリックして、照会期間中の平均使用量と最大使用量を確認できます。

- Pod内で実行されたコンテナのタイムラインを提供します。 コンテナのタイムラインを選択した場合、当該タイムラインが選択された<UI>コンテナタイムライン</UI>メニューが新しいタブで開きます。

</InDoc>

<InDoc pages="container-timeline">

<Cmdname sid="overview" className="uitext" />タブで選択したコンテナ情報と全体的なリソース使用量情報を要約して提供します。 当該コンテナの画像、コンテナが実行されているノード、コンテナが実行されているPodの`ownerReference`などのコンテナ情報を総合的に確認できます。 ユーザーは、コンテナのリソース使用状況と状態を把握し、性能と運用の安定性をモニタリングできます。 また、***AVG***、***MAX***ボタンをクリックして、照会期間中の平均使用量と最大使用量を確認できます。

</InDoc>
