### ノードの種類を選択する

<ImgLang img="npm-dashboard-topology-select-node.png" desc="ノード選択" />

チャートに表示されるノードの種類を選択できます。 それぞれが1つのタグに対応します。 タグは、_tagRule.yaml_ファイルに設定した**tagRule**オプションに従って収集されます。 <Cmdname sid="ITM06810" className="uitext" />、<Cmdname sid="ITM06811" className="uitext" /> または、2つの組み合わせでノードを表現できます。

:::note

**tagRule**オプションの詳細については、[次の文書](set-tagrule)を参考にしてください。

:::
