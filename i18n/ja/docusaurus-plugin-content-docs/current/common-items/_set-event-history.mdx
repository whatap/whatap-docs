-   <Cmdname sid="time_selector" className="uitext" />

    -   右側の緑色のボタンを選択して表示時間を選択することができます。

    -   ![左アイコン](/img/left-arrow.svg)または ![右アイコン](/img/right-arrow.svg)ボタンを選択すると、選択した表示時間だけ間隔を移動します。

    -   詳細な時間を選択するには、日付または時間の領域を選択してください。詳細な時間を設定し、<Cmdname sid="apply" className="uitext" />ボタンを選択します。

-   <Cmdname sid="filter" className="uitext" />: <Cmdname sid="title" className="uitext" />または<Cmdname sid="message" className="uitext" />内容に基づいてイベント履歴をフィルタリングできます。

-   <Cmdname sid="event_time" className="uitext" />: イベントが発生した時刻です。

    -   イベントが解消されずに進行中である場合、<Cmdname sid="in_progress" className="uitext" />タグが表示されます。

    -   整備中である場合、イベントが発生すると<Cmdname sid="ITM07315" className="uitext" />タグが表示されます。

    :::note

    <Cmdname sid="sms_maintenance" className="uitext" />メニューの詳細については、<Link to="../management/maintenance">次の文書</Link>を参考にしてください。

-   <Cmdname sid="TTL06583" className="uitext" />: 設定したイベントが解決された時刻です。もし該当カラムが見えなかったら<Cmdname sid="column_select" className="uitext" />ボタンをクリックし<Cmdname sid="TTL06583" className="uitext" />を選択してください。
