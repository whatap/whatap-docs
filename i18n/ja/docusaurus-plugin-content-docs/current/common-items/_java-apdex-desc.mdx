Application Performance Index(Apedx)はアプリケーションのパフォーマンス指標です。ウェブアプリケーションの顧客満足度を測定します。ユーザー満足度の指標として活用でき、0~1の間の値を有します。

> (満足回数+ (許容回数\*0.5))／全体リクエスト数

-   満足(Satisfied, S)：タスク処理に問題なし≤ 1.2秒(満足Sデフォルト値)

-   許容 (Tolerating, T)：ユーザーが遅延を感じるが、タスク処理は可能≤4.8秒(満足S\*4)

-   不満 (Frustrated, F)：タスク処理の不可能 > 4.8秒 
     (許容T超過及びエラー)
