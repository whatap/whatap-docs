## データ参照

- 画面の下までスクロールすると、次のデータを検索します。一度に、10,000個のログを参照できます。

- ![number-s](/img/number-03.png)ログテーブルの左上に検索されたログの総数を確認することができます。 

- ログデータを時間の順と逆順で照会できます。![number-s](/img/number-03.png)ログテーブルの右上の<Cmdname sid="past_order" className="uitext" />と<Cmdname sid="recent_order" className="uitext" />の中から照会方式を選択します。

- 時間範囲を指定した後<Cmdname sid="apply" className="uitext" />ボタンを選択して照会時間を設定し、![検索アイコン](/img/ic-search-on-press.svg) <Cmdname sid="search" className="uitext" />ボタンを選択してデータを照会します。

- ![number-s](/img/number-03.png)ログテーブルの右上の<LinkImage img='ic-fullscreen.svg' desc='全画面表示アイコン'/> <Cmdname sid="TLT06869" className="uitext" />アイコンを選択すると、<Cmdname sid="log" className="uitext" />と<Cmdname sid="TTL06444" className="uitext" />を全画面で確認できます。

## ログのタグオプション {#log-tag-options}

ログのタグを選択すると、ドロップダウンメニューが表示されます。<Cmdname sid="search" className="uitext" />、 <Cmdname sid="exclusion_search" className="uitext" />、<Cmdname sid="TTL06582" className="uitext" />、<Cmdname sid="TTL06527" className="uitext" />オプションが確認できます。 

<ImgLang img='log-search-tag.png' desc='ログタグドロップダウンメニュー'/>

|                     ログタグオプションメニュー                     | 説明                                                                                                                                                    |
| :---------------------------------------------------: | ----------------------------------------------------------------------------------------------------------------------------------------------------- |
|                        **コピー**                        | 該当するタグのログをコピーします。                                                                                                                                     |
|      <Cmdname sid="search" className="uitext" />      | <Cmdname sid="filter" className="uitext" />には、該当のタグを含む（'=='）条件で入力されます。                                                                                |
| <Cmdname sid="exclusion_search" className="uitext" /> | <Cmdname sid="filter" className="uitext" />には、該当のタグを除外する（'!='）条件で入力されます。                                                                              |
|     <Cmdname sid="TTL06582" className="uitext" />     | <Cmdname sid="TTL06582" className="uitext" /> の詳細画面では、選択したログのサーバーを対象に、選択したログと隣接する時間帯のログを照会できます。時間を選択して、周辺の時間帯に発生したログを確認することも可能です。基準となるログは青い背景で表示されます。 |
|                     **トランザクション情報**                    | ログに `@txid` や `@mtid` タグが含まれている場合は、**トランザクション情報** をクリックすると、**トランザクション情報** の詳細画面で内容を確認できます。                                                            |
