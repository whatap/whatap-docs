ホーム画面 > プロジェクト選択 > **_メトリクス_** > **_メトリクス異常検知_**

さまざまなメトリクス指標のパターンをAIによって学習された予想パターンと比較することができます。予想パターンから外れた異常検知をグラフチャートで確認できます。過去の履歴データに基づいて繰り返されるパターンを確認し、それらを使用して今後の指標値を予測できます。

:::note

-   画面に配置したウィジェットは、別のメニューに移動しても保存されずに初期化します。
-   パターン表示と異常値表示機能を除くと、**_メトリクス_** > **_メトリクスチャート_**メニューと類似しています。

:::

## ウィジェットを確認する

![メトリック異常検知](/img/metrics-detect-abnormal2.png)

-   ![数字1](/img/number-01.png)明るい色のグラフ領域は、AIによって分析した予想されるパターンです。 

-   ![数字2](/img/number-02.png) <span class='normal'>青色</span>グラフは、プロジェクトのメトリクス指標の推移です。

-   AIによって分析された予想パターンから外れると<span class='slow'>オレンジ</span>、<span class='vslow'>赤色</span>のステップでグラフにドットが表示されます。予想されるパターンの範囲から大幅に外れた値を<span class='vslow'>赤色</span>として表示します。

## ウィジェットを配置する

![メトリクス異常検知](/img/metrics-detect-abnormal-set-widget2.png)

1.  ![数字1](/img/number-01.png) **_時間_**で、目的の時間間隔を設定してください。最大1か月間隔まで設定できます。

2.  ![数字2](/img/number-02.png) **_カテゴリ選択_**で、メトリクス指標の範囲を選択します。

3.  **_ウィジェットの追加_**一覧からモニタリングするメトリクス指標を選択します。

選択したメトリクス指標を画面の右側に配置します。

:::note

-   モニタリング対象を選択し、メトリクス指標を区分して確認するには、![数字3](/img/number-03.png) **_タグ_**を選択してください。画面の右側にタグ選択一覧が表示されます。項目を選択した後、ウィジェットを追加してください。
-   画面に配置したウィジェットは、**_時間_**または**_タグ_**値を変更してもチャートに反映されません。
-   画面に配置したウィジェットを削除するには、ウィジェット右上の![削除アイコン](/img/ico-remove.svg)ボタンを選択します。
-   ウィジェットの左上を選択した状態でドラッグすると、ウィジェットの位置を変更できます。
-   ウィジェットの右下を選択した状態でドラッグすると、ウィジェットのサイズを変更できます。

:::

## 関連メトリクスを確認する

![メトリクス異常検知](/img/metrics-detect-anormal-details2.png)

ウィジェットでチャートの一部の領域をドラッグしてください。選択した領域で、該当メトリクス指標と関連する指標を纏めて確認することができます。
