遂行中のセッションが何の待機イベントで待機しているのか、頻繁に発生する待機イベントはどのようなのかを確認することができます。 5秒ごとに収集したアクティブセッションを待機(Wait)の観点から分析することができます。 

:::note

- この機能は、DBXエージェント1.9.0バージョン以降に対応しています。

* 収集された wait および stat 情報は、`v$system_wait_class`、`v$system_event`、`v$sysstat` ビューに基づいています。  
各指標の詳細な説明については、[次の文書](https://docs.oracle.com/en/database/oracle/oracle-database/19/refrn/V-SYSTEM_WAIT_CLASS.html)を参照してください。

:::
