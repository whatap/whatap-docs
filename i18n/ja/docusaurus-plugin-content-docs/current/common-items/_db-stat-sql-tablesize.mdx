ホーム画面 > プロジェクト選択 > ***統計/レポート*** > ***テーブルサイズ増減***

<ProdImg img='db-database-tablesize.png' alt='テーブルサイズ増減'/>

テーブルおよびインデックスの使用量を日別に収集し、期間の経過に伴う全体およびテーブル別使用量の増減情報を提供します。収集基準は、エージェント設定パラメータの`table_min_row`(既定値：1000件以上)、 `tables_hour `(既定値：午前5時) の値に従います。収集基準を変更するには、***管理*** > ***エージェント設定***メニューに移動してください。

1. ***基準日***を設定してください。今日の日付が既定値として設定されます。

2. データを比較する***比較日***を設定してください。

3. ***インスタンス***一覧で、データベース設定値を検索するインスタンスを選択します。

4. ![検索アイコン](/img/ico-btn-search.png)ボタンを選択します。

***テーブル増減***セクションでは***件数***、***データ***、***インデックス***別増減率と***全体***増減率を確認できます。***テーブル別の容量詳細***セクションでは***テーブル名***による詳細な増減率を確認することができます。 

:::note

- ***エージェント設定***メニューの詳細については、[次の文書](agent-settings)を参照してください。

- 収集したいテーブルの参照権限が必要です。次のコマンドを実行してアクセス権限を取得してください。

  ```sql

  grant select on [DB_NAME].[TABLE_NAME] to whatap;
  /* または */
  grant select on *.* to whatap;

  ```

- 参照した内容をcsv形式のファイルにダウンロードするには、![csvダウンロードアイコン](/img/ic-csv-download-ms.svg)ボタンを選択します。

:::
