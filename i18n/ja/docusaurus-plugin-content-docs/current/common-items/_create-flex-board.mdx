## ダッシュボード作成

Flexボードを作成し、ウィジェットを配置して独自のダッシュボードを作成できます。

1. <Cmdname sid="side_flexboard" className="uitext" />メニューから右上の<Cmdname sid="create_dashboard" className="uitext" />ボタンを選択します。

2. <Cmdname sid="create_dashboard" className="uitext" />画面が表示されると<Cmdname sid="name" className="uitext" />入力欄にダッシュボード名を入力します。

3. ウィジェットの配置方法を選択し、<Cmdname sid="create_dashboard" className="uitext" />ボタンをクリックします。

   - <Cmdname sid="fixed_flexboard" className="uitext" />：ピクセルベースで自由にウィジェットを配置できます。

   - <Cmdname sid="flexible_flexboard" className="uitext" />：ブラウザサイズを基準にグリッドレイアウトを提供します。<Cmdname sid="flexible_flexboard" className="uitext" />を選択後、グリッドを設定して<Cmdname sid="create_dashboard" className="uitext" />ボタンを選択します。

4. <Cmdname sid="widget_template" className="uitext" />選択画面でダッシュボードに配置するウィジェットを選択します。

   * メトリクスウィジェットを追加するには<Cmdname sid="widget_template" className="uitext" />から右側の**すべてのメトリクス**ボタンを選択します。

   :::note

   - <Cmdname sid="widget_template" className="uitext" />：一般的なモニタリング状況で重要な指標を要約し、事前に定義されたウィジェット一覧を選択できます。
   - <Cmdname sid="all_metrics" className="uitext" />：プロジェクトで収集しているすべてのメトリクスデータからウィジェットを作成します。メトリクスウィジェットの詳細については、[次の文書](flexboard-metric-widget)を参照してください。

   :::

5. 配置したウィジェットの位置を移動または、サイズを変更します。

   - 配置したウィジェットの上部にマウスカーソルを移動してください。カーソルの形が十字形に変更されましたら、ウィジェットをマウスでクリックした状態でドラッグすると位置を移動することができます。

   - 配置したウィジェットの右下にマウスカーソルを移動してください。カーソルの形が矢印の形に変更されたら、ウィジェットをマウスでクリックした状態でドラッグすると任意のサイズに調整することができます。

6. 追加するウィジェットのすべての配置が完了したら、<Cmdname sid="view_mode" className="uitext" />を選択してレイアウトを確認します。レイアウトを変更する場合は、<Cmdname sid="edit_mode" className="uitext" />を選択してウィジェットの配置を変更します。

7. すべての手順を完了したら、右上に![一覧へアイコン](/img/ico-exit.svg) (<Cmdname sid="go_to_list" className="uitext" />)ボタンを選択します。<Cmdname sid="side_flexboard" className="uitext" />メニューの<Cmdname sid="dashboard_list" className="uitext" />で生成したボードを確認できます。生成したボードを選択して、新たに生成したFlexボードを確認できます。

:::note

顧客の意見を反映してダッシュボードのテンプレートまたはウィジェットを追加する場合、必要なダッシュボードテンプレートまたはウィジェットがありましたら、[support@whatap.io](mailto:support@whatap.io)までお問い合わせください。

:::
