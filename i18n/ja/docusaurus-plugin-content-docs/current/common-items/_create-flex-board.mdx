## ダッシュボード作成

Flexボードを作成し、ウィジェットを配置して独自のダッシュボードを作成できます。

1.  **_Flexボード_**メニューから右上の**_+ダッシュボード作成_**ボタンを選択してください。

2.  **_ダッシュボード作成_**ウィンドウが現れると**_名前_**入力テキストボックスにダッシュボード名を入力してください。

3.  위젯 배치 방법을 선택해 **_대시보드 생성하기_** 버튼을 클릭하세요.
    -   **_固定レイアウトのFlexボード_**：ピクセルベースで自由にウィジェットを配置できます。
    -   **_反応型フレックスボード_**：ブラウザサイズに基づいてグリッド応答レイアウトを提供します。 **_反応型フレックスボード_**を選択するとグリッドを設定し、**_ダッシュボード作成_**ボタンを選択してください。

4.  **_ウィジェットテンプレート_**選択画面でダッシュボードに配置するウィジェットを選択してください。 メトリクスウィジェットを追加するには**_ウィジェットテンプレート_**から右側の**_すべてのメトリクス_**ボタンを選択してください。

    :::note

    -   **_ウィジェットテンプレート_**：一般的なモニタリング状況で重要に扱われる指標を要約して事前定義されたウィジェット一覧を選択することができます。
    -   **_すべてのメトリクス_**：ユーザーのプロジェクトで収集中のすべてのメトリック データに基づいてウィジェットを作成するときに使用します。 メトリクスウィジェットの詳細については、[次の文書](flexboard-metric-widget)を参照してください。
        :::

5.  配置したウィジェットの位置を移動するか、サイズを調整してください。

    -   配置したウィジェットの上部にマウスカーソルを移動してください。カーソルの形が十字形に変更されたら、ウィジェットをマウスでクリックした状態でドラッグしてください。位置を移動できます。
    -   配置したウィジェットの右下にマウスカーソルを移動してください。カーソルの形が矢印の形に変更されたら、ウィジェットをマウスでクリックした状態でドラッグしてください。希望のサイズに調整できます。

6.  追加するウィジェットのすべての配置を完了し、**_ビューモード_**を選択してレイアウトを確認してください。レイアウトを再修正するには、**_修正モード_**を選択してウィジェットの配置を変更してください。

すべてのプロセスを完了したら右上の![一覧へ行くアイコン](/img/ico-exit.svg)(**_一覧へ行く_**)ボタンをクリックしてください。**_Flexボード_**メニューの**ボード**一覧から生成したボードを確認することができます。生成したボードを選択して、新たに生成したFlexボードを確認することができます。

:::note

-   顧客の意見を反映してダッシュボードテンプレートまたはウィジェットを追加しようとしています。必要なダッシュボードテンプレートまたはウィジェットがあれば、[support@whatap.io](mailto:support@whatap.io)までお問い合わせください。
    :::
