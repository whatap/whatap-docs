## ログファイルのダウンロード

検索したログの結果を _CSV_ または _TXT_ ファイル形式でダウンロードできます。 

1. 画面中央右側の ![ダウンロードアイコン](/img/ico-download.svg) <Cmdname sid="download" className="uitext" /> ボタンをクリックします。

2. **CSVダウンロード** または **Logダウンロード** をクリックして、希望の形式のファイルをダウンロードします。
