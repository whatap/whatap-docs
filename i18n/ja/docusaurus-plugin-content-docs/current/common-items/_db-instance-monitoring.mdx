**_インスタンスモニタリング_**メニューでは、データベースサーバーの主要指標とアクティブセッションに関する詳細情報、ロックツリー(Lock Tree)、PQツリー(PQ tree)、プロセス(Process)情報をリアルタイムでモニタリングできます。また、問題となる可能性のある警告を確認することもできます。

## 指標チャート領域の活用

リアルタイムで収集するデータベースサーバーの情報をチャートで確認できます。各指標チャート右上の![設定アイコン](/img/ico-setting.svg)ボタンを選択します。チャートを選択すると**_カウント_**ウィンドウが表示されます。

<ImgLang img='db-select-count.png' desc='カウントウィンドウ' />

## レイアウト設定

指標チャートのレイアウトを設定するには、画面右上の![レイアウト設定アイコン](/img/ico-layout-set.svg)ボタンをクリックします。9つ(3x3)の指標チャートがデフォルトレイアウトとして設定され、ページに収まるチャートの数を変更できます。

<ImgLang img="db-layout-set.png" desc="レイアウト設定" />

## アクティブセッション領域の活用

画面の下に位置する**_アクティブセッション_**領域からリアルタイムで実行中のセッションとクエリ履歴を確認できます。**_query_**項目を選択すると、クエリ(**_query_**)のFullText及び**_Plan_**情報を確認できます。

## イベント領域の活用

インスタンスのモニタリング中にイベントが発生した場合、イベント領域に適切な履歴を表示します。イベントの履歴を選択すると、その時点での**_分析_** > **_カウント推移_**メニューに移動します。**_カウントの推移_**と**_アクティブセッション_**の履歴を確認できます。
