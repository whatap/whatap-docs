### レコード要約 {#anlaysis-trace-record-summary}

トランザクションの基本情報を確認するには、<Cmdname sid="TTL06125" className="uitext" />タブを選択します。 トランザクションの基本情報およびそのトランザクションが実行されたエージェント関連情報、メモリ割り当て使用量、クライアント関連情報を確認できます。

| 分類                 | プロパティ                          | 説明                                                                            |
| ------------------ | ------------------------------ | ----------------------------------------------------------------------------- |
| **トランザクション**       | <Cmdname sid="TTL06918" />     | アプリケーションで実行されたAPIまたはURL呼び出し、そのトランザクションのパスです。 正規表現式を使用して一定の形式に変換します。           |
| ^                  | <Cmdname sid="TTL06774" />     | 実際に呼び出されたURLパスです。 変数を含まない具体的なリクエストパスです。 元のURLがある場合は元のURL表示、元のURLがない場合は表示しません。 |
| ^                  | <Cmdname sid="TTL06077" />     | WhaTapモニタリングサービスに登録したプロジェクトの識別コードです。                                          |
| ^                  | <Cmdname sid="status" />       | そのトランザクションのHTTP応答ステータスのコードです。                                                 |

| ^                  | <Cmdname sid="TTL06086" />     | そのトランザクションで発生したエラー関連クラスです。                                                    |
| ^                  | <Cmdname sid="TTL06088" />     | そのトランザクションで発生したエラーメッセージです。                                                    |
| ^                  | <Cmdname sid="TTL06361" />     | トランザクションの実行時間です。                                                              |

| ^                  | <Cmdname sid="TTL06359" />     | クライアント関連情報の抽出に利用するブラウザ情報です。                                                   |
| ^                  | <Cmdname sid="TTL06108" />     | クライアントがブラウザで以前に利用したページアドレスまたは流入パスです。                                          |
| ^                  | <Cmdname sid="TTL06111" />     | 該当トランザクションの固有識別子(ID)です。                                                       |
| ^                  | <Cmdname sid="TTL06107" />     | マルチトランザクションの固有識別子(ID)です。 他のアプリケーションのトランザクションと並列に実行された場合に表示されます。               |

| ^                  | <Cmdname sid="TTL06587" />     | エージェントの固有識別子(ID)です。                                                           |
| ^                  | <Cmdname sid="TTL06356" />     | `okind`オプションで設定されたエージェントグループ名です。                                              |
| ^                  | <Cmdname sid="TTL06072" />     | エージェントグループの固有識別子(ID)です。                                                       |
| ^                  | <Cmdname sid="TTL06357" />     | `ondoe`オプションで設定したエージェントサーバー名です。                                               |
| ^                  | <Cmdname sid="TTL06309" />     | エージェント サーバーの固有識別子(ID)です。                                                      |
| **ステップ**(Step)     | <Cmdname sid="TTL06106" />     | HTTPメソッドです。 (GET、POST、PUT、HEADなど)                                             |
| ^                  | <Cmdname sid="TTL06362" />     | 外部Http Call時間です。                                                              |
| ^                  | <Cmdname sid="TTL06089" />     | 外部HTTP Call件数です。                                                              |
| ^                  | <Cmdname sid="TTL06972" />     | データベースに接続するまでにかかる時間です。                                                        |
| ^                  | <Cmdname sid="TTL06364" />     | SQLの実行時間です。                                                                   |
| ^                  | <Cmdname sid="TTL06097" />     | SQLの実行件数です。                                                                   |
| ^                  | <Cmdname sid="TTL06530" />     | レコードの照会にかかった時間です。 ネストが発生するか、他のロジックが含まれる場合があります。                               |

| ^                  | <Cmdname sid="TTL06080" />     | クライアントの国の情報です。                                                                |
| ^                  | <Cmdname sid="TTL06078" />     | クライアントの国内の都市情報です。                                                             |
| ^                  | <Cmdname sid="TTL06485" />     | クライアントが利用したブラウザの種類です。                                                         |

| ^                  | <Cmdname sid="TTL06420" />     | クライアントの固有識別子です。                                                               |

:::note

- アプリケーションの種類や設定、ステップの種類によって収集する情報は異なる場合があります。 **コンテナ**項目は、アプリケーションがコンテナ環境で実行されている場合にのみ表示されます。

- エラー関連項目は、そのトランザクションでエラーが発生した場合に表示されます。

- トランザクション属性の一部は、統計機能を提供します。 ![統計アイコン](/img/ic-static.svg)ボタンを選択すると、時系列チャートを通じて照会時間中に発生したトランザクション状況を把握できる画面が表示されます。 チャート内の棒グラフを選択すると、<Cmdname sid="side_analysis" className="uitext" /> &gt; <Cmdname sid="side_txSearch" className="uitext" />メニューに移動して詳細照会できます。

- WhaTapは、クライアントに関する情報をデフォルトで保存します。 ユーザーデータの収集に関するエージェント設定の詳細については、[次の文書](agent-number-of-user)を参照してください。
