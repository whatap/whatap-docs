<p>チャートおよび収集機能を強化した{props.product}モニタリングV2をアップデートしました。 最新の機能を利用するには、エージェントをアップデートしてください。</p>

:::note

<p>{props.product}モニタリングV2を利用するエージェントの最小バージョンは{props.ver}です。 エージェント更新の詳細については、<Link to='agent-manage'>次の文書</Link>を参照してください。</p>

:::

- **DB性能指標の情報提供**

  <ProdImg img='new-feature-01.png'/>

  指標のヘルプとチューニングに関するヒントを追加し、ユーザーが数多くの指標を簡単に理解できるようにカテゴリー化して利便性を提供します。

- **ダッシュボード機能の改善**

  <ProdImg img='new-feature-02.png'/>

  ダッシュボードでは、1つのチャートに複数の指標を追加する機能が追加されました。 また、プリセット機能を追加してユーザーが必要な指標を素早く確認することができます。

- **UI改善**

  インスタンスモニタリングフィルター機能の改善

  <ProdImg img='new-feature-03.png'/>

  セッションキル機能UIの改善

  <ProdImg img='new-feature-04.png'/>

  インスタンスモニタリングのフィルター機能を改善し、セッション強制終了機能と関連するユーザーインターフェース(User Iterface)を改善しました。

- **Flexボードテンプレートおよびウィジェットの追加提供**

  <ProdImg img='new-feature-05.png'/>

  DB指標の多くをメトリクスに変換し、追加のテンプレートとウィジェットを提供します。

- **ログ**

  <ImgLang img='new-feature-06.png'/>

  **_ライブテール_**と**\*ログトレンド**、**_ログ検索_**、ログ通知機能では、**DBログ**の検索とトレンド分析、ログ関連のアラートの受信を行うことができます。
