すべての権限を一括で適用するには、次のように*java.policy*ファイルに設定を追加します。

```java title='$JAVA_HOME/jre/lib/security/java.policy'
grant {
   permission java.security.AllPermission;
};
```

### java.io.FilePermissionエラーが発生した場合

![java.io.FilePermision error](/img/troubleshooting-permission-error.png)

次のように*java.policy*ファイルに設定を追加します。

```java title='$JAVA_HOME/jre/lib/security/java.policy'
grant {
   ...
   permission java.io.FilePermission {エラーメッセージから確認されたパッケージパス}, "read"
};
```

```java title='$JAVA_HOME/jre/lib/security/java.policy'
grant {
   ...
   permission java.util.PropertyPermission {エラーメッセージから確認されたパッケージパス}, "read"
};
```
