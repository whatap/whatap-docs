## DBXエージェントダウンロード

DBXエージェントファイルをダウンロードします。 次の2つの方法を利用します。

- `wget`コマンドを使用してダウンロードできます。

  <InDoc product="postgresql">

  ```bash title="BASH"
  wget -O whatap.agent.database.tar.gz "https://service.whatap.io/download/dbx_agent?type=postgresql&format=tar.gz"
  ```

  </InDoc>

  <InDoc product="mysql">

  ```bash title="BASH"
  wget -O whatap.agent.database.tar.gz "https://service.whatap.io/download/dbx_agent?type=mysql&format=tar.gz"
  ```

  </InDoc>

  <InDoc product="altibase">

  ```bash title="BASH"
  wget -O whatap.agent.database.tar.gz "https://service.whatap.io/download/dbx_agent?type=altibase&format=tar.gz"
  ```

  </InDoc>

- コマンドでダウンロードできない場合は、WhaTapモニタリングサービス画面で<Cmdname sid="download" className="uitext" />ボタンを選択します。

DBXエージェントファイルをダウンロードして圧縮を解凍してください。

```bash title="BASH"
tar -zxvf whatap.agent.database.tar.gz
```

:::note

セキュリティ設定によりtar形式のファイルをダウンロードできないユーザーのためにZIP形式のファイルも一緒に提供します。 インストール画面で<UI>.zip</UI> <Cmdname sid="download" className="uitext" />ボタンを選択します。

:::
