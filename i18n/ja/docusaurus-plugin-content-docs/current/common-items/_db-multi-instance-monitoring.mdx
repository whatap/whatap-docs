**_マルチインスタンスモニタリング_**メニューでは、多数のデータベースインスタンスのコア指標を統合して確認する機能を提供します。各インスタンスの間でロードバランシングの状況を比較できる機能も提供します。

詳細については、次の動画を通じて参照してください。

<Youtube ko="JWoW9v32-OU" en="JWoW9v32-OU" ja="JWoW9v32-OU" />

## 状態およびロードバランス領域の活用

特定の指標を比較およびモニタリングするには、**_状態_**と**_ロードバランス_**各チャートの右上の![設定アイコン](/img/ico-setting.svg)ボタンを選択します。比較指標を変更できます。

## アクティブセッション領域の活用

マルチインスタンスモニタリング中に特定のインスタンスで実行中の**_アクティブセッション_**、**_ロックツリー_**、**_プロセス情報_**を確認するには、インスタンス名ボタンをクリックします。
