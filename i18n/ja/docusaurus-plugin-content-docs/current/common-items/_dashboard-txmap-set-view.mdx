## 照会基準の変更

<Cmdname sid="transaction_map" />のチャート領域、左上のボタンを選択してデータ照会基準を変更することができます。

- <Cmdname sid="elapsed" className="uitext" />: トランザクションが実行された時間を基準にデータを表示

- <Cmdname sid="httpcTime" className="uitext" />: HTTP呼び出しに対する応答時間を基準にデータを表示

- <Cmdname sid="sqlTime" className="uitext" />: SQL実行時間を基準にデータを表示

- <Cmdname sid="TTL06401" className="uitext" />: CPU使用時間を基準にデータを表示

## 参照結果フィルタリング

<Cmdname sid="transaction_map" />のチャート領域、右上ではTotal、Error、Readフレーズを選択して結果を区分できます。

***Total***: 全体トランザクション表示

***Error***: エラーが発生したトランザクションのみ表示

***Read***: サーバーで読み込んだトランザクションの数を表示

例えば、エラーを選択した後、チャートのトランザクションをドラッグすると、エラー状態のトランザクションリストが表示されている<Cmdname sid="TTL06209" className="uitext" />画面が開きます。 
もう一度全体のトランザクションを照会するには***Total***をクリックします。

:::tip

**Y軸の調整**

- チャートの左上の![上方向アイコン](/img/up-arrow.svg)または![下方向アイコン](/img/down-arrow.svg)ボタンを選択すると、Y軸の最大値を簡単に変更できます。 また、キーボードの上または下方向のボタンを押して、同じ機能を利用できます。

  Y軸の区間によって![上方向アイコン](/img/up-arrow.svg)または![下方向アイコン](/img/down-arrow.svg)ボタン、キーボードボタンを利用する際の動作方法については、以下を参照してください。

  - 10秒から50秒の区間は10秒間隔に変更

  - 50秒から300秒は50秒間隔に変更

  - 300秒以上は100秒間隔に変更

- Y軸の最大値を直接入力して変更することができます。 ![編集アイコン](/img/ico-edit.svg)ボタンを選択し、希望する値を入力して<Cmdname sid="apply" className="uitext" />ボタンを選択します。

:::

:::note

<Cmdname sid="transaction_map" />のチャートに表示されるトランザクションの最大数(<span class="uitext">Limit</span>)は、2,000,000個です。

フィルターを適用して照会する時は、Total(画面に表示される個数)よりReadの値が大きい場合があります。

:::
