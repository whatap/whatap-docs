- WhaTapモニタリングサービスに登録していないアカウントには、<code><Cmdname sid="new_account" /></code>タグを表示し、会員登録メールを送信します。

- メンバーリストのEメールの右側には、MFA適用及びSSO連動可否によって`MFA`または`SSO`タグを表示します。

- 招待を完了する前にメンバー招待をキャンセルする場合は、<Cmdname sid="member_invite_list" className="uitext" replace="({length})" />から![削除アイコン](/img/ico-trash-red.svg)ボタンを選択します。

- 形式が正しくないメールアドレスを入力する場合は、招待されません。

- すでに招待されたメンバーは追加で招待できません。上位の組織やグループのメンバーは自動的に追加されます。
