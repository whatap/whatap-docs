## Error: Not found PHP ini directory

PHP環境の中で`Scan this dir for additional .ini files`項目の値を確認できなかった場合に発生します。PHPコンパイルインストールオプション`--with-config-file-scan-dir=PATH`が設定されていない場合、対応できる環境はありません。 

次の説明書を参照して、[PHPコマンド（CLI）の正しい場所](#check-php-cli-path)を確認した上で、[PHP拡張モジュールおよびwhatap-phpサービス選択インストール](#extension-modul-manual-setting)に進みます。

```bash
$ sudo php -i | grep '.ini files'

Scan this dir for additional .ini files => (none)
```
