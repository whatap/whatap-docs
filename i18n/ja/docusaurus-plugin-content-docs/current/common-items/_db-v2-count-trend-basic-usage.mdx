## 基本画面ガイド

<ProdImg img='db-compare-count-trend.png' desc='カウント推移' />

- ![数字1](/img/number-01.png) <Cmdname sid="TTL06202" className="uitext" />、![数字2](/img/number-02.png) <Cmdname sid="TTL06203" className="uitext" />、<Cmdname sid="instance" className="uitext" />項目を選択すると、すべてのウィジェットに選択した項目を基準に、データが自動的に反映されます。

- ウィジェットに表示されている指標の意味を知りたい場合は、名前の横の![情報アイコン](/img/ic-info-circle.svg)ボタンを選択するか、[次の文書](metrics-data-list)を参照してください。

  :::note

  ![情報アイコン](/img/ic-info-circle.svg)ボタンのツールチップ機能は多言語ではサポートされていません。

  :::

- 他のウィジェットの位置を変更する場合は、ウィジェットの上部を選択した状態でドラッグします。 ただし、ウィジェットのサイズは変更できません。

- ![数字3](/img/number-03.png) <Cmdname sid="active_sessions" className="uitext" />テーブルの左側には、<Cmdname sid="TTL06202" className="uitext" />、右側には、<Cmdname sid="TTL06203" className="uitext" />に該当するデータを表示します。

- ![数字3](/img/number-03.png) <Cmdname sid="active_sessions" className="uitext" />テーブル一覧の文字の色は黒 → <span class='slow'>オレンジ</span> → <span class='vslow'>赤</span>の順でセッションの実行速度が遅いことを意味します。
