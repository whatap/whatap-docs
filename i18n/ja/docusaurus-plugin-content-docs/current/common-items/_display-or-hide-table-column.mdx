<Title hashid='display-or-hide-column' level={props.level}>テーブルカラムの表示または非表示</Title>

<InDoc product="browser">

ブラウザーエラー一覧テーブルのカラム ヘッダー項目を表示または非表示できます。

</InDoc>

1. テーブルの右上の![カラムアイコン](/img/ico-column.svg)ボタンを選択します。

2. <Cmdname sid="column_settings" className="uitext" />ウィンドウが表示されたら、左側のリストで目的の項目を選択・解除します。

   右のリストから![削除アイコン](/img/ico-close.svg)ボタンを選択してカラムを非表示にすることもできます。

3. すべての設定を完了した後<Cmdname sid="BTN06053" className="uitext" />ボタンを選択します。 設定した内容がテーブルに適用されます。
