ダッシュボードメニューの機能を改善したデータベースモニタリングV2サービス画面を利用できます。 

1. <Cmdname sid="side_management" className="uitext" /> &gt; <Cmdname sid="side_projectManagement" className="uitext" />メニューに移動します。

2. <Cmdname sid="TTL06144" className="uitext" />セクションで<Cmdname sid="TTL06145" className="uitext" />を**V2**に選択します。

3. <Cmdname sid="save" className="uitext" />ボタンを選択します。

:::note

- 過去のデータを照会する際に画面に表示されないデータがある場合は、<Cmdname sid="TTL06145" className="uitext" />を**V1**に変更し、以前のバージョンの画面からデータを照会できます。

- 互換性を提供するためのこの機能は、2025年6月26日まで提供されます。

:::
