## テンプレート{#template}

**_警告通知_** > **_イベント設定_**で**_+ イベント追加_**ボタンを選択します。 **_テンプレート_**から希望する項目を選択します。 次のようにアラート通知テンプレートを提供します。

- <Cmdname sid="CNT06830" className="uitext" />

  平均**jitter**(`avg_jitter`)が30msを超えると、警告通知を送信します。

- <Cmdname sid="CNT06831" className="uitext" />

  平均**latency**(`avg_latency`)が200msを超えると、警告通知を送信します。

- <Cmdname sid="CNT06832" className="uitext" />

  データ伝送量(**bps**)が128Mib/sを超えると、警告通知を送信します。

- <Cmdname sid="CNT06833" className="uitext" />

  パケット伝送量(**pps**)が50Kpps以上の場合は、警告通知を送信します。

- <Cmdname sid="CNT06834" className="uitext" />

  セッション(session)集計量(`session_count`)が100以上の場合は、警告通知を送信します。
