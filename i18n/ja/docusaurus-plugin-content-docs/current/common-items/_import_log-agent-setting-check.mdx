**ホーム画面 > プロジェクト選択 > **ログ** > **ログ設定\*\*

1. WhaTapモニタリングサービスの初期画面でプロジェクトを選択した後、**ログ** > **ログ設定** メニューに移動してください。 

2. **ログモニタリング開始**タブの **1. エージェント設定の確認** の案内に従って、エージェント設定を確認してください。

### エージェント設定

1. プロジェクトの **管理** > **エージェントの設定** メニューをクリックしてください。

2. <Cmdname sid="write_options" className="uitext" /> の **オプション選択** ドロップダウンリストから <Cmdname sid="direct_input" className="uitext" /> をクリックし、`logsink_enabled=true` オプションを入力して **適用** ボタンをクリックして追加してください。

   - キー入力欄に `logsink_enabled` を入力してください。

   - 値入力欄に `true` を入力してください。

3. ログモニタリングを適用するには、アプリケーションを再起動します。
