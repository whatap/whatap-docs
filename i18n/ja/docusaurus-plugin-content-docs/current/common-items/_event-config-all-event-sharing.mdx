## すべてのイベント設定を共有する {#share-event-setting}

基本イベントを含むすべてのイベント設定を**JSONまたはExcel形式でダウンロード**できます. 同じタイプの製品設定を共有すると、繰り返されるイベント設定タスクを減らすことができます。

### エクスポート {#export-event-setting}

すべてのイベント設定を含むJSONファイルをダウンロードできます。 ダウンロードしたファイルはJSON一括修正機能で編集できます。 画面上部の![ダウンロードアイコン](/img/ico-download.svg) <Cmdname sid="BTN07762" className="uitext" />ボタンをクリックします。

- JSONファイル名は、*integrated-event-`YYYY`-`MM`-`DD`.json*の形式です。

- JSONデータの構造と`events`属性の下位属性は製品(`platform`)または製品タイプ(`productType`)によって異なる場合があります。

:::caution

JSONデータの属性のうち`productType`または`platform`属性の値を任意に修正すると、他のプロジェクトでイベントが動作しない場合があります。

:::

**JSONデータ構造**

```json title="integrated-event"
{
  "productType": "{productType}",
  "platform": "{platform}",
  "events": {
    "HITMAP": {
      ...
    },
    "LOG_REALTIME": [
      {...}
    ],
    "METRICS": [
      {...}
    ],
    "COMPOSITE_LOG": [
      {...}
    ],
    "BASIC": [
      {...}
    ],
    "COMPOSITE_METRICS": [
      {...}
    ],
    "ANOMALY": [
      {...}
    ]
  }
}
```

### インポート {#import-event-setting}

1. 画面上部の![編集アイコン](/img/ico-edit.svg) <Cmdname sid="BTN07761" className="uitext" />ボタンをクリックします。

2. JSON編集画面が開くと、画面の下の***インポート***ボタンをクリックします。

3. ![ダウンロードアイコン](/img/ico-download.svg) <Cmdname sid="BTN07762" className="uitext" />機能を通じてダウンロードしたJSONファイルを選択します。

4. <Cmdname sid="overwrite" className="uitext" />ボタンを選択します。

5. 確認メッセージが表示されたら<Cmdname sid="overwrite" className="uitext" />ボタンをクリックします。

:::caution

上書き後の以前のイベント設定では復旧できません。

- `productType`または`platform`属性の値を任意に修正すると、他のプロジェクトでイベントが正常動作しない場合があります。

- JSON編集画面には、JSON検証(Validation)機能が組み込まれています。 形式や値が正しくない場合は、編集画面の下にエラーメッセージが表示され、<Cmdname sid="overwrite" className="uitext" />ボタンは非活性化されます。

:::
