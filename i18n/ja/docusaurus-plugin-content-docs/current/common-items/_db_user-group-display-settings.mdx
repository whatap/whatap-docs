<Cmdname sid="users_group_display_settings" className="uitext" />は、特定のユーザーグループに対してのみメニューやプリセットを表示できるように制限する機能です。  
管理者は表示グループを作成し、目的のメニューやプリセットに関連付けて表示許可を設定することで、ユーザーの役割や権限に応じて不要な項目を非表示にし、必要な情報のみを表示できます。

:::note

この設定は、データベース製品のプロジェクトでメンバー管理権限を持つユーザーのみが可能です。

:::

## 基本画面

<Cmdname sid="users_group_display_settings" className="uitext" />は、タグ付けされたユーザーに対してのみ特定のプリセット、メニュー、機能を表示するように設定できます。

<ImgLang img='db-user-group-display-settings.png' desc='ユーザーグループ別表示設定の基本画面' />

### 表示許可の設定

表示グループが設定されていない場合、デフォルトでは **![有効アイコン](/img/ic-radio.svg)すべて表示** となります。**表示グループのみ表示** を選択すると、そのグループにのみ表示が許可されます。

- **表示グループのみ表示**：設定された表示グループに含まれるユーザーにのみページが表示されます

- **すべて表示**：すべてのユーザーにページが表示されます（表示グループ設定に関係なく）

### 表示ロールの追加

表示グループを追加する際に、どのページに対して表示ロールを付与するかを設定できます。

1. **ユーザーグループ別表示設定**画面の右上にある ![追加アイコン](/img/ic-s-plus.svg) **表示ロールの追加** ボタンをクリックします。

2. **表示グループの作成**で、**表示グループ名**と**表示グループの説明**を入力します。

    a. ![追加アイコン](/img/ic-s-plus.svg) **追加** ボタンをクリックし、メールアドレスまたはユーザー名でユーザーを追加します。

    b. **メニュー/プリセット**または**機能**に対して**表示グループに追加**トグルをオンにして、表示グループに追加します。

3. **確認**ボタンをクリックして**表示グループ**を作成します。

## 表示グループの追加

1. **ユーザーグループ別表示設定**で、表示グループを関連付ける**メニュー/プリセット**の表示グループ ![追加アイコン](/img/ic-s-plus.svg) をクリックします。

2. ドロップダウンから ![追加アイコン](/img/ic-s-plus.svg) **表示グループの作成** をクリックします。

3. **表示グループの作成**で、グループ名と説明を入力します。

    a. ![追加アイコン](/img/ic-s-plus.svg) **追加** ボタンをクリックし、メールアドレスまたはユーザー名でユーザーを追加します。

    b. **メニュー/プリセット**または**機能**に対して**表示グループに追加**トグルをオンにして、表示グループに追加します。

4. **確認**ボタンをクリックして表示グループを作成します。

5. 作成された表示グループを選択し、**適用**ボタンをクリックして反映します。

## 表示グループの編集

1. 表示グループを編集するには、表示グループ一覧から ![追加アイコン](/img/ic-s-plus.svg) をクリックします。

2. ドロップダウンから、編集したいグループの ![編集アイコン](/img/ic-edit.svg) アイコンをクリックします。

## 表示グループの削除

1. 削除したい表示グループが含まれる **メニュー/プリセット** の **表示グループ** をクリックします。

2. ドロップダウンから、削除対象の **表示グループ** の ![編集アイコン](/img/ic-edit.svg) アイコンをクリックします。

3. ![削除アイコン](/img/ic-delete-critical.svg) **表示グループ削除** ボタンをクリックします。

4. **適用** ボタンをクリックすると、表示グループが削除されます。

    - 表示グループがない場合、すべてのページと機能が表示されます。
