<Title level={props.level} hashid="create-db-user">DBユーザーファイル作成</Title>

<InDoc product="mongodb">

:::tip

userとpasswordを使用しない場合(認証なしでMongoDBに接続可能な場合) **_DBユーザーファイル生成_**する手順をスキップできます。

:::

</InDoc>

<InDoc product="redis">

:::tip

userとpasswordを使用しない場合 (認証なしでRedisに接続可能な場合)  **_DBユーザーファイル作成_**する手順をスキップできます。

:::

</InDoc>

データベース接続用に暗号化されたUIDを作成します。usernameとpasswordを入力し、シェルスクリプト(またはバッチファイル)を実行してください。 

<InDoc product='redis'>

- **Redis 6バージョン以下**: <em>uid.sh</em>シェルスクリプトファイル(または<em>uid.bat</em>バッチファイル)は、WhaTapデータベースエージェントをインストールしたパスで見つけることができます。userなしでパスワードのみある場合は、<code>DB_USER</code>を<code>""</code>に設定します。例）`./uid.sh "" whatap\!pwd`

- **Redis 6バージョン以降**: <em>uid.sh</em>シェルスクリプトファイル(または<em>uid.bat</em>バッチファイル)は、WhaTapデータベースエージェントをインストールしたパスで見つけることができます。userなしでパスワードのみある場合は、<code>DB_USER</code>を<code>default</code>に設定します。例）`./uid.sh default whatap\!pwd`

</InDoc>
