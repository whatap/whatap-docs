### 照会一覧をダウンロードする

<Cmdname sid="trace_list" className="uitext" />セクションで照会された結果をCSV形式のファイルにダウンロードできます。 ダウンロードしたCSVファイルの名前は、*profile*`{project_code}`*`YYYYMMDD`.csv*形式です。

1. トランザクションを照会した後に![ダウンロードアイコン](/img/ic-csv-download-ms.svg)ボタンを選択します。

2. <Cmdname sid="maximum_csv_lines" className="uitext" />オプションに希望する値を入力します。

3. <Cmdname sid="download" className="uitext" />ボタンを選択します。
