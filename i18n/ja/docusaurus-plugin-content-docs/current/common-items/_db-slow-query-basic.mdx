## 基本画面ガイド

<ProdImg img='db-slow-query.png' desc='slow query'/>

- ![指示線1](/img/number-01.png) <Cmdname sid="time" className="uitext" /> / <Cmdname sid="instance" className="uitext" />

  進行中のセッションの時間を変更したり、個々のインスタンスを選択できます。

  :::note

  <Cmdname sid="time" className="uitext" />オプションの利用方法の詳細については、[次の文書](usage-ui#timeselector)を参照してください。

  :::

- ![指示線2](/img/number-02.png)**_Slow Query_**

  SlowQueryが発生した時にチャートを表示します。マウスで特定の領域をドラッグすると![指示線3](/img/number-03.png)**_SlowQueryList_**で詳細情報を確認できます。

- ![指示線3](/img/number-03.png) **_Slow Query List_**

  **_Slow Query_** チャートでドラッグした領域のSlow Query詳細情報を一覧に表示します。**_SQL_**項目を選択すると、クエリ(**_query_**)のFull Textおよび**_Plan_**情報が確認できます。
