### チャートウィジェット

![number 3](/img/number-03.png)領域のチャートウィジェットの左上で指標名を確認できます。チャートウィジェットの右上で次のオプションを確認できます。

<Xclude product='cubrid,mysql,oracle,postgresql,redis,tibero,altibase,mongodb,mssql,mssql-v1,oracle-pro,oracle-v1,db2'><ImgLang img='metrics-chart-widget-upper.png' desc=''/></Xclude>

<InDoc product='cubrid,mysql,oracle,postgresql,redis,tibero,altibase,mongodb,mssql,mssql-v1,oracle-pro,oracle-v1,db2'><ProdImg img='metrics-chart-widget-upper.png' desc=''/></InDoc>

- タイムシフト：![left-arrow](/img/ic-caret-left.svg)![left-arrow](/img/ic-caret-right.svg)左矢印または右矢印ボタンを使用して、選択した時間範囲だけ-1または、+1ずつ移動することができます。 

  > 例えば、時間範囲が2月13日00:00\~2月14日00:00の場合、![left-arrow](/img/ic-caret-left.svg)左矢印を選択すると、2月12日00:00\~2月13日00:00までのデータが表示できます。

- 間隔/時間のマージ：![number 1](/img/number-01.png)トップメニューで指定した間隔と時間のマージを修正できます。

- モニタリング対象：![モニタリング対象アイコン](/img/ic-filter-bl.svg)アイコンを選択して、モニタリング対象を指定できます。選択しない場合は、全体として表示されます。

- 時間比較：![時間比較アイコン](/img/ic-hourgalss.svg)アイコンを選択すると、同じ指標の以前の時間帯の推移を比較できます。

- スナップショット：![スナップショットアイコン](/img/ic-camera.svg)アイコンを選択すると、ウィジェットのオプションを除くチャートのスナップショットを取得できます。

- CSV：![CSVダウンロードアイコン](/img/ico-csv.svg)アイコンを選択して、チャートを描画したデータをCSVファイルとしてダウンロードできます。

- <Cmdname sid="detail" className="uitext" />：![詳細アイコン](/img/ico-detail.svg)アイコンを選択して詳細を表示します。モニタリング対象が多い場合、モニタリング対象の指標推移を特定して確認できるため便利です。

:::note

メトリクスチャートウィジェットの上部にオプションが表示されない場合は、![その他のアイコン](/img/ico-more.svg)アイコンを選択します。

:::
