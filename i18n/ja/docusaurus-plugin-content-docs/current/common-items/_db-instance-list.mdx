ホーム画面 > プロジェクト選択 > **_ダッシュボード_** > **_インスタンスリスト_**

<ProdImg img="db-dashboard-instance-list.png" desc="インスタンスリスト" />

エージェントをインストールしたデータベースサーバーの一覧を確認できます。主なパフォーマンス指標とステータスを確認できます。パフォーマンス指標は5秒間隔で更新され、一覧のカラムは設定を使用して順序を変更したり、任意の指標に変更することができます。

:::note

**_状態_**カラムで正常でない状態アイコンをクリックすると**_イベント_**ウィンドウが表示されます。そのインスタンスで発生したイベントの履歴を確認できます。

:::
