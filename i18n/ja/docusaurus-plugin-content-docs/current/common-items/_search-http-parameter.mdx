### HTTPパラメータ照会 {#view-http-parameter}

<Cmdname sid="TTL06126" className="uitext" /> タブでそのトランザクションのHTTPパラメータを照会することができます。

1. ページの下にスクロールして、トランザクションの実行の最後のステップに移動してください。 

2. ![ロックアイコン](/img/ic-lock.svg) <Cmdname sid="TTL06520" className="uitext" />項目を選択します。

3. <UI>HTTP-PARAMETERS</UI>画面が表示されると、<Cmdname sid="BTN06764" className="uitext" /> ボタンを選択します。

4. 設定したParam Keyを入力します。

隠されたパラメータを確認できます。

<Xclude product='java,kubernetes,php,golang'>

:::note

- HTTPパラメータに関するエージェント設定は、[次の文書](agent-transaction#profile_http_parameter)を参考にしてください。

- <Cmdname sid="BTN06764" className="uitext" />：復号化されたパラメータの値を確認できます。 パスワードは<em><code>WHATAP_HOME</code>/paramkey.txt</em>ファイル内の6桁の文字列です。 別の文字列に変更できます。 <em>paramkey.txt</em>内のキーは、SQL変数の検索、HTTPクエリの検索、Threadの停止に必要です。

:::

</Xclude>
