Flexボードの画面モードによって提供する機能について説明します。

:::note

プロジェクト内の<Cmdname sid="side_flexboard" className="uitext" />メニューでは、ダッシュボードの変更権限のあるユーザーのみが<Cmdname sid="edit_mode" className="uitext" />および<Cmdname sid="admin" className="uitext" />モード、<Cmdname sid="filter" className="uitext" />機能にアクセスできます。アクセスできるメンバー権限は、次のとおりです。

  - プロジェクト**修正**権限 

  - <Cmdname sid="ITM06494" className="b600" />権限

  - **Site Admin**権限

メンバー権限の詳細については、[次の文書](../project/project-structure#member-auth)を参照してください。

:::

## 画面モード

<Cmdname sid="side_flexboard" className="uitext" />は、次のような画面モードを提供します。画面モードによって設定できる機能が異なります。

- <Cmdname sid="edit_mode" className="uitext" />：ウィジェットテンプレートの一覧からFlexボードのレイアウトにウィジェットを配置したり、ウィジェットを変更、削除、位置の調整ができます。

- <Cmdname sid="admin" className="uitext" />：ウィジェットの詳細なプロパティをJSON形式で管理できます。レイアウトに配置されたウィジェットの右上の![コードアイコン](/img/ico-custom-code.svg)ボタンをクリックします。

- <Cmdname sid="view_mode" className="uitext" />：ウィジェット配置、サイズ変更、詳細オプションなどの設定を完了した後、ダッシュボードなどの各ウィジェットのデータをリアルタイムでモニタリングできます。

<Cmdname sid="side_flexboard" className="uitext" />には、次の共通の機能を使用できます。

- データフィルタリング
- リアルタイムのデータ検索範囲を設定
- 印刷モード
- Flexボード管理
- データマージオプション

## データフィルタリング

Flexボードは数百種類の異なる種類のデータを処理します。データフィルタリング機能を使用して、モニタリング対象の概要を含む、Flexボードを設定します。

<ImgLang img='flexboard-filter.png' desc='Flexボード-フィルター' />

1. 画面の上の![フィルターアイコン](/img/ico-filter.svg)<Cmdname sid="filter" className="uitext" />をクリックします。 

2. 画面の左に<Cmdname sid="filter" className="uitext" />画面が表示されます。 フィルタリング対象を選択してください。

   - <Cmdname sid="agent" className="uitext" />オプションを選択した場合は、エージェントの種類またはアプリケーション一覧からフィルタリング対象を選択できます。フィルタリング対象をすべて選択するには、<Cmdname sid="select_all" className="uitext" />を選択します。必要に応じて、<Cmdname sid="search" className="uitext" />機能を使用して選択できます。

   - <Cmdname sid="metrics" className="uitext" />オプションを選択した場合は、<Cmdname sid="category" className="uitext" />オプションでプロジェクトまたはエージェントタイプからフィルタリング対象を選択できます。<Cmdname sid="category" className="uitext" />オプションは、<UI>+ And</UI>を選択し、追加できます。<Cmdname sid="contain" className="uitext" />、<Cmdname sid="exclude" className="uitext" />オプションを選択すると、文字列でフィルタリング対象を自動的に選択します。

3. フィルタリング対象の選択を完了し、<Cmdname sid="apply" className="uitext" />ボタンを選択します。 

4. フィルタリングの設定を保存して再利用するには、<Cmdname sid="save_filter" className="uitext" />ボタンを選択します。フィルター名を入力し、<Cmdname sid="save" className="uitext" />を選択します。<Cmdname sid="saved_filter" className="uitext" />は、<Cmdname sid="filter" className="uitext" />画面の下に表示されます。

#### フィルタリング条件

フィルタリング条件は、エージェントタイプとメトリプスタイプに分けられます。

| タイプ       | エージェント                                                                         | メトリクス                                                                                                                                  |
| --------- | ------------------------------------------------------------------------------ | -------------------------------------------------------------------------------------------------------------------------------------- |
| 選択オプション   | WhaTapプロジェクトのモニタリング対象<br/>(アプリケーション、サーバ、データベース、<br/>コンテナ及びKubernetes PODとNODE) | エージェントを含むすべてのメトリクスのタグ                                                                                                                  |
| 適用範囲      | すべてのウィジェット                                                                     | メトリクスデータを使用するすべてのウィジェット                                                                                                                |
| フィルタリング条件 | チェックボックス                                                                       | <Cmdname sid="equal" className="uitext" />、<Cmdname sid="contain" className="uitext" />、<Cmdname sid="exclude" className="uitext" />方式 |

:::note

- エージェントタイプは、モニタリングのデフォルト対象のみを選択オプションとして提供します。

- メトリクスタイプは、プロジェクトで収集されるすべてのメトリクスデータのタグを選択オプションとして提供します。メトリクスタイプは、クラウドモニタリングなどのエージェントなしでデータを収集し、フィルタリングを適用する場合に便利です。

- メトリクスの詳細については、[次の文書](metrics-intro)を参照してください。

:::

## 照会時間の設定

<Cmdname sid="side_flexboard" className="uitext" />メニューの画面右上のタイムセレクターを利用してウィジェットにデータを照会する時間を設定できます。![一時停止アイコン](/img/ic-pause-green.svg)ボタンをタップすると、リアルタイムモードを停止し、照会時間を設定できます。またリアルタイムモードに戻るには![再生アイコン](/img/ico-play-time.svg)ボタンを選択します。

## 印刷

Flexボード画面をPDFに保存および印刷できます。Flexボード画面の右上にある![印刷アイコン](/img/ico-print.svg)ボタンを選択します。画面を印刷モードに切り替えます。次のオプションを設定し、<UI>PDF</UI><Cmdname sid="download" className="uitext" />を選択します。

- <Cmdname sid="filter_visible_msg" className="uitext" />：画面でフィルターオプションをオン・オフできます。
- <Cmdname sid="change_landscape_mode_msg" className="uitext" replace="horizontal" />／<Cmdname sid="change_landscape_mode_msg" className="uitext" replace="vertical" />：画面を横向きまたは縦向きに変換できます。

印刷モードを終了するには、右上の![終了アイコン](/img/ico-close.svg)<Cmdname sid="close_print_mode" className="uitext" />を選択します。

## Flexボード管理

Flexボードのレイアウトを変更したり、ウィジェットの位置変更やサイズ変更、名前変更などの管理機能を提供します。 <Cmdname sid="flexboard_management" className="uitext" />機能は、<Cmdname sid="side_dashboard" className="uitext" /> &gt; <Cmdname sid="side_flexboard" className="uitext" />メニューの<Cmdname sid="dashboard_list" className="uitext" />で![編集アイコン](/img/ico-edit.svg)ボタンを選択して利用する方法もあります。

1. Flexボード画面の右上にある![設定アイコン](/img/ico-setting.svg)ボタンを選択すると、<Cmdname sid="flexboard_management" className="uitext" />画面が表示されます。 

    - <Cmdname sid="rename" className="uitext" />：Flextボードの名前を変更できます。変更する名前を入力します。

    - <Cmdname sid="layout" className="uitext" />：ウィジェットのレイアウトを変えたり、位置やサイズを変更できます。ウィジェットを削除するには![削除アイコン](/img/ico-trash.svg)ボタンをクリックします。

    - <Cmdname sid="dont_allow_users_to_modify" className="uitext" />：他のユーザーがFlexボードを変更できないように設定します。修正権限のあるメンバーのみが変更または削除、ウィジェットを編集できます。ボタンをクリックすると、<Cmdname sid="dont_allow_users_to_modify" className="uitext" /><UI>：On</UI>に変更されます。

    - <Cmdname sid="widget" className="uitext" /><UI>.json</UI>：Flexボードに含まれるウィジェットをJSON形式で編集および管理できます。

    - <Cmdname sid="option" className="uitext" /><UI>.json</UI>：Flexボードの設定をJSON形式で編集および管理できます。プロジェクト情報、Flexボードのフィルター、時間選択などの設定を編集できます。

    :::note

    <Cmdname sid="widget" className="uitext" /><UI>.json</UI>、<Cmdname sid="option" className="uitext" /><UI>.json</UI>機能は、ダッシュボードの管理者権限があるユーザーが利用できます。

    :::

2. すべての設定を完了した後、画面の右下にある<Cmdname sid="save" className="uitext" />ボタンをクリックします。

## ダッシュボードをお気に入りとして登録

ユーザが生成したFlexボードは、<Cmdname sid="side_dashboard" className="uitext" />メニューの下位にあるお気に入りのメニューとして登録できます。

<Cmdname sid="side_dashboard" className="uitext" /> > <Cmdname sid="side_flexboard" className="uitext" />メニューで<Cmdname sid="dashboard_list" className="uitext" />の![お気に入りアイコン](/img/ico-favorite.svg)ボタンを選択します。お気に入りに設定された項目は、![お気に入り設定アイコン](/img/ico-favorite-check.svg)に変更されます。
