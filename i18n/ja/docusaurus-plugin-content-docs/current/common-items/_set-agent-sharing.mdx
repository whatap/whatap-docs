### エージェント設定を共有する

エージェントの設定内容を json形式のファイルとして保存し、そのファイルを読み込んで他のエージェントに適用することができます。

1. <Cmdname sid="TTL06549" className="uitext" />で設定内容をJSONファイルに書き出しするエージェントを選択します。

2. 画面右上の![書き出しアイコン](/img/ico-export.svg)ボタンを選択します。 

3. jsonファイルをダウンロードしてください。

4. <Cmdname sid="TTL06549" className="uitext" />で他のエージェントを選択します。

5. 画面右上の![読み込みアイコン](/img/ico-import.svg)ボタンを選択します。

jsonファイルを通じて読み込んだエージェント設定を確認してください。
