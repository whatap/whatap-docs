![number 2](/img/number-02.png)領域は、オプションを参照する指標の一覧です。指標を検索するか、一覧で目的の指標を選択します。![number 1](/img/number-01.png)領域の上のメニューで指定した時間範囲のデータに基づいて![number 3](/img/number-03.png)領域でチャートウィジェットを使用してデータを表示することができます。

</InDoc>

:::note 

**データ結合**

データのマージは、<Cmdname sid="object_merge" className="uitext" />と<Cmdname sid="time_merge" className="uitext" />方法を提供します。 

- <Cmdname sid="time_merge" className="uitext" />は、ソースデータからフィールド値が同じデータを一定の間隔でマージします。

- <Cmdname sid="object_merge" className="uitext" />は、異なるフィールド値を持つデータの中からタグが一致する場合、そのデータをマージします。

:::
