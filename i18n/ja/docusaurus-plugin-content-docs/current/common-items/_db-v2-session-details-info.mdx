### セッション詳細情報を確認する{#session-details}

<Cmdname sid="active_sessions" className="uitext" />タブでセッション項目を選択すると<Cmdname sid="session_detail" className="uitext" />画面が表示されます。 選択したセッションについての詳細を確認できます。

<InDoc product="postgresql"><ImgLang img='db-session-details-info.png' desc='セッション詳細' /></InDoc>

<InDoc product="oracle-pro,oracle"><ImgLang img='db-session-details-info-oracle-pro.png' desc='セッション詳細' /></InDoc>

<InDoc product="mssql"><ImgLang img='db-session-details-info-mssql.png' desc='セッション詳細' /></InDoc>

- セッションに関する情報は、既定値でリアルタイムで照会されています。 タイムセレクターの![静止アイコン](/img/ico-pause.svg)ボタンを選択すると、リアルタイム照会を中止し、中止した時間ベースのデータを確認できます。 リアルタイム照会を中止した状態でデータを更新するには、![更新アイコン](/img/ico-refresh.svg)ボタンを選択します。

- ***Kill Session***：選択したセッションを停止できます。 <Cmdname sid="session_kill" className="uitext" />画面が表示されたらパスワード(***Parameter Key***)を入力し、<Cmdname sid="apply" className="uitext" />ボタンを選択します。

  :::note

  この機能は、リアルタイム照会モードでのみ利用できます。

  :::

<InDoc product="postgresql,oracle-pro,oracle">

- <Cmdname sid="TTL06721" className="uitext" />：選択したセッション履歴が確認できる<Cmdname sid="side_sessionHistory" className="uitext" />メニューに移動できます。

  :::note

  <Cmdname sid="side_sessionHistory" className="uitext" />メニューの詳細については、<Link to="analysis-session-history">次の文書</Link>を参照してください。

  :::

</InDoc>

<InDoc product="oracle-pro,oracle">

- ***Session Stat***: 選択したセッションの様々な性能メトリクスと統計情報を提供します。 セッションで実行された作業のタイプと回数、リソース使用量などが含まれます。

- ***Delta*** / ***Sigma***: 選択したセッションの変化率と統計的分析情報を提供します。 セッションのアクティビティに対する変化量を確認できます。

</InDoc>

- ***Session Info***: 選択したセッションの基本情報とステータスを含む詳細を確認できます。

<Xclude product="mssql">

- ***Wait Info***: 選択したセッションの待機イベントと待ち時間の情報を提供します。 特定の作業を実行するために、データベースシステムで待機する時に発生する待機イベントに関する情報も確認できます。

- ***SQL Text*** / ***Plan***：選択したセッションのSQLクエリ文とPlan情報を確認できます。 詳細については[次の文書](#query-details)を参照してください。

</Xclude>

<InDoc product="mssql">

- ***Stat***：選択したセッションの様々な性能メトリクスと統計情報を提供します。 セッションで実行された作業のタイプと回数、リソース使用量などが含まれます。

- ***Request SQL*** / ***Plan***：選択したセッションのSQLクエリ文とPlan情報を確認できます。 詳細については[次の文書](#query-details)を参照してください。

</InDoc>

<InDoc product="postgresql">

- ***Process Info***: 選択したセッションのプロセス情報を確認できます。

</InDoc>
