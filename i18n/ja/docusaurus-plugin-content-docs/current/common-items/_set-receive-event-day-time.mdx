## 曜日および時間別の通知を設定する

曜日別、時間別通知の受信有無を選択できます。<Cmdname sid="alert_receive_settings_by_member" className="uitext" />セクションのユーザー一覧の左端の![詳細アイコン](/img/ico-more-down.svg)ボタンを選択します。アラート通知を受信する曜日を選択するか、時間を入力します。通知の受信方法ごとに設定できます。
