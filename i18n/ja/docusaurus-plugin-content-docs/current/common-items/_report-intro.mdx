画面の上に<Cmdname sid="report" className="uitext" />で![down-arrow](/img/down-arrow.svg)ボタンをクリックすると、日刊、週刊、月刊に分類された各種分析レポートを確認できます。希望するレポート様式を選択し、照会したい日付と時間、フィルターオプションを設定してください。![검색 아이콘](/img/ico-btn-search.png)ボタンを選択すると、選択したレポートを確認できます。

レポートは、個々のプロジェクトのサービスの使用状況と障害の発生履歴を報告する文書です。サービスモニタリング担当者は、レポートを通じて関連部署の担当者とステータスを共有します。

モニタリングデータ分析は、サービスの改善方向を定める指標となるため、極めて重要です。ただし、複数のダッシュボードからデータを収集して文書化するには煩雑な場合があります。  

WhaTapの<Cmdname sid="report" className="uitext" />メニューは、**レポート作成業務の自動化**を支援します。毎週報告書を作成すること、決まった時間に報告書を共有すること、様々な書式を管理することなど、すべて<Cmdname sid="report" className="uitext" />メニューからできます。

:::tip

**レポート**のダウンロード、印刷やメール送信の予約を希望する場合は、そのボタンをクリックしてください。HTML形式でダウンロードできます。

:::

## レポートと統合レポートの違い

- <Cmdname sid="report" className="uitext" />は、1つのプロジェクトに関するレポートを作成するメニューです。したがって、プロジェクトを選択した後、サイドメニューの<Cmdname sid="report" className="uitext" />メニューから進入できます。

- <Cmdname sid="integrated_report" className="uitext" />は、複数のプロジェクトに関するレポートを作成するメニューです。プロジェクトを選択せずにホーム画面のサイドメニューから進入できます。<Cmdname sid="integrated_report" className="uitext" />の詳細については、[次の文書](integrated-report)で確認できます。

## レポートの種類

基本レポートは、大手企業、公共機関やITサービス企業などで実際に使用しているフォームです。 ご要望のフォームを作成したい場合は、support@whatap.ioまでご連絡ください。 要望されたレポートは、<Cmdname sid="report" className="uitext" />一覧に追加されます。

### レポートフォーム追加

:::caution

**制約事項**

- WhaTapモニタリングで現在提供しているレポートテンプレートの各項目を追加または修正、削除が必要な場合に提供できます。

- 現在提供しているレポートテンプレートにないデータや分析結果を追加するには、開発者によるレビューが必要です。

- データの追加および加工は、WhaTapモニタリングで収集および保存されたデータ(メトリクス)に限定します。他のツールや外部データは追加できません。

- 特定の顧客企業のニーズを満たす汎用的でないレポートフォームは提供できません。

- 詳細については、support@whatap.ioまたは営業担当者にお問い合わせください。

:::

#### **ディスクフィルタ対応テンプレート**

レポートメニューのレポート項目で**日間時間別サーバーの主要指標報告書**(ディスクフィルタ対応テンプレート)を選択した場合は、ディスクフィルターでマウントポイントを選択して適用できます。
