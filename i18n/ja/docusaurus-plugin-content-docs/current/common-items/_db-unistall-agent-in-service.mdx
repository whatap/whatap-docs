### サービス画面から削除する

[WhaTapモニタリングサービス](https://service.whatap.io)からエージェントを削除するには、**_エージェント一覧_**メニューに移動してください。エージェントを削除する前に、エージェントをインストールしたサーバーでエージェントプロセスを終了する必要があります。 

1. **_プロジェクトリスト_**からエージェントを削除するプロジェクトを選択してください。

2. 左側のメニューから**_管理_** > **_エージェント一覧_**を選択してください。

3. **_エージェント一覧_**で削除するエージェントの右側の![削除アイコン](/img/ico-trash.svg)ボタンをクリックしてください。

:::note

- 非アクティブ状態のエージェントのみ削除できます。

  <ImgLang img='db-disable-agent.png' desc='' />

- エージェントのバージョンによっては、削除機能をサポートしていない場合があります。最新バージョンのエージェントに更新してください。

:::
