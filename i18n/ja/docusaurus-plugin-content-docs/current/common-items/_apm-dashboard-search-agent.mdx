### エージェントを検索する{#searchagents}

プロジェクトに接続されたエージェントの一覧が多く、見つけるのが難しい場合は、エージェント検索機能を利用します。 画面の左上にある![検索アイコン](/img/ico-search.svg)<Cmdname sid="search" className="uitext" />ボタンをクリックすると、文字列を入力するボックスが表示されます。

<InDoc pages="dashboard"><ImgLang img="apm-dashboard-search-agent.png" desc="エージェント検索" /></InDoc>

入力した文字列に一致するエージェントのみが、エージェント一覧に表示されます。 検索したエージェントに基づいてダッシュボードのデータをフィルタリングするには、文字入力ボックスの右側にある<Cmdname sid="select" className="uitext" />ボタンをクリックします。

選択したエージェント項目を初期化するには、文字入力ボックスの右側の<Cmdname sid="close" className="uitext" />ボタンをクリックします。
