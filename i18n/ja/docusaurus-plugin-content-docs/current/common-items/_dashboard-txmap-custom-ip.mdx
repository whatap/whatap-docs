### ユーザーIPアドレスを追加

<UI>TOP 30</UI>セクションの<Cmdname sid="ip" className="uitext" />基準でトランザクションの実行件数を照会する時、特定のIPアドレスを追加して常時確認できる機能を提供します。

1. <UI>TOP 30</UI>セクションで<Cmdname sid="ip" className="uitext" />を選択します。

2. 一覧の一番上の入力欄に特定のIPアドレスを入力します。

3. ![追加アイコン](/img/ico-add.svg)ボタンをクリックします。

   <ImgLang img='txmap-filter-ip-added.png' desc='IP' />

一覧の一番上に入力したIPアドレスが固定されて表示されます。

:::note

追加したIPアドレスを削除するには、![削除アイコン](/img/ico-trash.svg)ボタンをクリックします。

:::
