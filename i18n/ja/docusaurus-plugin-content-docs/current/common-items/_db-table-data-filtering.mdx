<Title level={props.level} hashid="tablefilter">テーブルデータのフィルタリング</Title>

<ProdImg img='db-bottom-table-filter.png' desc='Filter' />

1. テーブルの右上にある![フィルターアイコン](/img/ico-filter.svg)ボタンを選択します。

2. テーブルのカラムヘッダー項目と条件を選択します。

3. <Cmdname sid="enter_condition" className="uitext" />テキストボックスに条件を入力します。

4. <Cmdname sid="save" className="uitext" />ボタンを選択します。
