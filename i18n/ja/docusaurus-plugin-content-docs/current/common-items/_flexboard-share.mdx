Flexボードを他のアカウントのユーザーと共有または再利用できます。

## メンバーとFlexボードを共有する

ユーザーが作成したFlexボードを他ユーザーと共有して同じダッシュボードを見ることができます。これにより、タスクを共有できます。共有したダッシュボードを修正した場合、共有されたすべてのユーザーは同じダッシュボードが表示されます。この機能はホーム画面 > **_Flexボード_**メニューからのみ使用できます。

1.  **_Flexボード_**メニューの**_ダッシュボード一覧_**で共有しようとするダッシュボードの![共有アイコン](/img/ico-flexboard-share.svg)ボタンを選択します。

    ![Flexボード共有](/img/flexboard-share-01.png)

2.  **_ダッシュボード共有_**ウインドウが表示されたら**_+アカウント_**ボタンをクリックします。

    ![Flexボード共有](/img/flexboard-share-02.png)

3.  ダッシュボードを共有するユーザーアカウントを選択します。

    ![Flexボード共有](/img/flexboard-share-06.png)

    -   共有対象を追加するには**_+アカウント_**ボタンを選択してください。ユーザーアカウントを追加で選択できます。

    -   ダッシュボードを共有できるユーザーアカウントは、選択したプロジェクトに属するユーザーアカウントです。

    -   ダッシュボードのすべてのプロジェクトに対する**修正権限**がない場合、「読み専用」のタグが表示されます。

4.  **_ダッシュボードを共有する_**ボタンを選択します。**_読み取り専用_**チェックボックスを選択すると、共有されたユーザーはダッシュボードを修正できません。

    ![Flexボード共有](/img/flexboard-share-03.png)

ダッシュボードを共有した項目は、**_ダッシュボード一覧_**で`共有`タグが表示されていることを確認できます。共有タグにカーソルを合わせると、ダッシュボードを共有したメンバーに関する情報が確認できます。

| ダッシュボードを共有している場合                          | ダッシュボードを共有した場合(ダッシュボードの所有者)               |
| ----------------------------------------- | ----------------------------------------- |
| ![Flexボード共有](/img/flexboard-share-04.png) | ![Flexボード共有](/img/flexboard-share-05.png) |

:::note 

**ダッシュボードの共有条件**

-   ダッシュボード共有機能は、ホーム画面 > **_Flexボード_**メニューからのみ利用できます。

-   ダッシュボードを共有するメンバーは、ダッシュボードのすべてのプロジェクトに対する**修正権限**を持っている必要があります。 

-   すべてのプロジェクトに対して**修正権限**がない状態でダッシュボードを共有する場合は、**_読み取り専用_**モードで共有します。アクセス権限の詳細については、[次の文書](../project/project-structure)を参照ください。

-   共有されたダッシュボードを別なユーザーに再共有する場合、ダッシュボードの属性を**_読み取り専用_**モードにすることはできません。

-   共有条件が満たす場合に、ダッシュボード共有ボタンが表示されます。

    ![Flexボード共有](/img/flexboard-sharable-list.png)

:::

:::note

-   ダッシュボード所有者がダッシュボードを削除する場合、ダッシュボードを共有しているすべてのユーザーアカウントもダッシュボードを削除します。
-   ダッシュボードを共有するユーザーがダッシュボードを削除した場合、そのユーザーの**_ダッシュボード一覧_**から削除されます。元のダッシュボードは影響を受けません。

:::
