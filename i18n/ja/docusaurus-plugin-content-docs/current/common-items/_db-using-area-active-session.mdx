## アクティブセッションのセクションを使用

画面の下にある**_アクティブセッション_**セクションからリアルタイムで実行中のセッション情報を確認できます。ここでは、テーブルの右上にあるボタン機能を説明します。

<Xclude product='postgresql,mysql'>

-   ![停止アイコン](/img/ico-pause.svg)：リアルタイムで収集するデータの変更を停止することができます。再起動するには![スタートアイコン](/img/ico-play-grey.svg)ボタンをクリックしてください。

</Xclude>

<InDoc product='postgresql,mysql'>

-   ![フィルターアイコン](/img/ico-filter.svg)：テーブルのカラムヘッダー項目に基づいて一覧をフィルタリングできます。

</InDoc>

-   ![更新アイコン](/img/ico-refresh.svg)：テーブルのリストを更新できます。

-   ![カラムアイコン](/img/ico-column.svg)：テーブル内のカラムヘッダー項目を表示または非表示にすることができます。

<InDoc product='postgresql,mysql'>

-   ![セッションキルアイコン](/img/ico-kill-session.svg)：稼働中のセッションを停止できます。

    :::note

    この機能は、**修正**権限のあるメンバーのみ利用できます。メンバー権限に関しての詳細な説明は[次の文書](../project/project-structure#project-auth)を参照してください。

    :::

-   ![ダウンロードアイコン](/img/ico-download.svg)：テーブルの内容をCSVファイルとしてダウンロードできます。

-   ![ポップアップアイコン](/img/ic-popup.svg):**_アクティブセッション_**、**_ロックツリー_**、**_プロセス情報_**を新しいウィンドウを開いて確認できます。一覧が多いため、大きな画面で一覧を確認する場合に便利です。

</InDoc>

<Xclude product='postgresql,mysql'>

-   ![フィルターアイコン](/img/ico-filter.svg)：テーブルのカラムヘッダー項目を基準に一覧をフィルタリングできます。ボタンをクリックすると、各ヘッダーカラムで**_含む_**、**_含まない_**、**_一致_**、**_一致しない_**の条件を設定できます。

    :::note

    フィルター条件のオプションを入力するテキストボックスが表示しない場合は、カラムの幅を調整してください。

    :::

-   ![ダウンロードアイコン](/img/ico-download.svg)：テーブルの内容をCSVファイルとしてダウンロードできます。

#### 進行中のセッションの停止

長期間進行中のセッションを停止することができます。

1.  **_アクティブセッション_**一覧の項目にマウスカーソルを移動してください。

2.  右側に表示される![セッション中止アイコン](/img/ic-kill-session.svg) ボタンを選択してください。

3.  **_セッション中止_**ウィンドウが表示されたら、パスワードを入力してください。

4.  **_適用_**ボタンを選択します。

そのセッションを停止します。

:::note

-   DBXエージェントのインストールパスの_paramkey.txt_ファイルからパスワードを確認できます。

-   この機能は、**修正**権限のあるメンバーのみ利用できます。メンバー権限に関しての詳細な説明は[次の文書](../project/project-structure#project-auth)を参照してください。

:::

#### グラフチャートセクションを拡大する

<ImgLang img='db-common-folding-table.png' desc='アクティブセッション' />

現在、表示画面が小さくてウィジェットを確認するのが難しい場合は、より広い画面でグラフチャートウィジェットを確認することができます。 画面の下部にある<Cmdname sid="active_sessions" className="uitext" />セクションから![下方向アイコン](/img/ic-caret-down.svg)ボタンを選択してください。<Cmdname sid="active_sessions" className="uitext" />セクションは縮小され、グラフチャート ウィジェットのセクションは拡大されます。

グラフチャートウィジェットセクションを縮小するには、画面下部に<Cmdname sid="active_sessions" className="uitext" />セクションの![上方向アイコン](/img/ic-caret-up.svg)ボタンを選択してください。

</Xclude>
