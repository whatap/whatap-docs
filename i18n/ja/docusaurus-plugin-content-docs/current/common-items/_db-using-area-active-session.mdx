  :::note

  </InDoc>

</InDoc>

- ![更新アイコン](/img/ico-refresh.svg)：テーブルのリストを更新できます。

### 進行中のセッションの停止

長期間進行中のセッションを停止することができます。

1. <Cmdname sid="active_sessions" className="uitext" />一覧で停止するセッション項目にマウスカーソルを移動します。

2. 右側に表示される![セッション中止アイコン](/img/ic-kill-session.svg) ボタンを選択します。

3. <Cmdname sid="session_kill" className="uitext" />画面が表示されたら、パスワードを入力します。

4. <Cmdname sid="apply" className="uitext" />ボタンを選択します。

そのセッションを停止します。

:::note

- DBXエージェントのインストールパスの*paramkey.txt*ファイルからパスワードを確認できます。

- この機能は、**修正**権限のあるメンバーのみ利用できます。メンバー権限に関しての詳細な説明は[次の文書](../project/project-structure#project-auth)を参照してください。

:::

</InDoc>

### グラフチャートセクションを拡大する

<ImgLang img='db-common-folding-table.png' desc='アクティブセッション' />

現在、表示画面が小さくてウィジェットを確認するのが難しい場合は、より広い画面でグラフチャートウィジェットを確認することができます。 画面の下部にある<Cmdname sid="active_sessions" className="uitext" />セクションから![下方向アイコン](/img/ic-caret-down.svg)ボタンを選択します。<Cmdname sid="active_sessions" className="uitext" />セクションは縮小され、グラフチャート ウィジェットのセクションは拡大されます。

グラフチャートウィジェットセクションを縮小するには、画面下部に<Cmdname sid="active_sessions" className="uitext" />セクションの![上方向アイコン](/img/ic-caret-up.svg)ボタンを選択します。

</InDoc>
