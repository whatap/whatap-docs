-   **whatap.server.host**<span class='type'>ip_address</span><br/>
    デフォルト値`127.0.0.1,127.0.1`<br/>エージェントが収集したデータを転送するサーバーを設定します。収集サーバーの二重化で2つ以上のIPアドレスを持っている場合は、コンマ（,）を区切り文字として利用してください。設定したIPアドレスには、収集サーバーであるproxyサーバーがリスニング（Listening）状態でサービスされなければなりません。

-   **whatap.server.port**<span class='type'>tcp_port</span><br/>
    デフォルト値`6600`<br/>収集サーバーポートを設定します。ポートは1つだけ設定できるので、`whatap.server.host`オプションに設定した収集サーバは同一ポートを使用する必要があります。

-   **tcp_so_timeout**<span class='type'>MiliSecond</span><br/>
     デフォルト値`60000`<br/>
     収集サーバーと通信するTCPセッションのSocket Timeout値を指定します。

-   **tcp_connection_timeout** <span class='type'>MiliSecond</span><br/>
    デフォルト値 `5000`<br/>収集サーバーと通信するTCPセッションのConnection Timeout値を指定します。

-   **net_send_max_bytes**<span class='type'>Byte</span><br/>
     デフォルト値`5242880`<br/>
     エージェントがデータを収集し、ネットワークに一度に送信できる最大バイトのサイズです。

-   **net_udp_port**<span class='type'>tcp_port</span><br/>
    デフォルト値`6600`<br/>WhaTapエージェントは、トレーサーからUDPを通じて収集したデータを収集サーバーに転送します。最初のUDPサーバーのポートを指定できます。デフォルトで提供される6600番ポートが使用されているときに、このオプションを使用します。

    <!-- IIS를 다시 시작해야 적용할 수 있습니다. -->
    {@include: ../common-items/_donet-iis-restart.mdx}
    :::

-   **net_send_queue1_size**<span class='type'>Int</span><br/>
    デフォルト値`256`<br/>TCPデータ処理のためのQueueバッファを設定します。優先順位の高いデータのためのQueueです。

-   **net_send_queue2_size**<span class='type'>Int</span><br/>
    デフォルト値`5000`<br/>TCPデータ処理のためのQueueバッファを設定します。トレースデータのためのQueueです。
