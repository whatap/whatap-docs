## 基本イベントを共有する {#share-basic-event}

基本イベント設定をJSONファイルとして保存し、他のユーザーと設定を共有したり、他のユーザーの設定を読み込むことができます。

### エクスポート {#export-json}

1. 画面右上の<span class="uitext">JSON</span> <LinkImage img="ico-export.svg" className="ico" />ボタンを選択します。 

2. JSON編集ウィンドウが表示されたら<LinkImage img="ico-export.svg" className="ico" /> <Cmdname sid="export" className="uitext" />ボタンを選択してください。

3. JSONファイルがダウンロードされたら、共有する他のユーザーに転送してください。

:::note

JSONファイル名は、*event-rules-`YYYY`-`MM`-`DD`.json*の形式です。

:::

### インポート {#import-json}

1. 画面右上の<LinkImage img="ico-import.svg" className="ico" />ボタンを選択してください。

2. <Cmdname sid="export" className="uitext" />機能でダウンロードしたJSONファイルを選択してください。

3. JSON編集ウィンドウが表示されたら<Cmdname sid="add_to_list" className="uitext" />または<Cmdname sid="overwrite" className="uitext" />ボタンを選択してください。

:::caution

この機能は、同じ種類の製品間で利用することをお勧めします。 他の製品のプロジェクトからイベント設定を取得することはできますが、正常に機能しません。

:::
