## アラート通知のユーザー設定

<Cmdname sid="MNU07360" className="uitext" />メニューからユーザーの通知受信レベル、受信手段、曜日・時間などを設定できます。

1. 画面の右上にあるプロフィールアイコンを選択します。

2. ポップアップメニューが表示されたら<Cmdname sid="MNU07360" className="uitext" />ボタンを選択します。

3. 画面を下にスクロールし、<Cmdname sid="notification_setting" className="uitext" />セクションへ移動してください。

4. 受信レベル、受信方法、曜日・時間を設定し、<Cmdname sid="save" className="uitext" />ボタンを選択します。
