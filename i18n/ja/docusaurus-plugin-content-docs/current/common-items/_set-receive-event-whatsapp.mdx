### WhatsApp通知受信を追加する

WhatsApp経由で通知を受信できます。

<ImgLang img='set-event-whatsapp-notification.png' desc='WhatsApp'/>

1. 画面の右上のプロフィールアイコンを選択します。 

2. ポップアップメニューが表示されたら<Cmdname sid="MNU07360" className="uitext" />ボタンを選択します。 

3. <Cmdname sid="account_user_phone" className="uitext" />セクションで<span class="uitext">WhatsApp</span>ボタンを選択します。

4. <Cmdname sid="phone" className="uitext" />に認証番号を受信する電話番号を入力します。

5. <Cmdname sid="BTN06578" className="uitext" />ボタンを選択します。

6. WhatsAppアプリケーションに送信された認証番号6桁を入力します。

7. <Cmdname sid="BTN06501" className="uitext" />ボタンを選択します。
