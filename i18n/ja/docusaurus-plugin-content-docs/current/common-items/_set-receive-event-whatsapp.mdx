### WhatsApp通知受信を追加する

WhatsApp経由で通知を受信できます。

<ImgLang img='set-event-whatsapp-notification.png' desc='WhatsApp'/>

1. 画面の右上のプロファイルアイコンを選択してください。 

2. ポップアップメニューが表示されたら<Cmdname sid="MNU07360" className="uitext" />ボタンを選択してください。 

3. <Cmdname sid="account_user_phone" className="uitext" />セクションで<span class="uitext">WhatsApp</span>ボタンを選択してください。

4. <Cmdname sid="phone" className="uitext" />に認証番号を受信する電話番号を入力してください。

5. <Cmdname sid="BTN06578" className="uitext" />ボタンを選択してください。

6. WhatsAppアプリケーションに送信された認証番号6桁を入力してください。

7. <Cmdname sid="BTN06501" className="uitext" />ボタンを選択してください。

:::note

登録した電話番号を変更するには、<Cmdname sid="BTN06594" className="uitext" />ボタンを選択し、上記の手順を繰り返してください。

:::
