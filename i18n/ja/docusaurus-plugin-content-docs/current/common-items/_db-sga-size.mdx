## SGAサイズ

ホーム画面 > プロジェクト選択 > **_統計／レポート_**>**_SGAサイズ_**

オラクルサーバーのメモリ領域であるSGA要約情報を毎時間保存し、時間別変化量をモニタリングすることができます。 

<ProdImg img='db-stat-sga-size.png' alt='SGA 사이즈' />

1.  **_時間_**で照会したい時間を設定してください。

2.  インスタンスでトレンド情報を表示するインスタンス名を選択します。

3.  ![検索アイコン](/img/ico-btn-search.png)ボタンを選択してください。

:::note

照会時間は、最大3週間まで選択できます。任意の日付と時刻を照会するには、時計の部分を選択してください。日付と時刻のテキスト領域をクリックすると、日付と時刻を選択するオプションが表示されます。

:::
