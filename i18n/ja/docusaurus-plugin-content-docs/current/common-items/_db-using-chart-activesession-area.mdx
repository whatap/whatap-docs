## チャートおよびアクティブセッション領域を活用

画面上の![レイアウトアイコン](/img/ico-layout-set.svg)ボタンをクレックしてください。**_レイアウト設定_**ウィンドウで**_1x3_**に設定し、**_保存_**ボタンを選択してください。**_時間_**を**_直前24時間_**として設定してください。次のように、その日のデータベースのコア指標を詳細に確認できます。

<ProdImg img='db-count-trend-1x3.png' desc='カウント推移' />

- ![数字1](/img/number-01.png)**_時間_**で目的の時間間隔を設定してください。

  - 緑色のアイコンをクリックすると、検索時間を選択できます。
  - 検索時間を選択した状態で ![左方向アイコン](/img/left-arrow.svg)または ![右方向アイコン](/img/right-arrow.svg)ボタンを選択すると、設定した時間間隔で検索時間を変更できます。
  - 日付と時間部分を選択することで、目的の時間を細かく設定できます。 設定を完了した後 **_確認_** ボタンを選択してください。

- ![数字2](/img/number-02.png)**_インスタンス_**：プロジェクトに接続したインスタンスを選択することができます。

- 指標チャートの特定のポイントをクリックすると(![数字3](/img/number-03.png))、ポイントを赤い線で表示し、収集されたアクティブセッションを確認することができます。 

- チャートの指標を変更するには、右上の![設定アイコン](/img/ico-setting.svg)ボタンをクリックしてください。 

- アクティブセッションデータは5秒単位で収集しています。画面の下の![数字4](/img/number-04.png)5秒単位のセルを選択してデータを表示することができます。

- 1分単位で時間を移動するには![数字5](/img/number-05.png)![以前の時間アイコン](/img/ico-left-time.svg)または![以降の時間アイコン](/img/ico-right-time.svg)ボタンを選択してください。

- ![数字5](/img/number-05.png)![コラムアイコン](/img/ico-column.svg)：テーブルのカラムヘッダー項目を表示または非表示にすることができます。

- ![数字5](/img/number-05.png)![フィルターアイコン](/img/ico-filter.svg)：テーブルのカラムヘッダー項目を基準に一覧をフィルタリングすることができます。ボタンをクリックすると、各ヘッダーカラムで**_含む_**、**_含まない_**、**_一致_**、**_一致しない_**条件を設定できます。

- ![数字5](/img/number-05.png)![ダウンロードアイコン](/img/ico-download.svg)：テーブルの内容をCSVファイルとしてダウンロードできます。
