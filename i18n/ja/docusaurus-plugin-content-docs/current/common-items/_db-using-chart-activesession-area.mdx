## チャートおよびアクティブセッション領域を活用

画面上の![レイアウトアイコン](/img/ico-layout-set.svg)ボタンをクレックしてください。<Cmdname sid="layout_setting" className="uitext" />ウィンドウで**_1x3_**に設定し、<Cmdname sid="save" className="uitext" />ボタンを選択します。<Cmdname sid="time" className="uitext" />を<Cmdname sid="TTL07218" className="uitext" />として設定してください。次のように、その日のデータベースのコア指標を詳細に確認できます。

<ProdImg img='db-count-trend-1x3.png' desc='カウント推移' />

- ![数字1](/img/number-01.png) <Cmdname sid="time" className="uitext" />で目的の時間間隔を設定してください。

  - 緑色のアイコンをクリックすると、検索時間を選択できます。
  - 検索時間を選択した状態で ![左方向アイコン](/img/left-arrow.svg)または ![右方向アイコン](/img/right-arrow.svg)ボタンを選択すると、設定した時間間隔で検索時間を変更できます。
  - 日付と時間部分を選択することで、目的の時間を細かく設定できます。設定を完了した後<Cmdname sid="confirm" className="uitext" />ボタンを選択します。

- ![数字2](/img/number-02.png) <Cmdname sid="instance" className="uitext" />：プロジェクトに接続したインスタンスを選択することができます。

- 指標チャートの特定のポイントをクリックすると(![数字3](/img/number-03.png))、ポイントを赤い線で表示し、収集されたアクティブセッションを確認することができます。 

- チャートの指標を変更するには、右上の![設定アイコン](/img/ico-setting.svg)ボタンをクリックします。 

- アクティブセッションデータは5秒単位で収集しています。画面の下の![数字4](/img/number-04.png)5秒単位のセルを選択してデータを表示することができます。

- 1分単位で時間を移動するには![数字5](/img/number-05.png)![以前の時間アイコン](/img/ico-left-time.svg)または![以降の時間アイコン](/img/ico-right-time.svg)ボタンを選択します。

- ![数字5](/img/number-05.png)![コラムアイコン](/img/ico-column.svg)：テーブルのカラムヘッダー項目を表示または非表示にすることができます。

- ![数字5](/img/number-05.png) ![フィルターアイコン](/img/ico-filter.svg)：テーブルのカラムヘッダー項目を基準に一覧をフィルタリングできます。ボタンをクリックすると、各ヘッダーカラムで<Cmdname sid="includes" className="uitext" />、<Cmdname sid="excludes" className="uitext" />、<Cmdname sid="equal" className="uitext" />、<Cmdname sid="unequal" className="uitext" />の条件を設定できます。

- ![数字5](/img/number-05.png)![ダウンロードアイコン](/img/ico-download.svg)：テーブルの内容をCSVファイルとしてダウンロードできます。
