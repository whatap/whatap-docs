## 大量通知発生の防止

通知が大量に発生すると、設定した時間の警告通知を一時的に停止します。**_警告通知_** > **_イベント受信設定_**メニューで大量通知発生の防止セクションに移動してください。

<ImgLang img='set-event-huge-message.png' desc='大量通知発生の防止' />

-   **_活性化_**トグルボタンを選択して機能をオン・オフにすることができます。
-   **_検知時間_**に**_検知回数_**を超えるイベントが発生すると**_停止時間_**に警告通知を停止します。

例えば、5分間20回のイベントが発生した場合、5分間警告通知を停止します。設定した**_停止時間_**時間が経つと大量通知発生の防止機能は解除されます。
