## メソッド性能状態 {#side_methodStat}

ホーム画面 &gt; プロジェクト選択 &gt; <Cmdname sid="side_agentsConfiguration" className="uitext" /> &gt; <Cmdname sid="side_methodStat" className="uitext" />

Javaアプリケーションで実行されたメソッド(Method)の性能統計情報を提供します。 

- <Cmdname sid="refresh" className="uitext" />: データを再呼び出しします。

- <Cmdname sid="reset" className="uitext" />: 登録されたメソッド統計データを初期化します。

:::note 

<Cmdname sid="side_methodStat" className="uitext" />メニューを使うには、<Cmdname sid="agent_configuration" className="uitext" />メニューから次のカスタムオプションを追加します。 そして再起動または<Cmdname sid="side_loadedClasses" className="uitext" />メニューから<Cmdname sid="redefine" className="uitext" />します。

> `hook_method_stat_patterns=io.home.test.baseapp.app.post.mapper.PostMapperImpl.*`

:::
