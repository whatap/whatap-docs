## カウント推移

ホーム画面 > プロジェクト選択 > ***分析*** > ***カウント推移***

<ImgLang img="analysis-count.png" desc="カウント推移" />

***カウントの推移***メニューでは、次の情報を確認できます。

- １日のデータベース稼働の推移

- 特定の時間帯のコア指標

- ***アクティブセッション***

  画面の下の***アクティブセッション***タブを選択します。現在進行中のセッションの情報(Query含む)を確認します。10秒以上経過したセッションの場合は<span class='slow'>黄色</span>、20秒以上のセッションの場合は<span class='vslow'>赤色</span>として表示します。Queryのプラン情報をまとめて確認することができます。

- ***ロックツリー***

  画面の下の***ロックツリー***タブを選択します。ロックに関する情報を確認できます。holderとwaiterの構造をツリー形式で表現して提供します。10秒以上進行しているセッションの場合は<span class='slow'>黄色</span>、20秒以上のセッションの場合は<span class='vslow'>赤色</span>として表現します。

- ***プロセス情報***

  画面の下の***プロセス情報***タブを選択します。XOSエージェントをインストールしている場合、データベースサーバーのプロセスに関する情報を確認できます。
