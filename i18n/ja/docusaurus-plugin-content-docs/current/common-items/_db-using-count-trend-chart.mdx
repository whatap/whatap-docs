## カウント推移チャートの使用

カウント推移関連メニューから次の機能を活用できます。

<ImgLang img="analysis-count-24.png" desc="カウント推移24時間" />

画面右上の![レイアウトアイコン](/img/ico-layout-set.svg)ボタンをクリックします。***レイアウト設定***ウィンドウで、***1x3***に設定し、***保存***ボタンを選択します。***時間***は***最近24時間***として設定してください。次のように、1日間のデータベースのコア指標を詳細に確認できます。

<ImgLang img="analysis-count-usage.png" desc="カウント推移の時間別確認" />

- 指標チャートの特定のポイントをクリックすると、(![指示線1](/img/number-01.png))そのポイントを赤い線で表示し、その時点で収集されたアクティブセッションを確認することができます。 

- アクティブセッションのデータは5秒単位で収集しています。画面下の![指示線2](/img/number-02.png)秒単位セルを選択し、5秒単位でデータを確認することができます。

- 1分単位で時間を移動するには![指示線3](/img/number-03.png)![以前の時間アイコン](/img/ico-left-time.svg)または![以降の時間アイコン](/img/ico-right-time.svg)ボタンを選択します。

- 詳細情報をデータセルに表示する項目を追加または削除するには![指示線3](/img/number-03.png)![コラムアイコン](/img/ico-column-set.svg)ボタンを選択します。***カラム編集***ウィンドウが表示されたら、必要な項目を追加または削除することができます。

- ![指示線3](/img/number-03.png)![フィルターアイコン](/img/ico-filter.svg)ボタンをクリックすると、データセルヘッダーにフィルターを適用して、必要なデータのみソートできます。

- 画面下に出力されたデータをcsvファイルとしてダウンロードできます。![指示線3](/img/number-03.png)![ダウンロードアイコン](/img/ico-download.svg)ボタンを選択します。
