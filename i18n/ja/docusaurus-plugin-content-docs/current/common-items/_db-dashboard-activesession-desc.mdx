<InDoc product="postgresql,mssql">

- <Cmdname sid="active_sessions" className="uitext" />：リアルタイムで実行中のセッションとクエリ内容を照会できます。

  - 実行中のセッション項目を選択すると<Cmdname sid="session_detail" className="uitext" />画面が表示され、選択したセッションの詳細を確認できます。[次の文書](#session-details)を参考にしてください。

  - テーブルの右側にある***query***カラムの項目を選択すると、<Cmdname sid="TTL06224" className="uitext" />画面が表示されます。選択したクエリ(Query)のFull TextおよびPlan情報を確認できます。詳細については、[次の文書](#query-details)を参照してください。

</InDoc>
