<Title hashid='basic-usage' level={props.level}>基本画面ガイド</Title>

<ImgLang img="analysis-tx-main.png" desc="トレースの分析" />

- ![数字1](/img/number-01.png)**トランザクション照会件数**:ヒットマップウィジェットからドラッグした領域のトランザクション一覧の数を確認できます。 最大1000件まで照会できます。

- ![数字2](/img/number-02.png)**トランザクションの分類**：![アクティブスタックアイコン](/img/ic-s-active-stack.svg) <Cmdname sid="active_stack" className="uitext" />と![マルチトランザクションアイコン](/img/ic-s-mult-transation.svg) <Cmdname sid="multi_transaction" className="uitext" />、<Cmdname sid="error" className="uitext" />項目を区分して一覧を確認できます。

- ![数字3](/img/number-03.png)**トランザクション検索**：トランザクション名、エージェント名、エラー メッセージに基づいてトランザクション一覧を検索できます。

- ![数字4](/img/number-04.png)**一覧ソート**：テーブルのヘッダーカラム項目をクリックすると、選択した項目に基づいてトランザクション一覧をソートできます。

- エラーが発生したトランザクション項目は、一覧に<span class="vslow">赤色</span>で表示されます。

:::tip

- テーブルでよく参照するカラムを前方に配置したり、カラムの追加および非表示などができます。 詳細は[次の文書](#set-table-column)を参照してください。

- <Cmdname sid="AA005" className="uitext" />ウィジェットから別の領域をドラッグすると、新たなトレースの分析画面を表示できます。 既存のトレース分析画面と比較して確認できます。

- 表示されているトランザクション項目を他のユーザーに共有できます。 トランザクション項目を選択した状態(![a icon](/img/char-a.svg))で、トレース分析画面![b icon](/img/char-b.svg) ![コピーアイコン](/img/ico-copy.svg) <Cmdname sid="DSC07542" className="uitext" />ボタンを選択します。 コピーしたURLアドレスを他のユーザーに共有します。 URLアドレスを受け取ったユーザーは、ブラウザアドレスバーにURLアドレスを入力すると、同じ項目をすばやく確認できます。

  <ImgLang img='share-trace-item.png' desc='TX共有' />

:::
