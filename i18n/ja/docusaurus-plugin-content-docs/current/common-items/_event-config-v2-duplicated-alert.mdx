## エージェント重複通知の設定{#duplicatedalert}

<Cmdname sid="TTL05989" className="uitext" />オプションは、モニタリング中のエージェントのうち、同じIDを設定したエージェントが2つ以上の場合、指標を重複して収集することによって発生する警告通知です。 エージェント名を重複して設定していないか、エージェント設定で関連オプションを確認します。 ブルー/グリーンなどのデプロイメント状況でエージェントが重複して警告通知が発生した場合は、この機能を利用することをお勧めします。

<ImgLang img="event-config-v2-apm-duplicated-alert.png" desc="イベント重複イベント" />

同じ名前のエージェントが同時に2つ以上実行されたとき、イベントリスト上部の<Cmdname sid="TTL05989" className="uitext" />オプションで選択した時間は通知を受信しません。
