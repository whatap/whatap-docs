-   ![現在のページのテーマアイコン](/img/ico-theme.svg) **テーマ**：現在のページのテーマを変更できます。 デフォルトは**_Light_**モードです。
-   ![リアルタイム通知icon](/img/ico-noti.svg)**リアルタイム通知**：最近発生したイベントが確認できます。トグルメニューをクリックしてブラウザ通知をオンまたはオフにできます。
-   ![顧客サポートアイコン](/img/ic-message-filled.svg)**顧客サポート**：オンラインで顧客サポートを受けることができます。
-   **プロフィールアイコン**：一番右側に位置するアイコンを選択してください。 **_アカウント情報管理_**、**_組織作成_**、**_組織管理_**、ユーザー言語を選択できる機能を提供します。
