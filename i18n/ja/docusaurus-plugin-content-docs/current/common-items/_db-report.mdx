## 日次データベースのレポート

日次データベースレポートは、1日あたりのデータベースのパフォーマンスを数値とチャートで確認できるレポートです。レポート作成時に必要に応じて開始日、開始時間、終了時間を設定することができます。特定のインスタンスのみ選択して表示することができます。すべてを選択すると、すべてのインスタンスを一度に表示します。

### 性能推移チャート

データベースインスタンスごとに指標チャートを確認できます。各指標の最大値を円形で表示し、最大値が発生した時間をまとめて表示します。

:::note

-   チャートで表現される指標は、Config設定によって異なる場合があります。

-   各チャートの右上にある**_CSV_**ボタンを使用して、データをエクセルファイルとしてダウンロードすることができます。

:::

### SQL TOP 10

検索期間中に全体または選択したインスタンスが実行したSQLクエリ統計を確認できます。クエリ実行時間の最大値を降順にソートし、上位10項目のみ表示します。

## 週次データベースレポート

週次データベースレポートは、1週間のデータベースのパフォーマンスを数値とチャートで確認できるレポートです。開始日を直接設定することができ、参照日の7日前から表示できます。

### 性能推移チャート

データベースインスタンスごとに、週間指標を数値およびチャートで比較できます。検索期間の最大値を円形で表示し、その数値が発生した時間も表示されます。

:::note

-   チャートで表現される指標は、Config設定によって異なる場合があります。

-   各チャートの右上にある**_CSV_**ボタンを使用して、データをエクセルファイルとしてダウンロードすることができます。

:::

### SQL TOP 10

検索期間に全体または選択したインスタンスが実行したSQLクエリ統計を確認できます。クエリ実行時間の最大値(elapse max)を降順にソートし、上位10項目のみを表示します。

## 月間データベースレポート

月間のデータベースの性能を数値とチャートで確認できるレポートです。開始日を設定でき、開始日から30日まで表示できます。月間の各指標別の変化量をインスタンス別に確認することができ、検索期間中に実行した上位10項目のSQLクエリ統計も確認できます。
