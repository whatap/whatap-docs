### オプションを修正・削除する

<ProdImg img='modify-agent.png' desc='옵션 수정' />

画面を上または下にスクロールするか、左側のオプション カテゴリを選択して修正・削除するオプションを選択してください。画面の一番上に移動するには、**_オプション追加へ移動_**ボタンを選択してください。

変更したいオプションで希望する値を選択・修正してください。オプションを削除するには![削除アイコン]\(/img/ico-trash.svg）ボタンを選択してください。変更事項を適用するには、**_適用_**ボタンを選択してください。

:::note

オプション値に何も入力していない状態で**_適用_**ボタンを選択すると、そのオプションが削除されます。

:::
