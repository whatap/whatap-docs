### オプションを修正・削除する

<ProdImg img='modify-agent.png' desc='オプションを修正' />

1. 画面を上下にスクロールするか、左側のオプション一覧から修正または削除するオプションを選択します。 

2. 変更するオプションから目的の値を選択または修正してください。オプションを削除するには![削除アイコン](/img/ico-trash.svg)ボタンを選択します。 

3. 変更した事項を適用するには<Cmdname sid="apply" className="uitext" />ボタンを選択します。

:::note

- 画面の上部に移動するには、![数字3](/img/number-03.png)<Cmdname sid="go_to_add_option" className="uitext" />ボタンを選択します。

- オプション値に何も入力せずに<Cmdname sid="apply" className="uitext" />ボタンを選択すると、該当オプションを削除します。

:::
