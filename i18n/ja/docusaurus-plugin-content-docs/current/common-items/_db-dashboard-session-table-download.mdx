### 照会リストをダウンロードする{#downloadcsv}

現在照会中の**_Session Table_**リストをCSVファイルでダウンロードできます。 **_Session Table_**セクションの右上の![ダウンロードアイコン](/img/ico-download.svg)ボタンを選択します。

ダウンロードしたファイル名の形式は次のとおりです。

</InDoc>
