2023年7月25日にリリースされたトレース分析機能の変更をご案内します。 トレース分析機能の詳細については、[次の文書](../java/dashboard-hitmap-trace)をご参照ください。

## TXリスト

-   カラムのタイトルを明確な名前に変更し、ユーザーが直感的に認識できるように変更しました。

-   重要度と優先確認が必要なデータに基づいてカラムの順序を変更しました。

-   頻繁に確認しなかったり、不要なカラム内容を削除しました。

| :---------------------------------------------------------: |
| ![before](/img/changes/changes-transaction-list-before.png) |
|                           **変更後**                           |
|  ![after](/img/changes/changes-transaction-list-after.png)  |

### カラム変更点

| 変更前                  | 変更後                 |
| -------------------- | ------------------- |
| Client OS Pro        | OS                  |
| CPUタイム               | CPU使用時間             |
| IPアドレス               | クライアントIP            |
| referer              | リファラー               |
| URL                  | トランザクション名           |
| メモリ割当て               | メモリ割り当て使用量          |
| エージェントグループ           | エージェントグループ名         |
| エージェントノード            | エージェントサーバー名         |
| エージェントノードID          | エージェントサーバーID        |
| エージェント名              | エージェント名(oname)      |
| エージェントID             | エージェントID (oid)      |
| クライアントID             | WClientID           |
| フェッチ時間               | SQLフェッチ時間           |
| 呼出しの深さ               | ❌ (削除）              |
| マルチコーラー              | ❌ (削除）              |
| ユーザーエージェントのハッシュ      | ❌ (削除）              |
| ユーザータイプ              | ❌ (削除）              |
| エラークラス               | ❌ (削除）              |

## レコード要約

-   **_レコード要約_**情報をタブで分離しました。

-   関連情報をグループ化して簡単に確認できます。

-   頻繁に確認しないか、不要な情報を削除しました。

## テーブルビュー

-   カラムのタイトルを明確な名前に変更し、ユーザーが直感的に認識できるように変更しました。

-   開始、終了、経過、エージェント名(oname)などの重要情報を上部に配置しました。

-   **累積時間**を削除し**_時間_**を追加しました。

|                                変更前                                |
| :---------------------------------------------------------------: |
| ![before](/img/changes/changes-transaction-table-view-before.png) |
|                              **変更後**                              |
|  ![after](/img/changes/changes-transaction-table-view-after.png)  |

| 変更前         | 変更後      |
| ----------- | -------- |
| ステップ        | No       |
| 累積時間        | 時間       |
| 時間差         | 時間差      |
| ステップ時間      | 経過       |
| ステップ内容      | 内容       |
