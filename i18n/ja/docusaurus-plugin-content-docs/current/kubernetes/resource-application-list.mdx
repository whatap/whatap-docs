ホーム画面 &gt; プロジェクト選択 &gt; <Cmdname sid="side_workload" className="uitext" /> &gt; <Cmdname sid="side_containerizedAppList" className="uitext" />

コンテナ化されたアプリケーションの一覧を照会します。 希望するカラムを選択すると、そのカラムを基準にアプリケーション一覧をソートして照会することができます。 

<ImgLang img='k8s-app-list.png' desc='アプリケーションリスト' />

- 10秒サイクルでリアルタイムのデータを更新します。

- 検索、カラム整列、詳細を見る、_CSV_ファイルのダウンロードを提供します。

  > CSV ファイル名、_kubernetes\_application\_list\_`{pcode}`\_`{timestamp}`.csv_
