<Cmdname sid="side_oomContainer" className="uitext" />メニューを通じて**Out Of Memory**(**OOM**)が発生したコンテナをすばやく検索し、その時点でのメモリ使用量の変化の推移を確認できます。変化の推移は急激または漸進的である可能性があります。これを確認し、適切なPOD数とメモリの臨界点に対する客観的な判断が可能です。

:::note 

**OOM発生例**

コンテナのメモリ使用量がメモリ制限を超えたり、コンテナが実行中のノードのメモリが不足している場合、Out Of Memoryが発生します。もしコンテナがコード`137`で終了したらOut Of Memoryによって`SIGKILL`を受信したものです。

:::

### 基本画面ガイド

<ImgLang img='whatap_kubernetes_analysis_oom_container_1.png' desc='Out of Memory Killedコンテナ' />

- 左上のタイムセレクターをクリックして照会したい時間帯を選択できます。最大2日まで照会できます。

- 右上の検索入力画面でコンテナ名を入力すると、照会したい特定のコンテナ情報をすばやく確認できます。

- リストの右の![詳細アイコン](/img/ic-static.svg)詳細アイコンをクリックすると、選択したコンテナのメモリ使用の推移を確認できます。 

- OOMが発生したコンテナ情報をリスト形式で提供します。コンテナの詳細については、次のガイドを参照してください。

### OOM Killedコンテナ情報

<Cmdname sid="side_oomContainer" className="uitext" />メニューで提供するカラム情報を次のように案内します。

| カラム                   | 意味                                 |
| --------------------- | ---------------------------------- |
| ***OomKilledAt***     | Out Of Memory Killedが発生したTimeStamp |
| ***ContainerName***   | コンテナ名                              |
| ***ContainerId***     | コンテナのID                            |
| ***Pod***             | コンテナが属するPod名                       |
| ***ApplicationName*** | コンテナで実行中のアプリケーション名                 |
| ***Created***         | コンテナが生成されたTimeStamp                |
| ***Command***         | コンテナの実行コマンド                        |
| ***Image***           | コンテナのイメージ名                         |
| ***Namespace***       | コンテナのPodが属するネームスペースの名前             |
| ***Node***            | コンテナが動作しているノード名                    |
| ***MemLimit***        | コンテナのメモリLimitサイズ                   |
| ***MemWs***           | コンテナで非活性化されているメモリを除いて実際に使用中のメモリ    |

:::note 

- ***ContainerId***をクリックすると[コンテナ一覧](/kubernetes/resource-container-list)メニューに移動します。

- ***Pod***をクリックすると[Pod一覧](/kubernetes/resource-pod-list)メニューに移動します。 

- ***ApplicationName***クリックすると、該当アプリケーションが検索された[コンテナアプリケーションリスト](/kubernetes/resource-application-list)メニューに移動します。 

- ***Node***をクリックすると[ノード一覧](/kubernetes/resource-node-list)メニューに移動します。 

:::
