ホーム画面 > プロジェクト選択 > <Cmdname sid="cluster" className="uitext" /> > ***ネームスペース状況***

:::note 

Kubernetesクラスタープロジェクト(<code class='my'>CP</code>)の場合、該当メニューを照会できます。

:::

***ネームスペース状況***メニューを通じてWhaTap Kubernetesクラスタープロジェクトおよび下位ネームスペースプロジェクトの状況を確認できます。ネームスペースプロジェクトではアクセスできないメニューです。 

<ImgLang img='k8s-namespace-status.png' desc='ネームスペース状況' />

***ネームスペース状況***メニューは、プロジェクトリスト画面と似ていますが、次のように違いがあります。

- **プロジェクトリスト**

  ネームスペースプロジェクトを作成した後、プロジェクトを別のグループに移動できますが、ネームスペースを一目で確認するのことは困難です。

- **ネームスペース状況**

  - クラスタープロジェクトと同じグループに属していないネームスペースも確認できます。

  - ひと目で関連プロジェクトをすべて確認できます。

クラスタープロジェクトの場合、Kubernetesプラットフォームを表す<span class='normal'>青色</span><code class='my'>Kubernetes</code>のタグが表示されます。ネームスペースプロジェクトには、<code class='my'>Namspace</code>タグが表示されます。ウィジェット左上のタイトルはプロジェクト名で、ネームスペース名は一番下の<code class='my'>Namspace</code>タグの横に表示されます。
