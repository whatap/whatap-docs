# GPUトレンド

**GPUトレンド**は、GPUデバイスごとの使用量を長期統計データとして可視化し、特定の時点でどのデバイスが非効率的に使用されていたかを迅速に特定できます。

## 基本画面

GPUの長期使用量統計およびパターンを確認できます。  
照会単位は **日次、週次、月次** をサポートします。  
デバイスごとのGPU使用率はブロック形式で表示され、色の濃さによって使用レベルを区分します。

* **50%未満**: 薄い青  
* **80%未満**: 青  
* **80%以上**: 濃い青

## 詳細情報の確認

各ブロックにマウスを載せると、その時点およびデバイスの詳細情報をツールチップで確認できます。

* **Label**: デバイス名およびGPUモデル  
* **UUID**: GPU固有識別子  
* **Time**: 基準時刻  
* **GPU Utilization**: 当該時点のGPU使用率(%)
