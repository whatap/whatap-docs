Kubernetesモニタリングを開始する前に、次の動作環境を確認してください。

## 対応プラットフォーム

-   EKS(Amazon Elastic Kubernetes Service) 1.16 ~ 1.23
-   AKS(Azure Kubernetes Service) 1.16 ~ 1.23
-   GKE(Google Kubernetes Engine) 1.16 ~ 1.23
-   OKE(Oracle Container Engine for Kubernetes) 1.16 ~ 1.23
-   Openshift 3.xx / 4.xx
-   Cocktail 1.16 ~ 1.23
-   Accordion 1.16 ~ 1.23
-   PaaS-TA 1.16 ~ 1.23
-   plugins項目：istio、GPU対応
