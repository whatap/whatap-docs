**_コンテナマップ_**メニューで選択した対象についての詳細情報を、<Cmdname sid="BTN07712" className="uitext" />画面で確認できます。

1. **_コンテナマップ_**メニュー画面で表示する特定のブロック(コンテナまたはPod)を選択します。 

2. 画面の右側に、次のように**概要ビュー**パネルが表示されます。 

   <ImgLang img='k8s-container-map-sum-select.png' desc='コンテナマップ要約' />

3. **概要を表示**パネル右上の![詳細分析アイコン](/img/ico-detail-mode.svg) <Cmdname sid="BTN07712" className="uitext" />アイコンを選択すると、より詳細な情報を確認できます。

## 基本画面ガイド

コンテナマップ<Cmdname sid="BTN07712" className="uitext" />画面は、次のように構成されています。基本情報は、カードと一覧形式で確認できます。核心モニタリングデータを照会できるように**_トレース_**、**_メトリクス_**、**_ログ_**、**_イベント_**タブを提供します。 

<ImgLang img='k8s-container-detail.png' desc='詳細分析画面' />

- ![number 1](/img/number-01.png) **概要情報カード** (コンテナまたはPod)

  選択した対象のグループ情報を確認できます。対象を追加で選択すると、![number 3](/img/number-03.png)領域から対象の指標が確認できます。コンテナマップで選択したカードと同じビューが表示されます。詳細については、[次の文書](container-map)を参照してください。

  :::note

  集計情報カード参照時にブロック数が多い場合、次のように表記方法を変更します。すべてのブロック、<span class='normal'>正常</span>、<span class='slow'>アラート</span>、<span class='vslow'>警告</span>状態ブロック数を確認して比較することができます。

  <ImgLang img='k8s-container-card-block.png' desc='ブロック数が多い場合は、集計情報カード' />

  :::

- ![指示線2](/img/number-02.png) **対象情報一覧**

  選択した対象のグループ情報を一覧で確認できます。対象を追加で選択すると、![number 3](/img/number-03.png)領域から対象の指標が確認できます。 

  - ![指示線2](/img/number-02.png)一覧の上段の**_選択項目を見る_**チェックボックスを選択すると、次のように選択されたコンテナまたはPodのみ表示されます。 

    <ImgLang img='k8s-container-detail-list-selected.png' desc='選択のみ表示' />

  - 一覧で選択した対象左側の![詳細を見るアイコン](/img/right-arrow.svg) **_詳細を見る_**アイコンを選択すると、![指示線a](/img/char-a.svg)以下のように**_Node_**、**_Namespace_**、**_Deployment_**、**_ReplicaSet_**、**_Pod_**、**_ContainerName_**、**_ContainerId_**、**_Image_**、**_Agent_**などのメトリクス情報を照会できます。詳細については、[次の文書](container-map-target-info)を参照してください。 

- ![指示線3](/img/number-03.png) **リソース関係**、**トレース**、**メトリクス**、**ログ**、**イベント**、**呼び出し情報**

  - **_リソース関係_**

    リソース関係タブを使用して、Kubernetesリソース間の接続関係を視覚的に確認できます。 

    :::note 

    選択したブロックに関する**_Namespace_**、**_Deployment_**、**_ReplicaSet_**(**_DaemonSet_**)、**_Pod_**、**_Container_**、**_Application_**などの関係を表現したチャートを次のように提供します。 

    ![リソース関係](/img/k8s-container-detail-rr-s.png)

    :::

  - **_トレース_** 

    **_トレース_**タブを使用して、クバネティスコンテナ上で実行中のアプリケーションのトランザクション詳細をモニタリングできます。**_ダッシュボード_**ビューと**_トランザクション一覧_**ビューを提供します。詳細については[次の文書](container-map-trace)を参照してください。

  - **_メトリクス_**

    **_メトリクス_**タブを使用して、選択されたコンテナまたはPodの主要なメトリクス情報とリソース使用パターンを確認できます。これにより、リソース関連の問題をすばやく特定し、最適化ポイントを見つけることができます。詳しい内容は[次の文書](container-map-metrics)を参照してください。

  - **_ログ_** 

    **_ログ_**タブを使用してクバネティスコンテナ及びコンテナアプリケーションで発生するログを表示することができます。ログ機能は、コンテナのリアルタイムログ検索機能と特定の時点での履歴ログ検索機能に分けられます。詳細については[次の文書](container-map-log)を参照してください。

  - **_イベント_** 

    **_イベント_**タブでは、クバネティスクラスターで発生する**_Kubernetes_**イベントとユーザーが設定した条件による**_WhaTap_**イベントを表示することができます。詳細については[次の文書](container-map-event)を参照してください。

  - **_呼び出し情報_**

    **_呼び出し情報_**タブを使用してクバネティスコンテナ上で実行中のアプリケーションの呼び出しフローと外部呼び出し対象の状態を確認することができます。詳細については[次の文書](container-map-call-information)を参照してください。
