## Node一覧

ホーム画面 > プロジェクト選択 > **_リソース_** > **_Node一覧_**

現在のプロジェクトのNodeのスタータス情報をテーブル形で確認できます。

<ImgLang img='whatap_kubernetes_resource_node_list.png' desc='ノードリスト' />

提供する主な情報は次のとおりです。

| 分類     | 重要情報                                                  |
| ------ | ----------------------------------------------------- |
| CPU    | コア数<br/>使用率<br/>Sys/User使用率                           |
| メモリ    | 容量<br/>使用率<br/>Swap使用率                                |
| ディスク   | 使用率<br/>I/O使用率<br/>inode使用量                           |
| その他    | Private IP<br/>Public IP<br/>稼働時間<br/>エージェントバージョンなど   |

次の機能を追加で提供します。

-   ![比較するアイコン](/img/ico-compare.svg) **_比較する_**：複数のNodeを選択し、チャートベースで使用量を比較できます。

-   ![CSVダウンロードアイコン](/img/ico-csv.svg) **_CSVダウンロード_**：画面に表示された一覧データをCSVファイルとしてダウンロードできます。

サーバー一覧ページのデータは、60秒ごとに自動更新されます。ユーザーは、![一時停止アイコン](/img/ico-pause.svg)ボタンをクリックして、データの自動更新を中止します。![再開アイコン](/img/ico-play-time.svg)ボタンをクリックして自動更新を再開できます。

## Node詳細情報

ホーム画面 > プロジェクト選択 > **_リソース_** > **_Node詳細_**

特定のNodeの詳細を確認できます。

<ImgLang img='whatap_kubernetes_resource_node_detail.png' desc='ノード詳細情報' />

Node詳細画面は、上部の4つのチャートとNode内のコンテナ詳細情報で構成されています。

-   トップチャート

    -   **_CPU Usage_**

    -   **_Memory Usage_**

    -   **_Network_**

    -   **_Disk Usage_**

-   コンテナ情報

    | 項目               | 説明                                                |
    | ---------------- | ------------------------------------------------- |
    | イメージ             | コンテナイメージ情報                                        |
    | Pod              | Pod名                                              |
    | コンテナ             | Container ID                                      |
    | コマンド             | コンテナ実行コマンド                                        |
    | CpuByLimit       | CPU Limit基準CPUの全体使用率                              |
    | MemoryWsByLimit  | コンテナメモリのLimit基準working set使用量                     |
    | CpuThrottledCnt  | コンテナCPU Throttled数                                |
    | CpuThrottledTime | コンテナCPU Throttled時間                               |
    | MemLimit         | コンテナのメモリLimitサイズ                                  |
    | MemRequest       | コンテナのメモリRequestサイズ                                |
    | MemWs            | コンテナのメモリworking set                               |
    | IoReadBytes      | コンテナ全体ブロックデバイスの1秒当たり読み込んだバイトの合計                   |
    | IoReadIops       | コンテナ全体ブロックデバイスの1秒当たり読み込んだ数の合計                     |
    | IoWriteBytes     | コンテナ全体ブロックデバイスの1秒当たり書き込んだバイトの合計                   |
    | IoWriteIops      | コンテナ全体ブロックデバイスの1秒当たり書き込んだ数の合計                     |
    | NetRxBytes       | コンテナ全体ブロックデバイスの1秒当たり読み取りバイトの合計                    |
    | NetRxDropped     | コンテナネットワーク受信dropped数                              |
    | NetRxError       | コンテナのネットワーク受信エラー数                                 |
    | NetRxIops        | コンテナのネットワーク受信数                                    |
    | NetTxByes        | コンテナのネットワーク送信データサイズ                               |
    | NetTxDropped     | コンテナのネットワーク送信dropped数                             |
    | NetTxError       | コンテナのネットワーク送信エラー数                                 |
    | NetTxIops        | コンテナのネットワーク送信数                                    |
