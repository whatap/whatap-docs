ホーム画面 > プロジェクト選択 > **_ダッシュボード_** > **_アプリケーションサービスダッシュボード_**

コンテナ内部で実行中のアプリケーションの状態をリアルタイムで確認できるダッシュボードです。各ウィジェットの正常のデータ出力では、Kubernetes環境にノードエージェントをインストールし、コンテナ内のアプリケーションに、WhaTapエージェントをインストールする必要があります。 

:::note

Java、Node.js、Pythonで開発されたアプリケーションは、WhaTapエージェントのインストールをサポートします。

:::
