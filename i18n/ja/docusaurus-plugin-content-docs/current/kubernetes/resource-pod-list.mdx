ホーム画面 &gt; プロジェクト選択 &gt; <Cmdname sid="side_workload" className="uitext" /> &gt; <Cmdname sid="podList" className="uitext" />

Pod一覧を照会します。 希望するカラムを選択すると、そのカラムを基準にPod一覧をソートして照会することができます。 

<ImgLang img='k8s-pod-list.png' desc='Pod一覧' />

- 10秒サイクルでリアルタイムのデータを更新します。

- 左上で全体のPod数と状態別のPod数を確認することができます。 

- 検索、カラム整列、詳細を見る、_CSV_ファイルのダウンロードを提供します。

  > CSV ファイル名、_kubernetes\_pod\_list\_`{pcode}`\_`{timestamp}`.csv_
