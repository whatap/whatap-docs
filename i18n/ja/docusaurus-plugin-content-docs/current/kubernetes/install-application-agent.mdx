コンテナ内のJava、Python、Node.jsなどのアプリケーションをモニタリングするためのエージェントをインストールする過程です。

マイクロアプリケーションエージェントのインストールは、WhaTapのアプリケーションモニタリングエージェントのインストールの手順と同じです。ただし、コンテナ化(Containerization) 作業の追加が異なります。 

アプリケーションエージェントのインストールを進めるには、**_管理_** > **_エージェントのインストール_**メニューの画面左の一覧**_追加インストール_**下に**_アプリケーションモニタリングの追加インストール_**タブを選択後、下段の**_アプリケーションモニタリングの追加インストール_**ボタンをクリックします。

<ImgLang img='k8s-install-master-node-application.png' desc='アプリケーションエージェントのインストール' />

## アクセスキーの確認

<ImgLang img='k8s-application-accesskey.png' desc='アクセスキーの確認' />

インストール手順ページの**_アクセスキーの確認_**セクションで、**_プロジェクトアクセスキーを発行する_**ボタンをクリックします。発行済みの場合、アクセスキーが表示されます。

:::caution

**アクセスキー**は、他のプロジェクトと国合わせて使用することはできません。**クラスタープロジェクト**認証キーと異なります。

:::
