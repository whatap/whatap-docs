ホーム画面 > プロジェクトを選択 > ***ダッシュボード*** > ***性能サマリー***

最近の時間別チャートを通じて、プロジェクト内のWhaTapノードエージェントおよびアプリケーションエージェントがインストールされている対象の情報を表示します。ネームスペースプロジェクトに分離された情報は、ネームスペースプロジェクトで確認できます。 

<ImgLang img='k8s-performance-summary.png' desc='性能概要' />

- **ショートカットオプション**

  ウィジェッ別の右上に位置する![詳細アイコン](/img/ico-more.svg)***詳細***アイコンを選択して![ショートカットアイコン](/img/ico-shortcut.svg)***ショートカット***オプションを確認できます。

- **ラインチャート設定オプション**

  ラインチャートウィジェット別の右上に位置する![詳細アイコン](/img/ico-more.svg)***詳細***アイコンを選択した後、またはすぐに![設定アイコン](/img/ic-setting.svg)設定オプションを確認できます。ラインチャートの場合、設定オプションを通じてidとなる値を変更できます。この設定は保存されません。  

  > 例、***Node CPU (%)***ウィジェットの場合***Node***ごとのデータ照会が可能です。メトリクスを変更して***ContainerName***ごとに照会できます。

:::note 

エージェントのインストールが正常に完了していない場合、データが表示されない可能性があります。 

:::

### ノード情報

ウィジェットを提供するNode情報は次のとおりです。

- 直前30分チャート

  - ***Node CPU (%)***、***Node Memory (%)***など

:::note 

- ノード数量ウィジェットの右上に![その他のアイコン](/img/ico-more.svg)***その他の***アイコン>![ショートカットアイコン](/img/ico-shortcut.svg)***ショートカット***アイコンを順番に選択すると***クラスター***>***ノード一覧***メニューに移動します。 

- ***Node CPU (%)***ウィジェットと***Node Memory (%)***ウィジェット右上の![詳細をみるアイコン](/img/ico-more.svg) ***詳細を見る***アイコン > ![ショートカットアイコン](/img/ico-shortcut.svg) ***ショートカット***アイコンを順番に選択すると***分析*** > ***メトリクスチャート***メニューに移動します。該当メニューを使用すると、より詳細な条件でチャートを確認できます。

:::

### コンテナ情報

ウィジェットによって提供するコンテナ情報は次のとおりです。

- 直前10分チャート

  - ***Container Status***、***Container CPU (%)***、***Container Memory (%)***など

### アプリケーション情報

ウィジェットによって提供するアプリケーション情報は次のとおりです。

- 直前10分チャート

  - ***Active TX***、***TPS***、***Response Time***など

:::note

アプリケーション情報は、アプリケーションエージェントのインストール対象が存在する場合に表示します。

:::
