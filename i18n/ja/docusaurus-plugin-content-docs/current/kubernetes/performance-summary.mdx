ホーム画面 > プロジェクトを選択 > **_ダッシュボード_** > **_性能サマリー_**

最近の時間別チャートを通じて、プロジェクト内のWhaTapノードエージェントおよびアプリケーションエージェントがインストールされている対象の情報を表示します。ネームスペースプロジェクトに分離された情報は、ネームスペースプロジェクトで確認できます。 

<ImgLang img='k8s-performance-summary.png' desc='性能概要' />

- **ショートカットオプション**

  ウィジェッ別の右上に位置する![詳細アイコン](/img/ico-more.svg)**_詳細_**アイコンを選択して![ショートカットアイコン](/img/ico-shortcut.svg)**_ショートカット_**オプションを確認できます。

- **ラインチャート設定オプション**

  ラインチャートウィジェット別の右上に位置する![詳細アイコン](/img/ico-more.svg)**_詳細_**アイコンを選択した後、またはすぐに![設定アイコン](/img/ic-setting.svg)設定オプションを確認できます。ラインチャートの場合、設定オプションを通じてidとなる値を変更できます。この設定は保存されません。  

  > 例、**_Node CPU (%)_**ウィジェットの場合**_Node_**ごとのデータ照会が可能です。メトリクスを変更して**_ContainerName_**ごとに照会できます。

:::note 

エージェントのインストールが正常に完了していない場合、データが表示されない可能性があります。 

:::

### ノード情報

ウィジェットを提供するNode情報は次のとおりです。

- 直前30分チャート

  - **_Node CPU (%)_**、**_Node Memory (%)_**など

:::note 

- ノード数量ウィジェットの右上に![その他のアイコン](/img/ico-more.svg)**_その他の_**アイコン>![ショートカットアイコン](/img/ico-shortcut.svg)**_ショートカット_**アイコンを順番に選択すると**_クラスター_**>**_ノード一覧_**メニューに移動します。 

- **_Node CPU (%)_**ウィジェットと**_Node Memory (%)_**ウィジェット右上の![詳細をみるアイコン](/img/ico-more.svg) **_詳細を見る_**アイコン > ![ショートカットアイコン](/img/ico-shortcut.svg) **_ショートカット_**アイコンを順番に選択すると**_分析_** > **_メトリクスチャート_**メニューに移動します。該当メニューを使用すると、より詳細な条件でチャートを確認できます。

:::

### コンテナ情報

ウィジェットによって提供するコンテナ情報は次のとおりです。

- 直前10分チャート

  - **_Container Status_**、**_Container CPU (%)_**、**_Container Memory (%)_**など

### アプリケーション情報

ウィジェットによって提供するアプリケーション情報は次のとおりです。

- 直前10分チャート

  - **_Active TX_**、**_TPS_**、**_Response Time_**など

:::note

アプリケーション情報は、アプリケーションエージェントのインストール対象が存在する場合に表示します。

:::
