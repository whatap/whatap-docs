ホーム画面 > プロジェクト選択 > ***コンテナー*** > ***コンテナリスト***

:::note

特定のカラムのデータが表示されない場合は、Kubernetes Nodeエージェントが古いバージョンである可能性があります。この場合、十分なデータが収集されるようにKubernetesエージェントバージョンを更新してください。 

:::

***コンテナリスト***メニューを通じて現在プロジェクトのコンテナの詳細情報をリスト形式で確認することができます。 

- 10秒サイクルでリアルタイムのデータを更新します。 

- コンテナのステータス要約、比較、フィルター、および_CSV_ファイルのダウンロード機能を提供します。

  > CSV ファイル名、_kubernetes\_contianer\_list\_`{pcode}`_`{timestamp}`.csv_

## 基本画面ガイド

<ImgLang img='k8s-resource-container-list-m.png' desc='コンテナ一覧メイン画面' />

![number 01](/img/number-01.png)領域から必要なオプションを選択し![number 02](/img/number-02.png)領域でデータを参照できます。 

- ![number 01](/img/number-01.png)領域の左上で、コンテナ全体の数とステータス別コンテナの数を確認できます。

- ![number 02](/img/number-02.png)領域で目的のカラムを選択すると、そのカラムを基準にコンテナ一覧を並べ替えて表示することができます。

- ![number 02](/img/number-02.png) 領域***Node*** カラムで ![詳細を見るアイコン](/img/ic-pageview.svg) ***ノード情報の表示*** アイコンを選択すると、 ***ノードリスト*** メニューに移動します。

## 比較する
