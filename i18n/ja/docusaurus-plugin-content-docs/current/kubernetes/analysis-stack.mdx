ホーム画面 > プロジェクト選択 > **_分析_** > **_スタック_**

<ImgLang img='whatap_kubernetes_analysis_stack_1.png' desc='スタック' />

WhaTapは、10秒(デフォルト値) 間隔で収集したThread Stackを利用してメソッドレベルの性能遅延間隔を分析します。この機能を使用するには、各コンテナの**_トップスタック_**、**_ユニークスタック_**、**_アクティブスタック_**を確認することができます。 

:::note

**_トップスタック_**、**_ユニークスタック_**、**_アクティブスタック_**の詳細については、次の文書を参照してください。 

- [Java](../java/analysis-apm)

- [Python](../python/analysis-apm)

:::
