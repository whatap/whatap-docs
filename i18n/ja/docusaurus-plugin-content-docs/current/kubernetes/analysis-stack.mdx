WhaTapは、10秒(既定値) 間隔で収集したThread Stackを利用してメソッドレベルの性能遅延間隔を分析します。この機能を使用するには、各コンテナの***トップスタック***、***ユニークスタック***、***アクティブスタック***を確認することができます。 

:::note

***トップスタック***、***ユニークスタック***、***アクティブスタック***の詳細については、次の文書を参照してください。 

- [Java](../java/analysis-apm)

- [Python](../python/analysis-apm)

:::
