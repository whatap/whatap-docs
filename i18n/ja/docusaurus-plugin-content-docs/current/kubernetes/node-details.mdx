ホーム画面 &gt; プロジェクト選択 &gt; <Cmdname sid="side_node" className="uitext" /> &gt; <Cmdname sid="node_detail" className="uitext" />

<Cmdname sid="node_detail" className="uitext" />画面は上段の4つのチャートとノード内の***コンテナ***及び***Pod***詳細情報で構成されています。 ノードセレクタを通じて、照会するノードを選択できます。

<ImgLang img='k8s-resource-node-detail.png' desc='ノード詳細画面' />

:::note 

5秒周期で最近の1分のデータを更新します。

:::

### トップチャート

次の4つのチャート情報を確認できます。

- ***CPU Usage***: ノードのCPU使用量を確認できます。

- ***Memory Usage***: ノードのメモリ使用量を確認できます。

- ***Network***: ノードのネットワークインターフェイスの入出力トラフィックとエラーパケットを確認できます。

- ***Disk Usage***: ノードのDisk使用量を確認することができます。 

### コンテナ及びPodの詳細情報

#### コンテナ

***コンテナ***タブの選択時、ノード内のコンテナ詳細情報の上部で、`cpu_quota`、`cpu_request`、`cpu_total_milli`、`mem_limit`、`mem_request`、`mem_working_set`メトリクスの合計を確認できます。 

- ***コンテナ***タブの選択時、ノード内のコンテナ詳細情報の上段で`cpu_quota`、`cpu_request`、`cpu_total_milli`、`mem_limit`、`mem_request`、`mem_working_set`指標の合計を確認できます。 

- <Cmdname sid="overview" className="uitext" />カラムを選択する時<Cmdname sid="TTL08230" className="uitext" />が確認できます。

- コンテナ選択時に***コンテナ一覧***メニューに移動します。 詳細については、[次の文書](resource-container-list)を参照してください。

#### Pod

***Pod***タブの選択時、Podの詳細情報を照会します。 詳細情報画面の上段から `cpu_quota`、`cpu_request`、`cpu_total_milli`、`mem_limit`、`mem_request`、`mem_working_set`メトリクスの合計を確認できます。 

- 希望するカラムを選択すると、そのカラムを基準にPod一覧をソートして照会することができます。 

- <Cmdname sid="overview" className="uitext" />カラムを選択する時<Cmdname sid="TTL08317" className="uitext" type="kube_pod" />が確認できます。

- Pod選択時に***Pod一覧***メニューに移動します。 詳細については、[次の文書](resource-pod-list)を参照してください。
