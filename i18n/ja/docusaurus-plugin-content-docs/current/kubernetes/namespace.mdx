ホーム画面 > プロジェクト選択 > ***管理*** > ***ネームスペース管理***

クラスタープロジェクト(<code class='my'>CP</code>)で***ネームスペース管理***メニューを通じて、ネームスペースプロジェクトを管理できます。ネームスペースプロジェクトでは、当該メニューへアクセスできません。

:::note

クラスタープロジェクトの***ダッシュボード*** > ***コンテナマップ***メニュー画面の上に、下位ネームスペースまたは***ネームスペース状態***および***ネームスペース管理***メニューから移動できるリストボックスを提供します。 

<ImgLang img='k8s-namespace-project-list-check.png' desc='ネームスペース移動' />

:::

## ネームスペースプロジェクト管理

<ImgLang img='k8s-namespace-mng.png' desc='ネームスペース管理' />

ページ上段の![number-1](/img/number-01.png)領域では、全体対象数および主要なリソースの数を確認できます。![number-2](/img/number-02.png) ***ネームスペース一覧***領域からネームスペースプロジェクトの検索、追加および削除ができます。

### ネームスペースプロジェクトの作成

***ネームスペース一覧***からプロジェクトを作成する項目の***+ プロジェクト追加***ボタンを選択します。***プロジェクト追加***ウィンドウが表示されると***確認***ボタンを選択します。

:::note 

**注意事項**

ネームスペースプロジェクトを追加すると、そのネームスペースはクラスタープロジェクト(<code class='my'>CP</code>)から別のWhaTapプロジェクトに分離されます。追加されたネームスペースプロジェクトには、新たな**プロジェクトアクセスキー**が付与されます。 

分離されたネームスペースにアプリケーションモニタリングがインストールされている場合、アプリケーションモニタリング設定内の**プロジェクトアクセスキー**を新たに追加されたネームスペースプロジェクトの**プロジェクトアクセスキー**に変更する必要があります。

:::

### ネームスペースプロジェクトの削除

1. ***ネームスペース一覧***で削除するプロジェクト項目の右側の![設定アイコン](/img/ico-setting.svg) ***設定***アイコンを選択します。

2. ***プロジェクト管理***メニューに移動します。

3. 画面右上の***プロジェクト削除***ボタンを選択します。

:::caution

ネームスペースプロジェクトを削除する場合、ユーザー一覧情報と収集したデータはすべて削除されます。削除したデータは復元できないので注意してください。

:::
