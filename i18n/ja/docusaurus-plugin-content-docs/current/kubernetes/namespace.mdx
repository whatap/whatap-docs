ホーム画面 > プロジェクト選択 > **_管理_** > **_ネームスペース管理_**

クラスタープロジェクト(<code class='my'>CP</code>)で**_ネームスペース管理_**メニューを通じて、ネームスペースプロジェクトを管理できます。ネームスペースプロジェクトでは、当該メニューへアクセスできません。

:::note

クラスタープロジェクトの**_ダッシュボード_** > **_コンテナマップ_**メニュー画面の上に、下位ネームスペースまたは**_ネームスペース状態_**および**_ネームスペース管理_**メニューから移動できるリストボックスを提供します。 

<ImgLang img='k8s-namespace-project-list-check.png' desc='ネームスペース移動' />

:::

## ネームスペースプロジェクト管理

<ImgLang img='k8s-namespace-mng.png' desc='ネームスペース管理' />

ページ上段の![number-1](/img/number-01.png)領域では、全体対象数および主要なリソースの数を確認できます。![number-2](/img/number-02.png) **_ネームスペース一覧_**領域からネームスペースプロジェクトの検索、追加および削除ができます。

### ネームスペースプロジェクトの作成

**_ネームスペース一覧_**からプロジェクトを作成する項目の**_+ プロジェクト追加_**ボタンを選択します。**_プロジェクト追加_**ウィンドウが表示されると**_確認_**ボタンを選択します。

:::note 

**注意事項**

ネームスペースプロジェクトを追加すると、そのネームスペースはクラスタープロジェクト(<code class='my'>CP</code>)から別のWhaTapプロジェクトに分離されます。追加されたネームスペースプロジェクトには、新たな**プロジェクトアクセスキー**が付与されます。 

分離されたネームスペースにアプリケーションモニタリングがインストールされている場合、アプリケーションモニタリング設定内の**プロジェクトアクセスキー**を新たに追加されたネームスペースプロジェクトの**プロジェクトアクセスキー**に変更する必要があります。

:::

### ネームスペースプロジェクトの削除

1. **_ネームスペース一覧_**で削除するプロジェクト項目の右側の![設定アイコン](/img/ico-setting.svg) **_設定_**アイコンを選択します。

2. **_プロジェクト管理_**メニューに移動します。

3. 画面右上の**_プロジェクト削除_**ボタンを選択します。

:::caution

ネームスペースプロジェクトを削除する場合、ユーザー一覧情報と収集したデータはすべて削除されます。削除したデータは復元できないので注意してください。

:::
