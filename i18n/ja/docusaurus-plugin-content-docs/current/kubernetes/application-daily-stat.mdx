ホーム画面 > プロジェクト選択 > ***アプリケーション*** > ***分析*** > ***日別APP現況***

<ImgLang img='whatap_kubernetes_application_daily_app_stat_1.png' desc='日付別のアプリケーション' />

日付別アプリケーションの状況を確認できます。検索条件は、開始日、分類、アプリケーションで、区分条件は、次のとおりです。

| 分類          | 説明                                              |
| ----------- | ----------------------------------------------- |
| エージェント      | コンテナ内部にインストールされているアプリケーションエージェント名               |
| エージェント種類    | エージェントがグループ化されたエージェントの種類(`OKIND`)               |
| エージェントサーバー  | エージェントが実行されているKubernetesノード名                    |

提供するチャートは次のとおりです。

- ***1時間当たりのトランザクション数***

- ***1時間当たりのユーザー数***

- ***1時間当たりのHTTP呼び出し件数***

- ***1時間当たりのSQL件数***

- ***1時間当たりのSQLフェッチ件数***

- ***応答時間***

- ***1秒当たりのトランザクション***

- ***アクティブトランザクション***

- ***5分間の合計エラー件数***

- ***エラー率***
