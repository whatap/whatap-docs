## 始める前に

<Cmdname sid="side_slowQueryLog" className="uitext" />メニューを利用するには*cubrid_broker.conf*ファイルに次のように設定する必要があります。

```ini title="cubrid_broker.conf"
SLOW_LOG=ON
LONG_QUERY_TIME=3 
```

:::note

モニタリング対象のDBサーバーにXCUBエージェントをインストールしないと、slow queryをモニタリングできません。 XCUBエージェントのインストールについての詳細は[次の文書](install-agent#setxcubagent)を参照してください。

:::
