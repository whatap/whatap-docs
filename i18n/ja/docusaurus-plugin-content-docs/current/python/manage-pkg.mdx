## パッケージインストールの確認

次のコマンドを使用して、WhaTap Pythonエージェントパッケージのインストール状態を確認できます。

```javascript title=SH
$ pip list | grep whatap-python
```

## パッケージアップデート{#update-pkg}

次のコマンドを使用して、WhaTap Pythonエージェントのパッケージをアップデートできます。

```javascript title=SH
$ pip install -U whatap-python
```

特定のバージョンを指定する場合は、次の点に注意してください。

```javascript title=SH
$ pip install -U whatap-python==[特定のバージョン]
```

## パッケージを削除

次のコマンドを使用して、WhaTap Pythonエージェントのパッケージを削除できます。

```javascript title=SH
$ pip uninstall whatap-python
```
