## Pythonエージェントの動作環境

WhaTap Pythonモニタリングエージェントをインストールする前に、サポート環境を確認してください。

- **対応バージョン**

  - WSGI Application (Python 3.6バージョン以降、3.13バージョン以下)

  - ASGI Application (Python 3.7バージョン以降、3.13バージョン以下)

- **OS**

  - Debian／Ubuntu 12.04(64bit) 以降

  - OSX 10.15 Catalina

  - Red Hat/CentOS 6.x(64bit) 以降

- **アプリケーションサーバー**

  - Bottle

  - Celery

  - CherryPy

  - Django

  - FastAPI (0.84.0以降）

  - Flask

  - Frappe

  - Nameko

  - Odoo (v16, v17)

  - Tornado

- **ライブラリ**

  - 外部呼び出し: HttpLib、Requests、Urllib、HTTPX

  - SQL Alchemy, GraphQL

  - smtplib, logging , loguru

  - Kombu、Pika

  - LLM: OpenAI

- **データベース**

  - MongoDB(pymongo)

  - MySQL(MySQLdb, pymysql)

  - Neo4j(neo4j)

  - Oracle(cxOracle)

  - PostgreSQL(psycopg2)

  - Redis(redis)
