## Pythonエージェントの動作環境

WhaTap Pythonモニタリングエージェントをインストールする前に、サポート環境を確認してください。

:::info
最終更新日: 2025-12-23
:::


### サポートバージョン

| アプリケーションタイプ | Pythonバージョン |
| --- | --- |
| WSGIアプリケーション | 3.6 ~ 3.13 |
| ASGIアプリケーション | 3.7 ~ 3.13 |

### オペレーティングシステム

| OS | 最小バージョン | アーキテクチャ |
| --- | --- | :---: |
| Debian / Ubuntu | 12.04 | 64bit |
| Red Hat / CentOS | 6.x | 64bit |
| macOS | 10.15 Catalina | - |
| Windows | Windows Server 2008 R2, Windows 7 | 64bit |

| フレームワーク | サポートバージョン |
| --- | --- |
| Bottle | - |
| Celery | - |
| CherryPy | - |
| Django | - |
| FastAPI | 0.84.0以上 |
| Flask | - |
| Frappe | - |
| Nameko | - |
| Odoo | v16, v17 |
| Tornado | - |

### ライブラリ

| カテゴリー | ライブラリ |

| **メッセージング** | Kombu, Pika |
