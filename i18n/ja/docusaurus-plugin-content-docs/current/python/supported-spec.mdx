## Pythonエージェントの動作環境

WhaTap Pythonモニタリングエージェントをインストールする前に、サポート環境を確認してください。

-   **OS**

    -   Redhat/CentOS 6.x(64bit) 以降

    -   Debian／Ubuntu 12.04(64bit) 以降

    -   OSX 10.15 Catalina

-   **対応バージョン**

    -   WSGI Application (Python 2.7 & Python 3.3 以降)

    -   ASGI Application(Python 3.7 以降)

-   **対応ライブラリ**

    -   外部呼び出し：HttpLib、Requests、Urllib

    -   データベース：MySQL、SQL Server、PostgreSQL、Redis

    -   Kombu, Pika

-   **アプリケーションサーバー**

    Django, Bottle, CherryPy, Flask, Tornado, Nameko, Celery, FastAPI(0.84.0 以降)
