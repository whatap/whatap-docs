Pythonアプリケーションサーバーの機能を制御します。

-   **enabled** <span class='type'>Boolean</span>

    デフォルト値 `true`

    全機能を活性化します。値が`false`の場合でも、サーバーと最小限の通信を維持するための情報は送信されます。

-   **counter_enabled** <span class='type'>Boolean</span>

    デフォルト値 `true`

    性能カウンターの追跡を活性化します。性能カウンタには、アクティブトランザクション、ユーザー、CPU、メモリ、SQL、外部HTTP呼び出しなどのデータが含まれます。`enabled`の値が`false`であれば無視されます。

-   **stat_enabled** <span class='type'>Boolean</span>

    デフォルト値 `true`

    統計情報の追跡を活性化します。5分ごとにトランザクション、SQL Query、外部HTTP呼び出し、User Agent、Client IPなどの統計データが収集されることになっていますが、これらの情報の収集が中断されます。 `enabled`の値が`false`であれば無視されます。 

-   **active_stack_enabled** <span class='type'>Boolean</span>

    デフォルト値 `true`

    アクティブスタックの追跡をアクティブにします。

    :::note

    `counter_enabled`値が無効になった場合、アクティブトランザクションデータが収集されないため、アクティブスタックも収集されません。

    :::

-   **eactive_stack_second** <span class='type'>Seconds</span>

    デフォルト値`10`

    アクティブスタックを追跡する間隔を設定します。

    :::caution

    値を変えないことをお勧めします。

    :::

-   **realtime_user_thinktime_max** <span class='type'>MiliSeconds</span>

    デフォルト値`300000`

    リアルタイムユーザーを測定するときに、同一ユーザーとして認められる最大の呼び出し間隔を指定します。

-   **time_sync_interval_ms** <span class='type'>MiliSeconds</span>

    デフォルト値`300000`

    エージェントとサーバー間の時間の同期化サイクルを指定します。同期化しない場合は0と指定します。  

-   **realtime_user_thinktime_max** <span class='type'>Int</span>

    デフォルト値`0`

    WhaTapエージェントは、一度送ったテキストタイプのデータはhash処理されるため、翌日まで再送信しません。以前の設定値と異なる値を設定する場合は、再送信します。

    :::note

    トランザクションURL、SQL Stringなどがテキストタイプデータに対応します。

    :::
