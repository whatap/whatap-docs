インストールスクリプトを使用してトレーサー、PHP拡張モジュール、およびwhatap-phpサービスデーモンを削除します。その後、パッケージ(yum、 apt-get) の削除に進み、必要に応じて*/usr/whatap/php*のログファイルおよびその他のファイルを削除します。

## Linux / Unix

1. PHP 拡張モジュールと whatap-php サービスを削除します。

    ```bash
    $ /usr/whatap/php/install.sh remove
    ```

1. パッケージを削除してください。

    <Tabs>
    <TabItem value="rdct" label="Red Hat/CentOS" default>

        ```bash
        $ sudo yum remove whatap-php
        ```
    
    </TabItem>
    <TabItem value="dbub" label="Debian/Ubuntu">

        ```bash
        $ sudo apt-get purge whatap-php
        ```

    </TabItem>
    <TabItem value="frbsd" label="FreeBSD">

        ```bash
        $ pkg delete whatap-php
        ```

    </TabItem>
    </Tabs>

## Windows

:::caution

Windows でアンインストールする場合、事前に Web サーバーを停止し、削除後に再起動することを推奨します。

:::

**GUI アンインストール**

```cmd
# 1. Web サーバー停止（推奨）
net stop Apache2.4
REM または
iisreset /stop

# 2. アンインストーラー実行
"C:\Program Files\WhaTap\PHP\unins000.exe"

# 3. Web サーバー開始
net start Apache2.4
REM または
iisreset /start
```

**CLI サイレントアンインストール**

```cmd
# 1. Web サーバー停止（推奨）
net stop Apache2.4
REM または
iisreset /stop

# 2. サイレント削除
"C:\Program Files\WhaTap\PHP\unins000.exe" /VERYSILENT

# 3. Web サーバー開始
net start Apache2.4
REM または
iisreset /start
```

**例：バッチスクリプト**

```batch
@echo off
echo Stopping web server...
net stop Apache2.4

echo Uninstalling WhaTap PHP Agent...
"C:\Program Files\WhaTap\PHP\unins000.exe" /VERYSILENT

timeout /t 3

echo Starting web server...
net start Apache2.4
echo Uninstallation completed!
```
