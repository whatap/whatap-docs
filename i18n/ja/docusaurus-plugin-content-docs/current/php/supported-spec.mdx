## PHPエージェント動作環境

WhaTap PHPモニタリングエージェントをインストールする前に、サポート環境を確認してください。

- **対応バージョン**

  - PHP extension moduleとして動作
  - PHP PHP 5 >= 5.2.0, PHP 7, PHP 8
  - NTS / ZTS (Zend Thread Safe)

- **OS**

  - CentOS／Red Hat 6.x(64bit) 以降
  - Debian／Ubuntu 12.04(64bit) 以降
  - Alpine Linux(64bit)
  - FreeBSD 10.x(64bit) 以降
  - Linux-aarch64 (ARM64)

- **アプリケーションサーバー**

  - Apache
  - PHP-FPM(Nginx)

- **ライブラリ**

  - PDO
  - MySQLi
  - cURL
  - oci8
  - mssql
  - sqlsrv
  - psql
  - phpredis
  - CUBRID
  - odbc

- **データベース**

  - Oracle(OCI8)
  - MySQL
  - SQL Server (mssql, sqlsrv)
  - PostgreSQL(psql)
  - Redis(phpredis)
  - CUBRID
  - Tibero(odbc)
