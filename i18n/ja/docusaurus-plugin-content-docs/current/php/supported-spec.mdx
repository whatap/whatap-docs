## PHPエージェント動作環境

WhaTap PHPモニタリングエージェントをインストールする前に、サポート環境を確認してください。

### サポートバージョン

| 区分 | サポート範囲 |
| --- | --- |
| **動作方式** | PHP拡張モジュール |
| **PHPバージョン** | PHP 5.2.0以上、PHP 7、PHP 8 |
| **Thread Safe** | NTS / ZTS (Zend Thread Safe) |

### オペレーティングシステム

| OS | 最小バージョン | アーキテクチャ |
| --- | --- | :---: |
| CentOS / Red Hat | 6.x | 64bit |
| Debian / Ubuntu | 12.04 | 64bit |
| Alpine Linux | - | 64bit |
| FreeBSD | 10.x | 64bit |
| Linux | - | ARM64 (aarch64) |

### アプリケーションサーバー

| サーバー | サポート |
| --- | :---: |
| Apache | ✅ |
| PHP-FPM (Nginx) | ✅ |
