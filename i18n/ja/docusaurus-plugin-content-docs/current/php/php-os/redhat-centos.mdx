1.  WhaTap **リポジトリ**を登録します。

    ```bash title=SH
    $ sudo rpm -Uvh http://repo.whatap.io/centos/5/noarch/whatap-repo-1.0-1.noarch.rpm
    ```

2.  次のコマンドを使用して**パッケージ**をインストールしてください。 

    ```bash title=SH
    $ sudo yum install whatap-php
    ```
