PHPエージェントは、必要な設定を*whatap.ini*ファイルに作成して変更することができます。ユーザーの利便性を高めるため、[WhaTapモニタリングサービス](https://service.whatap.io)もエージェント設定機能を提供しています。

:::note

一般的に設定はランタイムですぐに reload されます。ただし、設定の仕組みで Apache またはPHP-FPM の起動時に適用されるオプションは、再起動が必要な場合があります。

:::
