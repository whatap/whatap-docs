### JDBCダウンロード

DBサーバーの*/`db2_instance_path`/sqllib/java*経路から*db2jcc4.jar*ファイルをコピーして*/`agent_installation_path`/jdbc*フォルダに張り付けます。

```txt
/圧縮を解凍したフォルダ/jdbc
```

:::note

IBM DB2のJDBCドライバー構成の詳細については、[次のリンク](https://www.ibm.com/docs/baw/23.x?topic=installation-configuring-jdbc-drivers)を参照してください。

:::
