WhaTapエージェントのインストールが完了しましたら、[WhaTapモニタリングサービス](https://service.whatap.io)にログインしてください。WhaTapモニタリングサービスは、円滑なサービス提供と安定した運用、異常検知モニタリングおよび把握のための情報を提供します。

<!-- 주요 메뉴 안내 개요 위치 -->

サーバーのモニタリングは、リアルタイム収集によりトラブル原因を簡単に特定できます。利便性を高めるためにレポート機能を強化し、ユーザライクな通知方法を選択できます。

WhaTapサーバーモニタリング サービスは、次の主要な機能を提供します。

- **リアルタイム統合モニタリング**

  - 300台以下のサーバーリソースのリアルタイムな使用状況の情報
  - サーバー全体のサマリー指標を提供
  - CPU、Memory、Disk I/O、IOPS使用率による上位5台のサーバーリストを提供
  - CPU、Memoryの最大占有率による上位5台のプロセスのリストを提供

- **性能分析**

  - 収集サーバー全体の状況に関する情報
  - 1分間に10回以上収集されたCPU、メモリ、Disk、Networkの性能情報
  - 最大5台のサーバー間のメトリックスチャートによる性能の比較情報
  - プロセスのグループ化適用後のマルチプロセスリソース使用量の合計情報

- **イベント通知**

  - メール、SMS、SNS(Telegram、Slack) 送信
  - イベント継続期間およびしきい値設定によるイベント発生および履歴データのイベントシミュレーション
  - ミュート設定時間登録及びキーワードの繰り返しによる同一イベント通知の抑制機能
  - 個々の通知をまとめてPolicyとして提供し、特定サーバーに自動的に適用する機能
  - 再起動の有無、通信障害の持続時間、リソース使用量によるサーバーイベント設定の通知
  - プロセス数の増減、CPU使用量、メモリ使用量によるプロセスイベント設定の通知
  - ファイルログキーワードによるイベント設定およびイベントリスク設定の通知

- **統計とレポート**

  - サーバー日次・週次レポート
  - サーバー月次レポート

- **管理機能**

  - 権限管理
  - エージェント削除
  - メンテナンス計画の事前入力
