## OS選択

<Cmdname sid="side_agentInstall" className="uitext" />画面で、インストールするエージェントのOSを選択します。

:::tip

プロジェクトを作成すると、自動的に<Cmdname sid="agent_installation" className="uitext" />ページに移動します。<Cmdname sid="agent_installation" className="uitext" />ページが表示されない場合は、左側のメニューから<Cmdname sid="all_projects" className="uitext" />を選択し、新しく作成したプロジェクトを選択します。
