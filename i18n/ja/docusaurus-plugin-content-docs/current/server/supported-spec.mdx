## OS別のサポート環境

WhaTapのサーバーモニタリングエージェントをインストールする前に、サポート環境を確認してください。X86ベースで動作する **Linux**、**FreeBSD**、**AIX**、**Windows**に適用されます。

:::info
最終更新: 2025-12-10
:::

| エージェントバージョン | RHEL/CentOS 5.8+ | Rocky Linux 8.6+ | SUSE 10.1+ | Amazon Linux | Ubuntu 12.04+ | Debian 6.0+ |
| --- | :---: | :---: | :---: | :---: | :---: | :---: | :---: |
| v2.9.5 | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ |

### Unix

| エージェントバージョン | FreeBSD 10+ | AIX 6.1+ | Oracle Solaris 10+ | Oracle Solaris 11.2+ | HP-UX 11.23+ |

| v1.4.4 | ✅ | ✅ | ✅ | ✅ | ✅ |

### Windows

| エージェントバージョン | Windows 2008 R2+ |
| --- | :---: |
