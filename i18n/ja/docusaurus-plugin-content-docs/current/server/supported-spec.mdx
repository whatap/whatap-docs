## OS別のサポート環境

WhaTapのサーバーモニタリングエージェントをインストールする前に、サポート環境を確認してください。X86ベースで動作する **Linux**、**FreeBSD**、**AIX**、**Windows**に適用されます。

- **Linux**

  - CentOS 5以降
  - Red Hat Server5.0以降(RHELまたはRHEL基盤のディストリビューションをサポート)
  - Rocky Linux 8.6以降
  - SUSE 10.1以降
  - Amazon Linuxすべてのバージョン 
  - Ubuntu 12.04以降
  - Debian 6.0以降

- **Unix**

  - FreeBSD 10以降

  - AIX 6.1以降

  - Oracle Solaris 11.2以降

  - HP-UX 11.31(IA64), HP-UX 11.23(IA64, PA-RISC)

  - Oracle Solaris 10、Solaris 11.2、Solaris 11.3、Solaris 11.4

- **Windows**

  - Windows 2008 R2以降

:::note

使用環境がサポート一覧に含まれていない場合は、support@whatap.io までお問い合わせください。

:::
