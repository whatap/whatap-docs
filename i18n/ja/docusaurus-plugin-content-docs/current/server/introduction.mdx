WhaTapサーバーモニタリングサービスは1分以内にインストールでき、実務担当者の観点から迅速に導入可能です。CPUやディスクI/Oなど、しきい値を超える可能性のある指標は5秒間隔で収集し、平均値による検出の限界を克服します。また、20秒間隔のプロセスモニタリングも提供しています。

統合モニタリングのための複数のダッシュボードを提供しており、システム全体の状況を簡単に把握できます。クラウド環境に適した指標に対応しており、スケールアウト環境にも適用可能です。

WhaTapのサーバーモニタリングサービスを利用すると、運用担当者がいなくてもサーバーを安定して運用できます。

## 主な機能

- **多様な規模のサーバーモニタリング**

    さまざまな種類のダッシュボードを通じて、システム全体の状況をリアルタイムで確認できます。

- **サーバーインベントリ管理**

    表示したい項目を選択して表示できるテーブル形式でサーバーリストを管理し、リアルタイムモニタリングが可能です。

- **リアルタイム統合アラート**

    リソースの推移モニタリングや異常指標の検出にとどまらず、リアルタイムの統合アラート履歴を通じて障害に迅速に対応できます。サーバー、プロセス、キーワードに基づくアラートに対応しています。

- **システム担当者向けのWhaTapレポート**

    運用担当者に対してさまざまな種類のレポートを提供し、手作業の負担を軽減します。必要なレポートはフィードバックに基づいて追加することができます。

- **クラウド性能の脆弱性の検知**

    クラウド環境のアプリケーションで発生し得る脆弱性を自動的に検出し、アラートを提供します。

- **最適なシステム規模の予測**

    サーバー規模の変化をリアルタイムでモニタリングし、最適な規模を予測することができます。

- **プロセス性能の自動収集**

    すべてのプロセスのCPU、メモリ、I/O使用量を20秒単位で自動的に収集します。

- **マルチチャネルと個人用設定の通知**

    電子メール、SMS、3rd Party Plugin を使用して、パーソナライズされた通知サービスを提供します。

- **クラウド、物理サーバー、ハイブリッド環境に対応**

    クラウドおよび物理サーバー環境の両方に適した指標を提供し、スケールアウト環境でも活用できます。
