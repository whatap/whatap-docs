WhaTapサーバーモニタリングサービスは、1分以内にインストールが可能で実務者にとって簡単に適用することができ、モニタリングが容易です。 CPU、Disk I/Oなど、瞬間的に重要な間隔を超える可能性のあるメトリクスに対しては5秒単位で収集することで、平均化検出方式の制限を克服します。また、20 秒間隔でのプロセスモニタリングも提供します。 
統合モニタリングの複数のダッシュボードを提供することで、システム全体の現状把握が容易になります。 クラウド環境モニタリングに適したメトリクスを提供し、スケールアウト環境にも適用できます。 

WhaTapのサーバーモニタリングサービスを利用すると、オペレーターを必要とせずサーバーを安定して運用することができます。

## 主な特長

-   **多様な規模のサーバーモニタリング**

      統合モニタリング用のさまざまなタイプのダッシュボードを提供し、システム全体をリアルタイムで表示します。 

-   **サーバーインベントリ管理**

      管理対象サーバーの一覧は、ユーザーが表示したい項目を選択可能なテーブル形式で提供され、中小規模のサーバーの一覧表示の管理だけでなく、リアルタイムモニタリングも可能です。

-   **リアルタイム統合アラート**

      リソースの傾向をユーザーがモニタリングし、異常なメトリクスを検知するためのアクティブな応答からユーザーを開放するために、リアルタイムの統合通知履歴が提供され、通知履歴に基づいてトラブル対応が可能になります。サーバー、プロセス、キーワード検知方式の通知を提供します。

-   **システム担当者向けのWhaTapレポート**

      大量のリソースを安定して運用する必要があるオペレーター向けに、レポート用のデータを手動で作成する手間を軽減するために、さまざまな種類のレポートを提供します。 ユーザーが必要なレポートがある場合は、フィードバックに基づいてレポートタイプを追加します。

-   **クラウド性能の脆弱性の検知**

      クラウド環境のアプリケーションで発生する可能性のある脆弱性の自動検知とアラートを通知します。

-   **適切なシステム規模の判断**

      年によって必要なサーバーの規模は異なります。 クラウドを利用している場合、毎週ごと毎月ごとにリアルタイムで変化する場合があります。これはモニタリングを通じて予測され、適切なレベルの規模が算定されます。

-   **プロセス性能の自動収集**

      すべてのプロセスのCPU、メモリ、I/O使用率を20秒単位で自動的に収集します。

-   **マルチチャネルと個人用設定の通知**

      電子メール、SMS、3rd Party Plugin を使用して、パーソナライズされた通知サービスを提供します。

-   **クラウド、物理サーバー、ハイブリッド環境**

      物理サーバー以外にも、クラウド環境のモニタリングに適したメトリックスを提供し、スケールアウト環境に適用できます。
