ホーム画面 &gt; プロジェクト選択 &gt; <Cmdname sid="side_event" className="uitext" /> &gt; <Cmdname sid="side_eventHistory" className="uitext" />

警告通知が発生した履歴を確認できます。 発生したすべてのイベント履歴一覧の照会が可能です。 

<ImgLang img='event-history-server.png' desc='サーバーイベント履歴'/>

- ![number1](/img/number-01.png) **上段オプション**

  - <Cmdname sid="time_selector" className="uitext" />

    - 右側の緑色のボタンを選択して、表示時間を選択することができます。基本設定は12時間です。

    - ![左アイコン](/img/left-arrow.svg)または ![右アイコン](/img/right-arrow.svg)ボタンを選択すると、選択した表示時間だけ間隔を移動します。

    - 詳細な時間を選択するには、日付または時間の領域を選択してください。詳細な時間を設定し、<Cmdname sid="apply" className="uitext" />ボタンを選択します。

  - **_入力ウィンドウ_**  

    入力ウィンドウでサーバー名またはイベント名で検索できます。

  - <Cmdname sid="exclude_resolved_alerts" className="uitext" />

    イベント履歴の一覧から**進行中のイベント**のみ照会できます。

  - <Cmdname sid="BTN06585" className="uitext" />

    イベント履歴一覧に<Cmdname sid="TTL06583" className="uitext" />カラムを追加できます。 

    :::note

    進行中のイベントは、<Cmdname sid="TTL06583" className="uitext" />を照会しません。

    :::

  - **_CSV_**

    取得したイベント履歴結果をCSVで保存できます。<Cmdname sid="maximum_csv_lines" className="uitext" />を入力し<Cmdname sid="download" className="uitext" />ボタンを選択します。

- ![number2](/img/number-02.png)**イベント履歴一覧**

  - <Cmdname sid="sms_alert_severity" className="uitext" />

    リスクレベルは、 <span class='vslow'>Fatal</span>(危険), <span class='slow'>Warning</span>(警告)の色で表示されます。

  - <Cmdname sid="event_title" className="uitext" />

    イベントポリシーで設定した値に基づいて発生したイベント名です。

  - <Cmdname sid="event_time" className="uitext" />

    イベントが発生した時刻です。

  - **_+_** <Cmdname sid="sms_alert_acknowledge" className="uitext" /><ScrollToLink anchor='scrollTo-alert-acknowledge' />

    イベント処理履歴を追加できます。

  - <Cmdname sid="server" className="uitext" />

    サーバー名です。 

  - <Cmdname sid="sms_alert_snapshot" className="uitext" />

    発生したイベントに対する<Cmdname sid="cpu" className="uitext" />、<Cmdname sid="memory_pused" className="uitext" />、<Cmdname sid="disk" className="uitext" />、<Cmdname sid="network" className="uitext" />へのスナップショット情報です。 

### イベント詳細情報

イベント履歴リストから対象の<Cmdname sid="event_title" className="uitext" />を選択すると、イベント発生の前後5分間の<Cmdname sid="detail" className="uitext" />情報に移動します。

<ImgLang img='server-event-detail.png' desc='サーバーイベント詳細'/>

- ![number1](/img/number-01.png) **チャートウィジェット**

  アラーム発生の前後5分間の **_ CPU Usage_**、**_Memory Usage_**、**_Network_**、**_Disk I/O_** チャートを表示します。イベントに対象する部分の<Cmdname sid="sms_alert_severity" className="uitext" />を色付けして表示します。

- ![number2](/img/number-02.png) <Cmdname sid="sms_summary_process" className="uitext" />

  イベント対象プロセスの上位10個の情報を表示します。 

- ![number3](/img/number-03.png) <Cmdname sid="sms_alert_acknowledge_detail" className="uitext" />

  イベントの処理履歴を表示できます。<Cmdname sid="sms_alert_acknowledge_add" className="uitext" />ボタンを選択すると、履歴を作成して保存できます。
