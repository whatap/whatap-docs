## キューブとは？

WhaTapは5分単位で作成されたパフォーマンス統計を**キューブ**と呼びます。キューブ分析は、キューブに保存された5分単位のパフォーマンスデータを利用する分析機能です。

<ImgLang img='cube.png' desc='キューブ'/>

## ![](/img/number-01.png)キューブのセレクトパネル

左側のキューブセレクトパネルで、特定の時間を基準にキューブを選択します。 キューブは5分単位で保存されるため、任意の時間を選択する必要があります。 

- **パネル関心メトリクス**

  サーバーキューブセレクトパネルで関心のあるメトリクスは、 <Cmdname sid="cpu" className="uitext" />と <Cmdname sid="memory" className="uitext" />です。

- **キューブ表示色**

  キューブは、選択された関心指標の数値に応じて異なる色で表示されます。

  - <span class='highfg'>紺色</span>：<span class='highfg'>高い数値</span>

  - <span class='lowfg'> 水色</span>：<span class='lowfg'>低い数値</span>

## ![](/img/number-02.png)アプリケーションセレクター

上部![number 2](/img/number-02.png)領域のアプリケーションセレクターオプションを使用して、特定のアプリケーションのデータを選択することができます。右側の![down-arrow](/img/down-arrow.svg)ボタンをクリックすると、次のように選択可能なアプリケーション全体を確認できます。 

<ImgLang img='cube-select-app.png' desc='キューブ選択アプリケーション'/>

- <Cmdname sid="select_all" className="uitext" />：すべてのアプリケーションを表示します。既定値は<Cmdname sid="select_all" className="uitext" />と<Cmdname sid="oid_active" className="uitext" />が選択されています。

- <Cmdname sid="checked_only" className="uitext" />：アプリケーションからユーザーが指定したアプリケーションのみを表示します。

- <Cmdname sid="search" className="uitext" />：アプリケーションが多数の場合は、入力ウィンドウからそのアプリケーションを検索できます。

- <font color='5B7DEC'>数字</font>/<font color='979797'>数字</font>：<font color='5B7DEC'>指定されたアプリケーション数</font>/<font color='979797'>全体のアプリケーション数</font>を表示します。

## ![](/img/number-03.png)キューブデータ

:::note

- チャート右上の ![バーチャートアイコン](/img/ic-dashboard-bar-chart.svg)アイコンを選択すると、バーチャートが表示します。 
- チャート右上の ![個別アイコン](/img/ic-multiline-chart.svg)アイコンを選択すると、線形チャートを提供します。

:::

![number3](/img/number-03.png)領域で該当するキューブデータを<Cmdname sid="cpu" className="uitext" />、<Cmdname sid="memory" className="uitext" />、<Cmdname sid="disk" className="uitext" />、<Cmdname sid="network" className="uitext" />などのチャートから確認できます。

- <Cmdname sid="cpu" className="uitext" />

  プロジェクト内のアプリケーションをCPU使用量が多い順にソートして表示します。 

- <Cmdname sid="memory" className="uitext" />

  プロジェクト内のアプリケーションをメモリ使用量が多い順にソートして表示します。

- <Cmdname sid="disk" className="uitext" />

  プロジェクト内のアプリケーションをディスク使用量が多い順にソートして表示します。

- <Cmdname sid="network" className="uitext" />

  プロジェクト内のアプリケーションをネットワークの入出力トラフィック量が多い順にソートして表示します。

:::note

WhaTap uses the IP2Location LITE database for [IP geolocation](https://lite.ip2location.com).

:::
