**GPUインベントリ**では、サーバーに搭載されたGPUの仕様や状態を体系的に管理できます。

:::note

**サポートエージェントバージョン**

Linuxでのみサポートされており、エージェントバージョン 2.8.5 以上が必要です。

:::

## 基本画面ガイド

GPUインベントリは、システムに割り当てられたGPUおよびMIG（Multi-Instance GPU）リソースの全リストをモニタリングします。 各GPUインスタンスの詳細情報を確認できます。 

- 検索ボックスでキーワード検索が可能です。
- 表示結果の項目名をクリックすると、昇順または降順に並べ替えが可能です。

### **カラム設定**

画面右上の **列設定** ボタンをクリックすると、表示する列を選択できます。 カラム設定ウィンドウでカラムの選択およびカスタムカラムの追加後、**確認**ボタンをクリックして設定を保存してください。 表示項目からドラッグしてカラムの順序を変更できます。 

- **カラム選択**

    ユーザーはテーブルに表示するカラムを選択できます。**初期化**をクリックすると、デフォルトのカラム選択状態に戻ります。

    :::note

    選択した カラム はブラウザのクッキーに保存され、ページを更新しても状態が維持されます。 クッキーが削除されたり、他の理由で設定エラーが発生した場合は、デフォルト項目の選択状態に初期化します。

    :::

- **カラム追加**

    **+ カラム追加**ボタンを選択し、基本項目以外に任意のカラムを追加してカスタムカラム（`custom`）を構成できます。 必要に応じてカラムを追加し、サーバーの状態や性能をより詳細にモニタリングしてください。

### **修正**

**修正**ボタンを選択すると、サーバーインベントリリストを編集できます。

- **サーバー別名**と**基本グループ**のカラム内容を設定可能です。

- ユーザーが追加したカスタムカラム（`custom`）の内容も設定できます。
