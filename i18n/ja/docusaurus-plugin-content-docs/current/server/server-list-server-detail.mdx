ホーム画面 > プロジェクト選択 > ***サーバーリスト*** > ***サーバー詳細***

WhaTapモニタリングサービスの初期画面でプロジェクトを選択し、***サーバーリスト***メニューの***サーバー詳細***メニューを選択します。***サーバー詳細***は、サーバーのリソース現況と実行プロセスの詳細情報を提供します。

![サーバー詳細](/img/server-detail.png)

## ![](/img/number-01.png)上段オプション

上部![number 1](/img/number-01.png)の領域で***データ更新時間***および表示する***サーバー***を指定することができます。***データ更新時間***および***サーバー***指定オプションの下にあるサーバーの***サマリー情報***を確認することができます。

### データ更新時間

サーバー詳細ページのデータは5秒ごとに自動的に更新されます。 

<div style={{textAlign: 'center'}}>

![sc](/img/server-detail-data.png)

</div>

- ![一時停止アイコン](/img/ic-s-pause.svg)***一時停止***：ユーザーはデータの自動更新を停止できます。 
- ![再生アイコン](/img/ic-play-grn.svg)***再生***：ユーザーは停止した自動更新を再開できます。
- ![更新アイコン](/img/ic-refresh2.svg)***更新***：ユーザーは現在の時間を基準にデータを更新できます。

## ![](/img/number-02.png)サーバー詳細

### サーバーリソースの状態

![number 2](/img/number-02.png)領域では、CPUとメモリ使用率、ネットワークおよびディスクI/Oのチャートを使用して、サーバーのリソースの状態を確認することができます。  

- ![csvアイコン](/img/ic-csv-download.svg)：***CSV***アイコンを選択すると、チャート履歴をCSVファイルで保存できます。

- ![カメラアイコン](/img/ic-camera.svg)：***カメラ***アイコンを選択すると、イメージ形式でチャート画面を保存できます。

- ![詳細アイコン](/img/ic-zoom-in.svg)：***詳細***アイコンを選択して、CPU使用率、メモリ使用率、ネットワークおよび、ディスクI/Oの***詳細***が確認できます。

### プロセスリスト

#### ソート

- ***プロセス一覧***で選択された指標を基準に降順または昇順のソートを指定して照会できます。 

- ***プロセス一覧***の上で選択したメトリクスにより、上位の***10個***、***20個***、***30個***、***全体***のプロセスを指定して確認できます。 

#### グループ化

プロセスグループは、一つのインスタンスから同じく動作する複数のプロセスをまとめたものです。***グループ化***ボタンを選択し、プロセスグループを追加できます。プロセスのグループについては、[次の文書](server-detail-process-group)を参照してください。

### サーバー情報パネル

***サーバー情報パネル***で確認できる内容は次のとおりです。 

- ![通知アイコン](/img/ic-alert.svg) ***サーバーアラートリスト***

  ***サーバー情報パネル***の![通知アイコン](/img/ic-alert.svg)***サーバー通知一覧***ボタンを選択すると、サーバーに割り当てられている通知を確認できます。

- ***Display Name***/***ID***

  選択したサーバーの***表示名***(ONAME)と***サーバーID***(OID)が確認できます。![修正アイコン](/img/ic-edit.svg)***修正***アイコンを選択するとONAMEを修正することができます。

- ***OS***

  選択したサーバーの***OS情報***を確認できます。

- ***IP Address***

  選択したサーバーの***IPアドレス***を確認できます。

- ***Agent***

  選択したサーバーの***エージェントのバージョン***を確認できます。

- ***Memo***

  作成した***メモ***を確認するか![編集アイコン](/img/ic-edit.svg)***編集***アイコンを選択してメモを編集および追加できます。
