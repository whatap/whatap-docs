ホーム画面 > プロジェクト選択 > **_サーバーリスト_** > **_サーバー詳細_**

WhaTapモニタリングサービスの初期画面でプロジェクトを選択し、**_サーバーリスト_**メニューの**_サーバー詳細_**メニューを選択してください。**_サーバー詳細_**は、サーバーのリソース現況と実行プロセスの詳細情報を提供します。

![サーバー詳細](/img/server-detail.png)

## ![](/img/number-01.png)上段オプション

上部![number 1](/img/number-01.png)の領域で**_データ更新時間_**および表示する**_サーバー_**を指定することができます。**_データ更新時間_**および**_サーバー_**指定オプションの下にあるサーバーの**_サマリー情報_**を確認することができます。

### データ更新時間

サーバー詳細ページのデータは5秒ごとに自動的に更新されます。 

<div style={{textAlign: 'center'}}>

![sc](/img/server-detail-data.png)

</div>

- ![一時停止アイコン](/img/ic-s-pause.svg)**_一時停止_**：ユーザーはデータの自動更新を停止することができます。 
- ![再生アイコン](/img/ic-play-grn.svg)**_再生_**：ユーザーは停止した自動更新を再開することができます。
- ![更新アイコン](/img/ic-refresh2.svg)**_更新_**：ユーザーは現在の時間を基準にデータを更新することができます。

## ![](/img/number-02.png)サーバー詳細

### サーバーリソースの状態

![number 2](/img/number-02.png)領域では、CPUとメモリ使用率、ネットワークおよびディスクI/Oのチャートを使用して、サーバーのリソースの状態を確認することができます。  

- ![csvアイコン](/img/ic-csv-download.svg)：**_CSV_**アイコンを選択すると、チャート履歴をCSVファイルとして保存することができます。

- ![カメラアイコン](/img/ic-camera.svg)：**_カメラ_**アイコンを選択して、画像形式でチャート画面を保存することができます。

- ![詳細アイコン](/img/ic-zoom-in.svg)：**_詳細_**アイコンを選択して、CPU使用率、メモリ使用率、ネットワークおよび、ディスクI/Oの**_詳細_**が確認できます。

### プロセスリスト

#### 配置

- **_プロセス一覧_**上で選択した区分に従って、降順または昇順のソートを指定して参照することができます。 

- **_プロセス一覧_**の上で選択したメトリクスによって上位の**_10個_**、**_20個_**、**_30個_**、**_全体_**プロセスを指定して確認することができます。 

#### グループ化

プロセスグループは、インスタンスで同じ動作を実行する複数のプロセスをまとめたものです。**_グループ化_**ボタンを選択してプロセスグループを追加することができます。プロセスのグループ化の詳細については、[次の文書](server-detail-process-group)を参照してください。

### サーバー情報パネル

**_サーバー情報パネル_**で確認できる内容は次のとおりです。 

- ![通知アイコン](/img/ic-alert.svg) **_サーバーアラートリスト_**

  **_サーバー情報パネル_**の上の![通知アイコン](/img/ic-alert.svg)**_サーバー通知一覧_**ボタンを選択すると、現在のサーバーに割り当てられた通知を確認することができます。

- **_Display Name_**/**_ID_**

  選択したサーバーの**_表示名_**(ONAME)と**_サーバーID_**(OID)が確認できます。![修正アイコン](/img/ic-edit.svg)**_修正_**アイコンを選択するとONAMEを修正することができます。

- **_OS_**

  選択したサーバーの**_OS情報_**を確認することができます。

- **_IP Address_**

  選択したサーバーの**_IPアドレス_**を確認することができます。

- **_Agent_**

  選択したサーバーの**_エージェントのバージョン_**を確認することができます。

- **_Memo_**

  作成した**_メモ_**を確認します![修正アイコン](/img/ic-edit.svg)**_編集_**アイコンを選択してメモを編集および追加します。
