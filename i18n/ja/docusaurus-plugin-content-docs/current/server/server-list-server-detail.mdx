ホーム画面 > プロジェクト選択 > **_サーバーリスト_** > **_サーバー詳細_**

WhaTapモニタリングサービスの初期画面でプロジェクトを選択し、**_サーバーリスト_**メニューの**_サーバー詳細_**メニューを選択します。**_サーバー詳細_**は、サーバーのリソース現況と実行プロセスの詳細情報を提供します。

![サーバー詳細](/img/server-detail.png)

## ![](/img/number-01.png)上段オプション

上部![number 1](/img/number-01.png)の領域で**_データ更新時間_**および表示する**_サーバー_**を指定することができます。**_データ更新時間_**および**_サーバー_**指定オプションの下にあるサーバーの**_サマリー情報_**を確認することができます。

### データ更新時間

サーバー詳細ページのデータは5秒ごとに自動的に更新されます。 

<div style={{textAlign: 'center'}}>

![sc](/img/server-detail-data.png)

</div>

- ![一時停止アイコン](/img/ic-s-pause.svg)**_一時停止_**：ユーザーはデータの自動更新を停止できます。 
- ![再生アイコン](/img/ic-play-grn.svg)**_再生_**：ユーザーは停止した自動更新を再開できます。
- ![更新アイコン](/img/ic-refresh2.svg)**_更新_**：ユーザーは現在の時間を基準にデータを更新できます。

## ![](/img/number-02.png)サーバー詳細

### サーバーリソースの状態

![number 2](/img/number-02.png)領域では、CPUとメモリ使用率、ネットワークおよびディスクI/Oのチャートを使用して、サーバーのリソースの状態を確認することができます。  

- ![csvアイコン](/img/ic-csv-download.svg)：**_CSV_**アイコンを選択すると、チャート履歴をCSVファイルで保存できます。

- ![カメラアイコン](/img/ic-camera.svg)：**_カメラ_**アイコンを選択すると、イメージ形式でチャート画面を保存できます。

- ![詳細アイコン](/img/ic-zoom-in.svg)：**_詳細_**アイコンを選択して、CPU使用率、メモリ使用率、ネットワークおよび、ディスクI/Oの**_詳細_**が確認できます。

### プロセスリスト

#### ソート

- **_プロセス一覧_**で選択された指標を基準に降順または昇順のソートを指定して照会できます。 

- **_プロセス一覧_**の上で選択したメトリクスにより、上位の**_10個_**、**_20個_**、**_30個_**、**_全体_**のプロセスを指定して確認できます。 

#### グループ化

プロセスグループは、一つのインスタンスから同じく動作する複数のプロセスをまとめたものです。**_グループ化_**ボタンを選択し、プロセスグループを追加できます。プロセスのグループについては、[次の文書](server-detail-process-group)を参照してください。

### サーバー情報パネル

**_サーバー情報パネル_**で確認できる内容は次のとおりです。 

- ![通知アイコン](/img/ic-alert.svg) **_サーバーアラートリスト_**

  **_サーバー情報パネル_**の![通知アイコン](/img/ic-alert.svg)**_サーバー通知一覧_**ボタンを選択すると、サーバーに割り当てられている通知を確認できます。

- **_Display Name_**/**_ID_**

  選択したサーバーの**_表示名_**(ONAME)と**_サーバーID_**(OID)が確認できます。![修正アイコン](/img/ic-edit.svg)**_修正_**アイコンを選択するとONAMEを修正することができます。

- **_OS_**

  選択したサーバーの**_OS情報_**を確認できます。

- **_IP Address_**

  選択したサーバーの**_IPアドレス_**を確認できます。

- **_Agent_**

  選択したサーバーの**_エージェントのバージョン_**を確認できます。

- **_Memo_**

  作成した**_メモ_**を確認するか![編集アイコン](/img/ic-edit.svg)**_編集_**アイコンを選択してメモを編集および追加できます。
