ホーム画面 &gt; プロジェクト選択 &gt; <Cmdname sid="side_serverList" className="uitext" /> &gt; <Cmdname sid="side_serverList" className="uitext" />

WhaTapモニタリングサービスの初期画面でプロジェクトを選択し、<Cmdname sid="side_serverList" className="uitext" /> &gt; <Cmdname sid="side_serverList" className="uitext" />メニューを選択します。<Cmdname sid="side_serverList" className="uitext" />は、収集サーバーの全体状況を表示し、サーバーの管理機能およびサーバー間の比較機能を提供します。

<ImgLang img='server-list.png' desc='サーバーリスト'/>

## ![](/img/number-01.png)データ更新とサーバーステータス

### データ更新

サーバーリストページのデータは、60秒ごとに自動的に更新されます。

- ![一時停止アイコン](/img/ic-s-pause.svg)**_一時停止_**：ユーザーはデータの自動更新を停止できます。 

- ![再生アイコン](/img/ic-play-grn.svg)**_再生_**：ユーザーは停止した自動更新を再開できます。

:::note 

タイムセレクターの利用方法の詳細については、[次の文書](usage-ui#timeselector)を参照してください。

:::

### サーバー状態

対象サーバーのステータスごとの状況を表示します。ステータスは1分間隔で更新されます。

- ![非活性アイコン](/img/ico-disable-agent.svg) <Cmdname sid="status_inactive" className="b600" />：エージェントは、そのサーバーからデータを収集できない状態です。

- ![危険アイコン](/img/ic-warning-critical.svg) <Cmdname sid="status_fatal" className="b600" />：収集サーバーで危険イベントが発生した状態です。

- ![警告アイコン](/img/ic-warning-filled.svg) <Cmdname sid="status_warning" className="b600" />：収集サーバーで警告イベントが発生した状態です。

- ![正常アイコン](/img/ic-check-circle-filled.svg) <Cmdname sid="status_ok" className="b600" />：エージェントが収集サーバーで正常に動作している状態です。

- ![アイコン](/img/ic-s-pause.svg) <Cmdname sid="status_paused" className="b600" />：エージェントがデータ収集を一時的に停止した状態です。

## ![](/img/number-02.png)検索

上段右側![number-s](/img/number-02.png)の**_検索_**を活用してサーバー名(ONAME)、サーバーID(OID)、タグの中から入力値が含まれたサーバーリストをフィルタリングできます。

## ![](/img/number-03.png)サーバーステータス

エージェントが割り当てられたサーバー全体のステータスを示します。一覧からエージェントを選択すると、<Cmdname sid="side_serverDetail" className="uitext" />ページに移動します。<Cmdname sid="side_serverDetail" className="uitext" />メニューの詳細については、[次の文書](server-detail)を参照してください。

### カラム設定

- ![カラムアイコン](/img/ic-column.svg) <Cmdname sid="column_settings" className="uitext" />

  ![number-s](/img/number-03.png)領域の上部の ![カラムアイコン](/img/ic-column.svg) <Cmdname sid="column_settings" className="uitext" />ボタンをクリックすると、次のようにテーブルに表示するカラムを選択できます。

  <ImgLang img='server-list-cl.png' desc='サーバーリストカラム'/>

  :::note

  選択した**_カラム_**は、ブラウザのクッキー値として保存されます。また、ページの更新後もステータスが維持されます。クッキーの削除またはその他の理由で設定エラーが発生した場合、すべて選択に初期化されます。

  :::

- **カラム幅設定**

  カラムタイトルの端をドラッグして、カラム幅を設定できます。

- **カラムソート**

  カラムのタイトルをクリックすると、昇順または降順の並べ替えが可能です。

  :::note

  **カラム幅** および **カラムソート** 設定値は、ブラウザクッキーとして保存され、ページの更新後もステータスが維持されます。クッキーの削除またはその他の理由で設定にエラーが発生した場合は、未設定状態でリセットされます。

  :::

### リフレッシュ

![更新アイコン](/img/ic-refresh2.svg) <Cmdname sid="refresh" className="uitext" />ボタンをクリックすると、現在の時刻を基準にデータを更新できます。

### フィルター

![number-s](/img/number-03.png)領域の上段メニューから![フィルターアイコン](/img/ico-filter.svg) <Cmdname sid="filter" className="uitext" />ボタンをクリックすると、タグを使用してサーバーをフィルタリングできます。 

タグは、エージェントをインストール時に自動的に付与されます。 ユーザーが入力値を追加できます。 タグを使用して収集サーバーを識別するか、サーバーのステータスをフィルタリングします。

<ImgLang img='server-list-filter.png' desc='サーバーリストフィルター'/>

- ![number-s](/img/number-04.png)領域でタグを選択して、サーバーのステータスをフィルタリングできます。 

- エージェントごとにタグを入力して、新しいタグ機能を有効にできます。![number-s](/img/number-05.png)領域の入力ウィンドウを確認してください。

### モニタリング管理

エージェントコマンドを使用してサーバーを管理する機能です。![サーバ管理アイコン](/img/ico-setting.svg)<Cmdname sid="sms_management" className="uitext" />ボタンをクリックし、一覧から管理機能を適用するサーバーを選択します。右側の<Cmdname sid="paused_agent" className="uitext" />、<Cmdname sid="resume_agent" className="uitext" />、<Cmdname sid="closed_agent" className="uitext" /> ボタンが有効化されます。

- ![一時停止アイコン](/img/ic-s-pause.svg) <Cmdname sid="paused_agent" className="uitext" />：エージェントのデータ収集を一時停止します。サーバーリストで一時停止として表示されます。

- ![再起動アイコン](/img/ic-caret-right.svg) <Cmdname sid="resume_agent" className="uitext" />：エージェントを再起動します。

- ![削除アイコン](/img/ic-delete-critical.svg) <Cmdname sid="exclude" className="uitext" />：エージェントの動作を停止します。サーバーリストから除外され、サーバーのステータスを確認できません。

### エージェント移動

![移動アイコン](/img/ic-swap.svg) <Cmdname sid="BTN07897" className="uitext" />ボタンを選択して、エージェントを他のプロジェクトに移転できます。

<ImgLang img='server-list-agent-transfer.png' desc='サーバーリストエージェント移動'/>

1. 右側の画面で、移動する対象プロジェクトを選択します。

2. 左側の画面で、移動するエージェントを選択してください。

3. ![right-arrow](/img/right-arrow.svg)**_右矢印_** ボタンをクリックして、対象プロジェクトに移動します。

4. <Cmdname sid="save" className="uitext" />ボタンをクリックすると移転が完了します。

:::note

- 実行から10分以内に同じエージェントを再度移動する場合、クラッシュが発生する可能性があります。 
- 収集されたデータは移動されません。 履歴データを参照するために、既存のプロジェクトは削除しないでください。

:::

### サーバー間の比較

![チャート比較アイコン](/img/ic-multiline-chart2.svg) <Cmdname sid="TTL06636" className="uitext" />ボタンをクリックすると、サーバー間の比較チャートが次のように提供されます。 

<ImgLang img='server-list-compare.png' desc='サーバー間の比較 sc'/>

1. ![チャート比較アイコン](/img/ic-multiline-chart2.svg) <Cmdname sid="TTL06636" className="uitext" />ボタンを選択します。

2. ![number-s](/img/number-06.png)をサーバーリストの左側に<Cmdname sid="TTL06636" className="uitext" />カラムが追加されます。

3. <Cmdname sid="TTL06636" className="uitext" />カラムで比較するサーバーを選択します。

4. 比較機能を有効にすると![number-s](/img/ic-number-07.svg)領域で比較チャートを参照することができます。

:::note

- 比較サーバーは最大5つまで選択できます。

- サーバーごとに色が割り当てられます。色は![number-s](/img/ic-number-07.svg) <Cmdname sid="compare" className="uitext" />でサーバーを区別するために使用されます。

- <Cmdname sid="check_period" className="uitext" />ボタンをクリックすると、履歴データを照会できます。

:::
