WhaTapのGoアプリケーションモニタリングは、Goアプリケーションに対するモニタリングサービスを提供します。

- ウェブフレームワークに対応します。 ウェブトランザクションのURL、レスポンス時間、TPS情報、エラーメッセージ情報などを収集します。
- Go Runtimeパッケージの情報を継続的に収集します。メモリ、goroutine、gc関連情報を収集します。
- MSA環境で複数のAPI/RPCサービスを呼び出す場合、関連付け追跡を使用して呼び出し関係を収集します。
