Goウェブアプリケーションサーバーに接続したユーザーの数を集計します。

-   **trace_user_header_ticket** <span class='type'>String</span>

    HTTP Headerの特定の値でユーザー数を集計する場合は、対応するキー値を設定します。

-   **trace_user_cookie_keys** <span class='type'>String</span>

    設定したクッキーの値を固有のユーザー情報として選択します。複数登録するときは、コンマ(,)を区切り文字として利用してください。設定した順に検索します。
