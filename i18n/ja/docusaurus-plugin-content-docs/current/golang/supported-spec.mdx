## Goエージェントの動作環境

### サポートバージョン

| 言語 | サポートバージョン |
| --- | --- |
| Go | 1.17以上 |

### オペレーティングシステム

| OS | 最小バージョン | アーキテクチャ |
| --- | --- | :---: |
| CentOS / Red Hat | 6.x | 64bit |
| Debian / Ubuntu | 12.04 | 64bit |
| Amazon Linux | - | 64bit |
| Alpine Linux | - | 64bit |
| Linux | - | ARM64 (aarch64) |

### ライブラリ

| カテゴリー | ライブラリ |
| --- | --- |
| **Webフレームワーク** | net/http, gin, chi, fiber, gorilla, echo, fasthttp |
| **データベース** | database/sql, gorm, redigo |
| **その他** | grpc, sarama, k8s.io/client-go |

:::note

フレームワークまたはライブラリの適用方法および例は、[GitHubサイト](https://github.com/whatap/go-api-example)を参照してください。

:::
