Goエージェントサービスを再起動または一時停止、削除する手順を提供します。

## エージェントサービスの再起動

WhaTapエージェント(whata-agent)サービスが実行中またはエラーが発生した場合は、サービスを再起動します。

<Tabs groupId="golang">
<TabItem value="rc" label="Red Hat/CentOS">

```bash title='SH'
$ service whatap-agent restart
```

</TabItem>
<TabItem value="du" label="Debian/Ubuntu">

```bash title='SH'
$ sudo service whatap-agent restart
```

</TabItem>
<TabItem value="al" label="Amazon Linux">

```bash title='SH'
$ sudo service whatap-agent restart
```

</TabItem>
<TabItem value="apl" label="Alpine Linux">

```bash title='SH'
$ /usr/whatap/agent/whatap-agent stop
$ /usr/whatap/agent/whatap-agent start
```

</TabItem>
</Tabs>

## WhaTapエージェントサービスを停止する

<Tabs groupId="golang">
<TabItem value="rc" label="Red Hat/CentOS">

```bash title='SH'
$ sudo service whatap-agent stop
```

</TabItem>
<TabItem value="du" label="Debian/Ubuntu">

```bash title='SH'
$ sudo service whatap-agent stop
```

</TabItem>
<TabItem value="al" label="Amazon Linux">

```bash title='SH'
$ sudo service whatap-agent stop
```

</TabItem>
<TabItem value="apl" label="Alpine Linux">

```bash title='SH'
$ /usr/whatap/agent/whatap-agent stop
```

</TabItem>
</Tabs>

## エージェント削除

GoアプリケーションのソースコードからWhaTapライブラリを削除します。

<Tabs groupId="golang">
<TabItem value="rc" label="Red Hat/CentOS">

```bash title='SH'
$ sudo yum remove whatap-agent
```

</TabItem>
<TabItem value="du" label="Debian/Ubuntu">

```bash title='SH'
$ sudo apt-get purge whatap-agent
```

</TabItem>
<TabItem value="al" label="Amazon Linux">

```bash title='SH'
$ sudo yum remove whatap-agent
```

</TabItem>
<TabItem value="apl" label="Alpine Linux">

```bash title='SH'
$ /usr/whatap/agent/whatap-agent stop

# /usr/whatap/agentディレクトリ削除
$ sudo rm -r /usr/whatap/agent
```

</TabItem>
</Tabs>
