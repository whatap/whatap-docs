## エージェント削除

アプリケーションサーバーでJVMオプションの`-javaagent`設定を削除してからサーバーを再起動してください。次に、`$WHATAP_HOME`環境変数を削除します。

## エージェントのロールバック

1. [WhaTapモニタリングサービス](https://service.whatap.io)でロールバックするプロジェクトを選択します。

2. 左側のメニューから<Cmdname sid="side_management" className="uitext" /> &gt; <Cmdname sid="side_agentUpdate" className="uitext" />を選択します。

3. <Cmdname sid="other_versions" className="uitext" />ボタンを選択します。

4. ロールバックするバージョンを選択し<Cmdname sid="selected_version" className="uitext" />または<Cmdname sid="selected*VERSION*for_all" className="uitext" />ボタンをクリックします。

   <ImgLang img='agent_rollback_2.png' desc='管理 - エージェントのロールバック' />

5. エージェントをインストールする時に修正したスクリプトから`WHATAP_JAR`環境変数をロールバックするバージョンに変更します。

   ![管理 - エージェントロールバックJVM修正](/img/agent_rollback_3.png)
