Web Application Server（WAS）、バッチアプリケーションなど、JVM環境で動作するすべてのアプリケーションに適用できます。アプリケーションの障害分析のためにThread Dump（スレッドダンプ）を取得する手間をかけることなく、既に取得されたスタックを利用して、障害の再現なしに正確な原因を分析できます。
