ユーザーのアプリケーションでフレームワークやオープンソースを使用する場合、Javaエージェントを通じて追跡できる設定を提供します。 Javaエージェント設定ファイル(_whatap.conf_)に`weaving`オプションを追加して使用できます。

```ini title='${WHATAP_HOME}/whatap.conf'
weaving=  
```

```ini title='whatap.conf'
weaving=spring-boot-3.0,feign-11,okhttp3-4.4
```
