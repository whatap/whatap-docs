### エージェントのインストールファイルを構成{#agent-files}

エージェントファイルは、アプリケーションモニタリングに必要な情報を抽出し、WhaTap収集サーバーに送信するトレーサーとトレーサーがデータを送信するのに役立つ要素で構成されています。

<Filetree name="whatap">

<FiletreeItem name="whatap.conf" type="file">モニタリングエージェントの設定情報、アクセスキー、WhaTapデータ収集サーバーのIP情報などを保存する設定ファイルです。詳細については、[次の文書](set-agent)を参照してください。</FiletreeItem>

<FiletreeItem name="ping.sh(bat)" type="file">設定されたアクセスキーとWhaTap収集サーバーで通信を行い、設定およびネットワーク状況に問題がないか確認するためのスクリプト(バッチ)ファイルです。</FiletreeItem>

<FiletreeItem name="proxy.sh(bat)" type="file">*whatap.agent.proxy-`X.Y.Z`.jar*ファイルを開始するためのスクリプト(バッチ) ファイルです。</FiletreeItem>

<FiletreeItem name="resmon.sh(bat)" type="file">CPU、Memory、Disk情報が正しく収集されるかを確認するためのスクリプト(バッチ) ファイルです。</FiletreeItem>

<FiletreeItem name="whatap.agent-X.Y.Z.jar" type="file">アプリケーションサーバーの各種モニタリング情報を収集して収集サーバーに送信するモニタリングエージェントのファイルです。</FiletreeItem>

<FiletreeItem name="whatap.agent.proxy-X.Y.Z.jar" type="file">アプリケーションサーバーが稼働しているサーバーからWhaTap収集サーバーに直接接続ができない場合、その間のProxyサーバー(経由サーバー)で経由する役割を行うアプリケーションです。</FiletreeItem>

</Filetree>
