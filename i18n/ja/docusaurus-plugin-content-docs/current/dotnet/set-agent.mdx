WhaTapエージェントは、エージェントごとに必要な設定を*whatap.conf*ファイルに書き込みます。エージェントは、環境変数を使用して設定ファイルを見つけて読み込みます。ユーザーの利便性を高めるため、[WhaTapモニタリングサービス](https://service.whatap.io)は、エージェント設定機能を提供します。
