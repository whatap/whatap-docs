WhaTapモニタリングサービスを使用するには、[会員登録](https://www.whatap.io/ko/signup/)後、プロジェクトを作成し、ターゲットサーバーにエージェントをインストールする必要があります。

## エージェントダウンロード

**アクセスキー**を発行すると、**_エージェントファイルのダウンロード_**セクションでダウンロードボタンが有効になります。**_EXEダウンロード_**または**_ZIPダウンロード_**ボタンを選択してエージェントのインストールファイルをダウンロードしてください。

:::note

-   セキュリティ設定により.exe形式のファイルのダウンロードができないユーザーのために、.zip形式のファイルも提供できるようにしました。

-   サーバーのセキュリティのために、ブラウザーから直接インストールするのではなく、ダウンロードしたインストールファイルで実行することをお勧めします。

:::

## エージェントインストール{#install-agent}

1.  ダウンロードしたインストールファイルをダブルクリックして実行してください。

2.  画面の案内に従って、**_License Key_**に**アクセスキー**を入力し、**_WhaTap IP_**項目には、**WhaTapサーバIP**を入力してください。

    WhaTapモニタリングサービスのプロジェクト作成画面で、**_アクセスキー_**と**_WhaTapサーバIP_**の値をコピーすることができます。

    ![.NETインストール - 01](https://img.whatap.io/media/agent_dotnet/install/agent_install_02.png)

3.  **_Next_** ボタンをクリックします。

4.  次の画面で **_Install_** ボタンをクリックします。 インストールを続行します。

5.  **_Finish_** ボタンをクリックして、インストールを終了してください。

<details>
<summary>.NETエージェントインストール後の構成ファイル</summary>

モニタリング情報を収集するためのトレーサーと収集した情報をサーバーに送信するためのエージェントで構成されています。

-   _WhaTap.ClrProfiler.dll_：.NET CLR Profilerは、.NETトレーシングAPIに関連付けられています。

-   _WhaTap.Trace.dll_：プロファイラーから渡されたデータを処理し、エージェントに送信します。

-   _whatap_dotnet.exe_：エージェントおよび、トレーサーから渡された情報を収集サーバーに送信するプログラムです。

-   _whatap.conf_：エージェントの環境設定ファイルです。

</details>

## モニタリングの開始

アプリケーションサーバーを再起動すると、エージェントが情報収集を開始します。

エージェント更新の詳細については、[次の文書](agent-manage)を参照してください。

:::
