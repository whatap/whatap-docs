## エージェントダウンロード

**アクセスキー**を発行すると、**_エージェントファイルのダウンロード_**セクションでダウンロードボタンが有効になります。**_EXEダウンロード_**または**_ZIPダウンロード_**ボタンを選択してエージェントのインストールファイルをダウンロードしてください。

:::note

- セキュリティ設定により.exe形式のファイルをダウンロードできないユーザーのために.zip形式のファイルも一緒に提供します。

- サーバーのセキュリティのために、ブラウザから直接インストールするのではなく、ダウンロードしたインストールファイルで実行することをお勧めします。

- **.NET 4.6.1**バージョン未満または**Windows 2008**環境に対応する別のエージェント インストール ファイルを提供します。ユーザーの環境に合わせたエージェントインストールファイルをダウンロードしてください。 

:::

## エージェントインストール{#install-agent}

1. ダウンロードしたインストールファイルをダブルクリックして実行してください。

2. 画面の案内に従って、**_License Key_**に**アクセスキー**を入力し、**_WhaTap IP_**項目には、**WhaTapサーバIP**を入力します。

   WhaTapモニタリングサービスのプロジェクト作成画面で、**_アクセスキー_**と**_WhaTapサーバIP_**の値をコピーすることができます。

   ![.NETインストール - 01](https://img.whatap.io/media/agent_dotnet/install/agent_install_02.png)

3. **_Next_** ボタンをクリックします。

4. 次の画面で **_Install_** ボタンをクリックします。 インストールを続行します。

5. **_Finish_** ボタンをクリックして、インストールを終了してください。

エージェントは、モニタリング情報を収集するためのトレーサー、収集した情報をサーバーに転送するためのファイルで構成されています。エージェントファイルの詳細については、次を参照してください。

## モニタリングの開始

アプリケーションサーバーを再起動すると、エージェントが情報収集を開始します。

エージェント更新の詳細については、[次の文書](agent-manage)を参照してください。

:::
