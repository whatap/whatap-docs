## エージェント削除

**スタートメニューから削除**

1. ウィンドウズで**_スタート_** ![Windows Start](/img/btn-windows.svg) > **_すべてのアプリ_**を選択し、表示された一覧から**WhaTap .NET**アプリを検索します。

2. アプリを長押しまたは、マウスの右ボタンをクリックし、**_削除_**を選択してください。

**コントロールパネルから削除**

1. ウィンドウズのタスクバーから検索する場合は、**コントロールパネル**と入力し、結果から選択してください。

2. **_プログラム_** > **_プログラムと機能_**を選択してください。

3. **_WhaTap .NET_**プログラムを選択し、右クリックして**_削除_**または**_削除・変更_**を選択してください。

画面の指示に従ってエージェントを削除してください。

## エージェントのアップデート

エージェントをアップデートする場合は、既存のバージョンを削除します。新しいバージョンのエージェントをダウンロードして、インストールプロセスを再度実行してください。

:::note

.NETエージェントのインストールの詳細については、[次の文書](install-agent#install-agent)を参照してください。

:::
