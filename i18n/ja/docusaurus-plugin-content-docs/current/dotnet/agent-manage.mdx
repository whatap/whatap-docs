## エージェント削除 {#uninstallagent}

**スタートメニューから削除**

1. ウィンドウズで**_スタート_** ![Windows Start](/img/btn-windows.svg) > **_すべてのアプリ_**を選択し、表示された一覧から**WhaTap .NET**アプリを検索します。

2. アプリを長押しまたは、マウスの右ボタンをクリックし、**_削除_**を選択します。

**コントロールパネルから削除**

1. ウィンドウズのタスクバーから検索する場合は、**コントロールパネル**と入力し、結果から選択します。

2. **_プログラム_** > **_プログラムと機能_**を選択します。

3. **_WhaTap .NET_**プログラムを選択し、右クリックして**_削除_**または**_削除・変更_**を選択します。

画面の指示に従ってエージェントを削除してください。

## エージェントアップデート {#updateagent}

エージェントをアップデートするには、既存バージョンを削除する必要はなく、新しいバージョンをダウンロードしてインストール手順を再実行します。

:::note

.NETエージェントのインストールの詳細については、[次の文書](install-agent#install-agent)を参照してください。

:::
