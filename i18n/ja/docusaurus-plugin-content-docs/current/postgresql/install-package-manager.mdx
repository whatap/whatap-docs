パッケージマネージャーを利用したWhaTapデータベースモニタリングエージェントのインストール方法をご案内します。

:::tip

**インストール方法の選択**

* `apt`パッケージマネージャーを使用するDebian/Ubuntu系システムの場合は、このドキュメントを参照してください。
* Docker環境では、[Dockerインストール](install-docker)を参照してください。
* その他の環境では、[手動インストール](install-agent)を参照してください。

:::

:::note

**サポートされているデータベース**

MySQL、PostgreSQL、SQL Server
