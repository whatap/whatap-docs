ホーム画面 > **_統合メトリクスボード_**

ユーザーが属するすべてのプロジェクトのメトリクスデータを照会できるダッシュボードが作成できます。さまざまな製品群のメトリクスをダッシュボードの上にウィジェットとして構成し、一目で把握することができます。

:::note
**_통합 메트릭스 보드_** 메뉴를 처음 이용한다면 **_Flex 보드_** 메뉴를 사용할 것을 권장합니다. **_Flex 보드_**에 대한 자세한 내용은 각 상품별 문서를 참조하세요.

{@include: ../_link2flexboard.mdx}\
:::

## ウィジェットチャートを追加します。

1.  **_統合メトリクスボード_**画面上段の**_+ウィジェットの追加_**を選択します。 

2.  ダッシュボードに**_Dashboard_**ウィジェットが作成されます。ウィジェット右上の![編集アイコン](/img/ico-edit.svg)ボタンを選択します。

3.  **_ダッシュボードを作成する_**ウィンドウが表示されます。

### Seriesチャートの形を設定する

**_チャートタイプ_**で**_Series_**を選択します。**_ Series_**チャートでは、時間の経過に伴うメトリックスデータの流れを確認することができます。

![Seriesチャート](/img/int-metrics-dashboard-add.png)

-   **_プレビュー_**：設定したオプションによってチャートの形状が変わることが確認できます。

-   **_Title_**：ウィジェットチャートの名前を入力します。

-   **_X Axis_**
    -   **_Axis Line_**：X軸線を表示たは非表示にできます。
    -   **_Plot Line_**： 時間の経過とともに垂直線を表示または非表示にすることができます。
    -   **_Tick_**：X軸テキストの表示または非表示にすることができます。

-   **_Y Axis_**
    -   **_Axis Line_**：Y軸線の表示または非表示にすることができます。
    -   **_Plot Line_**：データ別の水平線の表示または非表示にすることができます。
    -   **_Tick_**：Y軸テキストの表示または非表示にすることができます。
    -   **_Inner Label_**：Y軸テキストをチャートの内側に移動します。
    -   **_uni_**：Y軸データの単位を設定することができます。

-   **_Style_**
    -   **_Area_**：グラフ領域の内側を色付けして表示します。
    -   **_Plot Verical Line_**
    -   **_Show Plot Max Y Value_**：チャートの最も高い数値をテキストとして表示します。

-   **_Hover tooltip_**
    -   **_Tick_**

### Staticチャートの形を設定する

**_チャートタイプ_**で**_Static_**を選択します。**_Static_**チャートは、メトリクスデータをテキストとして表現します。リアルタイムのデータ値をすぐに確認するときに容易です。

![Staticsチャート](/img/int-metric-dashboard-static.png)

-   **_プレビュー_**：設定したオプションによってチャートの形状が変わることが確認できます。

-   **_Title Style_**
    -   **_Title_**：メトリクスチャートウィジェットのタイトルを入力します。
    -   **_description_**：メトリクスチャートのウィジェットに関する説明を入力します。
    -   **_Title Font Color_**：タイトルの色を設定します。

-   **_Content Label_**
    -   **_Content Font Size_**：チャートに表示するテキストのサイズを設定します。
    -   **_Content Background Color_**：チャートに表示するテキストのサイズを設定します。
    -   **_Content Front Color_**：チャートに表示するテキストの色を設定します。
    -   **_Vertical Align_**：テキストの垂直方向の配置を設定します。
    -   **_Horizontal Align_**：テキストの水平方向の配置を設定します。

-   **_Formats_**
    -   **_format_**：メトリクスデータの単位を設定します。
    -   **_Separator_**：1000単位は、コンマ(,)で区切ることができます。
    -   **_Prefix_**：チャートに表示するテキストにプレフィックスを付けることができます。
    -   **_Suffix_**：チャートに表示するテキストの後にサフィックスを追加することができます。

### Color alertチャートの形を設定する

**_チャートタイプ_**で**_Color alert_**を選択します。**_Color alert_**チャートを使用すると、メトリクスデータの遅延間隔を簡単に識別できます。間隔は３つの段階で設定でき、減算または加算することができます。

-   **_Title_**：チャートのタイトルを入力してください。

-   **_Values_**

    -   **_Name_**：各セグメントの名前を入力して、現在のステータスを定義できます。
    -   **_Min_**：遅延時間を分単位で設定できます。
    -   **_Color_**：遅延区間ごとに色を設定して、ステータスを直観的に理解することができます。

    :::note
    不要なステップを削除するには、![削除アイコン](/img/ico-close.svg)ボタンを選択します。ステップを追加するには、**_+ Add_**ボタンを選択します。
    :::

### 基準時刻の設定

**_データ設定_**：設定した時間に基づいてチャートにデータを表示します。

-   **_リアルタイム_**：過去5分、10分、30分、1時間、3時間、6時間、12時間を基準にリアルタイムデータを確認できます。
-   **_過去_**：今日、昨日、2日前、3日前、1週間前を基準に直近5分のデータが確認できます。今日以前を選択すると、現在の時間を基準に5分間のデータが確認できます。
-   **_カスタム_**：日付と時刻を選択して時間を設定できます。

### タグデータの追加

**_タグデータの追加_**ボタンを選択して、メトリクス データを読み込むプロジェクトを選択してください。

![タグデータの追加](/img/int-metric-dashboard-add-data.png)

-   ![編集アイコン](/img/ico-edit.svg)ボタンを選択して、タグデータの名前を変更できます。

-   **_時間_**：**_データ設定_**で設定した時間を基準にデータを読み込んでチャートを描画したり、前回の時間と比較してチャートを描写できるように設定することができます。

-   **_プロジェクト_**：製品別にプロジェクトを選択できます。プロジェクトを選択すると、**_カテゴリ_**項目が表示されます。

-   **_カテゴリ_**：選択したプロジェクトからメトリクスをインポートします。カテゴリを選択すると、追加オプションが以下に表示されます。

![タグデータの追加](/img/int-metric-dashboard-add-data-tag.png)

-   **_タグ_**：収集対象を選択できます。エージェント名(**_Oname_**)、グループ(**_OkindName_**)、ホストIP(**_Hostip_**)、形態(**_Type_**)別に選択できます。

-   **_フィールド_**：チャートに表示するメトリクスを選択してください。

-   **_時間マージ_**：ソースデータ内のフィールド値の同一データに対し、時間範囲に応じて一定の時間間隔（5分または1時間）でデータをマージします。

-   **_オブジェクトマージ_**：タグ項目で選択した値に基づいてデータをグループ化するときに使用します。異なるフィールド値を持つデータの中でタグが一致する場合、そのデータがマージされます。

すべての項目に対する設定が完了したら、**_完了_**ボタンを選択します。

:::note

-   選択したプロジェクトによって**_カテゴリ_**で選択できる項目が異なる場合があります。
-   メトリクスの詳細については、[次の文書](../metrics/common-metrics-intro)を参照してください。
-   **_時間マージ_**および**_オブジェクトマージ_**の詳細は[次の文書](../java/flexboard-widget-manage#datamerge-option)を参照してください。
:::

## ウィジェットチャートを確認する

ウィジェットチャートを追加した後、リアルタイムでメトリクスの推移を確認するには、画面上部の**_ビューモード_**を選択してください。ウィジェットチャートを追加・修正・削除するには、**_修正モード_**を選択してください。

## ウィジェットチャートを修正する

ウィジェットチャートを修正するには、**_修正モード_**に切り替えてウィジェットチャート右上の! [編集アイコン](/img/ico-edit.svg)ボタンを選択してください。 **_ダッシュボードを作成する_**ウィンドウが表示されたら、項目を修正することができます。修正が完了したら、**_確認_**ボタンを選択してください。

## ウィジェットチャートをコピーする

同じタイプのウィジェットチャートでメトリクス値のみを変更して使用する場合、ウィジェットチャートをコピーして使用します。**_修正モード_**に切り替えてウィジェットチャート右上の![コピーアイコン](/img/ico-copy.svg)ボタンを選択してください。 メトリクスを変更するには、ウィジェットチャート右上の![編集アイコン](/img/ico-edit.svg)ボタンを選択し、項目を修正してください。

## ウィジェットチャートを削除する

ウィジェットチャートを削除するには、**_修正モード_**に切り替えてウィジェットチャート右上の![削除アイコン](/img/ico-close.svg)ボタンを選択してください。ポップアップウィンドウが表示されたら、**_確認_**ボタンを選択してください。
