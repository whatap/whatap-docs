ホーム画面 > **_サイトマップ_** > **_統合ダッシュボード_**

ユーザーが属するすべてのプロジェクトのデータを表示できるダッシュボードを作成できます。異なる製品群のステータスをダッシュボードの上にウィジェットとして構成し、一目で把握することができます。

:::note

**_統合ダッシュボード_**メニューを初めて利用する場合、**_Flexボード_**メニューの使用をお勧めします。**_Flexボード_**に関する詳細については、各サービス別の文書を参照してください。

{@include: ../_link2flexboard.mdx}  

:::

## 統合ダッシュボードを作成する

### テンプレートを使用してダッシュボードを作成する

WhaTapが提供するテンプレートを使用してダッシュボードを簡単かつ迅速に作成できます。

1.  画面の上の**_ダッシュボードを作成する_**を選択してください。

2.  **_ダッシュボードを作成する_**メニューの下にテンプレート一覧が表示されます。

    ![ダッシュボードテンプレート選択](/img/integrated-dashboard-sel-template-1.png)

3.  テンプレートを選択すると、**_ダッシュボードテンプレート_**ウィンドウが表示されます。

    ![ダッシュボードテンプレート](/img/main-int-dashboard-sel-template.png)

4.  ダッシュボード名と説明を入力し、データを読み込むプロジェクトを選択してください。選択できるプロジェクトの数はテンプレートによって異なる場合があります。

5.  **_確認_**ボタンを選択してください。

**_統合ダッシュボード_**一覧で作成したダッシュボードを確認できます。一覧で新しく作成したダッシュボードを選択すると、ダッシュボード画面に移動します。

### 空のテンプレートでダッシュボードを作成する

1.  画面の上の**_ダッシュボードを作成する_**を選択してください。

2.  **_ダッシュボードを作成する_**メニューの下にテンプレート一覧が表示されます。

3.  左端の**_空のダッシュボード作成_**を選択してください。

    ![空のダッシュボード](/img/integrated-dashboard-sel-template.png)

4.  **_ダッシュボードを作成する_**ウィンドウが表示すると、名前と説明を入力して**_保存_**ボタンを選択してください。

5.  **_統合ダッシュボード_**一覧で作成したダッシュボードを確認できます。一覧で新しく作成したダッシュボードを選択すると、ダッシュボード画面に移動します。

6.  画面上部の**_+ウィジェットの追加_**ボタンを選択して、ウィジェットを追加してください。

## ウィジェットを追加する

ダッシュボード画面にウィジェットを追加して、さまざまなプロジェクトの指標を確認できます。例えば、ユーザーがAプロジェクトのヒートマップチャートウィジェットとBプロジェクトのヒートマップチャートウィジェットをダッシュボードに追加して同時に確認することができます。

1.  **_統合ダッシュボード_**一覧からウィジェットを追加するダッシュボードを選択し、ダッシュボード画面に移動してください。

2.  画面の上の**_+ウィジェットの追加_**ボタンを選択してください。

3.  ウィジェット作成ウィンドウが表示されると、目的のウィジェット名とウィジェットチャートの種類、プロジェクトの順に選択してください。

    ![ウィジェットの作成](/img/int-dashboard-add-widget.png)

    :::note

    -   選択したウィジェットチャートの種類によって、選択できる製品カテゴリおよびプロジェクトが異なる場合があります。

    -   製品カテゴリでは、複数のプロジェクトを選択できます。他の製品カテゴリと組み合わせることはできません。

    -   **_Custom chart_**は、**_メトリクス_**および**_オブジェクトマージ_**、**_期間フィルター_**などのオプションを設定できます。

    :::

4.  **_次へ_**ボタンを選択してください。

5.  各プロジェクトに属するエージェントは、エージェントまたはグループID、Node ID を選択できます。プロジェクト項目の一番左の▶ボタンを選択してください。

    ![データタイプ](/img/int-dashboard-widget-data.png)

    -   ウィジェットで指標を表示するエージェントをクリックし、**_適用_**ボタンを選択してください。

    -   すべてのエージェントの指標を表示するには、**_全体_**をクリックし、**_適用_**ボタンを選択してください。

6.  **_完了_**ボタンを選択してください。

**_統合ダッシュボード_**画面で追加したウィジェットを確認してください。

:::note

-   ウィジェットの右下部分をカーソルに合わせてドラッグすると、サイズを調整できます。

-   ウィジェットのタイトル部分にカーソルを合わせてドラッグすると、位置を移動できます。

-   ウィジェットに関する情報を確認するには、ウィジェットの右上の![情報アイコン](/img/ico-widget-info.svg)ボタンをクリックしてダブルクリック
    してください。

:::

## ウィジェットを修正する

1.  ウィジェットの右上の![編集アイコン](/img/ico-edit.svg)ボタンを選択してください。

2.  **_ウィジェットの作成_**ウィンドウが表示されたら、ウィジェットチャートの種類とプロジェクトを選択して変更できます。

3.  **_次へ_**ボタンを選択してください。

4.  プロジェクトを使用して、別のエージェントを選択するには、プロジェクト項目の一番左の▶ボタンを選択してエージェントを選択してください。

5.  **_完了_**ボタンを選択してください。

**_統合ダッシュボード_**画面で修正したウィジェットを確認してください。

## ウィジェットを削除する

**_統合ダッシュボード_**画面から削除するウィジェットの右上の![削除アイコン](/img/ico-close.svg)ボタンを選択してください。メッセージが表示されたら、**_確認_**ボタンを選択してください。

## 統合ダッシュボード名と説明を変更する

1.  **_統合ダッシュボード_**一覧から名前または説明を修正するダッシュボード項目の![編集アイコン](/img/ico-edit.svg)ボタンを選択してください。 

2.  **_ダッシュボード修正_**ウィンドウが表示すると、名前と説明を修正してください。 

3.  修正を完了して**_保存_**ボタンを選択してください。

**_統合ダッシュボード_**一覧で修正した内容を確認できます。

## 統合ダッシュボードを共有する

**_統合ダッシュボード_**で作成したダッシュボードを他のアカウントのユーザーと共有して再利用できます。

### コピーする

ユーザーが作成したダッシュボードをコピーしたり、他のユーザーと共有したりできます。

**_統合ダッシュボード_**一覧でコピーしようとするダッシュボード項目の![コピーアイコン](/img/ico-copy.svg)ボタンを選択してください。 **_自分に_**コピーまたは**_他のユーザーにコピー_**ボタンを選択してください。

-   **_自分にコピー_**を選択すると、ダッシュボード一覧に「(Copied)」接頭語が追加された項目を確認できます。

-   **_他のユーザーにコピー_**の場合、他のユーザーのメールアドレスを入力し、**_送信_**ボタンを選択してください。**_読み取り専用_**オプションを選択した場合、共有するユーザーはダッシュボードを修正できません。

    :::note

    ダッシュボードの共有後、編集したダッシュボードの変更は共有されません。

    :::

### 書き出し、読み込み

JSONファイルとして保存した統合ダッシュボード設定を、他のユーザーアカウントから読み込みしてコピーします。ダッシュボード管理者権限の機能です。

1.  **_統合ダッシュボード_**一覧で他のユーザーと共有したいダッシュボード項目の![書き出しアイコン](/img/ico-export.svg)ボタンを選択してください。

2.  **_Export Dashboard_**ウィンドウが表示されたら、**_書き出し_**ボタンを選択してください。

3.  JSONファイルをダウンロードします。

4.  他のユーザーにダウンロードしたファイルを転送してください。

5.  **_統合ダッシュボード_**画面上の![読み込みアイコン](/img/ico-import.svg)**_読み込み_**ボタンを選択してください。

6.  **_Export Dashboard_**ウィンドウが表示されると、**_保存_**ボタンを選択してください。

ダッシュボード一覧に「(Imported)」接頭語が追加された項目を確認できます。

## 統合ダッシュボードを削除する

統合ダッシュボード一覧で削除しようとするダッシュボード項目の![削除アイコン](/img/ico-trash.svg)ボタンを選択してください。ポップアップメッセージで**_削除_**ボタンを選択してください。
