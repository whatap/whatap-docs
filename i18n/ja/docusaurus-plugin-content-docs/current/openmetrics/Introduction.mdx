> **統合モニタリングへの新しいアプローチ**

**WhaTap** <Cmdname sid="openmetrics" className="uitext" /> は、PrometheusベースのOpenMetrics指標を自動収集し、複雑な設定なしで簡単に統合モニタリング環境を構築できるソリューションです。  
新しいクラウドや古いサーバー環境に関係なく、さまざまな指標を収集し、ダッシュボードやチャートで一目で確認できます。

### 自動収集と簡単な設定

* [OpenMetricsエンドポイント](../openmetrics/install) を登録するだけで自動収集を開始  
* オンプレミス環境とSaaS環境の両方をサポート

### 直感的な可視化と分析

収集された指標はリアルタイムチャートで提供され、運用者は各インスタンスのHTTPリクエストスループットを即座に確認できます。グラフによりトラフィック急増のタイミングとサーバー別の負荷分散状況を直感的に把握できます。

<ImgLang img='openmetrics-intro.png' desc='OpenMetrics紹介' />

#### 主な観察ポイント

リクエストレートをエラー種別で可視化すると、特定のエラー（例: 404, 500）が集中的に発生した時点を容易に特定できます。これによりエラーパターン分析とトラフィック変動およびサーバー性能のリアルタイム監視が可能になり、迅速に対応できます。

### アラートと安定運用サポート

* 条件ベースのアラートポリシー  
* 閾値超過時にリアルタイムアラート送信  
* 障害の早期検知とサービス安定性向上

### 多様な活用シナリオ

* **インフラ運用**: CPU、メモリ、ネットワーク使用量の追跡と容量計画  
* **アプリケーション監視**: API処理量、エラー率、応答遅延の分析  
* **DB性能分析**: スロークエリ、コネクションプール、レプリケーション遅延、デッドロック検知

### 運用効率と拡張性

WhaTap OpenMetricsは単なる指標収集ツールではなく、運用効率を高める統合モニタリングプラットフォームです。  
グローバルSaaS環境で実証済みの拡張性により、数十万件以上のメトリクスも安定的に収集できます。

これにより障害対応時間を短縮し、サービス品質を向上させることができます。
