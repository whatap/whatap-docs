サーバーアプリケーションからログを収集するには、次の点を確認してください。

## エージェントのアップデート

サーバーエージェントの2.1.2バージョン以降から可能です。アップデート方法は[次の文書](../server/agent-update)を参照してください。

## エージェント設定の確認

ホーム画面 > プロジェクト選択 > ***ログ*** > ***ログ設定***

WhaTapモニタリングサービスのホーム画面からプロジェクトを選択し、プロジェクトメニューの***ログ*** > ***ログ設定***メニューを選択します。***エージェント設定の確認***タブの上にあるOS選択タブでサーバーOS(***Linux(shell)***、***Windows(Powersehll)***)を選択します。***基本インストール***または***カテゴリでのインストール***を参照して進めてください。 

:::note

次の設定は、エージェントを再起動する必要があります。
