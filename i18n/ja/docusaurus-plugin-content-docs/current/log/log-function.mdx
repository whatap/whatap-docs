## ログモニタリングを適用する

使用するアプリケーションによって適用方法を次のように提供します。ログモニタリングを適用する前に対応バージョン](#how-log-works)を確認してください。 

1. 対応するエージェントのバージョンを確認し、**アップデート**してください。

2. ログモニタリング**オプション**を設定してください。

3. ログモニタリングを**有効**にしてください。
