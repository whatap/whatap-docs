## ログモニタリングを適用する

使用するアプリケーションに応じた適用方法は以下のとおりです。ログモニタリングを適用する前に、[対応バージョン](#how-log-works)を必ずご確認ください。 

1. 対応するエージェントのバージョンを確認し、**アップデート**してください。

2. ログモニタリング**オプション**を設定してください。

3. ログモニタリングを**有効**にしてください。
