## Node.jsのサポート環境

エージェントをインストールする前に、Node.jsアプリケーションサーバーが以下の仕様を満たしているかを確認してください。

-   **対応バージョン**：7.10.0バージョン以降

-   **OS**: Windows、Mac (OSX)、Linux、Ubuntu、CentOS など、Node.Jsを実行できるすべての環境をサポートします。

-   **アプリケーションサーバー**：Node.js基本モジュールを使用するすべてのフレームワーク（Express、Hapi、Mean、Koa、Restifyなど）

-   **データベース**：MySQL、Microsoft SQL、MongoDB、Redisなど

:::note

WhaTapサーバーにデータを送信するには、6600番ポートが開いてある必要があります。

:::
