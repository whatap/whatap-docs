エージェントをインストールした後に確認する必要があります。

## アプリケーションサーバーの再起動

アプリケーションサーバーを再起動してください。動作ログに次のWhaTapログが表示されている場合は、すべてのインストールが完了しています。

```bash
$ node app.js
20210309 07:45:59 [WHATAP-001] <35632> Start initialize WhaTap Agent...
Example app listening at http://localhost:3000
_      ____       ______NODE-AGENT
| | /| / / /  ___ /_  __/__ ____
| |/ |/ / _ \/ _ `// / / _ `/ _ \
|__/|__/_//_/\_,_//_/  \_,_/ .__/
                        /_/
Just Tap, Always Monitoring
WhaTap Node Agent version 0.4.28, 20210309
```

エージェントが情報の収集を開始します。エージェントログは_logs/whatap-yyyymmdd.log_の形式で出力します。
