Node.jsアプリケーションサーバーの機能を制御できます。

import Nodejscontrolfunction from "../common-items/_nodejs-control-function.mdx";

<Nodejscontrolfunction />

- **ignore_nextjs_build_file_enabled** <span class='type'>Boolean</span> <code class="my">Node.js Agent v0.4.96 or later</code>

  既定値 `true`

  _Next.js_ビルドファイルに対するモニタリング可否を設定します。このオプション値を`true`に設定すると、_Next.js_ビルドファイルをモニタリングしません。

- **ignore_nextjs_build_file_path** <span class='type'>String</span> <code class="my">Node.js Agent v0.4.96 or later</code>

  既定値`/next/`

  モニタリングで除外する_Next.js_ビルドファイルの開始パスを指定できます。`ignore_nextjs_build_file_enabled`値が`true`の場合に適用されます。コンマ(,)を区切り文字として複数のパスを設定できます。

  ```ini title="whatap.conf"
  ignore_nextjs_build_file_path=/next/abc/,/next/def/
  ```
