whatap-telegrafサービス(Service)が実行されていない場合、または、エラーが発生した場合は、サービスを再起動します。

:::note

Telegrafプラグインおよびエージェントは、ユーザーが独自のサービスに直接インストールして設定する必要があります。

:::

- **Red Hat/CentOS**

  ```bash title=SH
  $ service whatap-telegraf restart
  ```

- **Debian/Ubuntu**

  ```bash title=SH
  $ sudo service whatap-telegraf restart
  ```

- **FreeBSD**

  ```bash title=SH
  $ service whatap_telegraf restart
  ```
