Telegrafは、データベース、システム、IoTセンサーからメトリクスとイベントを収集および送信するためのプラグインベースのサーバーエージェントです。TelegrafはGoで作成されており、外部の依存関係のない単一のバイナリとしてコンパイルされ、最小限のメモリリソースを利用します。

- **全種類のデータ収集・送信**

  - **データベース**

    MongoDB、MySQL、Redisなどのデータソースに接続し、メトリクスを収集および送信します。 

  - **システム**

    最新のクラウドプラットフォーム、コンテナ、オーケストレータースタックからメトリクスを収集します。 

  - **IoTセンサー**

    IoTセンサーとデバイスから重要なステータスの保存データ(圧力レベル、温度レベルなど)を収集します。

- **Telegrafプラグイン**

  Telegrafは、メトリクスを収集、処理、集計、および作成するプラグインベースのエージェントです。入力、出力、アグリゲーター、プロセッサーを含む4つのカテゴリのプラグインを提供します。

TelegrafのWhaTap output pluginを活用し、WhaTapのTelegrafエージェントをインストールして、Telegrafで収集するメトリクスをWhaTap収集サーバーに送信できます。
