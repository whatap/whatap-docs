イベント条件を設定し、メール、SMS、メッセンジャーなどのさまざまな経路で通知を受信することができます。

- **_イベント設定_**メニューにより、しきい値ベースの通知条件および受信設定を行うことができます。

- **_イベント受信設定_**メニューでは、個人通知の受信チャネルおよびタイムゾーンを設定できます。

- **_イベント履歴_**メニューでは、通知発生履歴を確認できます。
