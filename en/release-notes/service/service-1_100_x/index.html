Service 1.100.0​

April 26, 2023

Browser​

New Open the browser monitoring service. Thanks for your use. For more information about Browser Monitoring, see the following.

Common​

Search​

  • Change Query is performed by pressing Enter after entering the Sidebar > Search keyword.
  • Fixed Fixed the screen tearing symptom when Sidebar > Search > Agent Search has been performed.

Flex Board​

  • Feature In Flex Board, added the data merge summation function of the secondary chart as the chart setting option for series widgets.
  • Fixed Modified to display the date on the X-axis when searching the data for more than 3 days through the Series widget.

Analytics​

Fixed In Metrics Chart, fixed the CSV download bug for the overlay chart.

STAT/REPORT​

  • Feature In Integrated Report, added the function to select the next day as the end time upon creating the Daily HTTP Call Delay Report.
  • Fixed In STAT/REPORT > REPORT, fixed the bug where the Filter option fetches the list of agents only based on the current time rather than the start date selected by the user.

Log​

  • Change In Log Setting, modified the agent support versions for each product.
  • Fixed In Live Tail, modified to display the loading animation upon initial data loading.
  • Change In Log Setting, changed to represent the actual trial days remaining.
  • Feature In Log Search, modified to display the number of logs matched among all searches, if there are two or less search keys.
  • Feature In Log Trend, modified to display the searched logs.
  • Feature In Log Setting, added the Go agent configuration guide.

Alert​

  • Change In Notification Setting, changed the name of the Escalation option to Repeat Notification.
  • Feature In Notification Setting, added the Repeat Notification option in the Event Notification Settings Per-User and 3rd Party Plugin sections.
  • Fixed In Event Setting > Metrics, fixed the problem where a whiteout occurs when a project is initially created.
  • Fixed Fixed the validation logic bug in Composite Metrics Event.
  • Fixed In Event Setting > Metrics, fixed the problem where a text rolling occurs when reducing the browser screen size.

My Usage​

  • Change In Preview Invoice, added the Discount Amount column in the billing history table.
  • Change In Billing and payment history, deleted the product type column from the list in Display Billing History.

Management​

  • Fixed Through Manage Account INFO. Withdrawal, modified to check the membership withdrawal information (unpaid amount, granted roles).
  • Fixed Fixed the bug where the loading message is not displayed when clicking the button on the project management page.
  • Change If the project is in a restricted state, modified for a member of the project to access Management > Project Member.
  • Fixed In Select Account Icon > Manage Account INFO. > My Profile on the upper right of the screen, modified to display the unpaid amount and management support team's phone number during membership withdrawal.

MXQL​

Fixed Fixed the problem where CSV is not downloaded after MXQL data query.

Application​

  • Change Improved the design of the installation guide section in Management > Agent Installation.
  • Fixed In Dashboard > Application Dashboard, fixed the bug where the metric is hidden because the text in the summary widget does not wrap lines when the language is set to Japanese.
  • Feature In Analysis > HITMAP, added the following information on the Statistics pop-up window displayed by clicking the transaction list.

    • In case of traces of HttpCall type, if there is driver information collected, driver info appears instead of Http Call.

    • Additional SID values appear in the detailed information window that appears by clicking traces of SQL or DBC type.

  • Change In Analysis > Hitmap > Tx Trace List, added the SQL query copy button at the bottom when searching SQL steps in the Statistics window by selecting a transaction.
  • Fixed In STAT/REPORT > Statistics, fixed the bug where some caller agent types and caller URL columns are missing when searching for Caller.
  • Fixed In Analysis > Multiple Transaction Trace, fixed the bug where not entering the MTID / CUSTID value does not search data.
  • Change In Analysis > Multiple Transaction Trace, added the validation logic for the MTID / CUSTID search value.

Server​

  • Fixed In Server List > Server Detail, fixed a missing Grouping option role checking logic for process widgets.
  • Fixed In Dashboard > Compound Eye, fixed the bug where entering a number of 24 digits or more in Refresh Interval (sec) does not allow to draw Topology View.

Container​

  • Feature Dashboard > Container Map > button > Add Show Container created within 1 minute option: The containers that have been created within 1 minute are displayed in bright colors.

CloudWatch, NCP, OCI, Azure​

  • Fixed In Management > Agent Installation, added the descriptions and modified UIs.

    • Added a description at the bottom of the loading image displayed when entering the agent installation page immediately after a project creation.

    • Added a description at the bottom of the loading image displayed when searching and saving the service list.

    • Modified the height of the description so that the CloudWatch installation guide page appears immediately without being covered by the scroll.

OCI, Cloud Monitoring​

  • Fixed In Management > Agent Installation, disabled the RSA Public Key generation button when you do not have the role to modify the OCI project.
  • Fixed In Management > Cloud Monitoring > Oracle Cloud Monitor tab, fixed the bug where once the RSA Public Key has been generated, it is classified as registered even if authentication has not been made.

ECS​

Fixed In My Usage > Project Subscription, fixed the bug where a whiteout occurs.
