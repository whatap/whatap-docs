Service 1.102.00​

May 24, 2023

Common​

  • Fixed Among sidebar menus, modified to add Browser types in project type filters and display the text according to the selected language.
  • Change In Sitemap > User Behavior, modified to search the user behaviors for all accounts with the member management role.
  • Change Modified the Delete icon () and the Channel Inquiry icon () in all services.

Flex Board​

  • Change Modified the widget's interaction button state not to be changed regardless of the side menu On/Off under Dashboard.
  • Fixed In case of the rich text widgets, fixed the bug where the saved content are not applied after modification.
  • Fixed If the browser window has a specific size, when the JSON editing window of the widget is imported in Manager mode under Flex Board, a scroll appears and disappears repeatedly in the text box.
  • Change Modified to display the specific reason when there is no data response in the following widgets.
    • Log 4XX, 5XX Count
    • Request Success Rate
    • Response Time by Log
  • Change Updated the MXQL path versions of the widgets used in the Integrated Service Dashboard template.
  • Fixed Fixed the widget MXQL in the Server Resource Plus dashboard.
  • Change Added and reorganized the Log RPS Trend widget metric for the Log RPS Board dashboard template.

Management​

Fixed In Project Member, fixed the issue where the Edit Log role does not appear in the Role column.

My Usage​

  • Fixed In Preview Invoice, fixed the bug where the discount amount of a cloud product does not appear in the Discount column.
  • Fixed In Preview Invoice, added the guidance message for billing currency when the fixed price and discount amount appear in Discount Information.
  • Change In Usage, applied UI improvements for better user experiences.
  • Feature In Usage, added the function to search the project name.
  • Change In Usage, modified to provide the average usages for each time.

Log​

  • Feature The Edit Log role has been added and the related functions has been applied to Log Setting.
  • Change In Log Setting, improved UI/UX for better user experiences.
  • Change In Log Setting, unified the mixed terms.
  • Change In Log Search, modified to enter the value, `none` as the key when adding a category while only a key has been entered in the filter with no value assigned.
  • Change In Live Tail, fixed a typo error (for the term, `spacing`) among the terms in the content filter tooltips.
  • Change In Log Search, modified not to automatically attach the prefix, `+` when `-` has been entered as the search value's prefix upon entry of a filter.

Database​

Dashboard​

  • Change In Multiple Instance Monitoring, changed the selection option from Database to Instance.
  • Fixed In Instance List, added the function to align the table and adjust the column width to compensate for text truncation in the Event window that appears by selecting the Fatal or Warning icon in the Status column.

STAT/REPORT​

  • Change In Table Size Increase, changed the Download CSV button to .
  • Fixed In the follow menu path, fixed the problem where the download feature does not work depending on the region clicked on the drop-down menu item when the button is selected.
    • STAT/REPORT > Table Size Increase
    • STAT/REPORT > SQL Statistics > DB, USER, HOST tab

Container​

Container Map​

  • Feature The metrics whose unit is percentage among the chart tooltips that appear when user hovers the mouse over a container element, are displayed with the bar graph charts.
  • Change Added the selected metrics item in the > Label option.
  • Fixed Fixed the symptom where part of the Log Search and Event tabs in the summary view panel.
  • Change Modified to search logs based on the microOid when using the Log > Log Search function on the summary view panel in case of the app-related category.
  • Fixed When using Log > Log Search after selecting a group, fixed the bug where the app-related category logs are not searched.

Browser​

  • Fixed In Dashboard > Browser Error, fixed the flickering symptom while loading charts in the error message table.
  • Feature In Alert > Event Setting > Metrics, added the templates for metrics and composite metrics alert.

CloudWatch, AZURE, OCI, NCP​

Common​

Feature In Management > Agent CONFIG., added 1 minute as a collection cycle option.

CloudWatch​

Fixed In Management > Agent CONFIG., modified the default value for validity period to 1 Hour.
