Preview 1.109.X

In https://preview.whatap.io, the following new features and enhancements can be experienced in advance.

Preview 1.109.22​

September 20, 2023

Common​

Feature Added an icon menu to the left sidebar to access the product introduction page.

Application​

  • Feature The Application dashboard menu has been reorganized as follows:

    • Added the time selector option and improved the function to view the past data.

      Real-time mode
      Non-real-time mode

    • Any widget can be added to the desired position or deleted. The widget can be adjusted to the desired size.

    • The preset function has been added that allows saving and loading the positions, sizes, and layouts of the edited widgets.

  • In the Trace analysis window on the dashboard, the following functions have been added and modified.

    • Feature Added the project filtering function in table view and tree view modes under the Multi-transaction tab.
    • Fixed In chart view mode under the Multi-transaction tab, fixed the bug where the mode is switched to the table view if the transaction node is clicked.
    • Feature Added the function to display appropriate messages if there is no data in the Transaction log tab.
      • In case there is no actual log data
      • In case logs cannot be collected without agent configuration
      • In case logs are disabled

Container​

Feature In Management > Agent installation, added the function to display the confirmation message whether the Kubernetes agent has been installed successfully.

Browser​

Fixed Fixed the issue where columns cannot be set in full screen mode of the resource table widget.

Preview 1.109.15​

September 8, 2023

Common​

  • Feature Added the function to display the default interval value, if you select Default among the Interval selection options.
    BeforeAfter
  • Changed Modified to select Default among the Interval selection options, when any time search section is changed.

    When changing the search section, the Interval option is selected as Default in the following cases:

    • The previous selection in the Interval option is Default.

    • The previous selection in the Interval option is not Default and it is not in the changed Interval option list.

    • The previous selection in the Interval option is not Default and the selected Interval option value is less than half of the search period.

  • Fixed Fixed the error where, in some situations, pressing OK in the Column settings pop-up displays an unknown error message and it does not work.
  • Fixed In Manage account INFO., fixed the error where the country code number is defaulted to blank when entering a phone number using WhatsApp.
  • Fixed In the card view screen for the Project list of the initial screen, fixed the bug where the application count appears as 0.

Application​

  • Changed In Management > Agent CONFIG. for the Java project, changed the category for the biz_exceptions and ignore_exceptions options from Alert option > HTTP outbound call to Exception.
  • Fixed Fixed the issue where the layout is broken when the log search password has been set in the Transaction log tab of the Trace analysis window.

Server​

Fixed In Server list > Server list, fixed the bug where the colors of table rows do not change according to the server status change.

Log​

Feature In Log search, Log trend, and Live Tail, added the full screen switching function to see only the timestamps and logs. Select on the upper right of the screen.

URL​

Changed In Alert > Notification setting, enabled Avoids a large number of alerts.

Preview 1.109.8​

August 31, 2023

Common​

Dashboard​

Changed In the Transaction analysis window, added information on the transaction selection status to the URL for list transaction analysis.

This change can be useful when sharing the URL of the selected transaction with other users.

  1. Select a specific transaction from the transaction list, and then copy the URL of the web browser.

  2. Go to the URL in the web browser through a new tab or window.

  3. You can see that the transaction has been selected in the transaction list.
