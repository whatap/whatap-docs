Preview 1.93.1

Through https://preview.whatap.io, you can experience the following new features.

2022-12-15​

Fixed In Management > Integration, fixed the bug where you can install and set items when you do not have the Edit role.

2022-12-08​

Flex Board​

Fixed Modified to enable deletion when the dashboard is copied as read-only.

Log​

  • Fixed Fixed the bug where "==" is duplicated in the expression statement when "==" has been entered as a comparison operator in the filter.
  • Fixed Fixed the bug where search cannot be performed because "+" has been added as a prefix when entering a numeric field (.n field) in the filter.

2022-12-05​

Container Map​

Fixed Fixed the bug where the "Agent Update" guide message appears when selecting a Pod category after updating to Agent 1.2.0.

2022-12-02​

Multiple transaction trace​

Fixed In Analysis > Multiple Transaction Trace, fixed the bug where charts were not drawn when searching for some data.

Transaction search​

Fixed In Analysis > Transaction Search, fixed the bug where the data retrieved from the previous project appears when moving to another project.

2022-11-30​

Common​

  • Change Removed the Beta display in integration items for platform selection when creating a project.
  • Change In Event Setting and Notification Setting, modified not to delete Event Receiving Tag that is being applied to events.

Container Map​

  • New Added the Pending Pod function to the Laboratory menu for Kubernetes suite products.
  • Change Modified to expose the container information when one or more containers exist in a pod.
  • Feature Added CronJob to the related events to provide additional log data to the job-type pods.

Flex Board​

  • Feature Add the Log RPS Board template for the server suite products to Dashboard.
  • Feature Addition of widget templates * Log count per second * 4xx, 5xx Count * Request Success Rate * Response Time

Maintenance Plan​

Change Changed the style of the maintenance plan's guide text at the top of the screen and modified the locations of buttons.

Log​

Change Changed the data retention period setting in the log setting menu to the log data retention period by category setting.

Multiple transaction trace​

Fixed In Analysis > Multiple Transaction Trace, fixed the white out issue that occurs when `Open chart in new window` is clicked.

Alert message​

New Added a new menu, Site Map > Laboratory > Alert Message to the Application, Container, DB, and Server suite products.

Event Setting​

New In Alert > Event Setting > Metrics, added the button to move the Notification message settings menu under Laboratory.

Integration​

Changed the menu entry path and design after changing the number of installed integration items (Unlimited → 1).

  • Change Repositioned the Integration Items menu to the submenu under Management; modified to install and set only one integration item.
  • Change When entering into the project, modified the entry path depending on the installation status of integration items.

    • In case any item has been installed: Goes to the dashboard that matches the item type.

      • aws: Moved to Amazon CloudWatch > EC2.
      • ecs: Moved to Amazon ECS > Task.
      • Others: Moved to Flex Board.

    • In case the items have not been installed: Moves to Management > Integration Items.

  • Change Changed the project card design.
  • Change In case the integration item is ECS: In Management > Install Integration, repositioned the Activate monitoring button, which was fixed at the bottom of the Install guide page, to the inside of the installation process.

Integration, Cloud Monitoring​

Deprecate In Management > Integration items, deleted the Select All checkbox upon configuration of the AWS CloudWatch service.

Agent CONFIG.​

Fixed Fixed the issue where the agent name is not reflected in the server list when it has been set as an alias.

Billing Management​

In Billing Management > Subscription History, modified the following functions.

  • Change In the Monthly Estimation column, modified to additionally display the `session usage details`.
  • Change Modified to display the MYSQL RDS product on the Product Type column.
