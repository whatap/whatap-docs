Preview 1.101.X

In https://preview.whatap.io, the following new features and enhancements can be experienced in advance.

Preview 1.101.25​

May 22, 2023

Common​

Change Modified the Delete icon () and the Channel Inquiry icon () in all services.

Container​

Fixed In Dashboard > Container Map, additionally fixed the bug where the app-related category logs cannot be viewed when using Log > Log Search after selecting a group.Related: Preview 1.101.22

Browser​

Feature In Alert > Event Setting > Metrics, added the templates for metrics and composite metrics alert.

CloudWatch, AZURE, OCI, NCP​

Feature In Management > Agent CONFIG., added 1 minute as a collection cycle option.

CloudWatch​

Fixed In Management > Agent CONFIG., modified the default value for validity period to 1 Hour.

Preview 1.101.24​

May 19, 2023

Common​

  • Fixed In My Usage > Preview Invoice, fixed the bug where the usage fee does not appear when there is no discount.
  • Fixed Fixed the problem where the previous page does not appear because org=undefined has been entered in the query parameter when user clicks the profile icon, goes to Manage Account INFO., and then tries to go to the previous page (by clicking Back).

Application​

Change In Dashboard > Transaction Map, as the TX Trace table columns, added the items for SQL Count, Login ID, and SQL Fetch Count.

Log​

  • Change In Log > Log Search, modified to enter the value, `none` as the key when adding a category while only a key has been entered in the filter with no value assigned.
  • Change In Log > Live Tail, fixed a typo error (for the term, `spacing`) among the terms in the content filter tooltips.
  • Change In Log Search, modified not to automatically attach the prefix, `+` when `-` has been entered as the search value's prefix upon entry of a filter.

Preview 1.101.23​

May 19, 2023

Log​

Change In Log > Log Setting, unified the mixed terms.

Browser​

Fixed In Dashboard > Browser Error, fixed the flickering symptom while loading charts in the error message table.

Preview 1.101.22​

May 18, 2023

Container​

Fixed In Dashboard > Container Map, fixed the bug where the app-related category logs cannot be viewed when using Log > Log Search after selecting a group.

Preview 1.101.21​

May 18, 2023

Database​

  • Change In Dashboard > Multiple Instance Monitoring, changed the selection option from Database to Instance.
  • Fixed In Dashboard > Instance List, added the function to align the table and adjust the column width to compensate for text truncation in the Event window that appears by selecting the Fatal or Warning icon in the Status column.

Preview 1.101.20​

May 18, 2023

Container​

Change In Dashboard > Container Map, modified to search logs based on the microOid when using the Log > Log Search function on the summary view panel in case of the app-related category.

Preview 1.101.19​

May 18, 2023

Common​

  • Fixed Fixed the widget MXQL in the Server Resource Plus dashboard.
  • Change Added and reorganized the Log RPS Trend widget metric for the Log RPS Board dashboard template.

Database​

Fixed Added the missing `sql_param(query_param)` column in the lock tree table from the panel at the bottom of the following menus.

  • Dashboard > Instance Monitoring

  • Dashboard > Multiple Instance Monitoring

  • Analysis > Counts Trend

Preview 1.101.18​

May 17, 2023

Fixed In My Usage > Project Subscription, fixed the bug where a whiteout occurs.

Preview 1.101.17​

May 16, 2023

Change In Sitemap > User Behavior, modified to search the user behaviors for all accounts with the member management role.

Preview 1.101.16​

May 16, 2023

Common​

Change In My Usage > Usage, modified to provide the average usages for each time.

Container​

Fixed Fixed the symptom where part of the Log Search and Event tabs in the container map's summary view panel.

Preview 1.101.15​

May 11, 2023

Common​

  • Fixed Among sidebar menus, modified to add Browser types in project type filters and display the text according to the selected language.
  • Fixed While moving the tab in STAT/REPORT > Statistics, fixed the problem where the parameters in the selected organization are kept.

Database​

Fixed In Management > Agent List, fixed the error where set instance alias is not applied to the following menu.

  • Dashboard > Instance List

  • Dashboard > Multiple Instance Monitoring

  • Analysis > Counts Trend Comparison

Container​

Fixed In Resource > Node Details, fixed the bug where partial data in the container list does not appear.

Preview 1.101.14​

May 11, 2023

Common​

  • Fixed In Home > Flex Board, fixed the bug where the slow query widget does not appear.
  • Change Updated the MXQL path versions of the widgets used in the Integrated Service Dashboard template.

Container​

  • Feature In Dashboard > Container Map, the metrics whose unit is percentage among the chart tooltips that appear when user hovers the mouse over a container element, are displayed with the bar graph charts.
  • Change In Dashboard > Container Map, added the selected metrics item in the > Label option.

Preview 1.101.13​

May 10, 2023

Container​

Deprecate In Dashboard > Application Service Dashboard, deleted the Dashboard for Service button for each group panel under the Multi Service Dashboard tab.

Preview 1.101.12​

May 9, 2023

Container​

Fixed In Container Map, fixed the bug where the user without Log Read role cannot see the Container Stdout/Stderr data.

Preview 1.101.11​

May 9, 2023

  • Fixed Through Home > Flex Board > Hitmap widget, fixed the problem where the member with the Log Read role cannot see the Transaction Log tab in the Transaction Detail screen.
  • Fixed In Management > Project Member, fixed the problem where the icon button on the utmost right is hidden when the width of the browser screen is a specific size.

Preview 1.101.10​

May 3, 2023

Database​

  • Change In STAT/REPORT > Table Size Increase, changed the Download CSV button to .
  • Fixed In the follow menu path, fixed the problem where the download feature does not work depending on the region clicked on the drop-down menu item when the button is selected.
    • STAT/REPORT > Table Size Increase
    • STAT/REPORT > SQL Statistics > DB, USER, HOST tab

Preview 1.101.09​

May 4, 2023

Server​

  • Fixed In Server Setting > Server List, deleted the inactive server deletion feature.
  • Fixed In Server Setting > Server Environment Variable, deleted the inactive server deletion feature.

Preview 1.101.08​

May 4, 2023

Common​

  • Change Removed the Integrated Dashboard and Integrated Metrics Board from the side menu and modified to access them only through Sitemap.
  • Change Modified to display the specific reason when there is no data response in the following widgets.

    • Log 4XX, 5XX Count

    • Request Success Rate

    • Response Time by Log

Log​

  • Fixed Fixed the typo error in Log > Log Setting > Log Parser Setting > Input Filter > Filter Key (okineName → okindName).
  • Fixed In Log > Live Tail, fixed the problem where the previous log appears when there is no log data searched.

Preview 1.101.07​

May 3, 2023

  • Fixed In Management > Project Member, fixed the issue where the Edit Log role does not appear in the Role column.
  • Fixed In Management > Project Management, fixed the bug where the panel title color does not change depending on the theme set by the user.

Preview 1.101.06​

May 2, 2023

Log​

Feature The Edit Log role has been added and the related functions has been applied to Log Setting.

Preview 1.101.05​

May 2, 2023

Fixed If the browser window has a specific size, when the JSON editing window of the widget is imported in Manager mode under Flex Board, a scroll appears and disappears repeatedly in the text box.

Preview 1.101.04​

April 28, 2023

Common​

  • Fixed Fixed the bug where logs are not queried intermittently when searching log profiles.
  • Fixed In My Usage > Usage, UI improvements have been reapplied.

Application​

Fixed In STAT/REPORT Statistics, fixed the problem where a whiteout occurs when the Domain tab is accessed.

Preview 1.101.03​

April 28, 2023

Application​

Fixed In Analysis > Transaction Search, applied the numeric type validation logic for the options that require numeric data only.

Preview 1.101.02​

April 27, 2023

Container​

Fixed In Management > Agent Installation, fixed the collection server IP display error in Precautions.

ECS​

Fixed In Flex Board, Amazon ECS > Service, fixed the problem where only one service is displayed in the ecs_service widget.

Application​

Fixed In Management > Agent Installation of a Java project, fixed the typo error for the agent download command.

Preview 1.101.01​

April 27, 2023

Common​

My Usage​

  • Change In Usage, applied UI improvements for better user experiences.
  • Feature In Usage, added the function to search the project name.
  • Fixed In My Usage > Preview Invoice, modified not to display the host column data in case of Browser, Log, Cloud, and Amazon ECS products.

Flex Board​

  • Change Modified the widget's interaction button state not to be changed regardless of the side menu On/Off under Dashboard.
  • Fixed In case of the rich text widgets, fixed the bug where the saved content are not applied after modification.

AWS Log​

  • Feature Modified for only the users with log query role to access the Live Tail, Log Search, and Log Trend menus under Dashboard.
  • Change Modified for the users who do not have the role to view logs to access Alert > Event Setting (however, functions cannot be added or modified).

Database​

DB commons​

Change Hid a menu: Analysis > Metric Anomaly Detection.

POSTGRESQL​

  • Change Hid STAT/REPORT > PG_STAT_STATEMENT.
  • Fixed Fixed the problem where a blink page appears when the guide link is clicked in Dashboard > Slow Query.

MYSQL​

Fixed Fixed the problem where a blink page appears when the guide link is clicked in Dashboard > Slow Query.

Container​

Fixed In Management > Agent Installation, fixed the collection server IP display error in Precautions.

ECS​

Fixed In Flex Board, Amazon ECS > Service, fixed the problem where only one service is displayed in the ecs_service widget.

Preview 1.101.0​

April 26, 2023

Common​

Fixed In Log > Live Tail, fixed the problem where data is not output when initially searching for log data.

Browser​

Fixed In Project list, fixed the problem where a Browser project is not searched when switched to List View mode.
