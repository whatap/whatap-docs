Preview 2.13.0.a

Note

In preview.whatap.io, the following new features and enhancements can be experienced in advance.

Preview 2.13.0.a3‚Äã

January 14, 2025

Common‚Äã

  • Feature In Analysis > Metrics Chart New, improved the menu features.

    • Added a specific time when the Max value of the hint data is recorded upon mouse over in the chart data.

    • Added the Select/Unselect button to the entire legend list.

    • Added the feature to select colors for chart lines.

    • Added the Copy CSV button to the More menu.

    • Added the Copy chart image button to the More menu.

  • Feature In Alert > Notifications, added the feature to select the notification level via a third-party plugin.

  • Changed Replaced the time selector feature in the Integrated Report menu.

    • Daily report: You can select the start/end date and time.

    • Weekly report: Simply select a start date. Then the end date is selected automatically after calculation.

    • Monthly report: Simply select a start date. Then the end date is selected automatically after calculation.

    • If the start date is the 1st date, up to the end of the month is automatically selected.

    • If the start date is not the 1st date, up to the previous date of the next month is automatically selected.

    Note

    Example

    • If the start date is January 1, from January 1 00:00:00 to January 31 23:59:59 is automatically selected.

    • If the start date is January 10, from January 10 00:00:00 to February 9 23:59:59 is automatically selected.

  • Fixed In Integrated Flex Board or Flex Board, fixed the issue where the notification sound is generated for every update in the notification list widget in real-time query mode

  • Fixed In Usage > Balance, fixed the issue where the list does not appear.

Application‚Äã

  • Feature In Dashboard > Transaction Map, added the feature to adjust the maximum value of the Y-axis on the chart by directly entering it.

  • Changed In Dashboard > Transaction Map, when adjusting the maximum Y-axis value with the or button, improved the feature so that adjustments can be made only in 10-second intervals if the value exceeds 30 seconds.

    • The segment from 10 to 50 seconds is moved at 10 second intervals.

    • The segment from 50 to 300 seconds is moved at 50 second intervals.

    • For the segment of 300 or more seconds, it is moved at 100 second intervals.

  • Fixed Fixed the issue where when checking SQL steps in the Table view tab of the Trace analysis window, if one of the substitution variables or bind variables in the SQL statement is released, all of them are released.

Server‚Äã

  • New Added the Metrics Explorer menu that can analyze the past data more intuitively.

    • Added categories in the Server Inventory menu.

    • Added the modal to guide the Metrics Explorer feature.

    Note

    This feature can be seen in the Laboratory > Metrics Explorer path.

  • Feature Added the Active event status tab in the group summary panel and detailed view of the Dashboard > Server Inventory Map menu to view active events within the past 24 hours based on the current time.

  • Changed Changed the top of the group panel in the Dashboard > Server Inventory Map menu to display the counts by group and server status and the number of events by status.

  • Changed In Server List, changed the logic to determine the server status.

    • Previous: Server status is determined only by active events.

    • Change: Server status is determined by active events + one-time events (Resolved notification events with the option turned off).

      One-time events are treated the same as active events during the valid time (1 hour by default) after occurrence.

  • Changed In Server List, on the Server status window that appears when the server status icon is clicked, changed the snapshot chart to display the charts only in valid event categories.

  • Changed Improved the server name column to be fixed when horizontally scrolling the Server Inventory table in the group summary panel and detailed view of the Dashboard > Server Inventory Map menu.

  • Changed Changed the content of the introduction page to the Server monitoring product.

Kubernetes‚Äã

  • New Added the kube-scheduler Dashboard menu to view monitoring data for kube-scheduler, one of the system components of Kubernetes.

    Note

    This feature can be seen in the Cluster > kube-scheduler Dashboard path. It is supported in Kubernetes agent 1.8.1 or later.

  • New Added the Container Timeline menu that can display the container status over time.

    Note

    This feature can be seen in the Container > Container Timeline path.

  • New Added the Pod Timeline menu that can display the Pod status over time.

    Note

    This feature can be seen in the Workload > Pod Timeline path.

  • Feature In the Target information tab of the Dashboard > Container Map menu, added the feature to check the Pod Startup Analysis information also.

  • Feature Added guidance about No data when the widget of the Cluster > kube-apiserver Dashboard menu is for the managed Kubernetes.

  • Changed Separated the summary analysis feature into a tab in the following menus:

    • Dashboard > Container Map
    • Dashboard > Node Map

  • Fixed In Dashboard > Container Map, fixed the issue where the metric name of the filter.

  • Fixed In Container > Container Images, fixed the issue where tags are not displayed properly when the image name containes @.

  • Fixed In Cluster > Ingress List, fixed the issue where several incorrectly set rules are missing.

  • Fixed Fixed the errors for Request and Limit setting messages in the container summary analysis.

  • Deprecated Removed the feature to navigate to the tab when clicking the diagram in the following menu paths:

    • Cluster > Node Timeline

    • Cluster > Node List > Detail
