Preview 1.105.X

In https://preview.whatap.io, the following new features and enhancements can be experienced in advance.

Preview 1.105.21​

July 24, 2023

Common​

Alert​

  • Feature In Notification setting, added the fold/unfold function in the Event notification settings per-user section.
  • Change In Alert > Event setting > Metrics event tab, the Related category item has been deleted upon addition of any event.

Application​

.NET​

New Added the Analysis > Stack menu. See the following.

Container​

  • Fixed In Resource > Node list, fixed the bug where a loading event appears even when there is no container data to display.
  • Deprecate Alert > Event setting > Metrics event tab, the Related project function has been discarded upon addition of any event.

Preview 1.105.20​

July 21, 2023

Common​

Report​

  • Fixed In Report, fixed the bug where report creation is not made when timeout occurs.
  • Change In Report, modified the UI loading display format when creating reports.

My Usage​

Fixed If the language is set to English, modified to search countries with lowercase characters in Payment information.

Server​

  • Fixed In Alert > Event setting > Metrics event tab, modified to display the name-related data instead of ID on the Testing event rules chart.
  • Fixed In Server list > Server detail, fixed the time reset bug upon refreshing after the time range has been changed.

Preview 1.105.19​

July 21, 2023

Database​

V2 commons​

  • Feature In Dashboard > Instance monitoring, the sorting changes in the Active Session table can be saved in presets.
  • Fixed In Dashboard > Instance monitoring, fixed the bug where sorting is canceled after refreshing data.
  • Feature In Dashboard > Instance monitoring, added the view for the Active Session table on a new window.
  • Feature In Dashboard > Flex Board, the sorting changes in the Active Session table can be saved.
  • Change Changed the deadline date of the project version selection.
    • PostgreSQL: September 30, 2023 → October 31, 2023
    • MySQL: October 30, 2023 → November 30, 2023
  • Fixed In Analysis > Counts trend comparison, improved the issue where charts appear unnaturally due to differences in data calling and loading speed.

PostgreSQL​

Feature In Dashboard > Instance monitoring for V2 products, added the plan lookup to the session details in the active session table.

MySQL​

Feature In Dashboard > Instance monitoring for V2 products, the initial sorting in the Active Session table can be set.

Preview 1.105.18​

July 20, 2023

Common​

  • Feature Added the guidance message when throttle has been set for the agent to limit API requests.
  • Fixed When inviting members for the project, group, and organization, the guidance message has been changed for non-existent accounts.

Preview 1.105.17​

July 20, 2023

Application​

  • Fixed In the list under STAT/REPORT > Statistics > Transaction tab, fixed the bug where operation is not made after clicking the T or U button.
  • Change In Dashboard > Active transaction, changed the display colors for slow transactions and very slow transactions in the table.

Container​

Fixed In Dashboard > Container map, fixed the bug that causes intermittent whiteouts.

Preview 1.105.16​

July 18, 2023

Common​

Flex Board​

Fixed Fixed the issue where the button layout goes out of the menu when the width of the Hitmap widget becomes narrower as the screen width decreases.

My Usage​

  • Feature In Payment information, added `Overseas bank transfer customer registration` when registering the payment data.
  • Change In Payment information, modified the payment process based on the country selection when registering the payment data.
  • Change In Payment information, modified not to display `Register reverse issuance account transfer` in Korea when registering the payment data.
  • Change In Payment information, changed the promotion banner.
  • Fixed In Payment information, modified the placeholder of `Register overseas card` to be displayed according to the locale language.
  • Fixed Fixed the page number count setting bug in Billing and payment history.

Report​

Fixed Deleted the Visitor column from the Daily applications custom report.

Application​

GO, PHP​

  • Fixed In Dashboard > Application dashboard > Memory detail window, modified the following.

    • Text change: Heap total → System memory, Heap use → Process memory

    • Deleting the system GC button

Dashboard​

  • Fixed In Dashboard > Application dashboard > Memory detail window, modified the following.

    • When the size of the name column decreases, modified so that the tooltip is enabled by hovering the mouse over the ellipsis.

    • Fixed the bug where the tooltip of the Detail button does not disappear after the button is clicked.

  • Fixed In Application dashboard > dragging HITMAP widget > HITMAP transaction > selecting TX trace > Statistics window, in case the transaction log is not set under the Transaction log tab:

    • Language setting is Japanese: Fixed the line break bug in the displayed guidance message.

    • Language setting is Korean: Modified to apply the style in the displayed guidance message.

    • Modified to go to the Docs page for the set language when clicking the link in the displayed guide message.

Analytics​

  • Fixed If there is no result when searching for data in the MSA analysis menu, modified to display the `No data` guidance message.

Browser​

Change In Management > Agent Installation, added proxyBaseUrl as the config option of the browser agent script.

Preview 1.105.15​

July 18, 2023

Container​

Change Added the group title notations for each metric in Performance summary in case of a cluster project.

July 17, 2023

Fixed In Flex Board, added exception handling in the MXQL creation logic when using the logbits field in the app_user category for application monitoring products.

Application​

Dashboard​

  • Fixed Modified the notation of onode to Agent node in the Record summary tab on the Transaction analysis window in the Kubernetes application project.
  • Fixed Fixed the bug where the statistics modal chart does not appear on the Transaction analysis window.
  • Fixed In the Transaction analysis window, added the guidance message when any data is not viewed in SQL summary, Method summary, and HTTP Call summary tabs.

Analytics​

Fixed In Multi transaction trace, fixed the bug where characters other than number cannot be entered in Search by MTID / CUSTID.

Database​

Fixed In PostgreSQL V2 and Mysql V2 projects, English and Japanese guidance messages are supported in Slow query and Database size.

Dashboard​

  • Fixed In Multi container map, fixed not to move to the Namespace menu.

  • Change In the Display detail panel under Container map, changed the notation method when the block count is very large.

Preview 1.105.13​

July 13, 2023

Common​

  • Fixed In the project creation menu, modified the product names (Node.js, Altibase).
  • Fixed In the Japanese version, modified the sidebar texts for Create group and Create project buttons to be displayed in English.

  • Change In Dashboard > Transaction map, modified the chart design.
    Before modificationAfter modification
  • Change In Dashboard > Transaction map, improved to first display the error when a normal transaction and an error transaction overlap.

  • Feature In Analysis > Page load hitmap, added the page group filter and search filter.
  • Fixed In Dashboard > Browser Error > Error Detail, modified the background style.

Container​

Fixed In Management > Agent Installation, modified as follows.

  • Application Agent Installation > Docker PHP

    • Deleted the subtitle at the top of the access key copying area in the Verify Access Key section.

    • Deleted the subtitle from the Kubernetes environment variables and volumes section.

  • Application agent installation > all installation types: Unified wording in the Verify Access Key section and added an icon to the left of the Chat contact.

  • In the Install guide section, unified the icon on the Chat contact button with the channel talk icon.

  • In Kubernetes agent installation > KUBERNETES > Kubernetes agent creation section, added notices related to OOM issues.

Laboratory​

Change (Only for Container and Java products) In Remote Access Status, modified to use the function even by the members without the View log role.

Preview 1.105.12​

July 12, 2023

Common​

Alert​

  • Feature In Notification setting, added the User-Header function in the Webhook JSON addition form for the 3rd party plugin.

    User Header: The key and value pair can be checked in the header when receiving a webhook. The following is an example of the received webhook when the user header key and value are entered as User-Header-Key and TestValue.

    example
    POST /test HTTP/1.1
    Host: whatap.requestcatcher.com
    Accept: text/plain, application/json, application/*+json, */*
    Connection: keep-alive
    Content-Length: 234
    Content-Type: application/json
    User-Agent: Java/17.0.7
    User-Header-Key: Test-Value
  • Fixed Fixed the issue where the window closes after adding a 3rd party plugin.

Dashboard​

  • Fixed In the Transaction analysis window, modified to display as normal when the transaction name under the Record summary tab is longer than the screen.
  • Change In the Transaction analysis window, modified the layout width of the Tree view tab.

Flex Board​

  • Fixed Fixed the issue where the menu does not appear upon clicking the dashboard copy button.
  • Fixed Added missing multilingual (English) texts in the dashboard copy window.
  • Change When using the Copy to others function on a dashboard, added the _copied suffix in the dashboard title.

Laboratory​

Preview 1.105.05​

July 5, 2023

Common​

Change Modified the password validation logic.

  • Use at least 9 characters with 3 combinations among English lowercase and uppercase letters, numbers, and special characters.

  • The same password as your email cannot be used.

  • The same password as the one you used cannot be used.

  • A password containing consecutive keyboard arrangements is not allowed.

  • A password containing consecutive alphabets is not allowed.

  • A password containing consecutive numbers is not allowed.

  • The same password as the current one cannot be used.

Container​

Change In Log > Log setting, modified the node agent log setting so that it can be applied with a click instead of directly entering options for user convenience.

URL​

Fixed When using the Add URL function in URL list, deleted the duplicates in the Check location section.
