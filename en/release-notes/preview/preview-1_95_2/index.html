Preview 1.95.2

In preview.whatap.io, the following new features and enhancements can be experienced in advance.

Hotfix: 2023-02-16​

Application​

Analysis​

Fixed In HITMAP, fixed the bug where the End Time column in the TX Trace table appears before the Start Time column.

Hotfix: 2023-02-09​

Application​

Analysis​

Fixed In Multiple Transaction Trace > Tree View, fixed the bug where profile integration data is not represented in a tree structure.

Official Release: 2023-02-08​

Common​

Flex Board​

Feature In the dashboard list under Home > Flex Board, added the Share Dashboard () function.

Analysis​

  • Fixed In Sitemap > Analysis > Metric Anomaly Detection, replaced the UI logic due to the UI bug upon selection of category and field.

  • Fixed Modified not to change the start date, metrics of interest, and dashboard tab, if query is not finished in CUBE.

Transaction Detail Application​

  • Change Changed the SQL analysis tab name to SQL Summary.

  • Change Changed the method analysis tab name to Method Summary.

  • Feature Added the HTTP Calls tab: Provided the HTTP call count, time sum, and time average.

Management​

  • In Management > Project Member, the following functions have been added.

    • Feature Added the function to display the member affiliation type (group/project) for improved project member management.

    • Feature Added the tooltip (deletion is impossible) when the member deletion () button is disabled.

  • In My Usage and Billing Management, the following functions have been modified.

    • Change In Preview Invoice and Usage, changed the log display and expression units according to the change in log charging policy.

      • Change of the log display unit: LOGs ⇒ LOG UNITs
      • Change of the log representation unit: Modified to display the daily average for the summed log line count.
      • In Preview Invoice, changed to display the log usage in the Metered Units column.
      • In Usage, changed the metering summary column for usage details.

Application​

Active Transaction​

Change Modified to make the input value invisible in the secret key field to stop threading.

Server​

Fixed In Server List, fixed the bug where the modal content that appears when the server status icon is clicked does not match that of the selected server.

Container​

Container Map​

Feature Added DaemonSet and StatefulSet in the pod category view grouping conditions.
