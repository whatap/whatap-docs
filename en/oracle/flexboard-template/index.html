Application of the template

Use the pre-configured templates. You can easily perform the initial configuration and change the settings for use.

  1. Select Flex Board.

  2. Select a desired template from the template list on the left.

  3. If the Flex board management window appears, enter the name in Rename.

    • Select a widget from the Layoutand then drag it to the desired location.

  4. Click Save. The Flex board is created based on the selected template.

    • You can see the list of boards you have created in Flex Board.
