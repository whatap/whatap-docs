Server List

Home > Select Project > Server List > Server List

Select a project in the initial screen of the WhaTap monitoring service, and then select Server List > Server List under Project Menu. The Server List displays the overall status of the collection server and provides the comparison functions between server management functions and between servers.

Data refreshing and server status‚Äã

Data update‚Äã

The data on the server list page is automatically updated every 60 seconds.

  • Pause : You can stop automatic update of the data.

  • Playback : You can restart the auto update you have stopped.

Note

For more information on how to use the time selector.

Server Status‚Äã

It displays the current status of the collection server by status. The status is updated every 1 minute.

  • Inactive: The agent cannot collect data from the server.

  • Fatal: Any fatal events occur in the collection server.

  • Warning: Any warning events occur in the collection server.

  • Normal: The agent is running normally in the collection server.

  • Paused: The agent temporarily paused collecting data.

Searching‚Äã

You can search for agent names and IPs by using the search bar at the upper right.

Filter‚Äã

You can specify the filter by directly entering a value in the input field or by adding a filter. The filter tags are entered in the following order: filter key, condition, and search value.

Enter the filter key, condition, and search value.‚Äã

  • When you enter the filter key, it provides the following options: Agent Name, CPU Used %, Memory Used %, Disk Used %, Disk I/O %, Network Traffic, and private_ip.

  • When entering a condition, the options such as >, <, <=, >=, ==, and != are provided.

  • Upon entry of the search value for matched search (>, <, <=, >=, ==), it is highlighted in blue, In case of excluded search (!=), it is highlighted in red.

  • Upon entry of the search value, you can search using the case-sensitive option.

Note

  • If the filter tags exceed 2 lines, you can fold it by selecting the Fold icon.

  • After entering a filter tag, you can use the Shift and Tab keys simultaneously in the input field to go to the previous filter tag.

  • After entering a filter tag, you can use the Tab key in the input field to go to the next filter tag.

Deleting filter tags‚Äã

  • A tag can be deleted with the Backspace key.

  • You can delete a tag by selecting the X icon for it.

  • You can delete all tags by selecting the Delete All X icon in the input field.

Filter favorites‚Äã

You can select the Favorites icon on the right of the input window to add, delete, and select desired filter search conditions in the favorites.

Exceptional cases for application of filters‚Äã

  • In the tag with a numeric index (search key that ends with .n), only numbers can be input for the search value.

  • Duplicate ‚Äòsearch key‚Äô and ‚Äòsearch value‚Äô cannot be entered.

  • Searching is not possible when there is a tag that does not have either condition key or search value. Invalid tags are displayed in gray.

Server Status‚Äã

It displays the status of all servers to which agents are assigned. If an agent is selected in the list, the Server Detail page appears. For more information about Server Detail, see the following.

File system information‚Äã

File system information can be checked by clicking the icon on the left of the agent name to view in detail.

Column Settings‚Äã

  • Column settings

    In the menu at the top, if you click Column settings, you can select columns to displays in the table as follows.

    Note

    The selected Columns are stored as the browser cookie values so that the status is maintained after refreshing the page. If an error occurs between settings due to cookie deletion or other reasons, it is initialized to the full selection state.

  • Column Width Setting

    You can set the column width by dragging the edge of the column title.

  • Align Columns

    Sort Columns By clicking the column title, you can sort in ascending or descending order.

    Note

    The settings for Columns Width and Columns Align are stored in the browser cookies and the status is maintained after refreshing the page. If an error occurs between settings due to cookie deletion or other reasons, it is initialized to the unset state.

Preset management‚Äã

In Server List > Server List, if you click Column settings, you can manage and share the presets.

The selected column combination can be saved as a preset, and shared with other users who access the same project. Recently used presets are stored in the personal browser and automatically loaded when connection is made again. If the recent preset is deleted, it is reset to the default preset (Default). For information on how to set presets, see the following.

  • The widths and alignments for each column are saved in the user's browser and automatically applied when the menu is re-accessed. These settings are applied only to the user and cannot be shared with other users.
Caution

Only the users with the project edit role can save and delete presets. For more information about member roles, see the following.

Refresh‚Äã

If you click Refresh, the data is updated based on the current time.

Agent Transfer‚Äã

By selecting Transfer, you can transfer the agent to another project.

  1. Select a project to transfer on the right screen.

  2. Select an agent to transfer on the left screen.

  3. By clicking Right arrow, you can transfer to the target project.

  4. If you click Save, the transfer is completed.

Note
  • If you move the same agent again within 10 minutes just after execution, it may cause a conflict.
  • The collected data is not transferred. Do not delete the existing projects for a while to retrieve the past data.

Comparison between servers‚Äã

If you click Compare, the cross-server comparison chart is provided as follows.

  1. Select Compare.

  2. Like , check whether the Compare column has been added on the left of the server list.

  3. In the Compare column, select a server for comparison..

  4. To enable the comparison function, you can see the chart to compare in the area.

Note

  • The comparison servers can be selected up to 5.

  • Unique colors are assigned to each server. The unique colors are used to distinguish servers in Compare.

  • If you select Check period, the past data can be retrieved.

Monitoring Management‚Äã

This function manages servers through agent commands. Click Management and then select a server to apply the management features from the list. On the right, Pause, Restart, and Delete buttons are activated.

  • Pause: Data collection for the agent is paused. It is displayed as the paused state in the server list.

  • Restart: The agent is restarted.

  • Exclude: The agent operation is stopped. It is excluded from the server list and the server status cannot be checked.
