트레이스 분석 변경 사항

2023년 7월 25일 릴리스된 트레이스 분석 기능의 변경된 사항을 안내합니다. 트레이스 분석 기능에 대한 자세한 내용은 다음 문서를 참조하세요.

트랜잭션 목록​

  • 컬럼 제목을 명확한 이름으로 변경해 사용자들이 직관적으로 인식할 수 있도록 변경했습니다.

  • 중요도와 우선 확인해야 하는 데이터를 기준으로 컬럼의 순서를 변경했습니다.

  • 자주 확인하지 않거나 불필요한 컬럼 내용을 삭제했습니다.

테이블 뷰​

  • 컬럼 제목을 명확한 이름으로 변경해 사용자들이 직관적으로 인식할 수 있도록 변경했습니다.

  • 시작, 종료, 경과, 에이전트 명(oname) 등 중요 정보를 상단에 배치했습니다.

  • 누적 시간을 삭제하고 시간을 추가했습니다.
