3분기 안내

2023년 10월 31일
TW J
테크니컬 라이터

와탭의 2023년 3분기 서비스 신규 기능 및 주목할 만한 변경 사항 개요를 안내합니다. 와탭이 새롭게 제공하는 기능들을 한 눈에 살펴보세요.

공통

프로젝트 플렉스 보드 편집 권한이 추가되었습니다.

Application
  • 트레이스 분석 기능이 새롭게 개편되었습니다.
  • 트랜잭션 맵 메뉴 맵 차트가 개선되었습니다.
Server

리소스 보드 및 이벤트 설정 메뉴가 개선되었습니다.

Kubernetes
  • Docker PHP 설치를 제공합니다.
  • 통합 컨테이너 맵 메뉴를 제공합니다.
Database
  • PostgreSQL 상품 V2 서비스가 시작되었습니다.
  • MySQL 상품 V2 서비스가 시작되었습니다.
Browser
  • 페이지로드 히트맵 메뉴가 개선되었습니다.
  • 브라우저 에러 메뉴에서 에러 상세가 추가되었습니다.
AWS Log

AWS Log 모니터링 상품 정식 서비스가 출시되었습니다.

Network

네트워크 성능 모니터링(Network Performance Monitoring) 상품 베타 서비스가 시작되었습니다.

상품별 신규 기능 외 변경 사항 개요 및 해당 릴리스 버전, 에이전트 변경 사항은 다음 안내를 통해 확인할 수 있습니다.



공통​

와탭 2023년 3분기 공통 부분의 주목할만한 신규 기능 및 변경 사항으로 프로젝트 플렉스보드 편집 권한 추가 및 편의성 향상을 위한 개선을 소개합니다.

프로젝트 플렉스보드 편집 권한이 추가되어 이제 프로젝트 내 Flex 보드를 수정하기 위해서는 해당 권한이 필요합니다.

그외 전반적인 편의성 향상을 위한 업데이트가 이루어졌습니다. 와탭의 서비스 버전과 상품 목록을 확인할 수 있는 와탭 모니터링 정보 메뉴가 추가되었습니다.

또한 이벤트 수신 설정 시 WhatsApp 알림 설정이 새롭게 추가되었고 통합 보고서 메뉴에서 일간 보고서 생성 시 시간 선택 제한 방식이 변경되었습니다. 기존에는 종료 시간을 전일 23시 59분까지 선택할 수 있었지만, 이제 종료 시간을 현재 시간 기준 1시간 이전까지 선택할 수 있습니다.

다음 개요 안내를 통해 2023년 3분기 공통 부분의 신규 기능 및 주요 변경 사항을 해당 릴리스 버전과 함께 확인해 보세요.

신규 기능 및 주요 변경 사항 안내

  • Feature 에이전트에서 throttle이 설정되어 API 요청 제한 시 안내 메시지 추가 Service 1.104.18​

  • Changed 시간 조회 구간이 변경될 때 인터벌 선택 옵션에서 기본 항목을 선택하도록 로직 수정 Service 1.108.14

  • Feature 인터벌 선택 옵션에서 기본 항목을 선택한 경우 기본 인터벌 값을 표시하는 기능 추가 Service 1.108.14

  • Feature 와탭의 서비스 버전과 상품 목록을 확인할 수 있는 와탭 모니터링 정보 메뉴 추가 Service 1.108.15

경고 알림​​

  • Feature 이벤트 수신 설정 > 사용자별 이벤트 수신 설정 메뉴에서 WhatsApp 알림 설정 추가 Service 1.104.00

  • Feature 이벤트 수신 설정 메뉴에서 3rd 파티 플러그인의 Webhook JSON 추가 양식에 사용자 헤더(User-Heaer) 기능 추가 Service 1.106.00​

  • Feature 이벤트 수신 설정 메뉴의 사용자별 이벤트 수신 설정 섹션에 접기/펼치기 기능 추가 Service 1.106.00​

보고서​​

  • Feature 통합 보고서 메뉴에서 일간 타입 보고서의 시간 선택 제한 수정 Service 1.104.00

관리​​

  • Feature 계정 정보 관리 > 사용자 전화번호, 알림 수신 설정 메뉴에서 WhatsApp 알림 설정 추가 Service 1.104.00

  • Feature 멤버 목록 메뉴에서 멤버의 삭제 및 권한 수정 버튼 위치를 멤버 이름 컬럼으로 이동 Service 1.104.00

  • Feature 조직 생성 및 조직 관리 메뉴에서 로고 이미지 안내 문구 추가(이미지 권장 사이즈, 파일 용량, 확장자에 대한 안내) Service 1.104.00

  • Feature 프로젝트 플렉스보드 편집 권한 추가 Service 1.104.00

실험실​

  • Feature Kubernetes, Java 상품에 사이트 맵 > 실험실 > Remote Access Status 메뉴 추가 Service 1.104.00

  • New 실험실 메뉴에서 통합 로그 검색기 추가 Service 1.104.12

이용 내역​

  • Feature 결제 정보 메뉴에서 결제 정보를 등록할 때 '해외 계좌 이체 고객 등록' 추가 Service 1.104.17

User Interface​

  • Feature 사이드 바에서 프로젝트 이름에 프로젝트 전체 이름을 표시하는 툴팁 추가 Service 1.104.00

Applicatoin​

와탭 2023년 3분기 Application 상품군의 주목할 만한 신규 기능 및 변경 사항으로 트랜잭션 분석 기능 개편 및 트랜잭션 맵 화면 개선에 대해 소개합니다.

트랜잭션 분석 기능 개편이 시작되었습니다. 기존에는 히트맵 아래로 트랜잭션 목록을 불러오는 방식이었지만, 이제 히트맵 드래그 시 트랜잭션 분석 팝업창이 나타나 구간별로 해당 상세 정보를 분석하고 비교하기 수월해졌습니다. 또한 트랜잭션 목록 필터에서 에러 건만 바로 확인할 수 있는 옵션이 추가되었습니다.

트랜잭션 맵은 가시성을 높이기 위해 맵 디자인을 변경하였으며, 일반 트랜잭션과 에러 트랜잭션이 겹치는 경우 에러 건이 먼저 보이도록 변경되었습니다.

수정 전수정 후

또한 .NET 상품의 경우 기존에 Java와 Python 상품에서만 제공되었던 스택 메뉴가 추가되었습니다. 스택 메뉴를 통해 탑 스택과 유니크 스택, 액티브 스택을 간단하게 확인할 수 있습니다.

Kubernetes​

와탭 2023년 3분기 Kubernetes 상품의 주목할 만한 신규 기능 및 변경 사항으로 Docker PHP 설치 제공 및 새롭게 추가된 통합 컨테이너 맵 메뉴를 소개합니다.

이제 Docker 컨테이너 기반으로 실행하는 PHP 애플리케이션에 와탭 모니터링 에이전트를 적용하고 컨테이너 이미지를 패키징할 수 있습니다. whatap-php 패키지를 설치 후 설치 스크립트 및 컨테이너 환경변수를 설정해 해당 에이전트를 설치할 수 있습니다.

클러스터 프로젝트와 네임스페이스 프로젝트를 포함한 통합 컨테이너 맵 메뉴를 새롭게 제공합니다. 통합 컨테이너 맵은 프로젝트별 그룹화 옵션을 추가로 제공해 클러스터 프로젝트 및 네임스페이스 개별 프로젝트 내 컨테이너 맵을 조회할 수 있습니다.

  • Feature 컨테이너 맵 메뉴에 불일치 필터 옵션 추가 Service 1.104.00​

  • Feature 컨테이너 맵 메뉴의 컨테이너 요약 보기 패널에서 로그 > 로그 검색 탭에 로그 조회 권한 및 상태에 따라 메시지를 다르게 표시 Service 1.104.00​
