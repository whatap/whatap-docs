3분기 안내

2024년 10월 21일
TW J
테크니컬 라이터

와탭의 2024년 3분기 서비스 신규 기능 및 주목할 만한 변경 사항 개요를 안내합니다. 와탭이 새롭게 제공하는 기능들을 한 눈에 살펴보세요.

공통
  • 이벤트 설정 메뉴가 개선되었습니다.
  • Application, Kubernetes, Database 상품군에서 메트릭스 차트 New 메뉴를 새롭게 제공합니다.
Application

통계 및 인스턴스 성능 관리 등 공통 메뉴가 개선되었습니다.

Server
  • Unix HP-UX 설치 패키지를 제공합니다.
  • 서버 인벤토리 메뉴를 새롭게 제공합니다.
Kubernetes
  • 노드 맵, 노드 타임라인, 노드 디스크 목록과 같은 신규 메뉴가 추가되었으며, 기존 노드 목록 메뉴가 개선되었습니다.
  • 컨테이너 요약 분석, Pod 요약 분석 기능을 새롭게 제공합니다.
  • Service 목록 메뉴를 새롭게 제공합니다.
Database
  • Oracle, Altibase, Cubrid 상품 V2 서비스를 제공합니다.
  • Oracla Pro 상품 정식 서비스를 제공합니다.
  • 스크립트 매니저 메뉴를 새롭게 제공합니다.
  • 멀티 인스턴스 모니터링 메뉴의 기능 개선이 이루어졌습니다.
Browser

세션 리플레이 기능 안정화가 이루어졌습니다.

상품별 신규 기능 외 변경 사항 개요 및 해당 릴리스 버전, 에이전트 변경 사항은 다음 안내를 통해 확인할 수 있습니다.



공통​

와탭 2024년 3분기 공통 부분에서 이벤트 설정 메뉴 개선 및 새롭게 개편된 메트릭스 차트 New 메뉴를 소개합니다.

기존 이벤트 설정 및 새로운 이벤트 설정 New 메뉴에 검색 기능이 추가되었습니다. 이벤트 목록 오른쪽 상단의 검색 입력창을 통해 이벤트 이름, 지표 이름를 기준으로 문자열을 입력하거나 이벤트명 또는 이벤트 수신 태그 중 원하는 기준을 선택 후 문자열을 입력하여 이벤트를 검색할 수 있습니다.

Application, Kubernetes, Database 상품군 공통으로 메트릭스 차트 New 메뉴가 새롭게 개편되었습니다. 메트릭스 차트 New는 사용자 경험을 대폭 개선하여 모니터링 작업을 더욱 효율적으로 수행할 수 있도록 설계되었습니다.

사용자는 시간, 대상, 인터벌 설정을 한 번에 처리하고 쉽게 차트를 추가할 수 있습니다. 또한 대시보드 위젯의 시간 범위를 자유롭게 변경할 수 있으며 일괄 변경 기능과 프리셋 기능을 통해 설정 관리가 간편해졌습니다. 이와 더불어 대시보드 레이아웃과 위젯도 자유롭게 조정할 수 있어 차트 분석이 더욱 효율적으로 개선되었습니다.

다음 개요 안내를 통해 2024년 3분기 공통 부분의 신규 기능 및 주요 변경 사항을 해당 릴리스 버전과 함께 확인해 보세요.

  • Changed 계정 관리 또는 경고 알림 > 이벤트 수신 설정 메뉴에서 모바일 기기 알림 테스트 시 표시되는 성공 메시지 문구 변경 Service 2.7.0​

  • Changed 사이드 메뉴의 조직 목록에 조직 아이콘 노출하도록 변경 Service 2.7.0​

  • Changed 보고서 메뉴의 보고서 메일 발송 예약 창에서 시간 문구를 타임존이 포함된 문구로 안내하도록 변경 Service 2.7.0​

  • Feature 사이드 메뉴의 통합 검색에서 에이전트를 IP 주소로 검색할 수 있는 기능 추가 Service 2.7.0​

  • Feature 사이드 메뉴에서 전체 프로젝트를 선택해 프로젝트 목록을 펼치면, 프로젝트 검색 입력란에 자동 포커스되는 기능 추가 Service 2.7.0​

  • Feature 리치(Rich) 텍스트 위젯의 글꼴 색상 선택 옵션 추가 Service 2.8.0​

  • Feature 이벤트 설정 New 메뉴에서 이벤트명 또는 이벤트 수신 태그로 검색할 수 있는 기능 추가 Service 2.8.0​

  • Feature 이벤트 수신 설정 메뉴에서 PagerDuty를 통해 경고 알림을 받을 수 있는 서드 파티 플러그인 추가 Service 2.8.0​

  • Changed 통합 그룹 관리 버튼 위치 및 UI 변경 Service 2.8.0​

  • Feature 다음 메뉴 경로(관리 하위 프로젝트 멤버, 통합 멤버 관리, 멤버 목록)에서 멤버 목록을 CSV 형식의 파일로 다운로드할 수 있는 기능 추가 Service 2.8.0​

Application​

와탭 2024년 3분기 Application 부분의 주목할 만한 신규 기능 및 주요 변경 사항으로 공통 기능 개선 사항을 안내합니다.

와탭 애플리케이션 모니터링은 이번 3분기에 통계 메뉴에 특정 IP별로 호출한 URL 및 호출 건수를 조회할 수 있는 IP 트랜잭션 메뉴를 제공합니다. 또한 인스턴스 성능 관리 메뉴 하위의 스레드 목록/덤프 메뉴에 선택한 에이전트가 설치된 애플리케이션의 스레드 현황 조회 기능 및 CPU 증가량 컬럼이 추가되었습니다.

다음 개요 안내를 통해 2024년 3분기 Application 상품군의 신규 기능 및 주요 변경 사항을 해당 릴리스 버전과 함께 확인해 보세요.

  • Changed 사이트맵 > 토폴로지 > Netstat 토폴로지 메뉴의 예시 이미지를 영문으로 변경 Service 2.7.0​

  • Changed 인스턴스 성능 관리 > 힙 덤프 메뉴에서 힙 덤프 확보 관련 안내 메시지가 항상 표시되도록 변경 Service 2.7.0​

  • Changed 트레이스 분석 창의 테이블 뷰 및 트리 뷰 탭에서 트랜잭션 호스트 정보 없을 경우 포트 표시하지 않도록 변경 Service 2.7.0​

  • Feature 인스턴스 성능 관리 > 스레드 목록/덤프 메뉴의 테이블에서 CPU 시간 증가량을 표시하는 컬럼 추가 Service 2.7.0​

  • New 특정 IP별로 호출한 URL과 호출 건수를 조회할 수 있는 IP 트랜잭션 메뉴 추가 Service 2.8.0​

Server​

와탭 2024년 3분기 Server 부분의 주목할 만한 신규 기능으로 Unix HP-UX 설치 패키지 제공과 새로운 서버 인벤토리 메뉴에 대해 소개합니다.

와탭 서버 모니터링은 이번 3분기에 새롭게 서버 인벤토리 메뉴를 제공합니다. 서버 인벤토리는 서버의 물리적 특성 및 가상적 특성, 운영 환경 등을 체계적으로 관리하기 위한 기능입니다. 자동으로 수집된 서버 구성 요소와 사용자 정의 항목을 통해 서버의 구성을 상세하게 파악하고 관리하는 것이 가능합니다.

사용자 정의 항목을 통해 커스텀 컬럼을 구성하여 서버 상태 및 성능을 더욱 세밀하게 모니터링하고 이슈 상황에 대응할 수 있습니다. 예를 들어 각 서버 담당자와 직접적인 소통이 용이하도록 운영 담당자 정보를 추가하여 문제 시 빠른 대응이 가능합니다. 또한 서버 인벤토리 메뉴는 3단계의 다중 정렬을 통해 조건에 맞는 서버를 빠르게 필터링 할 수 있습니다. 기본 정렬은 OS 타입, 모델, 그리고 시리얼 정보 기준입니다.

Unix HP-UX 에이전트 설치 패키지를 제공합니다. HP-UX는 Hewlett-Packard(HP)에서 개발한 고성능 Unix 운영 체제로, Itanium 및 PA-RISC 시스템에서의 안정성과 성능으로 잘 알려져 있습니다.

와탭은 HP-UX Itanium 버전 11.23과 PA-RISC-UX 버전 11.23을 지원합니다. HP-UX 설치 패키지는 간단한 다운로드 및 실행만으로 쉽게 설치할 수 있으며, 와탭 서버 모니터링을 통해 HP-UX 운영 체제를 효과적으로 관리하고 모니터링할 수 있습니다.

Kubernetes​

와탭 2024년 3분기 Kubernetes 부분의 주목할 만한 신규 기능 및 변경 사항으로 새로운 노드 맵, 노드 타임라인, 노드 디스크 목록, Service 목록 및 컨테이너 요약 분석, Pod 요약 분석 기능을 소개합니다.

와탭 쿠버네티스 모니터링은 이번 3분기에 노드 모니터링 확장과 함께 여러 가지 새로운 기능을 제공합니다. 노드 맵, 노드 타임라인, 노드 디스크 목록과 같은 신규 메뉴가 추가되었으며, 기존 노드 목록 메뉴가 보다 효율적으로 개선되었습니다.

노드 맵을 통해 클러스터 관점에서 노드 상태를 모니터링할 수 있습니다. 노드의 실시간 상태, 리소스 사용 현황 및 노드 간 관계를 직관적으로 파악할 수 있으며, 그룹화, 임계치 설정, 라벨 등의 사용자 설정을 통해 사용자가 원하는 방식으로 대시보드를 구성할 수 있습니다.

노드 타임라인 메뉴에서 시간에 따라 노드의 상태 변화를 추적할 수 있습니다. 예를 들어 특정 노드의 제거 시점 및 해당 시점의 상태를 시각적으로 파악할 수 있습니다. 이를 통해 클러스터 운영 중 발생한 이슈 분석이나 과거 상황을 확인하는 데 유용합니다.

노드 디스크 목록을 통해 노드에 마운트된 개별 디스크의 상태를 실시간으로 모니터링할 수 있습니다. 그리고 기존 노드 목록 메뉴의 비교 및 상세 정보 조회 기능이 개선되었습니다. 노드 상세 메뉴로 이동하는 대신 선택한 노드의 구성 및 리소스, 메트릭스, 이벤트 정보를 노드 목록 상세 모달을 통해 확인할 수 있습니다.

또한 와탭 쿠버네티스 구성 요소 분석 기능이 강화되었습니다. 컨테이너 요약 분석 및 Pod 요약 분석 기능이 새롭게 제공되어, 각 구성 요소의 성능과 상태를 종합적으로 분석할 수 있습니다. 컨테이너 맵의 대상 정보 탭에서 원하는 요약 분석 정보를 확인할 수 있습니다.

더불어 와탭 쿠버네티스 모니터링 정보 확대의 일환으로 Service 목록 메뉴가 새롭게 추가되어, 각 서비스의 구성과 연결된 파드 상태를 확인할 수 있습니다. 조회 시간 기준 최근 1분간 수집된 서비스 정보를 제공합니다.

Database​

와탭 2024년 3분기 Database 부분에서 주목할 만한 신규 기능 및 변경 사항으로 Oracle Pro 정식 출시와 Oracle, Altibase, Cubrid 상품 V2 출시 및 새로운 스크립트 매니저 메뉴 추가, 멀티 인스턴스 모니터링 메뉴의 기능 개선에 대하여 소개합니다.

와탭 데이베이스 상품군이 V2로 전환되고 있습니다. 1분기 Redis와 MongoDB에 뒤이어 3분기에는 Oracle, Altibase, Cubrid V2 서비스가 시작되었습니다. V2는 데이터 전송팩의 구조가 변경되었으며 UI 역시 전반적으로 개선되었습니다.

Oracle Pro 정식 서비스가 시작되었습니다. 와탭 Oracle Pro는 데이터베이스 서버의 메모리에 직접 접근해 성능 지표를 모니터링하며, 최대 1초에 20번까지 최신 성능 정보를 수집합니다. 디스크 I/O를 거치지 않아 시스템 부하를 최소화하고, 쿼리 실행 시간과 버퍼 캐시 상태 등의 정밀한 성능 분석을 제공합니다.

그리고 실험실에서 새로운 스크립트 매니저 기능을 제공합니다. 스크립트 매니저는 SQL 스크립트를 관리할 수 있도록 도와줍니다. 사용자는 에이전트 설치 경로에 스크립트를 저장하고, 이를 모니터링 서비스에서 실시간으로 불러와 실행할 수 있습니다. 이를 통해 복잡한 데이터베이스 작업을 자동화하고 손쉽게 모니터링할 수 있습니다.

또한 멀티 인스턴스 모니터링 메뉴 개선과 함께 새롭게 사용자 정의형 위젯을 제공합니다. 대시보드에서 기본 제공되는 지표 외에 사용자가 원하는 지표를 추가할 수 있습니다.

다음 개요 안내를 통해 2024년 3분기 Database 상품군의 신규 기능 및 주요 변경 사항을 해당 릴리스 버전과 함께 확인해 보세요.
