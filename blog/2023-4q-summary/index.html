4분기 안내

2023년 12월 31일
TW J
테크니컬 라이터

와탭의 2023년 4분기 서비스 신규 기능 및 주목할 만한 변경 사항 개요를 안내합니다. 와탭이 새롭게 제공하는 기능들을 한 눈에 살펴보세요.

공통
  • 와탭 모바일 앱을 새롭게 출시했습니다.
  • 편의성 향상을 위해 상품 카탈로그 및 일부 메뉴의 화면을 개선했습니다.
  • 정비 계획 메뉴 V2를 제공합니다.
Application
  • 트레이스 분석 기능이 개편되었습니다.
  • 대시보드가 프리셋 방식으로 개편되었습니다.
Server

이벤트 수신 태그 옵션을 제공합니다.

Kubernetes
  • 컨테이너 맵 메뉴 화면을 개선했습니다.
  • 리소스 하위 목록 메뉴가 추가되었습니다.
Database

PostgreSQL과 MySQL 상품 V2 정식 전환 및 해당 V2 서비스 개선이 이루어졌습니다.

Browser

AJAX 히트맵 메뉴를 제공합니다.

상품별 신규 기능 외 변경 사항 개요 및 해당 릴리스 버전, 에이전트 변경 사항은 다음 안내를 통해 확인할 수 있습니다.



공통​

와탭 2023년 4분기 공통 부분의 주목할만한 신규 기능 및 변경 사항으로 와탭 모바일 앱의 출시, 화면 개선 사항 및 정비 계획 V2 제공에 대해 소개합니다.

이번 4분기에 와탭 모바일 앱이 새롭게 출시되었습니다. 앱을 통해 와탭 모니터링 서비스 대시보드 주요 차트를 확인하거나 실시간으로 알림을 받을 수 있습니다. 또한 OTP 기능을 활용해 로그인 시 2차 인증 도구로 활용할 수 있습니다. 향후 업데이트를 통해 지원 범위를 확대할 예정입니다.

더 나은 사용자 경험과 편의성 향상을 위해 서비스 화면을 개선했습니다. 와탭 모니터링 서비스의 정기 배포 소식 및 긴급 점검 등을 확인할 수 있는 서비스 공지 사항 기능이 추가되었고, 상품 카탈로그 화면이 개편되었으며, 사이드바 메뉴에서 상품 소개 정보를 확인할 수 있습니다.

그외 결제 정보 및 에이전트 설정 등 일부 메뉴 화면 역시 개선했습니다. 결제 정보 메뉴의 경우 결제 정보 입력 시 통화 선택을 제공합니다. 에이전트 설정 메뉴의 경우 옵션 목록을 간소화해 가독성을 높였으며, 옵션 작성 시 적용된 최종 옵션 항목만 보이도록 변경되었습니다.

기존 정비 계획을 개편한 신규 정비 계획 V2를 제공합니다. 신규 정비 계획을 통해 에이전트별로 정비 계획을 추가할 수 있고, 서버 프로젝트에서도 반복 옵션을 사용할 수 있습니다. 다만 기존 정비 계획과 신규 정비 계획은 호환되지 않습니다.

다음 개요 안내를 통해 2023년 4분기 공통 부분의 신규 기능 및 주요 변경 사항을 해당 릴리스 버전과 함께 확인해 보세요.

Applicatoin​

와탭 2023년 4분기 Application 상품군의 주목할 만한 신규 기능 및 변경 사항으로 히트맵 및 대시보드 고도화에 대해 소개합니다.

히트맵 고도화 작업의 일환으로 히트맵 드래그 시 트레이스 분석 창이 개편되었습니다. 멀티 트랜잭션 탭에 프로젝트 필터 기능이 추가되었고 또한 이제 트랜잭션 로그 탭에서 로그 데이터 미조회 시 상황에 알맞게 안내 메시지를 출력합니다. 예를 들어 로그 데이터가 없는 경우, 에이전트 미설정으로 수집을 하지 못 한 경우, 로그 비활성화 상태인 경우를 각각 확인할 수 있습니다.

애플리케이션 대시보드를 개편했습니다. 시간 선택자 옵션이 추가되어 실시간 뿐 아니라 과거 데이터를 조회할 수 있습니다. 그리고 대시보드 내 사용자가 원하는대로 위젯을 추가 및 삭제할 수 있고 크기 역시 조절할 수 있습니다. 편집한 위젯의 위치, 크기, 레이아웃을 저장해 불러올 수 있는 프리셋 기능이 추가되었습니다.

과거 데이터 조회
대시보드 프리셋

또한 일부 애플리케이션 중복 지표 명칭에 대한 통일화 및 표준화 작업이 이루어졌습니다. 예를 들어 URL, 트랜잭션, 트랜잭션 명으로 사용되던 지표명을 트랜잭션으로 통일했습니다. 또한 기존 HTTPC, HTTP 호출, HTTPC 건수 등으로 사용되던 지표명을 HTTP 호출 건수로 통일했습니다. 그외에도 다수 전반적인 용어 표준화가 이루어졌습니다.

Server​

와탭 2023년 4분기 Server 상품의 주목할 만한 신규 기능 및 변경 사항으로 서버 상세 및 이벤트 설정 메뉴 개선 사항을 소개합니다.

서버 상세 메뉴에서 차트 내 x축 표시 단위를 시간 기준으로 변경해 일자 기준 뷰의 가독성을 높였습니다.

이벤트 설정 메뉴 내 서버 및 프로세스 이벤트에 프로젝트 특정 멤버 또는 팀 대상으로 알림 수신 여부를 설정할 수 있는 이벤트 수신 태그 옵션이 추가되었습니다. 또한 기본 알림명을 수정할 수 있게 되었으며, 기본 알림의 경우 이제 기본 태그를 별도로 표시합니다.

  • Changed 이벤트 설정 메뉴에서 기본(Default) 알림의 이름을 수정할 수 있도록 변경하고, '기본' 태그를 별도 표시하도록 수정 Service 1.114.0​

Kubernetes​

와탭 2023년 4분기 Kubernetes 상품의 주목할 만한 신규 기능 및 변경 사항으로 컨테이너 맵 개선 및 추가된 리소스 하위 목록 메뉴에 대해 소개합니다.

와탭 쿠버네티스 모니터링의 컨테이너 맵은 지속적으로 고도화 작업이 이루어지고 있습니다. 이번 4분기에는 상세 보기 창에서 컨테이너 상 실행 중인 애플리케이션의 호출 흐름과 외부 호출 상태를 확인할 수 있는 호출 정보 탭이 새로 추가되었습니다.

또한 컨테이너와 Pod 뷰 모두 블록 색상 표시 지표에 애플리케이션 관련 필드 목록(app_counter)을 선택할 수 있습니다. 그리고 1분간 사라진 컨테이너 및 Pod의 블록을 표시하는 기능이 추가되었습니다.

뿐만 아니라 리소스 하위의 목록 메뉴가 개편 및 새로 추가되었습니다. 노드 목록 메뉴는 노드 정보 지표, 필터 검색 기능이 추가되었습니다. 또한 시간 갱신 간격이 1분에서 10초로 변경되었습니다. 클러스터와 관련된 버전 정보 및 각종 오브젝트 정보를 한 눈에 확인할 수 있는 클러스터 요약 메뉴가 새로 추가되었습니다.

새롭게 네임스페이스 프로젝트의 실험실에 Pending Pods 메뉴가 추가되었습니다. Pending Pods 메뉴는 Pending 상태에 놓인 Pod를 빠르게 파악하기 위한 실험 기능입니다.

다음 개요 안내를 통해 2023년 4분기 Kubernetes 상품의 신규 기능 및 주요 변경 사항을 해당 릴리스 버전과 함께 확인해 보세요.

Browser​

와탭 2023년 4분기 Browser 상품의 주목할 만한 신규 기능으로 AJAX 히트맵을 소개합니다.

Asynchronous JavaScript and XML(AJAX)는 비동기 방식으로 데이터를 교환해 웹 페이지를 업데이트하는 기술입니다. 사용자 경험을 개선하고 서버로의 요청을 줄이는 장점을 가진 AJAX는 JavaScript 코드로 구현되기에 모니터링이 필요합니다.

와탭 Browser 모니터링이 제공하는 AJAX 히트맵 메뉴를 통해 시간에 따른 AJAX 이벤트의 응답 시간을 히트맵 차트로 확인할 수 있으며 각 이벤트 구간별 세부 시간 정보, AJAX 요청 URL 및 호출 페이지 정보, 에러 메시지 및 상태 코드 등의 상세 정보를 조회할 수 있습니다.
