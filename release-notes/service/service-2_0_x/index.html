경고 알림​

  • 기존: 카테고리
  • 변경: 이벤트명, 카테고리, 레벨, 메시지

통계/보고서​

Fixed 보고서 메뉴에서 사용자가 설정한 종료 시간이 시작 시간보다 앞서 설정된 경우 종료 시간을 다음날 기준으로 조회하도록 수정

정보

예를 들어, 시작일 12월 03일, 시작 시간 05:00, 종료 시간 02:00으로 설정된 경우, 2023/12/03 05:00 ~ 2023/12/04 02:00 기간을 조회할 수 있습니다.
