Common​

  • Feature Occurrence of real-time notifications can also be checked through the browser.

  • Feature Added the horizontal line option to the TopN widget of Flex Board.

  • Feature In Event Configuration, added the event action time.

    • Event Setting V1

      Metrics/composite metrics/outlier detection/composite real-time log/server log → Added them above the event reception tag.

      Hitmap pattern → Added it at the bottom of the card.

      Basic/metrics → Added to the following items for the resolved notifications.

  • Feature When configuring the Alert > Event Configuration metrics events, added the metrics search feature in the category selection, metrics settings, and target event selection steps.

  • Changed In the chart tooltip of Flex Board, displayed the data for WriteLatency and ReadLatency up to 5 decimal places.

  • Changed In Flex Board, changed the tooltip position of Project from other widgets when configuring the widget Project selection.

  • Changed In Site management > Metrics > Rebuilding Metrics, modified so that re-request is impossible when a request has not been processed.

Application​

  • Feature Added the feature to select traces with keyboard arrow keys when the list is focused in the Trace analysis window.

  • Feature On the Active Transaction screen, a list of active transactions can be downloaded as CSV.

  • Feature Enhanced the Transaction map past data searching feature.

    • Extended the maximum searchable time range (10 minutes → 1 hour).

    • Adding filter items

      Added filters

      • General filters

        Login ID, WClientId, transaction ID, multi transaction ID, Referer, custom TID, error (including class, step, and message), city, country, HTTP method, user agent

      • Step filters
        DBC, SQL, HTTPC, METHOD, MESSAGE

      • Number input filter
        Elapsed time, HTTP call time, HTTP call count, SQL time, SQL count, status

    • Added display of total number of transactions (read items) searched by the filter data.

    • Added description tooltips for Read and Limit.

    • Added the pause feature when searching the past data.

      • When the search button is clicked while searching the past data (loading), searching is paused.

      • If you search again with the same conditions (time, filter), the data is retrieved from the stopped point.
        (when changing the search conditions, the chart is initialized and a new drawing begins)

  • Changed Changed the Analysis > Transaction Search CSV downloading method.

  • Fixed Fixed some query errors when trace information from other projects is included in the multi-transaction.

  • Fixed Fixed the issue where loading does not end intermittently when searching the Transaction map past data.
