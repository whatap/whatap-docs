Service 2.17.0​

May 28, 2025

Common​

  • Feature The Real-time Alert List and Project Member(Integrated Group Management) options and selection status are included in the URL so that they can be shared in the same state when sharing the link.

  • Feature Added the Report Excel downloading feature (provided only in some report templates).

  • Feature Added the feature to change the Flex Board sharing role.

  • Feature Added the Excel file downloading button for the Event Configuration New event configuration history.

  • Fixed When adding a new widget after dragging the Metrics Chart Old widget, modified to maintain the time selector settings.

  • Fixed Fixed the issue where the project tag in the Flex Board management widget cannot be clicked.

  • Fixed Fixed the bug for the Compare with previous time feature of the Flex Board widget.

  • Fixed Fixed the bug where organization information disappears when the Profile icon > WhataTap Monitoring Information menu is clicked on the upper right of the screen.

  • Fixed Fixed the individual time settings for series charts to be applied.

  • Fixed Applied resizing of the page table to add the linked project in.

Application​

  • Feature Added the opetion to display the Multi transaction Chart client IP information.

  • Feature Added the feature to display error properties out of Trace analysis > Timeline and Table view Message Step.

  • Feature Added the feature to display the table area loading when initial data is viewed in Transaction map and to display the vertical line loading if no data exists.

  • Feature Added the city filter into the filters that can be used in the stop state of Transaction map.

  • Feature Added the STEP_SQL_BIND and STEP_ALL filters in Transaction map (if used, a decryption key is required to be input).

  • Feature When viewing the profile data by dragging Analysis > Hitmap, added the feature to display a message indicating the possible period if the maximum search period is exceeded.

  • Feature Added the Transaction map widget and its setting items.

  • Fixed In Instance Performance Management > Agent Log, fixed the display error for the data between agents when the same agent log file is selected.

  • Fixed In Analysis > Stack, fixed the table column sharing issue between tabs.

  • Fixed Fixed the missing Trend comparison transaction name display issue in Statistics > Transaction.

  • Fixed Fixed the bug where all data is not displayed, if there is no initial data in real-time mode of Transaction map.

  • Deprecated The Topology and Custom Hitmap service are ended (2025/12/31).

Server​

  • Feature Added the Notifications > Webhook JSON variable.

  • Feature Provided the content drawer for the Server List > Server Detail page.

  • Feature Added the configuration popover in which the Y-axis maximum value in the charts in the Server List > Server Detail menu.

  • Feature In Alert > Event Configuration > Log File / Window Event, added the feature to add and modify Repeat (times) and Duration (seconds).

  • Feature Improved the display of Event ID in the event list of Alert > Event Configuration > Log File / Window Event.

  • Changed Disabled case sensitivity when searching for the Server List agent.

  • Changed Improved the Laboratory > Metrics Explorer metric list sorting criteria (predefined order has been applied).

  • Fixed Fixed the Server Name column display error in the Server name column under Dashboard > Group Summary > Events in progress tab and added the Agent Name column.

  • Fixed Fixed the Server Name column display error in the Server name column under Dashboard > Server Inventory Map menu > Display Detail's Events in progress tab and added the Agent Name column.

  • Fixed In Server List > Column settings, prevented duplicate entries when spaces are included.

  • Fixed In the Server List > Server Detail line chart panel, the Download button is hidden when there is no data in the query section.

  • Fixed In Server List > Server Detail > Server information, fixed the error where the Server Alert List event reception tag is wrongly displayed.

  • Fixed Fixed the issue with list staying open when adding the Metrics Explorer widget.

Kubernetes​

  • Changed In Workload > Deployment List, added new metrics (Kubernetes Agent 1.8.7).

    • Remove Columns

      • Replaced as ReplicaSets, TotalPods, Running Pods → PodPhase.

      • Replace TotalContainers and RunningContainers with ContainerState.

    • Additional Columns

      • PodPhase, ContainerState, Ready, deployUpdatedReplicas, deployAvailableReplicas, deployCreationTime

        Note

        It is exposed only in Kubernetes Agent 1.8.7 or later, and the previous names appear as before when using older versions of the agent.

  • Changed Changed to allow to modify the namespace project name.

Database​

Common​

  • Feature Added the column selection feature in the lock tree table.

    • Dashboard > Monitoring a Database Instance, Monitoring Multiple Instances
    • Analysis > Lock Tree

  • Feature Added the feature to download the chart data as a CSV file when the real-time mode is not applied to Dashboard > Monitoring a Database Instance.

  • Feature Added the feature to save the Dashboard > Monitoring Multiple Instances Disk Usage widget table status preset.

  • Feature Added the tags to display the instance's Replication Role or Cluster configuration.

    • Instance selector on the Dashboard > Monitoring a Database Instance screen

    • Instance tag on the Dashboard > Monitoring Multiple Instances screen

    • Role column of the Management > Agent List

  • Feature Improved the Dashboard > Monitoring a Database Instance New screen.

    • Added the click and drag events in the custom widget series chart.
    • Changed the chart option icon to the More button.
    • Added the guide text when a deactivated instance is selected.
    • Added the metric selection button in the default series chart.

  • Changed Changed the Dashboard > Monitoring a Database Instance widget's header icon to the More button.

  • Changed In Dashboard > Monitoring a Database Instance, the status colors for the Instance Selector and Instance List are displayed identically.

  • Changed In Stat/Report > Database size, Tablespace Size, if the selected dbName does not exist, no data in the table is displayed.

  • Fixed When the Dashboard > Monitoring a Database Instance New session table cannot be downloaded as a CSV file or while downloading the chart image, fixed the issue where the theme is not applied.

  • Fixed In case of Dashboard > Monitoring a Database Instance Multi DB, fixed the tag untagging error when clicking an instance again.

  • Fixed Fixed the index changing and data refreshing issue caused by Stat/Report > Database size, Tablespace Size dbName.

MongoDB, MySQL, Oracle, PostgreSQL, SQL Server​

Feature Added the Cloud agent for DB agent installation.

MySQL​

  • Feature In the Lock Tree table, added the guide text for the lock query role.

    • Dashboard > Monitoring a Database Instance

    • Analysis > Lock Tree

  • Feature In the Stat/Report > Table increment summary section, added the increase/decrease rate column in the Capacity details for each table table.

  • Changed In Management > Agent installation, provided the guide that the lock query role has been added when MySQL 8 or later is used.

Oracle​

Feature Added the guide for multi-tenant DB kill session command to the installation page.

Oracle Pro​

Feature In Laboratory > Script Manager, added the feature to check the execution results of the multi-script and instance-based multi-script.

PostgreSQL​

Feature In Monitoring a Database Instance, Monitoring Multiple Instances, and Lock Tree lock tree tables, added the holder_relation, waiter_relation, and state columns.

Cloud​

Feature When querying the profile data by dragging Analysis > Application Hitmap and Multi-service Hitmap, displayed the notification, if the maximum query period exceeds.

Log​

  • Changed In Live Tail, Log Search > Filter favorites, displayed the toast message when adding/modifying/deleted favorites, and avoided entering spaces in favorite names.

  • Changed In Live Tail, Log Search > Filter favorites, fixed the issue where adding a personal filter to favorites with no favorites specified causes infinite loading.

  • Changed In Log view detail setting > Tag Management, displayed all tags in the log list by default.
