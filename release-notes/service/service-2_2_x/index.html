Service 2.2.0​

2024년 02월 28일

신규 기능 출시​

Service 2.2.0 릴리스에 포함된 신규 기능을 소개합니다.

공통​

  • New 사이드 메뉴 개편

    사이드 메뉴를 새단장을 했습니다. 기존에 프로젝트 목록과 프로젝트 메뉴가 상하로 분리된 UI 구조에서 메뉴 집중형 구조로 개편했습니다. 서비스 화면에서 변경된 사항을 확인하려면 사이드 메뉴 아래에 튜토리얼 버튼을 선택하세요.

    • 사이드 메뉴 개편에 대한 자세한 내용은 다음 링크를 참조하세요.

    • 사이드 메뉴에 대한 자세한 내용은 다음 문서를 참조하세요.

  • New SSO 계정 연동 기능 추가

    화면 오른쪽 위에 프로필 아이콘을 선택한 다음 계정 관리 > SSO 연동을 선택하세요. SSO 계정 연동에 대한 자세한 내용은 다음 문서를 참조하세요.

  • New 메트릭스의 필드(Field) 중심의 사용자 경험을 강화한 이벤트 설정 New 기능 추가

    • 새로운 이벤트 설정 메뉴에 대한 자세한 내용은 다음 문서를 참조하세요.

    • 이 기능은 사이트맵 > 경고 알림 > 이벤트 설정 New 경로에서 확인할 수 있습니다.

Kubernetes​

  • New 마스터 및 노드 에이전트 설치 현황 확인할 수 있는 쿠버네티스 에이전트 목록 메뉴 추가

    이 기능은 클러스터 프로젝트 전용이며, 관리 > 쿠버네티스 에이전트 목록 메뉴 경로에서 확인할 수 있습니다. 자세한 내용은 다음 문서를 참조하세요.

  • New kube-apiserver의 상태 및 성능 메트릭 수집, 응답 시간, 처리량, 요청 실패 등을 추적하여 클러스터의 가용성과 성능을 모니터링할 수 있는 kube-apiserver 대시보드 메뉴 추가

    이 기능은 클러스터 프로젝트 전용이며, 클러스터 > kube-apiserver 대시보드 메뉴 경로에서 확인할 수 있습니다. 자세한 내용은 다음 문서를 참조하세요.

공통​

  • Feature 프로필 아이콘 선택 > 계정 관리 메뉴의 멀티팩터 인증 섹션에서 OTP 인증 수단을 분실한 경우를 대비해 복구 코드 기능 추가

    복구 코드를 이용해 인증하는 방법에 대한 자세한 내용은 다음 문서를 참조하세요.

프로젝트 생성​

  • Changed 프로젝트 생성 개수가 제한될 때 표시되는 안내 메시지를 모달 방식으로 변경
  • Changed 프로젝트 생성 시 상품 추천 배너에 Network Performance Monitoring 추가, Java 상품 제거

Flex 보드 / 대시보드​

  • Feature 그룹 위젯 복사 기능 추가
  • Feature 대시보드의 Apdex * 100 위젯에 복수 프로젝트 병합 메트릭스 항목 추가
  • Feature 위젯에 해당 제품의 대시보드로 이동 가능한 버튼 추가
  • Changed 위젯의 창모드 버튼 아이콘 변경( → )
  • Changed 클라우드 관련 지표에 대하여 5분, 1시간 통계 카테고리를 사용할 수 있도록 변경: aws_ / azure_ / ncloud_ / oci_
  • Fixed 이벤트 타임라인 위젯의 이벤트 횟수가 간헐적으로 실제 이벤트 발생 횟수와 다르게 증가하는 현상 수정
  • Fixed 대시보드 불러오기를 실패할 경우 대시보드 목록으로 이동하도록 수정
  • Fixed 대시보드의 차트에서 최댓값 표시 시 가장 높은 값의 라벨 텍스트가 잘리는 문제 수정
  • Fixed 차트에 표시되는 Y축 임계치 라벨 텍스트가 잘리는 문제 수정

경고 알림​

  • Feature 경고 알림 > 이벤트 설정 메뉴에서 메트릭스 이벤트를 추가 또는 편집 시 카테고리 항목의 제품 로고에 적용된 다크 테마 시인성 개선
  • Fixed 이벤트 수신 설정 메뉴에서 알림 설정 권한이 없는 사용자는 대량 알림 발생 방지 패널의 문자알림 중단 해제 버튼을 비활성화 상태로 수정
  • Fixed URL을 제외한 모든 상품의 이벤트 설정 메뉴에서 메트릭스 이벤트 설정 시 알림 규칙 테스트 차트의 데이터 시점이 하나만 있으면 차트가 그려지지 않는 이슈 수정
  • Fixed 이벤트 수신 설정 메뉴의 사용자별 이벤트 수신 설정 섹션 테이블에서 멤버 이름이 너무 길면 테이블의 열이 밀리는 문제 수정

이용 내역​

  • Changed 프로젝트 유료 전환 메뉴에서 결제 권한 없는 프로젝트 표시 문구를 모든 유료 프로젝트 표시로 변경, 툴팁 추가
  • Changed 결제 정보 메뉴에서 카드 결제 수단 등록 시 약관 동의 문구 변경

관리​

  • Changed 통합 멤버 관리 메뉴에서 결제 권한 및 멤버 관리 권한을 소유한 유일한 멤버를 삭제할 경우 경고 메시지를 표시하도록 수정
  • Fixed 통합 멤버 관리 메뉴에서 조직 하위 그룹의 멤버를 삭제할 수 없도록 삭제 버튼 비활성화
  • Fixed 통합 멤버 관리 메뉴에서 멤버 목록 테이블의 너비 수정

  • Fixed 계정 정보 메뉴에서 회원 탈퇴 불가 사유 복사 시 태그까지 복사되는 문제 수정
