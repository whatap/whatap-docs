Service 2.22.0​

2025년 10월 29일

공통​

  • New OpenMetrics 추가

  • Changed 라이센스 키가 없는 경우 표시 문구 수정

Application​

  • Feature 관리 에이전트 목록, CSV 다운로드 추가

  • Changed 통합 Flex 보드 액티브 트랜잭션 TopN 위젯에 여러 프로젝트를 선택할 수 있도록 변경

  • Changed 분석 > 성능 추이

    • 3일 초과 조회 시 동시접속 사용자 대신 방문자 지표 조회
    • 조회 버튼 누르기 전 조회하지 않도록 변경

  • Fixed 통계 SQL, 트랜잭션 컬럼 안내 아이콘/툴팁 복구

Server​

  • Feature GPU 인벤토리 테이블의 custom 컬럼 헤더에 custom 태그 표시하여 구분하기 쉽게 개선

  • Feature 이벤트 설정 > 파일 키워드 감시 / 윈도우 이벤트 감시 / 파일 감시 탭 > 파일 감시 이벤트 설정 기능 추가

  • Changed 이벤트 설정 > 기존 로그 파일, 윈도우 이벤트 → 파일 키워드 감시, 윈도우 이벤트 감시로 변경

  • Fixed GPU 성능요약 BAR1_Total_Memory 필드값에 comma 포매팅 적용되지 않던 이슈 수정

Kubernetes​

  • Feature GPU 트렌드 기능 추가

    • 범례에 단위(%) 표시

    • 차트 셀에 값 표시

    • 정렬 기능 추가

    • 필터 기능 추가

    • 프리셋 기능 추가(정렬, 필터 값 저장)

  • Feature Pod 목록에 와탭 커스텀 GPU 지표 추가

  • Feature GPU 대시보드 맵 차트 클릭 시 열리는 드로워의 메트릭스 탭에서 원하는 시간을 지정하여 메트릭스를 조회하는 기능 추가

  • Changed GPU 대시보드 맵 차트에서 도형이 아주 작아지는 경우도 M/P 라벨을 표시하도록 변경

  • Fixed 워크로드 Ingress 목록에서 일부 Service 정보가 연결되지 않는 문제 수정

  • Deprecated 컨테이너 맵에서 전체 단위의 상세 드로워 제거

Database​

공통​

  • New 메트릭스 추이 분석 추가

  • Feature 대시보드 > 카운트 추이 비교 위젯을 이동하거나 크기를 변경할 수 있도록 추가

  • Feature 대시보드 위젯 개선

    • [XOS] CPU, [XOS] MEM 위젯에 범례 표시 옵션 추가

    • 사용자 정의 위젯의 이퀄라이저 차트가 퍼센트 지표를 표시할 때 y축을 100으로 고정하도록 변경

    • 사용자 정의 위젯에서 기본으로 병합 없음이 선택되도록 변경

    • 사용자 정의 위젯의 대상 병합을 에이전트 병합으로 명칭 변경

    • 사용자 정의 위젯에 범례 표시 및 표시 여부 옵션 추가

  • Feature 분석 > 데이터베이스 파라미터 파라미터 변경 내역을 조회할 수 있는 기능 추가

  • Changed 인스턴스 셀렉터 동작 개선

  • Changed 스크립트 매니저 보안 요건에 따라 노출 조건을 수정하여, 개발기 및 설치형 환경만 표시되도록 변경

  • Fixed 페이지 진입 시 간헐적으로 인스턴스가 선택되어 있지 않은 현상 수정

  • Fixed 대시보드 > 인스턴스 모니터링 New

    • 1일 이상 조회 후 시리즈 차트 드래그 시 차트 데이터가 5분 통계 데이터로 유지되는 현상 수정

    • 화면의 일간 비교 위젯의 소숫점이 너무 길게 나타나는 현상 수정

  • Fixed 대시보드 > 인스턴스 모니터링 액티브 세션 테이블 필터 선택 시 스케일이 변동되는 현상 수정

  • Fixed 대시보드 지표 선택기에서 이벤트 버튼을 클릭해도 이벤트 설정 페이지로 이동하지 않는 현상 수정

  • Fixed 분석 > 스크립트 매니저 스크립트 목록에 스크롤이 생성되지 않는 현상 수정

MySQL, PostgreSQL, SQL Server​
