Service 2.13.0​

2025년 01월 22일

신규 기능 출시​

Service 2.13.0 릴리스에 포함된 신규 기능을 소개합니다.

Database​

New IBM DB2 모니터링 상품 출시

노트

IBM DB2 모니터링 상품에 대한 자세한 내용은 다음 문서를 참조하세요.

Server​

  • New 과거 성능 데이터를 보다 직관적으로 분석할 수 있는 메트릭스 익스플로러 메뉴 추가

    • 서버 인벤토리 메뉴의 카테고리 정보 추가

    • 메트릭스 익스플로러 기능 안내 모달 추가

    노트

    이 기능은 실험실 > 메트릭스 익스플로러 메뉴 경로에서 확인할 수 있습니다. 자세한 내용은 다음 문서를 참조하세요.

Kubernetes​

  • New 쿠버네티스의 시스템 컴포넌트 중 하나인 kube-scheduler에 대한 모니터링 데이터를 조회할 수 있는 kube-scheduler 대시보드 메뉴 추가

    노트

    • 이 기능은 클러스터 > kube-scheduler 대시보드 메뉴 경로에서 확인할 수 있습니다. 쿠버네티스 에이전트 1.8.1 버전 이상에서 지원합니다.

    • kube-scheduler 대시보드 메뉴에 대한 자세한 내용은 다음 문서를 참조하세요.

  • New 시간의 흐름에 따른 컨테이너 상태를 확인할 수 있는 컨테이너 타임라인 메뉴 추가

    노트

    이 기능은 컨테이너 > 컨테이너 타임라인 메뉴 경로에서 확인할 수 있습니다. 자세한 내용은 다음 문서를 참조하세요.

  • New 시간의 흐름에 따른 Pod 상태를 확인할 수 있는 Pod 타임라인 메뉴 추가

    노트

    이 기능은 워크로드 > Pod 타임라인 메뉴 경로에서 확인할 수 있습니다. 자세한 내용은 다음 문서를 참조하세요.

  • New 클러스터 하위에 Persistent Volume 관련 신규 메뉴 추가

    • 클러스터 > Persistent Volume 목록

    • 클러스터 > Persistent Volume Claim 목록

    노트

    • 이 기능은 쿠버네티스 에이전트 1.8.1 버전 이상에서 지원합니다.

    • Persistent Volume 목록 및 Persistent Volume Claim 목록 메뉴에 대한 자세한 내용은 다음 문서를 참조하세요.

공통​

  • Feature 분석 > 메트릭스 차트 New 메뉴의 기능 개선

    • 차트 데이터에 마우스 호버 시 힌트 데이터의 Max 값이 기록된 특정 시간 추가

    • 범례 목록 전체 선택/해제 버튼 추가

    • 차트 라인의 컬러 선택 기능 추가

    • 더보기 메뉴에 CSV 복사 버튼 추가

    • 더보기 메뉴에 차트 이미지 복사 버튼 추가

    노트

    메트릭스 차트 New 메뉴에 대한 자세한 내용은 다음 문서를 참조하세요.

  • Feature 경고 알림 > 이벤트 수신 설정 메뉴에서 서드파티 플러그인을 통한 알림 수신 레벨 선택 기능 추가

  • Feature 경고 알림 > 이벤트 기록 메뉴와 실시간 알림의 링크 변경 및 추가

    노트

    이미 적용된 Server, Database 상품과 마찬가지로, Application, Kubernetes, Browser, URL 상품의 알림에 관련 메뉴로 이동할 수 있는 링크를 추가했습니다.

  • Changed 통합 보고서 메뉴의 시간 선택자(타임 셀렉터) 기능 교체

    • 일간 보고서: 시작과 종료 날짜, 시간을 선택할 수 있습니다.

    • 주간 보고서: 시작 날짜만 선택하면 종료 날짜를 계산해 자동 선택합니다.

    • 월간 보고서: 시작 날짜만 선택하면 종료 날짜를 계산해 자동 선택합니다.

      • 시작 날짜가 1일인 경우: 해당 월의 월말까지 자동으로 선택합니다.

      • 시작 날짜가 1일이 아닌 경우: 다음 달의 전일까지 자동으로 선택합니다.

    노트

    예시

    • 시작날짜가 1월 1일이면, 1월 1일 00시 00분 00초부터 1월 31일 23시 59분 59초를 자동 선택합니다.

    • 시작날짜가 1월 10일이면, 1월 10일 00시 00분 00초부터 2월 9일 23시 59분 59초를 자동 선택합니다.

  • Fixed 통합 Flex 보드 또는 Flex 보드 메뉴의 알림 목록 위젯에 데이터가 없는 경우 해소된 이벤트 보기 스위치 UI가 표시되지 않는 문제 수정

  • Fixed 이용 내역 > 선납금 메뉴에서 목록이 표시되지 않는 문제 수정

  • Fixed 프로젝트 목록 검색 시 검색 결과 내 상품 아이콘 정렬 수정

Application​

  • Feature 대시보드 > 트랜잭션 맵 메뉴에서 차트의 Y축 최댓값을 직접 입력해서 조절할 수 있는 기능 추가

  • Changed 대시보드 > 트랜잭션 맵 메뉴에서 또는 버튼으로 Y축 최댓값을 조정할 때, 30초를 넘어가면 10초 간격으로만 조절할 수 있는 기능 개선

    • 10초에서 50초 구간은 10초 간격으로 이동

    • 50초에서 300초는 50초 간격으로 이동

    • 300초 이상은 100초 간격으로 이동

    노트

    트랜잭션 맵 메뉴에 대한 자세한 내용은 다음 문서를 참조하세요.

  • Feature 통계 > 트랜잭션 메뉴에서 트랜잭션 건수를 기준으로 계산한 평균 HTTP 호출 시간, 평균 SQL 시간, 평균 SQL 패치 시간 컬럼 추가

    노트

    변경 사항은 다음을 참고하세요.

    컬럼 이름 변경

    • 평균 HTTP 호출 시간 → HTTP 호출 당 평균 시간: 총 HTTP 호출 시간 / 트랜잭션에서 발생한 총 HTTP 호출 건수

    • 평균 SQL 시간 → SQL 당 평균 시간: 총 SQL 시간 / 트랜잭션에서 발생한 총 SQL 건수

    • 평균 SQL 패치 시간 → SQL 패치 당 평균 시간: 총 SQL 패치 시간 / 트랜잭션에서 발생한 총 SQL 패치 건수

    새로 추가된 평균값 컬럼 항목

    • 평균 HTTP 호출 시간: 총 HTTP 호출 시간 / 트랜잭션 발생 건수

    • 평균 SQL 시간: 총 SQL 시간 / 트랜잭션 발생 건수

    • 평균 SQL 패치 시간: 총 SQL 패치 시간 / 트랜잭션 발생 건수

  • Feature 통계 하위 메뉴에서 테이블의 컬럼 추가, 컬럼 설정 기능을 통해 컬럼을 추가하면 다운로드한 CSV 파일에도 노출하도록 개선

    하위 메뉴별로 추가한 컬럼은 다음을 참조하세요.

    • 도메인: 합계 시간 (ms)

    • 에러: 에이전트, 에이전트 서버, 에이전트 종류, 최초 생성 시간

    • HTTP 호출: 표준 편차 (ms)

    • 로그인: 합계 시간 (ms)

    • 리퍼러: 합계 시간 (ms)

    노트

    최초 생성 시간: 조회 기간 중 트랜잭션의 에러가 최초 생성된 통계시간(5분 또는 1시간)입니다.

  • Changed 사이트맵 > Netstat 토폴로지 메뉴의 초기 화면에 안내 메시지 추가

  • Fixed 통합 Flex 보드 메뉴에서 액티브 트랜잭션 TopN 위젯을 클릭하면 발생하는 에러 수정

  • Fixed 트레이스 분석 창의 테이블 뷰 탭에서 SQL 스텝을 확인할 때, SQL 문의 치환 변수 또는 바인드 변수 중 하나라도 해제하면 전부 해제되는 문제 수정

  • Fixed 분석 > 스택 메뉴에서 체크박스가 일부 잘려 표시되는 문제 수정

  • Fixed 조직을 선택한 상태에서 분석 > 트랜잭션 검색 메뉴로 진입하면 시간 선택자에서 발생하는 에러 수정

Server​

  • Feature 서버 목록 > 서버 인벤토리 메뉴에서 서버의 활성 상태를 표시하는 기본 컬럼 추가

  • Changed Server 모니터링 상품 소개 페이지의 콘텐츠 변경

  • Changed 대시보드 > 서버 인벤토리 맵 메뉴의 그룹 요약 패널과 상세 보기에서 서버 인벤토리 테이블을 가로 스크롤할 때, 서버 이름 컬럼이 고정되도록 개선

  • Changed 서버 목록 > 서버 인벤토리 메뉴에서 OS 타입 컬럼에 운영체제별 아이콘 표시

  • Fixed 서버 목록 > 서버 인벤토리 메뉴의 컬럼 설정에서 서버 이름 컬럼을 체크 해제할 수 있는 문제 수정, 기본 컬럼으로 변경

    노트

    기본 컬럼은 컬럼 설정에서 추가하거나 삭제할 수 없으며 순서를 변경할 수 없습니다.

  • Deprecated 서버 목록 > 서버 인벤토리 메뉴에서 삭제 기능 제거

    노트

    서버 목록 메뉴와 연동하여 서버를 삭제할 수 있습니다.

Kubernetes​

  • Feature 대시보드 > 컨테이너 맵 메뉴의 대상 정보 탭에서 Pod 시작 분석 정보도 확인할 수 있는 기능 추가

  • Feature 대시보드 > 컨테이너 맵 메뉴의 대상 정보 탭에서 Volume 정보를 확인할 수 있는 기능 추가

  • Feature 컨테이너 > Out Of Memory Killed 컨테이너 메뉴에서 연관 메뉴로 이동할 수 있는 링크 제공

    • podName: 워크로드 > Pod 목록

    • containerId: 컨테이너 > 컨테이너 목록

    • onodeName: 클러스터 > 노드 목록

  • Feature 클러스터 > kube-apiserver 대시보드 메뉴의 위젯이 관리형 쿠버네티스일 경우 데이터 없음에 대한 안내 추가

  • Feature 관리 > 에이전트 설치 메뉴에서 kube-scheduler 기능 활성화 안내 추가

  • Changed 다음 메뉴에서 요약 분석 기능을 탭으로 분리

    • 대시보드 > 컨테이너 맵
    • 대시보드 > 노드 맵
