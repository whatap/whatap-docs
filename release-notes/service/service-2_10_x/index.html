Service 2.10.0​

2024년 10월 30일

신규 기능 출시​

Service 2.10.0 릴리스에 포함된 신규 기능을 소개합니다.

Kubernetes​

New Ingress 리소스에 대한 메타데이터와 관련 정보를 모니터링할 수 있는 Ingress 목록 메뉴 추가

노트

클러스터 > Ingress 목록 경로에서 확인할 수 있습니다.

Database​

  • New 클라우드 데이터베이스 서버의 자원현황을 수집하고 모니터링할 수 있는 클라우드 설정 메뉴 추가

    노트

    • 지원하는 클라우드 서비스 및 상품은 다음과 같습니다. V2 프로젝트에만 지원합니다.

      • Azure Monitor: PostgreSQL, MySQL

      • AWS CloudWatch: PostgreSQL, Oracle, MySQL

    • 관리 > 클라우드 설정 경로에서 확인할 수 있습니다. 자세한 내용은 다음 문서를 참조하세요.

  • New MySQL, PostgreSQL 플랫폼의 V2 프로젝트에서 Java 플랫폼 프로젝트와 실시간 세션 연계 분석 기능 추가

    노트

    연계 분석에 대한 자세한 내용은 다음 문서를 참조하세요. MySQL, PostgreSQL

Browser​

New 페이지 URL 경로별 로드 성능 정보에 대한 통계를 제공하는 페이지 로드 통계 메뉴 추가

노트

이 기능은 통계 > 페이지 로드 통계 메뉴 경로에서 확인할 수 있습니다. 자세한 내용은 다음 문서를 참조하세요.

공통​

  • Feature 경고 알림 > 이벤트 기록 메뉴에서 이벤트 처리 내역 기능 추가, 이에 따른 UI 변경

    노트

    Server 상품의 기능을 모든 상품으로 확대 적용합니다. 이벤트 처리 내역 기능 추가에 대한 자세한 내용은 다음 문서를 참조하세요.

  • Changed 통합 Flex 보드 또는 Flex 보드 메뉴에서 대시보드가 없거나 목록을 불러오는 중, 표시하는 안내 메시지의 세로 정렬을 가운데로 변경

  • Changed 경고 알림 > 이벤트 수신 설정 메뉴의 서드 파티 플러그인 추가에서 Webhook JSON 입력 필드 내용 최신화

    • uuid, okind, onode 키 추가

    • Server 상품의 경우 serverAlertPolicyName 키 추가

    노트

    서드 파티 플러그인 추가에 대한 자세한 내용은 다음 문서를 참조하세요.

  • Changed 분석 > 메트릭스 차트 New 메뉴에서 카테고리와 필드 이름으로 지표를 검색할 수 있도록 수정

  • Changed 경고 알림 > 이벤트 설정 New 메뉴의 메트릭스 탭에서 이벤트 추가 및 수정, 이벤트 목록에 새로운 UI 적용

  • Changed 언어 설정 메뉴의 아이콘 변경

  • Fixed 대시보드 > Flex 보드 메뉴에서 대시보드가 반응형 레이아웃인 경우. 인쇄 모드에서 위젯이 표시되지 않는 문제 수정

  • Fixed 상품 소개 화면에서 시작하기 버튼을 클릭하면 간헐적으로 선택한 조직이 해제되는 문제 수정

  • Fixed 시간 선택자에서 시간 단위 제한이 있는 경우 일본어의 메시지 표기 오류 수정

    選択できる最大時間は3時です。 → 選択できる最大時間は3時間です。

Application​

  • Feature Java 플랫폼 프로젝트에서 MySQL, PostgreSQL 프로젝트와 실시간 및 과거 트랜잭션 연계 분석 기능 추가

    노트

    연계 분석에 대한 자세한 내용은 다음 문서를 참조하세요.

  • Fixed 통계 하위 메뉴에서 에이전트 별칭이 있는 경우 검색되지 않는 문제 수정

  • Fixed 통계 하위 메뉴에서 에이전트 검색 필드 오른쪽에 위치한 선택 버튼 기능이 적용되지 않는 문제 수정

  • Fixed 트레이스 분석 창의 테이블 뷰 탭에서 액티브 스택 버튼을 선택하면 발생하는 화이트 아웃 현상 수정

  • Fixed 통합 Flex 보드 또는 대시보드 > Flex 보드 메뉴에서 일부 환경에 따라 PDF 파일을 다운로드할 때 비정상적으로 그려지는 문제 수정

Server​

  • Changed 관리 > 에이전트 설치 메뉴에서 Solaris 운영체제 환경의 설치 안내 변경

    • Solaris 버전별 다운로드 링크 제공

    • install.sh 스크립트 파일을 사용한 설치 방식으로 변경

  • Fixed 언어 설정이 일본어일 때, 서버 목록 > 서버 인벤토리 메뉴에서 필터 입력란의 안내 문구가 잘려서 표시되는 문제 수정

  • Fixed 서버 목록 > 서버 인벤토리 메뉴에서 사용자 정의 필드로 입력한 값이 간헐적으로 저장되지 않는 문제 수정

  • Fixed 로그 > 로그 설정 메뉴의 로그모니터링 시작하기에서 리눅스(shell) 에이전트 설정 명령어 수정

  • Fixed 관리 > 에이전트 설치 메뉴에서 HP-UX 운영체제 환경의 에이전트 설치 내용 중 오타 및 링크 오류 수정

  • Deprecated 서버 목록 > 서버 인벤토리 메뉴의 컬럼 설정 기능 중 정렬 옵션 제거

Kubernetes​

  • Feature 다음 메뉴에서 노드 요약 분석 기능 추가

    • 대시보드 > 컨테이너 맵

    • 대시보드 > 노드 맵

    노트

    이 기능은 쿠버네티스 에이전트 마스터 1.4.1 버전 이상에서 지원합니다.

  • Feature 다음 메뉴 경로에 프로세스 정보 추가

    • 대시보드 > 노드 맵

    • 클러스터 > 노드 목록 > 상세 보기

    노트

    이 기능은 클러스터 프로젝트에만 적용되며, 쿠버네티스 에이전트 1.7.14 버전 이상에서 지원합니다.

  • Feature 클러스터 > 노드 목록 메뉴에 신규 컬럼 추가

    • 노드 정보: AllocatablePods, DiskPressure, MemoryPressure, PIDPressure, Unschedulable, HasKubelet

    • CPU: AllocatableCpu

    • 메모리: AllocatableMemory

    • 워크로드 > Pending Pod 현황

    • 분석 > Pod 시작 분석

  • Feature 대시보드 > 클러스터 요약 메뉴에 노드 상태별 개수와 개수 추이를 표시하는 위젯 추가

  • Changed 대시보드 > 클러스터 요약 메뉴에서 다음 위젯의 지표 변경

    • 클러스터 프로젝트의 쿠버네티스 클러스터 통합 퍼포먼스 위젯

    • 네임스페이스 프로젝트의 쿠버네티스 네임스페이스 퍼포먼스 위젯

  • Changed 대시보드 > 컨테이너 맵 메뉴의 컨테이너 요약 분석에 컨테이너 상태 및 애플리케이션 정보 보충

  • Changed 컨테이너 > 컨테이너 목록 메뉴에서 Deployment 컬럼을 통해 Deployment 목록 메뉴로 이동하면 선택한 Deployment 상세 정보가 기본으로 열리도록 수정

  • Changed 컨테이너 > 컨테이너 목록 메뉴에서 비교하기 기능의 UI 변경

  • Changed 클러스터 > 노드 타임라인 메뉴에서 최소 선택할 수 있는 시간 단위를 5분으로 고정하도록 수정

  • Changed 클러스터 > 노드 디스크 목록 메뉴에서 OnodeName 컬럼의 을 선택하면 노드 상세 메뉴로 이동하도록 수정

공통​
