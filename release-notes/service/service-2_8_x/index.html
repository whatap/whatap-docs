Service 2.8.0​

2024년 08월 28일

신규 기능 출시​

Service 2.8.0 릴리스에 포함된 신규 기능을 소개합니다.

공통​

New 분석 > 메트릭스 차트 New 메뉴 추가

  • 사용자가 시간, 대상, 인터벌 옵션을 설정한 다음 모니터링하려는 대상의 지표를 특정하고 차트를 추가할 수 있도록 사용자 경험(UX)을 개선했습니다.

  • 대시보드에 배치한 위젯을 삭제하지 않고 차트 조회 시간 범위를 변경할 수 있습니다.

  • 대시보드에 배치한 모든 위젯의 옵션을 일괄로 적용할 수 있는 일괄 변경 기능을 제공합니다.

  • 프리셋 기능을 통해 대시보드에 배치한 위젯의 정보를 저장하고 불러올 수 있습니다.

  • 제공하는 다양한 옵션을 통해 대시보드의 레이아웃 및 위젯의 기능을 조정할 수 있습니다.

노트

  • 이 기능은 Application, Kubernetes, Database 제품에 적용됩니다.

  • 메트릭스 차트 New 메뉴에 대한 자세한 내용은 다음 문서를 참조하세요.

  • 개편된 사항에 대한 자세한 내용은 다음 링크를 참조하세요.

Application​

  • New Node.js 제품에 로그 모니터링 기능 추가

    노트

    Node.js 애플리케이션 모니터링 환경에서 로그 모니터링 기능을 활성화하는 방법에 대한 자세한 내용은 다음 문서를 참조하세요.

  • New 특정 IP별로 호출한 URL과 호출 건수를 조회할 수 있는 IP 트랜잭션 메뉴 추가

    노트

    • 이 기능은 통계 > IP 트랜잭션 메뉴 경로에서 확인할 수 있습니다.

    • IP 트랜잭션 메뉴에 대한 자세한 내용은 다음 문서를 참조하세요.

Database​

New Altibase V2 서비스 시작!

노트

Altibase 모니터링 V2의 새로운 기능에 대한 자세한 내용은 다음 문서를 참조하세요.

공통​

대시보드

Changed 세로 이퀄라이저 바 차트에서 개별 그래프의 라벨 표시 제거

통합 Flex 보드 / Flex 보드​

  • Feature 메트릭스 타입 필터 내 조건 간 OR 선택지 추가

  • Feature 리치(Rich) 텍스트 위젯의 글꼴 색상 선택 옵션 추가

  • Fixed 제목을 수정할 수 없는 위젯에 수정 버튼이 나타나는 문제 수정

  • Fixed 위젯 템플릿 목록에서 제품 카테고리 하위에 표시할 위젯 목록이 없을 때 펼침/접힘 아이콘이 변경되지 않는 문제 수정

  • Fixed 모든 메트릭스로 위젯을 추가하면 지표에 대한 설명이 위젯 제목으로 표시되는 문제 수정

  • Fixed 리치 텍스트 위젯의 수정 화면을 열고 닫는 과정에서 저장하지 않은 내용이 유지되는 문제 수정

  • Fixed 리치 텍스트 위젯의 수정 화면에서 텍스트와 툴바 버튼의 툴팁이 겹쳐 보이지 않도록 툴팁 요소에 테마 설정

  • Fixed 통합 Flex 보드 메뉴의 그룹 위젯에서 선택한 필터가 없고 대시보드 필터를 적용할 때, 그룹의 하위 위젯에 대시보드 필터가 적용되지 않는 문제 수정

분석​

Fixed 메트릭스 이상 탐지 메뉴에서 필터 옵션을 적용할 때, 한 카테고리에서 여러 항목을 선택할 수 없도록 수정

경고 알림​

  • Feature 이벤트 수신 설정 메뉴에서 Office 365 커넥터 방식이 중지됨에 따라 Teams Workflows을 통해 경고 알림을 받을 수 있는 서드 파티 플러그인 추가

    노트

    Teams Workflows 서드 파티 플러그인 추가에 대한 자세한 내용은 다음 문서를 참조하세요.

  • Feature 이벤트 수신 설정 메뉴에서 PagerDuty를 통해 경고 알림을 받을 수 있는 서드 파티 플러그인 추가

    노트

    PagerDuty 서드 파티 플러그인 추가에 대한 자세한 내용은 다음 문서를 참조하세요.

  • Feature 이벤트 설정 New 메뉴에서 이벤트명 또는 이벤트 수신 태그로 검색할 수 있는 기능 추가

  • Changed 실시간으로 화면에 표시되는 알림 메시지의 브라우저 알림 끄기 기능을, 모든 알림을 닫을 수 있는 전체 알림 닫기 기능으로 변경

  • Changed 경고 알림 메시지 관련 이벤트 해결 용어를 이벤트 해소로 일괄변경

  • Fixed 이벤트 설정 V2 메뉴의 이벤트 추가 또는 수정 화면에서 Warning 또는 Critical 지표 중 하나만 활성화된 경우, 동일한 설정값을 입력하면 저장되지 않는 문제 수정

  • Fixed 이벤트 설정 V2 메뉴의 복합 메트릭스 이벤트 추가 또는 수정 화면에서 MXQL 데이터가 조회되지 않는 문제 수정

    이벤트 데이터 조회 섹션의 필터 옵션으로 문자열 타입 값을 갖는 태그 또는 필드를 설정 후 텍스트 모드로 전환하면 MXQL 데이터가 조회되지 않는 문제를 수정했습니다.

  • Fixed 이벤트 설정 메뉴에서 이벤트 목록을 이벤트 이름으로 정렬하도록 수정

  • Fixed 이벤트 설정 메뉴의 메트릭스 및 이상치 탐지에서 개별 이벤트 항목 제목에 마우스를 오버했을 때 표시되는 툴팁의 복사 버튼이 동작하지 않는 문제 수정

  • Fixed 실시간 알림에서 차트의 알림 표시 개수가 누적되는 문제 수정

관리​

  • Feature 다음 메뉴 경로에서 멤버 목록을 CSV 형식의 파일로 다운로드할 수 있는 기능 추가

  • Changed 통합 그룹 관리 버튼 위치 및 UI 변경

User Interface​

  • Changed 테마 색상 일부 변경

  • Feature 트레이스 분석 창의 테이블 뷰 탭에서 바인드 변수가 매핑된 SQL 문을 다운로드할 수 있는 기능 추가

  • Fixed 트랜잭션의 SQL 문에서 치환된 변수 # 을 $로 치환해서 표시하지 않도록 수정

  • Fixed 인스턴스 성능 관리 > 소켓 오픈 개수 메뉴에서 트랜잭션 통계 메뉴로 이동할 때 종료 시각이 시작 시각보다 빠르게 설정되는 문제 수정

  • Fixed 트레이스 분석 창의 트랜잭션 로그 탭에서 로그 데이터가 없으면 안내 메시지를 출력하도록 변경

    로그 관련 에이전트 옵션이 설정되어 있지 않거나 로그 데이터가 없을 경우 메시지를 확인할 수 있습니다.

  • Fixed Firefox 브라우저에서 대시보드 > 애플리케이션 대시보드 메뉴의 아크 이퀄라이저 위젯 중앙 텍스트 정렬이 깨지는 문제 수정

  • Fixed 화면의 테마가 다크 모드일 때, 분석 > 스택 메뉴의 버튼을 통해 스냅샷을 저장하면 일부 스택 정보가 표시되지 않는 문제 수정

Server​

  • Changed 프로세스 목록 메뉴에서 이름 컬럼 데이터를 서버 상세 또는 shell에서 식별할 수 있는 이름으로 변경
