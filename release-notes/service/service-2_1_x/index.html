Service 2.1.7​

2024년 02월 01일

Network​

New 네트워크 성능 모니터링(Network Performance Monitoring) 서비스를 오픈합니다. 네트워크 성능 모니터링은 네트워크 간의 혼잡 및 장애를 파악하고 사용자의 네트워크 성능 개선에 활용하기 위하여 프로세스의 네트워크 통신 정보를 수집하여 가시화하는 모니터링 솔루션입니다.

노트

많은 이용 부탁드리며, 네트워크 성능 모니터링에 대한 자세한 내용은 다음 문서를 참조하세요.

2024년 01월 24일

공통​

  • Feature 경고 알림 > 이벤트 기록 메뉴에서 다음 기능 추가

  • Changed 휴면 상태의 계정을 프로젝트 또는 그룹에 초대 시 표시되는 문구 변경

  • Changed 이용내역 > 프로젝트 유료 전환 메뉴의 유료 전환 창에서 프로젝트 이름이 긴 경우 말줄임 처리, 마우스 오버 시 툴팁으로 프로젝트 이름을 보여주도록 개선

  • Changed Flex 보드 메뉴에서 대시보드 내보내기 기능을 통해 JSON 파일로 저장할 경우, 파일명의 특수 문자를 기존에는 제거했으나 밑줄(_)로 변경
