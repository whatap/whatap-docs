Kubernetes​

    • Total Container CPU Request: 프로젝트 내부의 모든 컨테이너의 CPU Request(cpu_request) 값의 합산
    • Total Container CPU Usage: 프로젝트 내부의 모든 컨테이너의 CPU Usage(cpu_total_milli) 값의 합산
