Service 2.20.0​

2025년 08월 27일

공통​

  • Feature 계정 관리 > 계정 정보 UTC 오프셋 변경 기능 추가

  • Feature 계정 관리 접근 제어 메뉴 추가 및 계정별 IP 접근 제어 기능 추가

  • Feature 계정 관리 GPU, GPU MIG, Open Metric High, Open Metric Low 등 새로운 사용량 지표가 청구서, 과금, 프로젝트 구독, 미터링 요약 등에 추가

  • Feature Flex 보드 > 필터 메트릭스 필터, 태그 선택에 직접 입력 옵션 추가

  • Feature Flex 보드 일간 비교 위젯 추가

Application​

  • Feature 대시보드 > 애플리케이션 대시보드 액티브 HTTPC, 액티브 DB 위젯 추가

  • Feature 통계 > 트랜잭션 토폴로지 열기 버튼 추가

  • Feature 대시보드 > 비즈니스 대시보드 상세, 토폴리지 탭 추가

  • Feature 대시보드 > 애플리케이션 대시보드 이벤트 위젯 추가

    • 이벤트를 레벨별로 시각화하고, 에이전트별로 이벤트를 구분해 표시

  • Changed 애플리케이션 토폴로지 Beta UI 개선

    • 엣지 라벨 게이지 표현 추가 및 응답 시간 임계 색상 표현 추가

    • 특정 노드(상세)

      • TX 트레이스 컬럼 설정 추가 및 전체/액티브/멀티 트랜잭션/에러 필터 추가

      • 선택한 노드와 관련있는 노드만 나타내는 팝아웃 추가

  • Changed app_context_stat 관련 명칭 통일(컨텍스트, 컨텍스트별 → 업무, 업무별)

  • Changed 액티브 스택 및 멀티 트랜잭션 아이콘 색상 변경

    • 멀티 트랜잭션: 파란색 → 회색

    • 액티브 스택: 빨간색 → 파란색

  • Fixed 대시보드 > 비즈니스 대시보드 별칭 설정, 모든 별칭 삭제 후 저장 시 변경 사항이 적용되지 않는 현상 수정

Java, Node.js, PHP​

  • Fixed 로그 > 로그 설정 로그 모니터링 시작하기 탭의 '1. 에이전트 설정 확인' 항목 일본어 번역 수정

.Net, Python​

  • Changed 지원 버전 정보와 OS 정보 수정

Java​

  • Changed 에이전트 설치 페이지 개선

Server​

  • Feature GPU 인벤토리 키워드 검색 필터 추가

  • Feature GPU 성능요약 컬럼 설정의 필드별 설명 툴팁 추가

  • Feature 서버 상세 서버 목록에 검색 입력 필드 추가

  • Feature 대시보드 브라우저 크기에 따라 위젯 크기가 자동 조정되는 모드를 기본 적용하고, 스크롤이 생기더라도 위젯을 확대해 정상적으로 볼 수 있도록 위젯 높이 자동 조정 스위치를 추가

  • Changed GPU 성능요약 GPU Status 조회 시간 범위를 기존 5분에서 1분으로 조정

  • Changed GPU 성능요약 Performance_State 필드값이 P0~P15 사이 값으로 표시되도록 변경

  • Changed GPU 성능요약 > 비교하기 패널

    • 단위가 bytes/s인 차트 값이 자동으로 단위 변환되도록 변경

    • Performance_State 차트의 Y축 범위를 P0~P15 고정하도록 변경

  • Changed 서버 목록 하위의 GPU 인벤토리와 GPU 성능요약 메뉴를 GPU 그룹 하위로 이동

  • Changed Event v2 매트릭스 이벤트 룰 추가할 경우, 지표 설정 조건에 밸리데이션 추가 및 밸리데이션 문제 시 오류 문구 추가

  • Changed GPU 인벤토리, GPU 성능요약 에이전트 최소 버전을 2.8.9에서 2.9.2로 변경

  • Fixed 서버 목록 키워드 필터 검색 시 항목이 oid인 경우 필터 항목 선택되지 않는 문제 수정

  • Fixed 서버 상세 서버 선택 후 페이지 로드 시, 서버 목록에 마우스 호버해도 즉시 표시되지 않는 문제 수정

  • Fixed 메트릭스 익스플로러 by GPU Process 하위의 지표 선택 시, 차트의 범례 label의 형식을 process_name, pid, (process_type) - index(uuid)로 표시되도록 수정

  • Fixed 서버 인벤토리 신규 에이전트 수집 필드 추가 시, 테이블 헤더에 필드명 표시되지 않는 현상 수정

  • Fixed 서버 상세 차트 상세 보기 드로워가 닫힐 때, 선택 서버가 아닌 default 서버 선택으로 초기화되는 문제 수정

  • Fixed 서버 목록 전체 리프레시 버튼 클릭 시, 이벤트 발생에 따른 서버 상태가 반영되지 않는 이슈 수정

  • Fixed GPU 성능요약 > 비교하기 패널

    • 단위가 %인 차트의 y축 최댓값이 100%를 넘지 않도록 수정

    • 단위가 없는 차트에서 제목에 빈 괄호가 표시되던 현상 수정
