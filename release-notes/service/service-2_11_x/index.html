이 수정 사항은 DBX 에이전트 1.6.7 버전 이상에서 지원합니다.

Oracle, Oracle Pro​

  • Feature 대시보드 > 멀티 인스턴스 모니터링 메뉴에 Disk Usage 위젯 추가

  • Feature 분석 > SQL 분석 메뉴에서 다음 기능 추가

    • Plan Change History 탭에서 sql_id, sql_hash_value, PLAN_HASH_VALUE로 검색할 수 있는 필터 기능

    • Plan Change History 탭에서 Plan Diff 확인 시 새 창으로 보기

  • Changed 대시보드 > 멀티 인스턴스 모니터링 메뉴에서 다음 사항 변경

    • Disk Usage 위젯의 크게 보기 창에서 여백에 대한 UI 변경

    • 세션 상세 정보 창에서 세션이 종료되더라도 sid와 serial# 정보가 유지되도록 변경

    • 세션 상세 정보 창의 Session Info에 sql_hash_value 값 추가

    • 세션 상세 정보 창의 Session Info에 command 값을 상태명으로 표시하도록 변경

    노트

    Session Info에 command 값을 상태명으로 표시하는 기능은 DBX 에이전트 2.2.3 버전 이상에서 지원합니다.
