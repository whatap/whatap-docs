    • 청구 및 납부 내역 > 상세 - 과금집계량

  • Feature OpenMetrics 대시보드 목록 추가

    • 사전 정의 대시보드 제공: Istio, Nginx

  • Changed OpenMetrics 탐색기 조회 가능 기간 변경

Application​

  • Changed 관리 에이전트 목록, 테이블 칼럼 명칭 수정

    • 시작 시간 → 애플리케이션 시작 시각
    • 활성/비활성 → 에이전트 활성/비활성 시각

  • Fixed 애플리케이션 대시보드 금일 위젯에서 비교일자 데이터가 없는 경우, 기준일자 데이터가 회색으로 표시되던 현상 수정

Server​

  • Feature 서버 인벤토리 히스토리 조회 기능 추가

    • 인벤토리 과거 데이터 조회
    • 변경 이력 조회
    • 서버 히스토리 일자별 비교

  • Feature 서버 인벤토리 선택된 테이블 칼럼 대상으로 동작하는 데이터 검색 입력창 추가

  • Feature 에이전트 설치 Windows 2008 R2용 ZIP, EXE 파일 다운로드 버튼 추가

  • Changed GPU 인벤토리, GPU 성능요약 리눅스 에이전트 최소 안내 버전 변경(2.9.2 → 2.9.3)

  • Changed 서버 상세 > 프로세스 목록 '디스크 I/O'를 '프로세스 I/O'로 명칭 변경

  • Changed 서버 상세 > 프로세스 상세보기 '디스크 I/O (Bps)'를 '프로세스 I/O (Bps)'로 변경

  • Changed GPU 인벤토리 > 컬럼 설정 초기화 시 자동 수집 칼럼이 전체 선택되도록 변경하고 기본 칼럼 순서 조정

  • Fixed GPU 인벤토리 Physical GPU의 MIG Mode Current 값이 Disabled인 경우, 하위 MIG 인스턴스는 보이지 않도록 처리

Kubernetes​

  • Feature 애플리케이션 > 대시보드 > 비즈니스 대시보드 추가

  • Feature 관리 > 에이전트 설치 오퍼레이터 설치 방법 추가

Database​

공통​

  • Feature 분석 > 스크립트 매니저 CSV 다운로드 기능 추가

  • Feature 대시보드 > 인스턴스 모니터링 New 일간 비교 위젯 추가

  • Feature 대시보드 > 인스턴스 모니터링, 분석 > 카운트 추이 멀티 DB 구성 시 나타나는 dbname 셀렉터 개선

  • Feature 관리 > 에이전트 목록 에이전트 일시 정지 기능 추가(dbx 에이전트 버전 2.55.03 이상 지원)

  • Feature 통합 Flex 보드 인스턴스 목록 위젯 추가

  • Feature dbxc 리눅스 다운로드 Copy 박스 추가

  • Changed 분석 > 메트릭스 조회, 메트릭스 이상 탐지 사이트맵으로 이동

  • Fixed 대시보드 > 인스턴스 목록 인스턴스의 Replication 혹은 Cluster 상태가 변경될 때 M/S/C 아이콘이 정상 변경되지 않는 현상 수정

  • Fixed 대시보드 > 인스턴스 맵 Top SQL을 지원하는 데이터베이스 제품만 Top SQL 탭이 노출되도록 수정

  • Feature SQL 상세 통계 추이 차트 추가

  • Fixed SQL 상세 JSON 플랜이 기본으로 조회되도록 수정

  • New 대시보드 > 버터풀 모니터링 추가

  • Feature 대시보드 > 인스턴스 모니터링, 멀티 인스턴스 모니터링 액티브 세션 테이블에서 확인할 수 있는 세션 상세 정보 팝아웃 화면 개선
