Preview 2.10.0.a

노트

preview.whatap.io에서 다음 추가한 신기능과 수정한 내용을 미리 경험할 수 있습니다.

Preview 2.10.0.a4​

2024년 10월 22일

공통​

  • Feature 경고 알림 > 이벤트 기록 메뉴에서 이벤트 처리 내역 기능 추가, 이에 따른 UI 변경

    노트

    Server 상품의 기능을 모든 상품으로 확대 적용합니다.

  • Changed 경고 알림 > 이벤트 설정 New 메뉴의 메트릭스 탭에서 이벤트 추가 및 수정, 이벤트 목록에 새로운 UI 적용

  • Fixed 상품 소개 화면에서 시작하기 버튼을 클릭하면 간헐적으로 선택한 조직이 해제되는 문제 수정

Application​

Java​

  • Feature MySQL, PostgreSQL 프로젝트와 실시간 및 과거 트랜잭션 연계 분석 기능 추가

Server​

  • Changed 관리 > 에이전트 설치 메뉴에서 Solaris 운영체제 환경의 설치 안내 변경

    • Solaris 버전별 다운로드 링크 제공

    • install.sh 스크립트 파일을 사용한 설치 방식으로 변경

  • Fixed 관리 > 에이전트 설치 메뉴에서 HP-UX 환경에서 에이전트 업데이트 단계의 스크립트 오타 수정

  • Fixed 언어 설정이 일본어일 때, 서버 목록 > 서버 인벤토리 메뉴에서 필터 입력란의 안내 문구가 잘려서 표시되는 문제 수정

Kubernetes​

  • New Ingress 리소스에 대한 메타데이터와 관련 정보를 모니터링할 수 있는 Ingress 목록 메뉴 추가

    노트

    클러스터 > Ingress 목록 경로에서 확인할 수 있습니다.

  • Feature 다음 메뉴 경로에 프로세스 정보 추가

    • 대시보드 > 노드 맵

    • 클러스터 > 노드 목록 > 상세 보기

    노트

    이 기능은 클러스터 프로젝트에만 적용되며, 쿠버네티스 에이전트 1.7.12 버전 이상에서 지원합니다.

  • Feature 클러스터 > 노드 목록 메뉴에 신규 컬럼 추가

    • 노드 정보: AllocatablePods, DiskPressure, MemoryPressure, PIDPressure, Unschedulable, HasKubelet

    • CPU: AllocatableCpu

    • 메모리: AllocatableMemory

  • Changed 워크로드 > Service 목록 메뉴를 클러스터 메뉴 하위로 위치 변경

  • Changed 컨테이너 > 컨테이너 목록 메뉴에서 Deployment 컬럼을 통해 Deployment 목록 메뉴로 이동하면 선택한 Deployment 상세 정보가 기본으로 열리도록 수정

  • Changed 컨테이너 > 컨테이너 목록 메뉴에서 비교하기 기능의 UI 변경

  • Fixed 대시보드 > 컨테이너 맵 메뉴의 요약 보기에서 트레이스 탭과 차트에서 발생하는 메모리 릭(Leak) 문제 수정

Database​

공통​

  • New 클라우드 데이터베이스 서버의 자원현황을 수집하고 모니터링할 수 있는 클라우드 설정 메뉴 추가

    노트

    • 지원하는 클라우드 서비스 및 상품은 다음과 같습니다.

      • Azure Monitor: PostgreSQL, MySQL

      • AWS CloudWatch: PostgreSQL, Oracle, Oracle Pro, MySQL

    • 관리 > 클라우드 설정 경로에서 확인할 수 있습니다.

  • Changed 연계 분석 버튼의 위치를 세션 상세 정보 창에서 액티브 세션 테이블로 변경

    노트

    이 변경 사항은 MySQL V2, PostgreSQL V2, Oracle V2, Oracle Pro 상품에 해당합니다.

  • Changed 대시보드 > 인스턴스 모니터링 메뉴에서 시간 선택자의 시간을 변경하면 테이블 데이터도 같이 변경하도록 수정

  • Fixed 대시보드 > 멀티 인스턴스 모니터링 메뉴에서 실시간 조회를 중지하고 과거 시간을 조회하면 SQL Elapse Map 위젯의 그래프가 표시되지 않는 문제 수정

  • Fixed 대시보드 > 인스턴스 목록 메뉴에서 멀티 뷰 버튼으로 클릭해 멀티 인스턴스 모니터링 메뉴로 이동하면 선택한 인스턴스가 적용되지 않는 문제 수정

  • Fixed 에이전트를 설치하지 않은 프로젝트로 진입하면 조직 정보가 해제되는 문제 수정

MySQL V2, PostgreSQL V2​

New Application 프로젝트와 실시간 세션 연계 분석 기능 추가

MySQL, PostgreSQL​

Changed 관리 > 에이전트 설치 메뉴에서 RDS 옵션 설정을 클라우드 설정 메뉴 링크로 변경

Oracle Pro​

Changed 관리 > 에이전트 설치 메뉴에서 Oracle 계정이 아닌 곳에서 설치할 경우 버튼의 스타일 수정

Altibase​

Deprecated 관리 > 에이전트 설치 메뉴에서 RDS 옵션 설정 내용 삭제

Browser​

New 페이지 URL 경로별 로드 성능 정보에 대한 통계를 제공하는 페이지 로드 통계 메뉴 추가

노트

이 기능은 통계 > 페이지 로드 통계 메뉴 경로에서 확인할 수 있습니다. 자세한 내용은 다음 문서를 참조하세요.

Log​

  • Changed 로그 > 로그 설정 메뉴의 로그 1차 파서 설정에서 시뮬레이션 및 퍼포먼스 측정 결과 성공 시 파서 패턴을 적용할 수 있도록 동선 변경
