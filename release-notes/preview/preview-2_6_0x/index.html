Preview 2.6.0.a

노트

preview.whatap.io에서 다음 추가한 신기능과 수정한 내용을 미리 경험할 수 있습니다.

Preview 2.6.0.a7​

2024년 06월 17일

공통​

  • Changed 분석 > 메트릭스 이상 탐지 메뉴의 상세 데이터 조회 및 표시 기준 변경

    • 기존: 차트 영역을 드래그한 기간에 대해 조회

    • 변경: 최대 6시간 기간만 조회

  • Changed 통합 대시보드, 통합 메트릭스 보드 메뉴의 서비스 종료 안내 메시지 변경

Application​

Feature 대시보드 > 애플리케이션 대시보드 메뉴에서 Default 프리셋 변경 기능 추가

Server​

Fixed 관리 > 에이전트 설치 메뉴에서 Oracle Solaris 에이전트 설치 명령어의 변수 수정 및 적용되지 않은 다국어 내용 반영

Kubernetes​

Fixed 클러스터 > 노드 목록 메뉴에서 컬럼 선택 드롭다운의 체크박스가 아래 목록의 체크박스와 겹치면 체크박스를 선택할 수 없는 문제 수정

Database​

공통​

Fixed 분석 > 데이터베이스 파라미터 메뉴에서 비교 날짜 선택 시 오늘 버튼 클릭하면 데이터가 정상 조회되지 않는 문제 수정

노트

이 기능은 Redis, MongoDB, Oracle DMA 제품에는 적용되지 않습니다.

MySQL​

Changed 관리 > 에이전트 설치 메뉴의 설치 안내 스타일 개선

Log​

Changed 로그 > 로그 설정 메뉴에서 데이터 유지 기간, 데이터 보관일 항목의 명칭을 데이터 보존 기간으로 통일

Browser​

Feature 페이지 로드 히트맵, AJAX 히트맵, 사용자 세션 로그 검색 메뉴에 세션 리플레이 기능 추가

  • Changed 프로젝트 생성을 위한 제품 선택 화면에서 검색 결과가 없을 때의 안내 문구 개선

  • Changed 이용 내역 > 프로젝트 유료 전환 메뉴에서 상태 컬럼의 유료 전환 버튼을 컬럼 분리해 표시하도록 수정

Application​

공통​

  • Feature 대시보드 > 트랜잭션 맵 메뉴에서 TOP 30 필터 내 항목을 직접 검색해 추가하는 커스텀 기능 추가

  • Feature 통계 하위 메뉴의 기능 개선

    • 마지막으로 조회한 에이전트 선택 옵션의 선택값을 저장합니다.

    • 각 메뉴로 진입했을 때 자동 조회하지 않도록 변경합니다.

    • 각 메뉴의 최초 조회 기간을 금일에서 전일로 변경합니다.

    • 트랜잭션, SQL 메뉴의 기본 정렬 순서를 합계 시간으로 변경합니다.

  • Changed 인스턴스 성능 관리 > 로딩된 클래스 메뉴에서 클래스 조회 창의 제목 수정(클래스 시그니쳐 → 클래스 상세)

  • Changed 구버전의 애플리케이션 대시보드 메뉴 제거 예정

    • 사이트맵 메뉴에 Deprecated 태그 추가

    • 사이트맵 > 애플리케이션 대시보드 Old 메뉴를 진입하면 제거 예정 안내 메시지 표시

  • Changed 통계 하위 메뉴에서 필터를 적용한 후 버튼을 선택하지 않아도, CSV 파일을 다운로드하면 필터가 적용되도록 수정

  • Fixed 트레이스 분석 창에서 운영체제, 클라이언트 명, 상태에 대한 문자열이 번역 key 값으로 노출되는 문제 수정

  • Fixed 트레이스 분석 창에서 스텝 타임라인 그래프를 선택하면 해당 스텝으로 이동하지 않는 문제 수정

  • Fixed 분석 > 메트릭스 차트 메뉴에서 위젯의 차트 영역을 드래그한 후 상세 버튼을 선택해 열리는 화면의 조회 시간이 위젯의 조회 시간과 다르게 표시되는 문제 수정

  • Fixed 대시보드 > 애플리케이션 대시보드 메뉴에서 위젯의 버튼을 클릭해 안내 창을 열면 로딩 표시가 해당 위젯에서도 적용되는 문제 수정

  • Fixed 대시보드 > 애플리케이션 대시보드 메뉴에서 에이전트가 없는 경우 딜레이 없이 데이터 호출이 발생하는 문제 수정

  • Fixed 에이전트 선택(애플리케이션) 옵션에서 에이전트 목록이 화면의 세로 크기보다 많을 경우 스크롤이 적용되지 않는 문제 수정

  • Fixed 사이트맵 > 토폴로지 > 애플리케이션 통계 토폴로지 메뉴에서 시간 선택 옵션을 클릭해도 기능이 작동하지 않는 문제 수정 및 시간 선택 옵션의 UI 변경

  • Fixed 조직에 소속된 프로젝트에서 분석 > 스택 메뉴로 진입하면 발생하는 화이트아웃 현상 수정

    cpu_quota, cpu_request, cpu_total_milli, mem_limit, mem_request, mem_working_set

  • Fixed 대시보드 > 컨테이너 맵 메뉴의 요약 보기에서 로그 > 로그 검색 탭의 조회 결과가 상세 보기의 조회 결과와 일치하지 않는 문제 수정
