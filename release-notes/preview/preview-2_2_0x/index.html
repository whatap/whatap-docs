Preview 2.2.0.a

https://preview.whatap.io에서 다음 추가한 신기능과 수정한 내용을 미리 경험할 수 있습니다.

Preview 2.2.0.a6​

2024년 02월 19일

공통​

  • Changed 계정 관리 메뉴의 멀티팩터 인증 섹션에서 OTP 키를 발급하지 않은 상태이면 이메일로 OTP 키 보내기 버튼 비활성화
  • Changed 프로젝트 생성 시 상품 추천 배너에 Network Performance Monitoring 추가, Java 상품 제거
  • Deprecated 언어 설정이 영어 또는 일본어일 때 계정 정보 관리 메뉴에서 마케팅 옵션 제거

Application​

  • Changed 대시보드 > 애플리케이션 대시보드 메뉴에서 금일 TPS 및 금일 사용자 위젯에 All APPS. 태그 추가

    All APPS. 태그가 표시된 위젯은 항상 모든 에이전트에 대한 데이터를 표시하도록 고정되어 있습니다.

  • Feature 대시보드 > 트랜잭션 맵 메뉴에 새로운 UI 적용 및 기능 개편
    • 데이터 조회 구간 확장: 실시간 최대 30분, 과거 시점 최대 1일
    • 트레이스 분석 창을 통해 최대 조회 가능 데이터 개수 증가(100건 → 1000건)
    • 트랜잭션 및 에러 개수를 실시간으로 표시
    • Top 30 목록에서 키보드 이벤트 추가
    • 차트 영역에 대한 키보드 이벤트 추가
    • 과거 시점의 데이터를 조회할 경우 서버로부터 받는 데이터 필터링 가능
    트랜잭션 맵 메뉴의 개편과 관련한 자세한 내용은 다음 링크를 참조하세요.

Database​

공통​

  • Feature XOS2 지표 대체 관련 안내 사항 추가
  • Fixed 경고 알림 > 이벤트 설정 > 데이터베이스 탭에서 이벤트 상태가 해결되면 추가 알림 기능에 대한 툴팁 문구 수정(에이전트 버전 1.6.22 이상 → 에이전트 버전 1.6.30 이상)

MySQL V2​

  • Feature JSON 형식으로 Plan 조회 기능 추가
  • Feature 통계/보고서 > 데이터베이스 사이즈 메뉴의 테이블 사이즈 (TOP 50) 테이블에서 TABLENAME 컬럼 항목을 클릭하면 column, index 구성을 확인할 수 있는 기능 추가

Kubernetes​

  • Fixed 홈 화면 > Flex 보드 메뉴에서 컨테이너 performnace 템플릿 선택 시 발생하는 화이트 아웃 문제 수정
  • Fixed 대시보드 > 컨테이너 맵 메뉴에서 노드 블록 선택 후 상세 보기 버튼을 선택하면 발생하는 화이트 아웃 문제 수정

Log​

Fixed 경고 알림 > 이벤트 설정 > 로그 메뉴에서 복합 로그 이벤트 저장을 실패하는 문제 수정
