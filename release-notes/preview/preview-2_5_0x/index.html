Preview 2.5.0.a

노트

preview.whatap.io에서 다음 추가한 신기능과 수정한 내용을 미리 경험할 수 있습니다.

Preview 2.5.0.a7​

2024년 05월 14일

공통​

프로젝트​

  • Changed 제품 선택 및 프로젝트 생성 UI/UX 개선

  • Changed 에이전트 설치 메뉴로의 유입 경로에 따라 화면 상단에 노출되는 스텝바의 노출 여부를 다르게 적용

Flex 보드 / 통합 Flex 보드​

  • Feature 위젯에 대한 설명 첨부 기능 추가

  • Changed 리치 텍스트 위젯에서 Heading 스타일의 여백 제거 및 텍스트 분량에 따라 스크롤이 자동 노출되도록 변경

  • Fixed 보조 차트 설정 수정 후 그룹 내 전체 위젯에 적용 기능을 이용하면 데이터 병합 옵션이 적용되지 않는 문제 수정

User Interface​

Changed 다크 테마의 일부 색상 변경

Application​

  • Changed 대시보드 > 애플리케이션 대시보드 메뉴의 금일 TPS, 금일 사용자 위젯은 하루 내에서 데이터를 재조회할 때 새로운 데이터를 요청하지 않고, 이미 조회한 데이터를 재가공해서 사용하도록 수정

  • Fixed 대시보드 > 트랜잭션 맵 메뉴에서 맵 상단 영역 드래그 시 데이터가 조회되지 않는 문제 수정

  • Fixed 트레이스 분석 > 테이블 뷰 탭 > SQL 스텝 오른쪽 아이콘 선택 > SQL 통계 창에서 트랜잭션 검색 메뉴로 이동 시 선택한 에이전트가 유지되지 않는 현상 수정

Server​

Changed 에이전트 설치 단계로 진입하기 전에 OS 선택 단계 추가

Kubernetes​

  • Changed 관리 > 에이전트 설치 메뉴의 쿠버네티스 에이전트 설치 방식 개편

    • 사용자가 설치 파일을 수정하지 않고 옵션 선택만으로 환경에 맞는 파일을 다운로드할 수 있습니다.

    • 추가 기능 적용 옵션을 추가합니다.

    • 설치 마지막 단계에서 모니터링 주요 메뉴를 소개합니다.

  • Feature 실험실 > Pending Pod 현황 메뉴의 상세 탭 개편

    • 이벤트 메시지를 기반으로 빠른 원인 분석을 도와주는 Suggestions 탭 추가

    • Pod 연관 이벤트를 확인할 수 있는 Event 탭 추가

    • 컨테이너 로그를 확인할 수 있는 Log 탭 추가

    노트

    이 기능은 쿠버네티스 마스터 에이전트 1.7.0 버전 이상에서 지원합니다.

  • Fixed 관리 > 에이전트 설치 메뉴의 애플리케이션 에이전트 설치에서 Docker Python의 okind 설정 옵션 이름을 whatap.okind로 변경

Database​

공통​

  • Fixed 관리 > 에이전트 목록 메뉴에서 DB IP 컬럼 내 툴팁이 표시되지 않는 문제 수정

  • Fixed 통계/보고서 > SQL 통계 메뉴에서 query 텍스트를 선택하면 나타나는 SQL 상세 창의 쿼리문이 undefined로 표시되는 문제 수정

PostgreSQL / MySQL / Oracle DMA​

New 비교 시간을 설정해 해당 시간대에서 가장 많이 실행된 SQL과 해당 SQL의 등락을 확인할 수 있는 Top SQL 비교 메뉴 추가

노트

이 기능은 분석 > Top SQL 비교 메뉴 경로에서 확인할 수 있습니다.

PostgreSQL / Oracle DMA​

  • Feature 분석 > Wait 분석 메뉴의 테이블 위젯에서 마우스를 오버하면 항목의 전체 이름을 확인할 수 있는 툴팁 추가

Oracle DMA​

  • Feature Applicatoin 제품에서 Database 제품으로 연계하는 분석 기능에서 이벤트 값이 enq: TX - row lock contention이면 Lock 메시지를 표기하도록 기능 추가

  • Feature 대시보드 > 멀티 인스턴스 모니터링 메뉴에서 SQL Elapse Map 위젯의 차트 영역을 드래그하면 나타나는 Query List 테이블에 sid, serial# 컬럼 추가

  • Changed 통계/보고서 > SQL 통계 메뉴에서 테이블의 컬럼 순서 조정

  • Changed 대시보드 > 인스턴스 모니터링, 멀티 인스턴스 모니터링 메뉴에서 액티브 세션 테이블의 항목을 선택하면 팝업되는 세션 상세 정보 창에서 표시되는 지표 변경 및 수정

  • Fixed Applicatoin 제품에서 Database 제품으로 연계 분석하는 기능에 특정 키 값이 있으면 에러 메시지로 확인하도록 키 값 수정

  • Fixed 분석 > SQL 분석 메뉴에서 Operation Count 차트가 무한 로딩되는 문제 수정

  • Fixed 통계/보고서 > SQL 통계 메뉴에서 필터 기능이 정상 작동하지 않는 문제 수정

Tibero​

Changed 통계/보고서 > SQL 통계 메뉴에서 DB 탭 항목 제외
