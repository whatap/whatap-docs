Preview 1.111.X

https://preview.whatap.io에서 다음 추가한 신기능과 수정한 내용을 미리 경험할 수 있습니다.

Preview 1.111.9​

2023년 10월 23일

공통​

User Interface​

  • Feature 상품 카탈로그 페이지에서 더 나은 사용자 경험을 위한 디자인 개편
  • Fixed 사용자 언어가 일본어인 경우 사이드 바의 상품 소개 페이지에서 상품 제목을 영어로 수정

Flex 보드​

Fixed 리치 텍스트 위젯의 텍스트 편집기에서 사용 불가능한 편집 기능 버튼 제거

Container​

  • Feature 노드 에이전트 1.3.7 버전부터 신규 지표 추가 및 UI 디자인 개선된 노드 목록 화면 제공
  • Fixed 클러스터 프로젝트에 대한 조회 권한이 없는 멤버의 경우, 네임스페이스 프로젝트 화면 위에 프로젝트 이름이 프로젝트 코드(PCODE)로 표시되는 현상 수정
  • Feature 대시보드 > 컨테이너 맵 메뉴에서 블록 색상 표시 지표에 애플리케이션 관련 필드도 선택할 수 있도록 추가

Browser​

  • Fixed 사용자 언어가 일본어인 경우 대시보드 > 브라우저 모니터링 대시보드 메뉴에서 일본어 위젯 제목 수정
  • Feature 사이트맵 > 사용자 행위 메뉴 추가
  • Feature 분석 > AJAX 히트맵 메뉴 추가

Preview 1.111.8​

2023년 10월 19일

공통​

  • Changed 사이트맵 > 사용자 행위 메뉴의 테이블에서 Request URI, METHOD 컬럼 제거
  • Fixed 조직이 선택된 상태에서 프로젝트 생성 시 상품 카드를 클릭하면 선택된 조직이 풀리는 현상 수정

Application​

대시보드​

  • Fixed 트레이스 분석의 SQL 변수 치환 기능 버그 수정
  • Fixed 트레이스 분석의 멀티 트랜잭션 뷰로 진입 불가 버그 수정

분석​

Fixed 히트맵 메뉴에서 검색 버튼 연타 방지를 위해 다음 조회까지 0.5초 딜레이 추가

통계/보고서​

Changed 통계 메뉴에서 탭 이동 시 사용자가 설정한 검색 시간을 유지하도록 수정

Database​

공통​

  • Changed 대시보드 > 인스턴스 목록 메뉴에서 테이블이 브라우저 크기에 맞춰 채워지도록 변경
  • Changed 통계/보고서 > SQL 통계 메뉴의 사용자 편의성을 위한 UI 개선

    • 프로그레스바 추가

    • 테이블 영역에 Panel 추가

    • 테이블 서브 차트에 ChartIndex 추가

  • Feature 대시보드 > 인스턴스 모니터링 메뉴에서 비실시간 모드 상태가 되면 해당 시간의 SQL 통계 메뉴로 이동할 수 있는 버튼 추가

DB V2​

  • Changed 멀티 인스턴스 모니터링 메뉴에서 액티브 세션이 누락되던 문제 수정
  • Fixed 멀티 인스턴스 모니터링 메뉴에서 SQL Elapse Map 위젯 차트의 조회 범위 수정

MySQL​

Changed 분석 > 데이터베이스 파라미터 메뉴의 테이블 스크롤이 테이블 내부에서 생성되도록 변경

PostgreSQL​

Fixed 분석 > Wait 분석 메뉴에서 액티브 세션의 상세 정보 창을 통해 세션 히스토리 메뉴로 이동할 경우, Wait 분석 메뉴의 시간으로 세션 히스토리를 조회하도록 수정

SQL Server​

Changed MS SQL Server 상품의 이름을 SQL Server로 변경

Container​

  • Fixed 관리 > 에이전트 설치 메뉴에서 Istio 환경일 경우 추가 자원 생성에 대한 yaml 파일의 들여쓰기 수정
  • Changed 다음 메뉴에서 트레이스 분석 기능을 개선된 버전으로 교체, 변경 사항은 다음 문서를 참조하세요.
    • 애플리케이션 > 멀티 서비스 히트맵
    • 대시보드 > 컨테이너 맵
  • Fixed 다음 메뉴에서 검색 버튼 연타 방지를 위해 다음 조회까지 0.5초 딜레이 추가
    • 애플리케이션 > 히트맵
    • 애플리케이션 > 멀티 서비스 히트맵

Log​

  • Fixed 로그 필터 입력에서 값을 수정할 때 대소문자 구분 옵션이 반영되지 않는 현상 수정
  • Fixed 로그 설정 메뉴에서 사용자가 선택한 언어에 따라 로그 설정 가이드를 연동하도록 수정

Network​

Changed 사이트맵 메뉴에서 사용자 행위 메뉴 링크 노출하도록 수정
