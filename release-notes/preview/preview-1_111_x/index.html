Preview 1.111.X

Note

In preview.whatap.io, the following new features and enhancements can be experienced in advance.

Preview 1.111.9​

October 23, 2023

Common​

User Interface​

  • Feature On the Product catalog page, the design has been reorganized for a better user experience.

  • Fixed In case the user language is Japanese, modified the product title to English on the introduction of the sidebar.

Flex Board​

Fixed Deleted the Edit buttons that cannot be used in the text editor for rich text widgets.

Container​

  • Feature Starting from node agent 1.3.7, added new metrics and provided the redesigned node list screen.

  • Fixed Fixed the issue where the project code (PCODE) appears for the project name on the namespace project screen for the members who do not have the role to view cluster projects.

  • Feature In Dashboard > Container Map, added the function for users to select application-related fields for the block color display metric.

Browser​

  • Fixed In case the user language is Japanese, modified the Japanese widget title in Dashboard > Browser Monitoring Dashboard.

  • Feature Added the Sitemap > User Behavior menu.

  • Feature Added the Analysis > AJAX Hitmap menu.

Preview 1.111.8​

October 19, 2023

Common​

  • Changed In Sitemap > User Behavior, deleted the columns for Request URI and METHOD in the table.

  • Fixed Fixed the issue where clicking on the product card when creating a project with an organization selected causes the selected organization to be released.

Application​

Dashboard​

  • Fixed Fixed the bug in SQL variable substitution for Trace analysis.

  • Fixed Fixed the bug that prevents access to the multi-transaction view of Trace analysis.

Analysis​

Fixed Added a 0.5-second delay until next search to prevent repeated pressing of Search in HITMAP.

Stat/Report​

Changed In Statistics, modified to maintain the set search time when moving tabs.

Database​

Common​

  • Changed In Dashboard > Instance list, modified to fill in the table according to the browser size.

  • Changed In STAT/REPORT > SQL Statistics, improved the UI design for user convenience.

    • Added the progress bar.

    • Added the panel in the table area.

    • Added ChartIndex in the table sub chart.

  • Feature In Dashboard > Instance monitoring, added the button to move to the SQL statistics menu in non-real-time mode.

DB V2​

  • Changed Fixed the issue where active sessions are missing in Multiple instance monitoring.

  • Fixed In Multiple instance monitoring, modified the query range of the SQL Elapse Map widget chart.

MySQL​

Changed In Analysis > Database parameter, modified the table scroll to be created in the table.

PostgreSQL​

Fixed In Analysis > Wait analysis, modified to display the session history by using the time in Wait analysis when the Session History menu appears through the detailed information window for active sessions.

SQL Server​

Changed Modified the MS SQL Server product name to SQL Server.

Container​

  • Fixed Modified the indentation of the YAML file for creating additional resources in the ISTIO environment in Management > Agent installation.

  • Changed In the following menus, replaced the Trace analysis function to a new version. For the changes, see the following.

    • APP. > Multiple Service Hitmap

    • Dashboard > Container map

  • Fixed Added a 0.5-second delay until next search to prevent repeated pressing of Search in the following menus.

    • APP. > Hitmap

    • APP. > Multiple Service Hitmap

Log​

  • Fixed Fixed the issue where the Case Sensitive option is not applied when changing the log filter values.

  • Fixed In Log setting, modified to link the log configuration guide depending on the language selected by the user.

Network​

Changed In Sitemap, modified to expose the User Behavior menu link.

Preview 1.111.7​

October 16, 2023

Common​

Changed Deleted real-time update animation for line charts to improve the performance and reduce bugs.

Database​

MySQL​

  • Feature Added the new feature description modal in Statistics > Mysql SQL Statistics.

  • Changed In Statistics > Mysql SQL Statistics, modified to display the configuration guide if data cannot be searched.

PostgreSQL​

  • Feature In Analysis > Wait analysis, Session History, added the new feature description modal.

  • Changed In Statistics > PG SQL statistics, modified to display the configuration guide if data cannot be searched.

Preview 1.111.6​

October 16, 2023

Common​

  • Feature Added the sidebar width adjustment function.

  • Feature In My Usage > Payment information, added to select a currency upon registration of payment information.

  • Changed In My Usage > Project subscription, Preview invoice, and Billing and payment history, deleted the exchange rate toggle.

Application​

Changed In Dashboard > Application dashboard, added the new feature description modal.

Preview 1.111.5​

October 13, 2023

Log​

Changed In Log setting > Log primary parser setting tab, repositioned the descriptions for reserved words that cannot be indexed.

Preview 1.111.4​

October 12, 2023

Common​

Fixed In Event configuration > Metrics tab, fixed the problem where the content does not appear when repeatedly importing the same json file.

Server​

Fixed In Server list > Server details, fixed the issue where the previous metric values of the server before change appear when changing the server list with the real-time interval gauge full.

DataBase​

DB V2​

Changed In Log > Log setting, added the explanation of how to configure logs for the installed DB and AWS RDS respectively.

Cloud Monitoring​

Feature In Sitemap, added the User Behavior menu.

Preview 1.111.3​

October 6, 2023

Common​

User Interface​

Fixed Deleted the function to display the interval for the default items in the interval selection option.

In specific categories, the default interval value appears as (0 second), but the interval value applied to the chart is sometimes not 0 second. Accordingly, the time value has been deleted.

Notice​

Fixed In Notice, fixed the bug where the scroll does not appear when viewing content by clicking any post.

Dashboard​

Changed Under the record tab in the Transaction analysis window, added the description for the mCaller data.

Project list​

Fixed In case of the list view in Project list, fixed the issue where the CPM project has not been deleted.

Application​

Fixed In the Heap memory widget under Dashboard > Application dashboard, fixed the typo in the title of the detailed view window.

Server​

Fixed In Server list > Server list, fixed the problem where the background color of the agent list does not match the actual status when searching for agents.

Log​

  • Feature In case of adding the parser in Log > Log setting > Log primary parser setting tab, added validation for reserved words when entering the search value for Log detection condition.

  • Fixed If Search or Exclusion search is selected in the dropdown menu that appears when a tag is clicked in the log list in Log > Log search, fixed the issue where any duplicate value is included in the Filter field.

Preview 1.111.2​

October 4, 2023

Database​

Instance monitoring V2​

Changed Modified to enter the string auto-completed with changed metrics by leaving the widget title blank when changing, adding, or deleting the metrics of the selected widget.

Preview 1.111.1​

September 27, 2023

Database​

Fixed In Management > Agent installation, fixed the multilingual translation bug for phrases of the XOS installation directory setting.

Preview 1.111.0​

September 27, 2023

Container​

  • Fixed In Summary or Display Detail of Dashboard > Container map, modified the guide messages according to the viewability of the Log/Event tab.

  • Fixed In Management > Namespace management, a message that indicates unavailability appears if more than 10 namespace projects are created during the trial period.
