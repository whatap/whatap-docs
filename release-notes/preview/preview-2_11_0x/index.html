Preview 2.11.0.a

노트

preview.whatap.io에서 다음 추가한 신기능과 수정한 내용을 미리 경험할 수 있습니다.

Preview 2.11.0.a4​

2024년 11월 20일

공통​

  • Fixed 사이드 메뉴를 접은 상태에서 조직을 선택하고, 즐겨찾기한 Flex 보드가 있으면, 통합 Flex 보드 메뉴를 선택할 때 조직 선택이 해제되는 문제 수정

  • Fixed 조직 내에서 알림 메시지 설정 이동 후 조직 내 조회가 정상 동작하지 않는 문제 수정

Server​

  • New 클라우드 및 엔터프라이즈 온프레미스 인프라 모니터링을 지원하는 서버 인벤토리 맵 메뉴 추가

  • Changed 서버 목록 > 서버 인벤토리 메뉴명을 서버 인벤토리로 변경

Kubernetes​

  • Feature 클러스터 > 쿠버네티스 이벤트 메뉴에 상세 보기 기능 추가(Suggestions, 인접 로그, 이벤트 타임라인)

  • Feature 클러스터 > 쿠버네티스 이벤트 메뉴에 Type, Reason 개수 표시 및 시간별 이벤트 개수 차트에 Type 구분 표시 추가

Database​

공통​

Fixed 대시보드 > 인스턴스 모니터링 메뉴의 액티브 세션 테이블에서 컬럼 변경 사항이 CSV 데이터에 반영되지 않는 문제 수정

Database V2​

Fixed 분석 > 카운트 추이 비교 메뉴의 지표 선택 화면에서 지표가 보이지 않는 문제 수정

SQL Server​

New SQL Server V2 서비스 시작

Oracle, Oracle Pro​

Changed 대시보드 > 멀티 인스턴스 모니터링 메뉴에서 Disk Usage 위젯의 크게 보기 창에서 여백에 대한 UI 변경

Browser​

Feature 통계 메뉴 하위에 IP 로드 통계 메뉴 추가

Log​

Feature 로그 > 로그 설정 메뉴에서 다음 기능 추가 및 수정

  • 로그 데이터 보존 기간을 직접 입력하여 변경할 수 있는 기능 추가

  • 로그 데이터 보존 기간의 데이터 표시 방식 수정

Preview 2.11.0.a3​

2024년 11월 14일

공통​

  • Feature 실시간 알림 목록에서 CSV 다운로드 기능 추가

  • Feature 이용 내역 > 프로젝트 유료 전환 메뉴에서 프로젝트 전체 및 개별 프로젝트의 과금 단위 요약 추가

  • Changed 계정 정보 > SSO 연동 메뉴에서 Okta에 한정된 안내 문구 변경

  • Fixed 계정 정보 > 알림 수신 설정 섹션에서 조회 에러 발생 시 알림이 종료되지 않는 현상 수정

  • Fixed 이용 내역 > 결제 정보 메뉴에서 AWS Marketplace 재구독 버튼이 활성화되지 않는 현상 수정

  • Fixed Flex 보드 메뉴에서 특수 문자를 포함한 메트릭스 지표가 통합되어 표시되는 현상 수정

Application​

  • Fixed 트레이스 분석 창의 레코드 요약 탭에서 에러 트랜잭션이 아님에도 에러 단계 필드가 노출되는 현상 수정

  • Fixed 트레이스 분석 창의 레코드 요약 탭에서 Pod명, 컨테이너 ID 필드를 별도의 그룹으로 표시하도록 수정

  • Fixed 트레이스 분석 창의 테이블 뷰 탭에서 SQL 상세 조회 시 비밀번호가 해제되지 않은 경우에도 바인드 변수 매핑 기능이 활성화되는 현상 수정

Server​

  • Changed 사용자 행위 이력이 남는 API로 교체하여 서버 목록 > 서버 인벤토리 메뉴 삭제 기능 개선

  • Fixed 서버 목록 > 서버 인벤토리 메뉴에서 커스텀 컬럼 수정 후 저장되지 않는 현상 수정

Kubernetes​

  • New 클러스터 하위에 etcd 대시보드 메뉴 추가

  • Feature 컨테이너 맵 및 통합 컨테이너 맵 메뉴에서 외부 프로젝트 에이전트가 설치된 컨테이너 식별 및 해당 프로젝트 이동 기능 추가

  • Fixed 컨테이너 개수가 많을 때 메트릭스 대시보드가 로딩 상태를 반복하는 문제 수정

Database​

  • Fixed 통계/보고서 > SQL 통계 메뉴에서 차트 영역 테두리(border) 제거

  • Fixed 대시보드 > 인스턴스 모니터링 메뉴의 메트릭스 선택 창에서 이벤트 설정 선택 시 이벤트 설정 New 메뉴로 이동하도록 수정

PostgreSQL​

Fixed 통계/보고서 > SQL 통계 메뉴에서 빈 쿼리가 조회되는 문제 수정

노트

이 수정 사항은 DBX 에이전트 1.6.7 버전 이상에서 지원합니다.

Oracle, Oracle Pro​

  • Feature 분석 > SQL 분석 메뉴의 Plan Change History 탭에서 sql_id, sql_hash_value, PLAN_HASH_VALUE로 검색할 수 있는 필터 기능 추가

  • Feature 분석 > SQL 분석 메뉴의 Plan Change History 탭에서 Plan Diff 확인 시 새 창으로 보기 추가

  • Fixed 분석 > SQL 분석 메뉴의 Plan Change History 탭에서 시간 간 발생한 데이터가 명확히 구분되도록 차트 수정

  • Fixed 트레이스 분석 창의 테이블 뷰 탭에서 SQL 스텝의 높이가 실제 영역보다 크게 출력되는 문제 수정

  • Fixed 분석 > 멀티 트랜잭션 추적 메뉴에서 선택한 프로젝트가 검색에 반영이 되지 않는 문제 수정

  • Fixed Java 플랫폼의 관리 > 에이전트 설정 메뉴에서 sigar_enabled 옵션의 기본값을 true에서 false로 변경

Kubernetes​

  • Feature 클러스터 > Ingress 목록 메뉴에서 쿠버네티스 에이전트 버전이 1.7.13 미만일 때 에이전트 업데이트 안내 화면 표시

  • Fixed 사이트맵 > 인스턴스 토폴로지 메뉴에서 컨텍스트 메뉴의 링크가 동작하지 않는 문제 수정

  • Fixed 네임스페이스 프로젝트에서 대시보드 > 컨테이너 맵 메뉴에서 컨테이너의 Stdout/Stderr 로그가 표시되지 않는 문제 수정

Database​

  • Feature 경고 알림 > 이벤트 기록 메뉴에서 이벤트 이름을 선택하면 인스턴스 모니터링 메뉴로 이동하는 기능 추가

  • Fixed 실시간 알림에서 발생한 이벤트를 선택하면 인스턴스 모니터링 메뉴로 이동하고, 이벤트 발생 시각을 기준으로 데이터를 조회하도록 수정
