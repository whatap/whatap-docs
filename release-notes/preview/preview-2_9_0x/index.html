Preview 2.9.0.a

노트

preview.whatap.io에서 다음 추가한 신기능과 수정한 내용을 미리 경험할 수 있습니다.

Preview 2.9.0.a3​

2024년 09월 12일

공통​

  • Feature 계정 관리 > 계정 정보 메뉴에서 사용자의 등록된 전화번호 삭제 기능 추가

  • Feature 실시간 알림 목록 메뉴에서 해소된 이벤트 보기 기능의 On/Off 설정 추가

  • Changed 실시간 알림 목록 메뉴에서 다음 사항 변경

    • 이벤트 일괄 해소 버튼의 아이콘 변경

    • 실시간 알림 설정 옵션의 UI 개선

    • 브라우저 알림 끄기 기능의 On/Off 토글 버튼의 위치를 실시간 알림 설정 내로 위치 변경

  • Changed 경고 알림 > 이벤트 설정 New 메뉴의 이벤트 추가 화면에서 일시 중지 옵션의 설명 중 이벤트 상태가 해소되면 추가 알림 용어를 해소된 알림으로 변경

  • Fixed 분석 > 메트릭스 조회 메뉴에서 태그 선택을 해제했을 때 가독성이 떨어지는 문제를 스타일 수정으로 개선

  • Fixed 분석 > 메트릭스 조회 에서 테이블 목록에 마우스를 오버할 때 텍스트 표기 스타일 개선

Application​

공통​

  • Feature Application 플랫폼에서 연계된 Database 플랫폼의 인스턴스 정보를 확인할 수 있는 DB 프로젝트 연계 기능 추가

    • 분석 > 히트맵 메뉴에서 DB 추적 기능을 DB 프로젝트 연계로 변경

    • 트레이스 분석 창의 테이블 뷰 탭에서 SQL 스텝에 DB 프로젝트 연계 기능 추가

    노트

    지원하는 Database 제품은 PostgreSQL, Oracle, Oracle Pro, MySQL, SQL Server, CUBRID, Altibase, Redis, MongoDB입니다.

  • Changed 트레이스 분석 창의 레코드 요약 탭에서 테이블 디자인 및 데이터 분류 방식 변경

  • Changed Database 플랫폼에서 Application 플랫폼으로 연결하는 연계 분석 기능의 테이블 디자인 변경

    노트

    이 변경 사항은 PostgreSQL, MySQL 제품에 해당합니다.

  • Fixed 인스턴스 성능 관리 > 스레드 목록/덤프 메뉴에서 화면 상단의 현황 요약에서 전체 스레드 항목에 다국어 지원 적용

  • Fixed 트레이스 분석 창에서 SQL 상세 모달 하단의 치환 변수, 바인드 변수 토글 버튼이 함께 표시될 때 디자인 요소가 제대로 나타나지 않는 문제 수정

Java​

Feature 와탭 모니터링 서비스 내 MySQL, PostgreSQL 제품 프로젝트와 실시간으로 트랜잭션을 연계 분석할 수 있는 연계 분석 기능 추가

Server​

  • New 인프라 서버 운영에 필요한 정보를 에이전트가 자동 수집 및 저장, 사용자 정의 필드를 통해 추가한 정보를 조회하고 관리할 수 있는 서버 인벤토리 메뉴 추가

    노트

    • 이 기능은 서버 목록 > 서버 인벤토리 메뉴 경로에서 확인할 수 있습니다.

    • 이 기능은 Server Linux/Windows 에이전트 v2.6.6 이상, AIX 에이전트 v1.3.4 이상, Solaris 에이전트 v1.3.4 이상에서 지원합니다.

  • Feature HP-UX 운영체제 환경을 지원하는 에이전트 설치 제공

  • Changed 실시간 알림 목록에서 알림 항목을 선택하면 이벤트 기록 메뉴로 이동하도록 변경

Kubernetes​

  • Feature 관리 > 에이전트 설치 메뉴에 에이전트 버전을 선택할 수 있는 옵션 추가 (Stable(권장) /Preview)

  • Changed 워크로드 > Pending Pod 현황 메뉴에서 노드 리소스가 Pending 원인으로 추정되는 경우 제공하는 지표를 확대하도록 수정

  • Changed 관리 > 에이전트 설치 메뉴에서 설치 방식의 기본 선택값을 YAML로 변경

  • Fixed 경고 알림 > 이벤트 설정 메뉴에서 메트릭스 이벤트 추가 시 제공하는 할당 불가능 노드 알림 템플릿의 오타 수정

Database​

공통​

  • New SQL 스크립트를 실행해 결과를 확인할 수 있는 스크립트 매니저 메뉴 추가

    노트

    이 기능은 실험실 > 스크립트 매니저 메뉴 경로에서 확인할 수 있습니다.

  • Feature 대시보드 > 멀티 인스턴스 모니터링 메뉴에서 사용자 정의 위젯에 다음 기능 추가

    • 파이 차트 또는 텍스트 차트에 병합 없음 옵션 추가

    • 멀티 인스턴스 모니터링 메뉴를 진입하면 사용자 정의 위젯에 대한 안내 메시지 창 표시

  • Changed 연계 분석 버튼 위치를 세션 상세 정보 창에서 액티브 세션 테이블로 이동

    노트

    이 변경 사항은 PostgreSQL V2, MySQL V2, Oracle V2, Oracle Pro 제품에 해당합니다.

  • Changed 연계 분석 팝업 창의 디자인 변경

    • 연계 프로젝트 창이 기본 표시하도록 수정

    • 연계 프로젝트에 대한 플랫폼 아이콘을 표시하도록 수정

  • Changed 프로젝트 생성 화면에서 CUBRID, Altibase 항목에 지원 버전 추가

CUBRID​

New CUBRID V2 서비스 시작

노트

CUBRID 모니터링 V2의 새로운 기능에 대한 자세한 내용은 다음 문서를 참조하세요.

PostgreSQL V2, MySQL V2​

Feature Application 플랫폼과 실시간 연계 분석할 수 있는 기능 추가

MySQL​

Fixed 사용자 언어가 영어일 때, 관리 > 에이전트 설치 메뉴에서 설치 내용 중 오타 수정

Oracle, Oracle Pro​

Changed 분석 > SQL 분석 메뉴에서 화면 상단의 차트 표시 형식 변경

Oracle Pro​

Fixed 관리 > 에이전트 설치 메뉴에서 에이전트 설치 내용 수정

  • Oracle 계정이 아닌 다른 OS 계정에서 설치하는 방법에 대한 설명 보완

  • ZIP 파일 다운로드 기능 삭제

Browser​

Feature 페이지 URL 경로별 로드 성능 정보에 대한 통계를 제공하는 페이지 로드 통계 메뉴 추가

노트

이 기능은 통계 > 페이지 로드 통계 메뉴 경로에서 확인할 수 있습니다.

Log​

  • Feature 로그 > 라이브 테일 및 로그 검색 메뉴에서 필터의 입력 창에 커서가 위치해 있을 때 ECS 키를 누르면 팝오버가 닫히도록 개선

  • Fixed 로그 > 라이브 테일 및 로그 검색 메뉴에서 로그 검색 태그 필터 입력 시 입력 창에 남아 있는 입력값을 제거하도록 수정

  • Fixed 로그 > 로그 검색 메뉴에서 로그 필터 정보를 표시하는 툴팁의 콘텐츠가 제대로 표시되지 않는 문제 수정

  • Fixed 로그 > 로그 설정 메뉴를 진입할 때 선택한 조직의 id가 쿼리 파라미터에서 누락되는 문제 수정

Preview 2.9.0.a2​

2024년 09월 09일

공통​

Feature 로그 관련 메뉴에서 테이블 컬럼의 너비를 조절할 수 있는 기능 추가

Kubernetes​

  • New 노드(Node)에 대한 상세 정보를 확인할 수 있는 노드 맵 메뉴 추가

    노트

    이 기능은 대시보드 > 노드 맵 메뉴 경로에서 확인할 수 있으며, 다음 상세 정보를 제공합니다.

    • 노드별 리소스 정보

    • 개별 노드의 상태(CPU, Memory 상태)

    • 노드에 대한 환경 정보

    • 노드 내부에서 실행 중인 와탭 에이전트의 상태

    • 노드 관련 메트릭스 카테고리의 정상 수집 여부

  • New 시간의 흐름에 따른 노드 현황을 확인할 수 있는 노드 타임라인 메뉴 추가

    노트

    이 기능은 클러스터 프로젝트 전용이며, 클러스터 > 노드 타임라인 메뉴 경로에서 확인할 수 있습니다.

  • Feature 클러스터 > 노드 목록 메뉴의 비교 및 상세 보기 기능 개선

    디스크, 네트워크 컬럼 그룹을 제거하고 노드를 선택해 비교 및 상세 보기에서 관련 정보를 제공합니다.

  • Feature 관리 > 쿠버네티스 에이전트 목록 메뉴의 상세 보기에서 메트릭스 데이터를 확인할 수 있는 메트릭스 탭 추가

  • Feature 클러스터> 노드 디스크 목록 메뉴에서 비교 컬럼을 추가, 비교 컬럼 항목을 선택한 디스크의 사용량 추이를 확인할 수 있는 상세 화면 제공

  • Changed 사이드 메뉴에서 분석 > 메트릭스 차트 Old 메뉴를 노출하도록 변경

  • Deprecated 클러스터 > 노드 목록 메뉴에서 이전 버전으로 보기 기능 제거

Log​

  • Changed 로그 검색 메뉴의 검색 필터 조건에 category만 넣었을 경우 로그 건수를 전체 {n}건 조회로 표기

  • Fixed 로그 설정 메뉴의 로그 파서 설정에서 '문자:' 패턴의 prefix를 등록할 수 없는 문제 수정

Preview 2.9.0.a1​

2024년 09월 05일

공통​

  • Feature 통합 Flex 보드 또는 Flex 보드 메뉴의 뷰 모드에서 그룹 위젯 레이아웃 접기 기능 추가

  • Changed 실시간 알림 목록 메뉴에서 이벤트 발생 시각을 월/일/시/분/초까지 상세 표시하도록 변경

  • Fixed 실시간 알림 메시지의 내용이 긴 경우 레이아웃이 밀려나는 문제 수정

Application​

Fixed 토폴로지 > 애플리케이션 토폴로지 메뉴에서 애플리케이션을 표시하는 요소의 아이콘이 정상 표시되지 않는 문제 수정

Database​

Changed 대시보드 > 멀티 인스턴스 모니터링 메뉴에서, Vacuum Sessions 위젯의 버튼을 통해 열리는 Vacuum 히스토리 창에서 time 컬럼의 시간을 조회된 시간으로 표시하도록 변경

Preview 2.9.0.a0​

2024년 08월 29일

공통​

  • Feature 분석 > 메트릭스 차트 New 메뉴의 지표 패널에 WhaTap Docs로 연결하는 링크 버튼 추가

  • Changed 분석 > 메트릭스 차트 New 메뉴에서 다음 사항을 수정

    • 대상 선택 옵션이 말줄임됐을 때 UI 개선

      • 새로고침 텍스트 제거

      • 전체 펼치기/접기 텍스트 좌측에 아이콘 추가

    • 위젯이 없을 때 일괄 변경 및 다운로드 버튼 비활성화

    • 대상 선택 옵션에서 검색 결과에 대한 전체 선택 및 해제 기능 추가

    • 다중 선택시 Shift key를 사용할 수 있습니다. 문구를 툴팁으로 보여주도록 수정

    • 전체 선택 버튼 툴팁을 마우스 오버하면 표시하도록 수정

    • 시간 선택 옵션에 마우스 오버했을 때 표시되는 툴팁 제거

  • Fixed 통합 Flex 보드 또는 Flex 보드 메뉴에서 위젯 버튼의 드롭다운이 불필요하게 생성되는 문제 수정

  • Fixed 그룹 이동 창에서 그룹 이름이 길어서 모달 창을 벗어나 표시되는 문제 수정

  • Fixed 그룹 삭제 창에서 그룹 아이콘이 'X'로 표시되는 문제 수정

Application​

  • Feature 인스턴스 성능 관리 > 스레드 목록/덤프 메뉴에서 선택한 에이전트가 설치된 애플리케이션의 스레드 현황(RUNNABLE, WAITING, 전체 개수)을 표시하는 기능 추가

  • Fixed 트레이스 분석 창의 트랜잭션 로그 탭에서 로그 데이터가 없으면 안내 메시지를 출력하도록 변경

    노트

    로그 관련 에이전트 옵션이 설정되어 있지 않거나 로그 데이터가 없을 경우 메시지를 확인할 수 있습니다.

  • Fixed Netstat 토폴로지 메뉴에서 Go 제품 아이콘이 정상 표시되지 않는 문제 수정

  • Fixed 트레이스 분석 창에서 SQL 문 표시 상자의 색상 수정

  • Fixed 트레이스 분석 창에서 SQL 상세 조회 시 API가 2회 호출되는 문제 수정
