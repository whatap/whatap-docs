Preview 2.1.0.a

https://preview.whatap.io에서 다음 추가한 신기능과 수정한 내용을 미리 경험할 수 있습니다.

Preview 2.1.0.a4​

2024년 01월 19일

Application​

  • Fixed 분석 > 큐브 및 성능 추이 메뉴에서 디자인 수정

Application, Kubernetes​

  • Fixed 통합 APP.그룹 토폴로지 및 애플리케이션 토폴로지 내 버튼 그룹 사이 간격 오류 수정

AWS Log​

  • 관리 > 에이전트 설치 메뉴에서 다음과 같이 수정
    • Change 설치 화면에서 S3 안내 문구 수정
    • Feature WhaTap Forwarder 설치 탭에서 2번 AWS Region 목록에 Virginia(us-east-1) 항목 추가

Preview 2.1.0.a3​

2024년 01월 19일

공통​

  • Feature 경고 알림 > 이벤트 기록 메뉴에서 다음 기능 추가
    • 이벤트 발생 시각 컬럼에 진행 중, 정비 중 발생 태그 표시기능 추가
    • 이벤트 해소 시각 컬럼 표시 옵션 추가

    이 기능은 Server 상품에 이미 적용된 기능으로 Server 상품을 제외한 모든 상품으로 확대 적용합니다.

  • Fixed 경고 알림 > 이벤트 설정 메뉴에서 다음 기능 수정
    • 메트릭스 이벤트 탭 수정 창에서 가이드 바로가기 버튼 클릭 시 언어 설정별 가이드 문서로 이동하도록 수정
    • 메트릭스 이벤트 탭 JSON 내보내기 또는 가져오기 과정에서 저장 및 목록에 추가하기 버튼 클릭 시 내보내기 또는 가져오기 창이 닫히지 않는 이슈 수정
  • Fixed 실시간 알림 목록 메뉴에서 그룹 지정 시 알림 목록이 제대로 나타나지 않는 이슈 수정
  • Changed 일부 메뉴에서 적용되지 않던 요소에 다크 테마 추가 적용
  • Fixed Safari 브라우저에서 공지사항 메뉴 내 간헐적인 목록 미표시 이슈 수정

Server​

  • Fixed 경고 알림 > 이벤트 기록 > 이벤트 항목 선택 > 이벤트 상세 메뉴에서 처리내역을 추가할 경우 작성한 사용자 계정이 등록되지 않는 문제 수정
  • Changed 보고서 메뉴의 월간 서버 보고서(상세) 템플릿 선택 화면에서 에이전트 선택 옵션 이름을 애플리케이션에서 필터로 변경

Database​

DB V2​

  • Feature 분석 > 락 트리 메뉴에서 다음 기능 추가
    • Lock Wait Sessions 위젯의 오른쪽 위에 조회 시간 단위 및 최대값 표기 태그 추가
    • 시간 조회 범위가 1달 이전인 상태에서 3시간 이내의 데이터를 조회할 경우 조회 불가 메시지 표시 기능 추가
  • Fixed 분석 > 락 트리 메뉴에서 많은 양의 락으로 인해 로직 상의 부하를 초래하여 발생하는 화이트아웃 현상 수정

Kubernetes​

  • Changed 리소스 > 마스터 메타 정보 메뉴 이름을 오브젝트 매니페스트로 변경
  • Changed 리소스 > 오브젝트 매니페스트 메뉴에서 과거 오브젝트 매니페스트를 조회할 수 있는 기능 추가

Preview 2.1.0.a1​

2024년 01월 15일

공통​

  • Changed 분석 > 메트릭스 차트 메뉴에서 전체 에이전트 선택 해제 기능 추가
  • Fixed 통계/보고서 > 통계 메뉴의 SQL 탭에서 다른 탭의 필터가 적용되는 버그 수정
  • Changed 이용내역 메뉴에서 과금 고정가 정책 변경

    와탭 요금 체계 변경에 대한 자세한 내용은 다음 링크를 참조하세요.

Application​

  • Fixed 트레이스 분석 창의 트리 뷰 탭에서 버튼을 선택하면 설정된 다른 옵션이 해제되는 현상 수정
  • Changed 트레이스 분석 창의 멀티 트랜잭션 탭에서 차트 및 트리 설정에 대한 저장 기능 추가
  • Changed 트레이스 분석 창의 멀티 트랜잭션 탭에서 프로젝트 선택 상태를 저장하는 기능 추가

Kubernetes​

  • Fixed 다음 메뉴에서 누락된 다국어 번역 추가
    • 애플리케이션 > 멀티 서비스 히트맵
    • 통계/보고서 > 보고서

Log​

  • Feature 로그 트렌드 메뉴에서 필터 목록에 없는 값을 직접 입력해 선택 가능

Preview 2.1.0.a0​

2024년 01월 11일

공통​

  • Changed 프로젝트 목록 메뉴에서 프로젝트 삭제 시 표시되는 팝업창의 UI 변경(프로젝트 이름이 잘리는 현상 개선)
  • Changed 사이드바에서 프로젝트 목록에 마우스 오버 시 표시되는 툴팁을 클릭하면 해당 프로젝트로 이동하는 현상 수정
  • Changed 휴면 상태이거나 잠금된 계정을 프로젝트 또는 그룹에 초대 시 표시되는 문구 변경
  • Changed 이용내역 > 프로젝트 유료 전환 메뉴의 유료 전환 창에서 프로젝트 이름이 긴 경우 말줄임 처리, 마우스 오버 시 툴팁으로 프로젝트 이름을 보여주도록 개선
  • Changed Flex 보드 메뉴에서 Apdex 위젯에 차트 범례 설정 추가
  • Changed Flex 보드 메뉴에서 대시보드 내보내기 기능을 통해 JSON 파일로 저장할 경우, 파일명의 특수 문자를 기존에는 제거했으나 밑줄(_)로 변경

Database​

  • Fixed 대시보드 > 인스턴스 목록 메뉴에서 정상 상태 컬럼을 선택하면 나타나는 이벤트 창의 조회 기능 오류 수정
