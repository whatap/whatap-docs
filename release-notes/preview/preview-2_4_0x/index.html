Preview 2.4.0.a

노트

preview.whatap.io에서 다음 추가한 신기능과 수정한 내용을 미리 경험할 수 있습니다.

Preview 2.4.0.a2​

2024년 04월 11일

공통​

  • Changed 프로젝트 생성 페이지에서 리전 선택 동선 변경(지역 → CSP 선택 순으로 동선 변경)

  • Fixed 사이드 메뉴의 프로젝트 목록에서 그룹/프로젝트 이름이 긴 경우 정렬이 흐트러지는 문제 수정

  • Deprecated 사이드 메뉴의 튜토리얼 버튼 제거

Application​

  • Fixed 분석 > 일자별 애플리케이션 현황 메뉴에서 데이터 조회가 완료되기 전 검색 버튼을 선택해 다시 조회가 가능한 문제 수정

  • Fixed 대시보드 > 애플리케이션 대시보드 메뉴의 금일 TPS 및 금일 사용자 위젯에서 조회 기간을 변경해도 데이터가 유지되는 문제 수정

Kubernetes​

Changed 목록형 화면에서 이동 버튼의 UI 변경

Preview 2.4.0.a1​

2024년 04월 04일

공통​

  • Feature 계정 관리 메뉴의 사용자 전화번호 섹션에서 일반 휴대전화에 국가 코드 선택 기능 추가

    노트

    한국 및 일본 국가 코드만 입력할 수 있습니다.

  • Changed 경고 알림 > 이벤트 설정 New 메뉴에서 JSON 일괄 수정 기능 이용 시 메트릭스(METRICS) 이벤트 속성의 relatedPcode 검증 로직 제거

  • Changed SMS 및 WhatsApp 알림을 사용할 수 없는 프로젝트는 경고 알림 > 이벤트 수신 설정 메뉴의 사용자별 이벤트 수신 설정 섹션에서 SMS 알림 및 WhatsApp 알림 컬럼 항목을 보이지 않도록 수정

  • Fixed 경고 알림 > 이벤트 설정 메뉴에서 메트릭스 이벤트를 JSON 에디터로 편집할 경우 select 속성을 필수값으로 검증하는 로직 제거

  • Fixed 경고 알림 > 이벤트 설정 메뉴의 복합 메트릭스 이벤트 설정에서, 최초 시뮬레이션을 실행한 다음 날짜를 변경하여 재실행하면 데이터가 정상 표기되지 않는 문제 수정

  • Fixed Flex 보드 메뉴에서 그룹 위젯에 메트릭스 타입 필터를 적용하고, 하위 위젯에서 그룹 위젯 필터 적용 옵션을 선택하면 필터가 적용되지 않는 문제 수정

Application​

  • Fixed 대시보드 > 애플리케이션 대시보드 메뉴에서 금일 TPS 및 금일 사용자 위젯의 데이터는 에이전트 필터와 상관없이 항상 전체 데이터를 조회하도록 수정

  • Changed 분석 > 성능 추이 메뉴에서 다운로드한 CSV 파일의 헤더 컬럼에 단위 추가

  • Fixed 트레이스 분석 창의 테이블 뷰 탭에서 TXT 또는 SQL 버튼을 선택해 다운로드한 파일에 기입된 시간이 UTC+9가 적용되는 않는 문제 수정

  • Fixed 통계 > 에러 메뉴에서 상세 분석 기능 이용 시 에러 건수가 일치하지 않는 문제 수정

  • Fixed 통계 > SQL 메뉴에서 상세 분석 컬럼 항목을 선택하면 발생하는 414 에러 문제 수정

  • Fixed 애플리케이션 대시보드에서 프리셋 저장 후 덮어쓰기가 되지 않는 문제 수정

Server​

  • Feature 경고 알림 > 이벤트 기록 메뉴에서 이벤트 상세 정보 화면으로 이동하면, 알림 전후 항목의 시간 설정에 따라 프로세스 차트의 시간 범위가 변경되도록 개선

  • Fixed 서버 설정 > 서버 환경 변수 메뉴를 사이드 메뉴와 사이트맵에서 제거(실행 환경 변수 메뉴로 통합)

  • Fixed 관리 > 에이전트 설치 메뉴에서 Windows의 지원 버전이 잘못 표기된 사항 수정: Windows 2008 SP2 이상 → Windows Server 2008 R2 이상

  • Fixed 서버 목록 > 서버 상세 메뉴에서 차트 아래에 라벨 항목이 많은 경우 화면 크기에 따라 레이아웃이 흔들리는 문제 수정

Kubernetes​

  • Feature 클러스터 > 오브젝트 매니페스트 메뉴에서 쿠버네티스 이벤트 연동 기능 추가

    노트

    이 기능은 마스터 에이전트 1.5.8 버전 이상에서 지원합니다.

  • Feature 경고 알림 > 이벤트 설정 메뉴에서 다음과 같이 복합 메트릭스 이벤트 템플릿 추가

    • The kube-apiserver latency over 10 second
    • The kube-apiserver response increase/decrease rate for error codes

  • Changed 다음 메뉴 경로에서 아이콘 및 버튼 스타일을 규정에 맞게 수정

    • 대시보드 > 컨테이너 맵

    • 대시보드 > 통합 컨테이너 맵

    • 클러스터 > 오브젝트 매니페스트

  • Fixed 프로젝트 생성 후 에이전트를 설치하지 않는 상태에서 액세스 키만 발급한 경우, 관리 > 쿠버네티스 에이전트 목록 메뉴에서 발생하는 화이트 아웃 현상 수정

  • Fixed 클러스터 > 노드 상세 메뉴의 CPU 코어, Memory Total 지푯값을 kube_node 메트릭스 데이터의 CapacityCpu, CapacityMemory field 값으로 변경

Database​

Database V1​

Fixed 대시보드 > 인스턴스 모니터링 메뉴에서 SQL 상세 팝업창을 열면 query 텍스트가 보이지 않는 문제 수정

Database V2​

Feature 대시보드 > 인스턴스 모니터링 메뉴에서 액티브 세션 테이블의 데이터를 1분 간격으로 조회할 수 있는 버튼 추가

Network​

Fixed 대시보드 > 네트워크 토폴로지 메뉴에서 노드의 테두리 색상이 의도와 다르게 적용되는 문제 수정

노트

Outbound 트래픽을 발생시키는 노드는 녹색으로, Inbound 트래픽을 발생시키는 노드는 회색으로 테두리를 표시합니다.

Log​

Fixed 경고 알림 > 이벤트 설정 메뉴의 로그 탭에서 실시간 로그 이벤트의 검색 값 필드에 공백을 포함할 수 있도록 수정

Preview 2.4.0.a0​

2024년 03월 28일

공통​

  • Changed Flex 보드 메뉴에서 그룹 위젯에 하위 위젯을 추가할 경우 위젯 템플릿 목록이 닫히지 않도록 변경

  • Fixed 경고 알림 > 이벤트 설정 New 메뉴에서 JSON 일괄 수정 기능이 동작하지 않는 문제 수정

Application​

Changed 관리 > 에이전트 목록 메뉴에서 비활성 에이전트 삭제 시간을 비활성 에이전트 유지 기간으로 변경

Server​

Feature 경고 알림 > 이벤트 설정 New 메뉴에 로그 파일 / 윈도우 이벤트 설정 추가

Kubernetes​

  • Changed 대시보드 > 컨테이너 맵 메뉴의 블록 위에 마우스 오버 시 나타나는 UI 수정

  • Changed 대시보드 > 컨테이너 맵 메뉴에서 지표 옵션 사용 시 단위 표기 포함하도록 수정
