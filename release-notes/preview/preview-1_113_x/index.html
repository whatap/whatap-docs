Preview 1.113.X

https://preview.whatap.io에서 다음 추가한 신기능과 수정한 내용을 미리 경험할 수 있습니다.

Preview 1.113.11​

2023년 11월 21일

공통​

  • Feature 관리 > 정비 계획 메뉴의 기능 개선

    • 신규 정비 계획 메뉴 추가, 새로운 기능의 정비 계획 기능은 신규 버전으로 보기 버튼을 선택하세요.

    • 에이전트 별로 정비 계획을 추가할 수 있는 대상 에이전트 옵션을 추가했습니다.

    • 반복 옵션을 서버 프로젝트에서도 사용할 수 있습니다.
  • Fixed 실시간 알림 목록 메뉴에서 그룹화 옵션을 활성화하면 표시되는 테이블의 시안성 개선

Application​

Fixed 대시보드 > 액티브 트랜잭션 메뉴에서 트랜잭션 목록 중 하나를 선택하면 나타나는 액티브 트랜잭션 창의 실행중 TX 버튼 제거

Server​

  • Fixed 서버 목록 > 서버 상세 메뉴에서 프로세스 테이블의 최대(max) 수치가 간헐적으로 잘못된 값이 표시되는 문제 수정
  • Changed 서버 목록 > 서버 상세 메뉴에서 프로세스 테이블의 상세 화면 조회 시 디스크 I/O (Bps) 차트에 합계 차트선을 같이 표시하도록 변경
  • Fixed 서버 목록 > 서버 목록 메뉴에서 비교하기 옵션을 활성화한 다음 비교할 서버의 체크박스를 선택하면 스크롤 위치가 초기화되는 문제 수정

Database​

공통​

Feature 대시보드 > 인스턴스 모니터링, 멀티 인스턴스 모니터링 메뉴에서 액티브 세션 테이블 접는(folding) 버튼 추가

PostgreSQL / MySQL​

Feature 사용자 편의성을 위한 UI 개선

  • 쿼리 조회 영역과 Plan 조회 영역을 상/하/좌/우로 배치를 변경할 수 있도록 개선

  • Parameter 바인드를 기본 수행하도록 변경

PostgreSQL​

  • Feature Plan 조회를 json 형식으로 조회할 수 있는 기능 추가
  • Feature 경고 알림 > 이벤트 설정 메뉴에서 메트릭스 이벤트의 템플릿 추가
  • Changed 대시보드 > 인스턴스 모니터링, 멀티 인스턴스 모니터링 메뉴에서 Default 프리셋에 저장된 액티브 세션 테이블의 컬럼 순서 변경

Oracle​

Changed 통계/보고서 > 테이블스페이스 사이즈 메뉴에서 테이블스페이스 정보를 접어서(folding) 표시하도록 수정

Kubernetes​

  • Fixed Fixed 다음 메뉴 경로에서 트랜잭션 목록 중 하나를 선택하면 나타나는 액티브 트랜잭션 창의 실행중 TX 버튼 제거
    • 대시보드 > 컨테이너 맵 > 컨테이너 블록 클릭 > 상세 보기 > Active Transaction 위젯의 차트 클릭
    • 대시보드 > 액티브 트랜잭션 메뉴에서 트랜잭션 목록의 항목 클릭
  • Deprecated 네임스페이스 프로젝트의 Pod 초기화 성능 메뉴 제거
  • Feature 네임스페이스 프로젝트의 실험실 메뉴로 Pending Pods 추가

Browser​

Feature AJAX 히트맵 메뉴 및 위젯, AJAX 분석 기능에서 사용자 편의성을 위한 UI 개선

다음 메뉴 경로에서 AJAX 히트맵 위젯의 차트 영역을 드래그하면 새로워진 AJAX 분석 창을 확인할 수 있습니다.

  • 브라우저 모니터링 대시보드 > AJAX 히트맵 위젯
  • 분석 > AJAX 히트맵

  • Feature 리소스 > 클러스터 요약 메뉴 추가

  • Feature 대시보드 > 컨테이너 맵 메뉴에서 1분간 사라진 컨테이너/Pod의 블록을 표시하는 기능 추가

  • Changed 리소스 > 노드 목록 메뉴를 다음과 같이 개편

    • 노드 정보 지표 추가(클러스터 프로젝트)

    • CSV 다운로드 옵션 추가

    • 필터 검색 기능 추가

    • 텍스트 검색 제거하고 필터 검색으로 대체

    • 시간 갱신 간격을 1분에서 10초로 변경

    • 비교 컬럼을 기본 노출로 변경하고 비교하기 버튼 제거

    • 표시 컬럼을 최대 20개로 제한

    • 사용자 편의성을 위한 UI 개선

  • Changed 리소스 > 컨테이너 목록 메뉴를 다음과 같이 개편
    • CSV 다운로드 옵션 추가

    • 비교 컬럼을 기본 노출로 변경하고 비교하기 버튼 제거

    • 표시 컬럼을 최대 20개로 제한
