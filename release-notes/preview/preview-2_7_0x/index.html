Preview 2.7.0.a

노트

preview.whatap.io에서 다음 추가한 신기능과 수정한 내용을 미리 경험할 수 있습니다.

Preview 2.7.0.a6​

2024년 07월 18일

공통​

  • Feature 사이드 메뉴의 통합 검색에서 에이전트를 IP 주소로 검색할 수 있는 기능 추가

  • Changed 보고서 메뉴의 보고서 메일 발송 예약 창에서 시간 문구를 타임존이 포함된 문구로 안내하도록 변경

    예, 04~06 → (UTC+9) 04~06

  • Changed 계정 관리 또는 경고 알림 > 이벤트 수신 설정 메뉴에서 모바일 기기 알림 테스트 시 표시되는 성공 메시지 문구 변경

    모바일 기기와 성공적으로 연결되었습니다. → 테스트 메시지를 모바일 기기로 전송 요청 하였습니다.

  • Fixed 경고 알림 > 이벤트 수신 설정 메뉴에서 사용자 본인 계정이 아닌 다른 사용자의 모바일 기기 테스트 및 삭제 버튼을 선택할 수 없도록 비활성화

  • Deprecated 경고 알림 > 이벤트 설정 메뉴의 이상치 탐지 탭에서 이벤트를 추가 또는 수정할 때 오브젝트 병합 선택 UI 제거

Application​

공통​

  • Fixed 대시보드 > 트랜잭션 맵 메뉴에서 과거 시점으로 필터를 적용한 후 실시간 조회로 변경하면 이전 조회 시점의 필터가 적용되는 문제 수정

  • Deprecated 구버전의 애플리케이션 대시보드 제거

Node.js​

Fixed 관리 > 에이전트 설치 메뉴에서 에이전트 다운로드 관련 문구 수정

Server​

Fixed 경고 알림 > 이벤트 기록 메뉴에서 검색 버튼을 선택할 때, 실제 검색 로직을 수행하는 경우에만 테이블 내부에 로딩 UI를 표시하도록 수정

Kubernetes​

  • New Kubernetes 클러스터 내에서 실행 중인 모든 서비스들의 상태와 정보를 종합적으로 제공하는 Service 목록 메뉴 추가

    노트

    이 기능은 워크로드 > Service 목록 메뉴 경로에서 확인할 수 있습니다. 서비스 구성과 현재 상태를 상세히 파악하여 클러스터 내 네트워크 구성을 이해할 수 있습니다.

  • New 노드 관점에서 디스크 모니터링을 위한 노드 디스크 목록 메뉴 추가

    노트

    • 이 기능은 클러스터 > 노드 디스크 목록 메뉴 경로에서 확인할 수 있습니다.

    • 이 기능은 쿠버네티스 마스터 에이전트 1.7.8 버전 이상에서 지원합니다.

  • Feature 대시보드 > 컨테이너 맵 메뉴의 요약 보기에서 대상 정보 탭에 컨테이너 요약 분석 기능 추가

    노트

    이 기능은 한국어로만 제공됩니다.

  • Feature 대시보드 > 컨테이너 맵 메뉴의 요약 보기에서 대상 정보 및 리소스 관계 탭에 Service 정보 추가

  • Feature 컨테이너 > 컨테이너 목록 메뉴에서 다음 컬럼의 각 항목에 관련 목록 메뉴로 이동할 수 있는 버튼 추가

    • Deployment → Deployment 목록
    • Pod → Pod 목록
    • Image, ImageId → 컨테이너 이미지
    • ApplicationName → 컨테이너 애플리케이션 목록

    각 메뉴로 이동하면 선택한 항목의 데이터로 필터링된 조회 결과를 확인할 수 있습니다.

  • Feature 워크로드 > 컨테이너 애플리케이션 목록 메뉴에서 다음 컬럼의 각 항목에 관련 목록 메뉴로 이동할 수 있는 버튼 추가

    • CotainerId → 컨테이너 목록
    • ContainerName → 컨테이너 목록
    • Pod → Pod 목록
    • Node → 노드 목록

    각 메뉴로 이동하면 선택한 항목의 데이터로 필터링된 조회 결과를 확인할 수 있습니다.

  • Feature 워크로드 > 컨테이너 애플리케이션 목록 메뉴의 리소스 그룹에 Image 컬럼 추가

  • Feature 클러스터 > 쿠버네티스 이벤트 메뉴의 이벤트 목록 테이블에 이벤트 최초 발생 시간과 이벤트 발생 건수 컬럼 추가

  • Feature 트레이스 분석 창의 레코드 요약 탭에서 Container Id, Pod Name 정보를 제공하고, 해당 정보를 클릭하면 각각 컨테이너 목록, 파드 목록 페이지로 이동하는 기능 추가

  • Feature 쿠버네티스 에이전트가 비활성화되면 사용자가 바로 인지할 수 있도록 화면 상단에 에이전트 상태 표시 추가

  • Fixed 대시보드 > 컨테이너 맵 메뉴에 처음 진입하여 Pod 뷰 또는 컨테이너 뷰로 전환할 때 일시적으로 데이터가 잘못 출력되는 문제 수정

Database​

공통​

  • Changed 플랜 조회를 위해 쿼리를 선택하면 표시되는 SQL 상세 창에서 Full Scan 부분을 빨간색으로 표시하도록 변경

  • Changed 다음 메뉴 경로의 화면 상단 옵션바에 데이터베이스 이름을 표시하도록 수정

    • 대시보드 > 인스턴스 모니터링
    • 대시보드 > 슬로우 쿼리
    • 분석 > 락 트리
    • 분석 > 데드락
    • 분석 > 카운트 추이 비교
    • 경고 알림 > 이벤트 설정 > 데이터베이스

  • Changed 대시보드 > 인스턴스 모니터링 메뉴에서 플랜 조회 시 표시되는 오브젝트 상세 창에 에러 처리 추가

PostgreSQL / MySQL​

Feature 대시보드 > 멀티 인스턴스 모니터링 메뉴에 DB가 기동한 지 얼마나 시간이 지났는지 확인할 수 있는 Uptime 위젯 추가

Oracle V2 / Oracle Pro​

  • Feature 대시보드 > 멀티 인스턴스 모니터링 메뉴에 Global Lock을 확인할 수 있는 Main Waits 위젯 추가

    노트

    수집한 데이터베이스 카테고리의 지표를 통해 특정 이벤트의 총 대기 횟수를 조회하고, 각 에이전트(인스턴스)의 가장 최근 데이터를 추출합니다. 이를 통해 특정 시간에 대한 데이터베이스 대기 이벤트 상태를 파악할 수 있습니다.

  • Changed 대시보드 > 멀티 인스턴스 모니터링 메뉴의 Interconnect Bandwidth 위젯에서 interconnect_nic 및 interconnect_speed 지푯값을 표시하도록 디자인 수정

  • Fixed 통계/보고서 > 테이블스페이스 사이즈 메뉴에서 테이블스페이스 사이즈 상세 섹션의 항목을 선택하면 표시되는 세그먼트 사이즈 (TOP 50) 섹션에서 발생하는 다음의 문제 수정

    • 화면 가장 하단에 dbf 파일을 클릭하면 데이터가 변경되지 않도록 수정

    • 인스턴스 옵션을 변경하면 데이터가 표시되지 않는 문제 수정

    • 테이블스페이스 사이즈 상세 섹션의 항목으로 마우스 포인터를 가져가면 배경색이 변경되도록 수정

Feature​

Fixed KAKFA 프로젝트 진입 시 발생하는 화이트아웃 문제 수정

URL​

Changed URL 목록 안내 문구 디자인 변경

Log​

Fixed 로그 > 로그 트렌드 메뉴에서 조회 시간을 변경했을 때 조회 데이터가 없으면 이전 데이터가 남아 있는 문제 수정
