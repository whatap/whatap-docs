Preview 1.107.X

노트

preview.whatap.io에서 다음 추가한 신기능과 수정한 내용을 미리 경험할 수 있습니다.

Preview 1.107.17​

2023년 8월 21일

공통​

대시보드​

트랜잭션 분석 창에서 다음의 내용을 수정합니다.

  • Change 레코드 요약 탭의 표시 데이터에 원본 URL, URL/TX 추가

  • Change 트랜잭션 목록의 컬럼 설정에 원본 URL, URL/TX 추가

  • Change 테이블 뷰, 트리 뷰 탭의 트랜잭션 요약 정보에 클라이언트 IP 추가

  • Change 레코드 요약 탭에서 에이전트 명 클릭 시 환경 변수 메뉴의 해당 에이전트 정보 표시

  • Change 테이블 뷰 탭의 컬럼 영문 번역 수정

    • Memory Accumulation (byte) → MemAllocAcc (byte)

    • CPU Accumulation (ms) → CpuTimeAcc (ms)

  • Change 컬럼 선택 기능의 UI의 스타일 수정

Application​

  • Fixed 통계/보고서 > 통계 > 트랜잭션 메뉴에서 데이터 조회 범위가 5분 이하인 경우 조회되는 데이터의 검색 범위를 5초에서 5분으로 수정

  • Fixed 애플리케이션 > 인스턴스 성능 분석 메뉴의 에이전트 상태 탭에서 빈 데이터에 대한 예외 처리 추가

2023년 8월 17일

  • Fixed 서버 목록 메뉴에서 디스크, 네트워크 필터에 글자 입력 시 화면 깜빡임 현상 개선

New 네트워크 성능 모니터링(Network Performance Monitoring) 서비스를 오픈합니다. 많은 이용 부탁드립니다! 해당 서비스는 https://preview.whatap.io에서 이용할 수 있습니다. 자세한 내용은 다음 문서를 참조하세요.

인스턴스 모니터링 V1​

  • Change 테이블 컬럼 선택 모달 UI 변경

  • Change 프로젝트 멤버 중 수정 권한이 있는 멤버만 세션 킬 버튼이 노출되도록 변경

  • Feature 라인 차트에 데이터 최대값 표시 적용

멀티 인스턴스 모니터링 V2​

  • Feature 위젯별 메트릭스 정보를 나타내는 인포 아이콘 추가

  • Feature 제품 환경별(RDS, XOS) 탭 분리

  • Change XOS Disk Usage 위젯 내 테이블의 컬럼 중 Mount Source → Mount Point로 변경(최신 에이전트 업데이트 필요)

데이터베이스 사이즈 V2​

Feature CSV 다운로드 추가

PG SQL 통계​

Feature 조회 기준 선택 옵션 추가(temp blocks read, temp blocks written)

Log​

  • Fixed 로그 검색기 메뉴에서 pcode 태그 클릭 시 드롭다운 메뉴를 노출하지 않도록 수정
