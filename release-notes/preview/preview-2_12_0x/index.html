Preview 2.12.0.a

노트

preview.whatap.io에서 다음 추가한 신기능과 수정한 내용을 미리 경험할 수 있습니다.

Preview 2.12.0.a4​

2024년 12월 17일

공통​

  • Feature Flex 보드 또는 통합 Flex 보드 메뉴의 알림 목록 위젯에 다음 기능 추가

    • 해소된 이벤트 보기 옵션 추가

    • 대시보드에 설정된 시간과 별개로 조회 시간을 선택할 수 있는 옵션 추가

  • Fixed 계정 정보 메뉴에서 알림 수신 언어 설정이 'ja'로 표시되는 문제 수정

  • Fixed 일부 메뉴에서 화면 상단의 제목이 누락되는 문제 수정

Application​

Feature 경고 알림 > 이벤트 설정 New 메뉴에서 애플리케이션의 에이전트 중복 알림 옵션 추가

Server​

  • Feature 서버 목록 > 서버 인벤토리 메뉴에서 CSV 파일을 업로드해 서버 목록의 사용자 정의 컬럼을 교체하는 기능 추가

  • Changed 대시보드 > 서버 인벤토리 맵 메뉴에서 그룹화 옵션 선택 도구의 초기화 방식 변경

    Clear 버튼 클릭 방식에서 목록에서 미설정 항목을 선택하는 방식으로 변경

  • Changed 서버 목록 > 서버 인벤토리 메뉴의 필터 옵션을 서버 인벤토리 맵 메뉴의 필터 옵션과 동일한 UI로 교체

  • Fixed 서버 목록 > 서버 상세 메뉴의 에이전트 명 컬럼에 특정 운영체제의 아이콘이 표시되지 않는 문제 수정

Kubernetes​

Fixed 대시보드 > 외부 호출 상태 메뉴에서 실시간 모드를 일시 정지한 경우에도 실시간으로 데이터를 조회하는 문제 수정

Database​

공통​

  • Fixed 다음 메뉴 경로의 액티브 세션 테이블에 1분 범위가 초과되는 데이터가 표시되는 문제 수정

    • 대시보드 > 인스턴스 모니터링
    • 분석 > 카운트 추이 비교

Database V2​

  • Feature APM 프로젝트의 트레이스 분석 창에서 복사한 바인드 변수 값을 연계 데이터베이스 정보 창에서 붙여 넣는 버튼 추가

    적용 플랫폼: PostgreSQL, Oracle, Oracle Pro, MySQL

  • Changed 연계 데이터베이스 정보 창에서 인스턴스 모니터링 메뉴로 이동할 때 시작 시간, 종료 시간, 세션 시간을 쿼리 파라미터로 포함하도록 변경

    노트

    적용 플랫폼: PostgreSQL, Oracle, Oracle Pro, MySQL

  • Fixed 대시보드 > 인스턴스 모니터링 메뉴에서 과거 데이터를 조회할 때 세션 상세 정보 창을 열면 Full Text를 조회하지 못하는 문제 수정

    노트

    적용 플랫폼: SQL Server

  • Fixed 분석 > 카운트 추이 비교 메뉴에서 위젯의 툴팁에 [기준]/[비교] 정보 및 인스턴스 정보 추가

Oracle Pro​

  • Feature 대시보드 > 인스턴스 모니터링 메뉴의 SQL 상세 창에 액티브 세션 정보 추가

  • Feature 분석 > SQL 분석 메뉴의 필터 옵션에 object_name 조건 추가

  • Fixed 관리 > 에이전트 설치 메뉴의 계정 생성 단계에서 히든 파라미터 조회 권한을 부여하는 명령어의 잘못 표기된 내용 수정

MySQL​

Fixed 통계/보고서 > MYSQL SQL 통계 메뉴의 SQL 상세 창에서 hash 값이 undefined로 표기되는 문제 수정

Browser​

Feature 리소스별 성능 데이터를 시각적으로 제공하며, 로드 시간, 리소스 크기, 유형별 성능 추이 등을 분석할 수 있는 리소스 로드 통계 메뉴 추가

노트

이 기능은 통계 > 리소스 로드 통계 메뉴 경로에서 확인할 수 있습니다.

Preview 2.12.0.a3​

2024년 12월 12일

공통​

  • Feature 개별 메뉴에서 시간 선택자(타임 셀렉터)의 컴포넌트 교체

    • 키보드를 이용해 날짜 및 시간을 변경할 수 있습니다.

    • 날짜 및 시간 형식의 텍스트를 복사해 붙여넣기 방식으로 변경할 수 있습니다.

    시간 선택자 이용 방법에 대한 자세한 내용은 다음 문서를 참조하세요.

  • Feature 시간 선택자(타임 셀렉터)의 캘린더 팝오버 하단에 가이드 문서로 이동할 수 있는 링크 추가

  • Feature 경고 알림 > 이벤트 수신 설정 메뉴에서 알림 메시지를 AI를 통해 사용자 친화적인 내용으로 전송할 수 있는 Smart 메시지 (AI) 옵션 추가

  • Feature 경고 알림 > 이벤트 설정 New 메뉴에서 해소된 알림 옵션에 대한 설명 추가

  • Changed 경고 알림 > 이벤트 설정 > 메트릭스 탭의 이벤트 추가 또는 수정 화면에서, 이벤트 발생 조건 옵션의 선택 입력을 선택했을 때 모든 필드에 ${ } 적용하도록 변경

    • 단, 필드명에 { 또는 }가 포함되면 적용하지 않습니다.

    • 이벤트를 새로 추가하거나 수정하는 경우에만 적용합니다.

    • 테이블 뷰 > SQL 상세 모달에서 파라미터 바인딩이 되지 않는 문제 수정

    • 테이블 뷰 > 액티브 스택 모달에서 액티브 스택이 보이지 않는 문제 수정

  • Fixed 서버 목록 > 서버 목록 메뉴의 컬럼 설정에서 초기화 버튼을 클릭하면 전체 컬럼 선택이 해제되는 문제 수정

Preview 2.12.0.a2​

2024년 12월 05일

공통​

  • Feature 분석 > 메트릭스 차트 New 메뉴에서 다음 기능 추가

    • 대시보드의 위젯 범례 표시 여부에 따라 다른 UI 노출

      On: 위젯에 범례 패널이 고정된 형태

      Off: 위젯에 범례 패널이 드롭다운으로 제공되는 형태

    • 차트 라인이 많을 때, 그래프의 컬러가 흐려지지 않도록 처리

  • Changed 경고 알림 > 이벤트 설정 메뉴의 이벤트 추가 또는 수정 화면에서 수신 테스트 기능을 실행하면, 이벤트 수신 태그 선택 사항을 적용하여 테스트하도록 수정

    노트

    웹 브라우저에서 발생하는 알림은 이벤트 수신 태그와 무관합니다.

Application​

  • Feature 트레이스 분석 창의 SQL 요약, HTTP Call 요약, 메소드 요약, 액티브 스택 탭에서 텍스트 다운로드 기능 추가

  • Feature 트레이스 분석 창의 테이블 뷰 탭에서 SQL 상세 정보 확인 시 바인드 변수 복사 기능 추가

    노트

    • 이 기능은 연계 데이터베이스 정보에서 사용할 수 있습니다.

    • 비밀번호가 해제되어 바인드변수 값이 보일 때 사용할 수 있습니다.

Kubernetes​

  • Feature 다음 메뉴 경로에서 로그 개인정보 조회 권한 있는 사용자에게 개인정보 비식별화 기능 추가: 개인정보를 표시하거나 숨기는 기능

    • 대시보드 > 컨테이너 맵 > 로그 > 로그 검색
    • 대시보드 > 컨테이너 맵 > 이벤트

    • 클러스터 > etcd 대시보드
    • 클러스터 > 쿠버네티스 이벤트

  • Changed 네임스페이스 프로젝트의 관리 > 프로젝트 관리 메뉴에서 프로젝트 이름을 수정할 수 없도록 변경

  • Fixed 클러스터 > etcd 대시보드 메뉴에서 차트 인덱스에 불필요한 정보 값 제거

Database​

공통​

  • Feature 실험실 > 스크립트 매니저 메뉴에서 사용자가 SQL 실행 시 사용자 행위 메뉴에서 확인할 수 있는 기능 추가
