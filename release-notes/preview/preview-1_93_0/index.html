Preview 1.93.0

preview.whatap.io 페이지를 통해서 아래 추가한 신기능을 미리 경험해볼 수 있습니다.

2022-11-24​

공통​

Change 이벤트 설정 메뉴와 이벤트 수신 설정 메뉴에서 이벤트 수신 태그 기능 사용 중 수신 태그를 삭제하면, 알림 수신이 불가능한 버그가 발생해 다음 메뉴에서 태그 삭제 기능 임시 제한

  • 애플리케이션 상품군: 메트릭스(복합 메트릭스 포함), 히트맵 패턴, 이상치 탐지, 로그
  • 서버 상품군: 메트릭스(복합 메트릭스 포함), 이상치 탐지, 로그
  • 쿠버네티스 상품군: 메트릭스(복합 메트릭스 포함), 히트맵 패턴, 이상치 탐지, 로그
  • 데이터베이스 상품군: 메트릭스(복합 메트릭스 포함), 이상치 탐지

2022-11-16​

공통​

  • Fixed 서버 상품군 프로젝트 > 서버 목록 메뉴에서 데이터 로딩 시, '설치된 에이전트가 없음' 문구 삭제, 로딩 스피너를 표시하도록 수정

  • Fixed 초기 화면에서 그룹 선택 후 프로젝트 목록으로 이동 시, 딜레이가 발생하면 로딩 스피너를 표시하도록 수정

경고 알림​

Feature 프로젝트 > 경고 알림 > 이벤트 설정 > 히트맵 패턴 메뉴에서 각 항목 하단에 이벤트 수신 태그를 추가할 수 있는 기능 추가

로그​

프로젝트 > 로그 메뉴에서 다음의 기능을 추가 및 수정합니다.

  • Feature 설정 메뉴에서 로그 장기 보관 통계 패널 추가

  • Feature 설정 메뉴의 로그 Parser 설정에서 4xx, 5xx 상태 코드 파서, 상태 코드 성공률 파서 추가

  • Fixed 라이브 Tail 메뉴에서 콘텐츠 보기 기능 오작동 수정

  • Fixed 라이브 Tail 메뉴에서 콘텐츠 필터링(하이라이트) 기능 오작동 수정

일자별 애플리케이션 현황/성능추이​

Fixed 쿠버네티스 상품군 프로젝트에서 아래 메뉴의 조회 버튼 작동 이상 수정

  • 애플리케이션 > 일자별 애플리케이션 현황
  • 애플리케이션 > 성능 추이

에이전트 설치​

Fixed 쿠버네티스 상품군 프로젝트 > 관리 > 에이전트 설치 > Application Agent 메뉴에서 Java 에이전트 설치 jar 파일명 변경(whatap.agent-X.Y.Z.jar → whatap.agent.kube.jar)

정비 계획​

프로젝트 > 관리 > 정비 계획 메뉴에서 다음의 기능 추가 및 수정합니다.

  • Feature 정비 계획 공지를 화면 상단에 표시하는 기능 추가
  • Change 정비 계획 UI 개선
    • 반복 옵션 UI 텍스트 변경 : 일간/주간/월간/년간 → 매일/매주/매월/매년
    • 정비 계획 테이블 컬럼 순서 변경
    • 정비 계획 추가/수정 모달 UI 개선
    • 정비 계획 추가 시 한번만 기본 선택하도록 수정
    • 정비 계획 안내 문구 및 위치 수정
  • Fixed 서버 상품 정비 계획 삭제 불가 수정

메트릭스 차트​

Change 쿠버네티스 상품군 프로젝트 > 분석 > 메트릭스 차트 메뉴에서 일정 크기 이상의 태그 목록 조회 시 발생하는 미응답 상태 수정 : 태그 목록 조회 API 변경 search → tags

히트맵(트랜잭션 목록)​

Change 쿠버네티스 상품군 프로젝트 > 애플리케이션 > 히트맵 메뉴에서 컨테이너 ID, Pod 명 컬럼을 추가할 수 있도록 수정

Flex 보드​

프로젝트 > 대시보드 > Flex 보드 메뉴 또는 초기 화면 > Flex 보드 메뉴에서 다음의 기능 추가 및 수정합니다

  • Feature 정비 계획 위젯 추가
  • Feature 로그 장기 보관 통계 위젯 추가
  • Feature 대시보드 목록 화면에서 오른쪽 위에 정렬 기능 추가
  • Change 대시보드 즐겨찾기 아이콘 버튼에 안내 툴팁 추가
  • Change ECS 상품의 대시보드 템플릿에서 위젯의 가로 너비 조정

프로젝트 유료전환​

이용 내역 > 프로젝트 유료 전환 메뉴에서 다음의 기능 추가합니다.

  • Feature 유료 전환 시점이 7일 미만인 프로젝트의 경우 유료전환 취소 기능 추가

  • Feature 유료 전환시점으로부터 7일 이후인 프로젝트의 경우 프로젝트 삭제 기능 추가

청구서​

Change 사이트 관리 > 결제 관리 > 청구서 메뉴에서 월간 사용 Session 컬럼 추가

사용량​

Change 이용 내역 > 사용량, 사이트 관리 > 결제 관리 > 사용량 메뉴에서 범용 미터링 정보(MtypeDetail)를 사용량 > 상세 항목에 추가해 표시하도록 기능 추가
