와탭에 오신 것을 환영합니다

와탭 모니터링 서비스는 다양한 플랫폼을 아우르는 통합 모니터링을 제공합니다. 수많은 디지털 소스에서 데이터를 수집해 사용자가 시스템과 시스템 개선 방안을 파악할 수 있도록 돕습니다.

  • 사용자는 와탭과 함께 다양한 데이터를 통합할 수 있습니다. 와탭 에이전트와 확장 도구를 활용해 데이터를 수집하고 하나의 UI로 사용자만의 디지털 생태계를 구축할 수 있습니다.

  • 사용자는 와탭과 함께 데이터를 분석할 수 있습니다. 수집한 데이터를 기반으로 사용자 정의형 대시보드 및 각종 차트 또는 강력한 쿼리 언어를 활용해 장애의 근본 원인을 찾고 시스템을 최적화할 수 있습니다.

  • 사용자는 와탭과 함께 시스템 장애에 신속하게 대응할 수 있습니다. 장애 징후를 먼저 감지하고 장애가 발생하기 전에 사용자에게 알릴 수 있습니다.

와탭 Docs 안내​

와탭 Docs는 사용자가 와탭 모니터링 서비스를 효과적으로 사용할 수 있도록 돕고자 합니다. 단계별 에이전트 설치부터 대시보드, 사용자 정의형 대시보드인 Flex 보드 설정, 각 성능 지표 정보, 알림 설정까지 와탭의 다양한 기능 활용 방법 및 권장 지침을 제공합니다.

Docs 홈 화면에서 상품 바로가기와 상세 검색 등 기술 문서 구성을 확인할 수 있습니다.

  • WhaTap Docs 아이콘을 선택하면 Docs 홈 화면으로 이동합니다.

  • 고정된 상단 메뉴를 통해 사용자는 와탭 Docs 내 상품별 안내 문서, 관리 기능 문서, Open API 문 및 FAQ, Learning 문서를 자유롭게 이동할 수 있습니다.

  • 와탭 Docs는 다국어 안내를 지원합니다. 원하는 언어를 상단 언어 옵션에서 선택하세요.

  • 아이콘을 선택하면 와탭 Docs 페이지의 테마를 검은 배경으로 적용합니다.

  • 아이콘을 선택하면 와탭 Docs 페이지의 테마를 밝은 배경으로 적용합니다. 기본 적용 테마는 검은 배경입니다.

  • 상단 메뉴의 오른쪽에 있는 검색() 버튼을 통해 와탭 Docs 페이지에서 원하는 정보를 검색할 수 있습니다.

    • 입력창에 키워드를 입력하고 화살표 키를 이용해 원하는 페이지 위에서 Enter 키를 누르면 해당 페이지로 이동합니다.

    • 입력창을 확인 후 esc 키를 누르면 입력창을 닫습니다.

  • 와탭 Docs는 서비스 화면 UI 텍스트를 파란색으로 표기합니다.

    예, 홈 화면 > 프로젝트 선택 > 대시보드 > 애플리케이션 대시보드

  • 와탭 Docs는 경로 및 파일명을 녹색으로 표기합니다.

    예, /usr/whatap/php/whatap.ini

  • 와탭 Docs에서 사용하는 첨언의 종류는 다음과 같습니다.

    노트

    서비스 및 와탭 Docs 내 기능과 관련한 참고 사항을 안내합니다.

    팁

    기능 외 일반적인 참고 사항을 안내합니다.

    주의

    주의를 요하는 참고 사항을 안내합니다.

    위험

    사용자의 재산상 손해와 관련한 참고 사항을 안내합니다.

상세 검색​

기술 문서 홈 화면에서 상세 검색 기능을 사용할 수 있습니다. 검색 창에 텍스트를 입력하면 연관된 검색 결과 목록이 표시되며, Enter 키를 누르면 상세 검색 화면으로 이동합니다. 상세 검색에서는 제품 카테고리를 기준으로 검색 결과를 필터링하여 원하는 정보를 더욱 손쉽게 찾을 수 있습니다.

노트

검색창 하단에서 와탭 Docs의 추천 문서 및 새로운 문서를 확인할 수 있습니다.

시작하기​

와탭 모니터링 서비스를 소개하고 플랫폼 경험과 기능 이해에 도움이 되도록 서비스 흐름을 안내합니다.

빠른 시작하기​

와탭 모니터링 서비스를 처음 접하는 신규 사용자를 위해 회원 가입부터 프로젝트 생성, 에이전트 설치, 대시보드 조회하기 등 단계별로 서비스 사용 흐름의 개요를 안내합니다. 원하는 주제 선택 시 빠른 시작하기 문서 내 해당 안내로 이동합니다.

시작하기 전에​

  • 와탭 모니터링 서비스에 대해 알고 싶다면 와탭 모니터링 서비스 안내 문서를 참조하세요.

  • 지원 환경 문서에서 와탭 모니터링 서비스 상품별 지원 환경 개요를 확인할 수 있습니다.

  • 와탭 모니터링 서비스 이용이 처음이라면 빠른 시작하기 문서를 참조하세요.

  • 와탭 모니터링 서비스 모바일 앱 이용에 대해서 모바일 앱 문서를 참조하세요.

  • 홈 화면(UI) 안내 문서는 서비스 초기 화면 이용 방법을 안내합니다.

상품​

상품 메뉴는 사용자가 선택한 상품에 대해 설치부터 설정 및 활용 지침까지 통합 안내 문서 바로가기를 제공합니다.

  • 에이전트 설치 전 지원 환경을 확인하세요.

  • 와탭 모니터링 서비스를 이용하기 위한 에이전트 설치하기 문서를 참조하세요.

  • 설치 점검 사항을 확인하고 설치 중 문제가 발생한다면 설치 문제 해결 문서를 참조하세요.

  • 에이전트 설정 옵션과 업데이트부터 삭제 등의 관리 방법을 알아보세요.

  • 대시보드, Flex 보드, 보고서, 경고 알림 설정 방법 등을 알아보세요.

각 상품별 문서는 Docs 홈 화면 내 상품 바로가기와 혹은 고정된 상단의 상품별 문서 바로가기 메뉴에서 확인할 수 있습니다.

관리​

관리 메뉴는 와탭 모니터링 서비스의 계정, 프로젝트, 그룹 등을 관리하는 방법과 결제 방법을 안내합니다.

  • 보안이 강화된 와탭의 멀티팩터 인증 방식을 확인하세요.

  • 와탭 모니터링 서비스의 모니터링 단위인 프로젝트의 계층 구조 및 관리 방법에 대해 알아보세요.

  • 와탭 모니터링 서비스의 정비 계획 관리 방법에 대해 알아보세요.

  • 와탭 모니터링 서비스의 무료 이용 기간 15일이 지났다면 결제 정보를 확인하고 유료 서비스 전환에 대해 알아보세요.

What's New​

새로운 기능​

새로운 기능 메뉴에서 와탭 모니터링 서비스의 타임라인 기준 서비스 개요와 분기별 안내를 제공합니다. 서비스 개요의 경우 타임라인 기준으로 신규 기능을 한 눈에 조회할 수 있으며 상시 업데이트됩니다. 분기 안내의 경우 와탭 상품별 신규 기능 및 주목할 만한 변경 사항과 서비스 및 에이전트 개요를 분기별로 요약해 제공합니다.

노트

분기 안내의 경우 현재 국문 안내만을 제공합니다.

릴리스 노트​

릴리스 노트 메뉴에서 와탭 모니터링 서비스의 최신 기능 및 향상된 기능, 중요한 보안 패치에 관한 정보를 확인할 수 있습니다. 사용자 편의를 위해 서비스 릴리스 노트와 에이전트별 릴리스 노트를 구분하여 따로 제공합니다.

Reference​

참조 문서 메뉴는 다음과 같은 주요 자료를 소개합니다.

  • Open API: 모니터링 서비스의 Open API 활용 방법에 대한 안내

  • 라이선스: 와탭이 사용하고 있는 오픈 소스 라이선스와 관련 라이브러리 안내

  • 소프트웨어 프록시: 와탭 소프트웨어 프록시의 설치 및 구성 방법에 대한 안내

  • FAQ: 서비스 이용 중 자주 묻는 질문과 해결 방안 안내

노트

FAQ 문서는 지속적으로 업데이트해 제공할 예정입니다.

Learning​

학습하기 문서 메뉴는 와탭 모니터링 서비스를 효율적으로 이용할 수 있도록 기본 안내 문서를 보완한 대시보드 분석과 활용 사례 안내 및 MXQL 가이드와 주요 개념 및 용어에 대한 자료집을 제공합니다. 관련 내용을 지속적으로 업데이트해 제공할 예정입니다.

노트

안내 외에 와탭 Docs와 관련한 궁금한 사항이나 추가 내용 요청 등의 피드백은 docs@whatap.io로 보내주세요.
