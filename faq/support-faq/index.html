고객지원

와탭 모니터링 서비스 이용 중 고객지원과 관련한 FAQ 문서입니다.

채팅 문의​

채팅 문의 아이콘이 표시되지 않아요.

사용자가 사용 중인 웹 브라우저의 광고 차단 플러그인 또는 설정에 따라 Crisp 관련 자바스크립트 로딩이 차단되어 화면 오른쪽 위에 채팅 문의 아이콘이 표시되지 않을 수 있습니다.

웨일 브라우저​

웨일 브라우저를 사용 중이라면 다음의 안내에 따라 웹 브라우저 옵션을 설정하세요.

  1. 주소 표시줄에서 자물쇠 버튼을 선택하세요.

  2. 추적 방지된 항목이 있는지 확인하세요. 차단된 추적기를 선택하세요.

  3. 추적 방지된 콘텐츠가 client.crisp.chat인지 확인하세요.

  4. 이전 단계로 돌아와 추적 방지 옵션을 해제하세요.

  5. 웹 브라우저에서 화면을 새로고침하세요.

화면 오른쪽 위에 채팅 문의 아이콘이 표시된 것을 확인할 수 있습니다.

크롬 브라우저​

크롬 브라우저를 사용 중이라면 크롬에서 광고 차단 플러그인 확인이 필요합니다. 다음 링크에서 조치 방법을 참고하여 설정을 변경하세요.

회원 가입​

회원 가입 인증 메일이 오지 않아요.

인증 메일은 기본적으로 no-reply@whatap.io에서 발송됩니다.

  • 정상적으로 발송된 경우, 메일 특성상 추적이 어렵습니다. 반송된 이력이나 메일 발송 여부는 확인할 수 있습니다. 인증 메일이 스팸 메일함으로 수신되지 않았는지 또는 외부망에서 수신되도록 허용되어 있는지 확인이 필요합니다.

  • 인증 메일을 받지 못하였거나 인증 과정에 문제가 있는 경우, 로그인에 필요한 이메일 또는 인증 해제가 필요한 이메일에서 se@whatap.io로 계정 인증 해제 요청 메일을 전달합니다. 이메일 인증을 위해 본인 인증 절차를 진행한 후, 인증을 해제하고 안내해 드립니다.



로그인 계정 변경​

로그인 계정을 변경하고 싶어요.

현재는 계정 변경 기능을 별도로 제공하지 않습니다. SaaS 환경에서는 실제 사용하실 이메일로 직접 회원 가입 및 이메일 인증을 완료한 후 계정이 활성화됩니다. 이메일 변경을 원하실 경우 새로운 이메일로 회원 가입을 진행해 주세요.
